[D&TT] 幽默搞笑

昨日，TT媽要他幫忙把計算機拿過來，媽媽要算一下過年紅包錢夠不夠啊～～ :~~

媽媽指的是下面這個她從日本買的粉色可愛計算機….

TT煞有介事，不動聲色地… 走過去拿了個東西，回來媽媽身邊，手上拿著下面這張紙，眼神賊溜溜地，忍住笑意，故意看別處… 把媽媽笑得噴飯了～～

這是TT之前自己畫的計算機，他最近越來越會畫一些生活裡的事物了：電腦、時鐘、人物…. 而且還具有一定的相似度….

上面這一齣搞笑，也是他越來越樂在其中的活動… 有時候雖然會過度了，例如在趕時間時還在開玩笑（可能就被罵了），不過，其實小孩搞笑，如果適度且不傷害自己與別人，應該算是幽默感的展現，也可以說是智力發育的特徵。父母應該感到高興哪～～～

話說，看過一些兒童發育的書，確實，人類的幽默感，算是比較高級的功能，必須對環境裡的人事物的邏輯了解足夠，又能夠在當下反應出夠幽默的作為，需求條件是比較高的。

啊？什麼？我在自誇小孩聰明…. 沒有啦，沒有啦，他很多不行的地方。例如最近還在學注音符號，連認那37個注音符號，都還會錯不少個啊！

笨得很，笨得很，光會搞笑有啥用哪！進演藝圈啊！？

去去去，該幹啥幹啥去～～～

[社區閒聊] Google翻譯

歲末將屆，總是有一種想總結的念頭。

最近似乎總是嘆氣居多。可能是過年前這些紛紛擾擾，真的令人不勝唏噓吧？也可能是在這裡太久了，看過太多好想法不斷提出，卻總極少見成功的實現，甚或，成功的卻無法持續….

有些確實跟大環境的不好有關，但我覺得更多的是，在中心集權與社區自治的紛擾之中，我們始終還沒能找出更有效率的一條路吧～～

一方面，我一直認為，Steem這條鏈真的有很多很多的潛力可以發揮啊，我感覺還好多有創意的想法還等在未知之中，等待人們去發掘去實踐呢！另一方面，創意的想法，有時又很依賴於鏈的改革改善，提供基本建設機制，一方面可以讓許多想法更容易建置，另一方面，還能刺激出新的可能性…. 只可惜，這條路，也是漫漫長路….

我自己說過，許多人也說過，不要抱怨，不要說嘴，想改變，就去做。

我完全同意。所以，我只是在感概，不抱怨誰。包括我自己沒有辦法全心投入這裡，這一方面當然也是因為這裡收入是遠不足以讓我放下現實的工作的，跌價之後，更是如此了… 否則我還真想全職來好好做些什麼呢！（在此很敬佩那些堅持在這裡開發的朋友們！）

最近不是公司老是提要探索廣告之路來提高收入嗎？這讓我想起歷史上的一件事。

約莫是一年多前了吧？忘記那時Dan走了沒。我記得Ned到挪威還是北歐哪一國去演講吧，有視頻紀錄的。那次，也是Busy這個項目首次從Ned口中洩漏出來… 我還記得，那時候，大家都很關注這個年輕的區塊鏈項目又有什麼新變化了…. 當他說出Busy會是一個基金會的時候，我還想半天說為何要搞個非營利的基金會呢？隨著時間過去….. Busy，確實完成了另一個UI，確實也是他們開發出Steemconnect…. 但，似乎就是這樣了？Busy的介面，好像也都沒怎麼進步更新，還常常出問題…. 更別說什麼偉大的事了….. 基金會呢？不知道幹嘛去了… 項目的CEO也很少參與社區了….

但我要說的是那次，Ned提出了一個Attention Economy的概念。“不就廣告嗎”我心裡第一時間想… “不！一定是更厲害的概念，由區塊鏈驅動的創新！”我心裡辯護著….

然後，等到我兩鬢發白…. 等到公司開始在Steemit.com上放Google的廣告….

注意力經濟我的屁股….

誰！？誰在罵人哪！沒水準….. （p.s. Google翻譯真厲害啊 XD）

[社區閒聊] 大屁屁雷達

pixabay

這件事可能比較敏感一點，所以我就不用英文了。

前一陣子，那真狼先生非常熱情地把Steem上許多App都整理出來，做了個美觀的網站，我還推薦過給大家。後來，也許是在他努力之下，有一些關於Dapp的網站 把咱們這些SteemApps都給列上去了，感覺是個很好的宣傳與曝光嘗試，大家都蠻開心的… 這也是因為我們確實有一些還不錯的Apps啊… 讓更多人知道是好事…

不過，其實我心中一直有個疑惑，其實，更早之前，約莫四個月前我也提出過，還給了一些想法：

從大屁屁到阿屁屁的

上面一篇文章我引述的Dapp定義，感覺太複雜了點。剛好最近我聽說了一個網站是專門列Dapp的，據說很出名，所以我就去看了一下：DappRadar.com，然後沒看見任何SteemApps（能被列成為主要平台的有三個，分別是Ethereum, EOS跟TRON），所以我就去找了他們關於Dapp的定義，指向這一篇文章：

WTF is an Ethereum Dapp ?

我覺得這一篇的簡單定義很乾淨清楚：

DApp can be a website, mobile app or something else. As long as the code responsible for main app logic is running on blockchain, it is a DApp.

嗯，我終於感覺比較能抓住這東西的重點了….

不過，這樣一來，其實Steem上面，應該是永遠也不能有Dapps了吧？至少現在以及甚至SMT想像中出來之後？

我去看了一遍，那真狼整理的目前列入Steem的Dapp網站，約有三四個…. 稍微看過之後，與上面那個網站比較，不得說，DappRadar.com感覺是最專業的。而他們並未認可SteemApps是Dapp（當然也有可能是還沒有機會…）。看過他們的定義之後，我覺得真的去提交也應該是沒有機會的。

其實，如果你夠了解區塊鏈世界，就會知道，DPOS系的東西，常常被批評不夠中心化。甚至有份報告（不過，這份報告的動機有被質疑就是了），說EOS並不是一個區塊鏈（為何針對EOS？因為人家市值夠大啊….）。不過，EOS至少有這領域喜歡的智能合約這東西。Steem其實，既有EOS的不夠去中心化批評（只是不夠大，人家不太注意），又沒有智能合約（描述事實，不是說我們應該要有）… 說真的，其實在區塊鏈界應該是一直比較處於邊緣的地位。這些話，在這裡不容易看到，但我心裡，其實一直是有些擔憂的….

常常在想，Steem的優勢，會不會只是先行者這一項而已？如果現在重新來過，要打造一個全新的Steem，應該用什麼架構？跟現在又會有多少不同呢？太難了…. 畢竟我也不是很懂區塊鏈的結構啊….

說遠了，這話題可以下次再聊。

溝通

pixabay

電腦時代已經開啟好幾十年了，慢慢的，人類社會在這方面的發展已經越來越進化，數位化越來越深入與全面…

於是我們開始習慣越來越即時，越來越快速與豐富，越來越全面與多元的….. 溝通。

昨晚，我剛好要去赴一個約，到目的地之後，要走好一段長路。清朗的夜裡，走路很舒服，但我想起一些煩人的事，頓時覺得蠻煩悶的…. 突然之間，我感到手機一下震動，時間還早，我就打開看一下。原來是一位朋友幫我點讚的Partiko提示。

點讚其實也不稀奇了，每天總有上百個讚。不過，這位朋友，大家一起在社區也算是相識要兩年了，雖然沒有真的見過面，卻感覺對對方性格等應該蠻熟悉的，要是見面，鐵定可以聊上三天三夜吧！呵呵～～然後，我也知道這位朋友是親自在點讚，不是自動化的。所以，在那當下，突然感受到一陣溫暖，呵呵，實在是太神奇了～～跨越半個地球之外，一種來自朋友的關懷的感覺….

這就是現代科技的展現吧？

話說，今天早上來參加一個會議。會議嘛，可能大家都參加過吧，就是那個樣子。大家都習慣了，開場總會有漫長的長官/領導/主辦單位等的致詞，然後這些致詞通常，不會帶有太多，甚至任何，有用的訊息。不過，似乎總是要這樣過場… （如果去算集體時間成本，其實是很驚人的）

每一次在會議的這一段開頭時，我總是在想，在哪一個年代，大家會開始發現這樣真的好浪費時間？開始把這些過場，縮短成類似一人一分鐘之類的？甚至完全不要？

會議，其實，究其所以，就是要促成一群人來“溝通”，只是通常是幾位專業人士，向為數多的聽眾單向式傳遞訊息。然而，會議的安排，其實通常不會把“有效率溝通“放在很重要的位子。往往，會議看起來很專業，委託專業公關公司來辦，在高級場地，用多重投影，製作精美名牌，甚至用QR-code給你會議資料…. 會場裡可能很多媒體記者，很多人在攝影，看起來，這會議實在太重要的樣子…

不過，很可能會議過後，真正能留在人心，真正能被開會者吸收的訊息，也都很有限….

電腦之間的資料交換，明確而快速。人與人之間的資訊交換，卻充滿人味：多重考量、名實不符、形式主義….

電腦越來越快速，越來越聰明，人類，應該要及早醒悟，及早進化，否則…. 就要被電腦統治了啊…..
