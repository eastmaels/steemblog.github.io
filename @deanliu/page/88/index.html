[聲林點評] 不在乎今晚出爐的冠軍....

我之前在[聲林點將]系列介紹過兩位參賽歌手：李佳歡，跟孫肇谷。

經過這兩週，沒想到的是，兩位都被淘汰了，止步於七強的決賽門外。

對我來說，這幾乎就是不用繼續收看這比賽的意思了。說真的，這次的選手都很強，真的是近年來少見的優秀… 我才會推薦給大家看。不過，我覺得其實在節目製作上，真的有蠻多明顯的問題或瑕疵，這是隨著收看久了就可以看出來的。儘管我還是肯定有人願意製作這麼高水準的選秀節目，但是該批評的地方，應該也是要說一說啦～～

我對於李佳歡與孫肇谷的“偏愛”，其實並不算是太特別。我看網路上一面倒在批評說不看了，多半都是因為這兩位被淘汰了…. 在我心目中，這兩位拿冠軍我都覺得可以，沒拿也可以，但沒進七強，那是不可思議的….

這裡面的選手，能夠讓我真的被感動到的，其實只有這兩位。其他的選手，還是有很厲害的，例如許莉潔的唱功，例如林孟辰的Rap，還有李友廷的創作…. 這幾位我很清楚是我個人偏好不同，但他們很不錯我是認可的。

13強變成7強的過程，是跟外來猛獸PK。這就造成不公平問題，有的人的對手很強，有的比較弱，而配對是怎麼做的，又不是現場抽籤，似乎也有黑幕問題… 有幾位勝出，我覺得對手太弱當然贏啊…. 然後，又來了三位單日評審，一下子，結果都可能充滿變數….

當然，李跟孫，輸了就輸了，沒話說。我也覺得就單次較量，說冤枉也太過。只是從整個比賽看來，真的是安排有問題。李佳歡確實，可能選錯歌了。我其實覺得她唱得不差，甚至很不錯，只是那首歌這樣唱是比較怪一點沒錯，導致四位評審都給待定（表示沒有輸）。然後其他人有些贏得很輕鬆，所以李佳歡就GG了。

她輸了後，我怒上Youtube找她唱其他的歌來聽，彌補一下以後暫時聽不到的缺憾。然後聽到這一首，覺得，要是她選這一首【寵兒】，那一定過關的！實在是超好聽的啦！！！

https://www.youtube.com/watch?v=UmnvqgcSWcs

真的唱的太棒了～～ 我很久沒有聽到聲音如此有吸引力的歌手了。然後就被淘汰了。

老蕭說：不要來這裡做實驗。然後被罵翻了，因為他老是期待歌手們“自我突破”。自我突破，不就是要實驗嗎？

在這樣簡短的賽制裡，其實根本不應該“自我突破”，保持穩定，頂多略作變化讓人有一點新鮮感，那就夠了…

今晚九點，就要播出冠軍賽了。其實，冠軍是誰，我已經知道了。網路上已經有人爆雷，畢竟有現場觀眾，沒法封鎖消息，又不是做LIVE。但是，看或不看，也不是太在乎了～～

既然都捧了，那我就在這區塊鏈上撂下一句話來：這一批選手，未來五年後的發展，李佳歡一定會超越所有參賽者的！

關於工作者提案系統（WPS）的討論


pixabay

今天一早看到社區大大佬，發出一篇文章提案在Steem上創建一個工作者提案系統（Worker Proposal System, WPS）：

Proposing A Worker Proposal System For Steem

又是一篇社區討論熱烈的文章（最近常常有），想當然耳，這也是一篇對於最近關於社區治理解決方案的一個提議。請願意讀英文的人可以好好去看看文章跟下面的討論甚至辯論。

什麼是工作者提案系統（WPS）？

其實，之前，我也就是聽過跟大致猜測這是什麼而已。基本上就是創建一套區塊鏈上可以自動化管理（尤其是取得資金資助）各項社區提案之開發專案。你可以想像成是一種去中心化的社區自發支持的代碼開發方式，相對於例如公司的中心化式的開發推動工作。

所以，有技術能力的人可以自由提案（提案上架收取一定門檻金額避免濫用），再交由社區以投見證人的方式來投票選出支持的方案，給予資金進行開發。開發完了是否就能被見證人接受成為區塊鏈的未來更新版本（假設提案是要更新區塊鏈代碼的話）？基本上由於投票群體相同，接受度的問題應該不嚴重（雖然不是一定就成）。

聽起來不錯嗎？其實都需要很多細節設計，才有可能真的發揮功用吧！更別說Steem上的SP分布情況，基本上也是大戶們決定的。不過，這可能還是個不錯的思考方向，讓Steem開發工作可以既有中心化的Steemit Inc來做，也有社區渴望已久的自發推動部分。

其實，WPS，如上引述文所說，最早來自於Bitshares，甚至EOS也有這樣的機制在進行中。因此，Steem有人提出這樣的提議，根本就是來晚了一點啊～～ 因為都是BM的作品哪！不過，EOS如果算是正在進行中，那麼Bitshares的WPS運作的如何？當然 @blocktrades認為是很好的，也有其他不同意見。我想說，CN區也不少人比我懂Bitshares的運作，所以歡迎各路大神可以給出你的評論，我自己是根本沒有去投過，畢竟我的bts超少的，只是去玩過借錢爆倉的遊戲…. >_<~

另外，其實WPS概念不難懂，我有找到一篇中文解釋各位可以參考：

【一五一十】大白话解释EOS的工作者提案系统(WPS)是什么

不過，最重要的莫過於：實踐是檢驗真理的唯一標準嘛，到底實踐上的經驗如何，可能比概念上的理解更為重要了～～

另外，跟各位比較可能有關的是，上述引用文章內討論到WPS的資助資金來源，作者舉例說可以增加1%通膨來支付，不過這個例子被大部分人否決，我個人也不支持。多半意見是把獎勵池拿來用就好，也就是從作者身上的75%來支應，例如挪個10%過去之類的～～

不用不開心，都還在討論階段。畢竟總是要有人支付開發費用啊！比起各種項目先自己投入再尋求代理或是發代幣募資，WPS是用事前付費方式，確保對於社區或區塊鏈很重要的開發工作可以能被確定完成。確實是有其必要性，這是我個人意見。（例如就有人說一旦 WPS建置完成，第一個要提案的就是修復SBD的錨定機制，聽起來蠻好的～～）

大家說說你的看法跟意見？

PS. 不用再抱怨小魚沒話語權或是公司”應該”如何如何了…區塊鏈嘛，你懂的。

STEEM 在漲，為何帖子收入沒有明顯成長？

最近STEEM持續上漲走勢，我趁機來跟比較資淺的中小魚兒們，教育一下Steem經濟裡的基礎觀念。資深的朋友，可以直接跳過，你們應該都很清楚了…

其實這標題剛好是自己前兩天突然心裡有的疑問，但是只有一下子，我馬上想起了自己思考的盲點，所以就沒有問題了，覺得蠻適合跟新手或是還沒太清楚的中魚兒們分享一下～～

所以，問題是：『STEEM 在漲，為何帖子收入沒有明顯成長？』

在解說之前，要先解釋一下問題，可能真的很新的人，也不清楚問題的來源。問問題要先了解問題，所以，要先告訴你們，STEEM幣價上漲，會讓還沒結算的帖子獎勵顯示金額也跟著上漲，這是透過見證人對區塊鏈的STEEM餵價（feed price）所產生的效果（細節是過去3.5天餵價的平均數，所以上漲後還有一點延遲效果）

這樣知道問題的基礎背景了吧？

所以，當我看到，過去幾天（超過2, 3天就應該有感），STEEM漲價，不過，雖然帖子獎勵金額沒有下降，甚至有微幅上升“感覺”，不過，總覺得遠低於STEEM漲幅，這不是很奇怪嗎？

通常，會跟上去的，即使有一點延遲，也至少會有個六七成效果才對啊！當然，還有一些影響因素，例如參與點讚的SP總量突然有變化，那也是理論上可能影響的（獎勵池金額不變，參與指導分配的人變多，你的一份就會縮水），不過，通常這部分應該是穩定不會變化太大的….

所以，怎麼回事呢？

圖片過後告訴你，馬上回來！

【解答】

首先，有一個非常重要的基礎觀念，要請各位牢記，這是很多人到現在還容易忘記的觀念：『Steem經濟體系裡的”$“，通常指的是SBD。』

所謂通常是指Steemit.com上面，所有$，都是SBD，不是美元。文章獎勵顯示是，你錢包裡的也是，即便他這麼顯示：

Estimated Account Value
The estimated value is based on an average value of Steem in US dollars.

別被騙了（雖然不是故意要騙，但都沒改也是有點投機吧），這裡指明US dollars，其實也還是SBD。

不過，其他Steem UI，例如Busy.org裡頭的帳戶估值，就是採用美元計價的，不信你去比較一下就知道。但是文章獎勵因為算是系統參數，都還是以SBD的$為單位的。

你可能看過白皮書或什麼解釋的，要說這兩者差不多，所以差異不大。喔不，兄弟，差異可大了。SBD很久以前就脫錨了，原本是向上脫錨（甚至到達1 SBD = $20）這種大家開心的事，向下脫錨少見，頂多到$0.8（其實以穩定幣來說也是蠻嚴重的了），但是去年HF20之後，向下脫錨就成家常便飯了…

所以，這是第二點，『SBD，不再是Steem體系裡的穩定幣。』也就是說，SBD等於廢了，等於是STEEM的替代品，幾乎是沒有其他作用的（歡迎，期待有人反駁我）。SBD的價值，在STEEM價格太低時（此波漲價前的好一段時間），就是由SBD的量跟STEEM價格決定的，跟美元一點關係沒有。當STEEM價格高時，或許還能變成美元替代品，不過HF20之後，還沒太長時間能測試這件事，有待觀察。但是如果下檔風險沒法避，那麼其實也沒太大用處了。

至於為何是HF20之後產生這現象，是因為調整提高了SBD市值佔STEEM總市值比例上限的緣故。O哥應該有帖子說過這件事，我一時找不到， @oflyhigh要不要給一下，幫你打打廣告？

好。知道了上面兩個觀念之後，就可以解題啦！

為什麼『STEEM 在漲，為何帖子收入沒有明顯成長？』

簡單的答案就是：以SBD為單位的收入沒有明顯成長，但是其實以美元來看其實是有的。

原因就是SBD也跟著STEEM上漲了，所以當單位也變大時，自然顯示金額增加的幅度就降低了。

我們可以簡單計算一下，從CMC截了兩張圖：

我們就從起漲的Jan 16算到Jan 22好了。這段期間，STEEM美元價值漲了48.5%，SBD則是漲了22.4%。

因此，分母漲多，分子漲少，整體來說，實質增加應該是21.3%。所以，我原先錯誤預期的接近五成的漲幅，其實應該只有兩成，似乎比較符合”略有延遲的略漲”感覺了！（注意喔！實質，即美元，的收入，還是成長近五成的！）

不知道這樣說完，你是不是懂了？如果沒搞懂也沒關係，只要記住前兩個重要觀念就好囉！^_^

[幣市閒聊] 不確定因素消除之後....

昨天下午四點，韓國交易所Bithumb開放STEEM/SBD充提，我四點不到就坐在電腦前看好戲，順便也看看手上STEEM要不要賣一賣….

不少朋友都說開放後可能會小漲後，開始大跌… 四點之前半小時，確實盤比較弱勢，但是開放之後，並沒有明顯賣壓，反而還大漲，STEEM/BTC價格，從10000左右，漲到11280，40分鐘之內，又創新高到11855…. 之後回落，但都是高檔整理，好幾個小時，都沒有跌回起漲點….

我在一些高點賣光了手上的流動STEEM，不過，都賣得略早了… 然後，後來覺得，應該別賣完的，我總覺得還會有高點… >_<

大家可能會認為STINC都把那麼多籌碼給挪到交易所了，未來還持續會大量搬過去，怎麼可能不砸盤？就算他們不砸盤，光是各方的預期，就足夠砸不少了吧？這當然也有道理。只是，我總覺得，市場總是充滿爾虞我詐，第一是我認為公司真要砸盤的可能性不高，除了他們例行性的賣幣之外（純粹個人觀點）；第二是，就算要出貨，也要先拉高再慢慢神不知鬼不覺地出啊！
