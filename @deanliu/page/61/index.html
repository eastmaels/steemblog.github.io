繼續接著談：閒談 SCOT Tokens投資（上）

給定那四點SCOT Tokens投資的特性之後，我們就可以談一些相應的可能情境與做法。

消息與基本面需要花費精力去研究，去了解。去新手村微信群，那裡很多消息跟別人的經驗。去平台的Discord看看消息，有時候，最新訊息都在那裡先出來。比較認真的，可以從SE跟SCOT設定的參數，去理解平台發行團隊的想法跟設定。平台上的許多文章，也可以每天大致瀏覽，尤其是創始團隊的文章跟公告，如果是外文，可以用Google翻譯試著多少掌握…

其次，是了解其他參與者的行為動態。例如從SE的Richlist跟Transactions可以看到持股大戶情況，後者可以看到隨時這個代幣的交易情況，交叉比對就可以看到一些大戶動向… 這是籌碼面的掌握。透過籌碼面，還能夠間接推論參與者們的心態看法等等（例如拿到幣就拋，還是鎖？）

綜合以上的訊息後，做出自己參與這平台，投資這代幣的基本姿勢：擼？還是攢？還是賺審查？還是聯繫創始團隊談合作或深度參與？抓住基本調子，才能決定下一步。

下一步就是如何買賣幣？或是增加持有幣（不一定透過買），或是獲利了結，或是來回操作。凡是透過市場的，就得要善用一個好朋友：『時間』。這是因為，市場很初期，籌碼很少，掛單很少。要低接，你得掛一個合理的買單，不能低到永遠買不到，太高的買單可能也會讓你後悔掛太高… 這得設定好一個價格區間。賣單也是一樣。

有時候，沒用到的幣，掛一些誇張的高價，會有意想不到的橫財！但這可能也跟消息面有關，你有好的內線消息嗎？還是都是流言？最好的是，你透過經驗與判斷，感覺團隊會拉盤… 拉到多少，那又是得自己判斷了…

砸盤也是。有時候，砸盤的是意外拿到幣的群眾，這時候，掛低價買單就會有驚喜….

市場的拉與砸是短期策略，長期，還是得看好大趨勢，做好自己的設定方向。這樣，才能夠長短配合，日起有功。

說了很多，我自己也沒能做到多少。呵呵～～～～

願大家都比我說的更厲害。

結束。

To the Satellite!! ^_^

老實說，要談這題目，村長跟米高以及一些非常厲害的CN區朋友們比我更有資格…

不過，一來反正是閒談，我也玩一些，自然有自己的感想。二來，我會少談具體的細節，而是談一些這個過程的特點等等。

那為什麼談SCOT，而不談其他Steem Engine Tokens呢？主要是SCOT Tokens的變化多，要注意的面向多，因此特別容易讓人手忙腳亂的。尤其是，如果你同時要投資運作好幾個平台跟帳號的話…

對了，這裡的投資是廣義的，投入金錢，或是不投入金錢但投入時間參與也算，反正都是為了最後的報酬。

先談談SCOT Tokens投資的特點。

第一，市場都很Thin，意思是，買單賣單都不是很滿，很夠，很密… 你想買時，最後成交價（市價）可能買不到了，或是量很少。想賣時亦然。

第二，消息面很重。一個消息出來，馬上買單湧入，價格拉幾倍都可能，或是反之亦然。消息如何被你知道？就算是文章公告，也分成先看到跟後看到。但，消息有時候會不會是用來出貨？這也是得判斷的。

第三，基本面很不明確。新興平台嘛～～其實，通常重點是團隊。表面上的主題什麼的，頂多有兩三成重要而已，團隊的能力跟意志才是最重要的。偏偏，這很難得知，但可以在這早期過程慢慢觀察團隊的作法，自己可以試著有個基本的勾勒。

第四，SCOT的基本運作得要熟悉。有幣該賣還是鎖？該高價出低價接嗎？該投入挖礦還是賺審查？寫作跟審查該怎麼平衡？抱團作戰是否更划算？

哎呀，晚了，累了。那就下集繼續說吧～～～ ^_^

剛剛過午夜十二點，溫布敦的中央球場才空了出來，就是在這裡，前一場，球王Djokovic打敗了對手晉級決賽，等待這一場的勝者。

台北週五的夜晚，才剛剛幫溫布敦另一場女雙準決賽賽事加完油，台灣選手謝淑薇與搭擋勝出，進入女雙決賽。這個網球的盛宴之夜，太過豐盛，豐盛得讓人不敢相信，夜都深了，經典中的經典中的經典賽事，竟然才剛剛要開始。

我一直到開打前，都還覺得實在不敢相信，真的是這兩位，這十多年來叱吒網壇的這兩位，繼續對戰。雙方開賽前，都沒太多激情。畢竟，這樣的戲碼已經發生的太多次。只是這兩年少一點，但雙方都已經多麽習慣彼此在對方生命中的存在…

然後是熱身練球，鏡頭照著雙方，Nadal，頭髮稀疏了… Federer，臉上皺紋都稍稍加深了。歲月，顯然是留下些痕跡了。電視前的我，何嘗又不是呢？

Federer先發球。再怎麼不可置信的比賽，終究會開始。

然後，不可思議地，兩人像是根本沒有老過一樣的對戰，發球，移動，接球，反手，正手，殺球，截擊…. Oh My God，他們就是以前的他們！！！

也對，否則怎麼能在此時，還站在溫布敦的中央球場對戰廝殺呢？

兩人發球都超強，Ace連發，迅速保發，比賽快節奏的進行。在進入搶七之前，Nadal一度差點被Fed破發，但Nadal是何許人也？強心臟保住了發球局。

搶七大戰，是典型兩強選手的宿命。前面12局就像是沒發生過，一切就都在這搶七求之中決定。

Fed一度看起來有點恍神，Nadal非常強，但後來，Fed越來越穩健，以7:3拿下第一盤。

呼～～～ 他們兩人以往的弱點似乎都更有所改進，成為更為全面的超級賽亞人網球手了！只是特性不同，比賽很有看頭哪！

Federer : Nadal = 1 : 0

五戰三勝，賽事就要繼續。

對我來說，誰輸誰贏，都已經不重要了。能讓我再看到這一盤，跟著全球各個世代的網球迷一起見證，真的，真的，可以說嘴一輩子了。

突然很感動，這兩位選手身上所散發出的那種追求極致的精神力量。

他們早就不缺錢了，早就不缺名了，又有錢，又早已確定名留歷史，但他們仍在這裡，這不是只有這一天，這兩週賽事… 這需要每天每天的練習與精進，在他們繼續往前的人生裡…. 很感動，他們兩位，身上散發出一樣的力量。一位已經夠讓人心生敬佩，兩位，更是互相激盪，淬煉出人類運動史上少見的鼓舞人心的典範。

這才是這一場賽事，我們真正應該看的東西吧～～～～

到底該不該去睡覺呢？

傍晚看到這一則@aggroed的新公告，我也resteem了：

Steem good news: Steem Engine starts work on NFTs

對我來說，對我這個曾經想過要發幣的人來說，這確確實實是一個非常棒的消息啊！！！

我曾經想過為什麼SE上沒有non-fungible tokens (NFT)的設計，後來覺得也很正常，畢竟多數幣都是fungible的。再加上，我自己其實也對於NFT沒有多少實際操作經驗，只是知道這概念，所以，後來也覺得SE沒有根本是很正常的事，是我自己想太多了….

沒想到，SE今天就宣布要開始做這件事了！預計八月底完成。真是好期待啊～～～

那麼，我就稍微來說一下NFT為什麼可能具有更開闊的商業應用。

Fungible tokens就是大家很熟悉的這些crypto，幾乎99.9%都是。那有什麼知名的NFT例子呢？我能想到的就是Ethereum上面的CryptoKitties！

NFT就是：每一顆都是不一樣的。更精確地說，他們都是同一”類”，但是在這一類下，每一”顆”都不一樣。例如，撲克牌每一張都是撲克牌，但每一張都不一樣。

所以，交易不再是數量而已，而是每一顆NFT的集合。

舉個例子，我可以發一種幣，叫做CN名人幣，然後把CN區裡面的大家，都對應一顆：劉美女因此只會有一顆，具有收藏價值。村長，O哥，小冰，阿柔…. 都只有一顆。

所以，當有人跟我說你可以發劉美女幣時，其實發一堆劉美女幣有點怪，邏輯有點不通。但是上面的舉例，就變成有道理點。

當然，我發一套CN名人幣，他也可以發一套STEEM名人幣，裡頭同樣可以有劉美女幣。哪個代表我？嘿嘿，那就看什麼叫做代表，這裡頭都是商業操作的空間了～～～ （例如我授權公布我屬於誰…. XD 可能市價會高一點）

這樣一來，NFT就可以根據有的很多Token結合，可以玩的花樣，哎呀～～～～真的是海闊天空天空了啦！

這也難怪，大佬在文章最後喊了一句意味深長的話：

Hope you’re picking up some ENG at these prices keeping in mind that you might not see this again for a looooong time.

話可是他說的，不是我喔！^_^
