[146號檔案] 擁有

前一篇我寫上癮了，就是 @joythewanderer說的「意識流」寫作。 其實，之前我看某圓圈哥可以閒扯淡時就想要東施一下，但圓圈哥說是閒扯，卻還是寫得太不扯了，我也一直琢磨該怎麼笑貧好… 直到上一篇之後，靈感到就開了這個 [146號檔案] 系列，怎麼樣？這名字挺不錯吧？呵呵…

這就用來記錄我想寫想說，卻又沒把握或沒時間去把這些想法都給確認好，因此就只能停留在我的小筆記本或是腦海裡，常常這類東西太多了，實在可惜。所以，給自己一個藉口「意識流」，反正似乎就不用負責任，可以讓那個龜毛的完美主義者，改為專注在胡吹這新任務上，而不計較是否這樣寫不夠有組織之類的… 接招吧，看倌們~~~

擁有。

你要怎麼「擁有」些什麼呢？

你擁有誰的愛？不不不，不是這個，這是修辭學，我今天要論理的。擁有愛就是有人愛的意思而已。

你擁有你的美腿嗎？嗯，這也怪怪的，畢竟腿可以視為是「你」的一部分。今天別太血腥了，還扯什麼可以斷腿的…

典型的擁有，在目前世界，似乎可以分為「記名擁有」與「不記名擁有」。前者諸如汽車、房產、信用卡等等；後者則包括更多，任何你認為是你的但不記名的物品：你點的晚餐、你的衣服、床底下的金條、你家裡大部分的物品…

似乎沒太複雜？還可以更複雜。寫上你名字的杯子，你的手機，似乎是介於之間，看似是你的，但真有疑慮的話，你得去證明才行。

那你是「如何」擁有這些物品的呢？

假設存在一個人，存在一個物。這個「物」要如何「屬於」這個人，從而被「擁有」呢？

答案是：不能。

「擁有」是一個社會現象，需要多人才能成立（要多少人？意識還在流…）不知道。

總之，人擁有物的意思是，他人不得支配或使用該物品（先不考慮多人共有的情況）。

https://steemitimages.com/DQmePgfHdmWJztTyfG7JSrRgRjDNxe2zk2UCU49yEaST4gw/animal-1300138_640.png

讓我們先回到遠古時代，人類的原始狀態。

誰能擁有什麼？

似乎這詞不適用了。洞穴前擺著一把做好的石斧，你做的。你總不能整天看著它，哪天誰把它帶走，你也沒辦法。甚至你看著它，但是另一個兇惡的大塊頭來拿，估計你也不敢作聲。

但你可以藏起來，只有你知道在哪裡。這叫做「藏起來」，不太算是擁有。

….

野豬吃多了之後，人類變聰明一點了… 部落出現了，人類開始可以溝通，開始抱團生活。

部落裡，誰的石斧是誰的，誰的老婆是誰的，誰打的老虎皮是誰的… 開始可以有規則，也就是，開始可能「擁有」了。由於遵守這些規則有利於群體，所以大家會集體擁護它，你是兇惡大塊頭也沒用，不然你就是走人，離開這個部落。

只是，你要是帶著你的老虎皮到隔壁部落，人家可不認帳了，頭目喜歡一把搶去你在人地盤上大概不敢吭聲？畢竟那只是你家的規矩。所以，擁有的範圍，僅止於規矩的範圍。

兩部落要是真為了那張老虎皮開戰，或許最終會合為一部落，從而有一套新規矩出現。

到這裡，不知道你是否想起區塊鏈的「共識」呢？

https://steemitimages.com/DQmPCNJ1tC7d5HLodUR2sf2HVBg5qwpQNeHsPdtbtungVyH/lady-justice-2388500_640.jpg

人類開始種植，變得比較溫和（？），開始把這些規矩形成文字，甚至演變成法律體系。「權利」的概念出現，其中包括了「擁有」的權利，也就是所有權，或是產權等詞。

其實，本質上就是當年部落裡的規矩共識，只是後來的文明不斷演進，規矩本身的產生都開始有規矩，不斷適應人類複雜社會的變化。

直到今天，我們常常把「擁有」當成一種理所當然的概念，常常容易忽略了其實這概念本身就具有一種「社會共識」的隱含過程…

擁有，真的並不那麼天然的。其實是很人造的。

假設我跟蘇三跟李四去吃飯，蘇三先到了，我後到，李四還沒到。我看蘇三桌上擺了一個金戒子，我就把戒子拿走，蘇三甚至看到了，說你幹嘛拿我戒子？我說，你在說什麼啊，這戒子是我剛剛買的啊！蘇三肯定不可置信覺得我瘋了，大聲要跟我拿回去。

李四來了，看兩朋友吵架，問題是他也不知道戒子誰的啊？只要我夠會演，敢吵與不要臉，李四也無法作主。

就算李四知道是蘇三的好了，我要是跟李四串通了，分他點錢，那也可以變我的。但如果還有廖五、陳六、張七… 人多了，我要全串通就難了… 社會共識就會發揮，我大概就會打消念頭了…

但正常情況下，沒有人這麼做。其一是從小我們內建「擁有」觀念且被教導要捍衛它，其二是就算你沒有觀念，社會強制的會給你懲罰如法律審判等…

https://steemitimages.com/DQmQPVKmvgPR26Gqwnj1PgfJnN61aC4WEbz15Br9N3XbyGo/rings-2634929_640.png

只是，社會共識只能及於社會能夠處理的邊界。

餐廳裡有監視器可以看，人多處有目擊證人可以作證，分析指紋或是購買證明等，都是間接證據而可以證明的。

前面說的記名擁有，那沒問題。都有中心化的處裡機構去負責確認誰擁有什麼。你只要證明你是你就好。

不記名擁有，除非很容易透過其他間接證據可以確認，否則通常是你就是把它放在你可以掌握（傘拿手上），或是別人不能獲得的地方（家裡普通物品）。這種擁有，其實，還是跟遠古時代的「藏起來」沒太大差別就是了…

https://steemitimages.com/DQmfWMoZMmaZQ2MXZFwa2pjA8ojXXwXYrXkDMRMV2MohhiJ/keyboard-621830_640.jpg

說了這麼多… 其實是從我們如何擁有crypto assets開始146的…

前方有阻塞，今日先流到這裡。

區塊鏈真的很有趣，它讓你開始思考很多基本的問題。

[146號檔案]，我們下次見！

這一刻

寫什麼好？

我在心裡問自己。剛剛在沙發上躺了十分鐘，吃飽飯之後。因為今天很累。

還是起身，打開電腦。看了一下，上一篇文章已經進入”yesterday”。“噢，不行吧？可以這樣嗎？這樣對的起我的讀者嗎？”<–你如果相信我真的這樣想的話，我好愛你喔，好純真的小朋友啊！

我發呆了十幾秒。老婆剛好帶著她的筆電也坐下來上網（她剛剛買新電腦很開心）。所以，我就跟她說，好累喔，不知道要寫什麼，也不想去弄照片，因為挑照片也很累。

“那就不要寫啊！” 她隨口回我，顯然不太想理我。

我沒有回她，心裡想 “這樣對的起我的讀者嗎？” <–你如果又相信我真的這樣想的話，我又好愛你喔！^^ 我突發奇想，那我就寫一張借條貼上去好了，寫說我改天再補內容，先欠著。老客戶不是都這樣子的嗎？

老婆又白我一眼，完全無視於我的天才般的想法。”好吧！我看這樣的想法太前衛了，這世界還不懂得欣賞吧。” 我心底想。

https://res.cloudinary.com/hpiynhbhq/image/upload/v1519908036/y63o8lr3pelspthcfkak.png

此時，我發現我的筆電後面有一杯熱茶。“咦，這裡怎麼有一杯茶？“ 原來是老婆坐下來時，就順手幫我泡的。嗚嗚嗚，感動。

”那我來寫這杯茶好了！” 你覺得呢？當然，這次連白眼都沒了。真的，我自己也覺得這梗很爛。畢竟，這樣的文章該怎麼開頭啊？

打開電腦，發現後面有一杯熱茶等著我… 熱氣騰騰，像是老婆的殺氣… 我喝了一口，很好喝。因為我口很渴。不知道有沒有下藥？

爛到爆。若智到抱。而且寫不下去了。

於是我又回到，五分鐘前，還沒喝茶，不知道寫什麼，懶得動腦的，狀態。Shit，這五分鐘裡似乎只有喝那一口茶算是有意義的。

不！難道這一切就這樣流失在時光裡嗎！？啊啊啊！別吼，別吼！不過是五分鐘而已嘛。但，那是我的五分鐘啊！我高興用一分鐘，喔不，兩分鐘了，來哀悼它，那是我的自由啊！

就在我自己左腦跟右腦打架的當下，繆思女神總是在荒謬的時刻降臨。於是，我知道要寫什麼了。就是你看到這一篇。不假裝自己要為社會，為世界貢獻些什麼，不假裝有人會在遠端的螢幕前，對你吃的東西，去的地方，滿心歡喜的被你分享，不假裝這是一個什麼神聖的時刻…

就是記錄這五分鐘的我，不，六七八分鐘了，And counting …

於是我越寫越有感覺，越寫越覺得文思泉湧，我覺得還可以寫好幾篇，好幾天，好幾年… 我還要寫Live直播到底有什麼意義？要寫Steem是怎麼慢慢慢慢的改變了我，讓我的facebook已經荒煙蔓草（清明快到了，該去割割草了），出於貪財是怎麼讓我在一天一天裡變成一個blogger，if you allow me… 是怎麼這樣的一堆人在這裡遇見彼此，開啟人生的平行宇宙….

寫什麼好？

寫完了。我已經不累了。當然，這一篇，不是一張借條。我沒欠你了。

只是，確實是從一張借條開始的。

大述拒

昨天，看到一個廣告，把我惹毛了。

劉美女脾氣不好，可能是更年期到了吧！

當然不是啊！哼！我只是有正義感而已啊～～

是這樣的，這支廣告，拍的品質相當不錯，內容，從唬爛大數據開始 …

交錯發展各場景的畫面，男性低沉穩重的嗓音，緩緩而堅定地陳述大數據大數據每秒運算多少次多厲害，而在許多年份的豐功偉業之處：

諸如大數據選擇球員讓大聯盟棒球隊完成20連勝，大數據過關斬將擊敗人類頂尖高手變成棋王，大數據分析後改變警方巡邏路線大幅降低洛杉磯的犯罪率，大數據分析網路搜尋內容意外掌握H1N1疫情的發展，大數據會寫小說入選日本科技小說獎，甚至還有本土化題材，大數據讓新人不畏颱風將至如期在花蓮舉行戶外婚禮而圓夢，等等云云 …

好，真的，我沒少看過人唬爛大數據（近年來熱門的用來唬人的專業術語之一），在正式節目上亂說一通的我都不生氣了（只會笑），更何況是廣告呢？廣告本來就是用來胡說八道唬人用的了嘛，不會的，我的修養還沒那麼差。

我其實看到一半心中就有底這大概會是什麼類型的廣告了。

讓我來傳授一下經驗。但凡你看到那種，夠長，看到一半還搞不清楚在賣什麼的廣告，拍攝質感又不錯，而且重要的是內容還蠻感人或是觸動人心的，十有八九，是金融業的廣告。

說明一下，感動人心大概就是一些同理你、鼓勵你、支持你、為你加油，例如努力的你值得更好的對待，更值得的是支持守護你的家人。當然，還有”我們”。 買我買我～～～ （設計對白）

夠長，因為錢多。質感好，因為錢多。不清楚賣什麼，因為沒什麼可以說的，金融業要賣你的就是「降低理智」。感動人心，因為產品沒什麼好說的，金融業要賣你的就是「信任我們吧」。

正當我準備好我的嘲諷，準備在影片結束，給予不屑的嘲笑之際（話說這樣的功力可是看過多少這類影片，老是被感動而後感覺自己是傻子，不斷每次鍛鍊所修煉出來的呢！）。他們使出殺手鐧，拋出一句令我難以忍受的話。

生氣了，算我輸，你狠。這一句話是：

「想像一個沒有窮人的世界。」

哇賽，我X，夠XX，夠不XX（寫到這裡突然決定還是不要“讚美”得太露骨好了，畢竟人家可是有錢有勢的，我怕怕，即使在這個去中心化的平台上）。

不知道讀者們懂我的點嗎？

這句話說真的，還勉強可以是區塊鏈界的台詞，但我甚至不太覺得只要做區塊鏈就有資格說這句話。只是那個可能性跟邏輯是在那裡的。

金融業（我是針對整體，不是針對從業人員），其實，說真的，賺太多不該賺的錢了。我不能說窮人是他們造成的，但肯定他們在去除貧窮方面，只怕最後總結都是負面效果。

那也就罷了，世界目前就這樣，說真的，也不能說有多罪惡。

但請不要不知道自己的角色，請要賺錢默默地賺，不要高調地搶錢，聲稱你對於世界多有貢獻。這已經是無X了（我？知？心？^^）。

速食龍頭肯德基宣布經營鳥禽類動物樂園。（設計場景）

～～差不多就是這個意思。

讓我幫他們改改台詞：「「想像你子子孫孫一個都不窮的世界。」這個可以，反正就是要灌個人迷湯嘛！這我可以接受。

影片裡還有一個：大數據精準預測誰是下一任美國總統（指的當然是最近當選這一任）。哇靠！二選一，這也可以拿來說嘴啊！這真是“若智到抱”還不足以形容呢！1 …

令人驚喜的是個人頁面上最右邊的Activities ，這裡可以看到你自己的點讚紀錄，獎勵入帳，別人點你或是回覆你等等！好方便啊！我這老人家在Steemit住太久了，都不知道這裡有好東東哪！方便就是棒棒，不用切換網站來找尋遺落的記憶，無價。而且，右邊還能選擇過濾特定記錄類別哪！真的是只有一個字….

隨筆詩 — 感恩。A Poem about Gratitude

除了我家人之外，或許你就是我最親近的朋友了吧。

如果有一種永遠不變的付出與深切友誼，我想我的生命中，或許這樣的描述只適合套用在你身上吧！

我們在職場上認識，一起共事如此多的年頭，默契之好不在話下。
我記得我剛來時，其實不太意識到你的存在，你總是默默在一旁關心我。
一直到後來，漸漸地了解，我終於開始珍惜你對我的關懷，
只是還會常常因為習慣，所以忽略甚至忽視，你的存在。

今天，心血來潮，想寫一寫你，順道跟你說聲：謝謝！From the bottom of my heart.

你是只求付出不求回報的那種朋友。
關心我早餐午餐吃些什麼，咖啡飲料健不健康，尤其是在我熬夜加班，你總是默默陪伴，我都知道。
我總是忘記為你留下點什麼，即使剩下些什麼，你也毫無怨言的接手。

工作上的事，你或許不多說，但你都看在眼裡。
費盡心思卻被忽視的心血，莫名其妙吞噬你人生時光的繁雜流程，
尖酸刻薄充滿敵意的評論，錦上添花奉承你卻似乎暗藏企圖的話語 …
在不帶情感的白紙黑字裡，以新細明體的語調，一一道來。

你一一傾聽，充滿包容，把負能量都攬到自己身上去。

每天早晨來上班，我或許今日神清氣爽，改天卻睡眠不足，飄忽不定 …
但你總是律已嚴明，每天早晨都看到精神十足、準備好承受新的一天摧殘的你。
只有在業務高峰期的季節，才有機會看到你似乎也逼近承受極限的愁眉。

工作場合不總是壓力，也偶爾洋溢歡笑。
你不會跟著一起嗨，但總是像個母親，微笑地收拾我們的派對的殘留，
歡樂留影的合照裡，卻從沒見過你的身影。

在我辦公空間的小小一隅裡，職場上的一切橋段都曾上演。
你跟我一起見證過太多的戲碼，超越公司其他的同事們 …

老闆看似勉勵的話語，其實是要我承諾接手新的任務；
同事K又來串門，把供應商到客戶到同業，都給臭罵了一頓；
手下的G恭敬的請求，希望我能幫她寫下一個工作的推薦信；
老朋友M剛好到附近開會，順道過來找我聊聊，中年男人的苦悶與哀愁 …

你的低調與不存在感，讓人毫無防備。座位在角落，與牆面形成天然偽裝 …

我對你只有感謝。

謝謝你幾乎無條件的陪伴，一路走來。
謝謝你永遠不說不的包容，一直以來。

既然要說就說到底，說真的，儘管這麼感謝你，但，我從來沒嘗試過跟你說話。
或許是自己心裡的桎梏吧，我總是跨不過那道坎，希望你知道，這不是對你的忽視。

還要跟你道歉，其實。

我還記得，幾年前，那時或許還年輕氣盛一點。控制不好脾氣。
在一次不愉快的會議之後，經過你身旁，我竟然踢了你一腳 …

一直沒有正式跟你說抱歉，希望這次藉著跟你說感謝，你能一如既往地包容我原諒我的無心與莽撞 …

（叩叩叩！）
「請進！」
「打擾了，收垃圾喔！」（倒~~~~）
「謝謝！」
「不客氣！」

我看著你，重新容光煥發，準備好繼續上工 …

我知道，我不用開口，你是知道我心意的。
