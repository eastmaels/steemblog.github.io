Edit Hotel Yokohama 🍙 Dean's Tokyo Snapshots 🍙 橫濱文青風的Edit Hotel

這是我們在橫濱住的，最喜歡的酒店：Edit Hotel。雖然，地點不是面對著橫濱港未來，沒有無敵美麗海景，但也因此，價格合理許多。

其實，位置很好，走路不到十分鐘可以到櫻木町，最熱鬧的地方。這間算是隱身在後面一點的，有點雅痞風格，文青風格的一間酒店。

房間不小，只有一扇窗。不過，這窗外就是停車場跟鐵軌高架橋，很有一種日本生活風格。

簡潔俐落的設計風格。

整體設計感很強，很細緻。

我在窗前拍照。

我們在窗前吃早餐。很可惜，只待了一晚。下次到橫濱，這間鐵定會再列入考量。

下次介紹他們家的餐廳，也非常超值喔！～～～

[146號檔案] Steem能否與BTC價格脫鉤？


pixabay

最近我在想這問題，為什麼加密貨幣得跟著老大哥比特幣一起起伏？（好啦，最近都是下沈….）

算了，先不管其他幣，咱們就以Steem為思考對象：為什麼Steem價格要與BTC一起起伏？

我們先撇開現實的交易所交易對，主要是以Steem/BTC或是Steem/ETH為主，這樣限制。畢竟，這樣的現實條件，確實可能造成掛勾的效果。

比特幣，好好的去搞它的去中心化全球電子支付媒介的偉大任務，很好啊～～ 與此同時，Steem忙著用區塊鏈來打造全世界各式內容獎勵的新領域，也很好啊～～

啊，這兩者有什麼鳥關係？除了都用上區塊鏈這個新技術之外？不是完全不同的商業領域嗎？（雖然我也很想Steem去爭取全球支付的角色，不過很抱歉，目前為止都沒有一點點跡象顯示有進展…）

同樣都是用筆寫文字，諾貝爾文學獎得主寫的文章，難道就得跟你寫的文章水準相關連嗎？

沒道理。

當然你可以說，如果主要投資者都這麼預期，那麼這件事情就會實現。不過，這個說法基本上沒法解釋根本性的問題。

一個可能的，我想到的因素是：Steem主要投資者，通常也是比特幣投資者，甚至其Steem投資，是為了多元化區塊鏈資產，因此，如果比特幣下跌，他就會縮減非核心資產（Steem），來擴大核心資產的部位…

不過，我感覺這種說法也是有點怪….

或許，交易所一開始預設的交易對，還是蠻重要的？一開始就預設了投資人對於這些altcoins的想像…

如果是這樣的話，那麼，擴大Steem用法幣購買，強化投資者心中對於Steem價位的法幣印象，或許是長久之後，讓Steem價格脫離比特幣的重要步驟？

[146號檔案]，我們下次見！

時間度量：一個月 🍙 Dean's Tokyo Snapshots 🍙

轉眼之間，從東京回台，已經一個月了。

回台之後，到今天，其實也發生了不少事。讓我，已經幾乎不太記得東京的種種，彷彿，那場長長的旅行，已經是很久以前的事了….

才一個月哪！是啊～～

一個月，也差不多是我們待在東京的時間長度（略短，但差不太多）。還記得，在日本的後期，感覺出國好久好久了，久到都有一點忘記，台北的感覺，久到都以為，自己是住在東京的。

因此，回來之後，也不太習慣，對於生活中的很多事情。那時候就是覺得，出國前的事，恍如隔世，中間隔了一座異國城市，竟然能拉出這樣的時間感。

都是一個月的長度，似乎，就足以產生足夠的時間距離，細節開始遺忘，感覺開始模糊，於是頭腦自動賦予一個月前的事情一個tag，上面寫著：很久以前。

這其實也沒什麼不好，表示生活很充實，才一個月，就覺得很久了，表示經歷了很多事情，時間沒有白白流逝。

然後，一年，有十二個月。整整十二個月哪！那麼多呀！挺富有的嘛，大家～～

十二個，覺得很長很長的時間段。

能不開心嗎？

[DClick] 初體驗簡評


pixabay

昨天初體驗了DClick這個新的發帖平台：新時代的網路賺錢：My first DClick Post，蠻好玩的。

今天來寫一下初體驗的感想，順便簡介一下。先簡介一下… （參考上圖）

平台介面很簡單（也可以說有點陽春），分成Overview / Monetize / Advertise / Delegate / FAQ 等五大部分，分別功能大致是：資訊彙總 / 發帶廣告文章 / 廣告主區域 / 代理情況 / 常見問題。

從Overview可以看到我昨天發帖，總共目前有11次點擊，帶來了$0.53的價值（據說是會變成點讚或是可以轉變成日後的SMT-DCLICK）。以不抽成作者來說，這樣已經是蠻好的回報了喔！

發文部分，大致跟Steemit差不多了。我的小抱怨是：沒有草稿儲存功能，然後也沒地方看已經發出去的文章，只能從Steemit去看。

廣告主功能我尚且用不著，不說。代理部分，我也嘗試了代理200 SP，目前一天的收益是：$ 0.02465851，好還是不好，我也不去算了，呵呵～～

總結一下，我蠻喜歡這介面的，簡單就好，但是當然還有很多可以補強的地方，慢慢去加就好。重點是要能拉到外面的廣告，以及外部的流量～～

如此一來，有個功能就很重要：那就是廣告內容跟文章內容的關聯度。所謂置入行銷，就是要從內容去連結讓讀者想點擊廣告啊，否則，點擊/閱讀的比率會太低喔～～ 希望早點可以挑廣告類別之類的，這樣，我就不用這樣一直像下面這樣促銷了啦！>_<

歡迎讀者們用力點擊下面這個Steem區塊鏈上的新平台DClick！它結合了Steem加密貨幣提拱的寫作獎勵，並且引進了外部廣告點擊的概念，十分有趣喔！
