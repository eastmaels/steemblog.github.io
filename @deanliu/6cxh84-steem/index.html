[ Steem十講之題外話 ] 在區塊鏈上，你可以怎麼露？

各位朋友們新年好，

一個月前的一篇[ Steem十講之題外話 ]，原本只是一個偶然，但上次迴響熱烈，我後來就覺得應該要續寫一篇，儘管我的十講貌似又得拖下去了，就請再稍微等等囉！

今天這個 題外續話，延續題外話的話題，把幾個關於個人訊息的類別以及透漏的原則，做一點討論，希望可以做為大家的參考。

那就開始吧！

[ Steem十講之題外續話 ] 在區塊鏈上，你可以怎麼露？


恰好我昨天發了一篇一篇關於黑客與幣的報導，你可以去看看黑客們怎麼滲透入交易所，這還是他們得自己想辦法去找目標對象的網上資料紀錄的情況下。還有，咱們偉大的台灣黑道兄弟們，已經開始行動，看看這篇報導，有利益還沒法律，黑道當然是迅速跟上了…

我們在第五講時曾經帶到了創辦人Dan的偉大夢想，全透明的世界才是消除暴力的最佳方式。再怎麼樣被美夢所感動，也無法忽視這世界還需要非常久的時間來走到相對接近的程度。在那之前，只有你自己，才能保護你自己。

廢話不多說，進入主題吧！我們主要還是談Steem上的個人訊息問題，而因為Steem區塊鏈帶著個人內容，這是它競爭力的主要特點，卻因此也帶來了一些有別於其他幣的風險。

幾乎絕大部分的幣/鏈，都是建築在公鑰/私鑰的概念上。公鑰是你的帳號或地址，可以公開；私鑰代表運用這組帳號/地址的絕對權威。注意！公鑰是可以由私鑰導出的！所以不要以為你鑰匙掉了，歹徒反正不知道你家在哪裡喔！因為每個鑰匙上面都把地址給附上去了… 這點可以參考O哥的扯淡文章來學習一下。

私鑰的保護，好多人說了好多，我不再說了。我要說的是：再怎麼保護，你都不能防止你自己知道如何獲得你自己的私鑰，這句話可以好好參參。大致上來說，保護完私鑰，接下來就是要保護你是誰不能讓人知道，尤其是不能讓可能非常少數，甚至目前還不存在，且帶有惡意的人，有辦法知道你是誰，或是到哪裡去找你。

首先，是你的臉。臉其實就是一個私鑰的概念，全世界人的臉除了同卵雙胞胎之外，都是不一樣的。你一但露臉上鏈了，私鑰就外洩了。先別急著擔心，大概幾乎大部分人都露臉了啦！即使沒上鏈，大概也被許多這裡的朋友所認得了（例如我）。社交平台，不露臉（包括不放自己入鏡照以及不參加聚會）幾乎是不可能任務，怎麼辦？還好，目前刷臉科技還沒普及到有太大風險，認得臉大概還沒辦法直接找到你是誰吧？（除非是警方）只要你好好不做壞事，然後保護好其他訊息，那應該還沒關係。只是，要記得你曾經露臉這件事，不要以為街上你看起來像陌生人的人就不會認得你（哇，好像大明星喔！）。

其次，是你的真實名字。名字重複度高一些，但還是一個非常關鍵的訊息，尤其是你沒辦法Google臉，但你可以Google名字。或許你的名字不是網上找得到記錄的人，但是：1) 替換Google為你的學校、公司、問認識你的人呢？2) 洩漏之後，好了，從今天起，你就是「網上找得到記錄的人」啦！當然，如果你是例如在台灣，名字是志明啦，怡君啦之類的，安全度是高一點。

其三，你的活動地點。住家：你大概不會傻到把地址給放上網，但你有可能透漏你每天早上都在哪裡吃早餐，下班都到哪一間超市買菜，周末都在家附近哪間咖啡廳上網…. 公司：我們公司是做什麼產品的，在哪個城市，是世界五百強….

其四，你的行蹤。「大家好，很開心跟大家預告，這個周末我要出發去冰島玩了！好期待喔~~~~ 到時候再跟你們分享照片喔！」從香港(例如)到冰島的班機，一天能有幾班？周末是哪一天會不知道嗎？如果你也露臉了… 那我這個壞人就準時到機場去堵你就好了~~ 「我要參加台北Steem盃城市馬拉松喔，我會穿Steemit的T-shirt去幫平台打廣告的！」…. 沒說日期啊！對啊，沒說，但其實也說了。我這個壞人拿著望遠鏡在高樓看，指示我同夥說對就是那個穿溫泉標誌的女的，等跑完就把她給帶上廂型車去等我來逼問密碼 ~~

其五，你的財富。這個訊息不是跟你的個人資訊有關，而是跟歹人是否對你產生興趣有關。你Steem上的財富，那是沒指望隱藏了，但畢竟大家常討論加密貨幣，其他部分的財富，還是有可能說溜嘴的。你可別以為你都說虧錢就沒事啊！歹人很精明的，知道沒錢的人不可能虧錢的~~~~ 嘿嘿！

哈哈哈！大家又被我嚇得…. 真是抱歉哪！真的真的沒這麼可怕，我只是要激發你的危機意識。

那接下來說說一些我自己想的原則。

一，明確知道自己曾經在這裡分享過什麼個人訊息，想一想這些訊息可能怎麼被利用。如果有一天有人在你常去的咖啡廳湊上來說，Susan，好久不見，我是你小學同學啊！你可能一時之間覺得這人認識你，那還真有可能是同學只是你忘記了…. 其實他只是上Steem查過你所有留下的訊息跟照片而已…

二，與人相交往，貴在真誠，但真誠在於心意，不在於據實以告。你真的要去冰島，那肯定不好說是去普吉島了，但你可以不用說日期，或是故意說錯日期，或是說可能要去但還沒決定… 對於你的讀者來說根本沒差別，但對於想知道些什麼的人來說就難度高不少了…

三，留意發文內容之外，也要留心回覆。你可能發文時還真的乖乖地好好遵循這些原則了… 但是可能有人刻意針對你沒說的，而他需要的訊息來問你：「哇！好羨慕你去冰島喔！幾時回去呢？我聽說快要有颱風來了~~」哼哼！想藉由關心話術來套我話？「呵呵，真的喔，謝謝你的訊息啊，冰島真的非常好玩喔！你也快來吧！」本姑娘豈是這麼容易被你套出來的~~~

還有一點提醒：除了保護自己之外，希望大家也有保護朋友的意識。有人私下找你要誰誰聚會的照片什麼的，當然要好好保護朋友啦！至少要價錢夠高才出賣啊～如果大家都普遍具有防範意識的話，就像是打疫苗一樣，病毒就不容易攻陷我們這個大家庭啦！是不是啊？^_^

勤上Steem以來，我一直覺得我的世界有兩個：一個真實的，具體的，你所處在的社會與小環境；另一個就是Steem上面的網路世界，各位好朋友們所在的這個圈子。你說後者更虛幻嗎？是也不是。不是在於，許多我所謂真實世界的同事們、朋友們，我們都已經習慣了一種制式的互動方式，儘管天天次次說的話不同，但彼此的互動，大致上就是那樣，他們不會知道例如我早上去爬山時心裡想到的事與想法，但他們知道我中午吃了什麼，家住哪裡，哪個大學畢業的等等 …

到底誰認識的我更真實一點呢？這是一個很深刻的問題了….

但顯然如果這兩個世界混合了，或是兩方合作了，有心人士要對我下手就更容易了… ^_^… 這才是這一篇的主題啦！

希望這兩篇題外話，有助於你更安全地在這裡，自在快樂地與人互動交往囉！
