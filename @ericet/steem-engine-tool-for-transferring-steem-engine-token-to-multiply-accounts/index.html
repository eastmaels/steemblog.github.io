发送Steem-engine代币给多个账号的小工具/Tool for Transferring Steem-Engine Token to Multiply Accounts

我经常需要发送Steem-engine的代币给多个账号，但是一直没有找到一个像行长@justyy 写的给多帐号发送STEEM/SBD的工具

所以只能用蠢办法，一个个发。大家都知道Steem-engine转账有多慢的，所以每次给多帐号发送代币变得非常的浪费时间

昨天写完小工具一键清理Steem-engine代币后,想了想，只要再稍微修改一下就可以改成给多帐号发送Steem-Engine代币的小工具了. 所以今天有空就写了这个小工具。

网站：https://ericet.github.io/TransferToken/
代码开源：https://github.com/ericet/TransferToken

操作介绍：

1. 输入你的Steem用户名

输入后，Steem-engine代币那一栏会显示你钱包里面所有大于0的代币和数量，你可以点击选择你要转账的代币。

2. 输入你的活动密钥，要转给每人的代币的数量，附言和要转的账号

活动密钥，代币的数量和要转的账号是必须填的，附言可加可不加。

样板：

3. 点击“Transfer”转账

点击转账后，日志会显示转账的状态。

已经分别给@teamcn-shop @team-cn和@xiaoyuanwmm转了1 SPORTS

希望这个小工具可以给大家带来便利

未来也会把这个功能添加到steemcn.org的钱包里，这样不用记多个网站。

如果有任何意见或者问题，欢迎提问。
