怎么用JS写个STEEM转账程序？



(Image source: Pixabay)

有没有好奇NBC每日的自动分红程序是怎么实现的？或者那些给很多账号发送0.001 STEEM打小广告的程序是怎么实现的？

在steemjs里，提供了一个function用于转账：

steem.broadcast.transfer(wif, from, to, amount, memo, function(err, result) {
  console.log(err, result);
});

你只需填入相应的信息就可以进行转账了。比如我想给team-cn发送0.001 STEEM，并留言hello：

steem.broadcast.transfer(活动密钥, ericet, team-cn, '0.001 STEEM', 'hello', function(err, result) {
  console.log(err, result);
});

只需几行代码，一个简单的转账程序就写好了。

用金币兑换的零食

老婆一直喜欢在一个国外华人的晒货平台晒货来获取金币，而金币可以用来换取一些商店的礼物卡或者一些零食。

这次老婆用金币换了4包农心的零食：蜂蜜脆条、洋葱圈、原味虾条、辣味虾条。

每一款都是大爱最爱没有之一啊。

包裹是由农心美国公司寄出，除了四大袋零食之外，还赠送了他家的购物袋一个。非常贴心。

每袋零食的生产日期都很新鲜。迫不及待的拆了一袋最喜欢的辣味虾条。脆！鲜！一点点辣，有种童年记忆里的味道。

真羡慕老婆的金币可以换取零食或者礼物卡，而我的steem只能放着等着一天清零 XD

SteemKnights---不单单只是简单的石头剪刀布游戏

昨天写了一篇关于一款新的基于STEEM的游戏—SteemKnights

由于没有太多的介绍，只看规则，我认为这是一款高级版的剪刀石头布游戏。

SteemKnight的发起人在帖子回复说，这款游戏不只是简单的石头剪刀布游戏，还添加了策略，一场游戏可能需要好几分钟。

小镇赛马场观摩日

小镇的马场今天举办一年一度的欢庆开场活动。于是举家前往观摩。

由于出门有点晚，加上开错路，错过了开场的游行开幕式。进门的时候刚好和穿着整齐制服的队伍擦肩而过。好可惜！

第一次来马场，感到很新奇啊。没见过大场面的我表示和神马TVB里的差不多。室内到处都是电视播放着实时赛马的信息，到处都是带着耳机用笔在马经上写写画画的赌马者。

欢庆活动节目很多：

  • 开幕游行（错过）

  • 赛马表演 （不是很明白为啥骑手不是坐在马背上）

  • 音乐会
  • 农场小动物（可触碰）

  • 南瓜🎃图画

  • 老爷车评比

其他各种娱乐活动

看到有个项目比较有意思，100刀奖金的骑牛🐂比赛。只要在晃动的铁牛上坚持25s就能获得100刀的奖励。我上去试了一下，7s失败。😭😭😭。牛背是在太滑了，加上牛仔裤没有啥抓力，牛一甩就被抛出去了。

总之，还是挺欢乐的一天。

一款基于STEEM的高级版的剪刀石头布3D游戏

看到标题是不是以为我要说小卖部@teamcn-shop的剪刀石头布的游戏？

Big No No! Too young, too simple!

我要介绍的一款叫Steem Knights（Steem骑士）的游戏

据官方介绍：SteemKnights - For Steem we fight!，这是一款基于STEEM的3D游戏，将于9月28日正式上线

网站地址是：https://steemknights.com/ref/@ericet

先看看网站的页面：


是不是有点高大上？但是看看游戏的玩法:


游戏有3种战法：剑可以击败弓箭，弓箭可以击败魔法棒，而魔法棒击败剑

诶，这不就是剪刀石头布的游戏吗？只是改了一下名称。

那怎么通过游戏获取STEEM奖励呢？


有四种方式：

  • 通过战胜对手获取奖励减去10%手续费。
  • 每次对战，无论输赢，将获得1%忠诚度，最多120%。获得120%忠诚度后，你将获得额外20%的STEEM奖励。
  • 每天游戏将把他们获取的10%手续费中的20%放入奖金池。每次对战，胜利了将获得100骑士勋章，败了也会获得50勋章。每日按照持有的勋章比例分配每日奖金池奖励。
  • 推荐他人加入游戏将获得10%对方的勋章数量。

看了奖励方式，需要投注steem才能游戏。

还有一周时间游戏就上线了，到时在体验一下这个游戏。 —
