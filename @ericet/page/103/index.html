椒盐杏鲍菇

老婆喜欢做菜，所以经常会捣鼓新菜谱。

这道菜是老婆从下厨房上看到的，然后使用她最爱的空气炸锅进行制作。

下面是老婆的制作方法：

  1. 杏鲍菇洗干净，切成薯条状。最好切大一点哈。我切的太小了，烤完一缩水，体积减少一半。
  2. 加少许油，盐。用手或者摇晃碗，让杏鲍菇的表面都能薄薄沾上。如果后面还会加椒盐的话，盐要少一点。
  3. 放入空气炸锅，375华氏度8分钟。取出，加椒盐晃动均匀。继续400华氏度5分钟。搞定，撒葱花，开吃。烤的时间按照大小适当调整。

这道菜味道还是不错的，就是杏鲍菇炸的有点老了，没有啥嚼劲。废话那么多干嘛，有的吃就好了！

和小卖部猜拳赢的机率是多少？



最近很多人说小卖部@teamcn-shop 猜拳输多赢少，是不是小卖部的猜拳有黑幕？

为了证明小卖部不是黑店，这里和大家谈谈小卖部猜拳的机率。

可能有人以为猜拳的机率是50：50，因为不是赢就是输，为什么小卖部那么难赢呢？

其实玩家要猜赢小卖部的机率是1/3，因为有3种结果：胜，平，输

要获得每日小卖部的猜拳小能手的机率就更低了。

每日猜拳小能手需要至少获得2胜，要获得2胜，机率是1/3 * 1/3 = 1/9.
也就是说平均要猜9局才能获得2胜（小卖部是RC克星！～）。

说了那么多，如果你们还不信小卖部没有黑幕，这里是小卖部猜拳部分的代码，加了注释方便大家理解。

steem.api.streamTransactions("head", function (err, result) {
        if (result && !err) {
            let txType = result.operations[0][0];
            let txData = result.operations[0][1];
            var result;
            var choice;
            var machine;
            var count = 0;
            var rps_result = "";
            //查看是否有人在回复小卖部
            let includesMention = checkContent(txType, txData);
            if (includesMention) {
                //如果有，查看是否在黑名单里面
                if (blacklist.indexOf(txData.author) < 0) {
                    var lowerCaseBody = txData.body.toLowerCase();
                    //查看回复是不是包含‘rock','石头'，'石頭','软哥','軟哥'
                    if (lowerCaseBody.includes('rock') || lowerCaseBody.includes('石头') || lowerCaseBody.includes('石頭') || lowerCaseBody.includes('软哥') || lowerCaseBody.includes('軟哥')) {
                        choice = 'rock';
                        count++;
                    }
                    //查看回复是不是包含'paper','布','米高'
                    if (lowerCaseBody.includes('paper') || lowerCaseBody.includes('布') || lowerCaseBody.includes('米高')) {
                        choice = 'paper';
                        count++;
                    }
                    //查看回复是不是包含'剪刀','村长','村長'
                    if (lowerCaseBody.includes('scissors') || lowerCaseBody.includes('剪刀') || lowerCaseBody.includes('村长') || lowerCaseBody.includes('村長')) {
                        choice = 'scissors';
                        count++;
                    }
                    if (count == 1) {
                        //机器随机从石头，剪刀，布选一个
                        machine = choices[getRandomInt(3)];
                        result = results[choice][machine];
                        sendComment(txData, result);
                        //如果玩家赢了
                        if (result == rock_win || result == paper_win || result == scissors_win) {
                            //给玩家点赞
                            sendVote(txData);
                            rps_result = "win"
                        } 
                        //如果平局
                        else if (result == rock_tie || result == paper_tie || result == scissors_tie) {
                            rps_result = "tie";
                        } 
                        //如果玩家输了
                        else {
                            rps_result = "lose"
                        }
                    }
                }
            }
        }

雪的记忆

(image source:Pixabay)

今天下雪了，我住的地方大概1寸的雪。
问同事他住的那边下多大？
同事说1/4寸，这点雪他闭着眼睛都能开车!

这句话突然让我想起有一年的大雪，就说有一年大雪有3 feet（大概一米），就算我睁大眼睛也开不了车！

说起这场大雪，我的印象是非常深刻的，因为这场大雪直接让新泽西交通瘫痪了好几天。

在这场大雪之前，政府完全没有预料到会有如此大的大雪，所以减少了铲雪车的开支，以至于突然的大雪让政府没有足够的铲雪车处理路面积雪。

我记得那时我还在上大学，下雪天学校停课，所以在父母的餐馆帮忙送外卖。
一开始大雪不是很大，送外卖还是没啥问题的，但是在下午3，4点的时候，大雪越下越大，只看到雪一团团的从天上掉下。
大雪把路面全部盖住了，我完全找不到路了。最后很困难的开车回到餐馆，让父母早点打烊，毕竟这么大的雪不会有人出门买吃的。

隔天雪停了，但是路面积雪还是很厚。我送餐的途中，车因为雪太厚了，车卡住了。车怎么开都不能从雪里面开出去。
这时，在我停车附近的一家人看到我的车被卡住了，好心的帮我把车挖了出来。

说完感谢的话后，我开车离开。开了大概2栋房子，车又卡住了。。。

那家好心人家看到了，继续带着铲雪工具帮我挖车。那时真的又尴尬又感激的

挖出来后，车又可以继续上路了。这次学聪明了，只往比较浅的地方开，最后顺利的离开那地方。

一直以来不喜欢下雪天，就是因为下雪让开车增添了很多危险性。

为什么这几天STEEM价格飙升？



(Image source: pixabay)

相信大家都知道这几天STEEM价格逆市飙升,但是大家都不太清楚飙升的原因。


  • 是不是因为我们的指路明灯buddy爸爸@shenchensucc怒抛2000 STEEM引起价格飙升？

  • 是不是因为STINC power down，让社区不在一家独大，大家纷纷看到STEEM未来新希望？

  • 是不是好久没出来唱歌的ned，突然现身大谈STEEMIT的未来，大家对STEEM又有希望了？

可能上面这些原因真的引起STEEM价格飙升，但是最靠谱的解释是我下面要说的。

前段时间，韩国的交易所bithumb遭受黑客攻击，所以他们停止了很多货币的提现和充值，STEEM也是很多货币中一个。

不久前，bithumb发布公告说从1月15号开始，将慢慢恢复虚拟货币的提现和充值功能。（https://cafe.bithumb.com/view/board-contents/1639880）


这几天恢复的虚拟货币有： STRAT, WAX, AE, GTO
这些虚拟货币都爆涨了30%。因为这些虚拟货币在bithumb上的价格比其他交易所来的高，可以通过搬砖赚取差价。

STEEM也将要恢复提现和充值功能，所以大家抱着赚取差价的心态，买入STEEM，想着等恢复提现和充值后大赚一笔。(STEEM在bithumb目前价格$1.19,其他交易所平均$0.32. )

结合公告的时间点，和恢复提现和充值的虚拟货币的行为，很大原因是大家想赚差价。
所以如果现在跟随大流买入STEEM可能和buddy爸爸一样被割了韭菜

最后给大家一个投资秘诀，关注@shenchensucc！他发帖说抛掉STEEM时，大家就买入。他发帖说要买入STEEM时，大家就抛掉！至今buddy爸爸还没有让我们失望过！

buddy爸爸专注当韭菜数十载，他的专业值得我们信赖！
