勇敢的第一次线上见面会

由于新泽西的新冠肺炎疫情严重，勇敢已经半个多月没去学校了

每天，学校老师会发来邮件，告诉我们今天要做的功课，要做的游戏，还有一个老师自己录制的讲故事的视频让勇敢看

勇敢这几周来都还挺配合，唯一让他不开心的是，他想去学校见见老师和同学

所以老师就发起了一个线上见面会，也是勇敢的第一次线上见面会

整个见面会只有10几分钟，但是当勇敢看到同学后，兴奋不已

整场就他话不停，看到女同学就说，I Miss you!

看到不认识的人，就说My name is David. Glad to meet you!

听到有人说今天他生日，他赶紧说Happy Birthday! 还让Alexa(语音助手）给他唱生日快乐歌

还给同学介绍他的玩具，让他们等疫情结束了，到家里来玩

活脱脱的一个社交达人

曾经还担心他自闭，现在完全不用担心了

这次见面会唯一不足的是，老师没来！

老师可能第一次用视频软件，不懂怎么连上会议，所以隔天又发了一个视频会议。

由于老师没在邀请链接里面写上会议时间，所以这次的视频会议，只有3人参加，老师，勇敢，和一个女同学

不知道女同学太文静了，还是勇敢话太多了，整个会议就老师和勇敢在聊天，聊的居然还不错

老师问勇敢是否看过她发的讲故事的视频？

勇敢说都看过了

老师问，最喜欢哪个故事？我以为勇敢回答不上来了

勇敢说，最喜欢Bear on the chair。然后还解释为什么喜欢

这个还是让我蛮惊讶的，已经有了个人喜好和思想

这次见面会结束了，不知道什么时候勇敢才能回到学校和同学们玩耍

怎么直接用SteemCN打赏或者转账LIKE？

今天萍萍@annepink说like.co上也可以转账，经过测试真的可以，而且比用Liker Land转账来的方便!

怎么直接用SteemCN打赏或者转账LIKE

  • 打开SteemCN网页(http://steem.buzz或者https://steemcn.herokuapp.com/)

  • 如果账号旁边有个拍手的图标，说明那个账号是拍手公民，可以打开他们的帖子给他们拍手，打赏或者转账LIKE

  • 拉到帖子最下方，点击LIKE账号头像那里：

  • 跳出一个转账页面。输入你要转账的数量

  • 点击确定就可以转账了。

  • 如果不确定自己账户还有多少LIKE，点击页面右上角，会显示账号余额。

这个可比通过Liker Land App转账来的方便。App上你还要去记转账账号，而直接通过网页就不需要记对方的账号了。

编程者的神器---Code Server

之前看@ety001的帖子，无意中了解到这个神器，从此不能自拔，实在是编程者的神器啊！

Code Server其实就是云端版的Visual Studio Code.

一般我编程的环境就在3个地点：公司电脑，家里台式电脑和家里的笔记本

有时在公司写了一半的代码，走的匆忙没时间把代码push上去，到家后就没办法继续写下去

有了Code Server，代码都在云端，不需要push，到任意一台电脑打开浏览器，登陆Coder Server就可以继续编程了

还有就是，我的笔记本年代久远，配置有点低。代码在笔记本上运行缓慢又耗内存硬盘。但是在云端就没有这个烦恼，代码都在服务器上运行，使用的是服务器的内存和硬盘，笔记本上最多是费了点网络流量。

推荐编程爱好者使用！

Code Server的原代码：https://github.com/cdr/code-server

搭建Code Server可查看这个视频：https://www.youtube.com/watch?v=UeP6VQjMw8Q

丢块的糗事

前天，我们的社区见证人@cn-witnesses上线了!（如果还没给cn-witnesses投票，请支持一下！）

在大家的支持下，见证人排名也一路增长，到了一天可以出至少一块的排名

期待着出第一个区块，谁知没等来出块先等到了丢块。。。

赶紧询问@ety001是什么情况。在ety001的指导下，查看了一下日志，没有发现问题。

在查看config的时候，发现好多跑路到HIVE那边的seed node在config里面。这也导致有时候会获得HIVE那边的区块

按照@ety001之前帖子里面的seed node重新填写了STEEM上可用的seed节点: 记录下目前Steem链可用 seed节点

想着可能是seed节点的问题导致丢块。谁知几个小时后，又收到了丢块的讯息：

这可把我折腾死了。到底哪里出错了？@ety001让我对对见证人私钥和公钥是否正确。

在查看私钥的时候，发现了一个很蠢的问题。。。

见证人账号和私钥部分被注释掉了（就是不读取这一行的讯息），导致到cn-witnesses出块的时候，没有私钥来出块

赶紧去掉注释，重新运行了见证人。

早上6点多，鱼哥@maiyude通知说cn-witnesses出块成功~

Costco货源充足

前几周新冠肺炎疫情刚爆发的时候，写过Costco里的水，厕纸，面包等被一扫而空

几周过去了，家里的食物也吃的七七八八了，所以去了趟Costco补货

来到Costco后，发现和以往有了很大的不同。

首先进门的地方，有人会给你一张湿纸巾让你清理一下购物车的把手。清理完后并要求你给自己的手消毒一下

店里和平时比起来少了一大半人

以为来的比较晚，东西应该剩下的不多。但是看到下面画面，感觉自己是瞎担心：

肉类货源足足的！

之前一包难求的面包也多多的！

最恐怖的是水，到处都是水！.

厕纸也足！大家都不用担心没有厕纸擦pp了！

结账的地方也有变化。在结账地点每隔一段距离画了一条黄线，让大家按照这个距离排队

结账的柜台也添加了一个塑料的隔离板，这样收银员和顾客就没有直接的接触了

买了一车的东西，又可以继续在家宅个一个多月了

新冠疫情下，餐饮业受到重挫，公司打折应对

我的公司是做餐饮连锁的，自从新冠肺炎爆发后，公司业绩受到了很大的打击

老板上周发来邮件说，连锁店最近的订单少了将近40%，线上订单从9%增长到22%，第三方从8%增加到14%

看到线上订单增长的趋势，公司决定对线上订单提高打折幅度，线上订单全部5折

折扣上线第一天还没2小时，网站和APP都被挤爆了，服务器的CPU严重超负荷

公司下令关闭一些服务来释放部分CPU。一整天大家精神都在紧绷状态

第一天的订单是平时的10倍多，第二天15倍，第三天18倍，今天将是活动的最后一天

这个打折力度是有史以来最大的，公司会给连锁店补贴进行这次的折扣，所以基本公司都在亏本。但是好消息是，新顾客增加了几十万。只要留住一部分新顾客，未来还是可以赚回本的

疫情后，大家估计会着重发展线上这一领域了吧
