Keybase 总数20亿个XLM 空投

Keybase是一款类似Slack，Telegram, WhatsApp的加密聊天工具。

目前SDF(Stellar Development Foundation)将空投20亿个XLM给Keybase参与空投的用户。这20亿个XLM将分20个月，每个月空投1亿个XML。第一笔空投将在10月15号送出。

目前有18385名用户注册空投，按照目前的人数，每人可以获得5439 XLM（价值$315,会随着人数的增加而减少）

获得XLM的空投条件是以下之一：

  • 你在9月9日之前注册过keybase的账号
  • 你在9月9日前注册过github或者HackerNews账号

如果你符合以上任意一个条件，那你就可以往下看了。

下载并安装Keybase

下载链接：https://keybase.io/download
目前支持iOS，安卓，Windows, MaxOS, Linux

注册Keybase账号

要求手机号码来验收验证码

看看是否符合空投条件

点击“See if you qualify” 按钮查看自己是否符合空投要求

显示我有2项要求需要验证：

添加Paper Keys

点击下图像汉堡一样的图标，然后选择Devices

点击”Create a paper key”, 记住口令密码后保存。

再重复建立一个paper key。 账号拥有总数3个设备+paper keys就行了。

绑定github账号

点击下图像笑脸的图标，然后选择“Edit Profile”

点击Github选项

会要求你输入你的github账号id

输入账号id后，会自动生成验证的内容。

你需要登录你的github账号(https://gist.github.com/)，创建一个名叫“keybase.md"的gist文档。

创建完成后，回到Keybase，点击”Okay posted! Check for it“

如果验证失败，请等待10分钟后再试。按照下图所示重试。

如果重试后还是失败，登录你的github，在keybase.md文档的最后添加一行空白行。（这个办法对我有效）

再次查看是否符合空投要求

如果3项都打勾了，恭喜你~ 符合空投要求了~ 点击加入空投

参与空投后，每个月的15号（10月15日开始）就会自动获得XLM空投，持续20个月。

这是躺着赚钱的机会啊~ 还在等什么~ 还不快参与空投吧~

基于STEEM的产品测评Dapp---Review Hunt

加入STEEM有段时间的用户可能听说过steemhunt这个DApp

Steemhunt让用户发掘优质的产品项目，用户可以通过分享产品项目来获得steemhunt的点赞和他们自己的ERC-20代币HUNT。

目前HUNT币已经登陆了几家交易所，一个HUNT币目前$0.00696805


在HUNT的经济体系里面，将会有2个新项目：ReviewHunt和IdeaHunt


IdeaHunt将在第四季度上线，而ReviewHunt刚上线不久。

ReviewHunt主要是商家花费HUNT币来发布产品测评，而用户对产品进行测评来获取HUNT币。

ReviewHunt的网址：https://review.hunt.town/i/966

目前已经有很多测评上线了：


很多测评其实挺简单的，比如这个：



第一个任务，只需绑定你的steem账号就可以获得59.84 HUNT币

第二个任务要求截图任务的确认页面就可以获得119.68 HUNT币

完成以上任务，轻轻松松获得179.52 HUNT币（在测评结束后发送）

还有其他简单的测评，如果有时间也可以一一测评

目前对这个Dapp唯一不满意的是提现的流程。

如果你要提现，需要绑定一个ETH钱包(不是交易所钱包)，然后把HUNT币提现到钱包。

如果要交易HUNT币，需要把HUNT币转到支持HUNT币交易的交易所进行交易（转HUNT币的时候，需要消耗一些ETH）

这样对于虚拟货币小白来说，操作有点过于复杂。如果steemhunt可以直接把HUNT币转换成STEEM就可以减少操作。 —

Home Depot手工

又到一个月一次的Home Depot儿童手工活动～

这次是要弄一个小花盆

这是完成的作品：

看起来是不是很简单？

其实不简单，还费了我九牛二虎之力才完成！

由于手工有点难度，所以一般是我先把钉子钉入一半，然后勇敢小朋友拿着锤子把剩下一半钉子钉进去。

之前的手工都会把要钉的地方的孔挖好，这样方便小朋友钉。这次没有挖，所以当勇敢把钉子敲进去的时候，钉子都歪了。最后还把木板给钉坏了。

所以重新拿了一个新的开始弄。这次就没让勇敢参与，把坏的给他，让他随便怎么敲。

最后成品钉子还是有一个钉歪了，但是没啥影响。

有了小花盆就要思考要往花盆里面种什么？

家里都是手残星人，估计只能种葱了😂

Costco的模型吉普车

每年到了这个时候，costco就开始铺设玩具展区为下半年的各种大型节日做准备。

勇敢小朋友是爱车人士，所以就在车模区流连忘返。摸摸警车，看看老爷车，怀里揣着跑车，眼睛望着jeep车。车真的是男人（孩）永远的玩具。

Maisto 的车模，我们每年都会买来玩。costco售价15刀左右。模型的选择很多，不同品牌，不同车型。

车模比例1：18。车门，后备箱都可以打开，轮胎可以转动。车的细节部分也做的比较精致。还可以配合乐高玩偶一起玩耍。很适合送给孩子作为礼物。

鉴于勇敢小朋友最近良好的表现，以及对车日益增长持续不断的热爱，购入一辆 jeep rubicon作为奖励。

结账后，在出门的时候，看到一款和模型一摸一样的吉普车。颜色也还是红色。
