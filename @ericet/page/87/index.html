如何从火币交易所提现steem到steemit？

有新人问怎么从火币上提现steem到steemit上。 搜索了一圈没有找到答案，所以自己试着转了一下。 基本操作和币安提现是一样的。教程 | 如何把steemit里的steem转到币安

步骤如下：

首先登入: www.huobi.pro 右上角点击 资产->充币&提币 * 在列表里找到Steem，点击”提币“

提现地址：填写你的steemit id 地址标签：随便填写 数量：你要提现steem的数量 点击”提币“

好了，交易已经提交，等待steemit id ”huobi-withdrawal” 给你转steem。

最后说一下火币提币的费用，每笔提现需要0.5 steem. 如果有什么疑问，欢迎留言提问。
