【CN区美食大赛#2】家乡菜大葱油拌面


这是我老婆的家乡菜-葱油拌面。 夏天最方便的就是葱油拌面啦。下碗面，然后淋上葱油和酱油，简简单单，但是香气扑鼻。

下面是我老婆的做法：

  1. 葱摘下来，洗干净切断，然后用厨房纸擦干，然后再晾干。（这步绝对不能少，因为等会放进油里湿湿的会溅油。） 。

  2. 准备一个干净无水无油的容器，八角桂皮香叶洗干净晾干，一个无水无油的锅，一双干净的筷子。开始。我用的是BALL的宽口密封瓶，我觉得密封性有待加强，但是平时收纳还是非常方便的。

  3. 锅里放入油，小火慢慢加热，加入八角桂皮香叶熬一会，等到香气出来，马上把香料用筷子夹出丢了。然后加入葱白，熬制一会，看到软了，加入葱绿。然后小火慢慢熬到葱脱水变成黑色脆脆的，关火，把葱先夹出来放瓶子里，油放凉装瓶就可以了。放冰箱可以保存一个月。




友情提示： 1. 葱油可以做面，炖肉，做葱油饼，有些人还做饼干（这个我没做过，就听说，前三种我做过）。 2. 关火后离开炉灶后先把葱拿出来，免得余热使葱焦了，焦了会变苦，不要问我怎么知道的，都是泪。 3. 熬完葱油的锅，可以煮点面进去拌拌，不要浪费。如果像我大半夜熬葱油，又减肥的人。那就用厨房纸把多余的油擦干净再洗，免得堵塞水道。 4. 加香料是增加葱油的层次感。也可以不加，也可以加大蒜，加辣椒，加其他香料，还看到过下厨房菜谱里加海米仙贝等等的。我不太喜欢大蒜辣椒所以我没有加。还有就是蒜其实不能多炸，容易炸糊了。管家的日子还放了红葱什么的，但是我觉得要突出葱的香气，也没有放。 5. 也有些做法是熬好后再加酱油，耗油一起做，也可以。但是如果想要葱油做其他的比如葱油饼什么的。我觉得还是单一熬制比较好。 6. 不要熬制太多，一碗葱油面一勺就够了。 7. 感觉大葱熬出来的没有小葱香。

一碗葱油拌面，简简单单，一勺葱油几根脆脆的葱，加上一勺生抽拌匀就可以了。夏天必备啊。

【steempress小技巧】标签顺序问题

相信很多人像我一样，每次通过steempress发完贴之后,标签就会按字母顺序排序。 这样的问题是，有些机器人会首要查找第一标签的帖子，如果你没有放对第一标签，很多时候机器人会错过你的帖子。

如果你想自己设定主标签要怎么做呢？

方法很简单。 选择Settings->Steempress，然后填入你想要的第一标签，然后在“Always add the default tags before the post tags.” 前打勾。 这样你填入的标签就会放在首个位置。

很简单的一个小技巧，以后不需要发布后再到steemit上修改标签了。

为什么要炒SBD价格？

(image source: pixabay
昨天拉仔@davidke20的一席话，让我对之前为什么SBD价格比steem高那么多的疑惑一扫而光。 可能大家都听过或者经历过之前SBD价格达到$15而Steem价格在$2. 我刚加入Steemit的时候，对SBD价格为什么那么高感到疑惑？SBD不是绑定$1 USD吗？为什么会涨到$15?

这其实是利用的STEEM/SBD的经济系统的问题的投机行为。

在steem系统里，SBD是和$1 USD挂钩的。无论SBD价格怎么样，都是按$1 USD算。 比如，你选择帖子收益比例是50%SBD/50%SP 在正常情况下，如果你的帖子收益是$10, Steem=$2, SBD=$15. 不算点赞收益的情况下，你会获得： 5 SBD+2.5SP, 换算成市场价，帖子收益是$80美金。

有些投机者就会利用SBD挂钩$1 USD这个机制，把SBD价格炒高，这样SBD就可以换更多的SP。 换成SP后可以发帖，再赚取市价高于$1 的SBD。

现在因为Debt Ratio(SBD总数/STEEM市值）已经达到5%，所以steem系统上的sbd print rate为0%。 意思就是帖子收益将不会得到SBD, 只有Steem和SP。 在这种情况下，投机者将不会再炒SBD价格了，所以会有一段时间SBD价格将维持在$1 USD. 所以如果你想等SBD暴涨的机会，可能需要一段时间。

如果SBD长期维持在$1 USD是对Steem的机制起有利作用的。 创作者，投资者将会有稳定的收益，而投机者将会离开。

HF20中的一个更新是把debt ratio从2%-5% 提高到9%-10%。 这个提高可以让更多的SBD被生产出来，不知道会稳定SBD的价格？
