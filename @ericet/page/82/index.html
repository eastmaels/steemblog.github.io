【故事汇#1】 故事进行中

阿皮是个小学的学生。
今天他带着鲜艳的红领巾准备过马路的时候，突然踩到了一坨屎，屎溅到了小皮的脸上。

请大家发挥想象编写故事~ 因为不是结局轮，所有尽量在结尾的时候留点悬念让下周的人可以接着写故事。

不知道大家有没有看过一档竞赛类的综艺节目叫 “故事王”
该综艺的核心是故事接龙，就是先起个故事的头，然后选手们按照这个故事的开头，继续讲故事
如果故事得不到评审的满意，另外一位选手将会按照故事开头，继续讲故事
如果评审对这个故事满意，下一位选手将继续上一位选手的故事继续下去。

感觉这个是挺有意思的活动，加上看到机机@julian2013和软哥@softmetal每日的主题接龙的启发，
就想着发起这个【故事汇】的故事接龙活动，让喜欢写故事的人可以参与进来。
而且大家一起来编写一个完整的故事也是一件很有意思的事。

每期的【故事汇】的故事将分成4个部分（开头，故事，故事，结局），为期3周

  • 开头故事由我发布
  • 故事 （第一周）：接开头的故事编写故事
  • 故事 （第二周）：接上周的故事编写故事
  • 结尾 （第三周）：把上面故事挖的坑都填好，编个结局故事
    每周，我将会从所有的故事中选一个故事，作为整个故事的主线让后面的参与者继续创作。

【故事汇】规则：

  • 每周三北京时间23：59分停止收故事（本周活动将于03月20号 23:59截止）
  • 按照前面的故事继续编写故事，故事类型随意，只要不要脱离主线故事
  • 字数不限，类型不限
  • 可以直接在这篇帖子回复你的故事或者你可以发帖，但是必须加上#cn-story的标签，便于我查找

【故事汇】奖励：

迁移wordpress网站文章

@vornix 的wordpress服务将在6月1号结束

使用他们家服务的需要在这之前迁移自己在vornix.blog上的文章/照片等信息，不然6月1号后这些将会消失

如果你还没找好新的wordpress服务，可以看看我之前写的文章，里面推荐2家免费wordpress站点：Vornix.blog结业

如果你已经拥有新的wordpress站点，按照下面的步骤备份并且迁移到新的站点：

备份文章/照片


  1. 登录你的vornix网站，选择Tools -> Export


  2. 选择你要备份的内容，如果只要帖子就选Posts，如果想备份全部的，就选“All Content”。选择好后，点击“Download Export File”，会生出一个xml的文件。



迁移文章/照片


  1. 登录你的新的wordpress网站，选择Tools -> Import, 安装并且运行Wordpress的Importer


  2. 选择刚才备份的xml文件,然后点击 ”Upload File and Import“


  3. 按照要求选择。如果你要把照片也搬过来，请在最后一个格子上打勾。


等待新站点完成迁移

点击广告获取BAT token（目前只限Brave浏览器 Dev版本）

Brave是一款基于Chromium的网页浏览器，宣称阻止网站跟踪器和移除侵入式网络广告

有趣的是，他们创建了BAT(Basic Attention Token),靠用户点击他们投放的广告来获取BAT（目前市场1 BAT价值$0.19)

目前BAT有3种使用模式：

  1. 打赏喜爱的网站/youtube频道/twitch
    每个月Brave都会给用户30 BAT用于打赏你喜爱的的网站/youtube/twitch频道. 这30 BAT如果一个月内没用完，将会重新放回奖金池。
    获得打赏的网站需要通过Brave验证才能获取打赏的BAT
    要打赏喜爱的网站/youtube/twitch频道也很简单，只需点击地址栏那里BAT的logo，就可以打赏了


  1. 自动贡献
    如果你选择自动贡献这个选项，Brave会根据你浏览一个网站的长度，次数自动分配BAT


  2. 点击广告获得BAT
    这个功能目前只有在Brave Dev版本里面有
    在你使用Brave浏览器的时候，时不时会有广告的提示冒出来，你只要点击那个广告就会获得BAT token
    目前一个点击可以获得0.05 BAT
    你可以设置广告出现的次数，目前最多是1个小时5个广告




目前Brave有3个版本的浏览器：

  • Brave Dev: 还在开发，并且没有正式发布的版本。只用于体验。下载链接：https://brave.com/download-dev/
  • Brave Beta：发布正式版前，给想尝鲜的用户先行体验。下载链接：https://brave.com/download-beta/
  • Brave：正式版。下载链接：https://brave.com/son492

尝试Brave浏览器有一段时间了，已经完全抛弃chrome了。
阻挡广告功能超赞的! 不但能把youtube上的广告给阻挡掉，连我看华语视频的网站dnvod上的广告都能阻挡掉（每次广告都要播60秒）！
加上最新版本支持chrome插件了，让brave更加完美了~

就算没有BAT，brave也是一个超赞的浏览器~
