最近避免买赞

最近CN区很多朋友帖子无故收到了踩。


看了一下帖子，既不是搬砖，内容也不错，但是被踩的几乎整篇帖子的收益都没了。为什么突然收到那么多人的踩呢？

观察了一下被踩的几篇帖子，发现一个共通点。那就是都买赞了。

一篇向tipu购买了1 steem的赞，2篇向smartsteem购买了5 steem的赞。

估计这就是社区大佬发起的清理卖赞机器人的活动。通过踩买赞的帖子，让卖赞服务没有生意，从根本上解决卖赞机器人的问题。

活动其实没啥问题，但是大佬的心中可能认为世界只有黑和白。

买赞的都是黑的，坏的。不买赞的才是白的，好的。

不在乎买赞人的文章是否是好文，买赞的就是坏的，就应该踩。

其实世界上还有夹在黑和白之间的灰白。

有些人写了一篇自己觉得很好的文章，想让大家看到，通过买赞的方式获得更好的推广。

那些大佬肯定会说，买赞就是获取收益，破坏steem经济生态，就应该踩。

那你写了好文，没人看，怎么推广出去？通过steemit的promote功能吗？

前几天看到一个新的卖赞机器人，叫randowhale。这个卖赞机器人的理念是，你发送2 steem到randowhale，他会随机给你1-100%点赞，而这2 steem就会自动发送到null进行销毁。

我个人觉得这个想法不错，可以销毁通货膨胀率高的steem。但是很多人发现，向randowhale买赞后，自己的帖子被很多大佬踩了。大佬认为这是买赞，不管你是否有没有销毁steem。

这就是大佬认为的世界，只有黑和白。

吐槽到此为止，最后提醒大家一下，最近不要像任何买赞服务购买赞，比如tipu，smartsteem，minnowbooster，upmewhale，randowhale，therising，等等。

Steempeak的新更新

这次Steempeak对钱包进行了很多更新。比如一键领取STEEM/SCOT收益，查看SCOT转账记录，查看SCOT代理，锁仓，收到代理数量，一键查看Steemmonsters卡片的价值，等功能

具体可查看：SteemPeak Release - Update v1.18 - Wallet Improvements and Witness / SPS Proxy

一键领取STEEM/SCOT收益

这个用户需求比较高，SCOT平台，steemworld都支持一键领取SCOT收益后，steempeak终于也添加了~

登录后，进入你的钱包。在右边会显示你待领的收益，点击“Claim Rewards”就可以一键领取STEEM/SCOT收益。再也不需要登录steem-engine一个个领取了~


查看SCOT转账记录

进入钱包，在转账记录那里，选择“Steem-Engine", 就可以查看Steem-engine平台所有代币的记录了~ 但是有个小缺点，就是没有选项可以过滤一些不想看的记录，比如过滤掉领取收益的记录，市场交易记录等。


查看SCOT代理，锁仓，收到代理数量

在钱包里，把鼠标移到代币上，就会显示SCOT代理，锁仓，收到代理数量。

比如移到我的LEO上，显示我现有68.351 LEO，锁仓了4.196和代理了1277 LEO出去：


一键查看Steemmonsters卡片的价值

这个只适用于steemmonsters的玩家。在钱包里，有一栏”Splinterlands Cards“，点击”on this page"后, 会转到peakmonsters的网页。网页的上方会显示你所有卡片的价值。


看了一下我卡片的价值，目前$389.09.

除了以上一些更新，还优化了搜索，添加了设置见证人代理的按钮。

Steempeak越来越好用了~

鸡蛋的冷知识


(Image Source: Pixabay)

前几天和老婆讨论鸡蛋，她说鸡没有受精怎么生鸡蛋？

但是我记得小时候，我外婆有养过一只母鸡，那只母鸡每天会生一颗鸡蛋，我每天早晨最开心的事情就是去鸡窝里拿鸡蛋。从来没见过有公鸡让母鸡受精。所以母鸡是不需要受精产蛋的？

Google了一下，还真是。母鸡是不需要受精产蛋的，但是没有受精的鸡蛋是不会孵出小鸡的。

这个冷知识我还是第一次知道！所以我们在超市里面买到的基本一般是不能孵出小鸡的。

无所不知的所长@annzhao说，要辨别鸡蛋是否能孵出小鸡，只需照一下太阳，看看蛋里是否有影。如果有影，就是可以孵出小鸡的蛋。如果没有，就不能孵出小鸡。

所长还说，为什么要每日去鸡窝捡蛋呢？因为母鸡生够一窝鸡蛋后，就不会再生了，等小鸡大了才会继续生，所以要日日捡蛋，防止母鸡不生蛋。

好多第一次知道的冷知识啊！

了解了这些冷知识后，想到了一个世界级的难题：先有鸡还是先有蛋？

如果先有母鸡，母鸡在没有受精的情况下，产出的蛋不能孵出小鸡。所以先有鸡行不通。

如果先有蛋，孵出的小鸡长大后，没有公鸡让她受精也不能产出能孵出小鸡的蛋。先有蛋也走不通。

所以无论现有鸡还是先有蛋，都行不通。那鸡或鸡蛋是怎么来的呢？难道是变异来的？
