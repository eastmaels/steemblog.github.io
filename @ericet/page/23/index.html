Brave/Chrome一起崩溃了

今早像往常一样，打开Brave浏览器。突然，弹出一个小窗口说我装的steem keychain崩溃了，让我重新安装。同时，Brave的页面也崩溃了：

检查了一下，网络没有问题。并且点击任何页面都没有反应。想尝试清除一下缓存也没有办法。

没办法下，只能卸载了Brave浏览器，又重新下了新的brave。但是还是同样一个问题。

既然不让我用Brave，那我改用chrome好了。打开chrome后，发现也是同样的问题。

既然是同样的问题，那就应该是chrome方面出了问题，毕竟brave是基于chrome开发的。

在网上搜索了一番，找到了原因：Google Chrome/Microsoft Edge Chromium version 78.0.x error “Aw, Snap! Something went wrong while displaying this webpage.” when using Endpoint Protection

根据今日SEP发布的问题，如果你的Chrome/Brave浏览器的Chrome版本是78.0.x，那就可能和Symantec Endpoint Protection（SEP）有冲突。

目前发现几个环境可以引起这个问题：

  • 使用Chrome版本是78.0.x
  • 使用Windows Server 2016 或者Windows 10 RS1
  • 使用任何装有低于14.2版本的SEP的Windows操作系统

看了一下我的几个环境，发现我中了2个：chrome版本78.0.x，并且使用低于14.2版本的SEP

文章里面给了暂时的解决办法。如果没办法升级SEP或者Chrome版本，可以使用暂时的一个解决办法：

运行的时候，关闭RendererCodeIntegrity 功能

运行后一切恢复正常~

如果你不想每次都使用命令的模式打开浏览器，可以修改便捷方式，在Target那里加上“-disable-features=RendererCodeIntegrity”。这样每次只要点击图标就可以打开了。

State of the DApps上一些有趣的数据

今天无意之中逛了一下State of the DApps的网站, 看了一下各个区块链的DApps排名，发现了一些有趣的数据。

steemit是STEEM的第一应用，但是每日活跃人数不如Steemmonsters



steemit在所有区块链的DApps里面，综合排名第6。Steemmonsters紧随其后，排名第7.

虽然Steemmonsters的综合排名不如steemit，但是每日的活跃人数(3009)大于steemit（2479). 下周steemmonsters将提供免费试玩游戏，每日活跃人数将可能提高，并且有机会挤掉steemit，成为STEEM上的第一应用。

SportsTalkSocial 成为所有Tribes里面排名最高的社群


我对SportsTalkSocial平台是又爱又恨的。喜爱他的原因是，本身喜欢体育，并且觉得体育平台的发展潜力很大。恨他是因为本来有一个很好的起点，硬是被发起人还没开始发展就差点搞毁。以为SportsTalkSocial将会完蛋，没想到越发展越好。目前的排名和Partiki差一位，但是高过Busy和Esteem。还有一个没有想到的是，CN区的体育集资号@cn-sports 目前代理数量超过700万SPORTS，成为继SPORTS官方账号外，代理数量最多的账号。

查了一下其他的Tribes，发现居然没有LEO平台？如果有加进去的话，排名应该也不差。

区块链平台新贵Klaytn?



上一会我上State of the DApps的时候，前10应用里面，有至少4个应用来自STEEM。但是这次查看，发现已经变了天。
前10应用里面，有4个来自一个叫Klaytn的区块链。而且这4个应用的每日活跃人数也很多。

稍微查了一下这个区块链的信息，发现是一个在韩国很火的网络公司Kakao开发的（韩国人的微信KakaoTalk也是这个公司开发的）。

虽然目前Klaytn的应用不多，但是还算齐全。有化妆品应用，交易商品应用，视频内容应用，约会应用，区块链的instagram应用，宠物平台应用，等等。如果你对这些应用有兴趣，可以去了解一下。

虽然数据不完全代表平台的发展情况，但是可以用于了解一下平台的基本状态。

支持一下Steempeak的SPS

昨天收到steempeak的消息，说之前的SPS提案没有成功。。。

这么好的项目居然没有成功？发起人觉得可能之前的提案提的都是一些小更新，不足够获得大家的支持，所以这次他们改变方案，加入了steempeak未来的目标：

  • 支持社区功能的前端
  • 支持SMTs
  • Tribes更新
  • 游客账号功能
  • 从游客账号转换成正式账号
  • 支持赞助朋友新账号
  • 提供工具给社群，社区用于引入和支持新人
  • 一个叫”Steem Glossary” 的项目开源
  • 加入学习模式，让新用户了解STEEM
  • 更多的分析工具
  • 更多的方式赚取收益，比如打赏，广告收益分享，赞助创作人，付费阅读等
  • 添加自动点赞工具
  • 添加@提示
  • 手机APP的开发
  • 支持多语言
  • 更新搜索功能
  • 还有很多新功能

具体提案可以查看：The Always Up to Date Proposal to Support SteemPeak

Steempeak是目前功能最强大的STEEM前端，但是奇怪的是，这么好的前端居然SP数量不多。目前这个应用的主要收益来自他们的另外一个项目：peakmonsters和社区捐款。

如果这次的steempeak的SPS提案还是没有通过，我真的要怀疑SPS是否只是大鲸们用于圈钱的工具了。

废话少说，尽点自己的一点绵力来支持Steempeak这次的SPS

如果你也想支持一下steempeak，可以通过https://steemitwallet.com/proposals 或者https://steempeak.com/me/proposals 对这个提案进行投票。
