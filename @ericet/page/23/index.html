查看粉丝总SP的小工具

你是否知道要获得busy.pay 和esteemapp的点赞，除了需要用他们的平台发帖子外，他们的点赞比例取决于你的粉丝的实际SP总和。
所以创建了这个小工具让大家了解:

  • 自己粉丝的总SP
  • 如果使用busy发帖，busy.pay会给你多少点赞
  • 如果用esteem发帖，esteemapp会给你多少点赞

工具链接：https://ericet.github.io/Steem-tools/SteemianExplorer
开源代码：https://github.com/ericet/Steem-tools

怎么使用？

输入你要查的账号，然后点击“View”就行了

如果你想查找多个账号，可以同时输入多个账号，每个账号用逗号分开就行了：

*注： esteem使用的数据来自于steemdb.com, 这个数据有严重的延迟情况，所以esteem实际点赞比例和上面列表里面的比例会有点差别

Leodex V3更新（K线图，代币转换和空投服务）

SteemLeo在周一时发布了他们去中心交易所LeoDex的一些更新：Introducing Steemleo Dex V3 | Candlestick Charts, LeoServices, Token Swaps and More!

这些更新包括K线图，代币转换和空投服务

K线图

进入他们的市场交易页面，就可以发现多了K线图:

之前只有交易深度的图，所以看不出价格的走势。但是添加了K线图后，可以很明显看出代币的走势。

代币转换

点击上方的“Swap”就可以进入代币转换的页面

按照目前Steem-Engine的购买流程，如果你想买DEC，需要先把自己手上的Steem换成Steemp，然后用steemp购买DEC。如果你想卖DEC，需要把DEC出售，换成Steemp，然后再提现steemp换成steem。

这流程有点儿复杂，而LeoDex的Swap服务简化了这些流程。

你可以直接使用Steem购买DEC或者直接使用DEC换成STEEM, 省去了中间steem<=>steemp的过程

目前只支持DEC，未来会添加其他代币。

空投服务

很多新上线的代币都会给用户空投他们的代币。Steem-Engine也有提供空投代币的服务，但是费用比较贵，需要100 ENG(按照目前市场价格，大概60 STEEM）

LeoDex也提供了空投工具，但是费用比较低，只需10 STEEM！

目前Steem-engine在开发他们新版的Steem-Engine页面，但是从测试版上看，体验不如LeoDex，而且费用都比LeoDex来的高

不知道未来Steem-Engine会不会被LeoDex抢掉饭碗
