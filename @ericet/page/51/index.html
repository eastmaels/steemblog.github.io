Steemconnect v3 测试版上线（支持更换节点）


@fabien 发布了新的版本的Steemconnect的测试版本：SteemConnect 3, beta release looking for testers

这个新的steemconnect 3将会有让你通过网站，桌面程序和浏览器插件的方式登录steem账号。

国内用户最关心的登陆问题将会在这个版本得到解决

由于api.steemit.com被封锁（旧版的steemconnect使用这个节点来验证登陆steem的各种DApps），使得国内用户如果需要登陆上busy,partiko,steempeak需要首先翻墙登陆steemconnect。

这让steem的门槛变得越来越高，也不利于steem在国内推广。

这个steemconnect 3添加了节点设置的页面，大家可以修改节点登陆各种DApps.

测试版的节点设置连接：https://beta.steemconnect.com/settings


国内用户可以在这里修改节点连接，比如修改成还没被封锁的anyx.io 或者steemd.minnowsupportproject.org(steemconnect如果可以添加可选择的节点列表就更好了)

目前steemconnect团队将在二级域名上测试新版本的steemconnect，一个月后，如果没有什么问题，steemconnect 3将正式取代旧版的steemconnect 2.

上线后将进一步测试并给与修改节点的具体步骤 —

Likwid 帖子收益快速转换服务

热门上的一篇帖子介绍了一个新的服务：Announcing @Likwid, The reward liquifier!

大家都知道发帖时无论选择100%SP还是50%SBD/50%SP, SP部分需要13周才能全部转换成STEEM（每周1/13如果power down)

@likwid 提供的服务将让这漫长的等待时间缩短，从13周变成7天。

这个服务的原理是，把@likwid设置成你的帖子受益人，等帖子结算后，@likwid收取1.5%费用后，把剩下的收益全部转回给用户。

怎么使用这个服务呢？

通过steempeak.com 把likwid设置成100%受益人，然后发帖就好了。

帖子结算后，@likwid将自动扣取1.5%费用，把剩下的收益全部转回用户。比如帖子获得50SP/50 SBD, 你将获得49.25 STEEM和49.25 SBD。

目前@likwid账号拥有于6300 STEEM，背后的团队是创建Steemium（一个很便利的一键买赞服务）的人： @lightproject @dzivenu 和 @transisto

怎么查看自己在SCOT平台上的VP呢？

最近基于STEEM的SCT平台火爆CN区，大家除了发帖外也积极给sct帖子点赞。但是有一个问题一直搞不清楚的是，SCT平台上点赞的VP和STEEM平台上的VP是否一样呢？

根据ScotBot的设置参数
vote_power_consumption
vote_regeneration_seconds

ScotBot的设置是需要不同的VP的

那要怎样看自己的VP呢？

昨天在steemcoinpan.com上看帖子，看到这篇帖子내 SCT 보팅파워는 어떻게 될까?

虽然我看不懂韩语，但是图片我看的懂啊！这不是看VP的api吗？

API链接：https://scot-api.steem-engine.com/@用户名

比如我要查我在各个SCOT的平台的一些信息，就可以用这个api查看：https://scot-api.steem-engine.com/@ericet

这是从api得到的一些信息：

从上面的信息，我知道我的点赞VP还有64.09%，待结算的SCT有28.676个，总共Stake了88.874 SCT。

虽然SCT平台上的VP和STEEM平台上的VP不同，但是他们仍然使用相同的点赞比重（Voting Weight)

所以你在SCT上点赞帖子，同样耗费STEEM上的VP。

steem-engine团队有计划对这个问题进行改进，未来在SCT平台上点赞就不会耗费STEEM上的VP了

最后感谢SCT的平台，没有这个平台前，我很少会去看韩语的帖子。但是有这个平台后，发现kr社区讨论的很多事情CN区是不会讨论的。比如SMT vs SCOT。感觉发现了新世界！

Ethereum链上获奖项目Cache将登陆STEEM区块链


这个已经算是旧新闻了，但是很多CN区的朋友还不知道，这里就简单的介绍一下Cache这个项目。原文：From Ethereum to Steem

Cache是基于ETH链上的一款社交游戏DApp，是2018年ETHDenver的获奖项目。

这款游戏和drugwars毒品交易的设定差不多，但是主题是交易虚拟货币（开设交易所，发币，交易虚拟货币，赚取虚拟货币等）。

这次从ETH链搬到STEEM链上主要的原因是：

  • STEEM无费用转账。在ETH链上，转账是需要GAS的。
  • STEEM快速转账（3秒确认）
  • 社区。 STEEM拥有区块链上最多人数和最活跃的社区。

这次如果Cache顺利登陆STEEM区块链，将对STEEM是个大利好！其他区块链的DApps将会考虑迁移到STEEM链上发展。

目前Cache团队正在努力在STEEM上推出游戏，大家可以提前注册邮箱获取一件免费的游戏宝物。

注册链接：https://www.cache.gg/
