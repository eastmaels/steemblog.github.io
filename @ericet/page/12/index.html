编程者的神器---Code Server

之前看@ety001的帖子，无意中了解到这个神器，从此不能自拔，实在是编程者的神器啊！

Code Server其实就是云端版的Visual Studio Code.

一般我编程的环境就在3个地点：公司电脑，家里台式电脑和家里的笔记本

有时在公司写了一半的代码，走的匆忙没时间把代码push上去，到家后就没办法继续写下去

有了Code Server，代码都在云端，不需要push，到任意一台电脑打开浏览器，登陆Coder Server就可以继续编程了

还有就是，我的笔记本年代久远，配置有点低。代码在笔记本上运行缓慢又耗内存硬盘。但是在云端就没有这个烦恼，代码都在服务器上运行，使用的是服务器的内存和硬盘，笔记本上最多是费了点网络流量。

推荐编程爱好者使用！

Code Server的原代码：https://github.com/cdr/code-server

搭建Code Server可查看这个视频：https://www.youtube.com/watch?v=UeP6VQjMw8Q

Costco货源充足

前几周新冠肺炎疫情刚爆发的时候，写过Costco里的水，厕纸，面包等被一扫而空

几周过去了，家里的食物也吃的七七八八了，所以去了趟Costco补货

来到Costco后，发现和以往有了很大的不同。

首先进门的地方，有人会给你一张湿纸巾让你清理一下购物车的把手。清理完后并要求你给自己的手消毒一下

店里和平时比起来少了一大半人

以为来的比较晚，东西应该剩下的不多。但是看到下面画面，感觉自己是瞎担心：

肉类货源足足的！

之前一包难求的面包也多多的！

最恐怖的是水，到处都是水！.

厕纸也足！大家都不用担心没有厕纸擦pp了！

结账的地方也有变化。在结账地点每隔一段距离画了一条黄线，让大家按照这个距离排队

结账的柜台也添加了一个塑料的隔离板，这样收银员和顾客就没有直接的接触了

买了一车的东西，又可以继续在家宅个一个多月了

新冠疫情下，餐饮业受到重挫，公司打折应对

我的公司是做餐饮连锁的，自从新冠肺炎爆发后，公司业绩受到了很大的打击

老板上周发来邮件说，连锁店最近的订单少了将近40%，线上订单从9%增长到22%，第三方从8%增加到14%

看到线上订单增长的趋势，公司决定对线上订单提高打折幅度，线上订单全部5折

折扣上线第一天还没2小时，网站和APP都被挤爆了，服务器的CPU严重超负荷

公司下令关闭一些服务来释放部分CPU。一整天大家精神都在紧绷状态

第一天的订单是平时的10倍多，第二天15倍，第三天18倍，今天将是活动的最后一天

这个打折力度是有史以来最大的，公司会给连锁店补贴进行这次的折扣，所以基本公司都在亏本。但是好消息是，新顾客增加了几十万。只要留住一部分新顾客，未来还是可以赚回本的

疫情后，大家估计会着重发展线上这一领域了吧

ionomy上交易留点心

ionomy是第一个上线可以交易HIVE的交易所，加上这个交易所不需要KYC（身份验证），所以很多人使用这个交易所交易。

但是这几天发生在我身上的事，有必要让大家知道，对这个交易所留点心

前几天ionomy刚上架HIVE/BTC，我就赶紧去把手上的HIVE出售了

出售HIVE后，想着可能孙老板会来砸盘，所以就恶趣味的在0.00000003 BTC/HIVE 挂了50万个HIVE的买单（总价大概$90)

过了几天，HIVE也没有被砸下去的迹象，再加上ionomy的交易所看起来不太正规，孙老板不太可能会在这个交易所砸盘（要砸可能就在火币和币安砸），所以就把挂单的50万HIVE取消了

点完取消，发现挂单的BTC没退回我的钱包（因为当时钱包里面价值为0，如果取消挂单，价值不应该为0）。在网站上找了半天，也没找到哪里可以看到所有的挂单记录，所以发邮件问ionomy的客服

经过几天的交流，今天终于把BTC退回来了

虽然BTC退回来了，但是如果当时没有留点心，很有可能不会发现取消的买单的币没有退回来

所以大家在这个交易所交易，切记留点心，并且交易后把钱包里面币移到其他地方去。总觉得这个交易所某天会突然跑路

同步帖子到HIVE链，好文自动进驻HIVE CN中文社区

之前为了把STEEM的帖子自动同步到HIVE链上，写了一个很粗糙的程序自动同步：Steem2Hive：自动同步 Steem 帖子到 Hive

这个程序步骤有点繁琐（代理，授权，加标签），经过阿盐@robertyan的提点，改写了一下程序，把步骤缩减到了2步（授权和加标签）

写完刚巧看到甜心@sweetsssj在HIVE上创建了HIVE CN中文社区，希望中文用户把自己的好文发到这个社区，她会在社区里面挑选文章点赞

所以给同步程序又加了一个功能，就是如果要同步的帖子带有cn-reader或者cn-curation标签，同步到HIVE链上的帖子会自动发到HIVE CN中文社区里面。这样帖子会有机会获得甜心点赞！

怎么同步帖子并且自动发到HIVE CN中文社区？

  1. 授权 @steem2hive账号（Hive上）
    使用这个链接授权：https://hivesigner.com/authorize/steem2hive 这个操作只需一次
  2. 发帖的时候加上steem2hive标签
    任意STEEM的平台发帖，只要加上steem2hive的标签，就可以自动同步到HIVE链上

发布到 Hive链 的帖子，收益的 1% 将会给 @steem-drivers，用于开发和支持开发者。Steem 上的帖子不抽成。

为了避免过多水贴发到HIVE CN中文社区，帖子带有cn-reader 或者cn-curation标签的才会发到社区里面

如果滥用cn-reader或者cn-curation标签，账号将会拉黑账号，不会自动发到中文社区里。

希望这个程序会帮到大家
