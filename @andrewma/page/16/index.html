我估计要从busy转用steempeak了~

自从Steem-Engine上线以来，各种应用层出不穷，比如SCT,AAA,PAL,Sports等等，甚至有些让人眼花缭乱 ，随之而来的问题就是在使用上变得麻烦了，虽然发不同应用的帖子只要打上相应的标签就可以，但如果要查看收益，就需要登陆不同的平台去查看，很不方便。

今天从村长@ericet处得到消息，steempeak正在做整合各个应用的入口，迫不及待赶紧去看看。网址： https://steempeak.com/tribes。

果然，steempeak已经整合了PALnet和SplinterTalk，准备整合的还有TRIPLEA,SCTD等等。

只要把鼠标移到帖子收益处，就可以查看不同token的收益，还有比例分配和结算日期，真是不错啊。

这真是一件大好事情。我估计以后要放弃busy，转用Steempeak了。

Grab the doll machine

哥哥大概用了一个月时间完成这个作品“液压动力抓娃娃机”。

制作过程中，主要在性能，外观和操控性上考虑得比较多，主要设计体现在以下几个方面：

1.使用PPR管作为抓臂的支撑，开始还考虑使用PVC管，综合比较后选择PPR管，因为PPR硬度强，便于割锯，可以节省工作时间提高工作效率

2.选择的纸箱箱体适中，主要考虑适合六岁以下小朋友玩耍

3.选择使用10ml针管作为液压动力，还是主要考虑适合小朋友使用，比5ml针管动力更强，提高了操控的灵活性

4.为了美观，把纸箱外面涂成了彩色

不足和提升点：

1、由于美工设计不足，外观颜色搭配并不如人意，以后版本可以考虑继续提升

2、在力臂相接部分还要精打细磨，特别像前后动力臂和上下动力臂之间间距，伸缩的控制等方面要再努力，提升可操控性

This is mainly reference my previous post : https://busy.org/@andrewma/grab-the-doll-machine

我为什么买看空期权

我为什么买看空期权？

上周大概这个时候，BTC还在9000多刀徘徊，ETH在270刀左右。我当时第一次购入了ETH的看空计划委托期权，触发价285刀，成交价290刀，结算日是上周五，但是当周五的时候，币价突然快速上攻，当我收到触发价被触发的提醒时，不由得心情有些慌张，赶紧把订单撤销了，所以最终没能成交。谁知道，刚刚过去几天，周末时候BTC已经突破10000刀，最高到达11000刀，于是我在周一又下了一单ETH的看空委托期权，触发价325刀，成交价330刀。今天中午11点多，收到价格被触发的提醒，这次还没等我来得及详细看盘，ETH价格已经升到335刀左右，就这样，这次的订单成交了。总结一下，购买期权委托的经历。

为什么买ETH？

因为BTC价格太高，超过个人所能承受的风险范围；EOS受消息影响比较大，价格波动太大，而ETH价格走势相对比较平稳。其他如LTC，XRP没有太多关注和研究。

为什么买计划委托？
一是没有时间天天看盘，二是即使天天看盘可能更受干扰。买计划委托相当于买个希望，而且据最近一段时间观察，主流币走势往往会在周五小幅下跌，周末又会反弹向上。

为什么买看空？
主要是和自己的持仓作对冲，毕竟持仓占有比例更高，即使币价上涨，期权亏损，但持仓还是会涨的。如果币价下跌，持仓亏损，可以从期权找回一些安慰。

最后，希望Steem-Engine也能早日提供期权下单功能，如果开通，建议优先对交易量大的币种开通期权操作，如SCT，AAA，PAL等。
