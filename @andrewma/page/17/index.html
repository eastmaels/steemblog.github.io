戒酒防微

戒酒防微

Image Source:https://pixabay.com/photos/cobblestones-away-paving-stones-393455/

视其善者，取以为师；视其恶者，用以为戒
以史为镜，学皇帝所学

出自《资治通鉴外纪》卷二

大禹在位的时候，有一个人叫仪狄，特别擅长酿酒。有一次，他把刚刚酿好的酒进献给大禹，禹饮过之后，觉得此酒香醇美味，非常可口。但是，大禹却说道：“后世之人，肯定会有因沉迷于酒而误大事，甚至误国的”。此后大禹便疏远仪狄，并且不再允许他进见，同时，也不允许他进献美酒给政务官员。

古人用酒主要是在祭祀的时候使用，或者在士大夫以上的阶层人士的宴会上使用，这种礼仪是有必要的，但如果饮酒过度，不仅伤身体，更会妨碍政务，很多祸害都是因饮酒过度导致。所以圣人往往在开始的时候就小心谨慎，防微杜渐。

夏朝最后一个君主桀，传说他喝酒像牛饮水一样，因为总是饮酒过度而性情暴虐，最终导致夏朝灭亡。看来，祖宗的担心是很有预见性的，不听亡国啊！

Candy Machine

This is an candy dispenser made from waste materials.

Tools and materials are:
1, hot melt glue gun

  1. Ruler, pencil, scissors, carving knife: used for drawing and cutting
  2. Mineral water bottles
  3. Used cartons

Production process:

  1. Cut the used water bottle with scissors from the bottom, leaving the length about 12cm, then cut the top part from the outside of the bottle, do not cut, like a container with a lid, from which the candy can go
    2, calculate the size, first draw on the cardboard, and then according to the pattern of the drawing with a carving knife cut out the cardboard box two circles, leaving the rest of the part, and then cut out four, because the cardboard is still thin, stick the two flat, so that there are only two flat, two circular border. The circular frame is used as the base to hold the container, and the pincket is used to block the base.
  2. Use the hot glue gun to glue the water bottle cut out from the previous to the cardboard, so that the model of the candy machine is finished.

This is just the initial model of the candy machine, and there’s a lot of need to improvement, the next plan is:

  1. Increase the number of containers to 3 to 5. Use thin and long ones instead of short and thick ones
    2, the switch USES the button mode, one candy go out while pressing a button once, that’s more convenient than the hand pull mode
  2. Beautify the appearance

这是利用废旧物品作的一个糖果机原型。
工具和材料有：
1、 热熔胶枪
2、 直尺，铅笔，剪刀，刻刀：用于画图纸和裁剪
3、 矿泉水瓶
4、 废旧纸箱

制作过程：
1、 把用过的矿泉水瓶从底部用剪刀剪开，保留长度大概12厘米左右，再把上面部分从瓶子外沿剪开，不要剪断，像是一个带盖子的容器，糖果可以从这里面放进去
2、 算好尺寸，先在纸板上画出来，然后按照图纸样式用刻刀把纸盒子剪出两个圆型，留下剩下的部分，再剪出四个平托，因为纸板还比较薄，把两个平托粘在一起，这样就只有两个平托，两个圆型边框。圆型边框是用来作为盛放容器的底座，平托是用来挡住底座，以后一拉平托就会掉出一个糖果。
3、 使用热熔胶枪把之前剪出来的矿泉水瓶和纸板粘在一起，这样糖果机的模型就做完了。

这只是糖果机的初始模型，还有很多地方可以再完善，下一步改进计划：
1、 增加容器到3至5个，使用细长型代替短粗型
2、 开关使用按钮方式，按一下出一个糖果，比手拉方式更便捷
3、 对外观进行美化修饰

终于摆平了新版steemconnect~

上周折腾新版steemconnect没有成功（https://steempeak.com/cn/@andrewma/steemconnect），今天想再试一把。

首先还是想试一下sc推荐的desktop app，点击网站接直接到了https://github.com/bonustrack/steemconnect/releases.

用就用最新的，V0.1.2，我主要试了红框标识的两个应用，第一个是chrome扩展程序，下载，问了问度娘如何在chrome中增加扩展程序，增加成功，工具栏显示出了sc的小图标，小小一阵欣喜，但点击图标后没有任何反应？不知道为什么？是不是要梯子才行？白高兴了。第二个应用是exe文件，但必须要求64位机器，可惜我的机器太老，还是32位的，此路又不通。

没有办法用desktop，只能回去再试试登陆授权的方法了。重新回到sc的登陆首页，点登陆后发现需要输入keychain密码，keychain同样需要梯子（之前试过），但下面有个import account，点这个试一下，这里要输入master key 或者 active key，登陆成功再点击授权，还是报上次出现过的错误，本以为只是时间的提示。

这次仔细看了一下，前面是now，后面是trx.erp，不知道trx.erp是个什么东东，但感觉是不是时间不一致的提示呢？报着试一试的想法，修改了机器的系统时间，大概和提示中now的时间看齐，再返回重新登陆授权，这次终于成功了！

过鲁祀圣

过鲁祀圣

Image Source:https://www.pexels.com/photo/brown-concrete-wall-surrounded-by-trees-1653823/

视其善者，取以为师；视其恶者，用以为戒
以史为镜，学皇帝所学

汉高祖刘邦在位时，因淮南王英布谋反（其原属于项羽，后归顺刘邦），西汉初年，汉高祖亲自带兵讨伐英布，并旗开得胜，英布逃往江南，但最终被刘邦活捉。

汉高祖凯旋而归，途经山东曲阜县，这是孔子的出生地，并立有孔子的坟墓。汉高祖命令用牛、羊、豕三种性口作为祭品，并亲自拜祭孔子。

孔子虽然是一代圣人，但在当时官位并不高，仅是鲁国的一名大夫，而且在孔子死后，战国时期的各个诸侯并没有十分推崇孔子的儒家学说，后来秦始皇焚书坑儒，又烧毁了很多孔子的书籍。汉高祖刘邦以天子之尊，采用最高仪式（古代天子、诸侯、士大夫供奉祭祀祖先的方式）来祭拜孔子，可见其对儒家学说的认同和对孔子的尊敬。后人对孔子的敬仰和推崇，便是由此开始的。

出自《汉书-高帝纪》
