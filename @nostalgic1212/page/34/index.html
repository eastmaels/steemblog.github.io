「Vicky's 资讯」《天天向上》：优质偶像王一博，带大家熟悉日常防火安全知识

All Image Source

多年不看综艺的我，最近因为粉上王一博开始刷《天天向上》。

《天天向上》作为芒果台继《快乐大本营》以来受众最大的综艺节目，王一博能跻身成为常驻主持，起初也受到了不少质疑之声。

但他凭借自己的实力和人气，以及谦逊有礼的待人处事态度，后来已然成了节目的团宠。不仅汪涵哥爱护之，连晓明哥在这期节目中对王一博再三肯定。

在节目的最后，王一博努力克服身体不适，最终完成考验，成为湖南明星志愿消防救援队新队员。

常青综艺节目+人气明星，强强联手去科普消防安全知识。这样正能量的组合，请再给我来一打！

在节目之后，耶啵还为自己家乡的消防安全宣传工作出了一份力。

由@洛阳消防官微放出了王一博录制的消防安全宣传片，大家不妨跟着耶波一起学习消防安全知识。

1、起火不要乘坐电梯，以防断电时候无法顺利逃离；

2、居民楼道要保持畅通，杂乱物品不要随意堆放，尤其不要堵住消防设置和安全出口；

3、发生火情时，切记在楼梯间内俯身并用湿毛巾捂住口鼻，以防浓烟进入；

大家记住了嘛？

没记住？那就再看几遍嘛，哈哈。

（The End）

「Vicky's 资讯」《天天向上》：你有多久没听过电台了，最喜欢哪一档节目呢？

All Image Source

学生时期的时候，我们身边还没有手机、iPad、网络，连台式机的电脑还不是很普及。娱乐方式除了影院和CD机，我们能接触到最多的就是收音机。

收音机是个好东西啊，可以听卡带，也可以听电台。

随着物资的丰富，我们成天抱着手机享受着网络带来的便利，电台也逐渐被各类博客节目所取代。

大家在什么场景下还在听电台呢，我想大多数都和冰冰一样，在开车的通勤路上才会听吧。

最近一期的《天天向上》就怀旧了一把，带我们再次走进电台的世界。

节目组请来的都是业界重量级的主持人，有惊悚故事大师张震、电台情感快刀叶文、电台老歌知己李峙，还有电台高峰诗人国生和电台热点梗王伟胜。

顺便一说，冰冰之前就差点被《张震讲故事》吓到晚上不敢睡觉…

这些大咖们会和汪涵、王一博、大张伟等常驻主持们产生怎样的火花呢？

「Vicky's 资讯」《我们的歌》：肖战演唱《满足》和《跟着感觉走》，时隔四年重回舞台的他表现如何呢？

All Image Source

那英和蔡健雅问及肖战，演戏和唱歌如何选择的时候，耿直的肖战直言自演戏是目前工作的重心，同时音乐一直是他的一个梦想，不会放弃。

在《我们的歌》节目中，肖战选择了自己的歌《满足》，这首歌对于重回歌唱舞台的他来说意义非凡。他温暖、诚恳的声线，瞬间捕获了两位老师的耳朵。

“唱歌是我来到舞台的一个初心，所以这一次，时隔四年再回到这样的舞台，就像这首歌一样，跟着感觉走。”

舞台上，他真挚地这么对观众和老师们说道。

在四位老师激烈的争夺中，肖战选择了带过很多新人歌手的那姐。从后台的花絮中来看，那姐已经开始给萌新肖战很多技术上的指导。

除此以外，周华健与徒弟蒋一侨组合，相信有合作基础的两人应该会分外默契。

“落单”的阿云噶与蔡健雅组合，我个人很喜欢她的歌哒。

火箭少女李紫婷和黄凯芹搭档，这对年龄差距很大的组合会带给我们怎么样的惊喜呢，超期待的！

本期最后，四组风格各异的组合共同拿起出战的战麦，希望在后面的节目中可以看到他们精彩的演出。

「Vicky's 短评」《诛仙》：比电影更精彩的，是口水大战

看了中秋档的《诛仙 I》，我突然感慨颇多。

关于电影本身的讨论，我在长图文《《诛仙》口碑扑街是谁之过？流量明星：这个锅我们不背！》已经谈的比较深入了，有兴趣的童鞋欢迎移步那里阅读。

微头条里，我想谈谈这部电影以外的一些东西。

翻看电影宣传期时候的新闻，我很惊讶的发现大家讨论更多的并不是电影本身，而是各路人马间的互喷。

原著粉丝和演员粉丝互喷、演员粉丝和diss流量明星的影评人互喷、两位演员因为争番位导致粉丝间互喷、还没看过电影就无脑打出五星和一星的用户之间互喷…

在某位影评人的文章里，我看到他用了以下评论：上映当天呈现离奇分柱形态的豆瓣评分（6.7分得电影，确实打五星和四星的最多）。

即便是当时没有关注这部电影的我，都曾在热搜上看到类似“《诛仙 I》首日票房过亿”、“XX演技炸裂”的热搜。

观众们仿佛分成了两类人，狂热的互喷者们和一脸懵逼的吃瓜群众。

实话实说，那位喷流量的某知名影评人显然是错的。

这里面的主要演员，无论是肖战、李沁，还是孟美岐、唐艺昕，他们演的都不算是一塌糊涂，不应该成为这位影评人口中的背锅侠。

话又说回来，他们的演技依然非常稚嫩，其实并不算太出彩。

即便是我喜欢的演员肖战，在这部电影里中我看不到他在《陈情令》中所表现出来的放松和灵气，被粉丝们捧到“演技炸裂”的高度显然也并没见得有多理智。

一部电影的好坏，与导演、编剧、服化道、配音、配乐等休戚相关，而演员只是这诸多零件中的一小部分。

希望大家能理性对待电影和演员，不要一股子的谩骂和包容，回归那个理性看待影视作品的年代吧…

（本篇完）

「Vicky's 影评」《诛仙》口碑扑街是谁之过？流量明星：这个锅我们不背！

All Image Source

《诛仙 Ⅰ》中，也是有一些演技不错的实力派演员。但如果我不说破，我想大多数观众是认不出来的。

被人吐槽是照着灭绝师太来演的，陆雪琪的师傅水月大师，其扮演者是原版《新白娘子传奇》中许仙的扮演者叶童。

在《绣春刀2：绣罗战场》中因为饰演了信王这一角色开始崭露头角的刘端端，在《诛仙 Ⅰ》中基本变成了nobody。

一部作品中，演员最终呈现出来的样子，除去他们本身的演技和努力，很多时候还和拍摄团队、剧本等外在因素息息相关。

《诛仙 Ⅰ》恰恰没有给演员足够发挥的空间，使得老戏骨们演的束手束脚，更何况是流量新人们。

说句大不敬的话，我觉得这部电影的缺点，很大程度来都来自导演程小东。

这位导演的名字大家也许不算熟悉，但他确实是位实力派，曾经将金像、金马最佳武术指导拿了个遍。

他是大名鼎鼎的《大话西游》系列、《倩女幽魂》、《新龙门客栈》、《笑傲江湖：东方不拍》的武术指导，这份简历完全可以证明程小东的实力。

然而，他用自己拍武侠的老套路去拍《诛仙 Ⅰ》，被指出玄幻感太弱也就不是偶然了。

再者，我们可以看到这部电影拍摄团队的其他成员有：动作特技熊欣欣，配乐泰迪罗宾，造型师奚仲文。

他们确实是业界的老前辈了，但并没有向观众交出令人满意的答卷。

尤其是造型团队，从主角到配角千篇一律的服装和无力吐槽的发型让人大失所望，强烈推荐去学习下隔壁《长安十二时辰》、《如懿传》、《延禧攻略》、《琅琊榜》、《陈情令》这些剧组。

整部电影均选择用了配音，但配音团队毫无诚意，致使声音和演员口型对不上，分分钟让人出戏。

当下的互联网，无论是专业抑或是业余的影视评论，大家有了一种默契。

似乎一部电影中，我们只看到了流量明星，看不到导演、编剧、服化道团队、配乐、配音等等。

影视剧口碑好的时候，就夸流量明星们超水平发挥，演技炸裂；

反之，口碑不好的时候，流量明星们就被拖出来各种鞭笞。

为什么大家选择一上来就骂流量？

因为骂流量本身就能带来流量，哪怕是素人，借此火一把多涨几个粉也是幸事一桩。

《诛仙 Ⅰ》确实不是一部好片，但这个锅，流量们不背。

（The End）

「Vicky's 影评」《诛仙 Ⅰ》口碑扑街，是否完全是流量明星的锅？

All Image Source

张小凡作为《诛仙》的主角，本身就集结了一大部分书迷的眼光。其饰演者肖战，是时下当之无愧的当红明星，其流量不可小觑。

《诛仙 Ⅰ》上映的时候，《陈情令》的热度还未完全褪去。陈情女孩们还没有出戏，纷纷回忆着剧中那个潇洒自如又兼济天下的魏无羡。

导演选用肖战的时候，《陈情令》还未播出，他还只是刚出头的男团成员之一。不得不说导演相当的慧眼识珠，押对了这个宝。

《诛仙 Ⅰ》中，张小凡这个“火​头军”第一次亮相的时候，我相信很多人都会和我一样的反应。

这人…是谁？

一瞬间，我还以为直接穿越到了隔壁《捉妖记》片场。

观众熟悉的神颜不见了，取而代之的是一副灰头土脸、面色蜡黄的样子，哪里有半分魏无羡的样子。服装上，也再不是宽衣窄袖的仙家风范，反倒是有些褴褛的“丐帮”模样。

肖战，变丑了么？

别人化妆是为了遮丑，这事儿到了肖战这里，恰恰是反着来的。

涂黑粉、化晒伤妆、点雀斑，所有这些都是为了让肖战与张小凡这个角色更加贴合。

被观众所吐槽的丐帮服装，也许正是为了将肖战欣长的身型隐藏在人群之中，满足那个道行不高、天资极差、胸无大志的人设。

从《陈情令》到《诛仙 Ⅰ》，肖战的演技当然算不得当时被捧上热搜的“演技炸裂”，但作为观众我们依然可以感觉到他对每个角色的认真。

正像他之前说过的那样：把自己打散揉碎，然后融入到每个角色之中，想象自己就是这个人。

除了肖战，《诛仙 Ⅰ》中还有其他主要角色也都由流量明星担任。

陆雪琪是原著作者萧鼎笔下最美的女子，由学昆曲出身的李沁来饰演可以说是刚刚好。她的五官相较于别的女演员来说棱角更为分明，面部线条细长瘦削，非常符合“冷美人”所需要的那种感觉。

在武戏部分，她能做到身轻如燕，再带点儿倔强的味道；

在文戏方面，李沁版的陆雪琪在动情的瞬间，也显得内敛自持。

相较之下，孟美岐饰演的碧瑶，成了观众们争论的焦点。

粉丝们自然是首肯自家爱豆的演技，甚至上升到了演技炸裂的高度；

反对者则觉得孟美岐的碧瑶太出戏，根本不是原著中碧瑶的样子。

孟美岐想把碧瑶冷酷、狠劲、凌厉的特质表现出来，结果用力过猛，白眼翻得让人着实觉得凶了些。但我觉得，孟美岐之所以演的不像碧瑶，造型师的锅应该更大一些。

实话实话，孟美岐在《诛仙 Ⅰ》中的表现，是能让观众感知到她想要从女团成员向演员过渡的决心和诚意的。

错只错在，长相偏现代感的她本身就不适合演古装片和仙侠剧，是《诛仙 Ⅰ》在选角的时候过分考虑了流量的因素，选错了演员。

综合上面所说的，我觉得《诛仙 Ⅰ》口碑扑街并不全在流量明星之上。

（The End）
