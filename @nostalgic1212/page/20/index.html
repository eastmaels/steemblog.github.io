「Vicky's 短评」荧幕上被削发的女性们，是羞辱，不是荣耀

All Image Source

昨日，我被微博上一组非常魔幻主义的视频震碎了三观，不免想到了一些荧幕上被削发的女性群像。

《圣女贞德蒙难记》、《悲惨世界》、《少年的你》、《午夜守门人》、《V字仇杀队》、《西西里的美丽传说》、《驴得水》、《权利的游戏》、《女子监狱》、《水》、《巴黎烟云》…

有些人也许会说，女性短发也很美啊！

但这件事，根源并不在发型本身，而是这些女性被强行剥夺了建立在个人意志上的审美情趣。

我能理解出于便利的考虑，长发确实不容易打理。但只需要剪成利落的短发即可，犯得着要像公开处刑一般地剃成光头吗？

同为女性，我太清楚头发对我们是何其珍贵的存在。视频中那二十几个女孩子在哭泣，而他们绝不是因为内心的光荣而表现如此的！

所以我想问，是一线的男性工作者们，也需要剃光头吗？

相关部门这么做，还公开视频（且不打马赛克）的用意，到底在哪里？

除了作秀，我想不到任何别的理由。

心疼那些女孩，冒着生命危险赶赴一线之前，还要受到如此的羞辱。

「Vicky's 短评」《第三度嫌疑人》：让人背脊发凉的，人性中最矛盾的真相

All Image Source

《空气人偶》、《比海更深》、《步履不前》、《如父如子》、《海街日记》、《小偷家族》……

这些观众耳熟能详的电影，均出自今天我要推荐的电影《第三度嫌疑人》导演：是枝裕和

嫌疑人三隅高司（役所广司 饰）被控抢劫杀人，重盛朋章（福山雅治 饰）作为辩方律师开始对真相展开调查。随着案情的不断深入，受害者的女儿咲江、妻子、三隅的女儿等角色不断登场，看似板上钉钉的事渐渐变得不可捉摸起来…

一次次与三隅的会面中，重盛朋章的心态也有了很大的改变。他从一个一心只想要“赢”的辩护律师，转变为更在乎案件本身和嫌疑人的心路历程。

然而，纵使律师团队做了诸多努力，三隅依然被判处死刑。

电影的最后，重盛朋章走出看守所，天空还是如此湛蓝。

这个世界看上去似乎正在按照固定的规律有效运行着，但当他站在十字路口，心中想必充满了悲伤和迷茫。他觉得自己一直坚信的司法体系，似乎并没有那么公平正义。

本以为这会是东野圭吾笔下的《嫌疑人X的现身》这样温情的故事，但是《三度目の殺人》却是一部让我从头到尾都手脚冰凉的电影。

（The End）

「Vicky's 短评」世上的男人千千万，渣男的类型却如出一辙

All Image Source

歌剧《芝加哥》于1975年4月12日在巴西里约热内卢首演，截止到1977年8月27日共计演出了936场。电影于2002年上映获得了第75届奥斯卡金像奖最佳影片奖、第56届英国电影学院奖电影奖最佳影片(提名)以及第27届日本电影学院奖最佳外语片(提名) 。

这部电影最经典的一幕，是六个因杀死丈夫的女囚们所演绎的《Cell Block Tango》（牢房探戈），她们分别是「泡泡」、「六」、「咯吱」、「呃呃」、「西塞罗」、「李卜西斯」。

这个片段是女囚们的自述，随着感情及情绪的积累和叠加，慢慢演变成为了对男性伴侣的控诉。这掷地有声的呐喊并不是呼吁女性去手刃渣男，而是对女权主义崛起的呼唤，更是对现今社会女性所遭受到的各种不公平待遇绝望而又无奈的叹息。

去年，网路上有个年轻的团队在歌剧的基础上改编了一则短片，在微博和B站上非常火爆。由发起人分饰六角，分别用京、沪、川、粤、湘、鲁6种方言讲述了「胡同」、「萨克斯」、「龟儿子」、「滴滴」、「十四楼」和「妹儿睡着」6个主人公的故事。

从这六个故事中，我们不难看到PUA、家暴、性骚扰、性别歧视等诸多社会问题。

可惜的是，这个视频不久后就遭到下架，让人不禁感叹：

“一道光照进了坟墓，坟墓是肮脏的，光照亮了肮脏，于是光有了罪。”

「Vicky's 短评」《天气预爆》：现在的观众们，是不是对影视作品太过于宽容了？

All Image Source

今天看日历的时候，我才发觉自己已经连续上了14天的班。没有意外的话，这个状态在20日直接不会解除。

不知是否是疫情的关系，杭州的天空只有零星几日的阳光，剩下的不是风雨就是雾霾。

说到雾霾，我就想到了2018年，由肖央导演/主演的电影：《天气预爆》

《天气预爆》讲述了神仙被贬下凡间，马乐（肖央 饰）作为一介凡人无意间卷入一场神仙大战，最后也成为神仙并且拯救世界的故事。

电影的定义是都市奇幻喜剧，不同于过去我们看的那些存在于架空时代或者空间的神仙片，《天气预爆》的场景就是现在都市社会，还多多少少提及了雾霾等问题。

照理来说，这样的相对新颖的设定会带来不错的反响，但电影依然没逃过扑街的命运。

究其原因，我觉得有以下几点：

1、主配角没有鲜明的人物性格；

2、人物动机不明显；

3、情节突兀，在推进剧情方面作用不大；

4、毫无感情基础和CP感的男女主角

……

在我看来，中国电影最可怕的不是导演、编剧、演员无法交出令人满意的作品，而是观众对影视作品原来越“宽容”的态度。

当我和身边的人吐槽这部电影故事无逻辑、人物像从我家二三十年前家里年画跑出来的时候，听到最多的就是：

“就一部喜剧片而已，认真你就输了。”

好吧，一部喜剧片也能看的那么认真的我，大概是真的输了吧。

（The End）

「Vicky's 短评」《胖子行动队》：导演想告诉我们胖子也可以有梦想，但所做的确是背道而驰

All Image Source

每次我们提到“胖子”二字，总是会将其与“油腻”、“loser”、“猥琐”之类的印象联系起来。

今天要介绍的这部《胖子行动队》，则讲的是两个胖子拯救世界的故事。

《胖子行动队》由包贝尔导演携手文章领衔主演，郭京飞、克拉拉、辣目洋子联袂出演的动作喜剧电影。

从豆瓣评分来看，该片简直惨不忍睹，仅有4.6分。之所以想要谈谈这部电影，是因为我无意间在国外网站逛的时候，发现这上面的好评率居然高达80%。

是外国旁友们比较吃这个类型的电影嘛，迷思中…

影片中导演想要告诉大家“虽然我很胖，但我依然有梦想也可以实现梦想”这个励志故事，但他设置的喜剧笑点则是建立在对胖子的嘲讽之上。

除了对体型的全屏恶意之外，大boss郭京飞在电影中的角色说话有些结巴。电影以此来制造笑点，同样是对语言障碍群体的一种侮辱。

除此以外，电影还有诸多逻辑不清的致命缺点，并不推荐大家花时间观看。

（The End）

「Vicky's 短评」不负重望，高分韩影《寄生虫》成功拿下第92届奥斯卡“最佳原创剧本奖”

All Image Source

在最新公布的第92届奥斯卡获奖名单中，备受关注的高分韩影《寄生虫》成功拿下“最佳原创剧本奖”。

《寄生虫》由奉俊昊执导，宋康昊、李善均、赵茹珍、崔宇植、朴素丹等实力派演员主演。该片于2019年5月21日在戛纳国际电影节主竞赛单元首映，2019年5月30日在韩国上映，2019年10月11日在北美上映。

影片中，无业游民的一家四口，蜗居在廉价的半地下室出租房内，过着寒酸又没有尊严的生活。在长子基宇隐瞒真实学历，去一户住着豪宅的富有家庭担任家教后，一家人的生活渐渐起了变化 。

这天差地别的两家人，不断被卷入一个又一个的意外事件中。

此前，《寄生虫》就已将第72届戛纳国际电影节最佳影片金棕榈奖收入囊中，这也是韩国电影首次拿下金棕榈大奖。

除此以外，该片还获得了”第91届美国国家评论协会奖最佳外语片奖“及”第四届澳门国际影展亚洲人气电影大奖 “ 。

电影可看性很强，剧情不断翻转，强烈推荐！！！

（The End）
