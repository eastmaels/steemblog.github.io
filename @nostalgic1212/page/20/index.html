「Vicky's 短评」《诛仙》：比电影更精彩的，是口水大战

看了中秋档的《诛仙 I》，我突然感慨颇多。

关于电影本身的讨论，我在长图文《《诛仙》口碑扑街是谁之过？流量明星：这个锅我们不背！》已经谈的比较深入了，有兴趣的童鞋欢迎移步那里阅读。

微头条里，我想谈谈这部电影以外的一些东西。

翻看电影宣传期时候的新闻，我很惊讶的发现大家讨论更多的并不是电影本身，而是各路人马间的互喷。

原著粉丝和演员粉丝互喷、演员粉丝和diss流量明星的影评人互喷、两位演员因为争番位导致粉丝间互喷、还没看过电影就无脑打出五星和一星的用户之间互喷…

在某位影评人的文章里，我看到他用了以下评论：上映当天呈现离奇分柱形态的豆瓣评分（6.7分得电影，确实打五星和四星的最多）。

即便是当时没有关注这部电影的我，都曾在热搜上看到类似“《诛仙 I》首日票房过亿”、“XX演技炸裂”的热搜。

观众们仿佛分成了两类人，狂热的互喷者们和一脸懵逼的吃瓜群众。

实话实说，那位喷流量的某知名影评人显然是错的。

这里面的主要演员，无论是肖战、李沁，还是孟美岐、唐艺昕，他们演的都不算是一塌糊涂，不应该成为这位影评人口中的背锅侠。

话又说回来，他们的演技依然非常稚嫩，其实并不算太出彩。

即便是我喜欢的演员肖战，在这部电影里中我看不到他在《陈情令》中所表现出来的放松和灵气，被粉丝们捧到“演技炸裂”的高度显然也并没见得有多理智。

一部电影的好坏，与导演、编剧、服化道、配音、配乐等休戚相关，而演员只是这诸多零件中的一小部分。

希望大家能理性对待电影和演员，不要一股子的谩骂和包容，回归那个理性看待影视作品的年代吧…

（本篇完）

「Vicky's 影评」《诛仙》口碑扑街是谁之过？流量明星：这个锅我们不背！

All Image Source

《诛仙 Ⅰ》中，也是有一些演技不错的实力派演员。但如果我不说破，我想大多数观众是认不出来的。

被人吐槽是照着灭绝师太来演的，陆雪琪的师傅水月大师，其扮演者是原版《新白娘子传奇》中许仙的扮演者叶童。

在《绣春刀2：绣罗战场》中因为饰演了信王这一角色开始崭露头角的刘端端，在《诛仙 Ⅰ》中基本变成了nobody。

一部作品中，演员最终呈现出来的样子，除去他们本身的演技和努力，很多时候还和拍摄团队、剧本等外在因素息息相关。

《诛仙 Ⅰ》恰恰没有给演员足够发挥的空间，使得老戏骨们演的束手束脚，更何况是流量新人们。

说句大不敬的话，我觉得这部电影的缺点，很大程度来都来自导演程小东。

这位导演的名字大家也许不算熟悉，但他确实是位实力派，曾经将金像、金马最佳武术指导拿了个遍。

他是大名鼎鼎的《大话西游》系列、《倩女幽魂》、《新龙门客栈》、《笑傲江湖：东方不拍》的武术指导，这份简历完全可以证明程小东的实力。

然而，他用自己拍武侠的老套路去拍《诛仙 Ⅰ》，被指出玄幻感太弱也就不是偶然了。

再者，我们可以看到这部电影拍摄团队的其他成员有：动作特技熊欣欣，配乐泰迪罗宾，造型师奚仲文。

他们确实是业界的老前辈了，但并没有向观众交出令人满意的答卷。

尤其是造型团队，从主角到配角千篇一律的服装和无力吐槽的发型让人大失所望，强烈推荐去学习下隔壁《长安十二时辰》、《如懿传》、《延禧攻略》、《琅琊榜》、《陈情令》这些剧组。

整部电影均选择用了配音，但配音团队毫无诚意，致使声音和演员口型对不上，分分钟让人出戏。

当下的互联网，无论是专业抑或是业余的影视评论，大家有了一种默契。

似乎一部电影中，我们只看到了流量明星，看不到导演、编剧、服化道团队、配乐、配音等等。

影视剧口碑好的时候，就夸流量明星们超水平发挥，演技炸裂；

反之，口碑不好的时候，流量明星们就被拖出来各种鞭笞。

为什么大家选择一上来就骂流量？

因为骂流量本身就能带来流量，哪怕是素人，借此火一把多涨几个粉也是幸事一桩。

《诛仙 Ⅰ》确实不是一部好片，但这个锅，流量们不背。

（The End）

「Vicky's 影评」《诛仙 Ⅰ》口碑扑街，是否完全是流量明星的锅？

All Image Source

张小凡作为《诛仙》的主角，本身就集结了一大部分书迷的眼光。其饰演者肖战，是时下当之无愧的当红明星，其流量不可小觑。

《诛仙 Ⅰ》上映的时候，《陈情令》的热度还未完全褪去。陈情女孩们还没有出戏，纷纷回忆着剧中那个潇洒自如又兼济天下的魏无羡。

导演选用肖战的时候，《陈情令》还未播出，他还只是刚出头的男团成员之一。不得不说导演相当的慧眼识珠，押对了这个宝。

《诛仙 Ⅰ》中，张小凡这个“火​头军”第一次亮相的时候，我相信很多人都会和我一样的反应。

这人…是谁？

一瞬间，我还以为直接穿越到了隔壁《捉妖记》片场。

观众熟悉的神颜不见了，取而代之的是一副灰头土脸、面色蜡黄的样子，哪里有半分魏无羡的样子。服装上，也再不是宽衣窄袖的仙家风范，反倒是有些褴褛的“丐帮”模样。

肖战，变丑了么？

别人化妆是为了遮丑，这事儿到了肖战这里，恰恰是反着来的。

涂黑粉、化晒伤妆、点雀斑，所有这些都是为了让肖战与张小凡这个角色更加贴合。

被观众所吐槽的丐帮服装，也许正是为了将肖战欣长的身型隐藏在人群之中，满足那个道行不高、天资极差、胸无大志的人设。

从《陈情令》到《诛仙 Ⅰ》，肖战的演技当然算不得当时被捧上热搜的“演技炸裂”，但作为观众我们依然可以感觉到他对每个角色的认真。

正像他之前说过的那样：把自己打散揉碎，然后融入到每个角色之中，想象自己就是这个人。

除了肖战，《诛仙 Ⅰ》中还有其他主要角色也都由流量明星担任。

陆雪琪是原著作者萧鼎笔下最美的女子，由学昆曲出身的李沁来饰演可以说是刚刚好。她的五官相较于别的女演员来说棱角更为分明，面部线条细长瘦削，非常符合“冷美人”所需要的那种感觉。

在武戏部分，她能做到身轻如燕，再带点儿倔强的味道；

在文戏方面，李沁版的陆雪琪在动情的瞬间，也显得内敛自持。

相较之下，孟美岐饰演的碧瑶，成了观众们争论的焦点。

粉丝们自然是首肯自家爱豆的演技，甚至上升到了演技炸裂的高度；

反对者则觉得孟美岐的碧瑶太出戏，根本不是原著中碧瑶的样子。

孟美岐想把碧瑶冷酷、狠劲、凌厉的特质表现出来，结果用力过猛，白眼翻得让人着实觉得凶了些。但我觉得，孟美岐之所以演的不像碧瑶，造型师的锅应该更大一些。

实话实话，孟美岐在《诛仙 Ⅰ》中的表现，是能让观众感知到她想要从女团成员向演员过渡的决心和诚意的。

错只错在，长相偏现代感的她本身就不适合演古装片和仙侠剧，是《诛仙 Ⅰ》在选角的时候过分考虑了流量的因素，选错了演员。

综合上面所说的，我觉得《诛仙 Ⅰ》口碑扑街并不全在流量明星之上。

（The End）

「Vicky's 影评」《诛仙 Ⅰ》是否真的是部烂片？

All Image Source

在对原著小说进行整容式的删减以后，电影最终呈现在我们面前的是个非常套路的故事，里面的角色也都脸谱化地毫无新意。

张小凡（肖战 饰）幼年时全村遭到屠杀，被恰巧路过的青云门救下并带回去修炼。

上山十年，除了厨艺渐长以外，他的修行并没有什么精进。

仇，他是不想报的。

要说他心里有什么愿望，大抵就是和一直疼爱他的师姐田灵儿（唐艺昕 饰）谈恋爱。

就在这时候，为了应对魔教的卷土重来，青云门举办了一场比武大会，连常年吊车尾的张小凡也被要求参加。

而他也遇到了生命中另两个重要的女性，师姐陆雪琪（李沁 饰）和鬼王宗之女碧瑶（孟美岐 饰）。

虽说《诛仙》将会拍成三部曲，但我个人觉得，在第一部中张小凡这个角色还是过于弱化了。

身负血海深仇的他，不想报仇，不想比武，唯独想要和师姐相伴一生。

那么他为了追师姐做了什么呢？

唯唯诺诺地待在师姐身边，跑去问神龙要怎么表白…唯一一次具有爆发力的表现，还会得知师姐心有所属后跑到山顶大吼一声“我喜欢你啊”，结果还因为害怕被别人听到马上开溜了…

也许你会说，这是张小凡这个人心地善良，与世无争。

行吧，那么希望在接下来的第二、第三部（如果有的话）中，导演和编剧可以找到张小凡这个角色做每件事情时候发自内心的动力。而不是像第一部中，遇到什么问题了就让烧火棍自己启动跑来护主。

另外，刚才我们提到的神龙，在电影中只出现过非常短暂的两幕。

在我这个没有阅读过原著的人看来，这条神龙除了有点像张小凡的灵宠以外，大概也只是交代了《诛仙》中是有神兽这个背景了。

从第一部片尾神龙再次出现来看，接下来的续集中应该也会有它的戏份。

那么问题来了，第一部的故事都讲的仓促断层，观众们又怎么会有兴趣走进影院去看后续呢？

不可否认，《诛仙 Ⅰ》是一部成功的粉丝向电影，极尽可能地展现了几位流量明显的个人魅力，粉丝们也着实买单。

然而，电影并没有照顾到书粉及普通观众的感受。角色的素材就趋于扁平化，很多人物和情节都没有展开，使得观众很多时候感觉自己在看一段又一段的情景剧。

（The End）

「Vicky's 影评」《诛仙》：票房大卖，口碑却成两极化，你怎么看？

All Image Source

仙侠题材的小说一直深受编剧的喜爱，这些年也着实出了好些大IP：

《花千骨》、《青云志》、《三生三世十里桃花》、《香蜜沉沉烬如霜》、《陈情令》 …

其中大多融合了武侠、奇幻、探险、言情等诸多元素，无论是导演编剧还是服化道老师们，无不花了十二分心思凸显剧中人物的个人魅力。

赵丽颖、杨紫、迪丽热巴、邓伦、王一博、肖战…这些新生代年轻演员们，借由热门剧集迅速被观众们所熟知。

杨幂、周海媚、霍建华、李易峰、赵又廷等出道时间比较久的演员，也借着这股东风狠狠地翻红了一把。

这不，今年中秋档就有一部仙侠巨作被改编成了电影，并在上映当天拿下了一亿多票房。

《诛仙 Ⅰ》改编自同名小说系列，主要展现了几个主要人物跌宕起伏的修炼成长过程。其中，两败俱伤的正魔大战，错综复杂的情感纠葛吸引了一大批忠实读者。

这部被称为“后金庸时代武侠圣经”的著名IP，其原著总字数超过150万字。电影预计会分成三部，也就是说一部101分钟的电影要展现原著的1/3内容。

说真的，我对此报以质疑的态度。

而观影结束以后，《诛仙 Ⅰ》大体符合我对它的预期。倒不见得都是缺点，且听冰冰说道说道。

截止到我写这篇稿子的时候，《诛仙 Ⅰ》的豆瓣评分在4.9分，比之前的5.3分又低了一些。

在经过明星粉和书粉这两波极端势力的厮杀之后，最终得到这个分数，我个人觉得并不算太冤枉。

要将如此气势恢宏的小说塞进电影里，编剧需要有非常扎实的功底。很遗憾地说，这部电影的编剧并不具备这项技能。

（The End）

「Vicky's 影评」《少年的你》：正义可以迟到，但一定要来

All Image Source

「正义可以迟到，但一定要来」

据浙江大学《青少年攻击性行为的社会心理研究》调查则显示，49%的同学承认对其他同学有过不同程度的暴力行为，87%的人曾遭受到其他同学不同程度的暴力行为。

在我看来，校园霸凌和校园暴力并不一样。

念小学和初中的时候，一群不怎么念书的坏孩子会打架，甚至到学校来打群架，却很少看到这些孩子动手去惹一心只想好好念书的人。

但是校园霸凌是性质完全不同的存在，在我看来是一种纯粹的以欺负和自己不一样的人为乐的举动，这里面甚至不涉及到敲诈勒索财物。

影片中陈念的遭遇，正是校园霸凌中常见的几种手段。

言语暴力：从在受害者背后的窃窃私语到面对面的冷嘲热讽，随着互联网的迅速发展，言语暴力直接延伸为网络暴力。网络暴力有多可怕，相信看过《解除好友》的朋友们都知道；

人际关系暴力：独立受害人，约好了一样不和他们互动。在经过受害人身边的时候，施暴者会像躲避瘟疫一样避开，甚至受害人拿过的东西他们都会直接丢掉；

行为暴力：弄丢或者撕毁受害人的课本，向受害人丢掷物品，逐渐暴力升级到上下学路上的围追堵截。拳打脚踢抑或用到管制类刀具，对受害人身体造成实质性损害；

性暴力：污言秽语和当众撕毁受害人衣服，拍摄裸照或视频并上传至网络。触摸敏感部位，更严重的为强制发生性行为。

看到这里，你还能说校园霸凌只是一群不懂事的孩子在打闹吗？

不，这不是，这是赤裸裸的犯罪行为！

“大连13岁男孩杀害10岁女童”这起恶性事件，只是未成年人犯罪的冰山一角。其结果是这名男孩因为是未成年人，只得到了收容三年的惩罚，并未对其追究刑事责任。

让我们感到讽刺的是，《未成年人保护法》保护了未成年罪犯，却未能保护未成年受害者。

就像电影《伊甸湖》里那群熊孩子的父母那样，这位13岁男孩的父母并没有起到监护人应该有的样子。对孩子的不良行为非但没有及时矫正，甚至还对上门讨说法的女性辱骂。

我们不能把希望寄托于他人的三观和素质之上，而需要强而有力的法律支持。

所幸，就在10月26日上午，十三届全国人大常委会第十四次会议分组审议了《未成年人保护法（修订草案）》。

我们所关心的校园安全、学生欺凌、性侵害未成年人、未成年人沉迷网络等问题，都在这次提请审议的修改草案中。

希望这次大改能有实质性的作用，从源头上震慑住那些外表还是孩子，内心却阴暗无比的校园霸凌施暴者们。

毕竟，只有校园真正安全了，屏幕外的那些“陈念们”

才有勇气让孩子降临在这个世界上。
