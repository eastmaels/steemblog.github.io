「Vanilla‘s Diary」养猫如同养孩子，从小教育才是正道 191031

不知不觉，香草到我家也已经快要一个月时间了。小家伙就如同小baby一样见风就长，这孩子稚气褪去，曾经雌雄难辨的脸庞也越发显露出男孩子的特性来。

在接香草回来之前，我就听说有些规矩和习惯需要在小时候就开始“约法三章”，比如梳毛、剪指甲、洗澡、吃药…而不准咬人这点，自然也是列入在内的。

他从刚来的时候开始，似乎就对人类的手指特别感兴趣，时不时就要手脚并用的抱住开始啃咬。作为主人，必须要养成不用手逗弄小猫的习惯。如果要陪他们玩，使用玩具（逗猫棒、布老鼠等）会比较好。

那么问题来了，如果他们还是咬了你，应该怎么做？

我个人喜欢先礼后兵的思想，采取渐进式的“教育”方法。

最初的时候，如果他咬人，一般我会迅速停止游戏。把他晾在一边不同他说话，又或者指着自己被咬痛的地方板着脸告诉他，不可以这么做。

如果上述方法不启用的时候，一般我会抓住他后颈部分提起来，大声呵斥他。用简单的音节，类似“不要”、“NO”之类的词汇会比较好。猫咪对声音非常敏感，因此大声呵斥会让他们神经紧张。通常，出现飞机耳了就差不多了，主要不要太过了。

可惜，我们家香草在我用了前面两种步骤以后还是没有收敛，依然是逢梳毛、剪指甲的时候必咬人的状态。可能是最近玩疯了，后来逐步发展成陪玩玩儿的时候也会张嘴咬人。

眼见着事态正在朝着失控的方向发展，我终于下定了决心要纠正他的行为举止。

这么做以后，产生了几个结果：

1、香草变得很少仰天睡觉，以前他都是睡的四仰八叉的；

2、待在我身边的时间变少了，更愿意和我父母待在一起；

3、我的手一接近，香草就会举起手摆出防御的状态。

简单来说就是，打得太狠了！

父母自然是对我的行为百般指责，觉得我教育小孩子不应该下手那么狠。但是，看到最近几起未成年人暴力事件以后，我觉得小孩子的行为必须从幼年时期开始就被家长干预。在还能管教的年岁加以引导，后面还不至于失控到父母都无法矫正的地步。

小猫咬人是这样，小孩子的教育亦是如此。

「Vicky's 影评」《双子杀手》：从好莱坞华人之光，到接连票房失利，李安的神话是否还能延续？#3


All Image Source


「陈旧又稍嫌突兀的剧情」


有了父亲三部曲打底，让观众难免对这部带点“弑父”情节的《双子杀手》格外期待。


毕竟，小亨利的枪口不仅对准了老亨利，还指向了创造他的养父克雷。


正所谓期望越大，失望也就越大。


在知道自己是克隆人那刻起，小亨利内心几乎没有做什么挣扎，可以说是毫不犹豫地背叛了养父。


就我们所知，养父虽然把小亨利从小培养成杀手，但也是把他像亲生儿子一样对待的。这么火速地向DNA本体的老亨利倒戈，也是让人始料未及的。


此外，克隆人的电影我们已经看过太多。除了著名的《黑客帝国》系列以外，给我留下深刻印象的便是美日都曾拍过的《别让我走》。


该剧根据英国日裔小说作家石黑一雄的同名原作改编，讲述了一所寄宿学校中的孩子皆是为了器官捐献准备的克隆人。


剧集最后指向了以克隆人去弥补当下器官移植的空缺是否可行，以及随即而引发的社会伦理的讨论。


这一切，与《双子杀手》中用克隆人上战场代替普通人以减少牺牲，却又将面临伦理的诟病如出一辙。


就完成度而言，日版的《别让我走》显然更为出色。


不可否认，李安导演是深爱电影的人。


他本可以继续拍摄熟悉的题材，享受着观众的追捧。


但是，他却不是一个墨守成规的人。


《卧虎藏龙》、《绿巨人浩克》、《断背山》、《色戒》、《少年派的奇幻漂流》……


以65岁的年纪想要推动电影新技术的发展，他走上了一条又一条很少有人走上的道路。


李安导演的下一部电影《马尼拉之战》，将再次使用3D/4K/120帧技术。


它描绘上个时期六七十年代拳击界的景象，其重头戏就是乔·弗雷泽与阿里两个拳王之间那场被称为“马尼拉式震撼”的传奇之战。


这世上的所有事物都在变化，影视界日新月异风云万变，各种新技术源源不断出现。


但我知道，这个“老顽童”李安热爱电影的这份心不会变。


祝李安导演好运！


希望他的下一次尝试，可以在艰难中看到曙光！

「Vicky's 短评」《陈情令之生魂》：没有肖战和王一博的电影，粉丝会为此买单吗？ 191030

All Image Source

《陈情令》无疑是今夏最火爆的国产电视剧之一，优秀的原著、运营能力超强的主创团队、精致的服化道…种种努力，为这部剧的成功奠定了基础。

有人说，王一博和肖战是幸运的。借着《陈情令》这股东风，两人成功成为“新晋古装男神”。

我们都说，机会都是留给有准备的人的。他们除了先天外形条件以外，能够成功与他们的努力还有对演戏的执着是分不开的。

两人最近也是忙到不行，不仅要为各大一线品牌做代言，还有影视剧的拍摄任务。肖战和杨紫正在拍摄《余生请多指教》，而王一博和赵丽颖主演的《有匪》也将同样在2020年播出。

一部古装剧，一部现代剧，不知道两边的粉丝们会如何pick呢？

剧终后的两个月里，无论你去官微、豆瓣、贴吧，都能看到一大群大喊“走不出来”的粉丝们。于是乎，团队趁热打铁放出了电影版《陈情令之生魂》。

从演员配置上来看，主角并不是之前的“忘羡CP”，而是“小天使“温宁（于斌 饰）和蓝思追（郑繁星 饰）。

蓝思追本名温苑，是魏无羡带在身边的温氏一族的孩子。乱葬岗被围剿，魏无羡跳崖身死，蓝忘机回到乱葬岗发现了昏过去的温苑。他将孩子带回云深不知处抚养，并取名蓝思追。

在剧集版中，蓝思追与同为温氏一族近亲温宁相认，并相伴一同离开。不出意外的话，《陈情令之生魂》应该会就此展开，讲述二人行走江湖的中所发生的事。

从海报上来看，于斌与之前的妆容有较大改变，面部表情更为阴森可怖。郑繁星的装扮倒是没太多改变，颜值更胜之前。

电影版中，还加入了王一菲、高寒、何龙龙这三个新面孔，不知他们是否能成为新一波被喜爱的角色呢？

作为《陈情令》的粉丝，我对这次电影版的心情是复杂的。

一方面，喜爱的IP有了新的作品，自然是想看的；

另一方面，《陈情令》能够火爆和原著是分不开的。正是因为原著有大批的忠实读者，才在一开始奠定了观众基础。然而，电影版的剧情很可能并不是原著中有的，甚至不是延伸出去的，而是一个全新的剧本。

不管如何，《陈情令之生魂》上映的时候我还是会去贡献电影票的，你们呢？

（The End）

「Vicky's 短评」《我们不能是朋友》：这部霸道总裁剧，刷新了尺度的下限 191029

All Image Source

这几年的台剧一改霸道总裁爱上我的玛丽苏模式，推出了不少极具话题性的剧集。

《你的孩子不是你的孩子》专注于孩子的教育，将在父母禁锢之下孩子的众生相展露在观众面前；

《我们与恶的距离》将视角聚焦精神病人犯罪，向大众抛出一连串问题：这些边缘人士犯罪的源头是什么，社会福利制度又应该扮演怎么样的角色？施暴者家属的隐私是否应该被保护，受害者家属的安抚工作又当如何开展？

前段时间，我又看了一部话题性的8.2分台剧：《我们不能是朋友》

各大公众号安利这部剧的时候，无不用到了“大尺度”这样的字眼。

的确，无论是那花里胡哨极富暗示性的片头，还是几个主要角色三观不正的言论，怎么看这都不是一部正经的电视剧。

褚克恒：与未婚妻交往十余年，因为曾经对她母亲的承诺这些年一直在照顾容忍她。

周惟惟：一心和男友（未婚夫）按照计划来存钱、买房、生孩子、提前退休，剧集里三观最正的人；

黎皓一：周惟惟的男友，现实加强版凤凰男。有工作能力，却不懂如何真正维护好身边的爱人，让人不禁去想这家伙是否去参加过PUA课程；

高子媛：褚克恒的未婚妻，整个世界都围着对方来转。了解对方的喜好，什么都为他打点好，甚至为了未婚夫的事业和其上司的太太团们打得火热。除了买买买，没有事业没有喜好，甚至没有能说话的真心朋友。

当你放下这些成见去看以后会发现，《我们不能是朋友》其实揭示了诸多两性交往中极为常见的问题。

从周惟惟和黎皓一这段关系里，一方强加于另一方想法，要求对方跟着一起做计划，本身就是一种道德（情感绑架）。

两个人准备结婚，不仅仅是两个人的事情，而是两个家庭的大事。磨合是必要的，全盘接受对方的计划显然是不正常的。

从褚克恒和周惟惟这段关系里，直接暴露在观众面前的则是女性意识是否觉醒这点。

现代的女性，想要全职做阔太太的不能说没有，也绝不能说没有。但最起码的底线，就是不能绕着对方来转。学会独处、经营好自己的社交圈、有自己的爱好、有稳定的收入来源…

爱好自己，别人才会更爱你。

（The End）

All Image Source


「『李安对“父与子”主题的执着」


首先要提一句，《双子杀手》的编剧就是被《权力的游戏》影迷诟病的第八季的编剧之一戴维·贝尼奥夫。


剧本1997年就已成型，曾邀请《壮志凌云》的导演托尼·斯科特执导。但由于当时技术的限制，剧本中“一人同时分饰两角”及“返老还童”并未能展现出来，剧本就被暂时搁置了。


2008年，尼古拉斯凯奇又把制片人拉走了，我们就不说凯奇这几年被称作烂片之王这个梗了…


好不容易到了2012年，项目终于准备妥当，约翰尼·德普又因为某些原因拒了这个本子…


一直到近几年，李安导演接手后才有机会展现在观众面前。


李安深知一个好剧本对电影成品效果来说有多重要，纵使有为了制片做的妥协在内，但他为什么会选择这个剧本呢？


其中一个原因，也许正是《双子杀手》的主题。


就像我们熟知的那样，李安导演对「父与子」这个主题有着近乎执拗的偏执。他的“父亲三部曲”，就通过餐桌展现了各种中国式家庭的文化和冲突。


在《推手》中，跨国婚姻的朱晓生时常在父亲和妻子间左右为难；


在《喜宴》中，中国父权压制下传宗接代的思想与同性之爱产生了激烈的矛盾；


在《饮食男女》中，李安将“父子关系”延伸到了“父女关系”，讲述了一位老父亲在面对女儿纷纷离家时候的表现。


《双子杀手》看起来是“自己杀自己”的故事，但其实是一种更有新意的“父子冲突”。


无论是年龄还是经验，影片中的老亨利都更倾向于担任了父亲的角色。


另一方面，小亨利在面对创造自己的人克雷的时候，又将面临另一重父子关系。


从父子间的对立，到慢慢找到如何与其和谐相处的方法，某些程度来说也是寻找如何与自己相处方式的过程。

「Vicky's 影评」《双子杀手》：从好莱坞华人之光，到接连票房失利，李安的神话是否还能延续？#1


All Image Source
文/冰冰


“我超爱李安，告诉他，快点来找我拍片，不然我快变老了！”


2013年，史皇威尔·史密斯在台湾宣传《重返地球》的时候，曾经向李安导演隔空喊话过。


时隔六年，两人真的合作了一部新片，还在史皇在影片中见到了20多岁的自己。


那就是10年底唯一上映的好莱坞大片：《双子杀手》


美国国防情报局特工亨利（威尔·史密斯饰）在完成一个刺杀任务后，职业敏感性让他意识到这里面并不简单，于是萌生了退役的念头。


也许是唯恐他将国防情报局那些见不得人的秘密抖出去，刚到达休整地的亨利就发现组织派人在监视自己。随之而来的，是一个神秘黑衣人的疯狂追杀。


在激烈的交手中，亨利发现对方居然是年轻版的自己。组织在未经他同意的情况下，利用亨利的DNA暗自培养了一个他的克隆人。


就此，一场我与“我”的对局旋即展开。


「电影真的需要如此真实吗？」


在我看这部电影之前，就发现北美影评人对其评价不高。在烂番茄上，虽然观众喜爱指数有85%，但烂番茄指数仅有尴尬的29%，还不到及格线。


大多数影评人，无论评价褒贬，大多用到了一个词：


技术的尝试


纵观这几年李安导演的作品，他已经把最新的电影技术当作电影表达的重要形式。


这次，《双子杀手》沿用了卡梅隆（好莱坞另一位技术狂魔）称之为“新白金标准”的视觉技术：4K，120帧，3D。


翻译成大家更熟悉的词汇就是：更清晰、更流畅、更纤毫毕现。


那么《双子杀手》带来的第一个问题来了，电影真的需要如此真实吗？


电影自然是需要真实的，但只需要做到在特定情况下故事设定值得推敲即可，不是让在座的观众体验整个故事真实发生的过程。


《双子杀手》给人的感觉是所有演员都是贴在摄像机上面，他们每一个微表情、面部的毛孔哪怕是嘴唇的干皮都一览无遗，就像真实站在观众面前一样。


这种临场感带来视觉震撼的同时，也破坏了影片本身的电影感。


更有观众表示，其远不如《谍影重重》和《007》系列来得让人感到肾上腺素激增。


当然了，单从用影像去讲故事这点来看，李安导演对艺术的感知能力并没有衰退。追求视觉感受的观众来说，《双子杀手》确实提供了一出堪称浸入式体验酣的动作戏。


如果你的观影需求是这样的话，那么120帧版的《双子杀手》应该会是你的心头好。


话说回来，这部电影的问题究竟出在了哪里呢？


简单来说就是：剧本。
