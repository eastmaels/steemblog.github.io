All Image Source


本剧另一个亮点，就是强大的配角/客串卡司们。


在本剧刚开始的时候小三三和伊藤去换行头的那家理发店，店主是由小栗旬客串的。


快看看，他教育三桥不要做不良少年时候那苦口婆心的样子啊，简直就是长辈中的优等生。


大佬啊，你是不是忘记自己的过去了…


《热血高校》里的您老可是通过打架来制霸铃兰高中那个帅裂苍穹，霸气逼人的老大啊！


除了在第一集中玩票性质的出现一下的小栗旬和混混头目城田优，其他配角们也不可小觑。


饰演水谷先生老师的猪塚健太在《深夜食堂电影版》、《大奥》、《我无法恋爱的理由》中有不俗的表现，而去年推出的《老司机的爱情生活》更是深受腐女们的喜爱。


饰演椋木先生的室毅参演过多部影视剧，主要有《人生I字路》、《银魂》、《镰仓物语》、《重版再来！》等。同时，他还在《大恋爱：与将我忘记的你》中他饰演了男主角，与女主户田惠梨香展开了一段催人泪下的爱情故事。


饰演赤坂哲夫的佐藤二朗俨然走到哪里都是一幅佛祖的模样，推荐大家可以看看他参演的《银魂》和《全开女孩》。


饰演三桥父亲三桥一郎的吉田钢太郎应该都不会太陌生，我在弹幕中曾经看到有人说“部长怎么哪里都见得到你”的留言。没错，他就是耽美神剧《大叔之爱》里面的那个大叔，惊不惊喜意不意外？

「Vicky's 短评」《致命女人》：老娘为啥要手刃渣男？190904


上个周末，冰冰刷了一部刚播出的美剧《致命女人》（Why Women Kill），全程惊呼OMG（我的天）。

光是海报部分，就信息量巨大。

Beth身着粉色泡泡袖，手里端着的烤盘上赫然有一把手枪；Simone穿着高垫肩的衣服，一手举杯饮酒另一只手却拿着毒药；Taylor身穿亮黄色皮夹克，正对着锃亮的菜刀补妆。

不同的服装配色和款式，暗示了三位女主角不同的性格和年代背景。她们手中的道具，也许正是剧中所用到的“作案工具”。

《致命女人》的第一幕，就很容易让人想起若干年前的神剧《绝望的主妇》。当冰冰去看演职员表的时候，惊讶的发现编剧确实是同一个人！马克·切利，阔别多年以后再次奉上神级剧本。

本片讲述了三个不同时代的女人手刃丈夫的故事，神级的剪辑转场、富有时代特色的服化道、生动形象的表演，再配上小三、开放式关系、LGBT、不伦之恋等猎奇剧情，一下子就抓住了观众的眼球。

本片目前更新到第三集，豆瓣评分也从8.4飙升到了9.4分，目测会成为现象级热播剧。整部剧高能不断，尤其是女主角之一兼导演的刘玉玲，saly全场！

「Vicky's 短评」《平日下午三点的恋人》：知君用心如日月，恨不相见未嫁时 190903


根据日本知名“昼颜妻”主题日剧《昼颜：工作日下午3点的恋人们》改编的韩剧《平日下午三点的恋人》于上周末完美收官，豆瓣评分较原版有一定回落但仍保持在7.4分左右，可见观众对此次改编还是满意的。

《平日下午三点的恋人》揭开了婚姻里最后一块遮羞布，讲述了一个婚内出轨的故事。观众们在评论“三观尽毁”的同时又忍不住追着看下去，足以见得编剧在塑造剧中人物形象和拿捏观众心理方面有独到的本领。

韩版在改编的时候，总体上保持了日版的基调，在演员选择和服装造型上也丝毫没有败下阵来。故事和日版走向趋同，但在几个人物的结局上有了大幅的变动。总的来说，日版的更加现实克制，韩版的则很大程度上弥补了观众的遗憾。

安静地午后或是深夜，《昼颜》/《平日下午三点的恋人》会是你不错的观影选择:D

【Vicky's 影评】《我是大哥大》：主配角都是表情包？拜托，他们每个都是颜神好吗#1


All Image Source


今年夏天真是热力Max，不仅有吃不完的瓜还有超长待机中的暑气。


空调+冰镇西瓜成了续命法宝，煲剧自然也是不能落下的。


在追了《德鲁纳酒店》、《当恶魔呼唤你的名字时》、《凪的新生活》、《天国餐馆》、《轮到你了》、《长安十二时辰》等热播剧集以后，我发现这里边大多数不是小确丧就是恐怖片，着实需要找部沙雕剧来缓和下情绪。


这不，冰冰翻了去年在网络热议的爆笑神剧：《我是大哥大》


《我是大哥大》的演员明明都是颜值担当，但为了这部剧也算是豁出去了。


饰演三桥贵志的贺来贤人出生于日本东京，他参演过出演过《花牌情缘：结》、《海月姬》、《为了N》、《段田凛 劳动基准监督官》、《天堂之吻》、《Q10》、《极道鲜师》等电视剧。


然而，他在这部剧集里简直就是个行走中的表情包！


随时随地都可以摆出让人笑到面部抽搐的表情，表现力和表演欲极强。


女主角早川京子的扮演者桥本环奈，1999年出生于日本静冈。


她目前是偶像团体Rev. from DVL的成​​员，不足20岁就已经参演过《银魂2》、《FINAL CUT》、《暗杀教室》、《水球不良少年》等电影电视剧。


不管怎么看这都是个100%的萌妹子，冰冰是个女生看到她都会有初恋的感觉。
剧中的她是学校的大姐大，对待下属都是说一不二的那种。


京子在男友伊藤面前永远是一副小鸟依人、楚楚可怜的模样，但大多时候是…


冰冰只想说，明明可以靠脸吃饭，为啥要活成一个行走的表情包啊！

「Vicky's Life」具惠善婚变后称“他只是短暂的爱了我一下”，卑微的态度在婚姻中最不可取 190829


韩国艺人“具惠善安宰贤离婚风波”还在继续，算上之前“马伊琍文章离婚”及“双宋婚变”的新闻，让吃瓜群众们在茶余饭后多了许多谈资。不同于借由《失恋三十三天》进入大众视线的马伊琍和文章，以及因为拍摄《太阳的后裔》结缘的宋慧乔和宋仲基，具惠善和安宰贤的名字对大众来说似乎有些陌生。

具惠善是位多才多艺的韩国女艺人，同时拥有演员/导演/编剧等多重身份。2009年出演《韩版花样男子》的杉菜一角后一炮走红，成为亚洲韩流明星。

她亲自执导并出演的《具惠善的无业游民日记》于今年四月底播出后，受到很多粉丝的关注和喜爱。《具惠善的无业游民日记》以纪录片的形式展现了“艺人也是自由职业者”的一面，讲述了具惠善在事业空白期的无业游民生活。

这部仅有6集的纪录片风格轻松感性，让粉丝们见到荧幕以外具惠善的生活场景。该纪录片在豆瓣上的评分有9.2之高，观众们纷纷留下了“有才温暖的欧尼以后不管有没有再婚，都一定会幸福快乐”、“男人走了，还有她们陪着”之类的留言。

即使知道他们心中的“具大人”大概率不会登陆豆瓣，粉丝们仍忍不住替她加油打气。
