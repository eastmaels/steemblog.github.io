【Vicky's 影评】《德鲁纳酒店》：韩剧女主不再傻白甜，男主也不再精明腹黑霸道总裁-2


All Image Source


张弛有度的剧情


鬼神这类题材一直都是韩国的拿手项目，被中国观众所熟知的有《孤单又灿烂的神：鬼怪》、《主君的太阳》、《鬼客》、《王国》等。作为一部奇幻爱情喜剧，《德鲁纳酒店》被称作是养成系“女版”鬼怪，剧情有些像是《灵魂摆渡》和《第八号当铺》的综合体。


剧集的开篇非常出色，内容饱满、节奏紧凑地仅用了一集的时间就让观众对「张满月、具灿星以及德鲁纳酒店」前世今生的故事有了大致的了解。与此同时，编剧对剧情的设定及节奏的把控也十分得当，基本遵从了“一集讲一个故事”的模式。这样的编排除了能将故事讲清楚以外，编辑也可以有更多的发挥余地。


除此之外，剧中少有扁平化的人物形象。即便是看起来老实木讷的男主具灿星，也同时兼备了胆小、正义感、洁癖、固执、理想主义等多种特质。


从我个人来说，我特别喜欢女主角张满月的人物设定。她小财迷的形象加上豪车美酒、名牌包包来者不拒的嗜好，很容易让这个角色沦为“美则美矣，缺乏内涵”的花瓶形象。洪氏姐妹的编剧功底在于赋予一个角色双重性格设定，让这个角色同时充满了作为酒店女老板的霸气。


印象中有一幕，张满月受亡魂所托前往某议员的竞选会场。当她扛着枪大步流星熟视无睹地走进会场，举枪、上膛、扣动扳机，这一系列动作行云流水毫不迟疑，我相信很多观众都被这A到爆的一幕激的心里小鹿乱撞。




精致的服化造型


德鲁纳酒店的老板张满月是个时冷时热、性格阴晴不定的美人，男主常常会被她整的和尚摸不着头脑。剧集中的她不仅仅是“嗅觉灵敏”的小财迷，同时也是个极端的享乐主义者。虽然赚钱不少，却常年背着巨额贷款，眼见着酒店都要开不下去了逼的男主要强行限制满月的消费欲。


对于观众来说，简直就是在看一部不亚于《校对女孩河野悦子》、《First Class》、《了不起的麦瑟尔夫人》这样的时尚大片。行走中的衣架子IU光是开头六集就已经换了四十多套衣服，美轮美奂的配饰和咸淡相宜的妆容更是为整个人物增色不少。


她在剧中所穿搭的衣服、首饰、包包、鞋子等，目前也是很多女性观众争相购买的爆款。

「Vicky's Life」初遇施华洛世奇（中） 190810

打开的一瞬间还是让我惊艳了一下，包装很不错饰品本身也比较精致，有点超出我之前对它的预计了。星星和月亮的组合，配上人工水晶点缀，还真有点熠熠生辉的味道在里面了。这款项链低调，平时上班带着的话也不会被领导请去喝茶:D

想要赶紧看到它的全貌，于是我迫不及待地把项链从盒子里面拿出来。

Okay，这就是项链的全貌了。

总体来说还是非常漂亮的，很适合大学生或者刚入社会的年轻白领佩戴，青春又不失稳重神秘的气息。但我之所以说适合这个年龄阶层，是因为项链的材质。项链的正面总体还是比较好的，背面的锁扣处金属的部分看起来不是特别高级，有点儿街边小饰品店售卖的首饰那种感觉。

项链的锁扣部分可以调节，所以做一般长度的项链也可以当作锁骨链来佩戴。如果搭配得当的话，我觉得还可以和别的项链搭配一起佩戴（我的是反面教材）。

不过因为材质的问题，这条项链我最后还是给退了回去。不喜欢的东西不要多囤着，这是浪费钱的行为，这也是我最近才想明白的一件事。断·舍·离，这也是我们生活中都应该要时刻牢记的三个字

「Vicky's Life」极其恶劣的私自搬运行为 190809


前段时间极其忙碌，一边要完成给自己的“一周两篇大作文”的任务，一边组了一个16人的社群安排他们参加写作平台的培训班。对于我来说，自己亲身测试过可行的赚钱之路，是从来不吝啬于分享给身边的好朋友们的。

紧赶慢赶地终于在截止时期前把她们都送进了课程班，我得空了可以坐下来整理自己要发布的文章。在编辑好公众号以后准备发布的时候，突然提示说检测到类似的文章，因此我的文章不可以标注原创。这个情况曾经出现在《如懿传》：初看失望，看完却怅然若失，细水长流才是真正的好剧这篇文章发布的时候，只是当时我也只是提交了头条后台的作者页截图并没有多做追究。

今天发现的这篇，倒是让我认真研究了一番，问题出在一个名为“宵夜闲话”的公众号那里。我仔细核对了一下，那个公众号上的文章发布于2天前（8.7），而我的文章在头条（7.24）和简书（8.7）发布。从时间戳的角度来说，我的发布时间也是远远早于这个公众号主的。

对于这种极其恶劣的私自搬运行为，我已经给他们后台作出留言，要求删除这篇文章的内容。此外，针对这篇文章已在微信公众号后台作出申诉和投诉。不管对方是否申明原创，未经过原作者授权就私自搬运文章在公众号上发布的行为，足以激起任何文字工作者的愤慨！

另外，根据他私自搬运的文章内容，我猜测是从简书这个渠道找来的。我想我应该要调整一下各个平台上发文的顺序了，大家有什么好的意见吗？

All Image Source


该剧于7月31日播出，首播日就拿下了9.4的超高分。目前豆瓣评分虽有回落，但基本仍维持在8.4分左右。首播集的收视率高出同期剧集许多，是tvN本年度收视率的榜首，位列历史第四名。


本剧由韩国知名编剧洪氏姐妹操刀。你没听说过她们的名字？没关系，她们写的《主君的太阳》、《花游记》、《我的女友是九尾狐》、《九回时间旅行》、《我的女孩》、《豪杰春香》以及《原来是美男啊》这些剧集，喜欢刷剧的你一定看过其中一二。


不卖关子啦，今天我们要说的这部爆红韩剧的名字就是：《德鲁纳酒店》（又名：月之酒店/客栈）




All Image Source


剧情梗概


对于一般人来说，停止呼吸（脑死亡）的那一刻就代表着一生的终结。他们将经过忘川喝下孟婆汤，忘记前世今生的事继而等待下一世的轮回。有些人在世上还有未了解的心事，久而久之就成了游荡于世间的孤魂野鬼，而德鲁纳酒店就是这些亡魂死后暂时居住的地方。


酒店的老板张满月（IU·李知恩 饰）是个手上沾满鲜血的杀手，因为这一切的杀戮她被诅咒。虽然长生不老不死，却也永远地被束缚在月之客栈之中。酒店中的工作人员和客人均是往生者，但因它有真实的建筑和地址，故而需要活人负责打理营业执照、处理税务等凡间琐事。


机缘巧合之下，满月与男主的父亲定下约定，20年后她会带走当时还是幼童的具灿星（吕珍九 饰）。剧集的主线，正是从20年后开始的。成为酒店大堂经理的具灿星，开始用他的温暖和善良悄然改变着德鲁纳酒店和它的主人。那棵代表着张满月的千年大树，也因为具灿星的到来而重获新生。


正如同鬼怪新娘带给鬼怪阳光一样，前世起就颇有些渊源的两人，他们的故事也将在月之客栈里缓缓拉开序幕。
