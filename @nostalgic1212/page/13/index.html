【Vicky's 旅行】上海国际潜水展 / 2019 DRT SHOW Shanghai


去年四月份我在马来西亚·仙本那参加OW课程，由此也写了我人生中第一个长篇游记系列。之后的一年里，我偶尔会去室内潜水池“泡水”，倒是再也没有找到机会真的回到海里去。今年年初，师兄和我说上海国际潜水展快要开始了。

于是乎呢，我赶紧屁颠屁颠跑去上海和一票师兄弟妹们面基啦。

DRT是Diving(潜水)、Resort(度假村)和Travel(旅游)的缩写，其内容涉及了水下摄影、潜水旅游、自由潜水、技术潜水、医疗潜水、潜水教学等诸多方面。除了商业合作以外，DRT也是真正关注潜水事业发展以及海洋生物保育的组织。

因为其巨大的影响力，每年的展出都吸引了大量参展商和潜水爱好者的光临。

有不少人都是教练miumiu的学生或者认识的人，所以大家面基以后就像是老朋友一样聊开了。我们在现场还连线了当时正在宿务的miumiu，当即决定过年的时候就去巴厘岛找她玩。如果有机会的话，可能还会考虑参加AOW进阶课程。

展出人很多，我一路处于拿小册子/小礼品和买买买的节奏中…标准的中国式游客:D

看到娃娃就走不动的我大概买了十几只小企鹅（准备送人），算上之前买的抱枕和一些七七八八的东西，还好师兄开了车来！！话说，要不是妈妈耳提面命，我差点儿还买了潜水的脚蹼…冷静下来了一想，作为不怎么去的一项活动却是不需要坐那么大的投入。

大家结束以后还找了餐厅继续meet up，回头等写美食篇的时候我再慢慢分享给大家噢:D

【Vicky's 碎碎念】写作课程知识整理：如何获得机器更多推荐？（5）190718


All Image Source

B/ 合理的文章结构，巧用疑问和设问句。在文章的一开始设立疑问并于文章的四分之三处逐步揭晓谜底，一步步地引导读者去读完整篇文章/

C/ 运用简单易懂又不粗俗的语言，让广泛的读者愿意打开你的文章去阅读。

D/ 巧妙的运用配图，使得文章的篇幅显得不那么短内容更为丰富。不过需要注意的是配图必须符合或者靠近文章的主题，再者就是要注意配图的版权问题。


All Image Source

E/ 文章的篇幅不宜过长，推荐控制在5000字以内。文章并不是越长就越好的，如果文章在不是废话连篇的情况下确实超过了上限字数，反倒是拆分开发布会比较好。当然了，我的编辑也曾经提过说不推荐大家拆分上中下来发布文章，这会让读者有一种阅读体验不连贯的感觉。对于平台本身而言，也不方便他们做集中推荐。

F/ 减少与文章主题不相关的内容。有些作者在写文章的时候套用固定的模板，某些程度来说这样也无可厚非。在非传统平台写作时，作者为了凑够字数而将模板设置成段落多、篇幅长还配上了图片，这点在传统平台写作时是切记不可使用的。

All Image Source


任何一场分手都没有赢家


《失恋三十三天》是我看过次数最多的一部电影之一，另外两部是《他其实没那么喜欢你 He just not that into you》和《单身指南》。也许是因为感同身受，每次心情不好的时候去看一眼总能让自己感觉被开解抚慰了许多。


前年的时候，我谈过一场轰轰烈烈的恋爱也在分手时弄的两败俱伤。曾经如此了解彼此的两个人在争斗的时候却刀刀直戳要害，痛得我昏天暗地。


但你知道吗，天秤座真的太要面子了。白天我还是那个牙尖嘴利嘴上半点不饶人的斗士，到了晚上关起门来继续生不如死；面对对方的挑事儿，分分钟骂人不带脏字儿地怼得对方恨不得没来到过这个世上，却在挂断电话的那一刻冲到厕所号啕大哭。


除了王小贱在婚礼上替黄小仙出头怒怼陆然那场戏以外，我最喜欢的是小仙儿追着陆然狂奔的那一场。


在黄小仙突然意识到她在这段爱情中做错了什么，奋起追着陆然的计程车狂奔的时候却被王小贱摔了一巴掌拦下。这时候旁白中传来黄小仙的独白：「世上最肮脏的，莫过于自尊心。此刻我突然意识到，即便肮脏，余下的一生我也需要这自尊心的如影相随。」


是啊，在一段感情里只要两人实实在在地爱上了对方，就弥足珍贵。即便是到了结尾时，两人实实在在地恨上了对方，也不用去计较谁输谁赢。势均力敌的爱情就是在爱的时候愿意为对方去披荆斩棘，在分开老死不相往来永不相欠。


而如何做一个优秀的前任，则是我们下次要讨论的话题了。

  • 如何改进，提高推荐量和阅读量？

1> 点击率

文章需要有快速吸引读者阅读的标题，但又不可以是简单的标题党。部分比较成熟的平台，针对违规情况比较严重的标题党有专门的惩罚措施；

2> 文章读完率

作者写的文章可以被读者一字不落的读下来，这无疑是对作者辛苦写文一种莫大的肯定，所以我也很理解很多平台会将这个数据列入重要考量条件之一。此外，这块数据也会影响到文章的推荐率和文章所附的广告收益。


All Image Source

A/ 文章要做到“文题一致”，作者需要给文章配一个既符合现实又具有吸引力的标题。

早期我在自媒体平台发文的时候，还延续自己在区块链平台发表文章的习惯，用上类似于【Vicky’s 影评】和【Vicky’s 影视剧推荐】这样的字眼。但这除了占用标题有限的字符以外，对我文章的阅读量并没有任何帮助，关于这一块为仍在继续学习中。

「Life & Activities」一些老照片 / Some Old Photographs 20190717


上周六外婆家搬家，六点不到我就赶过去帮忙。小辈儿和搬家工人在忙活的时候，外婆外公坐在那里不言不语。现在想来，这是要搬离他们住了四十余年的家，换做是别人自然也是舍不得的。

听妈妈说，当年这套房子在分给外公以后，中途差点儿被走了后门的人给换走。其中费了不少事儿，这房子倒也最后回到了外公外婆手里。当年两位老人逃避饥荒从安徽老家漂泊到了杭州，没有什么家底的他们自然也没有财物置办家具。外公就去铁道边捡拾那些废弃的木条，一样样地自己打造成家具再搬上五楼。

在整理老物件的时候，我看到了压在茶几玻璃下的旧照片。细细看来，大抵是我的、妈妈和小姨的、舅舅的以及两位表妹的，却是我的照片最多。是啦，我是妈妈那边小辈儿中最大的，从小也是深得外婆外公的喜欢和照顾。

All Image Source

「负面影响」：举报、无效点击、过实效

1> 举报

除去作者本身抄袭因素以外，我个人觉得在用户基数特别大的平台运营，遇到不喜欢自己文章的读者真的太司空见惯了。我在这几个月的运营中，也偶尔会遇到语气不善的读者留言。通常建议大家抱着来者是客的心态去应付这类事情，如果实在控制不好脾气，那么不回复也比带着情绪回复来的强；

2> 无效点击

去年在Primas平台写作的时候，针对每篇文章是会监控停留时间的。打开文章迅速“点、评、转”这招是行不通的，打开文章静置不滑动自然也是不行的。在很多平台上，用户的真实阅读时间是系统判定这篇文章是否优秀的重要参数之一；


All Image Source

3> 热点时效

无论在Steemit上还是Primas上写作的时候，我很少顾及当下舆论的热点是什么。这两个平台上有一类文章最为受欢迎，就是平台本身关系收益的资讯类文章。收益是否高不可论定，但打开率应该较其他类型文章都要来的高出不少。

那么在传统平台中，又是怎么样一幅景象呢？大家要先清楚，国内传统平台的读者大多都是普通人。他们最喜欢看的大多是娱乐、情感、职场、育儿等类型的文章，并且常常紧跟当下社会舆论的脚步来走。比如最近某企业老板性侵幼童，那么无论是情感类、心理类作者还是育儿、影视类作者，想要抓住热点必须做到“快、狠、准”。

在热点发生的一周内文章大体都能比较受到关注，时间久了以后系统会检索到大量重复议题的文章，自然推荐量也就下来了。从读者的角度来说也不难理解这一点，同样的论题翻来倒去说的都是那么些道理和例子，谁又愿意再一篇篇打开认真去阅读呢？
