工作中的专业度

让斐塔美登上开篇说到的听证会席的，其实由韩国总统选举直播期间一场闹剧引起的。


李旭贤和徐明浩两位议员在竞选辩论的时候，因为互相揭老底而引发网络热议。


众所周知，韩国的政治和国内几大财阀有着千丝万缕的关系。独角兽的背后股东KU，其理事长正是独角兽代表宋佳景的婆婆。


基于利益需要，独角兽不出意料地删除了可能对李旭贤造成不良影响的词汇，这才有了斐塔美和李旭贤那方议员周胜泰的论战


对于李旭贤和徐明浩谁会如愿当上总统，斐塔美预测说是徐明浩。


事实证明，当选的正是徐明浩。


原因无它，因为「人们在搜索词面前是最坦诚的。」


人们会搜索自己真正支持的候选人，即使他们也许在人前不好意思承认支持这个人。


在被独角兽送出去坐上听证会席的时候，斐塔美也没有因为心中的愤愤而做出令公司蒙羞的行为。相反的，她依然在全力维护公司的形象。


在面对周胜泰议员咄咄逼人地质问时，斐塔美非但没有怯场，还依靠自己充分的准备扭转了局面。


原来，她偶然间发现周议员利用小号通过互联网招嫖，对方还是未成年人。


在职业道德范畴内，她收集证据坐实了议员的犯罪证据，将其一举送入监狱。


一时间，热搜词从「独角兽 受贿」转变成了「周胜泰 招嫖」，成功化解独角兽的公关危机。


即便是最后，她被当作独角兽替罪羔羊予以辞退的时候，斐塔美依然保持着自己作为职业媒体人的体面。


她没有大吼大叫，也没有谩骂同事上司，甚至没有在办公室砸东西。


她是怎么做的？


在收拾完办公桌后，走到昔日团队成员面前，告诉他们她准备去竞争对手巴罗。


她希望曾经的下属们记住她曾经教给她们的技能，希望有天能用这些知识打败她，因为最了解她的人是他们。


说完，潇洒的转身离开。


这才是专业人士才有的气度和品行，不是吗？

对于普通用户来说，网络的意义正如议员所说：


「网络应该是让小学生和专家平等讨论的地方，无关年龄、阶级、性别、人种、学历，不该有任何偏见。能够自由共享意见的空间就是网络，最中立、最多样，这才是网络。」


但事实上，热搜并不是很多人一起在搜索的结果

简单来说，不符合社会公约的词汇不能出现在热搜榜内。

对显而易见的违禁词，删除


当红演员出道前陪酒的视频呢？


里面的主角穿着整齐，似乎也不构成色情的因素，删除就等于将这波流量拱手让给了竞争对手。
不删？


蜂拥而至的网民将这条搜索词直接捧到了NO.1的位置，在毁了当事人人生和事业的同时还直接导致了他自杀未遂。
在删与不删做抉择，团队只有90s的时间。


这绝不是单个事件，可以说，办公室的日常就是场与时间的博弈和赛跑。

「Vicky's 短评」《花千骨》：喜欢仙侠剧，更喜欢里面的师徒恋 190927

All Image Source

一直以来都很喜欢看仙侠剧，里面飘逸的服装、精致的配饰、仙气十足的造型让我深深着迷。最近更是神使鬼差地打开了已经五刷，由霍建华和赵丽颖主演的仙侠剧《花千骨》，欲罢不能地再次看的津津有味。

这次看下来，突然意识到《花千骨》让我着迷的也许并不是霍建华这个演员本身，甚至不是这个故事。我喜欢的，只是白子画这个人设以及师徒恋这种恋爱模式。

白子画是长留上仙，一袭白袍穿的超凡脱俗。淡漠的表情、孤傲又严格的性格、心怀天下苍生…不愿杀死花千骨还反而收她为徒，想要以己之力扭转命运。有时候真的会疑惑，他到底是太兼济天下还是太过于自负了。

两人之前的感情，在我看来与其说是恋人更像是父女、兄妹。少了很多情欲的部分，更多的是守望相助。

讲真，当年看《花千骨》的时候着实让我哭的死去活来…

除了这部大IP以外，师徒恋题材的影视剧还有几部观众耳熟能详的。《神雕侠侣》、《择天记》和《三生三世十里桃花》，都是非常不错的剧集。

近期我还看了一部沧月写的小说《朱颜》，这是《镜》系列的前传。听说镜系列即将被拍成电视剧，男主角很可能是因为《陈情令》火到国外的肖战。如果消息属实的话，实在是太让人期待了！

最后，冰冰想安利大家去看看《朱颜》这部小说。

里面的男主角时影是九嶷山的大神官，阴错阳差之间和命中注定会杀死自己的赤之一族郡主朱颜在年少时有了一段师徒之情。两人彼此错过、否定、伤害，这和《花千骨》的故事架构非常类似。

如果你喜欢《花千骨》，那么不要错过《朱颜》这部小说喔:D

「总结」


开放式关系（Open Relationship）于十九世纪八十年代在美国被首次提出的，最先也只是流行于性思想比较开放的西方国家。随着女性性解放的不断推进及婚姻观念的不断改变，即便是比较保守的东方国家（中国、日本等）也逐渐出现这种生活方式。


在历史的长河中，的确有成功的开放式关系。2017年上映的电影《马斯顿教授和神奇女侠》，就是改编自真实成功的开放式关系。


著名漫画人物「神奇女侠」的创造者、知名心理学家威廉·马斯顿博士，是位拥有两位长期伴侣的BDSM（绑缚与调教、支配与臣服、施虐与受虐）爱好者。一位是同为心理学家的妻子，另一位是配合他和妻子实验的学生。这段关系中，既有同性恋和双性恋，也有异性恋。三人关系历经诸多磨砺后达到神奇的平衡，在他离世后两位女性还在一块抚养孩子。


要处理好一段的开放式关系，你必须是以后对自己有充分认知、心智成熟、充满理智、情绪稳定的人。除此以外，你还需要有较高的沟通技巧，有能敏锐感知到对方需求的天赋。即使像著名哲学家萨特和女性主义代表人物的波伏娃那段著名的开放式关系里，也依然充满了吃醋、争吵、猜疑等各种人类常有的情绪。


总的来说，开放式关系仍然是一种非常小众的生活方式，我对它的了解并不足以对其下任何定论。我身边有一些从一对一走向开放式关系的人，也有从开放式关系回到一对一关系的朋友。


对于这种生活方式，我抱着既不排斥诋毁也不盲目推崇的态度。

「Vicky's 短评」《网络谜踪》：老爹人肉搜索起来，比你前男友厉害多了190926

录屏电影是这几年还是流行起来的影片类型，其拍摄周期短却需要大量后期制作。无论是在电脑还是影院观看，观众都能以第一视角迅速被带入剧情（影院效果更加）。

我最先接触到这个类型的电影，是《解除好友》系列。

第一部告诉大家不要对身边的人太苛责，尤其不要网络暴力；第二部则是像我们揭示了暗网的可怕，这个平常人不会接触到的领域充满了血腥、暴力和大量的灰色交易。

去年又出了一部该类型的电影《网络谜踪》，口碑炸裂。

工程师大卫·金（约翰·赵 饰）原本有个幸福美满的家庭，但几年前癌症夺走了妻子的生命，也让很乖巧的女儿玛格渐渐和自己疏离开来。

某天，大卫发现女儿彻夜未归。在询问了身边的亲朋好友以后，他意识到玛格也许被卷入了某种事件之中，但前来协助调查的警察却推断女儿只是离家出走。

不满这一结论的大卫为了寻找女儿，独自展开调查。他打开了女儿的笔记本电脑，用搜索门户网站和社交软件顺藤摸瓜，终于找到了有关玛格的关键线索…

在观影的过程中，由于是第一视角，观众可以切身感受到大卫的紧张感。电影的故事情节通顺、悬念的设置恰到好处、细节的把控也十分到位，不可不谓是悬疑片中的佳作。

看完电影以后，我不由地感叹：

还好大卫用的是Google搜索，要是用了某度…能不能找到女儿还两说，怕不会被某系先坑一把吧…

现代人每天从被手机闹铃唤醒开始，几乎无时不刻和手机绑定在一起。


即使御宅族，打开手机就可以和朋友们天南地北胡侃，丝毫不觉得与社会脱节。


天气预报、交通情况、国内外热点事件…这些都是由我们使用的搜索门户网站算法决定。


在中国，对应的则是百度、新浪、腾讯、网易、头条…


《请输入关键词：WWW》中，也有两家公司分庭抗礼


它们分别是：Unicon（独角兽）和 Barro（巴罗）


就目前来看，Unicon的市场占有率略占上风，而Barro处于千年老二的尴尬位置。


开场听证会上，周胜泰议员质问搜索门户网站负责人：


“你们是否承认有过捏造热搜的行为？”


搜索门户网站负责：


“我们独角兽，对于搜索词，确实有过捏造行为。”


面不改色


出乎任何人意料，业内秘密就这么被揭开了。


里面提到的独角兽，一天累计搜索量最高达到四亿三千。


每一秒平均有五千个搜索词被搜索，每人每天检索门户网站的次数为63.8次。


「人们在搜索词面前是最坦诚的。」


用户在输入搜索词的时候，门户网站会将这些信息汇总传输到服务器。


通过AI算法处理之后，就是呈现在用户面前的搜索关键词，也就是我们所熟悉的热搜。


相比于国内热搜榜，韩国的热搜更加直观刺激


它是实时滚动的
