【Vicky's 影评】《平日下午三点的恋人》：婚后的女人如同“电冰箱”，这种丧偶式婚姻是否是出轨的原罪？

All Image Source

去年我看过一部叫《砂之塔》的日剧，讲的就是一幢豪宅内一群主妇们勾心斗角的故事。

越是成功的男人，身边越是有一群小帮派的太太们。如果出身家境不如他们，将会被无视甚至会被排挤，也有可能会影响孩子的交友以及丈夫的升迁。

当女主角高野亚纪（菅野美穗 饰）向丈夫哭诉这件事的时候，对方非常不耐烦地说：

“我在外面辛苦工作，你就是在家带带孩子，哪那么多事？能不能少给我惹事儿？！”

丧偶式婚姻，大抵就是这个样子的。

事业有成的男人需要一个好的贤内助支持，就像前阵子大家乐谈的徐峥·陶虹这对模范夫妻。但是，并不是每个主妇都能从家庭琐事中获取快乐和成就感。在日版《昼颜》中，利佳子（崔秀雅）对纱和（孙智恩）有这么一段经典对话：

“结婚就是用失去热情来换取安稳，过了三年丈夫只会把妻子当成冰箱一样对待。打开就有吃的，坏了也不去维修。”

“只有在外面有了男人，才会因为自知理亏而对丈夫和孩子更好，否则谁会甘心为男人去洗内裤呢？”

崔秀雅利用下午三点到五点孩子放学前的“自由”时光，放纵自己去寻找和享受“被爱”的感觉。因着有这样些许的平衡，她才能在丈夫把自己当作按步就按的“电冰箱”的时候，心平气和地点头说是吧。

当她当初在选择了地位和金钱的时候，就做好了生活会是如此的觉悟，就这点来说她不并不值得同情。

与此同时，她又是可悲的。丈夫在给了她经济保障的生活，却没有去尊重她。

然而，像孙智恩那样完全不考虑权钱和地位，一心嫁给爱情的女性们，婚后生活就一定是幸福的吗？

「Vicky's 资讯」人气过于火爆，肖战南京线下活动临时取消

All Image Source

双十二一大早，南京新街口德基广场就有很多人聚集在此。原来，是今夏因为一部《陈情令》开始广受网友关注和喜爱的知名演员肖战，将在该处参加某品牌活动。

闻讯而来的粉丝们，顶着寒风早早地就守在此处。有些甚至是从外地赶过去的，带着灯牌的粉丝们想要为自家爱豆应援。

为了保障现场秩序，商场方面出动了百余名保安维护现场秩序。德基广场更是在官微发布消息，希望来参加活动的粉丝们注意安全。

只是现场聚集的人们越来越多，上午十点半左右，商场遗憾地表示本次见面活动只能取消了。

对于此次活动临时取消，粉丝们在不舍之余，也都表示了理解。毕竟年末时期，如果因为见面会造成人员伤亡，对于艺人本身来说也不是一件好事情。

《陈情令》国风演唱会结束后，肖战等主要演员们纷纷正式和这部剧说了再见，场面可以说是相当感人。

这此之后，肖战明年将要播出的新剧《余生请多指教》顺利杀青，他紧接着投入了《我们的歌》录制以及各类商业活动中去。

从Olay、博朗、汰渍、雅诗兰黛等品牌宣传照来看，肖战的帅气比以往更盛，人气亦是频频攀升。

如今他在影视、唱歌、综艺等方面均有涉及，粉丝们也都对他抱有很高的期待，希望未来可以看到他更多更好的作品！

「Vicky's 短评」《庆余年》：年底最期待的国剧，范闲为了追“鸡腿夫妇”，太不容易了！

All Image Source

最近得空了就在刷《庆余年》，看到第10集发现了一个萌点。

范闲见自己干下的荒唐事，非但没有让皇帝、后宫退婚，自家父亲也是对此喜闻乐见。如此情况之下，他带着若若、范思辙还有护卫藤梓荆，一行四人杀到靖王府。

他的想法很简单，既然不能退婚，就先娶了“鸡腿姑娘”。到时候生米煮成熟饭，那尊贵的长公主之女自然是再不能嫁给他了。

场面自然是唐突的不能再唐突了，隔着屏幕我都能感到尴尬…

不出所料，范闲没有找到心里的白月光。靖王告诉他，诗会当天来客众多，那姑娘也许是别家的丫头。

范闲一拍大腿，正准备一家一家登门拜访，被身边的人堪勘拦住。

不得不说，《庆余年》真的是最近国剧中我最关注的一部剧。虽然第一集有点让人摸不着头脑，第二集妥妥就上头啦！

还有什么初恋，比得上“我喜欢的姑娘是我的未婚妻，但我不知道她就是要嫁给我的人。为了娶到她，我愣是使出了作妖大法，竭尽所能地去搅黄这桩婚事”这样曲折的剧情呀…

经常发生的情况是，上一秒钟我还在对着张若昀X李沁这对“鸡腿夫妇”嗑糖，下一秒就被紧张的剧情一颗心提到嗓子眼。

坐等更新，也期待“小言公子”上线啦，《庆余年》简直就是我一周的快乐源泉！

「Vicky's 资讯」《庆余年》开启付费加更模式，这把双刃剑，观众是否会接着买单？

All Image Source

这段时间讨论度颇高的热门国产剧，《庆余年》绝对可以算作一个。

截止到12月11日，该剧仅在腾讯视频的累积播放量已达到15.9亿次。猫眼全网热度榜上，该剧上线后半数以上日子长居单日热度冠军。随着官微频频放出全方位物料，《庆余年》在微博上也保持着相当高的热度。

根据今日腾讯视频和爱奇艺官微发布的信息来看，从12月11日晚20点开始，《庆余年》将开启VVIP超前点播模式。说白了，就是在VVIP会员在更新日比普通用户多看6集的基础上，可以再多看6集。

换句话说，充值VIP会员还不够。观众需要额外掏腰包，去享受提前观看后续剧情的福利。

事实上，这并不是视频平台第一次在热播剧集上采用付费超前点播的模式。《陈情令》、《没有秘密的你》、《从前有座灵剑山》等，均采用了该模式，并且收获了不菲的收益。

针对视频平台的这一举措，观众们抱有截然不同的两种态度。

一部分观众认为，在盗版如此猖獗的情况下，视频平台的举措无疑保护了创作者们和平台自身的利益。观众为了自己喜欢的剧集付费，也当属一种良性的市场行为。

另一部分观众则表示，视频平台这一行为冲击了他们作为会员的权益。这点，其实有点像是迪斯尼乐园的pass卡。

《庆余年》的观众们，你们会付费多看6集嘛，在评论区留言吧:D

（The End）

【Vicky's 影评】《平日下午三点的恋人》：是什么现实原因，使得中年人成为出轨的主力军

All Image Source

30岁以内的，不是恐婚未嫁就是刚结婚没几年还在蜜月期。70岁以上的，对权钱、欲望的追求会相对减弱，想要出轨也有些心有余而力不足。有些缠绵病榻的，估计还要倚仗于老伴照顾，庆幸老伴没有为自己年轻时候那档子破事发难。

三十到六十岁则是出轨的“黄金时期”，其中又以35到45岁之间为高发期。

经过一段时间共同生活之后，夫妻二人会感到新鲜感减弱，也会因为生活中的琐事怨怼陡增。现代人结婚大多晚于25岁，“七年之痒”恰好落在男性事业上升、女性怀孕育儿的时期，正是第三者趁虚而入的关键时期。

若是中年不得志的男性，在家中得不到慰藉（或是拉不下面子倾吐），很容易在外面寻找安慰和宣泄。男方家暴、性冷淡或不良嗜好（吸毒、赌博、嫖娼），亦会造成女性出轨以求平衡的情况。

此外，婚后夫妻两地分居也是出轨的主要原因之一。

《平日下午三点的恋人》的崔秀雅放弃梦想，嫁给出版社社长后做了全职主妇。丈夫对婚姻和妻子的要求很“朴实”，妻子上得厅堂下得厨房并且把孩子教育好，作为一家之主他只需要为这个家提供经济保障。

至于妻子的想法和情绪，不是他需要去照顾的部分。

在中日韩印等亚洲国家，男尊女卑是一种根深蒂固的观念。不少男性认定有些事是女性的分内之事，比如洗衣做饭带孩子。也就因为这样，衍生出了家庭主妇这么一项“职业”。

就拿我们的近邻日本来说，这个国家以精细和严谨闻名于世，家庭主妇的负担非常重。我曾经听过朋友担心嫁到日本以后，不能给孩子做出花样翻新的便当而害他受到排挤，也害怕因为无法融入邻居或者丈夫同事的太太团而影响丈夫工作。
