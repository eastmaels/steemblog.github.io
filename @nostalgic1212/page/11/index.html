「Vicky's Life」中年女性婚姻现状堪忧，原罪是配偶还是子女？ 190815


几个久未谋面的老同学约好在教师文伯及其妻子戴戴的家中聚会，到场的还是有小老板吴小江和老婆李楠、女强人韩笑、三流编剧贾迪和刚交往的富家女友娇娇。在众人的提议下他们决定玩一个游戏：用餐期间所有人都把手机放在餐桌中央，无论是哪部手机接到电话或是微信都必须当众接听或者朗读出来。

随着剧情的推进，看似外表光鲜亮丽的人们背后那些不为人知的秘密渐渐显露出来…

《来电狂响》改编自著名电影《完美陌生人》，后者曾经有十余个国家对其进行翻拍，其中以西班牙版和意大利版的最为出色。姑且不论中国翻拍的版本是否成功，我觉得《来电狂响》在拿捏男女间的情感问题和都市人的焦虑这点上非常出色。

戴戴是一名颇有威望心理医生，丈夫则是一名受人尊敬的人民教师。加上女儿漂亮聪颖，因此戴戴看上去令人羡慕的一位女性。她在游戏中对丈夫说自己想要去整形和隆胸，对方对此毫不知情也并不支持。二人其实早已协议离婚，却为了不影响还在青春期的女儿选择继续住在一起。

她在影片中的处境，代表了很大一部分中年女性婚姻的现状。与孩子沟通不良进而时常产生争执，我们常说孩子要和家长成为朋友，但做到这点谈何容易？得不到配偶的耐心倾听和支持，中年夫妻间的感情与其说是亲情不如说是搭伙吃饭，貌合神离的婚姻也比比皆是。

随着更年期的到临，如果再得不到来自家庭的关怀和理解，中年女性的处境就显得更加悲哀，无论她们在外是否是征战商场的女强人。中年女性婚姻现状堪忧，你觉得原罪是配偶、子女亦或是社会？

【Vicky's 影评】《德鲁纳酒店》：韩剧女主不再傻白甜，男主也不再精明腹黑霸道总裁-4


All Image Source


亡魂生死观和世界观


作为一部都市传说类韩剧，《德鲁纳酒店》的故事其实并不算非常新颖，但从它超高的收视率来看无疑它的尝试是成功的，这点也出现在洪氏姐妹的另一部剧集《主君的太阳》（苏志燮/孔晓振 主演）之上。神秘瘆人的都市鬼怪传说和浪漫的爱情喜剧融合在一起，同时收获了两波不同喜欢的观众。


究其成功的原因，很大一部分源于我们对死后世界的好奇。《德鲁纳酒店》中提及的生前做善事，去世时前往黄泉的接送车也会是高级车辆。再比如此生多做好事，来生也不会轮入牲畜道与人饭桌上相见。


德鲁纳酒店的工作人员更像是普渡众人的摆渡人，尽可能地满足来到这里的客人们。为饿死的亡灵提供一盘又一盘的美味佳肴、为冻死的故者添加一根又一根柴火、为此生还未把想要学的知识学尽的老太太提供海量书籍，甚至为因公殉职的女警官伸张正义…


在这家酒店里，亡魂已经不再是面目可憎的可怖存在，他们的故事甚至有些让人感到鼻头一酸近乎落下泪来。人生在世，我们永远不知道未来和意外到底哪一个会先到来，未能来得及好好和家人、朋友、爱人告别也是常事。看到这部韩剧以后，我觉得如果这世间真的能有这样一家酒店该有多好。


我们常常会说，人生只有一次一定要好好对待，但又何尝想过人死去这件事一生也只有一次呢？愿所有的逝者都能得偿所愿，无论多么细小的愿望都能被认真对待，而那些冤屈、背叛、屈辱也终将沉冤得雪。

「Vicky's Life」鱼摊门口的燕子一家 190814

周末我和妈妈去鱼摊买鱼的时候，发现摊位门口路灯上又新出了一窝小燕子。如果我没记错的话，这已经是第三窝小燕子啦。年初的时候遇到的那一窝，燕子妈妈和爸爸总是交替着出去觅食，而小燕子们长着嘴巴嗷嗷待哺的样子像极了小时候的我们。

鸟爸爸一起抚养下一代的习性，不同于很多别的鸟类，这让我对燕子这种生物好感度多了很多。夜幕降临，忙碌了一天的燕子爸妈们会休憩在卷闸门的边缘上，把鸟窝让给渐渐长大的孩子们。


All Image Source

让我觉得感动的是，待到小燕子们渐渐长大后鸟父母会离开，将这个鸟巢彻底让给他们。哥哥姐姐们会接过父母的重担，起早贪黑地为弟弟妹妹们寻找吃的东西。这点习性，甚至比人类里面的兄弟之情还要来的深厚感人。

鱼摊老板们说，如果卷闸门真的落下来了几只鸟儿都会睡在窝里面。早上开门的时候，几个小脑袋探出头来的样子实在是太可爱了。

「Vicky’s Life」《恋之运》：姐弟恋是否可取？ 190814


31岁的轻熟女平和子和同居男友相恋多年，已经到了即将筹备婚礼的阶段。炎炎夏日，她在打工的电影院里遇到了生命中的“限量款”，15岁的男高中生伊古梦明（神尾枫珠 饰）。一直等待被别人选择的和子终于有了选择的机会，是安心和男友走向婚姻还是选择如烟花般绚烂但也可能转瞬即逝的爱情？

本片根据新田章同名人气漫画改编，由Netflix和东京电视台联手打造。岛国的深夜剧一直以大尺度著称，细数一下我看过的大概有《大叔之爱》、《贤者之爱》、《双重幻想》、《Around 30 无修正》、《昼颜：工作日下午3点的恋人们》、《东京女子图鉴》…其中不乏以“姐弟恋”为主题的影视剧。

我认为姐弟恋未必就不靠谱，主要在于两个人的价值取向是否一致。影片中的女主角比男主角大了足足15岁，很多时候都是和子在迁就孩子气的伊古。30岁的女性更期待的是稳定踏实、气氛融洽、兴趣相近、能够朝着同一个方向努力的另一半，而年轻的男性也许更希望的是轰轰烈烈的爱情。

价值取向的不同让彼此间的磨合期变得异常痛苦，最终导致了两个人分道扬镳。影片的最后，和子找回了自己最初的梦想开了一家电影院，首映礼上播放的正是立志成为导演的伊古的新片。两个人最后是否有重新在一起似乎也变得没那么重要了，他们都找到了自己生活所要前进的方向，不是吗？

【Vicky's电影/电视剧推荐】胖子的老婆那么美？！我放弃减肥了！-4

All Image Source

第三，我们来说包贝尔的漂亮老婆克拉拉。她出现的场合，除了在医院为了丈夫据理力争那场戏以外，基本都是文章在无限意淫。我看过不少up主对这部电影的解说，均是用“恶俗”、“没眼看”、“up就不说了大家自己去看吧”来一笔带过。

克拉拉和包贝尔相遇的那场戏，也很值得玩味。因为身材的关系，一再失恋的克拉拉在医院顶楼决定跳楼，碰上了在楼下劝说的保安包贝尔。这个画面就很诡异了，克拉拉是日本人，楼下所有的围观群众也是日本人，包贝尔的人设也是在日本医院的小保安。

所以导演你倒是和我说说，这理论上应该全是讲日语的环境为啥克拉拉讲的是中文？此处不应该是克拉拉讲日文，包贝尔用蹩脚地日文笨拙应对，然后配上双语字幕更为合理吗？

最后，我要吐槽一下郭京飞这个终极大boss。一个实业家+黑帮老大，讲电话居然开美颜相机，平时的爱好还是吃咪咪虾条。好吧，你说这是他的个人喜好别人管不着，我也忍了。但是，这样一个响当当的人物跑路的时候能不能为自己准备一架好一点儿的直升机，好歹能负重量超过600斤那种？

我之前看过一句话，大致的意思是不能因为是喜剧片就可以抛弃基本的逻辑和社会常识，并不是说能让观众捧腹大笑就是全面的胜利，这句话用来《胖子行动队》是再恰当不过的了。

另外我还想说一句，隔壁家的邓超为了电影爆瘦40斤，阿米尔汗大叔为了电影一会儿爆肥一会儿爆瘦，这才是真正敬业的演员啊。如果包贝尔和文章真的有心往好演员的方向发展的话，建议还是向上述我提到的这两位演员好好学习一下吧。

毕竟比起爆瘦，增肥应该…并不算太难吧。

本片中最大的亮点我觉得就是郝英俊的日本老婆金秀珠，无论身材还是颜值简直就是所有男人的理想型。最关键的时候，变瘦变美的金秀珠并没有因为郝英俊还是300斤的大胖子去抛弃他，人家可是自始自终只爱郝英俊一个人。

有这样的老婆还减个肾肥啊！我身边的异性朋友们纷纷表示：“我不减肥了，请赐一个这样的媳妇给我！！！”

【Vicky's电影/电视剧推荐】《就算敏感点也无妨》：任何让你感到不适的异性举动，都是骚扰！


All Image Source

这部电视剧一共12集，每集5-10分钟。

本片主要讲述二十岁的年轻人在校园内两性关系的困扰，剧情将围绕性骚扰、女权主义、男女间的交往等问题展开。不同于以往死板的教科书式的说教，《就算敏感点也无妨》以轻松喜剧短片的形式告诉女孩子们在面对男生时，我们是可以大胆say no的。

郑信惠看起来就是一位呆萌的大学新生，在迎新会上就会被图谋不轨的学长盯上了。这位学长是学校的风云人物，当对方接近她的时候说实话她是心里小鹿乱撞的。只是在迎新会上，她渐渐地发现这个学长举止很奇怪。

学长会拉起并不太熟悉的郑信惠的手指去解锁手机要号码，也会带着酒气地靠近女生耳边说话，更会猝不及防的握住女主的手。这些画面，大家是不是会觉得很熟悉呢？

我们在生活和工作中，总也会遇到这种有些“自来熟”的人。看起来牲畜无害很热心，但总是有意无意地靠近着异性或者在对方肩膀上搭一下、腰上搂一下、手上摸一下。倒也不是什么太出格的行为，你也找不到什么特别的错处去反驳对方，但就是让你会觉得很不舒服。这时候，反而是女孩子会在心里犯嘀咕：“是不是我太敏感了，对方其实也没有恶意吧？”

但我很肯定地想要告诉心里在这么想的妹子没，你们想的没有错。因为你觉得那个人行为举止很奇怪，那么这个人就真的是奇怪的人。

现在无论是校园中还是职场里，性侵害的事件越来越多。究其原因就是因为我们实在是太不敏感了，再者也是家长对孩子的性教育、安全意识教育太差了。

作为孩子的家长，不要把这份责任寄希望于学校的老师。《熔炉》和《不能说的夏天》这两部电影以及《房思琪的初恋乐园》这本书，都告诉我们我们以为为人师表的老师很多时候正是孩子们的恶梦。

作为家长，教会孩子保护自己的技能和勇敢说“不”的能力，比上再多的艺术课程和奥数班都要有用的多。
