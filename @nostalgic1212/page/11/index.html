「Vicky's 影评」《双子杀手》：从好莱坞华人之光，到接连票房失利，李安的神话是否还能延续？#1


All Image Source
文/冰冰


“我超爱李安，告诉他，快点来找我拍片，不然我快变老了！”


2013年，史皇威尔·史密斯在台湾宣传《重返地球》的时候，曾经向李安导演隔空喊话过。


时隔六年，两人真的合作了一部新片，还在史皇在影片中见到了20多岁的自己。


那就是10年底唯一上映的好莱坞大片：《双子杀手》


美国国防情报局特工亨利（威尔·史密斯饰）在完成一个刺杀任务后，职业敏感性让他意识到这里面并不简单，于是萌生了退役的念头。


也许是唯恐他将国防情报局那些见不得人的秘密抖出去，刚到达休整地的亨利就发现组织派人在监视自己。随之而来的，是一个神秘黑衣人的疯狂追杀。


在激烈的交手中，亨利发现对方居然是年轻版的自己。组织在未经他同意的情况下，利用亨利的DNA暗自培养了一个他的克隆人。


就此，一场我与“我”的对局旋即展开。


「电影真的需要如此真实吗？」


在我看这部电影之前，就发现北美影评人对其评价不高。在烂番茄上，虽然观众喜爱指数有85%，但烂番茄指数仅有尴尬的29%，还不到及格线。


大多数影评人，无论评价褒贬，大多用到了一个词：


技术的尝试


纵观这几年李安导演的作品，他已经把最新的电影技术当作电影表达的重要形式。


这次，《双子杀手》沿用了卡梅隆（好莱坞另一位技术狂魔）称之为“新白金标准”的视觉技术：4K，120帧，3D。


翻译成大家更熟悉的词汇就是：更清晰、更流畅、更纤毫毕现。


那么《双子杀手》带来的第一个问题来了，电影真的需要如此真实吗？


电影自然是需要真实的，但只需要做到在特定情况下故事设定值得推敲即可，不是让在座的观众体验整个故事真实发生的过程。


《双子杀手》给人的感觉是所有演员都是贴在摄像机上面，他们每一个微表情、面部的毛孔哪怕是嘴唇的干皮都一览无遗，就像真实站在观众面前一样。


这种临场感带来视觉震撼的同时，也破坏了影片本身的电影感。


更有观众表示，其远不如《谍影重重》和《007》系列来得让人感到肾上腺素激增。


当然了，单从用影像去讲故事这点来看，李安导演对艺术的感知能力并没有衰退。追求视觉感受的观众来说，《双子杀手》确实提供了一出堪称浸入式体验酣的动作戏。


如果你的观影需求是这样的话，那么120帧版的《双子杀手》应该会是你的心头好。


话说回来，这部电影的问题究竟出在了哪里呢？


简单来说就是：剧本。

【Vicky's 影评】《祈祷落幕时》：连环杀人案背后另类的父女之情#5


All Image Source


被加贺恭一郎记恨了很多年对家庭对母亲毫不关心的父亲加贺隆正，其实并不如儿子所想的那样。直到他临死前对护士的一席话，观众们才知道这位父亲只是不知道如何表达自己对家人的爱。在老刑警加贺隆正看来，努力工作、努力挣钱养家就是对家人最好的爱意，却不曾腾出时间来关心妻子在家族中的处境。他的这种爱意，很容易让家人理解为对他来说工作比家人重要。


只是，即便是将死之时，加贺隆正依然没有放下对妻子加贺百合子的爱意。他一直非常介怀妻子去世的时候，身边并没有照顾她的人在，所以对来看自己的儿子说：「我死的时候你不要在我身边，我要自己死去」这样的话。


加贺父子之间的悲剧，在于二人都把爱意埋得太深太深了，那些没有说出口的亲人之爱恰恰成为了筑起父子之间隔阂的高墙。比起死亡来说，加贺隆正更害怕的是见不到儿子。也许正因为此，弥留之际的老爷子告诉护士自己终于可以摆脱肉身这个累赘，自由自在地看着儿子了吧。


「浅居博美」


东野圭吾笔下有很多角色都是为了爱而杀人，杀人是他们表达爱意的方式，这点用在浅居博美这对父女身上再合适不过了。无论是在看书还是观影的过程中，读者/观众会一方面恐惧于其杀人行凶时候的情景，又在另一端同情一些嫌疑人作案背后的隐情。


父亲的每一次作案背后都是情非得已，因为他不能让别人认出本该跳楼自杀却还活着的自己，更不能让自己的存在毁掉了努力爬到现在事业高度的女儿。浅居博美不忍看着父亲生生被火烧死，选择了掐死父亲让他不要在自杀的过程中受到过多的痛苦。


在被这段父女之情感动落泪的同时，他这种只顾天伦之乐而危及到他人性命的行为，又让观众觉得这份父爱未免显得过于沉重甚至有些畸形。在观影结束后，我们不经想问自己：如果我们深陷这对父女的处境之中，自己又会做出怎么样的选择呢？


这部电影中包含了凶杀、自杀、出轨、师生恋、抑郁症、校园霸凌、高利贷等多种畸形元素，但最后的核心依然是「爱与牺牲」。本片用多线叙事的方式，将这个错综复杂的案情呈现在观众面前。


加贺恭一郎无疑是幸福的。他有为了他的安危而避走他方，并在远方一直挂念他的母亲，也有不苟言笑却深爱着他的父亲。而浅居博美呢？虽然有父亲深沉的爱，但这份过于沉重的爱显得有些病态。而那个在外头养小白脸，因为躲避高利贷离家出走最终将她和父亲推入绝境的母亲，却从来没有教给她任何的母性。


如果有时间的话，冰冰非常推荐大家去看一下这部电影。

「Vicky's 短评」《生门》：每一对母子/母女，都是生死之交 191025

纪录片《生门》是一部关注中国早产儿家庭的纪录片，导演陈为军执导耗时三年多时间，于16年在国内院线上映后获得一致好评。17年导演放出了13集的剧集版，将几十位受访产妇及其家人在医院的待产过程展现在观众面前。

影片以四个经历极端情况的早产儿家庭为窗口，反映了一个个小生命和他周遭环境的关系。这个环境就包括了他出身的家庭、他辗转经过的医院、他在生产过程中遇到的医护人员以及这个大环境下的医疗保险制度和医院治病机制等等。

我们都知道生命是金钱换不来的，但是没有金钱连换生命的机会都没有。为此，作为女性一定要有自己的收入来源和社交圈子。如果有强势的娘家，在特殊时期可以作为坚强的后盾，那么就再好不过了。

女性在恋爱/结婚一定要擦亮眼睛，切莫被爱情冲昏了头脑。

不少未婚女孩子看了《生门》以后感叹，都说结婚等同于再投胎，那么生孩子就真的是在鬼门关走了一遭。正在备孕中的女性不敢看，而已经生产过的女性则庆幸丈夫还算照顾，也有父母足够的疼惜。

【Vicky's 影评】《祈祷落幕时》：连环杀人案背后另类的父女之情#4


All Image Source


《祈祷落幕时》整个时间跨度长，涉及的角色关系多，使得整个故事在推进的过程中显得扑朔迷离。虽然这是「加贺恭一郎」系列的最后一部，但即便观众没有看过之前的十几部小说，依然不影响看这部电影时的观影体验。


我们在看别的推理小说的时候，结局处常常是对人性的丑恶、社会的阴暗、体制的不完善的控诉，而《祈祷落幕时》恰恰体现出来的却是人与人之间最深的牵绊以及最真挚的爱意。这一些，是从「父母对孩子」和「孩子对父母」这两方面来展现的。


如果你经常阅读东野圭吾的小说就不难发现，在这位作者的笔下爱的本质就是「自残和牺牲」。影片中那种以爱之名所犯下的种种罪行，是最令读者和观众感到心酸的。世间有太多正义无法通过合法的手段伸张，“爱”便成了一切行为起始的动力。


加贺恭一郎的母亲因为不想让心爱的儿子受到伤害，毅然决定出走几十年都不曾再见儿子一面；浅居博美的父亲为了保护女儿来之不易的事业和生活，改头换面顶着别人的身份过了十几年，更是为了她手上沾满了鲜血。观众们即使隔着屏幕，都能感受到她父亲无言的父爱。


在《祈祷落幕时》里面，每个人似乎都在为爱奔走。这里面有些“以爱之名”所做出的行为看上去有些极端，在寻常人的眼中有些爱甚至沉重地让人几乎无法呼吸。但不可否认，这种爱是那么浓烈，以至于完全无法让人回避。


「加贺恭一郎」


加贺恭一郎的母亲加贺百合子因为患有严重的抑郁症，曾经在半夜起来走进厨房拿起一把菜刀，而年仅十岁的儿子目睹了这一切。儿子的出现让这位母亲突然意识在，自己如果继续再待在家里的话，很可能会危及到儿子的生命。


于是这位母亲毅然留书一封出走，再也没有回到家里。即便是多年以后看到儿子剑道比赛获得冠军照片刊登在杂志上，她都抑制住想要去见儿子的心。与浅居博美的母亲抛弃丈夫和女儿离家出走不同，加贺百合子正是因为对儿子的爱才选择放逐自己的。


在母亲去世后，加贺恭一郎去收拾母亲遗物的时候发现了在每一页上写着桥名字的日历。为了搞清楚母亲离开自己的这些年过得怎么样，他围绕着这些桥调查了许多年。作为警局才华横溢的明日之星，加贺恭一郎却自愿申请调到日本桥警署任职，这一待就是十几年。


对他来说，母亲是他与这个世界联系最重要的纽带之一。

在案件陷入胶着状态的时候，加贺恭一郎提醒松宫脩平不要从外部事物上提取DNA，因为那很可能是凶手设下的迷阵。果不其然，在警方重新提取DNA试验后立刻证明先前的推断是正确的，河边的流浪汉正是押谷道子那位消失的房东。


那么剩下的疑问就是，屋内的日历和加贺恭一郎母亲加贺百合子（伊藤兰 饰）遗物中的日历是否存在联系？经过笔记鉴定，两本日历上标注大桥名字的字迹出自同一人之手，情况就显得更加破朔迷离了起来。


与此次案件看起来有千丝万缕关系的加贺百合子，背后又有怎样的故事呢？原来，之前做过陪酒女的加贺百合子在加入夫家后一直不受待见。加上加贺恭一郎的父亲加贺隆正（山崎努 饰）因工作经常不在家，久而久之加贺百合子患上了严重的抑郁症，情绪一度崩溃。


于是有一天，加贺百合子留书一封后决定离家出走，直到几年前加贺恭一郎收到母亲的遗书，才勉强拼凑出这几年母亲在外的生活。原来离家后她在一个小酒馆继续做着过去的营生，与一位男士交往甚密。经过当时小酒馆老板娘的辨认，此人正是河边的流浪汉。


到了这里以后似乎再次进入僵局，加贺恭一郎决定还是从手头有的线索开始查起。他和松宫脩平沿着日历上标注的桥一路走访过去，发现日本桥附近每年都有「洗桥仪式」，并且留有大量的照片。加贺恭一郎通宵看了这些照片，以外的在多张照片中都发现了浅居博美的身影。


线索似乎都汇集到了这个女人身上，于是加贺恭一郎独自前往她的故乡进行调查。


看到这里，观众们的问题估计会比答案要来得多。


「浅居博美当年真的突然转学了吗？」、「她的父亲真的是跳楼自杀的吗？」、「她的父亲真的死了吗？」、「和她有暧昧关系的苗村老师为什么会突然人间蒸发？」、「她在这一系列的案件里面究竟扮演了怎样的角色呢？」


《祈祷落幕时》毕竟是一部悬疑推理剧，冰冰在本篇里面已经透露的过多了。关于剧情的部分就此打住，关于揭秘的部分就留给大家自己去看。

东京某公寓内发现一具死去多时高度腐烂的女性遗体，经法医解剖判定其是被勒死的。在警方的调查之下，发现死者是来自滋贺县的押谷道子（中岛博子 饰），而她的死亡属于他杀。同时，道子的房东也消失不见。


根据押谷道子的亲友们称，她不曾离开过自己所居住的城市。那么她为什么突然去了东京呢？她是要去见什么人吗？又为何会死在东京的一所公寓里呢？


与此同时，在道子遇害公寓不远处的河边，路人发现了一具被烧成灰烬的男性尸体。负责这起案件的警察松宫脩平（沟端淳平 饰）推断是否河边被烧死的尸体就是消失的房东，但由于二者DNA不一致证明推断不成立。


松宫脩平顺着线索来到押谷道子曾经去过的疗养院，从护工那里得知道子在这里见过一个201号房的怪人，认出此人正是同学浅居博美（松岛菜菜子 饰）的母亲浅居厚子（木村绿子 饰）。


获得此重要线索的松宫脩平赶紧去找浅居博美询问案情，通过对话二人得知浅居博美目前是一名知名话剧导演，且在两日前刚刚见过押谷道子。押谷道子希望浅居博美能去疗养院接回赖在那里的母亲，却被浅居博美断然拒绝了。


原来，早在几十年前嗜赌的母亲抛弃了父亲以及还在念书的自己，留下一屁股的债务离家出走。不断被债主上门逼债而走投无路的父女，在一个月黑风高的夜晚毅然决定逃走。后来，父亲终是不能面对这一切，抛下年幼的女儿原则跳崖自杀。


即将离开的时候，松宫脩平在浅居博美工作室的照片墙上偶然间发现了浅居博美和自己表哥加贺恭一郎的合影。于是松宫脩平不经发出疑问：原本不该有联系的两个人，又是为何会认识的呢？


在询问过加贺恭一郎（阿部宽 饰）之后得知，原来身为剑道高手的他曾经给浅居博美的演员们上过辅导课，但是两者其实并不算太熟悉。加贺恭一郎随口向表弟问及为何要调查浅居博美，对方于是说起了最近遇到的案件，并随口提了一嘴在公寓内什么也没有发现，就只有一本写了字的日历。


这让加贺恭一郎异常兴奋，在仔细比对上面的文字和笔记之后，他发现这是一本和母亲遗物里的日历一模一样的物品。
