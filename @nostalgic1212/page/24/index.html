「Vicky's 资讯」《陈情令国风音乐大碟》开售，令牌们这次会买单吗？

All Image Source

由著名音乐人林海老师亲自操刀的《陈情令国风音乐大碟》于今日开售！

里面除了29首原声配乐与人物单曲，还有肖战、王一博等演员的超独家未公开花絮照和人物专访。N刷剧集和花絮们的姐妹们，又有新的物料可以嗑啦。

除此以外，这张大碟内还附上了片尾曲《无羁》曲谱，最近在学笛子和古琴的小伙伴们有福咯。不仅如此，里面还会随机附赠各式人物的小卡片。悄咪咪地说，我最希望得到的当然是忘羡CP啦。

《陈情令》这个IP在经历了腾讯视频付费播放、音乐收费专辑、演唱会、相关周边售卖及最近即将上映的电影版之后，这次发行CD版《陈情令国风音乐大碟》也在网路上引起了不小的反响。

一部分网友认为，XP这波操作算是把一个知名IP的利用价值榨干到最后一滴，已经让曾经的影迷们感受到了厌恶；

也有一部分网友们认为，冲着林海老师制作精良的音乐，也会将这张专辑带回家好生收藏的。

令牌们，你们这次还会下单去抢购这张专辑吗？

「Vicky's 短评」《两只老虎》：这部黑色喜剧，你走进影院观影了吗？

All Image Source

低配版绑匪余凯旋（乔杉 饰）遇上极品人质张成功（葛优 饰），赎金居然被人质从100万还价到了200万？？？

威逼不成，反而被人质利诱+威胁？咱们这个绑匪啊，最后傻愣愣地替人质大哥办了这些年他想做，又一直没有勇气去做的事。

究竟是哪三件事儿呢？这里先卖个关子。

《两只老虎》中的双人模式，近年来已经有不少的电影使用过，比如大名鼎鼎的《人在囧途》、《泰囧》、《宿醉》等。

从喜剧效果来看，《两只老虎》完成度很好。笑点密集，在观影的过程中多次全场爆笑，我边上的小哥笑得差点背过气去…

对了，这里尤其要表扬一下闫妮。

从《武林外传》开始，闫妮一直在演技上与自己角逐。此次她在《两只老虎》中所饰演的彩霞，给人以一种“风情万种里悲喜交加”的暧昧感，超赞！

葛优在戏中的发挥一如既往的稳定，你葛大爷还是你大爷！

此外，肖战也是本片的另一个宣传点。

火爆一整个夏天的《陈情令》以国风演唱会作为完美落幕，舞台上肖战、王一博等主要演员向纷纷与成就他们的角色告别。

本月6日，肖战主演的第二部电视剧《余生请多指教》也顺利杀青。演戏、综艺通告不断的小赞，再度回归歌手的身份，于今日发布了电影《两只老虎》同名推广MV。

“我只是一只想你的老虎，我只是一只孤独的老虎，我只是想挣脱所有束缚。”

整首歌节奏感十足，风格轻快，旋律很容易让人听得随之摇摆。歌词近似经典童谣，非常朗朗上口。

肖战的粉丝们一定要坐到最后，他演唱的同名主题曲《两只老虎》将会在演职员表出现的时候响起。

有不少观众和冰冰一样，都是安静听完歌以后再离开的呢:D

这个周末，赶紧买张票去观影吧，值回票价！

「Vicky's 资讯」：最近的新闻，我想推荐给大家看这四部影视剧

All Image Source

崔雪莉因为网络暴力而患有重度抑郁症，具荷拉遭遇暴力和性侵害，让这两个原本前度无量的女孩子早早结束了生命。

Papi酱旗下艺人宇芽，曝光自己被前男友沱沱的风魔教多次家暴后，相关部门立即开展调查处理这两起事件。

目前，沱沱已被依法拘留，宇芽也申请到了人身安全保护令。

无独有偶，之前因为家暴日籍女友上过热搜的蒋劲夫，其另一位外籍女友也爆出其曾对自己多次家暴的证据。

蒋劲夫于27日亲自发微博澄清此事，并表示“我没做的，我也不想被冤。”

新闻真的不能连着看，我在气愤过后更多的是感慨，这里想要推荐给大家4部影视剧。

《解除好友》

当下的互联网，简直就是个暴露人性的动物世界。

前一秒还是粉丝的，下一秒钟也许就脱粉，甚至转为黑粉；前一秒还在散播对方不实流言、用最恶毒的语言攻击他人的，在对方死后又加入维护和悼念当事人的行列…

这是一部会让键盘侠们看完后，背后渗出丝丝凉意的电影。

《就算敏感点也无妨》

女孩子们，情到浓时会让我们做很多不理智的行为。但大家千万记住，某些私密的视频和照片千万不要留。即便是亲密爱人，当他们提出让你们不舒服的要求，也一定要果断的拒绝。

那些是男人拿去炫耀的资本，却可能是女人一辈子深埋心底的恐惧。

《监护风云》

很多人都觉得家暴离我们很远，但事实上，也许它生在一墙之隔的地方上演。

对于家暴，就像是出轨一样，就只有“0”和“无数次”这两种情况。如果遇到这种情况，尽快让周围的亲朋好友知晓，第一时间去报案并且去医院做好证据固定。

事实上，我并不能理解为何又“家暴”这种设定。

这明明…是赤果果的故意伤害，不是吗？

《丈夫得了抑郁症》

强调一句，抑郁症不是简单的心情不好，不是玻璃心，是一种疾病。

如果你身边有抑郁症的亲朋友好友，请对他们多一些耐心和关心。很多时候，你的一个善意的举动会拯救他们，反之可能会成为压死他们的最后一根稻草。

「Vicky's 资讯」：崔雪莉和具荷拉相继去世，抑郁症和故意伤害，到底离我们有多远？

All Image Source

2019年的最后一个季度，娱乐圈真的特别丧。

先是崔雪莉因长期遭受网络暴力患上抑郁症，被发现在家中自杀身亡，年仅25岁。

雪莉去世之后，她的生前好友具荷拉曾在直播中公开悼念，说一定要好好说下去。

可让人意想不到的是，11月24日下午，年仅29岁同样身患抑郁症的她也被人发现死于家中。

11月27日，具荷拉的哥哥晒出与妹妹生前的聊天记录截图，并表示：

“很抱歉我们都没再大一点声告诉你，希望你永远快乐永远幸福。虽然现在为时已晚，但是还是希望你能听到我们的声音，在没有坏人的净土要没有任何烦恼，比任何人都要幸福地生活呀。”

原本，17岁就出道的荷拉事业上可以说是顺风顺水。多栖艺人的她，在中日韩都有很高的人气。

但这一切，都在2018年急转直下。荷拉的发型师男友崔种范，将她拉入了地狱的深渊。

男方对荷拉进行暴力攻击之后，还公布照片，公布自己才是被打的那一方。事实上，荷拉的身上有多处瘀伤，医院的争端报告上甚至有「子宫及性器出血」的诊断结果。

哪一方才是受害者，一目了然了吧？

更过分的是，崔种范还以荷拉的私密照片及视频步步紧逼。即便他没有亲手杀了荷拉，其举动也极有可能促成了今天的结局。

交友需谨慎，远离暴力，远离渣男！

「Vicky's 资讯」《追我吧》：高以翔之死，为什么我们无法接受？

All Image Source

高以翔确定死亡以后，《遇见王沥川》导演陈铭章自言自语地否认：不信！不信！不信！

我想，这也是很多粉丝们的心声，大家都不能接受这件事情。

就我们所知，高以翔是在录制一档名为《追我吧》的综艺节目时，突发身亡的。等大家回过神来以后，不禁想去探究，这究竟是怎样的节目。

简而言之，《追我吧》是一档在规定时间内，明星跑素人追的高强度综艺节目。

为了赶时间，他们需要不停地奔跑并通过一个又一个高难度关卡。其中最可怕的一关，嘉宾们需要借助绳索，徒手攀爬70米的垂直高楼，再从顶楼速降到对面楼。

别说里面的女艺人们没有准备好，即便是奥运冠军们，都直呼“跑不动了”。要知道，咱们国家队的训练绝对是魔鬼级别的。

再者，为了体现「大型都市夜景追跑竞技秀」的效果，节目录制时间安排在晚间。

众所周知，入夜以后人类的身体机能会逐渐减弱，这个时间段本来就不适合做如此高强度的运动。

录制时间长，一不小心就会拍摄到凌晨，这可和我们平时捧着手机刷剧或者吃夜宵的那种熬夜不一样。

昨日，浙江卫视就高以翔录制《追我吧》突发意外去世发布了声明。

“对此意外造成的无法挽回的严重后果，我们深感遗憾和惋惜，并愿意承担相应责任。”

此外，节目组还表示会深刻反思原因，对节目录制所有环节进行全面检查，更周全地做好节目安全保障工作。

如此官方，看起来就是那种不知道重复了多少遍的通稿。

在这个“娱乐至上”的时代，观众们太难讨好了，所以各个节目组不得不寻找更能刺激观众，锁定收视率的方案。

细观这几年的真人秀节目，嘉宾出现事故并不是一次两次。

也许，高以翔的意外，正是给这个圈子创作者们一个审视自己的机会。

（The End）
