【Vickyの杭州印象】北山路上的西泠印社 The Xiling Seal Club in Beishan Road


杭州西泠印社（又名：著名金石篆刻研究学术社团）始建于清朝光绪年间，由几位浙派篆刻家牵头创建的。其位于杭州西湖景区北山路孤山南麓，附近分别有白堤、西泠桥、放鹤亭、北山公园等景点。放眼望去，引入眼帘的都是沁人心脾的绿色。

想到幼年时学习毛笔书法，篆刻自然也是必修科目之一。当时的师父是杭州碑林的馆长，与西泠印社第二任社长沙老颇有些渊源。也因为此，我曾经是印社年龄最小的名誉社员。只是后来课业加重，渐渐也是断了这条道路。

我曾在很多《盗墓笔记》书迷的笔下见到西泠印社这几个字，才意识到南派三叔真的吸引了很多人到杭州一游，兴许曾经还有过书迷们在此聚会也未尝可知。也因为这样，我在周末起了个大早搭车前往此处。

其实我已经有近20年没有真正踏入西泠印社了，每次也都只是草草经过牌匾。经过西泠印社四个大字后拾级而上，这山上7000多平方米都是印社的区域。转了好一会儿也没有见到书中所提及的「吴山居」，只好找到一位工作人员问路。

想来也是被问习惯了，店员熟门熟路地告诉我说现实中的西泠印社并没有小说中的吴山居。一定要说和书中的联系，边上的楼外楼倒是无邪和张起灵曾经吃饭过的新月饭店的原型。当然了，听说有人在附近开了一家名为「吴山居」的民宿，为千里迢迢赶来的书迷提供了一丝念想。

【Vicky's 碎碎念】关于结婚生孩子这件事的迷思（1） 190720


All Image Source

今天陪朋友去看医生，我们发现妇产科都是女性在被扎针或者做治疗。如果有男性患者需要扎针，居然还可以享受优先叫号的待遇。在等候区等待被叫号的时候，我们俩不禁开始讨论「如今女性在承担同等的社会责任的情况下，为何还要承担起生产/养育孩子的社会责任」这个问题。

在封建时期或者当今较为偏远地区，如果两个人结婚以后一直没有孩子，很多流言蜚语都会认为是女性不能生养。有时候即便是丈夫的基因或者是生理机能的问题，为了顾全其男性的颜面，也都会说是女性的问题。更常见的问题则是重男轻女的思维依然存在，我就听过很多人因为没有生男孩而被婆婆和小姑子们为难。

拜托，这都二十一世纪了怎么还有人搞不清楚状况。在生男孩还是女孩这件事情上，男性基因决定了一切女性根本插不上手好不好！


All Image Source

在回家的路上，我们俩接着聊到生活品质的话题。就我们自己而言，毕业五六年以后大多有了一定的存款。住在家里因此基本没有房租和吃喝费用上的压力，靠着自己的年薪和兼职，想要买衣服鞋子包包没问题，一年出去旅行两次自然也是没有问题。

有一定的生活常识，即便是租房子出去住也不会照顾不了自己。有自己的生活圈子，也有积极向上的兴趣爱好。如果要给我们自己目前的生活打分的话，基本可以打到6-7分左右吧。如果选择结婚，那么我们的生活会有怎么样的改变呢？

如果两个人结婚，在不考虑买房买车的情况下基本上生活品质是不会有太多的调整。然而，如果打算在没有父母支持下选择买房，那生活品质应该起码会降低一半。

【Vicky's电影/电视剧推荐】《游戏之夜》：一部全程无尿点的家庭温情悬疑喜剧电影（上）


All Image Source


时至今日，我身边依然有很多朋友还在怀念《老友记》这部经典美剧。


今天冰冰想推荐给大家一部被影迷称为电影版《老友记》的电影，就是这部《游戏之夜》。本电影由约翰·弗朗西斯·戴利执导，他本人是热播美剧《识骨寻踪》的常驻演员，饰演心理学家Lance Sweets这一角色。


马克斯（杰森·贝特曼 饰）和安妮（瑞秋·麦克亚当斯 饰）都是顶级的游戏痴迷玩家，两人在某个游戏之夜相遇并相恋。
就连求婚马克斯都是通过“我画你猜”这个游戏进行的，而他们的婚礼则是一个大型“跳舞机比赛”现场。志趣相同的两个人婚后生活很是羡煞旁人，还时不时邀请朋友来自己家进行一场游戏之夜。


一定要说美中不足的是，两人婚后久久没有baby。通过检查医生推测是马克斯长期在过于优秀的哥哥的阴影下，导致自己精子活跃度不足。Anyway，虽然这个理由有些牵强，既然是喜剧我们就不要过多去追究了。安妮心生一计，她准备举办一场游戏之夜，邀请那些经常在一起玩的朋友还有哥哥一同来参加。


这一次，她要确保丈夫一定会获胜。


他们的邻居黛比去世以后，他的丈夫加里（杰西·普莱蒙 饰）曾经参加过几次游戏之夜。但由于其奇怪的个性常常是话题终结者，马克斯他们渐渐就不喊他了。为了避开加里，好友们从窗户翻墙而入，只有哥哥布鲁克斯（凯尔·钱德勒 饰）大张旗鼓地前来。


不出所料的是，布鲁克斯再次大获全胜还抖了很多马克斯小时候的丑事出来。聚会结束后，布鲁克斯邀请大家去他位于富人区的house参加他举办的游戏之夜，他保证一定会让他们永生难忘。

【Vicky's 旅行】上海国际潜水展 / 2019 DRT SHOW Shanghai


去年四月份我在马来西亚·仙本那参加OW课程，由此也写了我人生中第一个长篇游记系列。之后的一年里，我偶尔会去室内潜水池“泡水”，倒是再也没有找到机会真的回到海里去。今年年初，师兄和我说上海国际潜水展快要开始了。

于是乎呢，我赶紧屁颠屁颠跑去上海和一票师兄弟妹们面基啦。

DRT是Diving(潜水)、Resort(度假村)和Travel(旅游)的缩写，其内容涉及了水下摄影、潜水旅游、自由潜水、技术潜水、医疗潜水、潜水教学等诸多方面。除了商业合作以外，DRT也是真正关注潜水事业发展以及海洋生物保育的组织。

因为其巨大的影响力，每年的展出都吸引了大量参展商和潜水爱好者的光临。

有不少人都是教练miumiu的学生或者认识的人，所以大家面基以后就像是老朋友一样聊开了。我们在现场还连线了当时正在宿务的miumiu，当即决定过年的时候就去巴厘岛找她玩。如果有机会的话，可能还会考虑参加AOW进阶课程。

展出人很多，我一路处于拿小册子/小礼品和买买买的节奏中…标准的中国式游客:D

看到娃娃就走不动的我大概买了十几只小企鹅（准备送人），算上之前买的抱枕和一些七七八八的东西，还好师兄开了车来！！话说，要不是妈妈耳提面命，我差点儿还买了潜水的脚蹼…冷静下来了一想，作为不怎么去的一项活动却是不需要坐那么大的投入。

大家结束以后还找了餐厅继续meet up，回头等写美食篇的时候我再慢慢分享给大家噢:D

【Vicky's 碎碎念】写作课程知识整理：如何获得机器更多推荐？（5）190718


All Image Source

B/ 合理的文章结构，巧用疑问和设问句。在文章的一开始设立疑问并于文章的四分之三处逐步揭晓谜底，一步步地引导读者去读完整篇文章/

C/ 运用简单易懂又不粗俗的语言，让广泛的读者愿意打开你的文章去阅读。

D/ 巧妙的运用配图，使得文章的篇幅显得不那么短内容更为丰富。不过需要注意的是配图必须符合或者靠近文章的主题，再者就是要注意配图的版权问题。


All Image Source

E/ 文章的篇幅不宜过长，推荐控制在5000字以内。文章并不是越长就越好的，如果文章在不是废话连篇的情况下确实超过了上限字数，反倒是拆分开发布会比较好。当然了，我的编辑也曾经提过说不推荐大家拆分上中下来发布文章，这会让读者有一种阅读体验不连贯的感觉。对于平台本身而言，也不方便他们做集中推荐。

F/ 减少与文章主题不相关的内容。有些作者在写文章的时候套用固定的模板，某些程度来说这样也无可厚非。在非传统平台写作时，作者为了凑够字数而将模板设置成段落多、篇幅长还配上了图片，这点在传统平台写作时是切记不可使用的。

【Vicky's 影评】《失恋三十三天》：时间是治愈一切的良药-8


All Image Source


任何一场分手都没有赢家


《失恋三十三天》是我看过次数最多的一部电影之一，另外两部是《他其实没那么喜欢你 He just not that into you》和《单身指南》。也许是因为感同身受，每次心情不好的时候去看一眼总能让自己感觉被开解抚慰了许多。


前年的时候，我谈过一场轰轰烈烈的恋爱也在分手时弄的两败俱伤。曾经如此了解彼此的两个人在争斗的时候却刀刀直戳要害，痛得我昏天暗地。


但你知道吗，天秤座真的太要面子了。白天我还是那个牙尖嘴利嘴上半点不饶人的斗士，到了晚上关起门来继续生不如死；面对对方的挑事儿，分分钟骂人不带脏字儿地怼得对方恨不得没来到过这个世上，却在挂断电话的那一刻冲到厕所号啕大哭。


除了王小贱在婚礼上替黄小仙出头怒怼陆然那场戏以外，我最喜欢的是小仙儿追着陆然狂奔的那一场。


在黄小仙突然意识到她在这段爱情中做错了什么，奋起追着陆然的计程车狂奔的时候却被王小贱摔了一巴掌拦下。这时候旁白中传来黄小仙的独白：「世上最肮脏的，莫过于自尊心。此刻我突然意识到，即便肮脏，余下的一生我也需要这自尊心的如影相随。」


是啊，在一段感情里只要两人实实在在地爱上了对方，就弥足珍贵。即便是到了结尾时，两人实实在在地恨上了对方，也不用去计较谁输谁赢。势均力敌的爱情就是在爱的时候愿意为对方去披荆斩棘，在分开老死不相往来永不相欠。


而如何做一个优秀的前任，则是我们下次要讨论的话题了。
