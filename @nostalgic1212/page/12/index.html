「Vicky’s 听剧笔记」《默读》：在黑暗的尽头，是你，那很好

简介：童年，成长经历，家庭背景，社会关系，创伤。

我们不断追溯与求索 犯罪者的动机，探寻其中最幽微的喜怒哀乐，不是为了设身处地地同情、乃至于原谅他们，不是为了给罪行以开脱的理由，不是为了跪服于所谓人性的复杂，不是为了反思社会矛盾，更不是为了把自己也异化成怪物。

我们只是在给自己、给仍然对这个世界抱有期望的人—寻找一个公正的交代。

原著：Priest

广播剧出品：729声工厂、声音气球

关键字：长篇/刑侦/犯罪/纯爱

主役：刘琮 x 杨天翔

短评：《默读》剧组交了一份近乎完美的答卷，给《默读》这本小说涂上了浓墨重彩的一笔。

「声优」刘琮略带一点烟枪的嗓音，可以说配骆闻舟刚刚好，所以我不太明白为何之前超话里有人喷他的配音。

骆队是个有案情就时常昼夜加班，三十好几的大烟枪。这里你不能指望他嗓音呈现出霸道总裁式的磁性，我知道大家喜欢听，但骆闻舟是个苦逼的警察。

刘琮的表演非常细致，对费渡、对陶然、对郎乔、对罪犯…对不同的人他总能做出不同微小的调整，让听众在不经意间就在脑海中做到了场景的转换。

同理，我觉得没有人比杨天翔老师更适合费渡这个角色了。

「师兄，我会爱你爱到不可自拔的。」

只要是听过《默读》广播剧的，相信都能立马脑补出这句话的有声版，足见小阿天的功力。

SE0107天幕那里，费渡带入部分自己经历劝说被害人家属不要跳楼。小阿天深情并茂地演绎在bgm的烘托下，简直太好哭了。

骆一锅配音好评！！一开始我以为真的是收录了猫咪的叫声，看了参演CV表，才发现居然是阿杰！！！

「后期」故事的背景是国内北方某座城市，后期们日常用的音效很多都来自国外音效库。为了音效更贴合实际情况，后期们死扣细节，据说很多都是专门去录制的。

「音乐」天幕那场戏的音乐是量身定做的，太好哭了！！

「整体感觉」《默读》原著在我心里一直占据神一样的位置，这是一个“即便两个主角不谈恋爱，光是走主线剧情都足够惊艳”的故事。

感谢729把读者们的白月光呈现出来，期待第五季！！！

【Vicky's 影评】随着中国综合国力的不断提高，学习汉语成了一种潮流趋势

All Image Source

汉字在朝鲜半岛的地位逐渐被韩文取代，大致是在甲午战争前后。

当时的中国国力大不如前，结束了在东亚文化圈璀璨了几千年的历史。日本、越南、新加坡等诸国开始放弃对汉文化的推崇继而发展本国文化，朝鲜自然也顺应历史的潮流，国民意识也逐步确立。

汉字和韩文是截然不同的语言体系，后者和日语一样是拼音/注音文字，这就使得其在具体生活应用场景中遇到很多麻烦。

很长的一段时间内，韩文和汉字都是混着在用（有点类似于日语），但汉字基本被限定在括号注解的部分并且字数上也有一定限制。

这样特殊的历史背景，造成了在游览朝鲜/韩国历史名迹或者观看历史剧的时候，中国人能看得懂，但是朝鲜半岛的居民们反而需要看翻译才能知晓这其中意义的现状。

虽然韩国政府对于汉字的态度摇摆不定，但随着中国综合国力的不断提高，不少韩国家长会选择课外补习班让孩子学习汉语以增加核心竞争力。

「Vicky's 短评」《颤抖的真相》：这部引发身心极度不适的动画片，你看懂了没？

All Image Source

真相是什么？什么样的真相又能让人感到颤抖？

抱着这个疑惑，我打开了这部《颤抖的真相》，没想到一看就停不下来了。

《颤抖的真相》是由著名18禁动画《南方公园》的导演维农·查特曼/卡特·索伦执导，后者是美国最长寿的动画片之一。

里面的人物造型不算特别精致（近似于简笔画），但胜在人物性格鲜明。虽然是一部动画，《南方公园》时常会分析时政趋势、经济现状、社会问题等等。除此以外，其放飞自我的台词也是一大亮点。

短片延续了《南方公园》的风格，人物大多以粘土人的形象出现。一个半小时左右的短片中，涵盖了三十几个小故事。

动不动就爆血浆、虫子断肢满天飞、随处可以见的性暗示。脑洞更是大的可以，伤口会说话还会耍泼要吃辣椒酱，R头会发射子弹射死警察，死去的警察又被伤口吃掉…

其中，我最喜欢的是以下两个故事：擅长“躲猫猫”的孩子、“脑袋有洞”的社畜

这部动画短片种有数不尽的隐喻、暗喻，在色情、血腥、暴力的包裹之下是对当今社会现状的诸多叩问和反思。

如果你看厌了批量生产的动画片，如果你对现实、真相、超现实等话题感情绪，那么冰冰觉得你们可以尝试着挑战一下这部《颤抖的真相》。

友情提示，不要在吃饭的时候观看:D

【Vicky's 影评】李氏朝鲜时代，是否认识汉字成为知识分子与普通人的界线

All Image Source

当时的朝鲜尚处于只有语言没有文字的时期，在与较为发达的汉文化接触的时候，自然而然地选择借用汉字来表意。

这一举动，不仅仅存在于朝鲜，东亚文化圈内很多民族（日本、越南）都是这样做的。在长达一两千年的时间里，朝鲜半岛上只有汉字，并且汉字是其唯一的书写系统。

李氏朝鲜时代，是否认识汉字成为知识分子与普通人的界线。

朝鲜世宗李祹是李氏朝鲜第四代君主，在位期间创造了「谚文」，对朝鲜之后的语言和文化发展带来深远影响，被后世的韩国史学家尊称为世宗大王。

但在谚文被创造出来的四百年内，朝鲜的主要官方用语依然是汉字，谚文仅仅流行于市井百姓之中。
