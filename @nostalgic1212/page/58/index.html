【Vicky's 美妆】悦诗风吟雪耳保湿修护精华霜 190729

前段时间和妈妈逛街，路过悦诗风吟的门店。不得不说他们家的门店设计清新简约，配色和灯光搭配的刚刚好，很让人有走进去看一看的想法。就光是这点，就赢在了起跑线上，毕竟顾客愿意走进去看是购买的第一步。

想起来目前有的面霜还是冬天款的，大夏天的继续用很可能闷痘，于是我就给了自己走进店里去看一看的理由。

我原本打算买绿茶系列，之前用过这款的精华乳，其轻薄的质地给我留下了深刻的印象。导购小姐根据我皮肤的状况，推荐我使用雪耳保湿修护精华霜。最近太阳毒辣，我的皮肤已经到了敷保湿面膜都会有刺痛感的程度…于是欣然接受了她的建议。

这款主打修复的系列主要分为三个部分：日常护理（沁润面霜、沁肤凝露）、“修修护力”（多效修护精华液、多效修护膏）和集中护理（夜间维护面膜、密集修护面膜）。我只买了沁润面霜，总体来说完全可以满足我的需求。

至于效果嘛，还要等空瓶的时候再做评价啦:D

【Vicky's 碎碎念】在简书平台第一次变现（下） 190728

要提现首先需要缴纳「诚信押金」，我觉得相当于就是保证金的意思。如果你准备交易100个简书贝，那就需要充值5个进入（总交易金额的5%）。想要卖出手上的简书贝有两种方法，一种是在现有挂单收贝的那家那里提交卖出申请，另一种是在市场中直接发布卖出信息。这点和Gate/Okcoin这些交易所的理论是差不多的，所以我不算事太陌生。

我首先选择了前者，因为比较方便（终极懒癌患者）。目前最高的价格是0.23/贝，选择了最少需要出售100贝的卖家。填写好想要卖出的金额，核对信息以后点击确认，这就算是需求提交成功了。接下来，就是等待对方打款了。

根据我以往的经验，买家会通过银行卡、支付宝或微信转账，Gate和Okcoin上也允许买卖双方即时通讯。简书因为有简信这个站内信的模式，因此沟通起来也是比较顺畅的。我提交申请半小时后过去，对方依然没有转账的动作，于是我取消订单打算尝试自己挂单。

如同很多交易所那样，当日取消订单的次数是有限制的，大家在挂单的时候务必慎重考虑。

在市场挂单以后，很快就有买家通过站内信联系我了。查询银行卡信息，发现100简书贝对应的23元人民币已经到账了。这是我第一笔通过简书赚到的钱，相比较过去的到1元打赏（不累计到一定数量还不能提现）就欣喜不已的日子，现在的幸福感简直就是爆棚。随着年龄的增长，我们获得幸福感的G点越来越高，有时候还真的挺怀念小时候的。

买家在简信中留言，说希望再收下600个简书贝。考虑到直接转账会有风险，我选择在市场中直接挂单并通知他尽快去购买。于是乎，这天我一共在简书这个平台上变现了161元。这笔钱就是我的手机基金啦，看看我多久可以通过写作把手机的分期给还清呢:D

【Vicky's 碎碎念】关于情侣间匹配度的沉思 190728


这几天和朋友聊天，听对方吐槽自己家的二狗子各种囧事。我也不知道为啥总是充当身边人“情感热线”的角色，光是同事的相亲史就够我水一个月的文章了…在之前关于「结不结婚」和「是否生孩子」这两点的讨论以后，我们最近又多了一个新的topic：情侣间匹配度

根据她的说法就是，她爱干净但是狗子特别讨厌收拾房间。打个比方来说，她喜欢东西从哪里拿就放在哪里，喜欢起床就把被子折好，喜欢吃完饭随手站起来把碗筷洗好、擦干及摆放整齐，喜欢洗完澡随手把衣服洗好，衣服晒干以后就折叠起来分门别类的放好…

她家狗子呢？用完东西永远在“异地”，起床从来不折被子还美其名曰“晚上回家不还要摊被子睡觉”（听到这里我忍不住打断差个嘴：反正也要如厕，干嘛多此一举还要费力进食呢？），吃完饭碗筷桌子上一丢（放进厨房水池已经是优质表现）以至于这些餐具发出恶臭，不爱整理房间和洗衣服、晒干的衣服不愿意整理收拾干净…

这还仅仅只是卫生方面的分歧就有那么多，关于为人处事、金钱、友情等方面的想法两人的差别更加大。她说家里的狗子想要结婚想要一个家，这类似于求婚的话语却着实吓得她落荒而逃。对于女孩子来说，莫欺少年穷。如果觉得对方是潜力股，很多女孩子其实愿意陪着对方走一生的。但如果是匹配度如此之低的话，确实婚后幸福的可能性很低。

照理来说，我们这里有句老话“劝和不劝离”，意思是听到这种抱怨大多是安抚而不是挑唆对方分开。对于她的这个情况，我还是建议她三思而行。

【Vicky's 影评】《如懿传》：初看怒其不争，看完后才怅然若失，细水长流才是真正的好剧（中）


All Image Source


关于《延禧攻略》和《如懿传》的对比，我始终觉得是一场闹剧。很多人喜欢《延禧攻略》，因为魏璎珞敢说敢做敢于向高于自己阶级的人说不，我想很多人看到她就想起了现实生活中不敢反抗的自己。但我们仔细坐下来想想看，生活工作中我们如果对上司对老板这么怼天怼地，会是怎么样的结果呢？


魏璎珞这个性格和行事的方式方法，放在皇宫里根本活不过一集。我身边看过这部电视剧的朋友说戏称，仿佛开了挂一样的魏璎珞自带主角光环。爽则爽以，但在现实中基本是不可能发生的。


「细水长流的剧情」


《如懿传》在剧情的走势上并不属于标准的爽剧，没有像当下别的剧集那样一两集就设置小高潮看大女主吊打“恶势力”，因此刚开播就被吐槽节奏慢地让观众很难进入状况。如何将一部正剧拍的让人不觉得沉闷，这就非常考验导演的把控能力。就我个人而言，


虽不能说《如懿传》剧情全部都能抓住观众的眼光（容妃那段我也看的走神），但网路上大量吐槽的原因并不出在剧集本身上。
究其原因，不过是现在的观众已经习惯了阅读微博140字的文章和观看抖音15秒自成一个梗的小视频。如此浮躁的文化环境之下，


很少有人愿意看花五集甚至十集剧情去铺设一个人物线的走向。通篇看下来，我觉得《如懿传》的剧情几乎没有与主线无关的镜头。


「值得推敲的台词」


也有人吐槽《如懿传》台词过于直白，但如果真用文言文来对台词，我想逼疯演员不说还得逼疯一大票观众吧。说到台词，我想和大家分享一个让我印象深刻的例子。在先帝先帝驾崩诸多丧仪之后，太后在她的寝殿中折着纸锭。她的贴身大宫女福迦见此不禁问道：“太后怎么还做这些，先帝的丧仪将毕，不用再烧这些纸锭了。”


太后太起眼瞧了一眼福迦以后，复又低头继续折着纸锭说道：“先帝灵前那些都是哀家亲手做的，如今这手闲不住，你说这是烧给景仁宫好呢，还是烧给她侄女？”


这句话的潜台词其实是“你说哀家是应该让孝敬宪皇后先死，还是让她的亲侄女乌拉那拉·青樱先死”，但剧集里面的那句话显然更具有让人去深思和推敲的余地。

【Vicky's 碎碎念】在简书平台第一次变现（上） 190727


All Image Source

简书作为国内比较大的全内容平台，已经运行了很长一段时间。在Steemit平台写文章之前，我主要经营的有个人博客和简书。简书有自己的一套生态体系和规则，使其得以在日益严格的国内内容审查制度之下依然可以正常运行。

让我最感到不可思议的是，上头很多专栏的管理者和编辑都是用户自发组织的（没有薪酬），不免和其他平台开口闭口就问好处形成鲜明对比。


All Image Source

去年下半年的时候，简书正式向区块链内容平台转变，提出了简书钻和简书贝的说法。一时间，过去写文章只看重阅读量和点赞回复量的简书作者们心态也开始改变。有利益的地方就有江湖，各路羊毛党们也涌入简书平台并把握了头部收益。

我在四个月前回归Steemit的同时，也重新开始打理简书、豆瓣、微博等诸多平台。做法无它，就是把过去发布的低质量、凌乱的文章全部删除，从头再来一遍。经过140多天的运营（日更），前几天发现我的总资产已经有1746.84个了，约等于尽400块大洋。刚好最近买了手机月供需要297月，这提现出来第一个月的钱不就有了嘛。


All Image Source

在提现之前需要做一下简单的解释，简书钻相当于是steem power，而简书贝相当于是steem/sbd。其实这个平台还有另一种货币，我没有用到就没有去做深入的了解。从简书钻变成简书贝需要13周，这和Steemit上的power down如出一则。其实之前简书推出白皮书的时候，朋友就说这套运行模式堪称翻版的Steeemit。

之前没有动提现的脑筋还有一个原因，就是变成简书贝以后还要变成另一种token，再拿去交易所交易。我是极其懒于去学习和注册新的交易所（至今还不会steem变现）的那类人，所幸的是现在简书有了内部交易市场，对于用户变现来说实在是个绝大的好消息了。

OK，那今天我就来试试看如何变现吧:D

「Travel & Food」两荤两素的完美搭配 190726


这几天换了新手机，顺便想要整理手机里上千张的照片。就像很多Steemit用户一样，我也会选择将图片整理好以后发布上来，此后就将照片从手机内存中delete掉勒。今天想推荐给大家四道菜，它们分别是：「土豆煲」、「凉拌秋葵」、「蒜香鸡块」和「香辣大虾」。

秋葵是非常富有营养的一种蔬菜，我在楼下小院子里辟了一小块菜园子刚有可以收成，所以最近吃的比较多。当然了，秋葵并没有那啥的作用，因此男性朋友们没必要逼着自己吃…土豆是我很喜欢的一种蔬菜，热量不算很高饱腹感还很强，只要不和蛋白质混吃的话个人觉得拿来减肥也挺不错的。

大虾的处理过程比较心酸…活虾的口感比较好，但是要活生生的挑虾线确实我很不愿意做的事情。于是我找了个折中的办法，就是将大虾稍稍过水以后再进行处理。可以做成油焖和香辣，个人觉得味道都超级好。鸡肉是我减肥的时候吃得最多的食物，除了水煮鸡胸肉以外我觉得任何做法都超赞的！

btw…这大概是我的厨艺巅峰时期了吧…
