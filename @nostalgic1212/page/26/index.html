「Vicky's 影评」《我告诉你女孩子一生都要害怕些什么》：重男轻女的旧思想，居然还未被打破

All Image Source

「重男轻女的旧思想，居然还未被打破」

我想，我们应该不是第一次在媒体上见到重男轻女这个话题了。

在很多地方，女婴连睁开眼看这个世界的机会都没有。即便是来到这个世上，顺顺当当长大成年依然要历经磨难。

某幼儿园事件过去已久，如果我没记错的话，最后的调查结果以硬盘损坏监控录像无法查证告终

「我有一个长长的望远镜，一直能伸到你的家里面。」

这句话，已然成为我这个成年人心中的阴影。我很难想象，当幼小的孩子们听到，内心将会产生多大的恐惧。

幼童是一个家的弱势群体，而有些女童惨遭家暴成为一家人的出气筒，着实让人心痛不已。

有些时候，明明是一家人，哥哥弟弟可以有单独的卧室，而女孩却为了自己能有一张睡觉的床而“感恩”不已。

在一些生存条件不那么宽裕的地区，女孩子能得到的资源原本就不多。有些人，却顶着女童的名义向社会讨要资助，转身就给了男童们。

我不是要仇视男孩，但是这样的（sao）操作，真的公平吗？

推荐影片：《狗十三》、《生门》、《赤月青日》

“你可以教孩子防备陌生人，提高警惕，但是不能让她怕穿碎花裙子，不然要我们干什么用的？

by Priest 《默读》

1月3日晚，ID为@窗帘它挡不住寒气的微博用户，发布了一则名为《我告诉你女孩子一生都要害怕些什么》的视频短片。

制作团队们第一时间进行转发，之后粉丝和路人们纷纷加入到转发和讨论的队伍中去。

截止到目前为止，该条微博转发量达到了24万次，用户留言点赞量超过40万。

随着视频的扩散，这个数字还在继续上升中。

遗憾的是，歌曲在网易云音乐和B站上，在上传后2小时内就下架了。目前视频在微博上还存留，网友们纷纷表示已缓存该短片。

我在写下这些文字的时候，其实并不确定读者们是否能看到这篇文章…

「能力摆在那里，何必拿长相说事」

在很多自媒体和主流媒体上，我们时不时会看到「美女外交官」等博人眼球的字眼。似乎人家勤学苦读了N年，倒是比不上这一身好皮囊。

在男女平等大旗都高举了N年的2020年，性别在职场中的区别待遇依然显而易见。

不信，去看看公司/单位管理层的男女性别比例吧。

女性在职场中，即使爬到了高位，也会被人指指点点称其是“睡”上去了。即便做出了什么成绩，也容易被一笔带过甚至功劳归于男性同事。

当女性工作上出了差错，等待她们的大多是“女人就是女人”、“就说事情不能交给女人去做”、“女人就是头发长见识短”、“女人就该在家带孩子，出来工作凑什么热闹”。

…

呵呵，直男癌们！

All Image Source

除此以外，剧中还有一个场景让我印象深刻。

大约在第九集的时候，被逼到零界点的深海在办公室里对老板说出了自己一直以来对诸多不满，希望对方可以有所改变。

老板并没有反省自己的错误，而是大着嗓门儿地说：

“如果你有那么多不满，那就辞职不干啊，这公司并不是没有你就运行不下去的！”

之前网路上有个很火的段子，作为领导不要惹怒刚刚工作的年轻人，因为他们随心所欲说辞职就辞职。

但作为领导却可以任意压榨那些中青年员工，因为他们大多有家要养要房贷要供。压在他们身上的重担，逼得他们不得不按耐住对生活对工作的不满，硬着头皮接着做下去。

无论在生活还是工作中，我们都需要有足够的能力和责任感。责任感和能力越高，自然要承担的事情就会变得更多。

然而如果片中的女主角那样有过多的责任感的话，只会让身边的人过度依赖而变成废柴。

工作就只是工作，就算公司少了某个员工依然可以找到新的替代人一样，某份工作也不会是一个人生命的全部。

反抗起来吧，野兽们！

《最美表演》作为微博的特别企划活动，受到了广大网友们的关注。

活动先是让网友们通过投票，选出他们心中10位优秀的演员，再由项目组联系入选人与金牌导演合作。据悉，最终呈现出来的10部影片大多会颠覆演员原本的荧幕形象，粉丝们均表示出十成十的期待。

【Vicky's 影评】《我们无法成为野兽》：工作中遇到太依赖自己的老板和同事，未必是件好事情

All Image Source

我之所以喜欢《我们无法成为野兽》，是因为它不去引导情绪的出口，主角中无论是深海还是恒星都不似别的剧那样自带主角光环。

相比较国产职场剧中很多主角如有神助的境遇，这部剧的主角们，就这样在现实社会的泥泞中摔爬滚打一身伤痕。

《我们无法成为野兽》没有用戏剧的手法，去粉饰太平告诉苦逼的上班族们「没事没事，一切都会过去的都会变得更好」，也没有用四海八荒的爱情去逃避我们现实生活中的碌碌无为。

它将我们会遇到的职场、爱情、亲情上的问题，如此赤裸裸的摆在我们面前，剧中很多场景的细节真实又扎心。

这部剧也让我回想起自己初入职场的情景，那时候的我也同深海那样是个有求必应的人，因为我觉得那样可以尽快融入这个团体也可以学到更多技能。

工作三五年后，我成了同事和朋友口中靠得住的同事和可以依赖的朋友。

这样的结果就是他们一遇到麻烦就想起来联系我，因为知道只要我接手了这个事情就约等于做成了。

这是一种什么样的体验呢？

就是老板娘要给老板生日惊喜，从想法到具体实施都是我；公司的总经理助理英文不好，只要是有外宾来临时被抽调去做总助也是分分钟会发生的事；老板家的户口本、身份证、护照、车钥匙、家门密码等等都在我这里…

有些人会说这是boss对你的信任，你应该要觉得开心，但事实上并不是。

周末和家人聚餐被老板一个电话call回去布置会场，结束以后又临时被告知外宾的介绍材料还是英文版需要翻译成中文，结果一个人在鬼都没有半个的办公室里通宵翻译。

因为被调配去支持别的岗位同事的工作，结果对方下班了，我还得加班去补上自己工作中落后的进度。

我只是个行政人事经理，却做着行政、人事、总经理助理、财务助理、销售助理、营业助理等等工作，当我看到深海这个角色深切的感受到了她内心的愤懑和无奈。

就像深海在片尾丢下了那颗“炸弹”，我也辞职了，活成为那个敢于对生活对周围的人说「不」的小野兽。

（The End）
