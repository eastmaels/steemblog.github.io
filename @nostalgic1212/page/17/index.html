【Vicky's 影评】《王国》：当年的捉虎营，是否暗藏玄机？

All Image Source
文/冰冰

「安炫（许峻豪 饰）」

安炫是世子李苍的老师，在世子年幼时悉心教导其成为一个合格的君王。

如此拥护世子成为新王的安炫大人，几年前因为征战而出宫，之后由于某些原因留在了当地，根据现有的剧情我有这样一些大胆推测。

三年前日本倭寇进犯尚州，安炫大人用五百名丧尸击败了三万倭寇并最终赢得了战争的胜利。

只是我现在还不确定的是，他是将当地村民做成丧尸并以此为武器还是拿当地的村民去供养丧尸。

但不管是哪一种，他都对当地的村民十分愧疚。

「永信（金成圭 饰）」

永信是捉虎营的一员干将，他的村子应该就在当年倭寇进犯之地不远处。

从闪回的场景来看，村里的小孩对他说自己经常会感觉到饿，这点让我怀疑村里的人是不是被培养成了丧尸。

这一季的后面几集出现过他回到村子的场景，从一座座坟冢可以看出这个村子里面没有人了。

这些坟冢看起来就是有人细心打理过的，并且永信去的时候墓群前面的香炉一直供着香火，这足以见得有人在满怀敬意地妥善料理这些人的后事。

以此，我们再联想一下安炫作为朝廷要员在战后一直没有班师回朝，而是在尚州原地守孝三年。

守的是何人的孝？

在我看来，他是为了当地的村民。

再者，我觉得永信会出现在李承熙开设的医馆也绝非偶然。

他村子的亲朋好友都可能与丧尸有关，而刚巧李承熙就是懂得如何让死人复活的医者。要说他不是为了探求真相而潜伏在医馆，我是怎么都不会相信的。

最后，大家不要忘记一点，当时让医馆其他病人吃下丹儿尸体的正是永信。

为什么同样饥饿状态下的他自己没有吃呢？别说是因为想省下来给别的病友吃，这么高风亮节的人设是女主徐舒菲。

（The End）

【Vicky's 影评】《王国》：与父亲二心的中殿娘娘，或将在第二季中放大招

All Image Source
文/冰冰

「中殿娘娘（金惠濬 饰）」

中殿娘娘的一张娃娃脸，和她剧中不输于其父亲的狠角色设定形成了强烈对比。

只是，随着剧集情节的推进，观众可以感受到她和父亲很可能不是完全同一条心。

剧情中有一个细节，不知道大家是否有关注到，她去大王寝殿“投食”，结果雪白的袜子上出了一抹猩红。

一开始我以为这是她沾到了大王咬死食物渗出来的血，但下一个场景就是她身边亲信的老宫女拿着她换下来的衣物到了焚烧炉并屏退了左右。

这个细节，让人不禁有一个疑问：

中殿娘娘莫非是来了月事？

中殿娘娘懂得如何拿捏人类的恐惧。在听到宫女们小声私底下讨论看到自己有带血的衣物拿去焚烧以后，当晚她不着痕迹地让宫女来寝殿侍奉自己沐浴。

她让宫女亲自一层层脱掉自己身上的衣物，让她看到自己用来伪装肚子的包袱和自己窈窕的腰身。

原来，中殿娘娘真的是假怀孕，并且这一点作为父亲的赵学洙并不知情。

身为国丈的赵学洙根本不关心女儿是否真的怀孕了，他想要的只是有个可以让自己进一步把持朝政的傀儡或者改朝换代的契机，而真正的掌权者是他自己。

与其不同心的是，作为女儿的中殿娘娘想要的则是垂帘听政，她才是那个手握实权的人。

在我看来，中殿娘娘是个极其矛盾的角色。

一方面她可以眼睛眨都不眨地每日为大王挑选少女“喂食”，将孕妇圈养起来，如果生的不是男孩就连孩子带母亲的统统杀死。

一方面当父亲带着拥护世子上位的老学究们去见大王时，又唯恐他们被大王咬死而动了恻隐之心。

（The End）

【Vicky's 影评】《王国》：仁心仁德的医女，是否是阻止这一切的关键？

All Image Source
文/冰冰

「徐舒菲（裴斗娜 饰）」

徐舒菲是位仁心仁德的医女，因为想把肉汤省下来给病者食用，才幸免于难没有变成丧尸。

她一直在找生长于「冻土之地」的汉阳草，在和世子相遇以后随着他们一起从东莱到了尚州。

剧中有一幕非常令人啼笑皆非，我想看过该剧的观众们一定还记得。

在给东莱府使把脉后，舒医女非常直接地答道“您是不是*茎时有疼痛”，并直言不讳地说他可能患了淋病之类性病。

这位东莱府使本因为徐舒菲救了自己而对她心生爱慕，但这一下子弄得他非常不好意思。

在礼法森严的李氏朝鲜，女人说出像现代女医生会说出的话简直是让人叹为观止。但是这话从一个医女嘴里说出来，到没有那么奇怪了。

这倒不是因为她是医女，而是李氏朝鲜时期医女的另一个职能——“官妓”。

本剧故事发生的背景差不多是17世纪，大约在此前百来年的十六世纪，李氏朝鲜出过一位暴君，将当时的医女制度给完全扭曲了。

在他执政时期，要求在职医女必须向妓女学习“侍奉之道”，当时的医女很多沦为了权贵阶层利益交换或者发泄性欲的工具。这其中，强占医女作为小妾偏房也是司空见惯的事儿。

因此，我们这位蠢萌的东莱府使大人，想以“娶舒医女回家做小妾”来报救命之恩这样的想法，也就让人觉得理所应当了吧。

（The End）

【Vicky's 影评】《王国》：“我和他们不一样”，这是一个野心王子改变的开始

All Image Source
文/冰冰

「 李苍（朱智勋 饰）」

世子李苍并不是任人揉捏的软弱之人，相反的，他开始是以一个准备叛变的野心家形象出现的。

无论是在中国古代还是在朝鲜古代，嫡出和庶出的区别是很大的。

如果在民间大户人家，庶出的子女可以说完全是奴隶奴婢的存在。在皇家吃穿用度勉强过得去，想要登基坐上皇位基本没可能。

李苍能做上世子，完全得益于元王后膝下无子，而他庶出哥哥脾气暴躁乖戾。如果中殿娘娘未能生下儿子且大王可以寿终就寝，那么他继承王位的可能性还是很大的。

但当时的局势是，他连继续活下去的机会渺茫。

因此，在儒生拥护之下，他决心策划一场政变。

随着逃亡的不断进行，李苍见到百姓因为饥饿去“易子而食”，这是从小在深宫中长大的他不曾见到过的。

渐渐的，他逐渐做出了「单纯想要活下去」向「具有君王之风」人设上的转变。

剧中有一幕，东莱的权贵阶层为了躲避丧尸来袭不惜丢下百姓，乘坐仅剩的大船仓皇逃离。

李苍带着百姓们准备去山腰处的医馆躲避，途中运送老人和孩子的板车被卡在石缝中动弹不得。

随着天色不断暗下来，山上的丧尸们逐渐苏醒过来。随侍武英劝李苍放弃板车和那些百姓赶紧向前行进，争取在日落前躲进医馆。

然而，李苍紧紧推着板车的手丝毫没有放开，并大声呵斥武英说道：

「我与那些抛弃了他们的人不一样！」

《王国》中几场李苍心理和人设转变的戏都拍的很好，导演和编剧把这点交代的明明白白，这也是国内很多影视剧所不具备的。

（The End）

【Vicky's 影评】《王国》：某些程度来说，真正吃人的并不是丧尸，而是权臣

All Image Source
文/冰冰

一部好的影视剧除了紧张刺激、逻辑清晰的剧情以外，还需要有定位明确的人物角色。

坏人就要坏得令人憎恨，善人就要善得心中没有一丝杂念，领导者自然也要有气势有能力…

《王国》（又名：李尸朝鲜）这部剧，显然做到了上述几点。

剧中的角色不算太多，除了奔跑中的大量丧尸群演以外，目前主要只有七个角色。

在第一季结束的时候，只有永信和安炫这两个角色背景还未完全明朗，但现有的剧情已经给了我们不少线索。

比如安炫大人三年前可以战胜倭寇，很可能是使用了丧尸；在徐舒菲所在的医馆养伤的永信，他之前所在的捉虎营应该在那场战役中发挥了不小的作用。

接下来，我们就逐一来看看这几个角色吧。

「赵学洙（柳承龙 饰）」

作为这部剧最大的反派，赵学洙可以说是权倾朝野，所向披靡。

女儿是李氏朝鲜王宫尊贵的皇后，她即将诞下龙子取代世子成为王位继承人。

身为国丈的他，自然可以进一步巩固在王朝中的地位，改朝换代也不是没可能的。

本来如同囊中取物一般容易的皇位，却因为王突然驾崩女儿还未诞下龙子而变得充满变数。为了可以保证可以挟天子令诸侯的野心，他不惜用禁术复活大王。

从这个设定上，《王国》的野心就远远超过了《釜山行》和《行尸走肉》。从权谋和剧情深度来说，我认为它更接近于《西部世界》和《权利的游戏》。

赵学洙在第一季第几集开场没多久的时候，就曾说过一句话：

「我绝不会原谅，妨碍我的所有人。」

面对为世子求情的官员，他直接“教育”对方到在座别的同僚都面露恐惧，再不敢多说一句为止；

想要女儿能继续诞下龙子为他所用，他不惜召回已经退居山林的太医李承熙，启用禁术复活大王；

为了铲除世子这个绊脚石，他私自让儿子带着追兵准备杀死李苍，甚至不惜以无辜百姓的性命为代价；

……

某些程度来说，真正吃人的并不是丧尸，而是这个身居高位的赵相。

（The End）

「Vicky's 短评」荧幕上被削发的女性们，是羞辱，不是荣耀

All Image Source

昨日，我被微博上一组非常魔幻主义的视频震碎了三观，不免想到了一些荧幕上被削发的女性群像。

《圣女贞德蒙难记》、《悲惨世界》、《少年的你》、《午夜守门人》、《V字仇杀队》、《西西里的美丽传说》、《驴得水》、《权利的游戏》、《女子监狱》、《水》、《巴黎烟云》…

有些人也许会说，女性短发也很美啊！

但这件事，根源并不在发型本身，而是这些女性被强行剥夺了建立在个人意志上的审美情趣。

我能理解出于便利的考虑，长发确实不容易打理。但只需要剪成利落的短发即可，犯得着要像公开处刑一般地剃成光头吗？

同为女性，我太清楚头发对我们是何其珍贵的存在。视频中那二十几个女孩子在哭泣，而他们绝不是因为内心的光荣而表现如此的！

所以我想问，是一线的男性工作者们，也需要剃光头吗？

相关部门这么做，还公开视频（且不打马赛克）的用意，到底在哪里？

除了作秀，我想不到任何别的理由。

心疼那些女孩，冒着生命危险赶赴一线之前，还要受到如此的羞辱。

（The End）
