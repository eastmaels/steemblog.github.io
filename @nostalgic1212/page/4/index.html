「Vicky's 资讯」“剧抛脸”的肖战，终于在最美表演里，成了青年演员肖战

All Image Source

《最美表演》作为微博的特别企划活动，受到了广大网友们的关注。

活动先是让网友们通过投票，选出他们心中10位优秀的演员，再由项目组联系入选人与金牌导演合作。据悉，最终呈现出来的10部影片大多会颠覆演员原本的荧幕形象，粉丝们均表示出十成十的期待。

毫无悬念的，因为《陈情令》大火的肖战，也在入选人名单之中。

在第一期的短片中，肖战颠覆了过去乖巧形象。戴着鸭舌帽，露出狡黠顽劣的表情，像极了20出头游手好闲的小青年。

皮归皮，小青年还是相当孝顺的，他给父亲准备了一份大礼。从预告片来看，可能是一辆小轿车，不想却遭到了来自父亲的一顿扫帚…

昨日，@X玖少年团肖战DAYTOY 在《买耳朵》里的精彩瞬间也曝光了。

继皮皮战以后，小赞化身重庆外卖小哥，不知道大家对他的表现是否满意勒？

康康九宫格里烟火气满满的战战，你最pick哪一张？

冰冰最喜欢小赞吃着盒饭，还微笑着盯着手机的这张。对亲人的寥寥几语，让同样在外务工的人们感同身受红了眼眶。

不愧是“剧抛脸”肖战，演技越来越好啦！

「Vicky's 资讯」肖战杨紫头条盛典同框，“余生夫妇”正式营业

All Image Source

时至年末，各大娱乐盛典不仅让明星们出尽了风头，更是让各家粉丝们看花了眼。1月9日晚的2019头条盛典，也是人尖叫连连的夜晚。

当青年演员肖战与杨紫同框的时候，现场的气氛更是到达了顶点。男帅女美，分外养颜。

作为这几年人气爆棚的一线演员，肖战和杨紫都有非常不错的作品。

杨紫在《欢乐颂》、《香蜜沉沉烬如霜》及《亲爱的，热爱的》等作品中的精彩表现，使她完成了童星到一线女星的转变。众所周知，杨紫自带热搜前潜质，她主演的电视剧收视率极高，还有粉丝认证的“旺夫体质”。

素人出生的肖战，也凭借《陈情令》、《庆余年》及《诛仙》等作品，让大家更全方位地认识这位笑容灿烂还还分外认真、努力的青年演员。

最近他参加了《最美表演》的演出，无论是之前被老父亲拿着扫帚追着打的皮皮战，还是让人瞬间泪目的外卖小哥，我们都看到了肖战在演技方面的精进。

据悉，两人合作的《余生请多指教》将于2020年播出，拭目以待！

PS.肖战饰演的顾卫，实在是太太太帅气了！！！！

【Vicky's 影评】《我们无法成为野兽》：工作中遇到太依赖自己的老板和同事，未必是件好事情

All Image Source

我之所以喜欢《我们无法成为野兽》，是因为它不去引导情绪的出口，主角中无论是深海还是恒星都不似别的剧那样自带主角光环。

相比较国产职场剧中很多主角如有神助的境遇，这部剧的主角们，就这样在现实社会的泥泞中摔爬滚打一身伤痕。

《我们无法成为野兽》没有用戏剧的手法，去粉饰太平告诉苦逼的上班族们「没事没事，一切都会过去的都会变得更好」，也没有用四海八荒的爱情去逃避我们现实生活中的碌碌无为。

它将我们会遇到的职场、爱情、亲情上的问题，如此赤裸裸的摆在我们面前，剧中很多场景的细节真实又扎心。

这部剧也让我回想起自己初入职场的情景，那时候的我也同深海那样是个有求必应的人，因为我觉得那样可以尽快融入这个团体也可以学到更多技能。

工作三五年后，我成了同事和朋友口中靠得住的同事和可以依赖的朋友。

这样的结果就是他们一遇到麻烦就想起来联系我，因为知道只要我接手了这个事情就约等于做成了。

这是一种什么样的体验呢？

就是老板娘要给老板生日惊喜，从想法到具体实施都是我；公司的总经理助理英文不好，只要是有外宾来临时被抽调去做总助也是分分钟会发生的事；老板家的户口本、身份证、护照、车钥匙、家门密码等等都在我这里…

有些人会说这是boss对你的信任，你应该要觉得开心，但事实上并不是。

周末和家人聚餐被老板一个电话call回去布置会场，结束以后又临时被告知外宾的介绍材料还是英文版需要翻译成中文，结果一个人在鬼都没有半个的办公室里通宵翻译。

因为被调配去支持别的岗位同事的工作，结果对方下班了，我还得加班去补上自己工作中落后的进度。

我只是个行政人事经理，却做着行政、人事、总经理助理、财务助理、销售助理、营业助理等等工作，当我看到深海这个角色深切的感受到了她内心的愤懑和无奈。

就像深海在片尾丢下了那颗“炸弹”，我也辞职了，活成为那个敢于对生活对周围的人说「不」的小野兽。

（The End）

「Vicky's 短评」《我不是药神》：钱不是万能的，但有时候，没有钱却是万万不能的

[All Image Source](https://movie.douban.com/photos/photo/2528752253/

2018年上映的《我不是药神》，成为打破了国内现实向电影市场的空缺，并且取得了口碑票房双收的好成绩。

山争哥哥饰演的程勇靠倒卖印度神油为生，是个我们口中常说的油腻的中年loser。

他家暴打走了老婆，一周见一次的儿子还要跟着后爸出国。老父亲血管瘤急着做手术，但他的神油卖不出去，店铺因为交不出租金和水电费即将面临关闭。

机缘巧合之下，他结识了白血病患者吕受益，对方希望他能代购一种高价专利药瑞士格列宁。

也许有人会说，吃药什么要搞代购，直接去医院里配药就是了。

但你们知道，这个药在医院里是什么价格吗？

四万人民币一瓶！

这只是一个疗程也就是一个月的价格，吃不起的人只能等死。

而就在邻国印度，有一种名为“印度格列宁”的药，在当地药店售价折合人民币才2000元一瓶。药效基本无异，价格却只有1/20！

他们又陆陆续续找到了其他几位成员，在赚钱的同时也切切实实地帮助到了病友们。

然而东窗事发，药厂发现了这件事，派了代表来华与专案组们彻查这起案件。

我们不能说药厂和警方立案调查这件事是错的，毕竟研发一种特效药所需要投入大量的人力物力财力，还需要经过长时间的临床实验才可以投入大批量的生产。

电影将病友们和药厂错误地对立起来，显然是这部电影最大的败笔。

然而，当患病的老太太期期艾艾地央求警方不要再查下去了的那番话，真正是戳到了观众们的泪点。

“领导，我求求你了，别再查假药了行吗，这药假不假我们这些吃药的人还不知道吗？”

“我吃了3年正版药，房子吃没了，家也吃垮了。现在好不容易有了便宜药，可你们非说这是假药。”

“不吃药，我们就只能等死。我不想死，我想活着。”

生而为人，都不容易。

希望这天下的病人们都能吃得起用得上药，早日康复。

（The End）

「Vicky's Life」突如其来的年度盘点——书影音

2019年对我来说忙碌（拖延）异常，很多plan被无限推迟，主观的客观的原因其实都有。

本来准备去旅行的年假，也是硬生生拖到了12月才兑现。这全国都天寒地冻的季节，又不想去几乎每年都去避寒的东南亚，灵光一闪我决定去拔牙。于是乎，一个月内我对仅剩的两颗智齿磨刀霍霍了。

拔了智齿，抽了牙神经，做了根管治疗，年前把牙给补了。这一系列操作结束后，我寻思着干脆趁着不结婚去把大学时候就想带的牙套给办了。

（明明是写总结，不知不觉的发散性思维开始做计划，不愧是天秤座本秤了:D）

作为朋友口中的移动影音库，有点惭愧的说，其实我今年的书影音成绩并不算太出色。

一方面是开始系统性地在自媒体运营，写文的时间替代了看片看书的时间；另一方面是后期在听广播剧，刷片频率降低了不少。

根据豆瓣电影给出的年度报告，我今年看过28部电影，36部剧集，其中34部是我本来就标注想看的（兑现率很高嘛）。

我平均5.7天看完一部影视，热衷于惊悚片。就速度来说不算很慢（有些剧集很长），惊悚推理片是我的最爱。

2019年，我一共阅读了40本书，算起来有13364页，其中有16本是我曾经想读的。

我平均1.4周读完一本书，习惯在夜晚读书，偏爱小说类。以前喜欢用Kindle看书，后来发现一些有声阅读小说的app做的挺不错的，渐渐用听书取代了看书。

这一年，我在豆瓣创作了147篇文章，写下了242365个字。这个数字之多不少，回头我得去看看别的平台的统计数字。讲真，我都没想到自己居然那么能掰扯。

04月13日那天我熬夜了，估计熬的挺晚的，虽然我不记得那天我为啥失眠。

我不爱考古看老片子，即便是比较经典的作品。在朋友的推荐下，我看了1997年上映的电影《千钧一发》。说真的，电影挺不错的，推荐。

不追热点影视剧是我的标签，但我也没想到自己居然看了那么冷门的影视剧和书…分别只有832人看过和6人度过…

2019年，李东旭的作品我看的最多。仔细想了想，我看了《触及真心》和《他人即地狱》，还回味了一遍《鬼怪》。不得不说，李东旭真的是可塑性特别的演员，可甜可盐太可以了！

在已经到来的2020年，我打算更加精选自己看的书和剧。不盲目追热点，也不刻意回避优秀的热门话题。

新的一年，Fighting❤️

（本篇完）

【Vicky's 影评】《我们无法成为野兽》：唯有彻底做出改变，才能拯救失控的人生

All Image Source

说这句话的酒友，就是根元恒星（松田龙平 饰）。

他是一名33岁的精英会计师，擅长处世的他很受周围人的欢迎。比表面截然不同，恒星内心其实什么人都不相信，常常冷眼旁观身旁的一切人事。

与他保持身体关系的好友橘吴羽（菊地凛子 饰），突然宣布要和某游戏公司社长橘海寺（饭尾和树 饰）闪婚。

这样原本平静的恒星，心里掀起了不小的波澜。

吴羽在本片中虽然是一个配角，却在剧情上起到了很大的推动作用，在观众中也有很高的人气。

因为，她是本片中唯一从始至终都做「野兽」的那个人。

身为模特的她，不愿向压榨她的经济公司妥协，冒着被起诉的风险毅然辞职。经过不懈地努力，她成立了自己的服装品牌有了自己的门店。

她遵从本心，做想做的事情、和喜欢的男人交往、同那个会让她感到幸福的男人结婚，活出了我们所有人想要的样子。

这些人里，自然也包括了深海。

以吴羽为偶像的深海，在她的潜移默化下慢慢开始蜕变。

连带着恒星、京谷、朱里还有京谷的妈妈、深海的同事们，「像野兽一样去表达自己做生活」这个想法像是有如病毒一样在他们之中扩散着。

（The End）
