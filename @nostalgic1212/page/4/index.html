「Vicky's 资讯」听说可以提前一天看《夺冠》啦，看女排姑娘们在线还原当年经典赛事

All Image Source

根据最新出炉的贺岁档电影预售排名来看，《夺冠》（原名：《中国女排》）位列第四，仅次于《唐人街探案3》、《囧妈》、《姜子牙》。

在北京举行的全球首映礼上，导演陈可辛，监制张一白，领衔主演巩俐、黄渤、吴刚、彭昱畅、白浪，扮演80年代女排队员的演员们及特别出演刘敏涛一同出席。

《夺冠》由陈可辛导演，主要讲述了女排成员艰苦训练最终夺冠的故事。其展现出来的女排精神，让不少观众，尤其是上一辈观众们不禁落泪。

作为贺岁档最燃的电影，首批看片的观众更是给出超高评价：“年度十佳预定”、“会被时代记住的作品”、“一个字，就是看！”。

服化道组也非常用心，不仅极富年代感，连赞助商百岁山都与当年一模一样。

原本大年初一上映的《夺冠》已确定提档，大年三十就可以看到女排姑娘的精彩表现，喜大普奔！！

期待陈可辛导演的用心之作，期待女排姑娘们的荧幕首秀！

加油！！！

「Vicky's 资讯」《默读》动画版正式开启，声优成粉丝热议焦点

All Image Source

去年，坊间就有一份耽改名单，大量读者所熟知的原著小说将影视化。其中，就有备受关注的《默读》，其作者Priest的另一部作品《有匪》剧版将于2020年播出，主演是小花旦赵丽颖和当红小生王一博。

1月19日，《默读者》（动画版名称）在官微正式发布PV，一时引发了书迷们的争相讨论。根据一天观察下来，冰冰发现争论主要集中在两点：

声优和画风

《默读》广播剧在猫耳独家放送，目前有4季。其主役为骆闻舟（cv：刘琮）X费渡（cv：杨天翔），两人都来自729梦工厂。

在微博超话中，有某些情绪激动的用户正对两位CV发布了较为偏激的言论，甚至闹到了【默读广播剧】超话下。

讲道理，这难道不是声优祭的后遗症之一么？

骆闻舟相较于费渡，本身就年长很多。费总念初高中的时候，骆队已经在基层派出所工作一两年了。在主线故事中，费总是22岁，那么骆队起码是30-35岁。加上他大烟枪，市局刑侦科工作昼夜颠倒，个人认为是大叔音并不奇怪。

中国的声优们，还是在近几年才慢慢从幕后走到台前。即便是现在大名鼎鼎的路知行、边江、阿杰等声优，亦是如此。

作为听众们，我觉得大家应该给这个行业多一点鼓励和包容。激励更多有志成为声优的新人，让他们有更多出场锻炼的机会，这才是不断传承地上上之选。

看到这出闹剧，我突然怀疑，声优祭是否是个正确的选择。

（The End）

「Vicky's 短评」《我们无法成为野兽》：不想做社畜？但为了生活，我们不能不作出妥协

All Image Source

社畜，是时下年轻人中，非常流行的一个词汇。

这个词源于“会社”（相当于公司）与“家畜”的结合，意为“公司的牲畜”，是日本企业底层上班族的自嘲用语。

在这之后，慢慢演变成白领用来自嘲或嘲笑他人词条，形容那种“为了企业放弃身为人类的尊严，以及舍弃私人生活无条件为公司卖命的生活状态”。

以职场为背景的优质日剧有很多，比如《半泽直树》、《Legal High》、《逃避虽然可耻却有用》、《我，准点下班》，还有这部新垣结衣主演的《我们无法成为野兽》。

深海晶是一名离开家乡到大城市打拼的30岁OL，每天醒来迎接她的是来自老板轰炸式的工作讯息。同事非但不能搭把手，还经常捅了篓子让深海去擦屁股。

工作不顺心，包子性格的深海在感情生活中也是有苦说不出。自家男友家里，还住着个“家里蹲”的前女友。在男友母亲和同事面前，深海要思索着怎么说谎才可能营造出他俩同居在一起的假象。

这场面，要多心酸有多心酸。

这么苦哈哈的深海，下班后唯一的慰藉就是去小酒馆喝两杯。这不就遇到了耿（du）直（she）的男主根元恒星，而对方也正为了自己女友突然结婚，新郎却不是自己而黯然神伤。

《我们无法成为野兽》能在霓虹本土收视一度冲到了11.5%，我觉得很大程度上是大家在里面的角色身上看到了那个明明心有不甘，却为了能活下去而向周遭不断妥协的自己。

看里面的角色们不断地尝试去反抗，一次次铩羽而归，却又契而不舍地再次尝试的时候，是不是也给了丧气慢慢的社畜的我们，一丢丢向阳而生的动力了呢？

btw，女配橘吴羽这个角色实在是太拉好感了，超A！！！

「Vicky's 影评」《我告诉你女孩子一生都要害怕些什么》：最怕世人都认定，女孩本就懦弱

All Image Source

「最怕世人都认定，女孩本就懦弱」

近几年，反应不同阶层的女性在现今社会中的现状的影视作品越来越多，类似「Me Too」的运动让女性站出来对不公平对待说不。

我们不得不说这确实是历史长河中行之有效的一步，但我们也不能否认这条路上依然有很多绊脚石，比如说「女德班」。

我不敢相信，屡禁不止的女德班居然一再一再地卷土重来。

比起杨永信那种下手狠毒的强硬方法，女德班更擅长用洗脑使人就范，大道理一套一套，比如她们所宣扬的婚姻四项基本原则：

打不还手、骂不还口、逆来顺受、绝不离婚。

这套言论最出名的“女德教母”丁璇，仅有初中文凭的说出了“女人最好的嫁妆是贞操”这种恶心言论。

我想说，这个短片我看了五遍，几近哽咽。

这个短片并不是要仇男或是厌男，制作团队中也有大量男性工作人员。如果短片的初衷真的是建立性别对立，我想那些男性staff们也不会愿意接下这个工作吧。

这几年，世界各地都掀起了「ME TOO」运动。

且不说这些运动最后是否能拯救尚在水生火热之中的女性们，但这个运动本身就极具正面意义。

女孩们别怕，走到阳光下面来吧！

All Image Source

就在最近，鼠年春晚语言类节目进入终审环节。虽然网传的节目单还不能完全做数，不过从此次参加终审的参与人员中我们还是可以看出一二。

让我们看看，名单里面有哪些人员。

谢娜、关晓彤、肖战、魏大勋、张若昀、吴磊、袁姗姗、张天爱、杨迪、张凯丽、郭冬临、沈腾、贾玲、马丽、沙溢、黄晓明、聂远、秦岚、许魏洲、惠若琪、王鸥、王迅、潘斌龙等。

大家可以看出，今年的春晚可以说是兼顾了流量和实力两个方面。

不仅有因为《陈情令》中魏无羡一绝跻身顶级流量的肖战，主演热播剧《庆余年》的张若昀，金句连连的黄晓明，还有郭冬临、沈腾、贾玲、马丽、杨迪等喜剧能人。

不得不说，肖战在cql之后还保持了一定的热度。不仅参加了很多时尚活动，更是在《最美表演》中show了一把演技。最美表演这个企划，让观众，无论是粉丝还是路人看到了一个截然不同的肖战。

我想，无论是参加春晚还是成为北京电视台的春晚的主持人，都是对他人气和实力的肯定。

目测今年的语言类节目会非常精彩，太期待啦！

「Vicky's 影评」《我告诉你女孩子一生都要害怕些什么》：女性到了年龄就该结婚，谁规定的？

All Image Source

「女性到了年龄就该结婚，谁规定的？」

无论是之前频繁被404的《天朝渣男图鉴》，还是著名电影《芝加哥》中的经典片段“牢房探戈”，所展现的都是女性在婚姻/两性关系中所遇到的出轨、家暴、重男轻女以冷血等现实问题。

女囚们的自述，随着感情及情绪的积累和叠加，慢慢演变成为了对男性伴侣的控诉。

这掷地有声的呐喊并不是呼吁女性去手刃渣男，而是对女权主义崛起的呼唤，更是对现今社会女性所遭受到的各种不公平待遇绝望而又无奈的叹息。

崔雪莉因为网络暴力而患有重度抑郁症，具荷拉遭遇暴力和性侵害，让这两个原本前途无量的女孩子早早结束了生命。

Papi酱旗下艺人宇芽，曝光自己被前男友沱沱的风魔教多次家暴后，相关部门立即开展调查处理这两起事件。

是谁发明了「家暴」这个词条，这分明是故意伤害。

我对婚姻没有信心，所以我选择不结婚，不想27、28岁以后就遭遇来自周遭的狂轰滥炸。

“女人过了25岁就走下坡，掉价了变成老姑娘就没人要了。”

“不结婚你还想干啥，还能上天了？”

“只有结婚才是人生的正道，你必须回到正道上来。”

……

一直觉得结婚也好生育也好，必须是自己身心都准备好了，也遇到了想要共度一生的人才会走进的人生下一阶段。

把自己的人生过好，不向世俗妥协，婚姻已经不是女人的唯一证明自己价值的途径。

推荐影片：《我选择了不结婚》、《春夜》、《芝加哥》
