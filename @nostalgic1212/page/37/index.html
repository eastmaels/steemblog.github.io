「Vicky's 影评」《陈情令》：极致匠心书写国风之美，彰显传统文化自信#4


All Image Source


「极致匠心书写国风之美」


原著《魔道祖师》属于架空小说，服化道等方面并无相关朝代可以借鉴。但观众在看《陈情令》的时候不难发现，这个年轻的团队花了很多时间精力去设计和制作这一切。


造型师在尊重原著的基础上，结合了中国传统服饰文化（如：宽衣薄带），打造了这一系列长线型飘逸的戏服。团队希望通过减弱剧中角色衣食住行这些烟火气十足的行为，赋予角色们符合仙侠剧的“仙味”。


五大仙门世家的选址极具地域风格，建筑物上皆有家徽。


弟子门生所穿服饰所用物件上均有家纹，如代表姑苏蓝氏的卷云纹，代表云梦江氏的九瓣莲花纹，代表兰陵金氏的牡丹纹，都是古代传统纹饰。这样的好处就是，即便来人没有自报家门，观众也可以一眼看出哪些人是来自同一家族的。


本剧大多数场面都采用实景拍摄的方式，使得全剧色调自然舒服，也减少了近些年来观众们被绿布抠图支配的恐惧。




All Image Source


姑苏蓝氏


蓝氏一族住在姑苏云深不知处，细究起来当是如今的苏州。外景卧松、绿竹、飞瀑、细流，内景则是细格窗棱、青纱帐幔，让观众仿佛置身于江南园林之中。


擅音律重规矩的蓝氏在服装上整体的基调是青色和蓝色，一眼看去就是清冷。家徽是是一卷云纹，门风只二字，雅正。眉上皆束有抹额，非父母妻儿不可触碰。


蓝忘机正是蓝氏一族的典范，服饰就多以蓝色、白色为主。




All Image Source


云梦江氏


魏无羡及其同门师弟江澄和师姐江厌离，均是来自位于湖北江汉平原一带的云梦。


云梦江氏整体的气质更开阔，家风偏向游侠式的自由感。在场景设置方面，建筑多以偏曲线形、暖色调为主。


服装师在设计这个家族服饰的时候，更多采用了淡紫色作为主色调。材质多用棉麻，服饰运用窄袖设计，给人以舒适、生活化的感觉。


前期魏无羡性格灵动活泼，服饰就以暖白色为主，加入草绿、压簧，贴近整个江氏家族体系。从乱葬岗归来后，该角色亦正亦邪的特性更加明显，服饰就转变为以黑色、红色为主。




All Image Source


兰陵金氏


这一族无论是出场还是所住的金麟台，无不透着“阔气”二字。


无论是家主金光善还是其族人金子轩、金光瑶、金凌，身着金色衣服，皆在领口和袖口处用金线绣着繁复的暗纹。其家徽更加简单直白，一朵盛放的牡丹。


用于夜猎的缚仙网价格不菲，金氏小公子金凌一出手便是400张，真正的大户人家。

「为剧集增色的配音团队」


说起边江这个名字，观众们可能并不太熟悉。


但如果我说《三生三世十里桃花》夜华君（赵又廷 饰）、《何以琛萧默》何以琛（钟汉良 饰）、《老九门》张启山（陈伟霆 饰）、《笑傲江湖》令狐冲（霍建华 饰）这些耳熟能详的角色呢？


边江擅长给冷清、孤傲、冷静的角色配音，《陈情令》剧组找他来给蓝忘机这个角色配音真可谓是押对了宝。


大家有所不知的是，动画版《魔道师祖》中蓝忘机一角也是边江配音的。


剧组另一个英明的决定，就是选择了路知行给魏无羡这个角色配音。


路知行的声线清澈时尚，性格多变。


他是《琅琊榜》中性格爽直的金陵小公子，《河神》里吊儿郎当的郭得友，也是《欢乐颂》中老实木讷的老好人王柏川。同时，他也是《魔道师祖》广播剧版中那个亦正亦邪的魏无羡。


路知行在配合肖战塑造魏无羡这个角色时，充分考量到肖战的吐字习惯，让声音的停顿和演员的动作搭配的天衣无缝。


当下网路上，充斥着“用配音即废物”的言论。


如我们所知，影视剧需要用到配音大多是两种情况：演员声音素质和台词功底太差或是剧组想要作品最后以更完美的样子呈现在观众面前。


就《陈情令》而言，我个人觉得是后者。


从场外肖战和王一博的采访来看，两人的声线虽谈不上“听了就怀孕”，但总体来说还是相当悦耳舒服的。两人在花絮中，更是被拍到走到哪里都在背台词的场景。


在演员不躲懒的前提下，剧组最终采用专业的配音演员，是出于一种对声音艺术的尊重。

「贴合原著的神仙选角」


《陈情令》在选角方面，首先做到的就是“养眼”。


小说改编剧集常被书粉吐槽“见光死”，男主之一的肖战却凭借一组魏无羡的定妆照，让无数粉丝纷纷表示“肖战就是我们心中的魏无羡”、“惊喜”、“良心选角”。


《陈情令》开播以后，肖战天生一副笑脸的俊朗外形使得魏无羡这个纸片人形象仿佛破壁而出。随着剧情的发展，喜欢捉弄友人的古灵精怪、潇洒飒爽到面对家破人亡时的万念俱灰，非科班出生的肖战之表现也着实可圈可点。


网络上有一张肖战读剧本时候泪流满面的路透图，由此不难看出他是用心去塑造这个角色的。他最大程度的还原了这个角色的乐天性格以及在逆境中始终对道义的坚守，展现出超然的与该角色共情的能力。


蓝忘机，姑苏蓝氏二公子。为人清冷最重礼数，高冷话少，最初没少被魏无羡气到吐血。


高冷男神简直就是古装剧中的标配，远的不说近的就有《花千骨》中的长留上仙白子画，《三生三世十里桃花》中的老铁树东华帝君。


人冷可以，表情总该有吧，加上过紧的发套和睡不醒的表情，使得这个角色的饰演者王一博最初没少受到观众的质疑。


随着剧情的推进，剧组在服装和道具方面的调整让王一博的面部更加自然。渐渐的，蓝忘机这个角色鲜活了起来，几场“神仙落泪”的戏更是让人震撼。


不得不说，年仅21岁的王一博真的是天赋型选手，未来可期。


「传统乐器诠释国风元素」


我们在看影视剧的时候，经常会用“视听盛宴”这个词来形容，足以见得在听觉方面的享受有多么重要。


《陈情令》是国内少见的，在配乐方面用心打磨的古装剧，由我国知名音乐家、制作人林海操刀。


林海在《陈情令》的配乐中大量使用了古琴、萧、笛、筝等中国传统乐器，曲风随着剧情推进而呈现出时而悠扬婉转、时而热血高昂、时而催热泪下的效果。将国风发挥到极致的同时，更在观众面前展现出一幅快意江湖的画作。


与此同时，如同《红楼梦》里每个人物都有自己对应的词，《陈情令》也做到了剧中主要人物皆有量身打造的词曲，且大部分由相关角色饰演者亲自演唱。


总体来说古风味十足，深得观众的喜爱。

「Vicky's 影评」《陈情令》：极致匠心书写国风之美，彰显传统文化自信#1


All Image Source
文/冰冰


2019年的夏天，是属于几部热播电视剧的。


《德鲁纳酒店》、《凪的新生活》、《长安十二时辰》、《小欢喜》、《亲爱的，热爱的》、《今宵大厦》…


当然还有这部：《陈情令》


不少观众从吐槽故事改编、角色妆容到熬夜刷剧，还充当自来水帮这部国剧做宣传，豆瓣评分更是从4.8分逆袭到了8.2分。上一次


实现豆瓣评分翻盘的热门剧集，还是周迅主演的《如懿传》。


《陈情令》作为一部题材稍显敏感的电视剧，能在同期竞争如此激烈的今夏杀出一条血路。不能不说，它的成功并不是偶然。


「快意江湖的新武侠世界」


抛开原著《魔道祖师》较为敏感的感情线以外，其故事架构本身还是十分宏大的。


云梦江氏、姑苏蓝氏、兰陵金氏、岐山温氏清河聂氏，天下由这五大世家共同治理，也度过了不少平静的岁月。


只是，只要是人就有欲望，而欲望会引发战争。岐山温氏的家主温若寒不顾众仙门反对，着手收集散落四方的阴铁。


乱世出英雄，首当其冲的便是姑苏蓝氏的二公子蓝忘机（王一博 饰）和云梦江氏首徒魏无羡（肖战 饰），二人合力联合四大家族共同讨伐温氏。


故事情节环环相扣，人物形象生动饱满。更难能可贵的是，剧情弱化了感情线，更多地去讨论了人性、道义和是非曲折。


剧组在对小说进行改编时，将原著较为敏感的内容做了删减，以期在能过审的情况下最大程度保留原著的主线。


《陈情令》采用顺序与倒叙穿插进行的方式，弥补了原著在叙事时间上较为凌乱的不足，使得即使没有看过小说的观众也不至于摸不着头脑。


不仅如此，剧组还调整了人物出场的顺序。薛洋、晓星尘、宋岚等主要角色提前登场，为剧集增加了更多矛盾冲突。以血洗莲花坞为界，将故事推上一个小高潮，给了观众更多的观影体验。


在叙事结构上，《陈情令》采用几集讲一个案件的模式，五大家族的年轻人在两位男主的带领下寻找阴虎符的下落。在其中穿插稀松平常的日常斗嘴，故事有张有弛，很大程度上提高了剧集的可看性，也使得角色的人物形象更为饱满。

「Vanilla‘s Diary」养猫如同养孩子，从小教育才是正道 191031

不知不觉，香草到我家也已经快要一个月时间了。小家伙就如同小baby一样见风就长，这孩子稚气褪去，曾经雌雄难辨的脸庞也越发显露出男孩子的特性来。

在接香草回来之前，我就听说有些规矩和习惯需要在小时候就开始“约法三章”，比如梳毛、剪指甲、洗澡、吃药…而不准咬人这点，自然也是列入在内的。

他从刚来的时候开始，似乎就对人类的手指特别感兴趣，时不时就要手脚并用的抱住开始啃咬。作为主人，必须要养成不用手逗弄小猫的习惯。如果要陪他们玩，使用玩具（逗猫棒、布老鼠等）会比较好。

那么问题来了，如果他们还是咬了你，应该怎么做？

我个人喜欢先礼后兵的思想，采取渐进式的“教育”方法。

最初的时候，如果他咬人，一般我会迅速停止游戏。把他晾在一边不同他说话，又或者指着自己被咬痛的地方板着脸告诉他，不可以这么做。

如果上述方法不启用的时候，一般我会抓住他后颈部分提起来，大声呵斥他。用简单的音节，类似“不要”、“NO”之类的词汇会比较好。猫咪对声音非常敏感，因此大声呵斥会让他们神经紧张。通常，出现飞机耳了就差不多了，主要不要太过了。

可惜，我们家香草在我用了前面两种步骤以后还是没有收敛，依然是逢梳毛、剪指甲的时候必咬人的状态。可能是最近玩疯了，后来逐步发展成陪玩玩儿的时候也会张嘴咬人。

眼见着事态正在朝着失控的方向发展，我终于下定了决心要纠正他的行为举止。

这么做以后，产生了几个结果：

1、香草变得很少仰天睡觉，以前他都是睡的四仰八叉的；

2、待在我身边的时间变少了，更愿意和我父母待在一起；

3、我的手一接近，香草就会举起手摆出防御的状态。

简单来说就是，打得太狠了！

父母自然是对我的行为百般指责，觉得我教育小孩子不应该下手那么狠。但是，看到最近几起未成年人暴力事件以后，我觉得小孩子的行为必须从幼年时期开始就被家长干预。在还能管教的年岁加以引导，后面还不至于失控到父母都无法矫正的地步。

小猫咬人是这样，小孩子的教育亦是如此。

「Vicky's 影评」《双子杀手》：从好莱坞华人之光，到接连票房失利，李安的神话是否还能延续？#3

「陈旧又稍嫌突兀的剧情」


有了父亲三部曲打底，让观众难免对这部带点“弑父”情节的《双子杀手》格外期待。


毕竟，小亨利的枪口不仅对准了老亨利，还指向了创造他的养父克雷。


正所谓期望越大，失望也就越大。


在知道自己是克隆人那刻起，小亨利内心几乎没有做什么挣扎，可以说是毫不犹豫地背叛了养父。


就我们所知，养父虽然把小亨利从小培养成杀手，但也是把他像亲生儿子一样对待的。这么火速地向DNA本体的老亨利倒戈，也是让人始料未及的。


此外，克隆人的电影我们已经看过太多。除了著名的《黑客帝国》系列以外，给我留下深刻印象的便是美日都曾拍过的《别让我走》。


该剧根据英国日裔小说作家石黑一雄的同名原作改编，讲述了一所寄宿学校中的孩子皆是为了器官捐献准备的克隆人。


剧集最后指向了以克隆人去弥补当下器官移植的空缺是否可行，以及随即而引发的社会伦理的讨论。


这一切，与《双子杀手》中用克隆人上战场代替普通人以减少牺牲，却又将面临伦理的诟病如出一辙。


就完成度而言，日版的《别让我走》显然更为出色。


不可否认，李安导演是深爱电影的人。


他本可以继续拍摄熟悉的题材，享受着观众的追捧。


但是，他却不是一个墨守成规的人。


《卧虎藏龙》、《绿巨人浩克》、《断背山》、《色戒》、《少年派的奇幻漂流》……


以65岁的年纪想要推动电影新技术的发展，他走上了一条又一条很少有人走上的道路。


李安导演的下一部电影《马尼拉之战》，将再次使用3D/4K/120帧技术。


它描绘上个时期六七十年代拳击界的景象，其重头戏就是乔·弗雷泽与阿里两个拳王之间那场被称为“马尼拉式震撼”的传奇之战。


这世上的所有事物都在变化，影视界日新月异风云万变，各种新技术源源不断出现。


但我知道，这个“老顽童”李安热爱电影的这份心不会变。


祝李安导演好运！


希望他的下一次尝试，可以在艰难中看到曙光！
