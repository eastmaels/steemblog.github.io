「Vicky's 短评」《囧妈》：爱不应该是「控制和索取」，爱应该是「接纳和尊重」

[All Image Source](https://movie.douban.com/photos/photo/2580034326/

在经历提档撤档之后，《囧妈》的导演徐峥让电影以史无前例的方式与观众见面。

大年初二，我和妈妈坐在iPad前，体验了一把足不出户看最新电影的乐趣。

《囧妈》讲述了小老板伊万（徐峥 饰）缠身于商业纠纷，却意外同母亲（黄梅莹 饰）坐上了开往俄罗斯的火车，一路上发生了啼笑皆非又感人至深的故事。

在旅途中，他和母亲发生激烈冲突，同时还要和竞争对手斗智斗勇。为了最终抵达莫斯科，他不得不和母亲共同克服难关，并面对家庭生活中一直所逃避的问题。

说到底，母子是同一个类型的人。

他心里有个幻想中的妻子形象，应该喜欢什么样子的台灯，应该如何待人接物；

他心里有个幻想中的儿子星系，应该吃什么东西，应该喝多少水，应该什么时候排水。

加上不擅长表达自己的性格，让彼此间的距离越行越远。

但事实上，两人的出发点都是基于对彼此的爱。

只不过，这份建立在「希望对方活成自己想要的样子」上的爱，实在是过于沉重了。

每个人都是独立的个体，每个个体也都应该是完整的。

爱不应该是「控制和索取」，爱应该是「接纳和尊重」。

别等到来不及了再明白这重要的相处之道，无论是对爱人、亲人还是朋友，恰到好处地表达爱的方式，尊重他人的性格和喜恶，比什么都来的弥足珍贵。

（The End）

「Vicky's 资讯」 肖战连轴转，热度毫无断层的他，收获了各个领域的喜爱

All Image Source

因为《陈清令》一切成为当红小生的肖战，真的可以说是从去年夏天红到了现在。

在《余生请多指教》杀青后，他参加了东方卫视《我们的歌》明星真人秀节目，展现出其唱歌天赋和综艺感。在不久后的春节联欢晚会中，肖战也将参加语言组的表演。

他参演的《庆余年》本身就是当下最火的热播剧，想必借着这股东风又收获一波路人粉。

除了奢侈品、游戏、美妆、电器、日化、手机多领域代言以外，肖战还深受时尚圈的喜爱。他频频作为封面人物出现在时尚芭莎、Vogue中，去年还参加了Cosmo的时尚盛典。

时至年底，肖战各种大奖简直可以说是拿到手软，更是马不停蹄地参加各种盛典。只要是他参加的活动，均获得颇高的关注度。

在下一部剧进组之前，肖战还参加了《最美表演》企划。在两个短片中，他丢掉了过去又仙又帅的形象，向观众展现了截然相反的另一面，演技颇受肯定。

能保持毫无断层热度，除了自身的努力，同样也和经济团队的策略得当有关。

演员肖战，未来可期！

（The End）

「Vicky's 资讯」《默读》动画版PV放出，观众是否对此买单？

All Image Source

上周日下午，根据Priest原著《默读》改编的动画版正式上线。

此前，在周六发布的预告中，宣发部门用了相当新奇的方式——摩斯密码。不得不说，这种方式既勾起了观众的兴趣，又紧密贴合了原著剧情。

2020.01.19 16：00

动画PV准时上线，仅仅5分钟内有了上千条用户留言。除了之前的声优问题以外，这次还多了关于画风的讨论。

不少观众反应，动画中的主角画风与动画设定稿有明显差异。根据原著，无论是骆闻舟还是费渡，都是相当美型的。相反的，动画中的费渡略显老态，而骆闻舟则面露凶相…

观众会有这样的心理落差也不奇怪，毕竟之前猫耳做的正版周边周边中，采用了广播剧封面的画师。此外，同人画手们也产出了大量符合读者心理预期的同人图。

希望动画版的制作团队可以听取观众们的意见，尽可能的作出调整。无论是《默读》这个IP本身，还是其作者Priest，都拥有大量粉丝和关注度。

如果运用得当的话，《默读》很有可能成为第二个现象级IP《魔道祖师》，让我们拭目以待吧！

（The End）

「Vicky's 资讯」《唐探3》预售23小时破亿创纪录，是否能成贺岁档最大赢家

All Image Source

1月18日0点，2020年春节档预售正式开启。

仅23小时，电影《唐人街探案3》预售就突破1亿，在发文之时已过2亿。其创下了华语影史预售最快破亿的新记录，大年初一当日，电影《唐探3》预售票房占据了总票房的60% 。

官微近日放出了《唐人街探案3》曝光一组“烧脑”版海报，王宝强、刘昊然、妻夫木聪、托尼贾、长泽雅美、浅野忠信、三浦友和、尚语贤、张子枫全员“烧脑”亮相。

面对东京篇的全新案件，9位演员纷纷陷入头脑风暴。不仅汗流不止，脑袋上还冒起热烟，“焦头烂额”的神情十分喜感。

这是陈思诚执导的《唐人街探案》系列的第三部，同时也是他担任导演的最后一部。

和黑帮大boss泡汤、与相扑选手较量、舅舅嘱托要照顾好泰美日三处的舅妈等场景，真的是笑料百出。

密室、本格、黑帮，小说都写不出来的丰富元素。

大年初一，《唐人街探案3》值得一看！

「Vicky's 资讯」电影《紧急救援》曝“势不可挡”终极预告

All Image Source

《湄公河行动》和《红海行动》这两部作品，让林超贤成功跻身大陆主旋律电影导演的第一梯队。

今年贺岁档，他带来的《紧急救援》是中国首部海上救援题材电影，由彭于晏、辛芷蕾、王彦霖、蓝盈莹、王雨甜、徐洋、陈家乐、李岷城等主演。

据悉，电影筹备期长达四年，辗转国内外多处进行拍摄，相信会有不错的视觉体验。

在最近发布的终极预告中，高谦（彭于晏饰）与队员只身闯入险境之中。冲天的烈火与震耳欲聋的爆炸声并没有阻挡救援队员的步伐，电影画面给人带来了强烈视觉震撼，使得观众们仿佛身处险境之中。

燃爆的火场，暗藏杀机的深渊，这一幕幕我们不曾经历甚至想象不到的灾难现场，却时时刻刻都在世上的某处真实上演中。

在此前的报道中得知，为了保证在实际拍摄中不会出现意外，导演林超贤要求演员们在拍摄前进行高强度、高难度的相关培训。演员们不仅要克服内心的恐惧，还要锻炼出与专业救援人员一般的心理素质。

目前，电影已开始预售。

大年初一，随海上应急反应特勤队亲临一线，用眼睛感受他们的勇气吧！

极限求生，紧急救援！

（The End）
