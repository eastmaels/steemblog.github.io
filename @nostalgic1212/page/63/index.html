【Vicky's 影评】《失恋三十三天》：时间是治愈一切的良药-7


All Image Source


别指望拥有一个能解救你人生的完美伴侣


小仙儿接了一个金婚的客户，客户陈书坤（魏宗万 饰）为了弥补当年没给妻子张玉兰（曹翠芬 饰）办一场婚礼，想让她一满穿婚纱的愿望。陈老师清楚张老师的身材尺寸，婚纱的事都是他忙前忙后地在张罗。这一对老夫妻，实在是羡煞了黄小仙。


这天小仙儿去给张老师看修改好的婚纱尺寸，躺在床上的张老师说了老伴当年一件上不了台面的事。当时在医院待产的张老师发现这档子事儿以后，应对方式俨然就是过去大房对待侧室的态度。


她并没有朝着老公大吵大闹，也没有去婆家或者娘家苦恼撒泼，更没有寻死觅活。她找到小三借着问候的名头骂人不带脏字儿地狠狠敲打了一番，接着回去就和老公摊牌，然后一家人和和美美地生活到现在。


我们看过很多模范情侣/夫妻，每每见着都会感叹“又相信爱情了”，背后默认他们从始自终都是这样和美幸福。我很喜欢范玮琪的「爱情没有捷径，只有经营」这句歌词，真正好的爱情并非都是成天爱来爱去，而是经历了欺骗、争吵、背叛、算计、诱惑之后还能通过沟通和包容继续走下去。


「像买台冰箱，保修期三年。你嫁了个人，还能要求他一辈子不出问题啊，出了问题就修嘛」


张老师聊聊数语，却每个字都是过来人的经历和智慧。如今的女性再不能是那种结了婚以后就如同签下长期饭票的心态了，而是必须成长成一个无论是人格还是经济独立的个体。在这样的心态下以平等为前提去面对爱人女性才会觉得更加的坦然。


当两人的关系出现状况的时候，带着脑子的沟通是必要的。这里的沟通指的是彼此冷静下来阐述自己的诉求，而不是单纯的情绪宣泄，然后剩下的就是你内心做的权衡。


如果你觉得这事儿在现有条件下能翻篇，那就坦然度过这个坎并且从此以后再不翻旧账；如果在内心始终觉得膈应，那就干脆的说再见并且今后再不后悔。我上面说的独立，就是让大家在做这个选择的时候不会被经济因素所裹挟。

【Vicky's 碎碎念】写作课程知识整理：如何获得机器更多推荐？（2）190715


All Image Source

  • 哪些因素会影响推荐量和阅读量？

「正面影响」：首发、关键词、标题、正文、用户反馈

1> 关键词

这里不得不感概一下平台庞大的推荐系统，它会根据用户在注册时候所选择的阅读习惯以及将阅读习惯相似的用户划归同一类的方式，将相关关键字（内容）的文章给推荐过去；

2> 标题/正文

其优秀与否的标准，是当朋友或者陌生人看到这篇文章的标题时就可以迅速锁定阅读人群。比如我写的是影评，那么标题中就要精准的提供可以吸引到常看影视剧的读者喜欢看到的信息量。

在标题和正文孰轻孰重这点上，以标题优于正文为标准。打个比方，有个闺阁女子特别有才华并且品行出众，唯一的缺点就是长相平庸甚至有些丑陋。那么换作是你，会有兴趣去了解对方“有趣的灵魂”吗？


All Image Source

3> 用户数据

简单来说就是，读者打开阅读、点赞、转发、留言这类互动量越多，文章的数据就会越出色。系统就会判定这篇文章可能会有更多的人想要看到，接下来也就会推荐给更多的人去阅读。

在文章刚发布后的一段时间内，系统会根据关键字识别将文章优先推荐给你账号的粉丝-同样兴趣点的用户-关注相近领域的用户-全平台用户。有前辈曾经提过一句话，就是如果可以在文章发布后的短时间内通过APP端打开文章，以此“诱导”系统给予这篇文章更多的推荐量也许是有迹可循的。

当然了，前提是这个作者本身就有一定的人脉和资源。如果是同为写作群内的作者互相帮忙打开文章阅读，那很可能在文章的冷启动阶段就获得不错的阅读量。不过话说回来，我也曾经有过文章发布一天后阅读数还是十位数的情况。但随着时间的推进，阅读量最终到达几百上千甚至几万的情况。

虽然无证可考，但我个人觉得作者和用户的互动是十分重要的。大家在发布文章以后如果看到读者的点赞和留言，应该尽可能的去和他们互动。

「Life & Activities」令人倍感困扰的下午茶文化 / The Things Bother Me So Much 20190715


不知道从什么时候开始，我们办公室开始流行起了下午茶文化。起先是新来的几个男同事请喝奶茶或者蛋糕，抱着不能吃白食的理念我和另一位女同事H也加入了这个行列。

某些程度来说，其实我不是特别好这一口。我对自己的身材管理很严格，成年以后的体重体脂一直维持在一个规定的数值，上下浮动不算很大。流行已久的各种奶茶和各色蛋糕，我也基本都没有尝试过。这办公室的下午茶文化，确实结结实实地让我对奶茶和甜食上了瘾，有时候晚上十点多一边写文章一边都要喝上一杯。

两个月前意识到这个问题的严重性，我果断限制了自己甜食尤其是快乐肥宅水的摄入量。同事点的下午茶一般我也很少喝或者直接拿回家给家人，轮到我点单的时候直接点了他们的pass我自己那份。久而久之，同事们也觉得不太好意思渐渐收敛了吃下午茶的习惯，或者干脆吃些水果、坚果等相对来说健康一点儿的食物。

你们的办公室有下午茶文化吗？

【Vicky's 碎碎念】写作课程知识整理：如何获得机器更多推荐？（1）190714


All Image Source

  • 文章在平台会经历什么样的处理流程？

1、消重：如果是抄袭或者非法转载的文章，会被系统列入不被系统推荐的行列。通过看这个机制其实不难了解，平台更推荐作者首发在该平台以换取更高的推荐量。毕竟，推荐量某些程度和阅读量以及文字变现的能力直接相关；

2、审核：一般来说，平台的审核采用机器+人工想组合的方式。色情、低质、违法包括旧文，原则上来说都是很难通过审核的。如果是标题党和软文，原则上会限制其推荐量，所以作者在选择写文方向的时候需要规避掉这一些。不过就标题党这一点来说其实我保有存疑，就我来看国内自媒体平台标题党的现象并不罕见；


All Image Source

3、特征识别：用我们的话来说就是，这边文章的目标读者是谁。系统会根据作者文章的关键字（内容）通过算法推荐给对该内容有兴趣的读者，这就是目前我正在着力学习的一个技能，即如何抓有尽可能多目标读者的关键字；

4、推荐：这就是特征识别休戚相关的一个环节，也是目前平台作者们为之努力的重中之重。

All Image Source


李可和魏依然这一对，正是时下快餐式爱情、交易式婚姻的缩影。一个千娇百媚其实是如假包换的山寨货，一个温文尔雅其实满肚子坏水。李可的要求很简单，就是一直要有奢侈品和高档会所可以供她挥霍，而魏依然则需要一个带得出门又不会管他太多的漂亮妻子。


当我们说起爱情，常听到身边的妹子会说“我长得漂亮学历也高，性格活泼开朗风趣幽默，下的了厨房还才华横溢，为什么我心里的白马王子找了个资质平庸的妹子？”每次听到这里的时候，我都会满脸无奈地和她们翻个大白眼。


某些程度来说，爱情其实是一场等价交换的博弈（生意）。要想获得最匹配的爱情我们首先要知道自己有什么，再去衡量以此能得到什么。一个长相、家境和学历都一般般却能收获从颜值到智商再到家庭都满分并且还会哄人的男神，这个只有袁湘琴才能做到。


顺便还要说一句，江直树和袁湘琴两家是世交，现实中的“袁湘琴”更是一个大才女。


我自己就是一个文艺青年，用我的话来说像我这样成天浸泡在书堆里面的女孩子其实最难把自己嫁出去，在恋爱关系中也最难守得住爱情。


你和我谈物质，我也许会大义凛然地对你说只要感情好物质不是必要的，但在要为房贷车贷伤神而不能买自己喜欢的衣服鞋子包包的时候，又会成天唉声叹气闷闷不乐。恋爱要求对象精神洁癖，还要对方安抚我偶尔出现的复杂性人格。


说实话，如果我是男人也会觉得李可们太招男人爱了。上的了厅堂也不会干涉丈夫太多私生活，回到家只要给她一张卡并且额度超高则万事皆休。


很多时候，在身边的妹子开启吐槽模式死命吐槽自己的男友/老公的时候，我都会对她们说放过他们吧。


当下社会中的女性承受了很多压力和社会责任，但在经济方面男性显然承担的会更多一些。他们要买房买车攒育儿费用，要隔三差五给另一半买小礼物让她们可以有面子，同时还要随时呵护对方那也许一碰就碎的玻璃心。有些有上进心的人还要学一些自我提升的东西，那么就更忙碌了。


那些觉得输在不够美丽、不够较弱、不够善解人意的妹子，在我看来只是输在了不够省事儿又太难取悦，输在了自己要的太多又能提供的价值太少。如果可以对自己有更切实际的认识，我想也许属于她们的Mr.Right就在下一个转角。

【Vicky's 碎碎念】关于SteemFest⁴的一些小想法（下）190713

「仍在考虑的问题」

语言：就我上次泰国旅行的经历来看，语言是个比较头大的问题…可能我说的是中国式英文，在问路、买东西的时候常常会出现沟通不良的情况。之前还曾出过迷路的情况，这点让我感到有些后怕…如果出行的话，我希望能找到语言能力还OK的同伴一起前往；

住宿：金鱼蜀黍的朋友michael曾经提到过会协助到泰国参加会议的steemian找到舒适又高性价比的房间，对此我感到非常感谢。如果此次选择参会的话，我想我还会去请教几位经常往返于杭州和泰国之间的朋友给出一些建议；

预算：机票酒店如果能控制在5000元以内，我想这趟行程应该没有太大的问题。根据我之前的出行经验，泰国的消费不算特别高，如果在吃喝玩乐方面节制一些的话预算可以控制在8000元以内。现在最大的问题依然是「门票」，之前听说需要好几千元，那么这远超我整个年度的预算了。

「总结」

我对这次SteemFest⁴还是非常抱有好感的，也很希望带着好朋友们一起来场榴莲盛宴。如果情况允许的话，我可能还会找出1-2天的时间去潜水。自从去年4月考出OW潜水证之后就没有再下水过，再不训练的话我很担心自己会把好不容易学到的技巧都忘记了…

虽然目前我还未能确定行程，但我对此持有乐观的态度。希望能穿着汉服（中国传统服饰）参加这次盛会，也希望见一见来自世界各地的steem用户们:D
