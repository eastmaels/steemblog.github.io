「Vicky's 资讯」《囧妈》：来自母上大人霸道总裁式的关怀，你get到了吗？

All Image Source

有一种冷叫你妈觉得你冷，有一种饿叫做你妈觉得你饿…

“别的孩子都做得到，为什么就你不行？”，“找女/男朋友了吗，什么时候带回来”，“我都说了应该XXX，你怎么就不听话？”…

OMG，多么熟悉的来自妈妈的灵魂拷问…

大年初一上映的这部由徐峥自导自演的《囧妈》，讲述了一对母子乘坐K3前往俄罗斯途中的爆笑故事。预告片中，观众不禁感叹这简直就是成年后的自己和母亲之间的日常互动。

山争哥哥的“囧妈”由黄梅莹老师饰演，这位妈妈简直就是个宝藏+斜杠青年。

她是疯狂投食的“幕后黑手”，是向往舞台的“solo歌手”，也是疼爱孩子的母亲。

和母上大人相爱相杀是种什么体验，大年初一，去看《囧妈》吧！

「Vicky's 资讯」2020年贺岁档影片扎堆，你会pick哪一部？

All Image Source

2020年贺岁档即将到来，这块兵家必争之地也使得众多影片都会扎堆在这个时段上映。

截止到目前，已经官宣将在大年初一上映的影片有《唐人街探案3》、《囧妈》、《姜子牙》、《紧急救援》、《中国女排》、《只有芸知道》等。

《中国女排》：青年陈忠和与郎平女儿白浪饰演的青年郎平，两代教练在年少时期互相扶持、互相激励的友情，不禁让人动容。

陈可辛导演+巩俐+黄渤+彭昱畅+郎平女儿白浪，中国女排阵容含金量太高了。

《唐人街探案3》：唐探组合与妻夫木聪、托尼·贾组成的“亚洲侦探联盟”，再现经典“风衣走”。张子枫、三浦友和、长泽雅美，这简直是“亚洲群星阵容”。

《紧急救援》：在《湄公河行动》和《红海行动》后，让观众熟悉了林超贤这位大陆主旋律电影导演。他带来的《紧急救援》是中国首部海上救援题材电影，筹备四年辗转国内外多处拍摄，相信会有不错的视觉体验。

《囧妈》：徐峥“囧”系列第三部，《囧妈》（俄囧）讲述了一对母子相爱相杀的故事。从路透出来的信息来看，阵容除了徐峥以外，还有沈腾和袁泉，笑点颇为密集。

《只有芸知道》：影片根据冯导好友的真实经历改编，讲述了中年丧妻的隋东风替亡妻完成遗愿的感人故事。虽然不是冯导擅长的喜剧片，但根据《唐山大地震》和《芳华》的催泪程度，相信本片会是一部很不错的片子。

《姜子牙》：2019年是中国动画爆发的一年，不仅有火到国外去的耽改动画《魔道祖师》，更是有口碑爆炸的动画电影《哪吒》。

《姜子牙》将延续《哪吒》的风格，进一步打造符合中国人口味的成人动画片，也许会成为下一个爆款哟。

2020年贺岁档影片扎堆，你会pick哪一部勒，快来给冰冰留言吧:D

（The End）

「Vicky's 资讯」郭敬明《阴阳师》全阵容曝光，赵又廷邓伦主演

All Image Source

郭敬明将执导日本知名IP《阴阳师》（分为《晴雅集》和《泷夜曲》）的消息一直在坊间热传，该消息在近日已经官宣，其阵容公布如下：

赵又廷 饰 晴明

邓伦 饰 博雅

王子文 饰 公主

春夏 饰 泷夜

汪铎 饰 鹤守月

在之前的路透中，网友们呼声最高的就是让邓伦出演晴明这一角色。就外形上来看，邓伦与晴明的契合度更高。

其实要是放在5-10年前，我个人觉得陈坤和严宽也是相当不错的晴明人选。

郭导选择赵又廷出演晴明，想必有他独到的见解版。毕竟，虽然郭导的作品颇受质疑，但其票房却相当高的。不少新人演员在与他合作以后，得到了很不错的曝光率。

当然了，不少网友也不禁替写而优则导的郭敬明捏了把冷汗。

《阴阳师》在国内有着非常高的粉丝群体，不仅有同名手游、音乐剧、话剧等衍生产品，还频繁出现在各大漫展之上。

小说本身在日本颇受欢迎，泷田洋二郎执导的电影版，由野村万斋、伊藤英明、今井绘理子、小泉今日子、真田广之等演员出演。至今，很多人提到安倍晴明，第一印象就是野村万斋主演的形象。

珠玉在前，郭导如果拍的稍有不慎，很可能遭到粉丝的抵制和批评。

「Vicky's 资讯」肖战《最美表演》形象颠覆，不再乖巧的他，贫嘴也惹人爱

All Image Source

年底各大颁奖活动都在陆续进行中，《最美表演》作为微博的特别企划活动，受到了广大网友们的关注。

活动先是让网友们通过投票，选出他们心中10位优秀的演员，再由项目组联系入选人与金牌导演合作。据悉，最终呈现出来的10部影片大多会颠覆演员原本的荧幕形象，粉丝们均表示出十成十的期待。

毫无悬念的，因为《陈情令》大火的肖战，也在入选人名单之中。

预告片中，肖战颠覆了过去乖巧形象。戴着鸭舌帽，露出狡黠顽劣的表情，像极了20出头游手好闲的小青年。

皮归皮，小青年还是相当孝顺的，他给父亲准备了一份大礼。从预告片来看，可能是一辆小轿车，不想却遭到了来自父亲的一顿扫帚…

对了，父亲的扮演者高曙光，在目前热播剧《庆余年》中出演范建一角。也就是说，两人在《最美表演》之前就已有合作。

在《陈情令》之后，看得出演员肖战做了不小的努力。台词功底更扎实了，表情和肢体语言也表现的更加游刃有余。难能可贵的是，如今已是顶级流量的他依然保持初心，在出席各大晚会活动中仍然谦逊有礼。

在《最美表演》中，南方人的他还要展现一口纯正的北京腔，简直不能再优秀啦。

老戏骨高曙光携手肖战，会擦出怎么样的火花呢，期待正片！

（The End）

「Vicky's 资讯」林超贤新片《紧急救援》，国内首部海上救援题材电影

All Image Source

《湄公河行动》和《红海行动》这两部作品，让林超贤成功跻身大陆主旋律电影导演的第一梯队。

今年贺岁档，他带来的《紧急救援》是中国首部海上救援题材电影。据悉，电影筹备期长达四年，辗转国内外多处进行拍摄，相信会有不错的视觉体验。

海报中，彭于晏在飞沙走石的峡谷中，竭尽全力伸手营救激流中垂死挣扎的人；王彦霖在海浪翻滚的大海中，沉着冷静地救出遇难者；辛芷蕾临危不乱地驾驶直升机，有序地进行空中救援。

这些救援场景虽然我们不曾面对，但却是时时刻刻都在世上的某处真实上演中。

为了保证在实际拍摄中不会出现意外，导演林超贤要求演员们在拍摄前进行高强度、高难度的相关培训。演员们不仅要克服内心的恐惧，还要锻炼出与专业救援人员一般的心理素质。

感谢《紧急救援》这部电影记录下这些英雄们的故事，我已经把这部电影列入贺岁档必看清单啦:D

【Vicky's 影评】《平日下午三点的恋人》：如果在婚后遇到了更喜欢的人，应该怎么办？

All Image Source

总的来说，韩版的演员颜值和演技双双在线，剧情较之原版更为丰满。在改编的过程中，编剧对细节的把控相当到位。不仅如此，在选角方面也丝毫没有落后于日版，单是看演员的服化道就能大致推测出剧中角色的人物性格。

然而，《平日下午三点的恋人》最后以7.4分收官。较之原版《昼颜》的8.6分，还是有些差距的。

《昼颜》整体给观众的感觉是“克制”，剧情大多集中在笹本纱和（上户彩 饰）这个角色的心理活动上。在仅有的叙述中，观众能感受到那些陷入不伦恋情后欲不断试探的心情，继而与剧中的角色们产生共情。

如果说《昼颜》中纱和和北野的婚外情唯美得像初恋，那么《平日下午三点的恋人》则是直接将这段不伦之恋拍成了初恋。韩式言情的法宝“柔光、慢镜头、磨皮、BGM”一个不落下，将孙智恩诠释成了全然不谙世事的形象。

如同很多国产类型片中，导演和编剧会树立不正确的阶级对立以求得到观众的热议和共鸣。在《平日下午三点的恋人》中，观众看到的是「对妻子冷淡对女同事侃侃而谈」、「不尊重妻子」这样恶劣的丈夫的形象。

这样的人设安排，难免有让出轨行为合理化的嫌疑。其突出了出轨的“恋爱性”，反之过于弱化了其中的“不伦性”，这也成了本次改编的最大弊端。

我不记得在哪里曾经看到过这样一个问题：“如果在婚后遇到了更喜欢的人，应该怎么办？”

如果确定真的是此生最爱，离婚重新开始，也许是个不错的选择。

（The End）
