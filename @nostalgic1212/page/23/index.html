【Vicky's电影/电视剧推荐】胖子的老婆那么美？！我放弃减肥了！-3


All Image Source


对于中国普通观众而言，《胖子行动队》颠覆了以往特工大多为帅气、精干、行动力十足的人物形象，不失为一种新鲜的创作途径。电影导演在《王牌特工》、《007》、《叠影重重》、《碟中谍》这一系列经典特工题材电影的基础上，加入了自己熟悉且擅长的喜剧元素。


这部电影以其搞笑的剧情、滑稽的动作留给观众深刻的印象，有着喜剧片本该有的那些部分。但是因为片中有多处情节与现实脱离过远加上老梗太多，让人多看过电影的人持负面评价。


影片中导演想要告诉大家“虽然我很胖，但我依然有梦想也可以实现梦想”这个励志故事，但他设置的喜剧笑点则是建立在对胖子的嘲讽之上。除了对体型的全屏恶意之外，大boss郭京飞在电影中的角色说话有些结巴。电影以此来制造笑点，同样是对语言障碍群体的一种侮辱。


国庆档的电影市场中，除了老牌大导演的《影》和类型片《找到你》，进入喜剧片角逐的只有《李茶的姑妈》和《胖子行动队》。某些程度来说，后者还拯救了一把前者，毕竟能让一部喜剧片得到观众反感的着实不多见。


除了上述讲的这些问题以外，《胖子行动队》还有个致命的问题，就是许多情节不符合逻辑。


本片的故事背景是在日本，郭京飞作为一个华人在当地是一名杰出的房地产商人。除了表面光鲜亮丽，背地里他还拥有一群黑帮打手并在日本制造和销售毒品。你们觉得这个逻辑上行得通么，日本当地的黑社会势力不要面子啊？


再者，通过电影最后反转的那一幕，我们会发现郭京飞边上的得力助手辣目洋子其实是一枚卧底。我们再回想一下，基地原本给郭京飞安排的任务是什么？那是一份基地安排给辣目洋子去执行任务的任务书啊！


那么我想问一下，在这个任务书被文章中图截胡的情况下辣目洋子是如何知道自己的下一步工作是什么的？如果辣目洋子本来就知道自己要做什么，那么是不是意味着基地给文章安排的“取文件”的任务本来就是一件毫无意义的工作呢？

七月对我来说，是自媒体写作方面收获颇丰的一个月。不仅学习到了不少专业的自媒体写作、运营知识，才切实地拿到了第一笔佣金。当然了，「焦虑」这个词依然是这个月的重点词汇，贯穿了整个月份始终。8月份开始要适当调整自己的写作计划，重心逐步向今日头条那里靠近。

Anyway，先看看这个我有做哪些事情吧。


All Image Source

「写作方面」

本月一共写了84篇，其中包括nostalgic61212（67篇）以及icebing925（17篇）。文章的数量比上个月少1篇，但是每日写作上花的时间由2-3小时增长到4-5小时。本月基本没有写tasteem（虽然在收益惨淡的时候这个比较好撸），但在925的账户下写了比较多SCT的文章。本以为7月会写很多书评，这点在实际操作中并未实现。

「书影音」

七月在豆瓣上有33个书影音记录，比上个月少了31个。

看过3部影视剧，发布了11篇评论共获得1个赞。目前在读的书籍2本，在看的影视剧2部，较上个月有显著减少（足以见得这个月的写作压力有多大）。

【Vicky's电影/电视剧推荐】胖子的老婆那么美？！我放弃减肥了！-2


All Image Source


原来J哥在居酒屋昏迷后被人送到医院，没有钱结账想要赖账的他在逃跑途中被医院保安发现被追赶。闻讯赶来的郝英俊看到J哥轻松撂倒了自己几个同事，果断选择识时务者为俊杰。就在他要装做什么事儿都没有发生溜走的时候，嗜睡症爆发的J哥直接倒在他面前。


J哥再度醒来的时候，就发现郝英俊把他绑得死死的并且就在身边看守着自己。紧接着，曾经的超A级特工J哥在郝英俊露了两手，不仅使自己摆脱了困境还成功收获了一枚迷弟。在日本单独行动还不会日语还没钱的J哥，当即决定收下郝英俊这个partner。


在郝英俊的帮助下，两人成功混进目标任务会参加的慈善晚宴。就在J哥招呼郝英俊跟上大毒枭的时候，两人变成了聚光灯的焦点。原来，他们冒充顶替的两夫妻是舞者，被邀请来晚宴表演节目的。


根本不懂表演艺术的他们自然是无乱跳了一通，被愤怒的晚会负责人从晚宴上赶了出去。


求表现心切的J哥果断决定放弃郝英俊，把他丢在外头后自己顺着通风管道爬了进去。在外头等了很久都不见partner回来的郝英俊也爬进了通风管道，却意外地发现提前出发好久的J哥因为太胖被卡在了通风管道之中动弹不得。


为了帮partner解除困境，郝英俊用力过猛之下不慎让两人跌进了一间密室，两人误打误撞之下竟然知道了下次毒枭交易的时间和地点。经历一些跌宕起伏的案件之后，J哥和郝英俊成功将这起毒品交易给拦下了，还受到了日本电视台的大肆表扬。


因为这次交易失败，毒枭大老板麦天佑（郭京飞 饰）损失巨大。为了夺回失去的毒品挽回损失，他下令让手下把郝英俊的妻子金秀珠（李成敏 饰）给抓了回来。同时他放话给郝英俊，要是他想要赎回妻子就带着毒品到指定地点来做交换。


J哥和郝英俊搞到了电击枪偷了老板的法拉利，变装超级特工的他们向着交易地点驶去。一番搏斗之后，他们俩最终用自身傲人的体重成功“拿下”大毒枭麦天佑，郝英俊也成功救回了自己的妻子。


两人虽然成功的破获了这单A级case，却均有负伤。在身体康复以后，打从心里认定郝英俊已经是一名合格特工的J哥带着他去了自己的基地，却在那里见到了麦天佑身边的得力助手初夏（辣目洋子 饰）。


原来这次追踪大毒枭的卧底任务是初夏的，却在阴差阳错之下被J哥给完成了。随后郝英俊也被正式邀请加入了特工部门，电影到了这里也就画上了圆满的句号。

「Vicky's Food」巧克力爱好者的好去处 Awfully Chocolate（下） 190802

原本也是饭后的闲逛，不算特别饿的我们隔着橱窗打量起这些美食。有「巧克力甘纳许纸杯蛋糕」、「白巧克力纸杯蛋糕」、「海盐奶油果糖布朗尼」、「白巧克力瑞士卷」、「超级巧克力蛋糕」、「榛心巧克力蛋糕」、「白巧克力烤布蕾」等等。

没打算再吃如此高热量食物的我们，实在忍不住买了「超级巧克力蛋糕」和「榛心巧克力蛋糕」。下次，我们打算一定要试试看「白巧克力瑞士卷」！

同时，Awfully Chocolate也售卖巧克力球和礼品盒。

巧克力球有11种不同口味，除了常见的一些口味以外，更有一些很新奇的口味等着大家慢慢去发掘。礼品盒的售价自然不便宜，但冲着她们精致的包装和让人舍不得下嘴的设计，也足以让人乖乖掏出钱包来。

综合点评：味道好颜值高，甜品爱好者必去的一家店！虽说一块蛋糕要30多块钱，不过花几十块钱就能买到满满的幸福感，现在上哪儿才能找到那么划算的事情呀！

「Vicky's Life」从《失恋三十三天》联想到的相处之道 190802

最近自媒体平台保持热度的话题，大概就是马伊琍和文章离婚的消息了。两人因戏结缘，曾经合作过脍炙人口的《奋斗》和《失恋三十三天》。说起后者，常常被吃瓜群众戏称为「有毒剧组」，从主演到配角不少都闹出过或大或小的事情。

《失恋三十三天》中的黄小仙是个性格不那么柔软的女性，她感情专一却不懂得两性关系的相处之道。某天她直击了现役男友后备役老公出轨女闺蜜以后，备受打击的她在老板、同事的鼓励和陪伴之下慢慢走出来。电影的剧情不算很拔尖甚至有些老套，但看完电影以后我觉得颇有些感悟。

李可和魏依然这对，让我们见识到了快餐式爱情、交易式婚姻。女方要的是体面的生活，男方要的则是女性保持美丽并且不过问他的私生活。这样的关系在我看来有些病态，但不可否认生活中有不少人追寻着这种相处之道。

让我印象最深的，是黄小仙接的一对金婚客户。张老师在听了小仙和男友分手的事情以后，语重心长地对她说：「像买台冰箱，保修期三年。你嫁了个人，还能要求他一辈子不出问题啊，出了问题就修嘛！」

张老师的这席话，让我想到了马伊琍离婚这件事。从「且行且珍惜」到「此情有憾，然无对错，往后各自欢喜」，我看到了一个女性正确对待婚姻和家庭的处理方法。遇到了问题不是歇斯底里地抒发情绪，也不是一拍两散老死不相往来的气势汹汹，而是尝试地去倾听、去缓解/修复关系、去维系。

但如果这一切努力之后，两个人还是朝着不同的方向的话，果断地放手并祝福对方未尝不是一个好的选择。

「Vicky's Food」巧克力爱好者的好去处 Awfully Chocolate（上） 190801

本来以为到了万象城，我们一定会跑尝尝Lady M这家店的蛋糕。要知道这可是被誉为“甜品中的爱马仕”的明星产品，当然价格也是相当不菲就是啦。结果走着走着，我们看到了这家新开的甜品店。仔细一看，原来是前段时间风靡朋友圈让人争相去打卡的巧克力店：Awfully Chocolate

那么，今次就随着冰冰一起去看看这家店吧:D

店内的装修非常简单明亮，走进门左手边是产品售卖区，右手边则是餐桌。虽然缺少个性化的设计，但是客人走进店内就有想要走走逛逛的欲望。敞开式的店面，有点儿像是美式的咖啡厅的样子。

店内的墙上贴着Awfully Chocolate主打的几款明星产品，如果你不知道吃什么的话不妨在里面选择其一。

Awfully Chocolate是新加坡著名的巧克力品牌，店内主营巧克力及巧克力口味的蛋糕、冰淇淋、面包及甜品。早期的Awfully Chocolate只主要售卖三种口味的巧克力蛋糕，即「原味巧克力蛋糕」、「巧克力香蕉蛋糕」和「巧克力樱桃朗姆酒蛋糕」。

如果是巧克力爱好者的话，在店里转一圈一定会幸福感爆棚吧。
