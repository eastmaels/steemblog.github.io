【Vicky's 碎碎念】在简书平台第一次变现（上） 190727


All Image Source

简书作为国内比较大的全内容平台，已经运行了很长一段时间。在Steemit平台写文章之前，我主要经营的有个人博客和简书。简书有自己的一套生态体系和规则，使其得以在日益严格的国内内容审查制度之下依然可以正常运行。

让我最感到不可思议的是，上头很多专栏的管理者和编辑都是用户自发组织的（没有薪酬），不免和其他平台开口闭口就问好处形成鲜明对比。


All Image Source

去年下半年的时候，简书正式向区块链内容平台转变，提出了简书钻和简书贝的说法。一时间，过去写文章只看重阅读量和点赞回复量的简书作者们心态也开始改变。有利益的地方就有江湖，各路羊毛党们也涌入简书平台并把握了头部收益。

我在四个月前回归Steemit的同时，也重新开始打理简书、豆瓣、微博等诸多平台。做法无它，就是把过去发布的低质量、凌乱的文章全部删除，从头再来一遍。经过140多天的运营（日更），前几天发现我的总资产已经有1746.84个了，约等于尽400块大洋。刚好最近买了手机月供需要297月，这提现出来第一个月的钱不就有了嘛。


All Image Source

在提现之前需要做一下简单的解释，简书钻相当于是steem power，而简书贝相当于是steem/sbd。其实这个平台还有另一种货币，我没有用到就没有去做深入的了解。从简书钻变成简书贝需要13周，这和Steemit上的power down如出一则。其实之前简书推出白皮书的时候，朋友就说这套运行模式堪称翻版的Steeemit。

之前没有动提现的脑筋还有一个原因，就是变成简书贝以后还要变成另一种token，再拿去交易所交易。我是极其懒于去学习和注册新的交易所（至今还不会steem变现）的那类人，所幸的是现在简书有了内部交易市场，对于用户变现来说实在是个绝大的好消息了。

OK，那今天我就来试试看如何变现吧:D

「Travel & Food」两荤两素的完美搭配 190726


这几天换了新手机，顺便想要整理手机里上千张的照片。就像很多Steemit用户一样，我也会选择将图片整理好以后发布上来，此后就将照片从手机内存中delete掉勒。今天想推荐给大家四道菜，它们分别是：「土豆煲」、「凉拌秋葵」、「蒜香鸡块」和「香辣大虾」。

秋葵是非常富有营养的一种蔬菜，我在楼下小院子里辟了一小块菜园子刚有可以收成，所以最近吃的比较多。当然了，秋葵并没有那啥的作用，因此男性朋友们没必要逼着自己吃…土豆是我很喜欢的一种蔬菜，热量不算很高饱腹感还很强，只要不和蛋白质混吃的话个人觉得拿来减肥也挺不错的。

大虾的处理过程比较心酸…活虾的口感比较好，但是要活生生的挑虾线确实我很不愿意做的事情。于是我找了个折中的办法，就是将大虾稍稍过水以后再进行处理。可以做成油焖和香辣，个人觉得味道都超级好。鸡肉是我减肥的时候吃得最多的食物，除了水煮鸡胸肉以外我觉得任何做法都超赞的！

btw…这大概是我的厨艺巅峰时期了吧…

2、排版需要美观，增加文章的可读性。必要时可以使用与内容相关的图片来作为文章段落内容的切分，缓冲阅读时的疲倦感。

3、内容需要有理有据，条理清晰。鼓励答题者在写问答的时候，以总-分-总的格式。第一部分简单阐述一下对这个问题的理解，第二部就该问题进行分析，第三部分总结或针对该问题给出一定的建议或意见。


All Image Source

4、紧扣热点，结合实际。比如前阵子的宋慧乔·宋仲基离婚，不仅仅让自媒体作者们沸腾了（大料啊），连平时我们打开朋友圈都可以看到不少相关文章。遇到这种热点，最害怕的就是看到大量重复的内容，毕竟知识点和切入点就那么些。要破这个情况，除了在热点出现的第一时间就动笔开始琢磨怎么写，平时内容和话术的积累也是极为重要的。

5、巧妙地选择问题。建议答题者选择那些有一定回答量的问题，因为这样的问题往往是经过一定考量和验证过的问题。但不推荐大家去回答几千上万回答的问题，不容易被机器推荐入信息流读者也就看不到了。此外，如果是靠近热点的问答，答题者在回答的时候务必要check下该热点是否有超时。

【Vicky's 影评】《如懿传》：初看怒其不争，看完后才怅然若失，细水长流才是真正的好剧（上）


All Image Source


八年前《甄嬛传》开播，让孙俪、蔡少芬、蒋欣这些后宫娘娘的扮演者们着实红透了半边天，剧集中的金句更是频频出现在我们的生活之中。我因为某些原因终是回避了这部电视剧，倒是将流潋紫的原著找出来翻看一边，真正是最优秀的宫斗小说之一。


去年，她的另一部长篇宫斗小说《如懿传》也被拍成了电视剧。与《甄嬛传》的顺风顺水不同，《如懿传》一上来就深陷几番负面舆论之中。


先是关于原著涉嫌抄袭，将该剧推到了道德的风口浪尖。接着传出两位主演拿走了天价片酬，使得剧组在服装、道具、后期等资金不足。然后由于档期排在热门IP《延禧攻略》之后，被指出配色大红大绿、剧情拖沓不吸引人等问题。让观众印象最深的，还是一场关于“周迅扮演的女主角没有少女感”和“中年女演员的尬与惑”的讨论。


作为文字工作者，没有人比我们更讨厌抄袭这件事。如今原著小说我还在阅读当中并不好分辨什么，但我始终觉得将原著小说和电视剧混为一谈甚至殃及池鱼是不理智的行为。既然是改编作品，何不给电视剧一个相对独立的品评环境呢。


至于两位主角拿走了1亿片酬，我觉得也是理所因当。这部电视剧拍摄时长接近1年，两位主角的戏份之重也是有目共睹，更况论这部电视剧最终以13亿的价格卖出。我是周迅和霍建华的粉丝，虽不及死忠粉的地步，但这二人的影视剧这些年我看的也不在少数。可以说，他们俩在本片中的演技一定程度上有了新的提升。


关于《延禧攻略》和《如懿传》的对比，我始终觉得是一场闹剧。很多人喜欢《延禧攻略》，因为魏璎珞敢说敢做敢于向高于自己阶级的人说不，我想很多人看到她就想起了现实生活中不敢反抗的自己。但我们仔细坐下来想想看，生活工作中我们如果对上司对老板这么怼天怼地，会是怎么样的结果呢？


魏璎珞这个性格和行事的方式方法，放在皇宫里根本活不过一集。我身边看过这部电视剧的朋友说戏称，仿佛开了挂一样的魏璎珞自带主角光环。爽则爽以，但在现实中基本是不可能发生的。

一般来说，越是符合上述要求的问答在红包现金分成的时候得到的也就越高。很多平台都会鼓励内容创作者持续地提供优质内容，在系统后台机器会识别这类账号为重点关注信任账号。这不仅仅可以得到更多的推荐，也可以使得作者得到更多切实的好处。

「Travel & Food」在西藏旅行途中的一些趣事#3：洁癖患者大改造 190725


「片段三」：洁癖患者大改造

二十多年来，我都是个精准的洁癖患者，没想到这么一趟西藏之行居然还治好了这个毛病…

徒步的路上自然没有什么公共厕所，能遇到个带着顶棚的简易蹲坑就不错了。记得有一次，我上完厕所出来看到地上一片白嘟嘟的东西。那时候眼睛还没有做激光，没戴眼镜的状态下可以说看谁都是帅哥美女，于是好奇心驱使下的我蹲下身子去看。嗯…这是我人生中第一次见到活的蛆，还是一大片:D

尖叫自然是必不可少的因素，不过适应力超快的我马上就可以做到在这些家伙间找到一块脚尖踮地就可以通过的路径。就算真的踩到上面，除了对他们深表歉意一样也不再会惊慌。

在德钦出来前往芒康的路上，我们经过一个感觉是大货车补给站/过夜的中转站的乡。名字叫什么我不甚记得，印象最深的是我遇到了床上有跳蚤（活的）的情况！这一夜自然是不敢睡了，坐在椅子上靠了会儿。听着楼下彻夜熙熙攘攘的人声和大货车经过的声音，总也害怕会有人破门而入。

第二天太阳还未升起，我就背着行李逃也似的退房离开。这时候，我还没有遇到后来一起出生入死的驴友小伙伴们。
