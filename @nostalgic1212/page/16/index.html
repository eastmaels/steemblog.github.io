「Vicky's 短评」《零的执行人》：变成科幻动作大片的柯南，还是你爱的那一款吗？

All Image Source

朋友里有柯南的死忠粉，在闲谈间提起了几年前的剧场版《零的执行人》。用她的话来说就是，这是最不像柯南的「柯南」。

近几年来，《你的名字》、《天气之子》、《火影忍者剧场版：博人传》等日本动画片引进大陆影院，大多受到了不错的反响。

长篇动漫《名侦探柯南》剧场版《零的执行人》（即：M22）在日本国内创下了历史最高记录，在中国大陆上映仅七天票房也已经过亿，并在日本本土最终以90亿日元完美收官。

虽然M22在票房上取得了不错的成绩，但是口碑上却出现了严重的两极分化。支持者认为其是迄今为止最好的剧场版更是推理的回归，持否定态度的观众则是觉得其不再是侦探推理片而是动作科（魔/玄）幻大片。

如果你只是想看一部爽片，那么M22是个不错的选择，里面的动作戏和爆破场面非常精彩。

但如果你想看懂，那恐怕有些费力。

片中会出现警视厅、警察厅以及他们下设的警备局、警备企划课、公安部、刑事部等等科室部门，因此这部剧场版被戏称为「又一部只有日本人才看得懂的SP」。

值得一提的是，《零的执行人》的并不是围绕柯南展开的，而是一个名叫「ZERO」的人。他的本名叫「降谷零」，于第667话结婚前夜这一集中初次登场。

他以「波本」之代号潜伏在黑衣组织内部做秘密搜查官，又用「安室透」这个名字在毛利侦探事务所楼下的波洛咖啡厅里打工，同时还是一位米花町侦探。最后，他还是隶属于警察厅警备局企划课的公安警察「Zero」。

在本部剧场版之后，安室透和怪盗基德一样成为非常驻角色中的台柱子，非常具有观众缘唷:D

【Vicky's 影评】律师与犯罪嫌疑人初次交锋，谁是凶手，犯罪动机又是什么？

All Image Source
文/冰冰

嫌疑人三隅高司（役所广司 饰）被控抢劫杀人，重盛朋章（福山雅治 饰）作为辩方律师开始对真相展开调查。

在与三隅的沟通下，重盛朋章得知道了几个讯息。

1、三隅在案发当天喝了很多酒，在醉酒后冲动作案杀死了老板；

2、三隅是在老板死后，抢走他身上的钱包并烧毁尸体。

重盛朋章回办公室后，和同僚们商量如何对这个案件进行辩护。

他们认为如果三隅是因为故意抢劫而杀人的，那么八成就是死刑（在日本，死刑犯非常少，但是如果是性质特别恶劣的案件的话也不排除会定为死刑的情况）。

但他们可以将三隅打造成一个饱受老板压榨的形象，心怀怨念的他在喝酒后才失控杀人。之后他才顺手盗走老板的钱包，并由于太过紧张才放火烧了对方的尸体。

如此这般的话，三隅就不是什么丧心病狂的杀人狂，而是一个可以让人接受的情绪失控误杀事件。

在这种情况下，三隅则不容易被判死刑。

【Vicky's 影评】神级导演是否走下了神坛？看看不一样的导演是枝裕和

All Image Source
文/冰冰

导演李安因为《双子杀手》口碑不佳，被不少影评人质疑，这位“华人之光”是不是要走下神坛。

看到这些报道的时候，我不免想起了曾经看过一篇名为《神级导演是否走下了神坛》的文章。

里面提到的，是日本知名导演：是枝裕和

说真的，我对这位导演的名字并没有什么深刻的印象。

他被人指出走下坡路，是因为电影《第三度嫌疑人》。网路上的争论实在太精彩，给了我去了解这个导演的契机。

翻看资料的时候，我惊讶的发现这位导演真的拍了不少口碑颇好的作品。

比如裴斗娜参演的《空气人偶》、阿部宽参演的《比海更深》和《步履不前》、福山雅治参演的《如父如子》、绫濑遥、长泽雅美参演的《海街日记》以及前段时间超火爆的《小偷家族》。

这些很棒的温情家庭片，突然让我有了好好看一看《第三度嫌疑人》这部电影的想法。

（本篇完）

「Vicky's Life」肖战的粉丝们，究竟触犯了什么底线？（下）

NO.3 是已经被逼到退无可退，原创作者们的创作空间

作为文字工作者，我简直不能再理解“退无可退”这四个字。

我在很多平台发布影评，每个平台的红线都不尽相同，每次修改文章的时候要斟酌再三。

哪些题材不能碰，哪些字眼要用别的词汇替代掉，哪些剧照不能贴上去…要是我真的很任性一把，结果可能就是不推荐、被限流、卡审，甚至是直接被删除。

渐渐的，我看了很不错的影片和剧集，想要写下一些文字和大家分享，却在刚打开编辑器后就关闭了。

为什么呢？

我担心自己辛苦码下的的文字不能被人看到，那写作的这些时间，我宁可拿来多看几本书多刷几部剧来充实自己。

文化的创作者们需要开放自由的创作空间，在一些平台上原创作者们已经发起了#创作不死#的话题。

我们不是要网暴谁，更没有想要什么人消失。恰恰相反的是，那是作为受害者的我们在自己最后的小糖罐被人无情夺走后，发自内心悲怆的恫哭和拼死的反抗。

艺术创作不是孤岛，它诞生于群体之中。敬文学，敬热爱，敬自由。

——AO3雨果奖颁奖词。

总的来说，粉丝引发了如此大的争议，不是偶然，而是必然。

如今顶流的粉丝们大多太年轻了，如果缺乏相应的规束和引导，“闯祸”也是迟早的事情。

这次被波及到的是肖战，也许以后还有张三、李四、王五…

我们需要的不是下场参加骂战，也不是资本入场去做公关、撤热搜，最应该去想是该打造怎样的明星、怎么样的粉丝以及怎样的星粉关系。

希望我曾经喜欢过的男孩能度过这次难关，毕竟我深爱的ao3和产粮的太太们，目测短时间内不会再回来了。

（本篇完）

「Vicky's 短评」《大人物》：优秀的本土化改编，国产警匪片就该这么拍！

All Image Source

近年来，国内翻拍风气渐长，其内容涵盖了电影、电视剧、综艺等方面。大部分作品效果都不尽如人意，比如前阵子被很多观众诟病的，翻拍《粉雄救兵》的国产某综艺。

当然了，我们也没必要那么悲观，翻拍作品中依然有不少成功案例。比如《超时空同居》、《重返20岁》和《找到你》，在票房和口碑方面都收获了不错的反馈。

今天想给大家安利的《大人物》，也是一部优秀的本土化改编作品。

其导演五百曾经执导和监制了大名鼎鼎的网剧《心理罪》、《白夜追凶》、《古董局中局》，有他操刀这部电影，某些程度上来说已经给这部电影定了一个基准分。

男主角之一的王千源，是演技在线的实力派演员，之前在《解救吾先生》中有令人惊艳的表现。

其他配角也一水儿的戏骨熟脸：潘粤明、乔振宇、刘敏涛、梅婷、王迅……就冲着这个主创班底，也值得我们把这部几年前的电影找出来看一看。

值得一提的是，无论是演戏还是导演都不及格的包贝尔，在《大人物》中贡献了职业生涯中演技最高峰。当然了，他与电影《老手》中饰演赵泰晤的刘亚仁，还是有一定差距的。

小包童鞋，继续加油，看好你喔。

我非常偏爱警匪题材的作品，《大人物》也从头侧面展现了基层民警之殇——工作时间长、收入低、社会地位低、没资源等等。

希望以后可以看到更多优秀的翻拍作品，在原作的基础上加入接地气的本土化改编，而不再是一味的生搬硬套。

（The End）
