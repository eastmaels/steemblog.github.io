「Vicky's 短评」《游戏之夜》：宅在家里无聊？来部家庭温情悬疑喜剧电影呗

All Image Source

去年9月，经典美剧《老友记》开播25周年，一时间网上掀起了一波怀旧的热潮。

今天想推荐给大家的这部《游戏之夜》，则被影迷们称作电影版《老友记》。

电影由约翰·弗朗西斯·戴利执导，他本人是热播美剧《识骨寻踪》的常驻演员，饰演心理学家Lance Sweets（小甜甜）这一角色。

马克斯和安妮都是顶级的游戏痴迷玩家，两人在某个游戏之夜相遇并相恋。

一天，安妮如往常一般约了几个好友和丈夫哥哥来家里举行游戏之夜。游戏正酣之际，他们意外的发现自己被卷入了一场绑架案中，耳奇怪的邻居加里似乎也牵涉其中…

作为一部惊悚片，它无疑是成功的。

在你以为“原来是这样”的时候，下一秒钟就会直呼“怎么会这样”。

作为一部喜剧片，它无疑是上乘的。

片中有些很奇怪的笑点，比如为了在加里的电脑里找人员信息却不小心将他做的关于妻子的“圣殿”搞成了谋杀现场，比如米歇尔以为自己出轨睡了名人丹泽尔·华盛顿实则只是给路人甲…

作为一部家庭片，它无疑也是满分的。

在布鲁克斯以为自己要被撕票的时候，双手抱头趴在地方对着同样姿势的弟弟说自己其实很羡慕他。他羡慕弟弟有正经的工作、美丽又有趣的棋子、一群志同道合的朋友。反观他自己，只是一个一事无成的骗子+loser（失败者）。

最后我想说，不要小看那些不起眼甚至有些讨人厌的朋友。就算你平时不想和他们打交道，也别对他们太苛责啦。

因为你不知道他们在千钧一发的时候，是推你一把让你跌进深渊，还是拉你一把救你于水火之中。

冰冰就废话到这里啦，一起来看电影吧:D

（The End）

「Vicky's 短评」《女生一生要害怕些什么》：有多少女性因为害怕，不敢结婚生育？

All Image Source

“你可以教孩子防备陌生人，提高警惕，但是不能让她怕穿碎花裙子，不然要我们干什么用的？

By Priest 《默读》

小说《默读》第二卷 亨伯特·亨伯特，讲述了未成年少女被拐卖、性侵、杀害的故事。

让人觉得背脊发凉的是，因为家庭及社会原因，其中一名女孩从受害者走上了施暴者的道路。这项犯罪甚至成为了“家族事业”，贯穿了祖孙三代人…

时至今日，关于女性受到侵害的案件依然不在少数。

新年伊始，ID为@窗帘它挡不住寒气的微博用户，发布了一则名为《我告诉你女孩子一生都要害怕些什么》的视频短片。

我几乎是哭着看这个短片的，很长一段时间想起来都止不住地瑟瑟发抖。

家长捧在手里视若掌上明珠的孩子，也许会成为幼稚园里老师虐待甚至猥亵的对象；

好不容易长大成人，在职场和生活中，依然会面对职业歧视、性骚扰/性侵等问题；

来自七大姑八大姨的催婚，比工资还来的准时准点。甚至有不相熟的人对你大放厥词，告诉你结婚生孩子才是“正确的道路”；

更可怕的是，这个时代居然还有人开设“女德班”，来宣扬他们所谓的婚姻四项基本原则：

打不还手、骂不还口、逆来顺受、绝不离婚。

这套言论最出名的“女德教母”丁璇，仅有初中文凭的说出了“女人最好的嫁妆是贞操”这种恶心言论。

这个短片最终还是逐渐消失在各大平台上，它的热度似乎还不及某些富豪结婚生子。

但我相信，这些愿意去做一些事的人，将带来女性的觉醒。

那些已经觉醒的女性们，将如星星之火一般，影响更多的人。

（The End）

「Vicky‘s Life」如果这场疫情是一场战役，那我们终将会胜利，大家加油！！

这个年过的格外tough，近几年锐减的年味已经不是最大的问题，而是这场所有人都始料未及的疫情。

看着朋友圈的朋友们，从7天大假变成超长假期，再到无休止的在家隔离。很多人从狂欢，到不解，到愤怒。

作为从过年就开始加班的一份子，我不明白这种不解和愤怒的源头是什么。

你们要知道，不让大家出门本质上来说是为了大家的健康和不让病毒交叉感染。大家想想看，如果你居家14天，期间不出门或者很少出门，没有被感染到病毒。那么在整件事结束以前，只要你能保持这个状态，你就是安全的。

有什么紧要的事情，是比身体健康更重要的呢？

别再和我说在家里好无聊想出门的了…这句话真的会被此刻在一线昼夜颠倒工作，连睡觉都是件奢侈的工作人员们唾弃的。

家里不好蹲吗？是电视剧不好看、小说不好读，还是游戏不好玩？

你们看看医院里医生护士们，他们在拿命工作！下沉到基层社区村舍的工作人员，他们也在冒着被感染的风险，严格按照要求工作着。

我不太明白那些一定要闯关出去的人，究竟有什么事情那么紧要。如果真的那么重要，早你们干嘛去了？

我也不明白那种在家里就要吵架家暴的，一家人在这种情况下不是更应该互相鼓励扶持，共同渡过难关吗？

已经那么艰难了，希望大家多一些包容，少一点戾气。

还有某个平台的娱乐领域，我真的不关心哪个韩星和谁交往了，也不想知道香港哪个富豪的给了儿媳妇几亿豪宅。

「Vicky's 影评」《我不是药神》：这个世界上只有一种病，那就是穷病

All Image Source

同时值得一提的是，尽管影片开头特意强调了本片是艺术创作，但其实所有人都清楚，程勇的原型真是当年热议的「陆勇案」。

终于，我们也有了一部真实事件改编的电影。

虽不能说它可以改变国家，但起码这是个不错的开始，不是吗？

最让我动容的是，《我不是药神》描绘出一副底层病患的群像。

在面对困难的过程中，底层人群互相帮助。他们组建了许多病友群，相互鼓励、同步信息，甚至可以一同分享价格不菲的药物。

如果说求生是一种自私的欲望的话，那么这些病患对生的渴望，维系着他们共同进退。

《我不是药神》是一部极具勇气的电影，看病难、看病贵是中国人的日常生态，却一直是电影人不愿去也不敢去碰触的敏感话题。

为了通过现有的电影审查机制，能够让一个团队人的心血最终可以与观众见面，电影必然是做了很多妥协的。

反过来说，《我不是药神》这个直指现实，揭露黑暗的故事能够过审上映，也代表着内地电影审查制度的进步。

希望这部电影会是一个契机，将来会有更多更好的现实向电影被搬上大荧幕。

（The End）

「Vicky's 影评」《我不是药神》：谁家还没个病人，谁敢保证自己一辈子不得病？

All Image Source

主导搜查的警官曹斌（周一围 饰），是程勇前妻的弟弟，也就是他的小舅子。

对于这位多次对姐姐动手，还阻碍小外甥出国的前姐夫，他厌恶至极。

一次抓捕搜查时，所有患者都不愿“出卖”他们的救世主。有位患病的老太太哀求曹斌希望他们不要再抓“药贩子”了，她期期艾艾地说道：

领导，我求求你了，别再查假药了行吗，这药假不假我们这些吃药的人还不知道吗？

我吃了3年正版药，房子吃没了，家也吃垮了。现在好不容易有了便宜药，可你们非说这是假药。

不吃药，我们就只能等死。我不想死，我想活着。

老太太的另一句话，更是让曹斌动摇到质疑自己的追捕行动是否是正确的：

谁家还没个病人，你敢保证你一辈子不得病？

作为警察，他必须要严守法力，不能意气用事。

但是，面对这些苦苦哀求的病患，他的内心震动了。随后，他去领导处请求允许他退出这次搜查。

看到《我不是药神》海报的时候，我第一感觉就是韩国电影《辩护人》。

海报上的演员们笑的眼睛鼻子都皱到一起，一度给人以一种这是一部喜剧片的感觉。

正所谓表面笑的多开心，背后哭的就有多惨烈。

整部电影强烈的情绪对比，使得剧情走到最后1/3的时候，整个放映厅内都萦绕着抽泣声。

（The End）
