「Vicky's 影评」《少年的你》：受害者、施暴者和旁观者相互转换

All Image Source

这个善意的举动，让陈念从「旁观者」转变为「受害者」，成为下一个被欺凌的对象。

椅子上的红墨水、体育课被排球砸中、课间被推下楼梯、放学路上被围追堵截…曾经胡小蝶遭受的，一一在陈念身上再现。

到这里，电影向我们展现了第二个事实：校园霸凌要从施暴者的角度来矫正，如果只是疏导和隔离受害者，很快会出现第二个受害者顶替上。

在这个事件里，以魏莱（周也 饰）为首的三个女孩是施暴者，但他们同样也是原生家庭的受害者。

因为霸凌事件被警方调查，三个女孩的家长都被叫来学校了解情况。

魏莱的母亲只会高傲地说自己女儿是不一样的，就算做错了事也只是因为交错了朋友。魏莱的父亲因为她高复整整一年没有同她说话，后来在太平间看到女儿的尸体，这个做父亲的也只是愤然离场。

而那两位女孩的家长呢，一位母亲只会哭着跪地请校方再给女儿一次机会，另一位父亲直接对女儿拳脚相加。

过于溺爱、期望过高或者充满暴力的家庭，对正在人格形成关键时期的孩子来说，其危害是不可估量的。

《少年的你》中的角色转换，除了上面提到的以外还有好几处。

三个女孩中的其中一个，因为对陈念的同情心，在欺侮的过程中放了水。因为些许的善意，导致她成了新一轮被欺凌的对象。

在小北（易烊千玺 饰）保护下的陈念，再次回到了旁观者的角度。有了胡小蝶的前车之鉴，又因为自己亲历了霸凌事件，让她这一次站在了那个女孩子的边上。

那个女孩为了摆脱受害者的身份，出卖陈念将她引到了小巷之中。她从受害者再次回到了施暴者的角色，而陈念也再次从旁观者变成了受害者。

兜兜转转，暴力升级，也就有了后来的悲剧。

为了保护陈念的小北，从旁观者变成了施暴者；错手杀了魏莱的陈念，从受害者变成了施暴者；而施暴者魏莱呢，却在电影的最后变成了受害者…

（The End）

「Vicky's 影评」以校园霸凌亲历者的眼光，告诉你《少年的你》优秀在哪里

All Image Source
文/冰冰

有一部电影，经历了入围柏林电影节，却因“后期制作原因”退出展映。

也曾定档暑期，却因为“制作完成度和市场预判”问题，距离上映前三天临时撤档。

四个多月过去，官微毫无预兆地宣布定档10月25日上映，仓促地甚至连一张新海报和预告片都没来得及发。

上映当天，就成为了年度爆款。

看到这里相信大家也一定猜到了，就是这部命运多舛的国产青春现实电影

在观影的过程中，我身边不少观众从起初小声的啜泣到猛吸鼻涕的抽泣，似乎有一种情绪在放映内蔓延开来。

不是所有人的青春都在阳光之下，而《少年的你》打开了他们尘封多年不愿回忆的年少时光。

我在走出影院的时候，喉头发紧，眼泪直流。

回家以后打开电脑，准备整理写下这部电影的观后感，却千头万绪久久不能落笔。

这是一部值得二刷，但我却不敢二刷的电影。

而我，将以一个校园霸凌亲历者的角度，谈谈这部电影。

「角色相互转换的施暴者、受害者和旁观者」

我在写《我们与恶的距离》的时候就曾说过，要讨论一个事件大多绕不过这三个角色。

《少年的你》的编剧，另辟蹊径地并没有用这三个身份来框定角色，而是让他们间彼此转换。

此刻的你也许是天使，但下一个转口也许就会变成恶魔。

受害者胡小蝶从教学楼一跃而下，一时间学校的天井似乎成了「动物世界」。

同学们是事不关己的旁观者。他们围观讨论，还看热闹似地拍下照片和小视频上传到社交网络上。

陈念（周冬雨 饰）曾经也是旁观者的其中一人。胡小蝶之前向她示好希望彼此能够成为朋友，甚至在跳楼前向陈念含泪控诉“为什么你不阻止她们欺负我”。

陈念，就像很多其他旁观者那样，害怕霸凌延伸到自己身上影响学习，选择无视了对方的这份求救。

虽然我们不能说陈念是压死胡小蝶的最后一根稻草，但也说明了一点事实：很多时候，受害者的悲剧是施暴者和旁观者共同作用下的结果。

因为内疚，陈念在同学们都在看热闹的时候，只身一人走向前脱下校服，盖在了躺在地上的胡小蝶身上。

（The End）

「Vicky's 短评」《少年的你》：撤档、争议、逆袭！少年的你，太猛了！

All Image Source

截止到11月5日下午，豆瓣上共有47.2万人标注看过，有11.2人标注想看。

豆瓣评分8.4分，票房已达11.89亿元。

从普通观众到专业影评人，口碑几乎成一边倒的好评。

不出意外的话，《少年的你》可以被列入「年度十佳华语片」。

故事发生在高考前夕。

一位名叫胡小蝶的女生突然跳楼身亡，事件最后被认定为自杀。想要找出源头的班主任被迫离职，知道内情的同学们也都噤若寒蝉。

陈念（周冬雨饰）于心不忍，脱下校服替胡小蝶盖上。正是这个些许善意的举动，使得她成了下一个被校园霸凌的目标。

这时候，距离高考还有42天，却成了无比漫长的地狱…

大多数人对这部电影的期待，大多来自《七月与安生》的原班团队。

导演曾国祥的代表作《七月与安生》，向观众展示他出色的小说原著改编成影视作品的能力。这部作品不仅让他被更多内地观众所熟知，也创下了金马奖史上第一个影后“双黄蛋”。

《少年的你》沿用了《七月与安生》的主创团队，男女主角分别是影后周冬雨和第一次登上大荧幕的易烊千玺。

观影的那天，说真的我对这类敏感题材的电影并没有抱有多大的希望。由于审核的关系，这类题材的电影通常妥协下来就变成了诸如《悲伤逆流成河》这样的爱情片。

心情平静的走进放映厅，我却在里面泪流满面。不过并不用觉得丢人，身边偷偷拭泪的人大有人在。

强烈推荐大家走进影院去支持这部电影，它是国内真正意义上的首部校园霸凌题材的电影！

（The End）

「Vicky's 短评」《双子杀手》：下一次，李安的粉丝还会为他买单吗？

All Image Source

美国国防情报局特工亨利（威尔·史密斯饰）在完成一个刺杀任务后，职业敏感性让他意识到这里面并不简单，于是萌生了退役的念头。

也许是唯恐他将国防情报局那些见不得人的秘密抖出去，刚到达休整地的亨利就发现组织派人在监视自己。随之而来的，是一个神秘黑衣人的疯狂追杀。

在激烈的交手中，亨利发现对方居然是年轻版的自己。组织在未经他同意的情况下，利用亨利的DNA暗自培养了一个他的克隆人。

就此，一场我与“我”的对局旋即展开。

李安在介绍《双子杀手》的时候称，这部电影多多少少还带有一些“自我”的主题。我想，导演想要借由电影去弄明白，究竟是什么让我成就了“我”。

自然人和克隆人相比起来，都可以经历生活的喜怒哀乐、受到高等教育，也都要受到社会制度和法律的制约。

人之所以为人，这个标准究竟是什么？

这似乎，成了哲学范畴的问题。

再说回技术层面，《双子杀手》中的小亨利是个完全的虚拟人物。这里面不是简单的换脸，而是更为先进的一项技术。

从《双子杀手》的完成度来看，这项技术已经相对成熟。

这就意味着，团队可以做出任何需要这个角色做出的表情和动作；

这就意味着，团队甚至可以不启用真实的演员，全靠尖端技术拍摄任何影片；

这就意味着，一部电影以后都不需要什么外景和影棚，可以直接用电脑完成拍摄。

……

李安，这个65岁的“老顽童”，出于对电影的热爱再次走上了很少有人踏上的探索之路。

他的下一部电影，你还会买单吗？

（The End）

「Vicky's 影评」《陈情令》：极致匠心书写国风之美，彰显传统文化自信#5


All Image Source


「出色的宣传运营能力」


《陈情令》出品方在宣传和热度维持方面做的十分到位，官方微博的小编更是展现了出色的运营功底。


在剧集播出的过程中，各种宣传物料多得让让书粉和视频制作者们直呼“逼死同人系列”。甚至还大有走上官方沙雕之路的趋势，各种衍生话题源源不断。


在剧集播出结束之后，每逢演员过生日或是该剧有新的动向，官微都会继续放出物料，吸粉及热度堪比二三线明星。


其实，出品方新湃传媒在买下《魔道祖师》本子的时候就做好了长线打算。


系列花絮和番外“陈情令花絮之‘陈’语大全”、“陈情令之还原名场面”在腾讯视频上部分是采用付费收看的方式播出的，《陈情令》的国风音乐专辑在相关平台同样采用付费播放的模式。


著名时尚杂志《时尚芭莎》为几位主要演员拍摄了时尚大片，粉丝们争相为偶像买单。其他相关产品，如口红、首饰、服饰这些并不便宜的周边产品在官方平台上屡屡售罄。两位主角所代言的产品，更是在最近购物节中深受粉丝们的追捧。


不得不说，热门剧集的衍生产品与剧集本身是相辅相成的关系。影视剧的热度可以激活粉丝经济，带动他们强大的购买力。粉丝们付出的心甘情愿，一整条产业链上的相关人员也可以赚的盆满钵满。


与此同时，层出不穷的衍生产品也会让《陈情令》在很长一段时间内持续发力。


《陈情令》能在口碑和评分上逆转，绝不是简单的“卖腐”那么简单。


魏无羡和蓝忘机纯粹相信彼此的同袍之情，“能够一生锄奸扶弱，无愧于心”的赤子之心，面对危难时能不忘初心的大仁大义，这些都很容易让当下年轻人产情感和审美上的共情。


《陈情令》对中国传统文化的尊崇是值得肯定的，彰显了国人对国风的自信和热爱。文化自信是一个民族、一个国家对自身文化价值的充分肯定和积极践行，并对其文化的生命力持有坚定的信心。


这个夏天，《陈情令》通过书写国风之美、传递中国传统礼仪、展现做人大仁大义，为观众们带来了一部有内容、有深度、有价值的优秀剧集。


毕竟，内容为王的理念在任何时代都不会过时。

「Vicky's 影评」《陈情令》：极致匠心书写国风之美，彰显传统文化自信#4


All Image Source


「极致匠心书写国风之美」


原著《魔道祖师》属于架空小说，服化道等方面并无相关朝代可以借鉴。但观众在看《陈情令》的时候不难发现，这个年轻的团队花了很多时间精力去设计和制作这一切。


造型师在尊重原著的基础上，结合了中国传统服饰文化（如：宽衣薄带），打造了这一系列长线型飘逸的戏服。团队希望通过减弱剧中角色衣食住行这些烟火气十足的行为，赋予角色们符合仙侠剧的“仙味”。


五大仙门世家的选址极具地域风格，建筑物上皆有家徽。


弟子门生所穿服饰所用物件上均有家纹，如代表姑苏蓝氏的卷云纹，代表云梦江氏的九瓣莲花纹，代表兰陵金氏的牡丹纹，都是古代传统纹饰。这样的好处就是，即便来人没有自报家门，观众也可以一眼看出哪些人是来自同一家族的。


本剧大多数场面都采用实景拍摄的方式，使得全剧色调自然舒服，也减少了近些年来观众们被绿布抠图支配的恐惧。




All Image Source


姑苏蓝氏


蓝氏一族住在姑苏云深不知处，细究起来当是如今的苏州。外景卧松、绿竹、飞瀑、细流，内景则是细格窗棱、青纱帐幔，让观众仿佛置身于江南园林之中。


擅音律重规矩的蓝氏在服装上整体的基调是青色和蓝色，一眼看去就是清冷。家徽是是一卷云纹，门风只二字，雅正。眉上皆束有抹额，非父母妻儿不可触碰。


蓝忘机正是蓝氏一族的典范，服饰就多以蓝色、白色为主。




All Image Source


云梦江氏


魏无羡及其同门师弟江澄和师姐江厌离，均是来自位于湖北江汉平原一带的云梦。


云梦江氏整体的气质更开阔，家风偏向游侠式的自由感。在场景设置方面，建筑多以偏曲线形、暖色调为主。


服装师在设计这个家族服饰的时候，更多采用了淡紫色作为主色调。材质多用棉麻，服饰运用窄袖设计，给人以舒适、生活化的感觉。


前期魏无羡性格灵动活泼，服饰就以暖白色为主，加入草绿、压簧，贴近整个江氏家族体系。从乱葬岗归来后，该角色亦正亦邪的特性更加明显，服饰就转变为以黑色、红色为主。

兰陵金氏


这一族无论是出场还是所住的金麟台，无不透着“阔气”二字。


无论是家主金光善还是其族人金子轩、金光瑶、金凌，身着金色衣服，皆在领口和袖口处用金线绣着繁复的暗纹。其家徽更加简单直白，一朵盛放的牡丹。


用于夜猎的缚仙网价格不菲，金氏小公子金凌一出手便是400张，真正的大户人家。
