【Vicky's 影评】《准许》：开放式关系没那么滥交，它比传统两性关系更有原则#2


All Image Source


当我们说起开放式关系的时候，人们大多以“水性杨花”、“私生活混乱”评论之。我在写这篇文章之前和身边的亲人、同事及朋友就该问题做了小采访，他们给我的答复主要集中在“男人这样子还可以，女人这样子有点不检点吧”、“交往的时候可以多选择，婚后绝对不允许”、“周围的人要是知道了怎么办”等方面。


这也足以见得，社会上对这种多偶关系理解并不到位，甚至是带有一定程度的偏见的。在和身边处于多偶关系的朋友聊了许久以后，得到了意料之外的不少干货。原来，开放式关系并不等于滥情，这其中要遵守的原则性问题比很对一对一的关系更为严格。


「什么是开放式关系」


开放式关系（Open Relationship）是一种非单一型的亲密关系。


这种关系建立在相同的基础上，任意一方都享有在对方知情并许可的前提下与第三者发生亲密关系的自由。另外一种“开放式关系”是指双方目前没有其他伴侣的情况下，同意保持传统柏拉图式的纯友谊或是含有性行为的炮友/性伴侣关系。


用更为通俗的语言来说，他们并不全盘接受现代两性关系中一夫一妻制或恋爱关系，不要求另一半对自己绝对忠诚。在彼此坦然说出自己的想法的前提下，双方就相处模式达成共识，在保持彼此关系的前提下双方都可以和他人发生性关系。

「Vicky's 短评」《杀人回忆》：时隔28年后，真相大白，终于可以让逝者们安息了… 190920

前天深夜，韩国警方公布《杀人回忆》凶手原型确认被抓，这条消息在中国瞬间登上热搜排行版首位，足以见得该案件在亚洲地区的影响力之深远。

由奉俊昊导演，宋康昊主演的《杀人回忆》于2003年上映后引发民众热议。

这部以真实事件改编的电影，向观众们展示了「华城连环杀人案」这起韩国历史上最诡异的刑事案件。

10名被害人均为女性

最小的14岁，最大的71岁

雨夜、红衣、长筒袜、碎桃块、文具…

残忍的手法和诡谲的氛围，很长一段时间都让我雨夜不敢外出。

「华城案」得以告破，与舆论的关注度休戚相关。

无论是《杀人回忆》还是后来基于本案改编的《信号》和《岬童夷》，使得民众即便是诉讼期已经过去，依然对该案件保持相当的热议度。

这点，也促使了警方长期就此案有专项调查组。

DNA，即便是过去10年还是20年，只要能找到比对型就能为案件找到突破口。

科技的改变，也成了该案得以告破的另一个关键点。

在电影《杀人回忆》的最后，奉俊昊导演在结局处做了如下安排：

宋康昊面对着镜头双眼含泪，凝视了整整6秒，没有眨眼…

他们希望以此让正逍遥法外的凶手胆寒，让那些模仿犯们在黑暗中瑟瑟发抖。

正义虽然迟到，但终会来到。

【Vicky's 影评】《准许》：开放式关系没那么滥交，它比传统两性关系更有原则#1


All Image Source


一般来说两个人恋爱/结婚了好多年，或多或少都会有想要尝试更多相处模式的想法。这里面有些人会选择分手另选他人，有些人会选择隐瞒另一半的情况下找小三小四小五。有一种比较特殊的情况，则是在配偶双方都认可的情况下与他人交往，这也就是我们平时略有耳闻的“开放式关系”。


前段时间我看了一部名为《准许》的电影，就是以这种两性相处模式为主题的。


影片的女主角安娜（丽贝卡·豪 饰）和威尔（丹·史蒂文斯 饰）是一对从学生时期就开始交往相恋多年的情侣。他们是初恋情人又都是对方的初吻和第一次，关系十分稳固。美中不足的是，也许是相处（睡）了太久，两人的sex life缺乏激情（起码女方是这么认为的）。


就在两人筹备婚礼的时候，安娜按耐不住早在自己心中百转千回的念头，向威尔提出自己想要开放式关系。内心百般不愿意的威尔为了让安娜开心，答应了未婚妻的请求并在她“出战”的第一夜鼓励她放手去做。在那以后，威尔也因为工作的关系遇到了一个富婆。


就这样，即将结婚的二人展开了一场各自狩猎之旅…

「Vicky's 短评」《我是大哥大》：社畜太苦？不如做个沙雕青年吧！ 190918

说起高校的不良文化，你会想到什么影视剧？

《热血高校》和《极道鲜师》，想必是大家最先在脑海中浮现的名字吧:D

去年一部名为《我是大哥大》的日剧在网路上爆红，主配角们明明都是颜值担当，但为了这部剧也算是豁出去了。

贺来贤人饰演的三桥贵志一头浮夸的黄色卷毛·，贱贱的表情和土拨鼠式的咆哮成了他的招牌动作；伊藤健太郎饰演的伊藤真司，那海胆头让观众们不由得怀疑他是不是洗剪吹从业者；桥本环奈这样的颜值，在里面成了随时翻白眼爆粗口的女高大姐大…

除此以外，这部剧里面的配角也是众星云集。

在让腐女们尖叫不已的《老司机的爱情生活》出演的猪塚健太，《大叔之爱》中饰演部长的吉田钢太郎，被人称作佛祖的佐藤二朗还有参演了《人生I字路》、《重版再来！》、《大恋爱：与将我忘记的你》的室毅。

更别说，还有小栗旬和成田优也来客串了一把。

沙雕的剧情、浮夸的演技、密集的笑点让《我是大哥大》瞬间火爆网路，成为观众们的下（pen）饭神剧。

如果你还没有看过这部剧的话，速度刷起来喔！

「Vicky's 短评」《他人即地狱》：韩版《轮到你了》，男神荧幕首次饰演变态，太带感！190917


热播韩剧《鬼怪》不仅让孔侑/金高银这对CP为观众所熟知，也让李栋旭/刘仁娜这对荧幕情侣大放异彩。在此之后，二人更是在《触及真心》里面让观众吃足了狗粮。

最近，冰冰又挖到一部由栋旭主演的韩剧《他人即地狱》。

故事根据金容基的网络漫画《惊悚考试院》改编，以考试院为背景。

什么是考试院？

这可不仅仅只是提供给学生们备考的地方，某些程度来说有点像是外来务工者临时落脚的地方。在很多城市的城郊，有些当地农民会将自家小楼中的以较为亲民的价格租赁出去。根据地段和配套设施的不同，价格也有较大的区别，这大概就是中国版的“考试院”吧。

从仁川到首尔生活的少年尹钟宇（任时完 饰）囊中羞涩，不得不一再调低期望值找到了勉强可以落脚的考试院。皮笑肉不笑的房东、虐猫为乐的双胞胎、一看就很危险的强奸犯，还有故意接近尹钟宇的牙医徐文祖（李栋旭 饰）…

里面每个人看起来都神神叨叨的，而危险也正在一步步逼近钟宇。

《他人即地狱》（以下简称《他人》）被称作是韩版《轮到你了》，区别在于后者让观众去猜谁是凶手，而《他人》一开始就告诉大家大boss是谁。本剧从第一集开始就营造了黑暗压抑的气氛，一副人间地狱的景象毫无保留地在观众面前铺陈开来。

冲着已经放出的4集，这部8.4分的韩剧我追定啦！

【Vicky's电影/电视剧推荐】新垣结衣上演乒乓版《摔跤吧爸爸》，这部电影既热血又治愈！


All Image Source


如果说刚开始重拾球拍是为了和渣男前任赌气，那么这一次多满子是为了找回自己人生的意义。思考清楚以后的她，开始全身心的投入到比赛的准备当中去。在比赛中无论怎么对对方碾压，他们都励志要拼劲自己最后一点力气。当落败的时候，众人互相打气的场面也温馨的让我一度落泪。


多满子和萩原久为了更大幅度的提高自己的乒乓球技巧，主动去找周围的高手挑战。有酒吧奇奇怪怪的女装大佬，也有学校里的兵乓球小高手。当他们击败最后一个之前无情碾压他们的高手邻居，众人来到平常吃麻婆豆腐的中华料理店。


这才意识到，老板张（森崎博之 饰）和店员杨（苍井优 饰）居然是中国国家队运动员。两人只是因为没有入选一线队伍，在退役后背井离乡来到日本开了家料理店的。这里要大赞一下苍井优的演技，平时看起来柔柔弱弱森女风的她，在这里演出了中国女乒的霸气。这一段，应该是导演对中国乒乓球的致敬片段。


在张和杨的严厉训练下，多满子和萩原久的乒乓球技术有了更进一步的提升。随着黑板上记着比赛日的倒计时，两人解开了对对方的误解并渐渐情愫暗生。


终于到了比赛日，多满子和萩原久一路过关斩将，终于要在球桌上正面迎战渣男前任组合。一场本来大家以为会在渣男前任组合会压倒性胜利的比赛，在多满子和萩原久的顽强抵抗下显得精彩异常。最后，导演并没有落入俗套，而是让多满子以一分惜败给了对方。


全场观众并没有为胜者喝彩，而是纷纷起立为斗争到最后一刻的多满子和萩原久鼓起了掌。那一刻，在观众们的心里，他们才是这场比赛真正的胜者，也是这个球场上真正的王。


电影也在这里接近了尾声。


虽然只得了第二名，但重拾球拍的多满子突然明白了母亲当年的执着和期望，她开始认真经营起母亲留下的花儿俱乐部。于此同时，她还收获了萩原久“患难见真情”的爱情。


弥生经过这次的比赛，与丈夫也进行了有效沟通，丈夫从过去的不理解不支持到最后变成弥生的忠实粉丝，她终于过上了和富太太们背道而驰却快乐的生活。农场夫妇继续兢兢业业地经营着农场，还说要不断扩大规模。那个有些奇怪的高中生，则成了俱乐部的经理人。


生活中总是有各种磕磕绊绊等待着我们，但大家一定不要惧怕和气馁。毫不松懈地持续去拼去抗争，总有一天生活会以我们意想不到的方式反馈给我们。
