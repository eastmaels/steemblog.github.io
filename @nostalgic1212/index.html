「Vicky's 影评」《少年的你》：大众将影视作品与原著绑定讨论，这么做是否合理？

All Image Source

《少年的你》确实改编自小说《少年的你，如此美丽》，但原著小说在三位编剧的鬼斧神工之下已经被改的“面目全非”。

电影不仅仅将原著中形象扁平化的陈念、小北、魏莱等角色塑造得更加立体，更为他们添加了所需的人物背景，让观众在观影的过程中产生“XXX会变成这样，原来是因为…”的感觉。

在出场人物和剧情上，编剧们不仅仅做了加法，更做了很多减法。网络文学中常见的冗长的“废话”被删除，对主线故事帮助不大的人物也被删减或者合并，让整部电影在紧扣「校园霸凌」这个主题的基础上显得更为合理。

说了一大段空泛的话，我想还是在这里用个比较具体的例子来说明。

在观影的过程中，难得让放映厅笑成一片的，是一个看起来有点污的段子。

陈念因为被欺负，跟着小北第一次回到他家。小北进家门就迅雷不及掩耳地把散落四处的衣服收拾起来，是一个青涩的少年特有的慌张。

这一夜，陈念洗了澡穿着小北的衣服睡在床上，小北和衣睡在沙发上。两个少年都对屋里的气氛有些似懂非懂的感觉，这时候陈念开口问了句：硬不硬？

小北反应过来以后，嘴角带了一丝笑意地半起身子：啊？

陈念突然意识到这么问实在是太暧昧了，于是加了句：我是问你沙发硬不硬。

听到这句话，小北略显遗憾地答了句：还行。

原著小说中也有类似的文字，冰冰截取了了部分放上来给大家看看。

陈念咬一口，蓬松温软，奶香四溢。她身体猛地一僵，北野的手从她裙摆下探进去，沿着大腿内侧的肌肤往上摸。
　　
陈念扭头，与北野的目光相触，他的手在她裙下得寸进尺，低声问：“害怕么？”
　　
陈念躲避着踮起脚尖，他的手尾随而上。她发着颤，眼珠一转不转盯住他，懵懂而惶惑。

一时间，那个青涩的少年荡然无存，剩下的只是一个小混混形象。

而后来会为了陈念顶罪坐牢的小北与那个小混混的形象格格不入，不得不说，这里《少年的你》的编剧们改得非常妙。

电影中这样的细节改动非常多，无比包含了编剧们的心血。

在我看来，即便是原著真的抄袭或者融梗，电影也不应该被网友们拿来diss。当然，法律方面的责任分割不在这篇文章的讨论范围之内。

我知道有些观众会担心，如果一个抄袭或者融梗的作品都可以被拿出来翻拍，一不小心居然还成为了爆款，是否会形成一种行业潜规则，让越来越多的人去仿效。

如果你有这个担心，请尽可能多的收集相关证据并组织起来，向相关部门去提交，专业的事交给专业的人去裁定。在法律作出裁定之前，先别急着赌咒别人，更不应该去做殃及池鱼的事情。

退一万步说，如果原著真的抄袭，电影方本身也是受害者，并不应该受到道德的绑架和审判。

（The End）

「Vicky's 影评」《少年的你》：原著《少年的你，如此美丽》，是否真的存在融梗现象？

All Image Source

说真的，我在看电影《少年的你》的时候除了几度被激到哭得上气不接下气以外，真的没有感觉其内容和《白夜行》以及《嫌疑人X的献身》有什么关系。

虽然我没有看过《少年的你，如此美丽》，但我是东野圭吾的书粉。他的书我几乎都有看过，很多影视化的作品一眼我就能看出他的风格来。

我也看过很多在拿融梗说事的评论，主要集中在两点：

1、小北和陈念一个走在阳光之下，一个坠入无尽黑暗，极度类似于《白夜行》的经典桥段；

2、小北为了让杀人的陈念脱罪，为她做了不在场证明，这和《嫌疑人X的献身》中的高潮部分也非常相似。

但是，经典桥段不是仅有一人能想出来，也不是有作者用了以后别人就不能再用了。

融梗的标准是什么，又有谁可以给出明确的定义呢？

融梗和抄袭的界定是存在一定标准的，这方面我不是专家，在这里就不做冗述。

我只是想说，当你拥有相当大的阅片量和阅读量的时候，就不难发现，无论是影视剧还是书籍的创作中，借鉴（融梗）的情况真的非常常见。

比如第38届香港电影金像奖《我不是药神》热映的时候，就有网友指出其部分情节与《达拉斯买家俱乐部》相似；

陈思诚导演的《唐人街探案2》，就不止一次被人说起和电影《双瞳》如出一辙；

去年由周润发、郭富城主演的高分电影《无双》，与1995年拍摄的《非常嫌疑犯》多处相似，虽然导演出面澄清自己并没有借鉴；

……

这样的情况，相信在小说中会更加常见。

我想，现在无论哪个自媒体都不敢说《少年的你》原著抄袭，这可是一不小心会被起诉的大事。

专业的事应该交给专业的人去做，我们在没有足够的知识量的时候不应去做过多的揣测，也是身为自媒体人应有的素养。

而我们作为普通读者和观众，欣赏眼前的作品，能够就事论事的谈眼前的作品就最好不过了。

当然，我们也非常呼吁相关部门可以出台关于借鉴和融梗的标准，让很多似是而非的争论有个标尺。

（The End）

「Vicky's 影评」《少年的你》：电影口碑炸裂，原著却深陷融梗风波

All Image Source

4日，《少年的你》原著作者玖月晞发长文回应抄袭争议，表示无论人物、主题还是背景都和任何一本书不一样。

“我从未抄袭，也从未违背一个文字工作者应有的初心、原则和底线。我不认同网友所说的“融梗”指责，我的作品中或许有着共通的思考，但没有任何抄袭融梗。” ​​​​

是的，随着《少年的你》口碑和票房爆红，其原著小说《少年的你，如此美丽》，原文涉嫌抄袭。

这几年，冰冰都不太看现代题材的小说，因此对这件事并未做过多的关注。

一周前看到同行讨论这件事，我才去网络上检索了相关词条，不料世态已经开始向着意料之外的方向发展。

起初，网友将矛头指向作者玖月晞，指出对方的原著小说《少年的你，如此美丽》涉嫌多处融梗《白夜行》、《嫌疑人X的献身》等作品；

接下来，不明所以的吃瓜群众们开始指责《少年的你》片方，称涉嫌抄袭的原著所改编的影视作品也应该被定性为抄袭，不应当被如此推崇；

再到后来，当网友夸奖周冬雨、易烊千玺、周也这几位年轻演员演技炸裂的时候，也有莫名其妙的黑子拿着原著融梗的事情说事。

……

所以，一部电影的原著涉嫌抄袭，电影、制作团队、演员就应该要被怼吗？

之前我在写《如懿传》：初看失望，看完却怅然若失，细水长流才是真正的好剧这篇文章的时候曾经提过，作为文字工作者，没有人比我们更讨厌抄袭这件事。

文章时不时地被不经授权地全文搬运，我发现后去举报和与对方理论，非但得不到道歉还会被冷嘲热讽一番，也实在是不能苟同当下搬砖党们的素质与底线。

但是，将影视作品和原著混为一谈打包谈论，我个人觉得是非常不理智的行为。

《少年的你》的编剧主要有林咏琛、李媛、许伊萌，而玖月晞仅以原著作者的身份出现，由此可以看到电影的剧本相较于原著是有不少改动的。

（The End）

「Vicky's 资讯」《人生选择题》：人生中的18种特质，周冬雨选择了哪9个？

All Image Source

《人生选择题》是一档由今日头条、西瓜视频与新世相联合出品的明星访谈微综艺，主持人华少将会带着八位嘉宾体验一把平行人生。

本期的嘉宾，是最近主演了《少年的你》的周冬雨。

节目的一开始，华少在周冬雨面前的展板上呈现了人生的18种特质，它们分别是：乐观、智商、颜值、健康、勇气、抗压能力、友情、亲情、爱情、自律、幽默、才华、好奇心、梦想、赚钱能力、诚实、自尊以及自信。

古灵精怪的水瓶座女孩，在考虑良久以后，最后选择了以下9项：健康、梦想、诚实、颜值、才华、自律、亲情、友情和爱情。

在做选择的时候，周冬雨直言亲情、友情和爱情对她来说是十分重要的。她还自嘲选择颜值是因为自己“没啥所以才想要啥”，这个耿直girl让人一下子就好感度倍增。

纵观这18个人生品质，我想我会选择以下几项：

「颜值」：毕竟是爱美如命的天秤座，我会毫不犹豫地选择这个选项。无论身边的人或者社会价值观灌输给我们，看人要注重内在和有趣的灵魂，但外型的重要性没有人可以否认。连周冬雨都戏称，有了颜值很多东西自然而然就回来了，哈哈；

「智商」：不想做个脑袋空空的草包，所以我们还需要智商来武装头脑。这里的智商倒不是要解奥数题或者争夺诺贝尔，而是明辨是非的能力以及接受、学习新鲜事物的能力；

「健康」：良好的身体状况必不可少，不然有美貌和智慧却无福消受，多可惜呀；

「友情」：身边有可以失恋时陪我们哭闹、大醉一场的闺蜜，是人生难得的幸事。如果再能有一些玩到一起的男性朋友，看事物的眼光和角度也会变得不一样。有点遗憾的说，我身边哥们比闺蜜多太多，反思中…

「亲情」：原生家庭非常重要，父母更加是孩子第一任老师，很多品质会影响孩子的一生；

「自律」：我自问还是属于比较自律的人，无论是生活还是工作上，虽然容易翻车过度消费…

「才华&梦想」：没有人会喜欢一事无成的草包的，一定要找到自己喜欢并有能力做到最好的事情。毕竟没有人会陪着另一个人走完一辈子的路，成年人更应该学会如何更好的独处。

「诚实」：这是一个人安生立命非常重要的品质，可以说上面的一切都可以没有，也不能没有诚实。

如果你是，你选择哪几样特质呢，在下方留言区留言讨论吧:D

（The End）

「Vicky's 资讯」《人生选择题》：节目组请来了周冬雨做嘉宾？火速围观！

All Image Source

由今日头条、西瓜视频与新世相联合出品的明星访谈微综艺《人生选择题》，近期邀请到了周冬雨作为嘉宾。

谁是周冬雨？

大家的第一印象，大概是谋女郎。

当时还在因为念高中的她，因为青涩淳朴的外形一下子被老谋子相中，主演了了催人泪下的《山楂树之恋》。

之后的几年，她作为内地上升期的小花旦，在影视方面做了诸多尝试。

《指甲刀人魔》、《倾城之恋》、《阳台上》、《幕后之王》、《后来的我们》、《同桌的你》、《少年班》、《春风十里》、《麻雀》、《喜欢你》…

合作者大多是知名导演和经验丰富的演员，照理说应该可以收获不错的成绩。但很可惜的是，不知是否是因为剧本不对路，周冬雨的这些作品虽然名气很大，口碑却平平。

直到2016年的一部根据安妮宝贝同名原著改编的电影《七月与安生》，才让观众们认识到这个小个子女生在演戏上的执着和爆发力。

今年，她和易烊千玺一同演绎的，讲述校园霸凌的电影《少年的你》，更是让不少人在电影院内就忍不住地擦眼泪。

由此可见，一个演员在演艺事业上的成绩，除了自身的努力以外更和选剧本眼光、合作的导演及同事密切相关。

说回《人生选择题》这档综艺，其讨论的重点是如果你进入一段“平行人生”，会选择怎么样度过呢？当遇到成长中不可避免的磨难时，会选择接受还是逃避？

我很喜欢周冬雨，冲着她做嘉宾这点，《人生选择题》这当节目必须火速Mark！！

每周三12:00，跟着超能主持人华少与八位嘉宾，一起体验奇妙的旅程吧:D

「Vicky's 资讯」《天天向上》：优质偶像王一博，带大家熟悉日常防火安全知识

All Image Source

多年不看综艺的我，最近因为粉上王一博开始刷《天天向上》。

《天天向上》作为芒果台继《快乐大本营》以来受众最大的综艺节目，王一博能跻身成为常驻主持，起初也受到了不少质疑之声。

但他凭借自己的实力和人气，以及谦逊有礼的待人处事态度，后来已然成了节目的团宠。不仅汪涵哥爱护之，连晓明哥在这期节目中对王一博再三肯定。

在节目的最后，王一博努力克服身体不适，最终完成考验，成为湖南明星志愿消防救援队新队员。

常青综艺节目+人气明星，强强联手去科普消防安全知识。这样正能量的组合，请再给我来一打！

在节目之后，耶啵还为自己家乡的消防安全宣传工作出了一份力。
