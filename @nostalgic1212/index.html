「Vicky's 资讯」《追我吧》：送别高以翔，人间再无王沥川

All Image Source

这几天，各大互联网的热搜话题之一，就是高以翔在录制浙江卫视追跑竞技秀《追我吧》时，喊出一句“我不行了”就突发去世。

据台媒消息，高以翔的遗体安放在当地殡仪馆灵堂内。高妈妈为儿子请来法师诵经，希望他可以一路走好。

网友们猜测的父母向节目组发难的情景并没有发生，当工作人员在机场接到二位老人的时候，他们反而安慰节目组：“不是你们的问题，你们也不会愿意发生这种事。”

目前，网络上盛传同为嘉宾的黄景瑜在大声呼救后，节目组医务人员很久才作出反应的消息。不少网友和自媒体作者作出推测，是否因为这点才延误了救治工作，要求彻查此事的呼声在互联网上越演越烈。

高以翔的经纪人Mary，在今日就此事作出澄清：

“没有这回事，没人救是乱说的。当他们看到监控时，很快跑过去，跑到高以翔身边需要一点时间，但不是大家说的做效果。大家都及时赶到，及时救援。”

经纪人呼吁大家不要再让这件事继续发酵，以高以翔的性格也不会希望大家再争论不休。

后续事宜将在经纪公司的帮助下与节目组进行协商，尚未决定是否在当地火花。

R.I.P

（The End）

【Vicky's 影评】《王国》：人怎么能吃人？人和恶鬼，究竟哪一个更为可怖？

All Image Source

舒菲突然想起师父回来那天一脸沉重，将一口薄棺交给他们下葬，他们打开时看到了与师父一同出去的丹儿的尸体。

那具尸体面容可怖，仿佛是被什么野兽撕咬过一般血肉模糊。

难道…永信给那些病人吃的是丹儿的尸体？

人怎么能吃人呢？

舒菲从厨房冲出来想把实情告诉大家，看看着那些病人正吃得如痴如醉，话到嘴边她却怎么也说不出口了。

愤怒的她只能把永信拉到柴房里头理论，不知不觉外头天已大黑，他们俩被外头的响动打断。

当舒菲打开柴房的门往外看去时，外面已是一片哀鸿遍野，师父也未能幸免于难。

这些吃了丹儿尸体的病人仿佛得了瘟疫一般，变成了嗜血嗜肉的丧尸。

本剧采用双线叙事的方式，一条线在东莱我们刚才已经看过，另一条线则在汉阳皇宫。

公元1392年，李成桂推翻高丽王朝改国号为“朝鲜”，开启了长达500多年的朝鲜（李氏）王朝。

被贪腐已经蛀空了的朝鲜王朝早就破败不堪民不聊生，年轻的王后赵氏和她的父亲赵学洙（柳承龙 饰）以及哥哥企图把持朝政。

此时民间早就流传起大王已驾崩的消息，但赵氏一族不惜启用秘术“复活”大王其实另有所谋。

他们希望借大王之手彻底阻止儒生簇拥的世子李苍（朱智勋 饰）继位，并希望赵王后可以成功诞下龙子，从此成为他们一族摆弄朝鲜王朝的傀儡。

李苍如果继续留在宫中坐以待毙，一旦赵氏如愿生下龙子，必然取代他成为大王顺理成章的继位人，那么自己必死无疑。

某日夜里，在听到民间关于“大王已驾崩”的传言后他夜访康宁殿，确实也是察觉到了不少值得推敲的细节。

为了查出父王的下落也为了自保，李苍决定带着心腹武英（金相浩 饰）冒死出宫前往东莱。

根据武英涉险从太医院寻来的药房日记来看，最后为他诊治的就是前太医李承熙。只是各种细节无从得知，仿佛是有人故意不将其记录在册。

一方面他要躲开赵氏一族派出来想要置他于死地的追兵，另一方面他要面对昼伏夜出嗜血成性的丧尸。

李世子是否能摆脱这些困境和危险，如愿找到真相并解救天下苍生于水火之中呢？

（The End）

【Vicky's 影评】剧集版《猖獗》，韩国版《行尸走肉》，古装版《釜山行》，这部高分韩剧你值得拥有！

All Image Source

毫不夸张地说，我是看了第一眼就立马决定要追这部剧的。

原因无他，就冲着片头展现的这套繁复有序的工序：针刺眉心、浇药入口、吹烟入鼻

从衣着配饰来说，片头中躺着的正是朝鲜王国的大王。不知道什么原因他已经驾鹤西去了，而有人煞有其事地在对他做着某种法事。

大王的口中含着珍珠和大米，经常看这类影视剧的朋友应该都知道一个常识，在古时候大米是用来辟邪的。

我们不难猜出这一套流程大致就是某种秘术，其目的就是利用古法炼尸。

果然，当大王真开眼睛的时候，这分明已经不是一双活人的眼睛了。

就此，剧情从这里展开了。

相传李氏朝鲜的大王身染恶疾，衣锦还乡在东莱开设医馆的前太医李承熙奉旨，带着徒弟丹儿前往汉阳的皇宫医治国王。

一老一少顺着微弱的宫灯走进康宁殿，那正是大王养病的地方。刚走进正殿，李承熙就对徒弟丹儿说：

“你千万不能偷看殿下的寝所，从现在开始你什么都看不见…”

丹儿手上端着要献给大王的汤药，一面战战兢兢地站在寝殿之中。那厚重的帷幕后面，时不时传来如同猛兽般的低吼声和链条撞击的哐啷声，在寂静的宫殿中显得尤其突兀。

这时候，李承熙喊他给大王送药过去。

他小心翼翼地靠近纱幔，想要将要从帐子下面递进去，隔着纱幔他隐隐约约似乎看到了什么。就在他隐约看到了什么的时候，纱幔后面伸出一双瘦骨嶙峋的手将他拖了进去。

一时间他声嘶力竭的惨叫声打破了皇城寂静，但谁也不会关心这个少年就此消失在了这黑洞洞的宫殿之中。

原来，少年的作用不是过来喂药的，他本身就是一味药材。

画面来到了李承熙在东莱开设的一家医馆，徐舒菲（裴斗娜 饰）是里面的医女专门治疗那些看不起病的穷人。

条件贫寒的他们已经把山里能跑的东西都找来吃掉了，包括树上地里的虫子。附近山上的草药也早就被挖空，医馆里面的医者们不得不去更远的山中寻找药材。

这天，舒菲外出采药回来发现医馆里头难得一片欢天喜地的场景。

她走近一看，发现这些伤员们正在大口吃肉喝汤，打听得知这些肉是医馆的一位伤者永信打回来的鹿。

可这山上哪里还有鹿呢？

舒菲还没来得及细想，一位病人就央求他能否再打一碗给正在哺乳的妻子。

舒菲拿着碗走进厨房，闻着锅中香味扑鼻的肉汤味她咽了咽口水，还是想把这些好吃的留给病人们吃来补充体力。

当拿着勺子想把汤汁倒入碗中的时候，她分明看到了碗中一小节断了的手指！

（The End）

【Vicky's 影评】奈飞出品，古装+权谋+丧尸，这部韩剧居然比《行尸走肉》还过瘾

All Image Source
文/冰冰

根据以往的经验来看，韩国通常会在夏季档播出惊悚灵异主题的影视剧。

去年夏天，就出了《鬼客》、《司祭》以及《赤月青日》。无论哪一部，都被观众们交口称赞。

然而就在年初的时候，我被一部反其道而行之，Netflix出品美韩合拍的剧集吸引。

那就是我今天想介绍给大家的这部：《王国 킹덤》（又名：李尸朝鲜）。

因为口碑炸裂，《王国》的续集在第一季播出之后立刻投入了紧锣密鼓地拍摄当中。

最近，《王国》剧组在官方ins上发布了一则短视频。状态非常好的三位主演纷纷现身，公开宣传第一季的开播日期为2020年3月，并对这部死亡史诗回归表示非常期待。

这里，冰冰就和大家回顾一下第一部的内容，梳理下各条人物线间的关系。

《王国》改编自网络漫画《神的国度》，影迷倾向于称之为剧集版《猖獗》和韩国版《行尸走肉》。

故事讲述了16、17世纪古朝鲜王国大王病死后，王后一族为保自己一脉的利益启用禁术将其复活。此后，国内开始出现奇怪的瘟疫。染病之人会变成不死的丧尸，以啃食尚存的人类为生。

本剧的导演金成勋执导过韩国高分电影《隧道》、《走到尽头》，目前他正在拍摄电影《绑架》（翻拍《解救吾先生》），影片预计将于2022年上映。

编剧金恩熙这个名字大家可能不是很熟悉，但她参与编剧的韩剧基本都在8.0及以上，大名鼎鼎的《信号》就是出自她之手。

除了强大的主创班底，本剧的卡司也是相当豪华。不难看出，网飞想要打入韩国市场决心之坚定。

女主角是被中国观众称为“韩国版王菲”的裴斗娜，她参演的《秘密森林》、《云图》、《我要复仇》、《道熙啊》、《隧道》、《空气人偶》等都在豆瓣上属于高分电影。

同时，她还参演了由Netflix出品的《超感猎杀》系列，亚裔的她获得了国内外影迷的一致肯定。

男主角朱智勋主演过影迷们耳熟能详的《与神同行》系列，同时他也参演了诸如《特工》、《魔王》、《宫》、《好朋友们》、《阿修罗》等高分电影。

大反派赵学洙的扮演者柳承龙在《7号房的礼物》影中精湛的演技，至今仍经常被影迷津津乐道。

他还参演了热门剧集《来自星星的你》、《Iris》、《双面君王》及电影《孩子们》、《哭声》的演出，均获得不错的反响。

（The End）

「Vicky's 影评」《少年的你》：大众将影视作品与原著绑定讨论，这么做是否合理？

All Image Source

《少年的你》确实改编自小说《少年的你，如此美丽》，但原著小说在三位编剧的鬼斧神工之下已经被改的“面目全非”。

电影不仅仅将原著中形象扁平化的陈念、小北、魏莱等角色塑造得更加立体，更为他们添加了所需的人物背景，让观众在观影的过程中产生“XXX会变成这样，原来是因为…”的感觉。

在出场人物和剧情上，编剧们不仅仅做了加法，更做了很多减法。网络文学中常见的冗长的“废话”被删除，对主线故事帮助不大的人物也被删减或者合并，让整部电影在紧扣「校园霸凌」这个主题的基础上显得更为合理。

说了一大段空泛的话，我想还是在这里用个比较具体的例子来说明。

在观影的过程中，难得让放映厅笑成一片的，是一个看起来有点污的段子。

陈念因为被欺负，跟着小北第一次回到他家。小北进家门就迅雷不及掩耳地把散落四处的衣服收拾起来，是一个青涩的少年特有的慌张。

这一夜，陈念洗了澡穿着小北的衣服睡在床上，小北和衣睡在沙发上。两个少年都对屋里的气氛有些似懂非懂的感觉，这时候陈念开口问了句：硬不硬？

小北反应过来以后，嘴角带了一丝笑意地半起身子：啊？

陈念突然意识到这么问实在是太暧昧了，于是加了句：我是问你沙发硬不硬。

听到这句话，小北略显遗憾地答了句：还行。

原著小说中也有类似的文字，冰冰截取了了部分放上来给大家看看。

陈念咬一口，蓬松温软，奶香四溢。她身体猛地一僵，北野的手从她裙摆下探进去，沿着大腿内侧的肌肤往上摸。
　　
陈念扭头，与北野的目光相触，他的手在她裙下得寸进尺，低声问：“害怕么？”
　　
陈念躲避着踮起脚尖，他的手尾随而上。她发着颤，眼珠一转不转盯住他，懵懂而惶惑。

一时间，那个青涩的少年荡然无存，剩下的只是一个小混混形象。

而后来会为了陈念顶罪坐牢的小北与那个小混混的形象格格不入，不得不说，这里《少年的你》的编剧们改得非常妙。

电影中这样的细节改动非常多，无比包含了编剧们的心血。

在我看来，即便是原著真的抄袭或者融梗，电影也不应该被网友们拿来diss。当然，法律方面的责任分割不在这篇文章的讨论范围之内。

我知道有些观众会担心，如果一个抄袭或者融梗的作品都可以被拿出来翻拍，一不小心居然还成为了爆款，是否会形成一种行业潜规则，让越来越多的人去仿效。

如果你有这个担心，请尽可能多的收集相关证据并组织起来，向相关部门去提交，专业的事交给专业的人去裁定。在法律作出裁定之前，先别急着赌咒别人，更不应该去做殃及池鱼的事情。

退一万步说，如果原著真的抄袭，电影方本身也是受害者，并不应该受到道德的绑架和审判。

（The End）

「Vicky's 影评」《少年的你》：原著《少年的你，如此美丽》，是否真的存在融梗现象？

All Image Source

说真的，我在看电影《少年的你》的时候除了几度被激到哭得上气不接下气以外，真的没有感觉其内容和《白夜行》以及《嫌疑人X的献身》有什么关系。

虽然我没有看过《少年的你，如此美丽》，但我是东野圭吾的书粉。他的书我几乎都有看过，很多影视化的作品一眼我就能看出他的风格来。

我也看过很多在拿融梗说事的评论，主要集中在两点：

1、小北和陈念一个走在阳光之下，一个坠入无尽黑暗，极度类似于《白夜行》的经典桥段；

2、小北为了让杀人的陈念脱罪，为她做了不在场证明，这和《嫌疑人X的献身》中的高潮部分也非常相似。

但是，经典桥段不是仅有一人能想出来，也不是有作者用了以后别人就不能再用了。

融梗的标准是什么，又有谁可以给出明确的定义呢？

融梗和抄袭的界定是存在一定标准的，这方面我不是专家，在这里就不做冗述。

我只是想说，当你拥有相当大的阅片量和阅读量的时候，就不难发现，无论是影视剧还是书籍的创作中，借鉴（融梗）的情况真的非常常见。

比如第38届香港电影金像奖《我不是药神》热映的时候，就有网友指出其部分情节与《达拉斯买家俱乐部》相似；

陈思诚导演的《唐人街探案2》，就不止一次被人说起和电影《双瞳》如出一辙；

去年由周润发、郭富城主演的高分电影《无双》，与1995年拍摄的《非常嫌疑犯》多处相似，虽然导演出面澄清自己并没有借鉴；

……

这样的情况，相信在小说中会更加常见。

我想，现在无论哪个自媒体都不敢说《少年的你》原著抄袭，这可是一不小心会被起诉的大事。

专业的事应该交给专业的人去做，我们在没有足够的知识量的时候不应去做过多的揣测，也是身为自媒体人应有的素养。

而我们作为普通读者和观众，欣赏眼前的作品，能够就事论事的谈眼前的作品就最好不过了。

当然，我们也非常呼吁相关部门可以出台关于借鉴和融梗的标准，让很多似是而非的争论有个标尺。

（The End）
