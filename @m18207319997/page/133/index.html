如何甑选去中心化项目的思考

昨天在@runicar的文章中BEWARE! NEW Game On STEEM Raises Some Serious RED FLAGS!看到他列举的近段宣传力度较大的Daiesthaiwar游戏的几个疑点：
一是将steem和steemit搞混；
二是只强调折扣少提游戏本身；
三是使用假账户伪造游戏购买。

所以朋友们在玩区块链项目的时候，一定要擦亮眼睛，以免被骗。但是作为非专业人士怎么才能辨别真假，暂且提供一下思路供参考。

图片来自pixabay
一是基本常识判断，作为一个能持续经营和专业游戏团队是不应该有常识性的错误；
二是看项目信息披露是否详尽，一个项目如果没有周密的计划是不可能走得长远的;
三是看项目团队的开发经历，如果是个门外汉投资的项目，你不得不考虑是否向该项目投资;
四是看已投资的账户或领投人，如果已投资账户有问题，那整个项目团队也干净不到哪去。

在决定投资一个去中心化项目之前，一定要做足功课，被骗之后那必定是血本无归！！！
