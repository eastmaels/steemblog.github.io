历史第三大融资内容社交项目TATATU | 区块链内容社区报告

之前在社区看到有steemit的小伙伴们发了关于TATATU项目的信息，我也尝试了解了下，这里简明把项目描述一下。对于内容平台，我认为感兴趣就直接用起来，相比作为一个旁观者解读思考白皮书，不如直接参与体验，你会获得更多，理解更深。

前言：

一个由好莱坞发起的视频内容社交项目TATATU，获得了5.75亿美元融资，是继EOS、Telegram之后，有史以来第三大融资，是内容社交项目的第一大融资。投资者来自BlockTower资本、Lvna资本以及一位皇室王子。那么这个项目到底怎样，让我们通过本文简要了解一下。

一、TATATU项目简介：

TATATU是一个视频点播和社交娱乐平台、视频内容覆盖动作、音乐视频、体育、游戏和KOL内容；用户可以通过观看、发布、分享等行为获得TTU奖励。

创始人CEO ANDREA IERVOLINO，意大利好莱坞电影制片人和企业家。 年仅29 岁的他被认为是电影史上最卓越的意大利电影制片人，也是好莱坞最具影响力的人物之一。从16岁开始，ANDREA IERVOLINO便开启了他的电影之旅。时至今日，ANDREA IERVOLINO已参与制作了60多部各类题材的电影。

可以看出创始人在电影内容领域有不错的成就和资源。

Flixxoo、YouNow、Videocoin、steem

详细了解，可以访问官方网址：https://tatatutoken.io/

我认为，这个项目及其项目逻辑并不是创新，很多类似项目都有。基于以太坊的通证经济模式，并不需要团队在加密货币方面有多少技术方面的能力。白皮书描述的项目逻辑及通证玩法，在很多项目中也都可以看见类似的。

那么TATATU为何能融资5.75亿美元。我认为，白皮书人人都可以写，但是创始人在视频内容社交领域特别是好莱坞电影市场的各种资源能力或许是无法比拟。基于创始人好莱坞制片人的背景，或许可以打通视频内容正规市场（这里正规市场，指传统电影视频官方发行方，而非个人视频社交内容），改变目前正规市场的视频内容利益格局。

麻烦的事，往往是机遇

最近@angelina6688询问我如何买steem，玩steemit的大部分对加密货币其实并不了解，@angelina6688算是一位，所以，对于他们来说如何用人民币购买steem还是颇有难度。

  • 你要懂点ETH或者BTC（steem的常用交易对）
  • 你要懂科学上网
  • 你要会场外交易
  • 你要会用币币交易所
  • 你最好还要会点英语
    ……

17年前，这些大部分我都不会，交易所的K线图都看不懂。然后，时代逼着自己前行，steem逼着你了解更大的世界。

我认识一朋友，17年10万资产入场目前身价过亿，在加密货币牛市里靠疯狂的搬砖赚取差价起家。据说他注册了至少上百家交易所，英语不好用翻译软件，注册大量的邮箱，别人觉得无比麻烦的事情，他都去做……

他说过一句话让我印象深刻的话：当一个事情你觉得很麻烦时，所有人都不想去做时，他反而看到的是机遇。如果是要问他成功起家的秘诀是什么，就是不怕麻烦，越麻烦越上。

那么，你也回忆一下，是不是某些时候因为自己怕麻烦，而错过了一些机会。我感觉我好像错过了很多。

之前空投的byteball麻烦不麻烦，我都弄了好几天，就是这样的麻烦，也收获了30+的steem。

Steemit也是如此，注册麻烦不麻烦？使用麻烦不麻烦，密码还那么长？早期的时候还没有汉化，看懂看不懂？就是因为这样麻烦，所以才是部分人的机会！不敢说所有人都在steemit里有收获，但是一定有一部分不怕麻烦的人收获颇丰！

反过来想，当人人都可以很可以很简单做的事，为什么好处就轮到你！

说道这，昨天steem的8.5元的低位，就尝试定投了一次！

有多少人按照活着的方式去想|《怎样改变自己的心智模式》读书笔记

关于这“心智模式”个词，我最早接触是来自“得到”专栏作者吴伯凡，他经常提到这个词，后来罗胖也经常说到这个词。

那么什么是心智模式，百度上是这样解释的：

心智模式这个名词是由苏格兰心理学家Kenneth Craik 在1940 年代创造出来的。心智模式又叫心智模型。所谓心智模式是指深植我们心中关于我们自己、别人、组织及周围世界每个层面的假设、形象和故事。并深受习惯思维、定势思维、已有知识的局限。

每个人都有自己的心智模式，因为它深值我们的内心，通常我们不易察觉它对我们行为的影响。它指导我们思考和行为的方式，有些心智模式是有利的，有些是不利的。

这词真的太过抽象和还是难以去捕捉它的要点。观察下自己的生活周遭，或许可以发现一些。

比如，身边有工作上的朋友自尊心强，任何他完成的任务遇到反馈过来的问题，第一反应就是辩解，“那是因为……我错是因为…….”，对于别人来说，并不关心他为什么错，反馈问题只是为了修正，只关心后面怎么做。但是，给人的感觉，对于任何“问题反馈，建议”第一反应都是一种抵触和反抗，这或许就是他的心智模式，并且一直在影响他的行为，而自己并不察觉。

有人对于心智模式说过这样一句话：你不赶紧按照你想的方式去活，那迟早会按照你活的方式去想。

这句话对我的触动很大，想想身边中也是有不少的“单身”朋友，他们从最开始的主动到后来的相亲，再到恐婚，最后说服自己“单身其实也没什么不好”，以及努力说服自己和证明“单身也许更好”……然后，排斥改变……

不讨论“单身”好不好的问题，这里我看到“我们按照自己活的方式去想”的心智模式。这应该是一种坏的心智模式，按照自己活的方式去想，把自己的现状合理化，对外部世界封闭，排斥自己的改变和人生更多可能性。

POB又一个内容激励平台（2）

上一期POB又一个内容激励平台，简单介绍了关于PoB平台，这一期继续聊聊。

一、PoB的通证玩法

1、核心玩法
可以看出，POB在设计玩法时，参考了很多类似项目。提问和问答是产品的核心功能，采用IDO模型与脑力挖矿进行设计。核心功能包括：

  • 对问题进行回答
  • 对回答进行评论
  • 对提问和回答点赞/点踩
  • 对提问者和回答者进行打赏
  • 对提问和回答进行收藏
  • 对提问和回答进行分享
  • 邀请其他人回答
  • 关注提问者和回答者。

2、更多玩法

  • 付费咨询
  • 话题
  • 文章专栏
  • 大V课程
  • 大V直播
  • 私密圈
  • 知识出版
  • 广告投放
  • 数字钱包
  • 信息存证
  • 基于智能合约的问答场景
  • 市场问卷调查
  • 第三方项目推广
  • 预测市场

基于通证的玩法，在加上IDO设计，玩法会比较多元。相比目前币问的“单调”还是值得期待。

二、未来计划

除了丰富的通证玩法，未来计划，我认为也是亮点之一，在项目的未来计划中，包括：公链计划、数字资产兑换、完全开放平台接入第三方应用。

项目的公链计划，主要是针对未来，有影响力的人可以在POB Network 发布自己的token，并进行售卖，token 则代表了发行者未来在POB Network 所产生内容收益的分红权。

数字资产兑换计划，先期，POB Network 将接入其他交易所的API，在平台内部实现token 与其他数字货币的转换。后期基于影响力的个人token 发行平台实施后，将会有大量的个人token 出现，有价值的token 最终需要通过其他数字货币，甚至法币的形式体现出来，token 的可交易性将会是用户的刚需，无论是选择在平台内部嵌入一个微型交易所，还是开通一条与其他交易所的对接渠道，POB Network 将致力于解决token 的流通难题。

POB Network 在未来将建立内部的交易市场，实现token 与BTC 和ETH 的直接兑换。

关于内部交易所，跟steem是类似的。当然，未来的币乎也可以借鉴构建自己的内部交易市场。

完全开放平台，平台运营成熟，功能完善、内容有一定沉淀后，将会考虑开放平台，将平台部分代码开源，为第三方dapp 提供SDK、开放API、成熟的功能组件。开发者可以基于开放平台开发出更多更好玩的应用，丰富社区生态。

不过我可能并不特别看好这点，YOYOW也是这个思路，但是目前发展并不是很顺利，国内更多项目团队喜欢自己发币自己单干。第三方应用，或许应该定位为更加小的组织团队才是出路。
