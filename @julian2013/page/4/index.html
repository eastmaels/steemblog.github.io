明星的高薪

现在全国都在声讨明星高薪，但明星高薪占的人数比例大吗？何况他是个体户啊，自筹资金，自担风险。只看到几个星拿高薪，没看到绝大部分演员在挣扎，群众演员只能温饱，有的还在饿肚子。

你可以用税收的经济手段限制明星，但你不能打垮这个行业。娱乐文艺是企业(企业)，它肩负着大量人员的就业，并为国家创造税源税收，这是一个商业王国。所以要按价值规律办事，而不是用行政手段拍死了之。你可以这样做，但围绕着这一行业的资源与资本就没了。

大量的人生活怎么办？都由政府来养？政府连自己都要停摆……这会形成：只要我不满，我就拍死它。请问之后怎么办？

鲁迅有一篇杂文叫《娜拉走后怎么办？》，意思是说娜拉出走容易，但走出后会怎样？回答是：要么惰落，要么还是回家。

所以，别再嚷嚷着明星如何如何了，人家也是凭本事吃饭的。

消费的最大问题

近期省部级领导干部“下馆子”助力复工复产的消息在国内多地都有报道。

例如，重庆市政府副市长李波、市商务委主任张智奎以普通市民身份，带头消费，堂食吃火锅，给餐饮行业鼓劲打气；安徽省委常委、常务副省长邓向阳，省委常委、合肥市委书记宋国权陪同调研，并在罍街自费品尝合肥小吃。
更早之前，3月9日，江苏省委常委、南京市委书记张敬华到夫子庙景区的“南京大牌档”饭店，品尝了特色鸭血粉丝。

有人可能要问了，这什么呀？干什么不行，怎么号召大家赶紧消费起来？钱存下不对吗？新三年，旧三年，缝缝补补又三年才对呀！这不是鼓励浪费吗？

同志，你外行也。发展经济要按经济规律办事，我们早就强调一个重要的经济法则：即消费促进生产，享受促进发展。这是市场经济的铁律。

我们现在假设一个经济模型来看一下：假设我们全国从3月1日起下令每人为了节约，停止购物消费半年为a，那么结果全国的各行业大小企业工厂超市直到角角落落里的鸡毛小店业主听到此令立马会想到立即在第二天关门，解散职工，辞退店员，停止进货，或同时降价抛售物品，越快越好。

聪明的人会首先必须这样做为，因为令下之后将无一人购物，那么生鲜，食品，万物将成垃圾占用租房，水电，一切开支性服务资金将断裂，商业这一中间环节首先崩溃为b。

反过来，同时这一天各企业生产环节立马作出反应，停产，结账，辞员，统统滚蛋，越快越好，聪明的企业主必须是这样，不然多一天他都要付出工资成本，所以变脸如同翻书。此行为为c。

由于a停，通过中间商业环节b不订货，导致企业c停产歇业，那么一切社会成员都不付出体力，因为工资没了。按马克思新的资本产生的原理，即工人工资这一部分是产生剩余价值的资本(马称可变资本)，社会财富资本不是材料成本变来的，(马称不变资本)，也就是成本不会增加财富，而只有工人动起来(生产起来)，将产生剩余价值……

持续生产，财富资本此时才不断膨胀，人工这个物化的资本在生产中增加，这时马克思把它表述为c+c1，如果生产是持续不中断，那么公式就是：C+c1+c2+c3……如此社会财富(资本)将由小人成长为巨人。如果有了巨人资本，那么扩大再生产才能持续，同时社会办学，救济，科研，医疗，等等所需要的资本将源源不断充实进来，假设这个持续链条任何一环中断(a，b，c，) 那么可变资 本(C+c1+c2+c3……)便萎缩，蛋糕变小至零。结果是整个社会一片萧条，前期表现为通货膨胀，(人们无币购物)，后期表现为滞膨(开始毁灭产品，因货堆积)。

这就是为什么现代社会要鼓励消费的原因。不消费导致不订货，不订货又导致不生产(失业出现)，反过来无钱更不会消费，而不消费更更不会生产，如此循环往复，王朝就该崩溃了。

道理就是这么个道理，为了社会，为了我们自己，也需要大力消费。唯一的问题是——

##没钱……

steemconnect api 那边如果默认用的 anyx，就会发到 Hive，如果用的 api.steemit.com，就发到 Steem。。。现在明白他们搞 steemlogin 是什么意思了。。。
发到哪个链，不受 busy 本身控制，受steemconnect.com 的管理者控制。。。

原罪

小时候，看电视新闻听到的很多的一句话就是：“不要干涉别国内政！”。当时觉得挺有道理，别人的家事，外人有啥资格插手呢？人家过得怎么样，是人家自己的选择，和你又没关系，应该尊重。

不过，现在想想，可能有点问题。

现在的世界连接越来越强，绝不是古老的社会上那种“非礼勿视，非礼勿听，非礼勿言，非礼勿动”就可以了，很多时候，别人的“无礼”，会像蝴蝶煽动的翅膀，引发别处的山崩海啸。换句话说，自己作死可以，但别连累别人。

这个时代，几乎没有孤家寡人了，每个人的所作所为都应该注意影响，而不是打着“家事”的幌子拒绝别人检视。

回想一下，有些组织的诞生，恰恰就是有人干涉别国内政的结果。

成立于1919年3月的第三国际，是一个国际性组织。每个国家的某某党都是当时第三国际的一和支部，听从它的指挥。第三国际中有不少终身以革命为饭碗的职业革命者，他们被派到全世界各个有革命需求的地方寻找革命代理人，为他们提供经费，组织政党。这不就是赤裸裸直憋憋的干涉他国内政吗？
