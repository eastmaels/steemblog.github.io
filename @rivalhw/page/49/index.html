令人失望的媒体APP(自媒体)

前几日的时候，我卸载了手机上的某头条新闻APP，原因先前已讲过，就是我发现那玩意太浪费我的时间了，而对我有用的信息却很少。

　　忍了几天后，闲暇时没有新闻看，总感觉心里有点空落落的，想到国内还有其他媒体如新浪、凤凰网等，这些都是比较传统的媒体，相信内容应该不会像头条那么杂乱吧。

　　于是安装了新浪app，刚打开后，发现对方给我推送的所谓新闻资讯，更是令我感到失望。广告我并不很反对，毕竟是企业要赚钱盈利，但其它的内容，多数不是标题党，就是打开后发现内容哗众取宠，总而言之，多数内容空泛无意义，当然，如果你纯属无聊打发时间除外。 　　


　　新浪新闻的APP打开界面，嗯，启动页面做的还是很不错哦，大气！


　　但推送的资讯和内容，就很令人失望了！

　　改试着安装了ifeng凤凰网，


　　ifeng凤凰网新闻APP的打开界面

　　ifeng的界面很简洁，这点跟我一贯的的风格很像。


　　对比看了下，ifeng还能好些。

Mac系统下No such file or directory的问题解决

　　我在Mac下安装了个软件，需要手工对文件进行配置下，于是打开terminial终端直接进行操作。

　　有一个很怪异的地方，就是我进行操作的时候，明明当前目录下有那个目录文件，但我使用命令的时候，却死活进不去，而且系统显示：

　　No such file or directory

　　这真是奇了怪了。刚开始时，我以为是自己输入的目录名称有误，还专门核对了下，再次输入操作，问题依旧，显示无该目录。

　　于是我改复制该目录名称，然后再粘贴进行操作，发现仍然无济于事？

　　难道是操作权限的问题？我心中疑惑到，但想了下觉得不对呀，

　　我这个当前操作身份本身就是系统管理员的身份啊。

　　百思不得其解，只好上网百度下，发现在Mac系统下，空格要转义符，比如一个目录名称为：

　　my directory

　　操作的时候应该为：

　　cd my \ directory

　　在空格前边加多个”\”转义符即可。

　　又或者另外一种方法，即将目录名称用””起来，如上述目录操作为：

　　cd”my directory”

　　这样就可以了。

　　返回Mac重新操作，果然问题解决。
