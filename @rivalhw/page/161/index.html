关于组织整理steemit操作和帮助的一些思考


　　晚上在机场候机的时候，@dapeng发来信息，问我，


　　大伟哥! 我们在编写 《steem 指南》书，想邀请大伟哥做顾问，宏观上为我们把把关，不知有没有时间？


　　我回复他说：好！并进一步跟他讲，只是我个人时间不固定。刚好也想组织安排人做个steemit从入门到精通的类似内容，你这就捷足先登了。


　　其实在两个多月前，那时候还没有中文版，就已经有人建议我，希望能做个简洁的、方便的steemit指引功能，方便大家尤其是那些刚开始接触steemit的新人。


　　对于这件事，我也一直在考虑，我想的不是具体如何去做这件事，具体的内容对于我来说不是难事，只需要列个提纲，然后按部就班去做就可以，但我们都知道，steemit是区块链存储，在过了7天后内容就无法再修改，这对于一直都在变化的网站平台来说，如果你完成了初步的内容编写，但后续却无法更改的话，到时候可能就不一定是帮到新人，还很有可能会因为过旧的内容反而误导了大家，而如果要撇开steemit另外建立个网站索引，这又不是我想要的，如果每个新项目都需要做个新网站的话，那样岂不是很是繁琐和不方便。


　　就是因为一直未能想到更好的办法，所以这件事情就迟迟未能开始。直到刚才@dapeng提醒我，我才想起来的确需要是时候提供这么个内容了。


　　我现在初步的想法是，可以先建立个索引，这样即使内容有变动或更新，我们只需要更换下内容即可，而不是无法更改或者重新建立个新的。


　　@dapeng 跟我讲到，他自己也在摸索，目前也没有找到合适方法，只是想着先列个目录，然后找两个人分头去做就可以。我表示赞同，我再谈下我个人的建议：


　　1、在做新的内容时，要注意尽量利用好先前的一些已经有的内容，包括但不限于steemit官网提供的内容，尽量避免重复造轮子，如果能结合steemit官网的帮助文档做些差异化的互补内容提供就更好了;


　　2、尽可能在一些大家碰到问题的地方，多花些精力和时间，可以分重点和普通内容，避免一些如大家都知道的内容上边都有，而大家碰到问题的，或者重点想了解的却内容少之又少甚至没有。


　　3、完成整体内容后，如何保持内容的最新和及时准确，这个需要重点考虑，是否可以利用好现有的如电子存储等优势？


　　再次感谢 @dapeng 和 那些愿意并对社区作出贡献的同学，即便是很小的贡献，我们都非常感谢你的付出，也希望社区今后能涌出更多的热心人士！

人在旅途


　　唐代大诗人李白曾在《早发白帝城》中书，朝辞白帝彩云间，千里江陵一日还。两岸猿声啼不住，轻舟已过万重山。我非常喜欢这首诗，也期盼着有一天，自己也可以像李太白那样游历大江南北。


　　早在一周多前，知道自己要去上海参加会议，刚好年前有些时间，我看了下地图，发现可以趁这此出行时，顺便去周边一些地方转下，于是便有了这次的行程地图安排：广州出发-上海-南京-合肥-南昌-返回广州。



　　因为多年养成的习惯，我已经习惯于每次出行前都安排好主要的行程等，这次也不例外，只是刚好赶上春节，票不太好买，于是这次的出行就成了各种交通工具的大杂烩了。


　　想着这次出发，不用那么赶，边走边看下祖国大好河山，于是首选了高铁出行，还特地选择了靠窗户的位置。待高铁驶出广州后不久，我沮丧地发现，透过窗户我很难看到较远处的景色，只是每到一站时，站台的牌子还是可以清晰可见。


　　这次出行，因为在短短的几天内要去好几个城市，虽然从时间上安排的稍微紧了些，但也能粗略地感受下不同地方的上至人文地理，下至美食佳肴，总体上而言我还是很满意，我会在随后的时间里，将去过各个地方的感受和碰到的一些奇妙事情等整理出来，一方面是自己多年来养成的日志记录习惯，另一方面也供那些想去这些地方感受的朋友作为些参考和娱乐。


　　列车行驶了两个钟后，我突然觉得肚子有点饿，本想在车上吃点东西，突然想起来铁路12306好象先前已经开通了可以在高铁上订餐的功能，遂打开手机APP体验了把。



　　因为预定线下要做准备，所以每到一站的提前约一个小时多就需要提前订好，距离站点一个小时内系统就自动禁止在即将到的站点下单，因为下一个是长沙站，我看了下APP，麦当劳和肯德基这样的国际连锁巨头自然榜上有名，国内知名的连锁餐饮店如真功夫、毛家饭店等也在上边开通了订餐功能，即将到湖南，自然要尝下湖南菜了，于是我点了毛氏红烧肉一份。


　　列车快到长沙站时，我就开始在留意，因为我是第一次在高铁上订餐，不清楚对方是否会送到我的座位上，在列车停稳一分钟后，我终于还是忍不住站起身，离开我的座位向车厢出口走去，刚好看到送餐的快递员，在报了我的车厢和座位号后，很顺利地拿到了自己点的餐。

区块链项目代币，何去何从？


　　一次偶然的机会，认识了个90后，跟对方交谈中，无意中得知对方在挖矿。本来挖矿这事目前不算新鲜事，前几日我在转车的时候，在星巴卡里边喝咖啡边候车时，就听见周围好几个人在交谈关于数字虚拟货币代币等情况，去年年初到今年以来，挖矿代币等名词已经变的非常的热门，稍微有点这方面了解的人，无一不在讨论或聊着相关话题。




　　等稍微深入了解后，我才猛然发现，跟我在聊天的已不是仅仅只是一个挖矿的90后小伙子这么简单，而是一个已经财富自由的人。




　　所谓财富自由，相信都是我等追求和努力的。什么是财富自由，我的理解是你不一定非得要赚够多少钱，简单讲是可以理解为你的被动收入超过了你所有的开支，这即财富自由。而对于这一条，我显然还差的很远。




　　我先前讲过，在过去的近二十年中，至少出现了两次财富机会，一是互联网泡沫，国内一些老资历的如新浪、网易、搜狐，甚至腾讯（腾讯晚一波）等均在那一波起家并发迹；另一波则是近十年前的移动互联网革命，典型的如小米等企业。




　　如果说过去的两波财富机会多是在企业中诞生，而这次的区块链革命，则是个人和企业并存的机会，我认识多个因为区块链或代币发迹的朋友，其中有其敏锐的市场头脑，或者勇于all in的赌博精神，但更多时候，对于大多数人来说，你不一定因此能够像他们般发迹，但如果抓住这次机会，至少可以实现部分财富的增长。




　　对于虚拟货币代币钱包，我认为目前尚且处于灰色地带，灰色不一定代表非法，但也不意味着你就可以大张旗鼓；对于去中心化交易所，我认为也尚且处于中间地带，或者我个人认为，随着2018年的到来，越来越多的区块链项目会大量涌入到这些交易所，这也就不难理解，为何币安等交易所会在短短几个月内突然一撮而就变成全球第一大交易所。几个月前上币安交易所还是相对较容易，而现在，也就在前几日堂而皇之发出不再继续服务大陆民众的公开信息。保守估计，至少有成百个区块链项目尚且在排队等候中，市场一度传闻，某区块链项目投资人打算拿百万美金给币安，希望能够优先安排均被其婉言谢绝，其市场影响力可见非一般。




　　我们都知道，对于ICO，国家在去年9.4明确出台文件是禁止的，本来对市场就很敏感的各个项目发起人或投资人，在此之后对于上交易所就更加迫切了，普遍的想法就是尽快能够上交易所，因为谁都不清楚政府下一步什么时候会出差新的政策。又或者，即便不禁止，但出台相关政策后，如国家队正式进场，有背书，有交易保证金，合法的手续等，现有的交易所将何去何从，胜负最终如何，相信大家心中必然有自己的答案。




　　对于那些参与或主导过代币发行尤其是项目发起人，我相信多数都是谨慎或者心中或多或少有一丝不安的，如同一个有入睡困难的人，在深夜里，楼上的一只鞋子已经掉落在地板上，而在等待另一只鞋子掉下来好踏实继续睡觉般。


　　而非常令人不安的是，这另一只鞋子，却迟迟始终未能落下来。

高铁上的一出李鬼闹剧




 　　坐车的时候，忘记是途径哪个站了，我正在车厢里专注地用电脑追着最近比较热门的电视剧《和平饭店》，突然听到了一阵吵杂的声音。




 　　这声音不似一般旅客上车的那种，显然是高出许多分贝，我循声望去，只见11B位置上，一中年男子，约莫四五十岁样子，在那里大声地叫嚷着：




　　你信不信我把你拷起来？




　　我顿时有些惊诧，谁在这公众场合如此嚣张跋扈，究竟是怎么回事？竖起耳朵听了会，明白了事情的大致经过。




　　原来这中年男子，连同边上一个中年妇女（两人一起，貌似夫妇吧），在上车后看到有个年轻小伙子（约莫二十出头样子）坐了他的靠窗位置，便要求对方(小伙子)起身让开，谁知不知怎样那小伙子说希望跟对方调换下位置，这中年男子不同意，那小伙子可能也坐在那没起来，于是那中年男子便大声叫嚷起来，之后便发生了开头的那一幕。




 　　不一会，靠窗那小伙子起身了，只听见他在那里小声嘟囔说，




　　你把我的证件还给我啊。。。




　　中年男子这时候有些得理不饶人，继续大声嚷嚷，




　　你信不信我把你拷起来？。。。




　　之后我没太留意，大约一分钟后，列车这时候已经开始行驶了，我顺着过道，透过来往的人，看到中年男子似乎拿着对方(小伙子)的证件，右手好像拿着手机，似乎在（严肃认真地）记录着对方的证件信息。。。




 　　看到此情景，我不由感觉哑然失笑，诚然先前那小伙子占着对方的位置不对，你不答应就是了，可以礼貌要求对方让开，如果不让开，也完全可以找列车乘务员协助解决，没必要在那里大声叫嚷，一副老子牛逼天下第一的样子，更何况还口口声称因此要将对方拷起来。如果真因为对方没有让位置就将对方要“拷起来”那岂不是滑天下大稽，老百姓常云，爱叫的狗不咬人，嗓门那么大，又那么炸炸呼呼的，摆明了就是给自己壮胆用来吓唬没有经过世面的小伙子而已，如果真的要碰上了内行或真李逵，这李鬼不知要如何将这场闹剧演下去？




  　　备注，看完文章后，有人可能会问那中年男子手中怎么会有对方(小伙子)的身份证？其实很简单，现在很多地方高铁站已经支持直接用身份证验票，而在车上时，当时小伙子给对方看证件时被中年男子顺势拿到手，所以才有了后来中年男子那出闹剧。

再谈STEEMIT上密钥的重要性！3分钟内彻底弄明白STEEM/SBD/SP/SAVING/POST key/ACTIVE key /MASTER key


　　早晨刚醒来时，就看到群里有人讲自己的steem和 sbd 被盗了。


　　这个事件之前就已经发生过几次了，鉴于steemit的所有操作均建立在密钥上，并且提供了三种不同类型的密钥，我在这里再强调下其作用和重要性。


　　steemit为了方便用户，提供了三种类型密码，分别为post key,active key和master key。坦白讲，我刚开始接触时，都有点懵逼，平常网站都是一个账户对应一个密码，谁见过一个网站里给用户的同一个账户提供三个密钥？对于刚开始接触steemit的小白，我相信一定也会碰到这个困惑。


　　post key主要是用来提供给作者发文章使用，仅用来发文章，包括点赞和评论。而active key主要是用来给作者提供转账交易操作的。steemit上提供了四个不同类型的账户，分别为STEEM,STEEM DOLLARS（SBD）,STEEM POWER(SP) 和SAVING，其中steem和sbd我们大家都熟悉，你可以把它理解成两种不同类型的代币，事实上steem和sbd的确可以在交易所上进行交易，如bittrex.com上可以进行交易steem和sbd，而作为刚刚晋升为全球第一的暴发户币安交易所，也在前几天刚刚上线了steem。


　　在讲解SP和SAVING前，我们先简单了解下STEEM和SBD是如何产生，这样会有利于我们了解后边的内容。


　　作者在发文章之后，按照STEEMIT的规则，7日后文章将被锁定无法修改(区块链永久不可篡改性)，同时进行该帖的结算，并根据作者发帖时的设定进行STEEM和SBD的奖励。如果作者选择50% 50%，系统会将奖励分为两部分，一部分作为SBD奖励，另一部分作为SP奖励；如果作者选择发帖时power up 100%，则系统将会将奖励全部按照SP进行power up。


　　而在STEEMIT上发文章的权重，也就是无论你发帖或者点赞的影响力，均取决于这个SP(STEEM POWER)。


　　SP你可以通过STEEM来进行1:1的power up，也就是提升你的SP；你也可以通过发文章时产生的SP来提升，又或者你也可以通过SBD购买STEEM来进行power up。


　　SAVING是用来存储你的STEEM/SBD,你可以把它想象成一个冻结操作，一旦被转入SAVING账户里，你至少需要提前三天才可以取出里边的STEEM／SBD。


　　而SP里边的STEEM，如果你想取出来，必须先power down，一般一个周期需要13周。


　　三个密钥，分别三种不同类型场景功能。如果骗子只拿到你的post key，他只能发文章而已。如果骗子拿到了你的active key，你的STEEM 和 SBD账户里余额可以瞬间被他们转走，而如果你的master key被盗，你不但会失去你所有的金额，包括STEEM和SBD，你还将会彻底失去这个账户，因为骗子一般会很快修改主密钥。


　　说了这么多，你应该明白了吧。日常生活中发帖要用post key，尽量不要使用master key发帖！尤其不要在公用电脑等种保存master key!!!重要的事情，我觉得讲三遍都不够！


　　如果你的账户里有SBD或STEEM，你不着急使用的话，完全可以存入到SAVING账户，而且系统还会根据情况给予你一定利息呢。


　　最后再次强调，如果你真丢了你的master key，按照steemit目前的规则，基本你那个账户就废弃了！我一直认为建立在这种纯粹密钥基础上的规则不是很好，这也就是未来即将会出现一些其它的改进措施，如auth 2.0授权登陆，或者硬件钱包等类似授权操作，确保你安全的情况下，可以免受这些被盗的后顾之忧。
