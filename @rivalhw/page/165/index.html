关于茶馆的一些想法-茶馆歇业筹备中，公开对外招募"店小二"


　　过两天，即将迎来茶馆的一周年纪念日。我们也将迎来茶馆的重大改革，今天这篇文章，就是谈下接下来的一些思路想法。


　　我最近这几天多数在外边，一方面是因为公事，另一方面也同朋友等交流，比如前日去白云区那边同 @lemooljiang 兄面对面交流，便颇有收获。


　　茶馆过去一年的做法，就是由几位茶东轮流做东，刚开始，纯粹灌水，后来开始有主题，但时间久了，一直没能有更好的模式等，所以也未能有大的起色。


　　这几日很多人都写了自己对茶馆的建议和看法，我大致看了下，有些建议是对的，直接采纳，比如大家提到的建议，建议茶馆应定时开关；有些建议是不错，可以考虑采纳，比如吸引其他人开主题贴；有些建议可能不是很好，暂时就先不考虑了。


　　另外，我们几位茶东之间也做了沟通，基本大家的看法也是一致的，有好的模式，大家都愿意尝试下，所以这里我讲下接下来的想法。


　　一、茶馆定时开馆；


　　我们初步决定于每周一至周日，晚上6时-8时(根据最新民意，要修改，具体时间茶东开会后定)开贴。


　　二、茶馆的主题；


　　主题不限，包括但不限于，比如社会新闻、社区动态、娱乐八卦、吹水等；


　　三、面向社区公开招募店员;


　　店小二负责每日开贴，当然是轮值，如张三负责周一，李四负责周二。。。以此类推；一般周日为茶东来开帖，包括我 @rivalhw 、@lemooljiang 和 @ace108 ;


　　四、关于茶馆开主题；


　　我们引入新的模式，即每个人都可以在当日茶馆帖子下开设自己喜爱的主题帖，包括设置茶桌等，自由讨论，内容和形式都不做限制。


　　五、建议一套奖励机制；


　　对于店小二，我们会给予一定的奖励，主要是根据店小二的当日营收所得sbd分成收益，暂定可以拿到当日sbd收益50%，每月结算一次。


　　六、未来考虑


　　未来我们还有可能引入代理sp给到店小二，除了完成主要工作后，其它由店小二自行决定代理的sp收益，有点固定底薪的意思，当然这个暂时未定。




　　想要一起参与茶馆管理和收益的同学们，我代表我们几位茶东，非常欢迎！你可以在本文评论区中留言，或微信我 rivalhw (备注茶馆)都可以。




　　如果你看完了如上，发现还有更好的建议，请尽管补充。:)

我的steemit之路-Steemit 广州粉丝聚会开始筹备啦


　　Steemit在中国社区以来，先后有台湾、香港、上海、苏州等地的小伙伴们自发组织过当地的聚会，仔细想来，广州这边还没举办过一次steemit粉丝聚会呢，下午的时候，同 @lemooljiang 兄聊了下，感觉很有必要在广州也举办一次Steemit粉丝聚会，因此在这里开始筹备广州的聚会啦。


　　说聚会，必然要涉及到地点、时间、人物和主题。


　　地点当在广州，嗯，市区某的地方，具体待定，大家可以各抒己见。


　　时间，考虑到有些小伙伴们要上班或上学，我们计划安排在周末的下午时间，初步定在下周六(3月31号)下午3点钟。


　　参会人员，当然是steemit上的小伙伴们啦，屏幕跟前的你，如果想参与的话，可以在评论区留言，也可以发动你周围的steemit小伙伴们一起来参与。


　　主题，这是广州的第一次粉丝聚会，所以大家互相以认识为主，所以主题暂定《我的steemit之路》，主要讲下自己从开始如何接触steemit到过程的一些个人分享。


　　感兴趣或想要参与的小伙伴们，赶快过来一起讨论吧，包括聚会的形式和内容等，有好点子的更受欢迎哦。




　　本次活动接受赞助，内容和形式不限，如SBD、STEEM、ETH、字画、书各种礼品等，有意者请在评论区中留下你要赞助的内容，非常感谢！

来，干了这碗自己亲手煲的心灵鸡汤


　　1、不做极端的事情;


　　2、不说过激或太过的话；


　　3、少看视频多看书；


　　4、勤思考，多行动；


　　5、尽可能多些运动，如走路、骑车等，在家跑步也可以；


　　6、吃饭一次不要吃太饱，一般八成就可以了；


　　7、做事情如果没把握，可以多些推演，多些准备，不能盲目的自信；


　　8、不要轻易打断别人的话，尽量让别人把话先讲完；


　　9、不要先入为主，时刻保持“半桶水状态”，太满则溢，容易听不进和学不进去新的东西；


　　10、虚心听别人讲，尤其是那些你初听起来似乎很刺耳的话；


　　11、不要纸上谈兵，不要夸夸其谈，理论跟实践相结合后的想法才是最适合的；


　　12、那些与众不同的才是机会，如果一件事情大家都还没怎么做，很可能就是个机会，反之则刚好相反；


　　13、做事要有格局，做人要有胸怀；


　　14、保持一颗好奇的心态，尤其是对那些新生事物等；


　　15、不要随便否定别人。

一次会议活动举办的总结 -打造广州最具影响力的区块链项目和社区推广


　　前天下午的时候，我们举办了广州区块链万里行的会议。虽然万里行线下活动已经举办过多次，这次广州算是第29次会议了，然而这次会议的举办，还是有着特殊意义，因为这次是我们这个团队独立承办，从项目方到筹备到承办，全都是。


　　总体上而言，会议的举办过程都很顺利，也是较为成功的一次会议，但在这里我不想说太多其它的，只是想讲下不足的地方，有待提醒我们迅速或尽快完善改进。


　　俗话说，打铁要趁热，在会议刚刚结束后，我们的团队立马就开始了会后的总结，因为我很想听到不同的改进意见，所以有意地告诉大家，我们开这个会议，主要是为下次的会议做准备和改进，所以好听的、赞扬的话就不要多讲了，重点讲下不足之处。


　　果然，在这种氛围引导下，大家都敞开了讲，每个人都从自己的视觉和感受出发，讲了自己的感受，我们相关的小伙伴们会后整理总结了下，主要存在以下地方需要改进：


　　1.报名表要提前发信息通知，建议提前3天和提前1天做通知，如短信提醒等；


　　2.资料统一放行政组归档；


　　3.vvip没有体现自我，具体方案我们可以想


　　4.会场地点，下次找个更佳场地最好；


　　5.整理会议流程，为下次会议做准备；


　　6.会场可以安排休息时间，下次可安排提前准备茶点、水果等；


　　7.工作落实都具体责任人；


　　8.可通过问卷对受众进行了解，根据会议内容受众可针对性的选择会议；


　　9.下次协作方面更明确；


　　10.直播需预先定好主题，定好地方，如一直播，再找备选；


　　11.人员架构，缺少外勤，如vip的信息联系，内勤过多，项目方时间过长，一般控制15-20分钟；


　　12.团队内部带头做到，再去影响辐射观众；


　　13.现场气氛不够热烈，建议粉丝组组长带头活跃气氛，奖品没体现价值，需思考奖品物有所值；


　　14.下次准备大的那种背景签到墙，既可以方便与会人员合影留念，又美观大气；


　　15.提前审核嘉宾ppt，看是否有自动播放等,提前做好嘉宾水牌；


　　16.提前掌握会议嘉宾的心理，提前预告后面神秘大奖；


　　17.接待过程中，明确指引，让大家更了解我们，对我们有印象，在报名时在活动行可设计相应客户感兴趣的问题，提高客户体验值；


　　18.签到时可扫码验证即可，打印流程表发送给所有嘉宾；


　　19.专人负责人员进出，最好不要在会议演讲过程中进进出出，影响会场氛围。


　　我觉得以上建议都很好，我们接下来会针对其一一进行改善，期待着我们下次的会议更精彩，能够帮到区块链项目方做好宣传和推广，实现我们的“打造广州最具影响力的区块链项目和社区推广”目标！
