人生的成功，如同一道夹杂了太多随机因素最后得出成功结果的复杂方程式


　　这段时间来，币市行情逐渐好转，很多币陆续都开始回涨，一些知名币甚至一度突飞猛进一路高歌，用暴涨还是有些过分，但相比之股市，尤其是近两年国内A股惨不忍睹的答卷来说，还是非常不错了。




　　私下里会被人问多最多的几个问题，就是你觉得哪个token会涨?能否推荐一些token？




　　据我的观察，一般提问问到这类问题的，绝大多数都是小白，这里说的小白，指的是接触区块链相关内容，有些甚至连小白都称不上，可以说就一门外汉，在他们眼里，区块链几乎等同于token。




　　诚然，追求财富是人之本性，本无可厚非，但需要你的努力和付出，无论是金钱或者时间，都是有成本和付出的，但现实生活中却有那么一些人，总想走捷径，将期望建立在一些小道消息上，试图通过获取这些小道消息猛捞一把或暴富的想法。




　　这世上成功的人有两种：一种是那种坐拥真正财富的成功人士，而另一种是那种“教”大家如何“致富”的部分成功人士。




　　为什么我说是“教”大家如何”致富”的部分成功人士?原因有几点。一是这类人士原本并非成功人士，但会将自己包装成一副成功人士样子，让不明就里的群众相信他(她)就是成功人士，而这类人则通过一些杜撰、夸大、甚至海阔天空的胡乱吹嘘制造出一系列让你眼花缭乱的成功案例，对于很多缺少阅历和一定社会经历的，尤其是那些又急于“暴富”的人，“你敢说，我就敢信”， 这招经常屡试不爽，但又不是所有靠包装就一定能让人相信的，所以我又说是“部分”成功人士 。




　　 而至于那些原本的成功人士，很多时候讲话又模棱两可，说话又不讲透，甚至也有很多虚构水份在其中，这类人说话往往也很容易让人信服，毕竟是成功人士嘛，让人一开始就有了先入为主“你说我就信”的想法。成功的人生，如同一道夹杂了太多随机因素最后得出成功结果的复杂方程式。你可以说某个因素可能是其成功的因素之一，但不能简单武断地讲成功就必然要靠某个因素，这显然也是荒谬的，大多成功的人士应该是明白这个道理，但讲出来的内容多未必就是如此。如同我们看三国演义，看着挺激动，但翻开真正的三国历史，很多事情就未必那样，甚至两码事。成功人士也一样，很多故事可能是真的，即便夹杂了些其他成分，但若真正放到现在的现实生活中，未必就一定能再现昔日的成功，原因很简单，成功不是一个简单的方程式，在这过程中有太多的随机因素。 




　　因此在我看来，想要获取更多的财富，真正有效的捷径还需靠自己的努力和付出，如前文所提，无论是金钱，或者是时间，都是有成本和代价的，付出了才可能有收获，但如果不想付出太多，总将这一理想建立在他人不劳而获的消息上，那显然是荒诞的。而真正那些坐拥财富的成功人士，其实也不必完全迷信，没有谁能完全预测未来，很多所谓的“成功人士”站在那高台上大谈特谈，在我看来其背后多数也是有其目的，如同电视里的股票”专家”，大家都是成年人了，闲来听下无妨，但若真以为自己获得了“灵丹妙药”发财致富的“窍门”，就很可笑了。




　　过去的经验可能会对你未来有一定的帮助，但若完全依赖或相信经验，也会令你丧失掉更多机会。真正有效通往成功的道路唯有不断学习，不断总结，依靠自己最新的判断做出自己认为最适合的决策。




　　如果你看好某个项目，借用小辉说的一句话：在没有执有一万个币之前，人生目标应该就是埋头屯币，屯币屯币再屯币，不参与是非，不争论不发声；当然，等你有了一万个币后，也不屑于去争论发声了。:-)

世界很大，世界很小



　　大概是在十七年前的时候，我在北方一座二线城市里做教师，学校里的生活是比较单一的，那时候我每天最大的乐趣有两个，一是每天下晚班后，黄昏的时候去学校附近的夜市上吃上一碗擦疙瘩(面食)，然后喝上一大杯扎啤；另一个乐趣就是闲暇的时候去网吧上网，打会游戏，浏览些网页。

 


　　就是在那个时候，我浏览到了一个网站，名字至今我还清晰地记得，程序员小辉的网站。我知道小辉应该是个人名，也知道程序员是个高大上的职业，准确讲那时候做程序员是个很神秘的职业，为什么我要这么说？因为在我眼里，能够让一行行代码编程形形色色的我们使用的电脑软件，如魔术般神奇，真是让人非常惊叹。

 


　　后来我换了几份工作，期间从前访问过的很多网站都打不开了，其中不乏一些曾经的知名网站，程序员小辉网站好像也改了几次版，但一直都坚持在更新着，我也陆陆续续一直保持着在闲暇的时候也会翻看下这个网站，直到前年七八月份吧，一次无意中我又访问了程序员小辉网站，看到小辉提到了一个神奇的网站(steemit)，发文章可以获得收益，从此之后仿佛打开了一个新的世界，我也从这个时候开始认识和了解到了虚拟货币、区块链等相关知识。

 


　　时至今日，从我开始接触区块链至今也有近两年，在这过去的近两年中，我收获了许多，不但是学到了区块链相关知识，更是在很多如思维方面改变了我过去的一些思考方式，认识了很多朋友，总之，我因此受益非浅。

 


　　有时候我也想过，如果没有我十七年前看到过程序员小辉的网页，如果我没有一直陆续都在坚持阅读他的网站，如果我没有那种时刻对周围新鲜事物保持的那点好奇心，如果。。。我想到今天，我一定不会收获如上很多。

 


　　因此我的内心是非常感谢他们，感谢程序员小辉带给我新的东东，不但让我学到很多也让我获益许多。我也感谢那些曾经支持和帮助过我的朋友们，虽然很多朋友至今都未曾见过面，但是我仍然要发自内心的对你们说声谢谢。

 


　　今天跟小辉聊，末了他发了个信息给我：世界太小，不是这是(里)，就是那里，都能遇得到:)。

 


　　我深表赞同。这个世界，真的很大，然而这个世界，真的又很小哦。

“月旦评”第九期(五月)正式开始


各位社区小伙伴们，



　　大家好！


　　今天是五月份的第一天，五一劳动节，也是国内五一连续三天放假的最后一天。“月旦评”第九期(五月)正式开始了。


　　从本期开始，我们正式做了如下改进：


　　一、为了更公平、公正地给到每一篇推荐入选的“月旦评”作品，评委组老师们经过讨论和沟通后决定，由原先的每周打分改为每期公众投票环节时统一打分，分值取所有评委老师们的打分均值。


　　二、为更好让所有人随时参与“月旦评”，避免每期投稿的空档时间，我们将月旦评每期的三次(周)推荐该为四次(周)推荐，这样每期的投票“评”将会基本固定在每月月初时间，比如现在正在进行的第八期公众投票。这样以来，将不会再出现投稿时发现恰逢暂停投稿的空档尴尬，让所有社区成员随时均可参加“月旦评”。


　　为了更大程度活跃社区氛围和鼓励更多创作，只要文章符合“月旦评”的参赛规则，随时都有机会被我们的评委老师们推荐入选并有机会获奖，具体参赛规则如下：


　　1、所有参赛的作品均需原创，杜绝盗版、抄袭，参赛作品用中文(简体、繁体均可)。


　　2、参加“月旦评”文章标题需要包括“月旦评”三字，所有“月旦评”的文章需加标签cn-reader;目前暂设文学cn-literature和艺术cn-art两项，即如果你认为自己的作品属于文学类，投稿时需要增加标签cn-literature,如果你的作品是艺术类型，则增加标签cn-art，每个对应的标签都会有相应的评委组老师们分别审阅。当然，如果你无法判断自己的作品类型，可以只需仍然按照先前加cn-reader即可。


　　参考例子如下，


　　如文学类作品加标签:cn-literature cn-reader


　　艺术作品标签:cn-artcn-reader


　　未知类型或其它作品暂归:cn-reader




　　3、每篇的文章内容文字在500-3000字(我们建议一般保持在1000字左右最佳)，诗歌文字不做限制；




　　4、本期入围文章以评委组推荐为主，评分标准：评委打分+公众投票；




　　5、“月旦评”优秀作品有机会被翻译成其它语言包括但不限于英文等，并向非中文区进行推广宣传；


　　本次活动设立如下奖项（奖金在第八期公众投票前公布）：


　　最佳文学入围奖3名（最佳文学奖从中最后决出）


　　最佳艺术入围奖3名（最佳艺术奖从中最后决出）




　　所有入围“月旦评”的作品，均会得到一定程度点赞鼓励。


　　“月旦评”活动不设主题，文章内容不限，包括但不限于文学、短篇、小说、散文、诗歌等；


　　再次声明，“月旦评”活动是社区里长期举办的重大活动，旨在鼓励大家多创作、发现优秀作者，每月三轮(即每月前几周每周推荐一次)，由评委组老师们推荐优秀文章，入选的文章会在下一月初(即最后一周)统一做评比，评比结果由评委打分+公众投票来选出最终获奖者。




　　“月旦评”活动的宗旨皆在鼓励社区更多创作作品、发现优秀作者、推广社区发展更繁荣璀璨。我们在这里欢迎社区的每一位小伙伴们都来参加“月旦评”活动。



　　“月旦评”第八期优秀文章公示暨正式投票(投票截止北京时间5月6日23:59) 正在如火如荼进行中，猛击这里一起来为你喜爱的作品投票吧！

为什么我们会感到 焦虑 / 愤怒 / 迷惘 ？


　　我们为还未发生的事情感到焦虑，为发生的一些事情而愤怒，不清楚自己想要什么而感到迷惘，这些很多不良的情绪，如同疾病一样不时会出现在我们身体中，令我们感到失望、失落，让我们的生活变得无序甚至混乱。




　　为什么我们会焦虑？我们为什么会愤怒？我们为什么又会迷惘？




　　很多时候，我们的焦虑主要是来自一些尚未发生的事情，我们总是陷于焦虑的情绪中，以至于夸大了其所带来的不良后果，但是却很少去花多些时间去分析这焦虑背后的原因，并尝试去努力解决它。




　　比如我们准备要做一件重要事情，随着时间的推移，越来越临近时，我们会愈发觉得焦虑，焦虑可能是因为我们对自己准备的不充分，甚至可能还未开始正式准备，又或者对已准备好的缺乏信心。我们陷入一种焦虑的情绪中，反复出现，我们试图通过其它一些事情来转移这件事情对我们的注意力，甚至麻痹自己，但屡次之后都会失望，而且更让我们的焦虑情绪愈发加重。




　　避免这些负面情绪的解决办法，首先就是找到问题的来源。要想减缓焦虑情绪的继续蔓延，就应该立即开始着手准备要做的事情，哪怕是刚刚开始准备，也会让你的焦虑情绪很快有所缓解，直到你完成准备工作，并再三确认过后，这时候你会如释重负般不再焦虑。




　　焦虑的另外一个原因，就是源于我们通常对未知事物的“恐惧”，因为长期进化的原因，我们多数时候习惯于无论是身体或者是思想总待在让自己觉得舒适的位置里，相比较那些未知的事物，我们内心更习惯和偏向于那些已经熟悉的认知内容，甚至有些条件反射般排斥新生或陌生事物。避免内心对未知事物产生“恐惧”的最好办法也是马上着手，向着你内心那块先前不熟悉的区域进军并熟悉掌握它。




　　通常情况下，熟悉的人让我们产生的愤怒，要远大于那些陌生人让我们产生的愤怒。我们习惯于在外对陌生人客客气气，将自己最友好的一面展示给他人，却往往将自己最不耐烦和最易动怒的坏脾气给了自己最熟悉的身边人。




　　为什么我们会这样？最根本的原因有两点。一是我们对自己熟悉的人太缺乏耐心，二是我们对自己熟悉的人要求过于迫切。前者毋需我解释，这里主要解释下第二点，对自己熟悉的人要求高本不是件坏事，但因为太过于迫切和操之过急，显然违背了事物的发展规律，从而一但未能达到自己内心中的想法时，这个时候最容易情绪失控而感到愤怒。




　　弄清楚产生愤怒的原因后，解决的方法就是对症下药，即做事尤其是对熟悉的人时，锻炼自己要更有些耐心，要坚信无端的抱怨指责和批评只会让双方的情绪变得更坏，让事情变得更糟，甚至一发不可收拾而却于事无补。要从内心真正意识到罗马真的不是一天就能建成，凡事都要有个过程，急于求成的结果往往是心急吃不了热豆腐反而更容易揠苗助长，让自己得不偿失。




　　迷惘背后的主要原因，多是来源于我们对自己目标的模糊和不确定。如同一艘船在海上行驶，如果没有一个目标的话，这艘船多半会在海上任流漂泊。没有明确目标或者目标设立不实际的人，多会在碰到问题的时候，容易陷入对未来生活的反复迷惘中。

真是有些奇怪，按了Ctrl+c终止后，又出现如下信息：





以为allow操作出了问题，于是换个命令:listkeys 试下，仍然同上错误。




改换了下输入：steempy –help 测试了下，见下图：







看这样子steempy是正常的呀，那会是哪里出了问题呢？




毛主席教导我们，看问题一定要善于抓住主要矛盾，于是仔细在错误的异常里查看，发现有如下一句异常提示：


Failed to establish a new connection: [Errno -3] Temporary failure in name resolution




看这样子是出现在网络连接方面的原因，于是输入命令测试下网络：ping baidu.com




系统返回:


ping: unknown host baidu.com


果然是外网不通的原因引起的。


于是修改 /etc/resolv.conf 文件，增加如下:


nameserver 8.8.8.8


然后再输入命令 ping baidu.com ，返回如下图：





网络恢复正常。




这时候再重新操作steempy 同时进行授权操作，一切正常。
