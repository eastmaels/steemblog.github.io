[EOS学习笔记]database dirty flag set (likely due to unclean shutdown): replay required 问题的解决方法

今天在开始打开 EOS: tail -f nodeos.log
结果发现提示：
database dirty flag set (likely due to unclean shutdown): replay required 见下图，

what?这个是什么问题？我啥也没做呀！

在网上查阅了下资料，才知道这个问题是由于上次 nodeos 非正常退出时引起的。

查询下nodeos –help，发现里边有这三个参数，

于是采用第一个参数：
nodeos –replay-blockchain

见下图

重新查看后，发现开始正常出块了。

“卒中优先” 是什么意思？

昨天上午带老妈去医院看腿关节，取药的时候，看到窗口前有个小牌子，上写：

　　卒中优先


　　我一时有点不明白，这个“卒中优先”中的“卒中”是什么意思呢？

　　回去的时候，特意在网上查阅了下，大意如下：

　　卒中包括脑梗死脑出血，广义上讲就是脑血管疾病和脑血管动脉硬化引起的疾病。

　　虽然可能不是完全准确，但也大致明白了意思。

　　八卦下，在网上搜索时，“卒”这个字我一直读“zou”，但却始终打不出来，后来查阅后才发现这个“卒”的正确读音是：zú，在象棋中用于兵的意思，但另外一层也有病了死亡的意思。

　　现在的医院，从预约挂号到缴费、化验单、片子等，多数都开始采用机器了，一台台机器放在过道，用的时候非常方便。


　　不知道过N年之后，连看病的医生会不会也都是机器人了呢？：）
