Mac系统下No such file or directory的问题解决

　　我在Mac下安装了个软件，需要手工对文件进行配置下，于是打开terminial终端直接进行操作。

　　有一个很怪异的地方，就是我进行操作的时候，明明当前目录下有那个目录文件，但我使用命令的时候，却死活进不去，而且系统显示：

　　No such file or directory

　　这真是奇了怪了。刚开始时，我以为是自己输入的目录名称有误，还专门核对了下，再次输入操作，问题依旧，显示无该目录。

　　于是我改复制该目录名称，然后再粘贴进行操作，发现仍然无济于事？

　　难道是操作权限的问题？我心中疑惑到，但想了下觉得不对呀，

　　我这个当前操作身份本身就是系统管理员的身份啊。

　　百思不得其解，只好上网百度下，发现在Mac系统下，空格要转义符，比如一个目录名称为：

　　my directory

　　操作的时候应该为：

　　cd my \ directory

　　在空格前边加多个”\”转义符即可。

　　又或者另外一种方法，即将目录名称用””起来，如上述目录操作为：

　　cd”my directory”

　　这样就可以了。

　　返回Mac重新操作，果然问题解决。

谨慎的司机

我刚开始学会开车的的时候，有一次去清远，路上不小心走错了路，不知道走到什么地方了，感觉像到了农村，两旁是庄稼，水稻🌾什么的，路也很窄，时不时马路上还会出现一头牛🐂，对于我这种当时刚拿到驾照没多久的人，心里紧张自然不消说，好在我的驾照不是买来的 囧。

　　不知道走了多久，前边马路上出现了类似石桥那种，两边是石墩，中间留了一条路，我目测了下，感觉那个宽度不是很宽，但又貌似车身能通过的样子，想直接冲过去，又担心被卡在那里，一时犹豫不决，不只担心那个石墩中间的宽度，更对自己的驾驶信心不足。

　　感觉犹豫了好一阵子，这个时候马路对面过来一辆车🚗，只见那车经过石墩的时候，稍微放缓了下速度，随即很顺利地通过了。我在旁边看到这景象，顿时来了信心，既然对方能过，我一定也能通过。

　　于是开车从石墩中间，小心翼翼有惊无险地通过了。


谨慎的女司机，有图有真相

　　今天看新闻，说深圳南山区登良路，有一女司机倒车入库时，因为不能确定倒库是否可以，于是下车用卷尺丈量停车位宽度，确定没问题后才小心将车倒库。

　　有网友评价说，这才是最谨慎的女司机，也有人为她的行为点赞，认为这是为自己也是为他人负责。

　　看到这里，我不由心想，要是当初我也随身带把卷尺，经过那个石墩路口的时候，提前下车丈量下，说不定现在也会成为“史上最谨慎的男司机”了吧？

　　可惜我当初随身没带那把卷尺，错过了一次成为网红的机会 :(。

无人超市关门了!

小区附近的那家无人超市关门了。

早在几个月前，我已经预感到了，这家无人超市可能要关门，因为我时常经过这家店门口时，留意到店里总是几乎没什么顾客，店门口用门可罗雀形容也不合适，因为压根门口就没人。事实上，这家无人超市从开业至今，已经勉强撑了一年多了，对于一般的小店，比如小饭店等，一般的存活观察期也不过三至六个月，能够撑过去，说明暂时还有存在的价值，比如饭店的饮食种类，客户群体价格定位等。反之，如果很快关门，要么是定位不清晰，要么是顾客不买帐，总而言之，成功的店各有各的成功之道，而失败的店，大多都略同。

近几年来，提起创业，很多人首先会想到开店，比如超市(士多)，原先那种单打独斗的店面，在竞争白热化的今天，远已经无法满足城市发展的节奏，以及人们日益对生活和品质的更高要求。

于是许多出现了两种分化，一种以沃尔玛、百佳、家乐福等这种国际大超市，驻扎在人口密集或人流量巨大的附近，另一种以加盟连锁的小型超市，如7-11，美宜佳、芙蓉兴盛等，专门选在较远离大型超市，却靠近居民的位置。人们每天下班或闲暇时候，又或者临时需要购买些东东，却又不想兴师动众跑到较远处的大超市，对于有些人来说，比如像我，毕竟去大超市里，看到货架上玲琅满目的东东，甚至一种商品出现非常选择时，每次选择都不见的是件轻松的活。

前两年的时候，随着电子支付的越来越普及，无人超市的概念也开始兴起，小区楼下的这家小超市，就是在这样的趋势下诞生了。

刚开业的时候，和大多数人一样，我也感觉很新鲜，专门跑去体验了下，从进门到购物，一直到付款后离开，无人工干预，真是件挺稀奇的事情。后来人脸识别支付出来后，也是一样，冲上去体验了几把，几次下来后，就渐渐平淡了许多。

毕竟，新鲜感不是一直都会有，见多几次后，也就觉得稀松平常不那么新鲜了。

慢慢的，我还发现了这家超市的一些其它缺点，最主要的莫过于人们普遍关心的价格问题了。虽说每次去超市，绝大数东西我一般都记不住价格，但对于经常购买的一些东西，我的印象还时很深的，比如我常喝的百威500ml罐装啤酒，小区其他超市一般售价6元或6.5元，这家超市售价7元；又比如脉动，一般4元，而这家超市售价5元，虽然只多1块大洋而已，但这个价格对比其他超市，没有半点优势，相反都是劣势，价格高嘛。

这家超市存在的另外一个问题，也是绝大多数超市都普遍存在和面临的问题，就是商品过于同质化。你有的商品，其他品牌的超市也都有，你有，我有，大家都有，而且都是同样的厂家和同样的包装商品，别人为什么一定要来你这里购买？从前我们说7-11的有些东西卖的比超市贵，但7-11的定位很明确，就是瞄准了一些人们日常疏忽的商品来销售，比如晚上你正在做饭，却发现家里的鸡精没有，你不可能走几百米之外的大超市去购买吧，绝大多数人的选择，一定是就近买，而且这个时候因为你刚好需要这个东东，你也不会太介意价格，即便平日里家庭主妇们在超市购买时总会货比三家，但这个时候，因为是刚需，你迫切地需要，你就买了。另外，就是瞄准了年轻人的“懒”而生计，比如7-11里卖的最好的车仔面，从煮到递给你，不过两三分钟的时间，对于许多人来说，来不及吃早餐，或者饥肠辘辘的时候，车仔面就是一个不错的选择，又快又方便，味道比泡面还是好许多的嘛。

而这家超市，因为商品的同质化太严重，或者说，压根没有拿得出手的比如像7-11那样自己拳头产品，店面也不比一般的超市大，价格还贵，都是劣势或缺点，除了一个无人超市的新鲜感，其他没有任何优势了。

而在新鲜感结束了之后，果然光顾的人就逐渐很少了。

说实话，这样的店能够撑到今天才关门，也算一个奇迹了。
