**一、前期准备**

1、https://wallet.yoyow.org/#/create-account ，注册一个YOYOW 账号，获取资金密钥（active key），以备后用。

2、运行见证人节点需要至少10000个YOYO做抵押,同时创建时还有1000的手续费，所以，最少要准备11000个YOYO在钱包里。

**二、设置运行节点**

这里我是以Ubuntu系统做测试，官方教程中也是以Ubuntu系统做为讲解的。

1、下载软件包
https://github.com/yoyow-org/yoyow-core/releases/tag/v0.1.2-171026
下载文件名字为“yoyow20171026.tgz”的文件，保存到本地电脑（搬个梯子下载得快些）

在home目录下新建文件夹“ubuntu”，将下载好的软件复制到这个目录下面，解压好。

![a1.jpg](https://steemitimages.com/DQmf8nnQGMLYQEy7xK64AKq77CcNVRB9SWJgbsyVaQ59FVP/a1.jpg)

2、运行节点

![a2.jpg](https://steemitimages.com/DQmRxTx7Z5rXeyBfNnuv82U4u5Bi3Bk1Zn99UG37tnPDHG7/a2.jpg)

打开终端，依次输入下面的命令（如上图所示）：
```
$ cd /home/ubuntu
$ cd yoyow20171026
$ ./yoyow_node --rpc-endpoint
```
<br/>
等待网络连接成功，屏幕上会有每3秒一个块的显示信息（如下图所示）。大约同步半个小时左右即可。现在区块链数据较少，约为615MB：

 ![a3.jpg](https://steemitimages.com/DQmcLg8QzjM6BTXqEoeJSBefHLWv86FcTBPGEFk1gzqzaD2/a3.jpg)

**三、测试钱包命令**

![a6.jpg](https://steemitimages.com/DQmaWTeHa8NRRzwXLpeEiB6hV22qjp69p9ZskzNtg8C7dTL/a6.jpg)

1、再新开一个终端（如上图所示），输入下面的命令：
```
$ cd /home/ubuntu/yoyow20171026
$ ./yoyow_client
new >>>
```
<br/>
2、显示new的时候先设置钱包密码，比如密码是yoyow-pass，如下
```
new >>> set_password yoyow-pass
set_password yoyow-pass
null
locked >>>
```
<br/>

3、设置密码完成后，解锁钱包（输入之前输入的密码），钱包显示Unlock>>>状态即可对钱包进行操作。
```
locked >>> unlock yoyow-pass
unlock yoyow-pass
null
unlocked >>>
```
<br/>
 
**下面就可以逐个测试下钱包的命令了：**

4、查看帐号。比如账号是25638，可以用命令查看（注意一定要在解锁状态下查看）：
```
unlocked >>> get_account 25638
get_account 25638
{
 "uid": 25638,
 "name": "init",
 "owner": {
   "weight_threshold": 1,
   "account_auths": [],
"account_uid_auths": [],
```
<br/>
 
5、导入私钥，即前面保存好的 “资金密钥（active key）”。
import_key 后面跟了两个参数，一个是yoyow账号名，一个是私钥（此处导入的是你的YOYOW钱包资金密钥）。
```
unlocked >>> import_key25638 5Hqwx3xXMYZ55Pko9nzw34234234nXHcGfNQjNEL23424w7Py
 ```
<br/>
6、 测试转账（此命令是25638账号转100个yoyow给账号test，可以转给自己的第二个yoyow账号进行转账测试，）
```
unlocked >>> transfer 25638 test 100 YOYO "" true
```
<br/>


**准备和测试工作基本完成了**

到了这一步，YOYOW见证人准备和测试工作基本完成了。下一步就是创建见证人和抵押YOYO了，由于会产生手续费（1000YOYO），所以，各位可以考虑自己的情况，能否正常运行节点（不正常运行会有惩罚），对各项突发情况是否有应对之道，各个问题都要有所考虑才行。
