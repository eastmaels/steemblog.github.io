&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Markdown 是轻量化的标记语言，一种适用于网络的书写语言。Markdown 是一个面向写作的语法引擎，Markdown 的最终目的都是解析成 HTML 用于网页浏览，所以它兼容 HTML 语法，即你可以在 Markdown 文档中使用原生的 HTML 标签。
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;本文分三部分：基础语法，高级点的语法和好用的HTML标签。
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;完成本课程你可以很轻松地进行排版和编辑，steemit上的文章也可以排版得很美观，图片、视频等可以轻松得混排。
