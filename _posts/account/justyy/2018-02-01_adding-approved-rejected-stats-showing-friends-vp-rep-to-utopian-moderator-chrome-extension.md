# What is Utopian Moderator & Supervisor?
`Utopian Moderator & Supervisor` is a Chrome Extension that aims to help the Utopian moderation work easier and enjoyable!

# This Feature
Version 0.0.4 adds these [new features!](https://helloacm.com/adding-approved-rejected-stats-and-showing-friends-vp-rep-to-utopian-moderator-chrome-extension/)

### Friends ID List
You can specify a list of steem IDs so that along with @utopian-io they will be displayed in the general tab.

![image.png](https://res.cloudinary.com/hpiynhbhq/image/upload/v1517514041/vgwwegrxicelhspputtn.png)

![image.png](https://res.cloudinary.com/hpiynhbhq/image/upload/v1517514123/z7mvhgrhbiqxpwtv3ehk.png)

### Show Approved/Rejected Statistics
You need to configure your moderator steem ID first, and it will show how many posts you have approved and rejected.

![image.png](https://res.cloudinary.com/hpiynhbhq/image/upload/v1517514189/xkahlsuhdomeieuc7ma6.png)

By the way, I have [moderated](https://helloacm.com/being-part-of-utopian-moderating-team-is-great-intro-post/) **100** posts so far and the approved ratio is 50% (not too bad, huh?)

![image.png](https://res.cloudinary.com/hpiynhbhq/image/upload/v1517514214/xpigq72ksuhswbhhx5d1.png)
