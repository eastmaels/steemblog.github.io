<p>24 Poker Game is an easy and fun math games. 4 cards (exclude the kings) are randomly picked. And the player who solves the equation to 24 first wins. For example, if the 4 cards are 10, 10 and 5 5. The formula to 24 is: (5 * 5) - (10 / 10).&nbsp;</p>
<p>I have made an online 24 Poker game and an API is provided using PHP.</p>
<p>The online 24 Poker Game: &nbsp;https://helloacm.com/24/</p>
<p>You can find the source code below. The algorithm used is just bruteforce.&nbsp;</p>
<h3>API (Application Programming Interface)&nbsp;</h3>
<pre><code>https://helloacm.com/api/24/?a=10&amp;b=10&amp;c=5&amp;d=5</code></pre>
<p>It will return JSON-encoded data:</p>
<pre><code>{"errorCode":0,"cnt":1,"result":["(5 * 5) - (10 \/ 10)"]}</code></pre>
<p>&nbsp;If <em>$_GET</em> parameter <em>s</em> is not specified, this API will use the <strong>$_POST</strong> variable <em>s</em> instead.</p>
<pre><code>curl <strong>-X POST</strong> https://helloacm.com/api/24/ -d "a=1" -d "b=1" -d "c=2" -d "d=3"</code></pre>
<p>老小的时候就玩这个 24点扑克<a href="https://justyy.com/archives/237">游戏</a>. 说来很简单 就是随机抽取4张扑克牌(除掉大小王), 然后需要以最快的速度用这4张牌 加减乘除 来得到 24的一个算式. 这需要脑子运算快, 是个很好的脑力训练.&nbsp;</p>
<p><img src="https://justyy.com/wp-content/uploads/2016/03/poker.jpg" width="601" height="344"/></p>
<p>&nbsp;周末闲来无事, 刚好那天在电视上看到 花样少年 里明星也在玩. 于是就很快的写了这么一个程序. 当然提供<a href="https://justyy.com/archives/2626">免费 API</a> 地址是 <strong>https://helloacm.com/api/24/</strong> &nbsp;</p>
<h3>原理和代码</h3>
<p>代码在 英文网页上: <a href="https://helloacm.com/24/">https://helloacm.com/24/</a>原理很简单, 就是<a href="https://justyy.com/archives/2047">暴力搜索</a>. 才4个整数. 而且几种运算符而已. 注意加入了 pow 函数也就是可以有 a 的 b 次方. 整体来说4个数字只有两种方式. 就是 2个一组 即 (a, b), (c, d) 或者 ((a, b), c), d). 代码是用PHP实现的, 因为要提供API, 而且<a href="https://justyy.com/archives/2616">PHP</a>里的<a href="https://justyy.com/archives/1830">数组</a>很好用. 可以将值做为索引 (key). 这样就可以通过 array_key_exists(“=24”, $try1) 这样的方式来判断是否有结果为 24的算法.&nbsp;</p>
<p>&nbsp;最关键的可以通过 array_values(array_unique($rr)); 来去掉一些明显的重复. 关键代码如下:</p>
<pre><code>// 返回 两数 A, B 可能的值数组 (value, how) 键值对.<br>
 &nbsp;function tryA($a, $b, $aa, $bb) {<br>
 &nbsp;&nbsp;&nbsp;$a = str_replace("=", "", $a);<br>
 &nbsp;&nbsp;&nbsp;$b = str_replace("=", "", $b);<br>
 &nbsp;&nbsp;&nbsp;$r = array();<br>
 &nbsp;&nbsp;&nbsp;$r["=" . ($a + $b)] = "$aa + $bb"; <br>
 &nbsp;&nbsp;&nbsp;$r["=" . ($a - $b)] = "$aa - $bb";<br>
 &nbsp;&nbsp;&nbsp;$r["=" . ($a * $b)] = "$aa * $bb";<br>
 &nbsp;&nbsp;&nbsp;$r["=" . ($b - $a)] = "$bb - $aa";<br>
 &nbsp;&nbsp;&nbsp;$r["=" . pow($a, $b)] = "pow($aa, $bb)";<br>
 &nbsp;&nbsp;&nbsp;$r["=" . pow($b, $a)] = "pow($bb, $aa)";<br>
 &nbsp;&nbsp;&nbsp;if ($a != 0) $r["=" . ($b / $a)] = "$bb / $aa";<br>
 &nbsp;&nbsp;&nbsp;if ($b != 0) $r["=" . ($a / $b)] = "$aa / $bb";<br>
 &nbsp;&nbsp;&nbsp;return $r; &nbsp;&nbsp;&nbsp;<br>
 &nbsp;}</code></pre>
<p>计算机最厉害的就是重复的工作. 这点搜索对计算来说是小 case. 而且能把几乎所有的可能搜索出来(不排除程序可能有漏掉的)</p>
