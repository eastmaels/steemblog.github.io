今天晚上，公司两大股东（其中一个是英国总公司的HR，另一个是中国公司的总经理）请我吃饭。我本来以为我犯了什么事呢，提心吊胆，结果原来是公司为了招人而犯愁呢。

![2017-09-20 19.17.28.jpg](https://steemitimages.com/DQmdrhnbRRGV6bCkCUhCDbhiTyt4VPnWVByLEFiZMiSfzVo/2017-09-20%2019.17.28.jpg)
*鱼烤焦了*

我们公司最近在招一个 C++ 软件工程师 （C++ Software Technical Expert），但是一直招不到合适的人选。但项目启动迫在眉睫，所以想听听的我意见。客观来说，我们公司的待遇还是不错的，但是[C++](https://justyy.com/archives/2257)的牛人一直招不到有一大原因就是：剑桥市有很多很牛的大公司，比如苹果、微软。能力强的很多不会到剑桥郡的一个村庄里上班。

先说好处：
- 我们公司给办工作签证（五年[英国绿卡](https://justyy.com/archives/924)），所以如果你想来英国工作或生活，这个机会难得，因为据我所知，英国并不是很多公司愿意给外国人发工签。
- 我们公司属于发展迅速的高科技公司（无线通讯规划优化），你来了不像大公司里是螺丝钉，在我们这里可是会身挑大粱的，甚至可能会 Full Stack。
- 灵活的工作时间，生活质量会更高一些。
- 很有可能你的孩子将来上[剑桥大学](https://justyy.com/archives/2085)（至少物理距离更近了）
- 你会遇到很多能力很强的人（不一定要在 Google, Facebook里，我们公司牛人也很多），能学到不少。

具体职责：
- 用C++重构一些老的代码、模块、重写一些算法。
- 改进开发流程。
- 预言新算法，解决困难的问题。
- 领导小开发团队。
- 成为相关模块的负责人。
- 撰写高质量的设计文档。

要求：
- 很强的C/C++程序设计能力：需要对多线程、分布式、数据结构与算法都有很深刻的理解和相关经验。
- 使用 [GIT](https://justyy.com/archives/3684) 的经验
- 有领导团队的经验
- 有过参于开源项目的经验 e.g. github
- 熟悉 敏捷开发流程
