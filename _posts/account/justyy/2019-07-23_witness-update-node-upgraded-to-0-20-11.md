I replayed on my backup witness node a week ago, and it took 230 hours + this time to finish the replay. The replay was slow because:

1. the version is 20.11
2. the command "./run.sh replay" is used instead of a faster approach "./run.sh memory_replay". The replay "with memory-replay" failed twice i.e. the process hangs somehow when it seemed about to end.

Now, I 've switched to the backup node and it starts to be producing blocks on 0.20.11!
