Previously, there are four API servers: [Tokyo Japan](https://happyukgo.com), [Fremont USA](https://steakovercooked.com), [New York, USA](https://helloacm.com), and [London, UK](https://uploadbeta.com/picture-gallery/faq.php). Now, I am adding another API server in Frankfurt, Germany.

The API server will be responsible for all the API calls in https://SteemYY.com - The Steem Tools.

![image.png](https://ipfs.busy.org/ipfs/QmSu2UKvhw5nJt9hUyeC7U6kdrQ1gF7Jw6qcANMqa1vExy)

By default, a random API server will be picked up however you can bookmark your favorite server by adding the query parameter &server.

The Frankfurt API Server is currently a VPS that has 1 core only (but may upgrade depending on the traffic) - it is served by [Digital Ocean](https://anothervps.com/out/do) - **currently they offer $10 free credit.**
