## SteemTools
[SteemTools](https://helloacm.com/steemtools-update-adding-account-witness-data-option-to-save-key-and-bug-fixes/) is a Chrome Extension that aims to provide useful tools and data for SteemIt Users.

## Github
https://github.com/DoctorLai/SteemTools/

## Chrome Webstore
It is online, and you can install SteemTools via:
https://chrome.google.com/webstore/detail/steem-tools/emjfpeecopppojbhkigjjmcahbfahhbn

If you are using Firefox, you can still install this Extension by [Chrome Extension Foxified](https://addons.mozilla.org/en-GB/firefox/addon/chrome-store-foxified/).

## Technology Stack
Javascript that runs in the Chrome Browser (Chrome Extension)

## New Features of SteemTools v0.0.10
[This commit](https://github.com/DoctorLai/SteemTools/commit/47844d0d94660c3f4a7fcc433a634f1d224de7f8) adds the following changes:
1. Account Witness Tab, which allows you to view a list of witnesses that you vote
2. Option to Save Private Key so the users don't have to worry about the security. Use it once and Dispose it.
3. Bug fixes.
