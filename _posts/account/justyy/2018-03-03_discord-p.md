O 哥的公众号里有一个 ?pu 命令很方便，可以根据当前的行情，市场价 feed price 等信息自动给出是否应该是 100% 发文或者 50/50。今天有点时间，整合到 Discord 机器人，还有[公众号](https://justyy.com/archives/6086)，还提供了API对接。

# 公众号 justyyuk
![](https://justyy.com/justyy-wx2.jpg)

直接输入P，大小写都可以。
![image.png](https://res.cloudinary.com/hpiynhbhq/image/upload/v1520081937/vgbraunqkaibcywrmi4r.png)

或者 P 后面加个数字，代表 SBD的数量

![image.png](https://res.cloudinary.com/hpiynhbhq/image/upload/v1520081978/y12x6lpekiaetktwgd0e.png)

最后面会跟据两种情况下的 美元数量给出 三种建议，其中如果两者差别小于1美元，则会显示无所谓。

# API
可选参数 amount 默认为100
`https://helloacm.com/api/steemit/pu/?amount=500`

返回：
```

```
