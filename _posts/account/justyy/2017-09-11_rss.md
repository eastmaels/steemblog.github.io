I have recently upgraded one of my VPS server (UK), which is used to improve the user experience of Steemit-Wechat Group RSS/Ranking/API services. It costs around 45 USD (inc. VAT) per month.

我的朋友都知道我从2012年开始就自掏腰包租用 VPS 主机，用于写博客和各种小工具程序。为了提高更快更好的[RSS](https://justyy.com/archives/5237)文章服务，我把我英国的[VPS](https://justyy.com/archives/1344)升级了，这样每个月费用是45美元（包VAT）。

After upgrade,
升级效果为：

![](https://steemitimages.com/DQmYc7cBhbCWb2oYsg1Bq1CftZtDBsLSXX9Rmo57Qa2ctKd/image.png)

Four cores, 8GB RAM, 100GB [SSD](https://helloacm.com/vps-review-vultr-high-performance-ssd-cloud/), this is pretty high-specs for blogging. This server currently is used for the following:

1. SteemIt Wechat Group RSS: https://helloacm.com/tools/steemit/wechat-ranking/rss/
2. Steemit Wechat Group Ranking: https://helloacm.com/tools/steemit/wechat-ranking/
3. Some SteemIt [APIs](https://helloacm.com/steemit-api-two-apis-to-get-the-followers-and-following-list-in-the-wechat-group/)
4. [Good Content Upvoting Bot](https://steemit.com/cn/@justyy/a-good-content-upvote-bot-cn) @justyy
5. My own Image Hosting with API: https://uploadbeta.com/picture-gallery/upload-photo.php

4核，8GB 内存，100G [固态硬盘](https://justyy.com/archives/3948)，这在博客界算是土豪配置了。这英国的主机现在主要用于以下服务：

1. 微信群 RSS 文章列表：https://helloacm.com/tools/steemit/wechat/rss/
2. 微信群 名单及排行榜数据：https://helloacm.com/tools/steemit/wechat/
3. 一些 STEEMIT [API](https://justyy.com/archives/5241)
4. [优质内容点赞机器人](https://steemit.com/cn/@justyy/a-good-content-upvote-bot-cn)
5. 还有就是我自己弄的免费图床和API：https://uploadbeta.com/picture-gallery/upload-photo.php

With this specs, now I can focus on experimenting,  writing some [useful tools](https://helloacm.com/tools/), which will be shared with you soon.
有了硬件配置的基础，我就也可以大胆的做研究、写程序、写点有意思的[工具](https://justyy.com/archives/3404)分享于大家。

Of course, the VPS is also equipped with the [CloudFlare CDN](https://helloacm.com/cloudflare-launches-flexible-page-rules-plan/), which provides a better caching. It is always pleasant to know that your server runs smoothly and provides good services.
当然，我还付费了20美元的[CloudFlare Pro](https://justyy.com/archives/3057)帐号用于提供各种缓存，这样一来，服务器响应速度快了，心情很棒~！(说真的，看着服务器配置好，健康状态良好是很令人愉悦的事情，这远远比天天盯着[收益](https://justyy.com/archives/5269)强多了）
