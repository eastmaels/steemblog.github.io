加入银行股东需要 自己的 SP，新号一般会得到系统代理的20多SP 不是你的，所以不能加入股东，但是却可以加入足球队，完成梦想。

# 加入足球队的好处
你需要把 posting key 给我（加我微信 ACM-er) 你会得到：

1. 每分钟自动领赏，从此你不需要自己去点那个 claim rewards.
2. 您答应跟随 @justyy 点赞，点赞比例由程序设定，点赞收益归你。当足球队员能量少于20%将会休息不跟随点赞。
3. 您将得到 和 股东不一样的点赞比例，当前暂时设定2-5%左右（根据能量调整），并且每24小时只能被点赞一贴。
4. 您得到的是大约10几个机器人的点赞。**其中 @justyy  等级69**，对于新人升级是非常有帮助的。
5. 您的隐私将得到保障（我不会公开说你把 KEY给我了）
6. YY足球队只会用你的 posting key 来点赞和收益，承诺不会做其它的事情（比如发文）。

# 退出足球队
最好，你告诉我你要退出了，我就会将你的KEY删掉，当然你也可以更改您的密码，这样 posting key 将会重设。

这是抱团取暖的一种方式，适用于一些不爱自己点赞的人，当然你也可以考虑加入 YY银行，成为股东，两者不冲突。
