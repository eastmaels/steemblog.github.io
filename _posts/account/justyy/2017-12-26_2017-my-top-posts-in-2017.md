# 博文统计
我有一个个人博客 [https://justyy.com](https://justyy.com) ，通过以下[SQL查询](https://justyy.com/archives/4048)：

```
SELECT 
   Count(1) 
FROM 
   `wp_posts` 
WHERE 
    `post_type` = "post" and 
    `post_status` = "publish" and 
    date_format(`post_date_gmt`, "%Y") = "2017"
```
