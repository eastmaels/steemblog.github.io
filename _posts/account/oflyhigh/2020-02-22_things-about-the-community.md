# 社区 = 账号+category

首先，社区其实就是一个账户+一个category，比如我花3 STEEM创建了一个新的社区，那么就会得到一个新的账户，比如当前的社区@hive-169270，steemd.com上看一下，会看到如下信息：
>![image.png](https://cdn.steemitimages.com/DQmeVt6aoAextkRY7hY7jMJ4eraG24TjoCd3Na5H8SYrsZi/image.png)

然后我们可以像查看其它类目(category)下的内容一样，通过如下链接查看此类目下的内容：
>https://steemit.com/trending/hive-169270

而所谓的类目(category)，其实就是我们之前发帖时用的第一个标签(tag)

# 社区并不排它

我们知道现在已经有好多社区了，假设已经有了一个“Friends”社区，那么我也想建一个“Friends”社区，那是否可行呢？

测试了一下，这并没有任何问题，所谓的社区名，描述等等，不过是附着在社区账户上的一种属性，所以可以任意修改：
>![image.png](https://cdn.steemitimages.com/DQmSXPrScuHKS8Ewx1YqHbrrqofyei86jnFrtVqb5AyXg56/image.png)

所以，不存在像域名抢注一样，抢先注册社区，完全可以弄一样的社区名，然后看哪个更善于经营。

# 社区的中文名

我想中文区的朋友，肯定更倾向于使用中文名称的社区，看着会更亲切一些，那么社区是否支持中文名呢？

我试了一下，将我之前创建的***DIY***，后来改为***Friends***同名社区的社区，再次改名为***STEEM 中文社区***，发现并没有任何问题。
>
![image.png](https://cdn.steemitimages.com/DQmQhxLgXd4nNN5J72872ccVSN59f3PB8h9z4YDaQbFSHPH/image.png)


所以大家可以放心的创建诸如***电脑论坛***、***汽车论坛***，等各类论坛了，当然了，要是不怕被踩的话，还可以创建一个***色情论坛***啥的，哈哈哈。

# 社区的管理员

社区可以有管理员/版主/成员/访客/黑名单几种级别，这已经和论坛差不多啦。

>![image.png](https://cdn.steemitimages.com/DQmaYAvwjmyqX7aRrjNoNgcRfNRXgBWtaCAxf7q7ioMopUB/image.png)

所以依赖于这个社区功能，可以兼顾区块链的上链&不可篡改以及论坛的强大管理功能。

另外，因为社区有自己的账户，所以如果有需要的话，管理员也可以转移社区给其它人，这点比较好。
