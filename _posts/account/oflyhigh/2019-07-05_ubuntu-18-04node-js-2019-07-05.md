尝试在一台Ubuntu 18.04的VPS上部署一个别人弄的项目，需要安装node.js以及npm等。可是使用Ubuntu自带的包管理器安装的node.js以及npm版本太低，导致部署项目时出一堆乱七八糟的问题。

![](https://steemitimages.com/DQmRkLq6rRew3mHfx4vYGWyqpC8wSebLPeC2iZCXAdpuGkR/image.png)
(图源 ：[pixabay](https://pixabay.com/))

# 旧的安装方式

安装nodejs与npm使用的命令为：
>`sudo apt install nodejs`
>`sudo apt install npm`

安装之后查看版本号：
>`node -v`
输出为：v8.10.0

>npm -v
输出为：3.5.2

# 新的安装方式

为了解决版本的问题，我搜索了半天，最终在[NodeSource Node.js Binary Distributions: Debian and Ubuntu based distributions](https://github.com/nodesource/distributions/blob/master/README.md#debinstall)找到了安装指南。

>`curl -sL https://deb.nodesource.com/setup_10.x | sudo -E bash -`

上述代码下载并执行setup_10.x这个脚本，这个脚本做了许多工作，比如说：
>* Populating apt-get cache...
>* Confirming "bionic" is supported...
>* Adding the NodeSource signing key to your keyring...
>* Creating apt sources list file for the NodeSource Node.js 10.x repo...
>* Running `apt-get update` for you...

执行完上述几项工作后，我们需要手动执行以下代码来安装Node.js 10.x 以及npm
>`sudo apt-get install -y nodejs`

安装完成后查看版本号：
>`node -v`
输出为：v10.16.0


>`npm -v`
输出为：6.9.0
