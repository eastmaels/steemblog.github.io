If I was able to install only one extension on my browser and stay with it for the rest of my life, the answer will definitely be the holy <a href="https://chrome.google.com/webstore/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm?hl=en">uBlock Origin</a>. It is available on multiple famous browsers and goes ultra popular in the community. Found this very handy extension out not long ago and immediately it beats all the adblockers I've been used.

Not only it blocks the annoying ads, it provides some degree of privacy as well. Occasionally it bypassed some country restricted sites for me without even informing me. Every browser vendors should adapt this extension into their build seriously(of course it won't happen).
