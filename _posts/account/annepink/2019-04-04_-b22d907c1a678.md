正好下午提早回到家
突然接到妈妈打来的电话
电话那头妈妈问我在哪 ... 我说刚到家呢...然后妈妈说：我到了你家门口的车站这里 ...我心想妈妈肯定是送蔬菜来给我吃了...妈妈接着说：你亲自下来一下，我要看下你 ... 听到这句话...顿时感到太惭愧了 ...😳

因为是我已经几周都没回家跟爸妈团聚了...
妈妈确亲自登门来看我这个不孝女...😥
于是我马上想着拿个红包装点零钱给妈妈 ...等下好塞给她~
与妈妈见面后 ...我连叫了几声妈~ 看着妈妈精神还不错的样子 ... 我泪目了😭写到这...我又落泪了... 真的 ...我很难过 ... 几天来好像压抑已久地情感不禁迸发出来 ...

然后突然感觉妈妈从来没有过的急切疼爱 ...一直在问我怎样怎样 ...我真不知怎么回答是好 ...鼻酸心颤起来 ...
我说 ：妈，我有空会回家看您们的，你本来腿脚就不是很好了 ...才刚恢复就不要跑这么远了... 可是妈妈说：我坐公交🚌反正不要钱，就当溜达玩下 ...唉...妈妈这么说虽说也对，但她腿因为年前发生了比较严重的病，我真不忍心她跑这么远来给我送菜 ... 我心里更难受了😭
接着妈妈说给我带来了小青菜和大蒜 ...还有我稀饭的香辣豆腐乳...还说菜都帮我清理好了...洗洗就可以了
然后接着拿出一大红包袋 ，说这些钱全部给我 ... 自己存起来... 不要跟其他人说，自己保管好写我的名字 ...
我，，瞬间眼泪夺眶而出 ...长这么大 ...真的是第一次听到妈妈说给这么多钱给我 ...足足九千块都是妈妈的退休工资省出来的 ... 她还说她自己有钱够用 ... 说自己年纪大了 ...  不需要这多钱 ...叫我留着...我不知道我该用神马言语来拒绝 ...我知道这都是妈妈为我...希望我好 ... 浓浓的母爱直蔓延我的心底 ...

真实温暖的爱意 ~~从来不局限于大风大雨 ... 而在于小小的关怀和惦记 ... 记得你缺什么 ... 就给什么 ... 只要我能给得起 ...这就是妈妈 ...妈妈的爱永远心意满满 ...甜在心里 ~~💓
然后我带妈妈到家附近的华润万家逛了超市，就给她买了一箱酸奶和塞了个小小的红包给她，她都很开心~
我和妈妈就像两姐妹似的有说有笑 ...

送她到回去的车站，看着妈妈上了公交🚌我的心疼啊 ... 余生希望我能尽最大努力陪伴她 ... 爱妈妈💓💓💓
祝妈妈身体健健康康💕
