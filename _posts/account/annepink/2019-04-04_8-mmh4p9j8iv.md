所有的相遇和回眸都是缘分 ... 当你爱上了某个人 ... 贪恋某个眼神 ... 意味着你心系一份铭心刻骨地缘分 ... 爱...💓 是不经意地对视眼中的彼此 ... 不必言语地表达 ... 一举一动滴眼神皆可示爱 ~~~此时无声胜有声啊 ~~~

每次听到这首歌的时候都会有心软的感觉 ... 我很享受这个过程 ... 旋律优美动人心弦 ...酒不醉人歌自醉啊 ... 特别是那无敌的萨克斯🎷前奏 ... 于第一句歌词衔接超稀饭 ... 词作人将爱人的眼神喻成一阵细雨洒落心底里 ... 生动地道出了眼神传递出的百转千回 ... 不管是明亮的 ... 美丽的 ... 还是多情 ... 妩媚的 ... 在恋人间都是世界上虽无声却动人的风景✨

这首歌听着太有魅力了 ... 还是老歌最经典👍 无论什么时候听都使人心中感慨万千啊 ...
