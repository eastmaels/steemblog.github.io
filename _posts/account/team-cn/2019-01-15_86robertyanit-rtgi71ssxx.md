阿盐@robertyan加入steem快一年了，但是最近才开始使用steem，算是个新人。

这个新人也是有趣，在自我介绍处@了一群大佬，就是没人理他。。。最后看他那么惨就把他捡回村里来了。

和阿盐聊天，发觉他具有老K @kelvinzhang的理性分析能力还具有机机@julian2013能说会道的瞎掰能力,是个有能力的人才！

大家来欢迎阿盐～

&nbsp;

<strong>问：@robertyan 您居住在哪里 ?</strong>
<div>上海和杭州待的比较多，据长辈称世代居住江南</div>
<div></div>
<strong>问：您的职业？</strong>

IT公司工作，参与管理、产品和技术等。我的工作和生活哲学是以创业者的视野、黑客的精神、研究者的态度，追求天人合一、促进人类文明的发展。

&nbsp;

<strong>问：希望大家怎么称呼您？</strong>

正在阅读此文的新人你们好，村霸龙虎豹及麾下一众热血少年团组成的黑帮会帮你们命名或重命名。。。所以不用想太多，这个问题是个坑，请相信我。。。

&nbsp;

<strong>问：什么原因让您加入Steemit?</strong>

之前注册的时候主要在写下一代互联网发展的研究报告，因Steem是一个很典型的区块链生态类型，故而注册了；注册以后一直搁着忘记使用。最近想起来应该实际体验一下，于是开启了这段Steem之旅。

&nbsp;

<strong>问：您主要分享什么类别的内容？</strong>

我觉得人生每个阶段想创作的内容会不太一样。这个阶段我觉得主要以冒险者、探索者和侦探的姿态，通过写作去更深入的发现和认识世界，并帮助充满好奇心的人们更好地认识世界。具体的内容可以参见我的写作计划。

&nbsp;

<strong>问：您想什么类似的朋友关注您 ?</strong>

龙、虎、豹、狗、浣熊、猫、兔、机器人、西瓜、妖怪、老道什么的，各种动物、植物、微生物、无机物都可以啊。。。

&nbsp;

<strong>问：您有什么爱好 ?</strong>

可以分成爱和好来讲：
<ul>
 <li>爱：
<ul>
 <li>（1）大爱：爱人类这个种群和她的生存环境，希望可以发展地更好；</li>
 <li>（2）小爱：爱家人朋友，希望可以给他们带去幸福快乐。</li>
</ul>
</li>
 <li>好：
<ul>
 <li>（1）格致：尽可能地通过学习、思考和观察了解与认识真实世界，以求天人合一；</li>
 <li>（2）利万物：希望自己在经历了时空之后创造的东西可以帮助人类的生存和发展。</li>
</ul>
</li>
</ul>
&nbsp;

<strong>问：能谈谈您的 Steemit ID 由来吗 ?</strong>

可以分成robert和yan来讲：
<ul>
 <li>Robert：小学的时候喜欢机器人，所以取了一个和robot最近似的英文名；</li>
 <li>Yan：严光（严子陵）本姓庄，后人避汉明帝刘庄讳改其姓。南华真人庄周为吾先祖。</li>
</ul>
&nbsp;

<strong>问：还记得您刚在Steemit起步时的情况吗？</strong>

上周（2019/01/03）正式开始起步吧，之前随便刷了几下。感觉进了新手村以后真正开动了，这里是个有爱、包容、多元的大家庭，村民的精神状态很吸引人。我平时并不怎么参与在线社交网络，但新手村的氛围很特别，和很多论坛不一样；这里世界各地的华人思想和精神很活跃，令人心向往之。或许Steem的华人社区以后可以承载更多的意义，现在这样已经很棒。

&nbsp;

<strong>问：您想过放弃吗？</strong>

好像还没来得及想放弃的时候就被拉到新手村了。。。希望可以和大家一起快乐和坚定地前进吧，贡献我的一些微薄的力量；尽量多做好事，然后写到日记里，像雷锋学习。

&nbsp;

<strong>问：您是怎样走出起步的困局的？</strong>

好像还没来得及陷入困局的时候就被拉到新手村了。。。于是得到了很多人的指点，像态度平和、回复及时的村长@ericet，冷静客观、建议中肯的米高@honoru，扶危济困、慷慨解囊的拉仔@davidke20，还有其他很多很多朋友们的赐教和帮忙，感激涕零！我零零碎碎的问题问了好多，实在是麻烦和感谢村民们。

至于真正走出起步的困局，我觉得唯有“热爱”和“坚持”可以，在和社区的互动和交流中传播智慧、收获友谊吧。多说一句，对于希望从Steem中获利的同学来说，我希望他们了解这是个竞争激烈的市场，如果想要获得大胜需要有top 0.1~0.5%的商业洞察、毅力与创造力，非等闲与投机之辈可以窥伺。

最后，我把在Steem的工作和探索过程，当成一个重要的思想过程，或许真正的读者不在这里，区块链保存的数据也未必永恒，但求无愧而已，更何况能够结识几位好友，实乃幸事；鞠躬尽瘁、死而后已，至于成败利钝，非臣之明所能逆赌也。
