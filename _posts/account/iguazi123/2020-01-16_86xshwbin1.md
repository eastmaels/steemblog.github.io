<strong><em>闲时碎言碎语：</em><em>关于社群与区块链，还没有什么人分享，就让我先来吧。</em></strong>

<section style="line-height: normal;"></section>社群，我们一般会想到构建“人与人的连接”。区块链，被《经济学人》誉为构建“信任的机器”；
<p style="line-height: normal;"> </p>
同样是连接与信任，它们有什么联系呢？社群作为一种自动签合同的技术，能否为社群赋能呢？
<p style="line-height: normal;"></p>
让我们先从数字资产说起。
<p style="line-height: normal;"> </p>
<p><center><strong>一、数字资产化</strong></center></p>

<section style="line-height: normal;"> </section>区块链的核心是数字资产，它从两方面展开，一方面将现实世界中的货币、股票、土地所有权等资产进行数字化，另一方面将网络中的数字进行资产化。这也是我们称区块链为价值互联网的原因。

<section style="line-height: normal;"> </section>
<center><img class="rich_pages js_insertlocalimg" title="我眼中的社群资产化：当“人与人的连接”遇见“信任的机器”" src="http://cocozl.cn/wp-content/uploads/2020/01/beepress8-1579154555.png" alt="我眼中的社群资产化：当“人与人的连接”遇见“信任的机器”" /><br/></center></p>
<strong>1.数据作为新的生产要素</strong>

我们知道，数字经济的引擎，像Facebook、Google、Twitter、微信和支付宝，免费收集我们在网络互动中留下的数据，这是它们作为世界上最有价值公司创纪录利润的来源。

<section style="line-height: normal;"> </section>
<section style="white-space: normal;">
<section style="font-family: Arial; border: 0px none; color: #319369;">
<section style="margin: 10px auto; padding: 1.5em; background-color: #ffffff; -webkit-box-shadow: rgba(0, 0, 0, 0.298039) 0px 1px 4px, rgba(0, 0, 0, 0.0980392) 0px 0px 60px inset; box-shadow: rgba(0, 0, 0, 0.298039) 0px 1px 4px, rgba(0, 0, 0, 0.0980392) 0px 0px 40px inset; background-position: initial initial; background-repeat: initial initial;">
<section style="margin-top: 0pt; margin-bottom: 0pt; font-size: 11pt; color: #494949; line-height: 1.5em;">Facebook每年只向员工（程序员）支付其价值的1%，因为有广大的我们免费为它做了其余的工作。人们作为数据生产者的角色，没有得到公平的对待或适当的补偿，数字经济的收入顶多分配给了少数富有的KOL而不是广泛的大众。普通大众唯一得到的好处是，获得了免费进入互联网的机会。</section>

 

然而，数字世界在支付上的缺失使任何人都不可能专门从事利用他们的数据增加价值的工作：人们无法依靠Facebook和google提供的免费服务生活。这削弱了围绕着数字贡献发展技能或职业的可能性，因为科技奴隶知道，他们所做的任何事情都将被平台剥削。

<p style="white-space: normal;">数字封建主义阻碍了个人的发展，就像封建主义阻碍了人们获得教育的机会以及阻碍了人们改良土地的机会一样。此外，社交平台的推荐算法都不开源，用户没有选择的自由。社交平台还面临着信任危机，例如，Facebook隐私泄露丑闻一次又一次发生。</p>

刚刚过去的十九届四中全会确定，将数据作为新的生产要素，与土地、劳动、资本等齐名。数据“由市场评价贡献，按贡献决定报酬”。数据必须确权，以便于公开、共享、交易，数据资产化和数据资产交易势在必行，副总理刘鹤这样写道。


<section style="line-height: normal;"></section><strong><strong style="font-size: 15px; white-space: normal;">2.</strong>数字资产化的好处</strong>

Facebook、Amazon、Google取得巨大的成功，源自于他们高效、简单地将我们贡献的数据进行了资产化。互联网平台在数据处理和利用方面，拥有巨大优势，所以我们将数据交给了他们。那么，区块链中的数字资产化有什么优势呢？

<section style="line-height: normal;"> </section>
<section style="white-space: normal;">
<section style="font-family: Arial; border: 0px none; color: #319369;">
<section style="margin: 10px auto; padding: 1.5em; background-color: #ffffff; -webkit-box-shadow: rgba(0, 0, 0, 0.298039) 0px 1px 4px, rgba(0, 0, 0, 0.0980392) 0px 0px 60px inset; box-shadow: rgba(0, 0, 0, 0.298039) 0px 1px 4px, rgba(0, 0, 0, 0.0980392) 0px 0px 40px inset; background-position: initial initial; background-repeat: initial initial;">
<section style="margin-top: 0pt; margin-bottom: 0pt; font-size: 11pt; color: #494949; line-height: 1.5em;">其一：无摩擦的交易市场。换句话说，就是这个市场更开放，任何人、在世界的任何地方，都可以参与进来；市场也更高效，一手交钱一手交货，只要交易达成，卖家和买家都会分别拿到钱和货。</section>
<p style="white-space: normal;"></p>
<p style="white-space: normal;">其二：开放的集成市场。传统的集成，我们需要中间公司的「许可」。比如，我从京东买了台笔记本电脑。如果这是一项区块链交易，有交易就有token（令牌）。我可以许可信用卡公司访问这个Token，我还可以给出我拥有这个Token的证明。这样，如果笔记本电脑要退货，信用卡公司可以直接走受理流程，不需要京东「许可」信用卡系统接入他们的系统。如果电脑坏了，保险公司也可以认可Token走流程，不需要跟京东集成。</p>
<p style="white-space: normal;"> </p>
<p style="white-space: normal;">这真正实现了将Token（令牌）交予用户，而不是托管在京东。想想看，凡是想跟用户做生意，受集成凭证烦扰的服务商，现在都可以直接「集成」到用户身上，集成难度很低，跟「信用卡接驳京东」不是一个等级。这种，我们叫做用户端集成，用户的数据、行为和关系由用户所有。</p>

</section>

<p style="line-height: normal;"></p>
<center><strong>二、社群资产化</strong></center></p>

<section style="text-align: center; line-height: normal;"> </section>社群作为一个原始的概念，自人类诞生之初就存在我们的生活和工作之中。用马克•格兰诺维特（Mark Granovetter）的话来说，人类是捆绑在各种不同的社会群体当中的——家庭、邻里、社交网络、商务、教会和民族，基本上所有的经济活动都是由群体而非个人来完成的。

<section style="line-height: normal;"> </section><strong><strong style="font-size: 15px; white-space: normal;">1.</strong>社群的价值</strong>

社群拥有真实的关系、紧密的社会信任，以分散的形式存在于国家、城市、小区、学校之中。小群效应、私域流量、社交电商、社群经济，无一说的不是社群。bilibili跨年晚会、LGBT的团结、马蜂窝和知乎的崛起、豆瓣的持续运营，这都是社群的力量，只不过这些更偏线上。

<section style="line-height: normal;"> </section>社群经济被各界人士倡导多年，在社交电商等取得了较大的成功，然而却没有发挥出更多的价值，究其根源，还在于没有一个适合的技术或工具将成员对社群的贡献进行量化。互联网平台的发展，正是大型平台量化其社区成员创造的数字价值，进入打包销售，维持平台持续运转。他们将数字资产进行了简单、高效、成熟的量化，比如粉丝经济、流量思维、广告营销等等，为互联网经济做了巨大的贡献。

<section style="line-height: normal;"></section> 微信圈子正在做这样的尝试，豆瓣、百度、QQ部落也尝试了多年，现在需要新的解决方案。

<section style="line-height: normal;"> </section><strong><strong style="font-size: 15px; white-space: normal;">2.</strong>社群资产需要被量化</strong>

我们经常谈到社群和商业的时候，有点像区块链。区块链分链圈和币圈，链圈专注于技术，币圈青睐于数字资产对经济的发展。两个圈层之间有着各自的鄙视链，链圈认为技术推动社会变革，商业模式是结果不是原因。币圈认为商业订购技术，一个商业问题等于一个技术方案，比如快速结账需要POS机，POS机就应声而出。

<section style="line-height: normal;"> </section>区块链的链圈和币圈交集很少，一个不知道接下来怎么赚钱的技术专家和一个不知道技术上根本创新的商业领袖在一起很少谈得来，一谈就是空对空。我们区块链充满了这种空对空的对话，其实社群也存在。

<section style="line-height: normal;"> </section>很多人认为将商业与社群强关联，难度很大，觉得人与人的信任关系不能被商业量化。其实，这样的困惑也发生在其他行业，最后实践证明，二者可兼得。

<section style="line-height: normal;"> </section>
<section style="white-space: normal;">
<section style="font-family: Arial; border: 0px none; color: #319369;">
<section style="margin: 10px auto; padding: 1.5em; background-color: #ffffff; -webkit-box-shadow: rgba(0, 0, 0, 0.298039) 0px 1px 4px, rgba(0, 0, 0, 0.0980392) 0px 0px 60px inset; box-shadow: rgba(0, 0, 0, 0.298039) 0px 1px 4px, rgba(0, 0, 0, 0.0980392) 0px 0px 40px inset;">
<section style="margin-top: 0pt; margin-bottom: 0pt; font-size: 11pt; color: #494949; line-height: 1.5em;">有人说，管理无法科学化，但泰勒带来了管理科学与工程这门学科。正是管理科学与工程的流水线生产方式，福特的T型车早期风靡美国；伴随着丰田生产方式（美国叫做精益生产），日本车走向全世界；稻盛和夫实践的阿米巴经营管理模式亦缘起于丰田的“Just in time”和“全员生产维护”，最后被众多公司作为管理宝典而应用。</section>
<p style="white-space: normal;"></p>
<p style="white-space: normal;">这些均缘起于，泰勒开创的用秒表卡工人工作的时间，优化流程和生产方式，科学对待管理所带来的。虽然将管理科学化，与工程进行结合，这带来了一些负面的影响，但推动了工业生产和互联网革命的长久持续发展。</p>

<section style="margin-top: 0pt; margin-bottom: 0pt; font-size: 11pt; color: #494949; line-height: 1.5em;"></section>
</section>
</section>
</section>
<p style="line-height: normal;"></p>
量化人们的工作，将管理科学化，一定程度上能更好地组织人们协作生产。社群作为松散的组织，一样需要被量化，需要更快速的流动，这些离不开社群资产化。

<section style="line-height: normal;"></section>那如何进行社群资产化？

<section style="line-height: normal;"> </section><strong><strong style="font-size: 15px; white-space: normal;">3.</strong>社群资产化的三个维度</strong>

•<strong>社群数字资产化</strong>：将社群成员创造的数据确权，由社群成员所有，并由社群里的成员审核确定其价值，将其资产化。这方面，类似于诸多去中心化的社交媒体平台，Steem在这方面实践了三年多，取得了不错的成果。

<section style="line-height: normal;"> </section>•<strong>社群行为资产化</strong>：行为产生内容，社群资产化不仅仅是将创作的内容进行资产化，而是将交易行为、出席活动的行为、投票的行为、贡献价值的行为等等，进行量化，并给与奖励。

<section style="line-height: normal;"> </section>•<strong>社群关系资产化</strong>：社群最重要的就是人与人之间的关系，如何将这种关系转化为数字，并随时携带，这是需要研究的。如果能将这样的关系进行量化评估，比如将一次推荐、一次转发、一次信任背书作为资产得到认可，这会带来巨大的改变。
<p style="line-height: normal;"> </p>
那社群资产化有啥好处呢？

<section style="line-height: normal;"> </section><strong><strong style="font-size: 15px; white-space: normal;">4.</strong>社群资产化的好处</strong>

社群资产化的好处是明显的，主要有以下三个方面：

<section style="white-space: normal;">
<section style="font-family: Arial; border: 0px none; color: #319369;">
<section style="margin: 10px auto; padding: 1.5em; background-color: #ffffff; -webkit-box-shadow: rgba(0, 0, 0, 0.298039) 0px 1px 4px, rgba(0, 0, 0, 0.0980392) 0px 0px 60px inset; box-shadow: rgba(0, 0, 0, 0.298039) 0px 1px 4px, rgba(0, 0, 0, 0.0980392) 0px 0px 40px inset;">
<section style="margin-top: 0pt; margin-bottom: 0pt; font-size: 11pt; color: #494949; line-height: 1.5em;"><strong style="color: #319369; font-size: 17px;">自金融</strong>：将社群资产化之后，数字的流转就是价值的流转，行为和关系的流转也是。那样，无需中间可信第三方进行担保，任何人在任何地方都可以参与进来，无摩擦的金融市场将形成。社群将随着价值的流动，将社群成员更紧密地联系起来。</section>
<section style="white-space: normal; line-height: normal;"></section>
<p style="white-space: normal;"><strong>自穿透</strong>：社群资产化之后，将不再分“场内”和“场外”，打破社群与社群、社群与个体、个体与个体之间的数据孤岛，就像拆除围墙一样，数字资产在空间中自由可穿透、可验证。这无疑带来更自由、有活力的市场，也带来了更开放的集成市场。</p>
<p style="white-space: normal;"></p>
<p style="white-space: normal;">现在微信数据库的数据只有微信App能读取，抖音的只有抖音App。基于区块链的社群数据，数据为用户所有，微信App、抖音App、微博App在得到用户授权后可读取链上公开数据，这就是所说的打破数据孤岛。</p>

<section style="white-space: normal; line-height: normal;"></section>
<p style="white-space: normal;"><strong>自生态</strong>：社群资产化将依靠共识机制、加密算法、智能合约等可行区块链技术，将组织自动化运转起来。这样的组织实现了社群一直的目标——自组织，区块链技术保障的合约7*24小时运行着，维持社群高效运转。一个社群就是一个生态，生态内自金融运转，生态之间高效协作。这样开放包容的组织形式，将让价值回归价值的创造者，诞生一个分配更加公平、平等的新的网络世界。</p>

</section>
</section>
</section>
<section style="line-height: normal;"></section>
<center><strong>三、区块链是一个创造信任的技术，更是一个量化贡献的技术</strong></center></p>

<section style="line-height: normal;"> </section>机器人越来越像人，人也朝着机器人进化。

<section style="line-height: normal;"> </section>区块链的核心在于签署自动运转的合同，可以把它当做公正、透明的机器人，它自动为协作的双方签署合同，监督完成过程，给予奖励。

<section style="line-height: normal;"> </section>区块链的自动化有两层意思，区块链技术能保证自动化的部分交给代码，不能自动化的交给人类和智能合约，人类即将扮演一个个分布式计算机，拥有强大的计算能力。区块链通过智能合约，将分散在各地、不同语言、不同文化的个体自由匹配起来，让地球这个超级计算机更好地运转。

<section style="line-height: normal;"> </section>自动化和工具的益处是降低成本、提高效率。可以把区块链当做是一个公开、透明、公正的法官，它帮助社群签署自动的合同，让社群成员彼此协作，为社群创造价值。这样的过程，就像公司依靠合同和会计账本一样，组织员工，生产产品或提供服务。

 

区块链技术让社群资产的量化、评估、交易变得有可能，进而可能诞生一个个小型数字社区。无数数字社群组成自生态圈，带来全新的未来。
