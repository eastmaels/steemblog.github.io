<p style="color: rgba(0, 0, 0, 0.87);"><b>初心</b></p><p style="color: rgba(0, 0, 0, 0.87);">STEEM已经稳定运行两年，但STEEM在中国的应用还很少，社区用户也较为不活跃。今年2月份，我们就像打造一个去中心化的旅行社区，给大家提供一个分享旅行内容的平台。到现在，我们更加清晰我们要做什么，我们需要基于STEEM打造一个图片社区，在这个社区内大家分享自己的生活，自己的旅行，在这里面互动。全球图片社交以Instagram的做的最棒，活跃用户达10亿，但国内没有出现具有特别优势的对应平台，我们看中这块相对空白的市场，和STEEM一同出发。</p><p style="color: rgba(0, 0, 0, 0.87);"><br></p><p style="color: rgba(0, 0, 0, 0.87);"><b>简介</b></p><p style="color: rgba(0, 0, 0, 0.87);">其中是一个奖励人们分享他们美好生活旅行方式和视觉体验的平台，您可以通过它与家人，朋友分享精美的照片和故事，所有人在这里真实地表达自己。最重要的是，您可以通过点赞，评论或推广您朋友拍摄的照片，获得加密货币奖励！</p><p style="color: rgba(0, 0, 0, 0.87);">其中引入了一种新的生态系统，将原汁原味生活旅行方式的展示与获得的报酬融合在其中的工具中，人们通过发布故事来绘制世界。故事就是一张表达自己的画布，其中将故事视为创作者和粉丝，朋友之间的双向联系，设计工具让普通人表达他们对周围世界的感受。其中让你记录自己的生活，做自己，分享每一个对生活心动的时刻！</p><p style="color: rgba(0, 0, 0, 0.87);">其中是一个应用程序，你可以发布自己的照片和故事，类似于Instagram的和小红书，但有一些差异。一个不同之处在于，当其他成员点赞您的帖子时，您可以得到平台奖励的STEEM加密货币，在这可以兑换成现金。当您点赞照片时，您也可以获得STEEM作为推广奖励。此外，上传到其中的任何照片都会同时发布到您的Steemit博客上，这使得关注您的任何人都可以看到它。</p><p style="color: rgba(0, 0, 0, 0.87);">STEEM 已经证明了将“点赞”转化成货币概念的可行性和能力，正如搭建在Steemit.com上的系统所示。其中将是首批搭建在Steem区块链的元通证之一，旨在消除普通用户进入加密</p><p style="color: rgba(0, 0, 0, 0.87);"><br></p><p style="color:
