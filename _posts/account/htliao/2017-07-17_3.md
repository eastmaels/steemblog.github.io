# 圓底（Rounding Bottom)
圓底是一種不常見的逆轉形態，這形態多數出現在一個由熊市轉為牛市的長時間整固期，常在長期跌市後出現，暗示大戶在靜悄悄收集。

![IMG_5515.JPG](https://steemitimages.com/DQmVeage7BpvtacTXyTscwVAYm7LqSLBVpVY1MnNd9hdzC2/IMG_5515.JPG)

圓底的威力十分可怕。如圖中所見，白銀價格就在2016年年初約$14位置左右做了一個圓底，之後最高還升至$21左右呢。

個人意見認為2016年年初白銀價格形成的圓底有極大機會暗示往後長達數年的金銀牛市展開。


<hr>

# 雙底（Double Bottom)

雙底是一種逆轉型態，當價格在某段時間內連續兩次下跌至相約的低點後出現，常以英文字母W的形狀出現。

![IMG_5783.JPG](https://steemitimages.com/DQmPxxdKjhewo4oupCng41wDwz7EQzcSvNeig4YvCimhYgs/IMG_5783.JPG)

這是道瓊斯工業平均指數的圖表，指數在2016年初在約16000點出現一個雙底形態，那時還很多人大嗌熊市來臨呢！雙底形成後不久後便大升，現在還超越了21000點。

<hr>


# 雙頂(Double Top)

![IMG_5789.JPG](https://steemitimages.com/DQmU6g7yi6KKrstB1RGMpXtGFVU4bnRWhDJGRtG5kxtXk1K/IMG_5789.JPG)

雙頂，與雙底相反，價格在某段時間內兩次觸及高點急回，暗示在該位置有強大沽壓，價格很大機會已見頂，投資者還是避之則吉。

<hr>


# 三底(Triple Bottom)



與雙底相似，價格在某段時間內三次下跌至相約的低點後出現，顯示下方對價格構成強大支持，後市利好。

![IMG_5777.JPG](https://steemitimages.com/DQmWKqhuiDwRDSNvF8sooRGbe27VokN7jKYsKC2riwHpb4e/IMG_5777.JPG)

這是歐元的圖表，可以觀察到的是歐元價格三次到達低點後都大幅反彈，正在形成一個三底，只要這個升勢持續就基本可以確定三底形成，往後會有不錯的升幅。

<hr>

# 三頂(Triple Top)

![IMG_5788.JPG](https://steemitimages.com/DQmTvwcv5P4dR7ny5JuK3qbKNCPyBk7e3o5rjDFEEV2aMvq/IMG_5788.JPG)

三底的相反形態，就不詳述了，趕快逃命吧。

# 頭肩頂(Head and Shoulders Top)



經典的見頂形態，圖形以左肩、頭、右肩和頸線組成，一旦跌破頸線後就發出強烈的沽出訊號。

![IMG_5784.JPG](https://steemitimages.com/DQmc4g6rHRAsfjUBBC5dyUovJYedoAYv7WQ4ExDqe1WHuh9/IMG_5784.JPG)

以上是美元指數的圖表，不難發現頭肩頂的形態已形成，如不能有效向上突破頸線，你對美元往後的趨勢發展便不能太樂觀。

<hr>

# 頭肩底(Head and Shoulders Bottom)

![IMG_5787.JPG](https://steemitimages.com/DQmNtxaavKK9pe9b7Z7c6mJ5KWEWBzApzRBho8NpFXRzbDQ/IMG_5787.JPG)

頭肩頂的相反形態，就不詳述了，突破頸線後會有一波大升浪。

<hr>

# 下降楔形(Falling Wedge)



這圖形向上突破後視為看好的訊號，雖然楔形的底線往下傾斜，說明市場承接力不強，但抛售的力度也越來越弱，向上突破後形成一個新的升浪。

![IMG_5778.JPG](https://steemitimages.com/DQmZuRZhxKC29tD7LN3ExiuMW6PDHw672nmKNVQ4MAUriR8/IMG_5778.JPG)

這是日圓的圖表，在六月中價格突破了下角三角形就形成了一波升浪。

<hr>

# 上升楔形(Rising Wedge)

![IMG_5786.JPG](https://steemitimages.com/DQmWKbb4eRDQVN71GuGK9vu1hBnPrT3fJg67KvwGYQWFxKJ/IMG_5786.JPG)

和下降楔形相反，被視為看淡的訊號，突破楔形下方後會形成強大沽壓。


![divider-1024.png](https://steemitimages.com/DQmb9i2KiKpKHoj63jj482Z2HpRPSbx1UyAPXrRGYxXzhEj/divider-1024.png)
