# (English version below :) ) 

不久前 @jubi 和 @coldhair 都定制了自己的Steemit裇衫，害我超羡慕的。於是我也想自己製作一件，希望~~往後的帖子能騙到更多投票~~能大大力宣傳Steemit，為Steemit社區出一份力，建造更美好的明天吼！

但說真的，才玩了兩個多月我也沉迷Steemit了，每天也在想有什麼特別有趣的寫作題材。弄件T恤好像也能令整件事情變得更好玩！我覺得要親自設計才有趣嘛！

還有，香港畢竟是個小城市嘛，來個聚會也是有可能的，那到時沒有T恤充充場面又怎夠霸氣呢？這樣就不用在電腦前穿了。
