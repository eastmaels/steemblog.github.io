Steem Handbook Newsletter |《Steem 指南》快讯：币圈投资组 + 校对组 + 写作规范 + 周边产品

This post is a contribution to the open source project Steem Handbook. Details could be found in the post Steem Handbook Calls for Support.

1. 币圈投资组和校对组招兵买马

这两个组面临人手不足的困境，日前开始招兵买马（看这里）。自发出公告后，两组均有新生力量加入，尤其是币圈投资组，已经拟定了编写大纲，鸿篇巨制，本书有可能容纳不下，有望成为《Steem 指南》的姊妹篇。

2. 写作规范增补

最近书稿增加迅速。面对稿件格式上出现的意外问题，@dapeng 对写作规范进行了增补（看这里）。

3. 产品开发部呼之欲出

卖鱼的奶爸 @maiyude 设计了《Steem 指南》一书的多款周边产品（看这里和这里），包括 T 恤衫、钱包、纸袋、胸针、名片、马克杯、钥匙扣、滤茶器等。

我们欲征集壮士成立“产品开发组”，把这些产品做出来出售。有没有哪位壮士自告奋勇？请留言。
