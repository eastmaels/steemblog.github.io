Steem Handbook Newsletter |《Steem 指南》快讯：八卦征文 + FAQ 搜集 + 书稿索引

This post is a contribution to the open source project Steem Handbook. Details could be found in the post Steem Handbook Calls for Support.

最近过年，感觉书稿的编写节奏放慢了，僧少粥多，用来给大家点赞的 SP 每天用不完，所以有闲的作者们可以趁机多投稿。等将来大家都来投稿了，就僧多粥少了。

不过，自上期快讯发布以来，回顾一下这一周的进展，仍然发生了很多事情。

1. Steem 八卦征文

@towardsthesun 发帖征集［Steem 八卦］，即「生动有趣地说说 Steem江湖的爱恨情仇」，按照要求投稿的作品有望收录到“附录”一章。投稿者可以直接在她的帖子后面留言说八卦，或者发帖也行，帖子要求：

  • 标题注明［Steem 八卦］；
  • 添加 #steem-guides 标签；
  • 在发起帖的留言区留下链接。

2. 常见问题搜集

@susanli3769 开始搜集 Steem 常见问题，将作为附录的一部分。欢迎大家（尤其是新人，这是答疑解惑的好机会）为她提供素材。

3. 书稿增添索引功能

@dapeng 为书稿增添索引功能。目前，书稿的 pdf 版末页列出了一个索引条目作为示范。为了避免混乱，我们将在最后统稿时统一添加索引。

除此之外，书稿的多个章节内容均有更新。经过本书协作，估计我们会把 GitHub 给普及了……

感谢大家的贡献！
