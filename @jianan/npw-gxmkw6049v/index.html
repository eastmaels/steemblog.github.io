NPW，基于用户画像的去中心化互联网流量引擎



本文谈到的是一个可能许多人没怎么听说过的节点币，新力量币（NPW）。新力量币也是一种Master Node（主节点）币种。


Master Node（主节点）是一种在区块链上执行专门功能的节点，它们是社区成员在满足特定条件后建立的服务器，因此主节点也称为绑定验证器系统。


这一说法最开始是来自Dash，从Dash开始引入了这一概念，同时这也作为一种额外的激励措施。本文谈的新力量币（NPW）也是属于此类。



项目简介


新力量币（NPW) 是一个采用了主节点和PoS两种持币挖矿生息混合机制的节点币。这也是一款去中心化基于用户画像的全球在线流量引擎。


NPW的目标很宏大，也很实际。其目标是打造一个庞大的去中心化互联网流量交换、计费、投放、统计、用户画像、标签的底层引擎。


于此同时，NPW还接入和支持多种去中心化广告平台、社区自由发布各类插件应用，把NPW用作为平台支付货币。从钱包应用开始，NPW项目目前相继推出了经典QT钱包、未来派桌面钱包、自助广告投放平台、点多多手机钱包，并且将广告业务集成在钱包中。


关于NPW经济体系


（1）NPW的分配方式


新力量币都通过挖矿挖出的，产出的币分为两个阶段，PoW阶段为基础的挖矿阶段，初始的币都通过这个阶段挖出，后续则是PoS阶段。新力量币在四年后的总供应量约为7300万，十年后的供应总量约为1亿。


所有人均可以搭建自己的主节点，只需则锁定20000个新力量币。新力量币的区块产出时间定义为2分钟，区块大小为2M。从2018年5月份挖出创世区块，至2019年5月份，全网运行了900+的主节点。




NPW是属于区块链技术的原生代币，也是应用型数字币而非证券型代币，在新力量币广告平台的经济体系内，NPW主要通过挖矿和权益积累获得，所有人都可以公平地获取它。普通用户可以通过钱包观看广告的方式获得。


（2）NPW的应用场景


当NPW网络的基础设施搭建完后，会成为基础流量引擎，NPW币也将作为底层基础加密货币，进入多个广告平台进行流转和消耗。在广告流量平台中，可以通过充值及广告投放对新力量币进行消耗，而消耗掉的币将作为投放流量主的奖励，重新进入经济循环。




NPW再打造一款去中心化基于用户画像的全球在线流量引擎，在引擎中有三个环节需要消耗或交易 NPW。




  • 广告投放：广告主可以消耗 NPW投放广告。



  • 流量交易：普通用户查看广告完成任务获得 NPW 奖励。



  • 货币流通：双方通过交易所或者线下交互换取 NPW。也可以通过 NPW 进行其他应用场景进行消耗或者交易。


NPW网络通过将广告主、广告平台、流量主，以及普通用户串联在一起，并自主形成经济闭环。早期会通过基于Web钱包来投放广告，而且钱包基本开发完成，这是目前市面上，少有的有真实使用场景的节点币。



竞争对手对比分析


除了NPW之外，市场上做区块链广告平台的类似项目：DTA 和 DAD。


DTA 偏向做一套基于区块链的广告投放底层工具，NPW 从钱包开始做起，直接提供广告业务形成闭环，落地比 DTA 快很多。根据 DTA 的白皮书描述，18年Q4 DTA 也会上线主链，转为主节点+子节点，而 NPW 天生就是主节点架构。DTA 当前市值约为 NPW 的两倍。


DAD 号称基于 ONT 公链的区块链广告平台，目前只上线了一个网站，并且只是NEO 的一个代币。DAD 的白皮书讨论了关于投放、反欺诈、透明度、授权的内容，技术文字说明较多，但其落地性成疑。对于 NPW 后续发展有一定参考意义，不构成威胁。DAD 还未上交易所。


竞争对手的出现说明区块链广告平台的市场有需求，而目前NPW的落地更快些，800+节点的建立，经典QT钱包、未来派桌面钱包、自助广告投放平台、点多多手机钱包等众多应用的落地。广告业务集成到钱包里，也初步形成了商业落地。



价值组件的涌现，与生态的繁荣


「王川: 从组件的角度看正反馈循环和区块链产业」一文中提到，生态的发展应该是以下路径：


「扩展性 / 零边际成本 / 开放系统 → 数量增加 → 各种连接指数倍出现 → 有价值的创意涌现 → 形成组件 → 系统复杂化 → 数量继续指数倍增加 → 正反馈循环 → 生态大繁荣。」


NPW同时也是一款程序，可以复制给成千上万的人使用，不限语言和国籍，边际成本只有几百兆账本的存储空间。


NPW 取名「新力量币」，原因是项目定位就是非币圈用户和商家，是一个足够开放的系统。扩展性、零边际成本、开放系统的基础条件已全部具备，逐步落地的应用正式上线后，提供平台让用户和商家产生有意义的连接。


当有部分用户涌现出创意，消耗一定的 NPW，投放广告，传播自己的影响力，把广告应用当作服务或物品交易平台，甚至做出更加开脑洞大开的事情。逐渐一些活跃的社区成员会贡献自己的力量，为 NPW 流量引擎沉淀出有价值的组件……



小结：


值得一提的是，作为一个不那么高调和知名的节点币种，却拥有非常优秀的节点参数，极高的锁仓率。NPW 的锁仓率一直非常高，锁仓率是评估主节点币最重要的参数，市面上的700多个主节点币，锁仓率长期超过 70% 的不超过 20 个。


再关于通胀利息收益，只要保证节点和钱包在线参与记账，每天就会有利息收入，目前节点年化收益率依然不低，将近100%，当然随着节点数量增多，会逐渐减少。


NPW尽管目前币价很低，但参与的用户特别是节点用户并没有收到市场价格波动的影响，依然有足够多的稳定的节点提供服务。另一方面NPW的应用也在持续不断的落地，一切都在朝着好的方向发展，总之，NPW是一个被低估的，值得关注的区块链项目。
