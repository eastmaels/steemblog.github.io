SMTs 和 Yoyow 3.0

        SMTs 和 Yoyow 3.0这两者都是最近推出的。前段时间 @ned宣传了一下SMTs ，社区内的小伙伴们都是激动万分的。SMTs 也许可能成为大一统的内容生成平台。不过很快，Yoyow 3.0发布了新的白皮书，《YOYOW 中文白皮书 3.0》。它也是剑指大一统的内容平台。这两者之间有什么联系和区别呢？我们可以试着来看下。

SMTs 和 Yoyow 3.0都是想做成基础平台，基础链

        在基础链上是一致的。后面的网站都要基于这两者。意思是你不用开发链了，做前端的内容平台就好了。在这一点上这两者是一致的。

区块链是一个去中心化的价值网络，YOYOW 将会成为一条以内容评价为核心的公有区块链网络，吸引第三方开发者以及厂家在 YOYOW 之上构建出各种各样的内容激励平台。而在初期，YOYOW将会对平台建设者提供各方面必要的帮助。

YOYO 代币是 YOYOW 网络的唯一基础代币，用户在各个内容平台上的评分权重以及收益都将使用YOYO 代币。同时在中期开发计划中，YOYOW 网络将会允许内容社群平台以 YOYO 代币为基础,发行的属于自己社群平台的智能代币。

在发行的代币上

        SMTs 在发行代币上是可以自由创建的，Yoyow则只能使用Yoyo。SMTs有点类似以太坊的智能代币，依托基础网络生成自己的代币。用户的自主权能强点。在这点上Yoyow是不能选的，必须抵押Yoyo才能做自己的平台。

YOYO 代币作为系统的基础代币，主要有以下几个作用： 1. 用于产生评价内容的权重 2. 用于抵押成为见证人 3. 用于抵押成为平台 4. 用于产生免转账手续费的积分 5. 作为传递价值的媒介

在治理的模式上

        在治理的模式上好像差别不大。DPOS的算法，见证人出块等。这点上Yoyow是参考了steem的模式和算法。其实说起来，两者的渊源还是很深的，有时把这两者算做一个系列也是可以的。

采用 DPOS 机制的 YOYOW 网络中，通过理事会投票决定见证人的数量，由普通用户投票来决定见证人人选。入选的活跃见证人按顺序打包交易并生产区块，在每一轮区块生产之后，见证人会在随机洗牌决定新的顺序后进入下一轮的区块生产。见证人负责处理交易以及维护社区，因此见证人处理交易可以获得相应的 YOYO 代币奖 励。
