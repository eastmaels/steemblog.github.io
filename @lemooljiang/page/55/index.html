新人生存指南之十五：密码即一切

密码即一切！

司第目（steem）是区块链应用，和比特币一样有去中心化的意思，这也就意味着你的资金只能你自己保管！如果你的密码丢失，那可没有银行来帮你取回！所以，在区块链的世界，密码即一切！

司第目是社交性的平台，它的操作很频繁，再加上很多用户会用不同的设备登陆，这就导致司第目上资金的风险性。而资金被盗也发生过好几起，所以，司第目就设计出了一套复杂的密码机制。如果你帐户上没什么钱的话，可以不用去多管。如果你帐户上的钱被偷会让你肉疼的话，那我建议你要好好学习一下本篇文章，学会如何保管密码。

一、密码的分类和级别

密码分成四种：
Owner Key—主控密码，最高级别的，修改它其它密码会自动修改
Active Key–资金密码，用于转帐
posting key—发帖密码，用于发帖
memo key—备注密码，用于备注

二、查看和保存分级密码

用主密码登陆后，如上图所示。公钥是公开可见的，不用去管。点击查看密码，保存好私钥！

三、修改主控密码

这里是修改主控密码的页面，修改它，其它密码会自动修改！所以，如果你觉得你的密码不太安全了，你就去修改主控密码，再回到上面的第二步，分别记录其它密码！

四、退回登陆并清除记录

主控密码改好之后，其它密码也记录好了。那就先退出吧！

事情还没完！浏览器都有用户名和密码的记录功能，它由“cookies”来实现，它一般保存在本地！所以，你下次登陆同样的网站不用重新输入用户名和密码。如果你的电脑中了木马，造成密码泄露，很可能是它引起的。一般公用电脑上登陆的话，用完了，最好完成一次“清除浏览记录”，清理掉自动生成的“cookies”！

回到正题。完成一次“清除浏览记录”，再使用你的用户名和posting key（发帖密钥）登陆司第目就好了！

备注：平时使用都用posting key（发帖密钥）就够了！转帐时需要使用Active Key–资金密钥，Owner Key—主控密钥尽量少用，一般情况下也用不上，所以找个地方好好保存就可以了。

如何练就抗跌神功？

        “欲练神功，必先自宫”！哟，这不是《葵花宝典》，可没有这一项。你想太监清心寡欲、心无旁鹜，倒也比常人在武功这个领域容易取得成功。所以呢，你要把自己的欲念摈弃、贪欲制止，才有可能练就神功。哈，扯远了！

        一片血海中求生不易！

        我想这是最近一段时间来大伙共同的感受。因为大家手里头都有些币，看到币价下跌、资产蒸发，心里也不痛快。不过，可以掰解掰解。你受损失的其实只有你流通中的那部分币值，比如你今天提现了1个比特币，而这个比特币你是在10万买的，那么你确实是亏了。不过如果你只是提出了0.1个，另外0.9个还在钱包里好好呆着，那么你亏损的资金（0.1个）还是很有限的。赶明儿比特币涨到20万，你不就盈利了嘛。

        其实分析到最后，抗跌神功最关键还是远离炒币这一条。如果用逆推的方法更好理解，那些炒币赚了百万千万的人手里的钱从哪来呢？还不是割韭菜割来的！币市和股市一样，大部分时间是零和游戏。你赚到的钱，其实是别人亏的钱。

        那如何实现最终的共赢呢，大家都能赚到钱？那就是一个长期价值增加的过程。比如比特币，比特币的用户增加，使用增加，应用增加，那么它的总体价值也是增加的。从比特币的总市值在增长的曲线就能看出来。长远的增值才是抗跌神功的基础，所以，很多人会建议别人做价值投资，而不是去炒作。

        前段时间比持币涨到十几万一个，我的一些朋友心动了，问我能不能买？ 我是直接拒绝的。你不懂的东西你不要去碰！一般我会建议他们去看看比特币的白皮书，试试比特币是怎么转帐的，怎样跨地域地传输价值。基本的原理你懂了，该何去何从，你自己也能判断了。

        参与区块链的事情很多。我也建议多参与一些基础的工作较好，比如挖矿、见证人节点、写作等。这两年挖了些以太坊，虽然没有卖过大钱，但也是不错的路子。前段时间和朋友一起搞定的YOYOW的见证人节点，收益也不错（《YOYOW见证人炼成记-成功篇 / 挖矿大业 #4 （顺便拉个票）》）。另外，在司第目（steem）上写作也不错啊，没事就多学习，多写作，这总不会错。

        总结：基础建设，技术优先。长期持有，价值投资。

《徒步中国》寻找灵魂之旅 / 品乐书会#19--阅读经典，分享创作

        最近玩徒步寻找灵魂的事好像有点多啊，《冈波仁齐》《七十七天》这些个电影在圈子里讨论得也挺多的。这倒让我想起早几年曾看过的《徒步中国》。作者是个德国人，雷克。老外来玩穿越中国，就有那么些新鲜感了。玩徒步的初衷都是体验生活、灵魂之旅 之类的高尚信念。我也一般不把它们当游记，而是当小说看。《徒步中国》本质是本不错的小说，这个老外的语言功底很深，当然也不排除一些营销手段。但好在是个好故事，都还蛮有趣的。

雷克是一个来自德国的交换生，在北京的大学念书。他徒步从北京回到德国。所谓徒步：就是指步行走出北京，进入河北、山西、陕西、甘肃、宁夏，再从宁夏进入甘肃，过河西走廊，再穿过新疆的茫茫沙海，过中亚地区，进入欧洲。

        我们经常说要徒步上班，这哥们厉害了，竟然徒步从中国回德国！

2008年10月
吐鲁番，中国西部戈壁
我站在国道上，前方有收费站和几家店铺，四周是茫茫戈壁。我用汗淋淋的手紧握着把手，拉着我的拉拉车向前迈步。
脚伤、风、沙漠，还有不允许我通行的警察，所有这一切对我而言，全都无所谓了。
我的心，痛。头脑中的想法仅是，这样的心痛应该停止。

        似乎每一个开始都是伤心开头，任何一个疯狂的开始都是一个自我背叛！

两位警察互换了一个惊诧的眼神，转而上下打量起我来：个头一米九多的老外，衣服有几处磨破了，胡子、头发因为长期没有修剪而蓬乱不堪，眼里布满血丝，拖着一辆白色的拉拉车走在戈壁滩上。突然，钥匙串好像想起了更为重要的事情，“护照！签证！”他吼起来。

        这个描写挺妙的，很有即视感。

当他走进山西乡村，住进一户农家窑洞时，一个七八岁的小女孩问，雷叔叔，你有北京的照片吗。“有”。当小女孩说：“北京可漂亮了，那儿很干净”。这个德国人不知道说什么好，在他的印象里，北京或许没有这样的概念，他第一天到达北京的情景，却是那散不开穿不透的尘雾！小女孩又接着说：“在北京，白色的裙子可以穿一整天，在这，几个小时就黑了”

        看到这里我也是沉默的。里面有关于煤窑的记录，很是让人胆战心惊的。国人对资源的掠夺，对环境的破坏已达到了极限。现在有些媒体报道，很多以前的富裕村现在都成了鬼村了！环境破坏了，水也不能喝了，也没经济收入了，只剩下老弱病残了！

        雷克这一路的见闻很是详细，也是国内真实的面面观。读来让人深思，让人兴奋，也让人伤心。

冈波仁齐

        读万卷书，行万里路！有机会也去做做背包客，徒步中国！

韦斯币种评级 (steem B+)/ Weiss Cryptocurrency Ratings

https://weisscryptocurrencyratings.com

        在新的韦斯币种评级（Weiss Cryptocurrency Ratings）中，司第目（steem） 得到了最好的 B+ ！司第目转帐速度很快（3秒），手续费为零，友好的帐号名称，庞大的用户群，融洽的社区氛围，得到好的评级也是理想当中的。@clayop 文章《Steem has Achieved the First B+ Grade on Weiss Cryptocurrency Rating》有提到此事，还有其它一些币种的信息：

ADA went up to B
EOS and NEO went down to B-
BTS went up to B-
ETH kept B-

        这是2018.1.26的评级报告，大家可以细看下！

        比特币只得了C+的评级，确实有点古怪。相对的，司第目（steem）却能得 到 B 的评级，有点看不明白。不知此等评级是基于什么因素，不过看到司第目能得到这么优秀的评价，还是挺开心的。虽然现在币价下跌，大家心里有戚戚然，不过，春天就快到了。

http://moonnocoin.com/weissratings-sokuhou/Weiss-Cryptocurrency-Ratings.pdf （这是2018.1.24的报告）

        去扒下韦斯这家评级机构，看看他们的测算模型：

The Model
The Weiss Cryptocurrency Ratings model is built from the ground up with five basic layers:
Layer 1. Current data on each currency’s technology, performance and trading trends
Layer 2. Proprietary formulas that convert the data into comparable ratios.
Level 3. Proprietary sub-indexes that aggregate the ratios to measure key factors and features considered critical to the potential success or failure of investments in each cryptocurrency
Level 4. Aggregation of the sub-indexes into four key indexes, each meriting a separate letter grade
Level 5. Aggregation of the four key indexes into an overall letter grade

Thus, each Weiss Cryptocurrency Rating represents the pinnacle of a pyramid built from tens of thousands of calculations that feed up to a final grade.

        嗯，看起来是那么回事！总算除了看币价曲线之外有了一个更多的参考了。以前只看https://coinmarketcap.com上的币值参考，现在多了一个视角，也是不错的选择。不过，好像韦斯评级只有付费成员才能看到最新的报告，并且现在还只有74个币种。期待更新吧。
