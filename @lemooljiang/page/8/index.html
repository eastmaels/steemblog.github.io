《兄弟》虽是一体，终将背驰/ 品乐书会#37--阅读经典，分享创作

余华的小说写得不错，《活着》写是虽悲剧了点，但是深刻动人。找来了这本《兄弟》读着也不错，很有种魔幻现实的色彩。在种种荒诞的情节下，是对现实的冷漠反讽。我想起了《百年孤独》，在一种现实与虚幻之中行走，叙述了一个貌似喜剧但却悲情的故事。

《兄弟》当然是两兄弟的故事，李光头和宋钢这对兄弟。当然他们并非亲兄弟，是父母再婚结合的名份。这两兄弟是性格迥异，人生境遇也是截然不同。在时代背景中展开了他们不同命运的叙述。

在人格设定上，李光头是个小流氓，宋钢是个好青年。但在现实中却是好人不长命，坏人活千年。故事刚开头就是李光头在厕所偷看女人屁股而一战成名。到后来下海经商，倒卖服装，举办处女大赛……种种荒诞之事，不一而足。而相反宋钢却是个老实人，本该是个幸福的小家庭。却因时代变化，失业、下岗、卖假药等，都是因生活所逼。因性格所致，不能调和，最终卧轨自杀，是为可叹。

总体来说，小说的描写较为夸张，背景又在文革时期，所以会产生很多荒诞的印象，可能现实也确实荒诞吧。在这个失落的时代，一切都显得那么不真实，却又是无比真实。

一入前端深似海，聊聊vue.js / 网络研习社#11

https://cn.vuejs.org

一入前端深似海，从此阿花是路人！

运行着nodejs，要开始多学下js， 一拿到js ，又要多学下vue.js ！貌似坑是越挖越大啊。我这最喜用wordpress神器建站的人，学起js来，还是有那么点费劲。

什么是Vue.js

  • Vue.js 是目前最火的一个前端框架，React是最流行的一个前端框架（React除了开发网站，还可以开发手机App， Vue语法也是可以用于进行手机App开发的，需要借助于Weex）
  • Vue.js 是前端的主流框架之一，和Angular.js、React.js 一起，并成为前端三大主流框架！
  • Vue.js 是一套构建用户界面的框架，只关注视图层，它不仅易于上手，还便于与第三方库或既有项目整合。（Vue有配套的第三方类库，可以整合起来做大型项目的开发）
  • 前端的主要工作？主要负责MVC中的V这一层；主要工作就是和界面打交道，来制作前端页面效果；MVC 是后端的分层开发概念；MVVM是前端视图层的概念，主要关注于 视图层分离，也就是说：MVVM把前端的视图层，分为了 三部分 Model, View , ViewModel。
  • Vue.js 的目标是通过尽可能简单的 API 实现响应的数据绑定和组合的视图组件。

简单来一段

vue通过数据绑定和渲染，不再操作DOM元素，而是专注于业务逻辑。它的优秀的地方还有很多，大家有兴趣可以玩玩。

掷钱币的“神秘主义”叙事（四）

孔明曰：“亮虽不才，曾遇异人，传授奇门遁甲天书，可以呼风唤雨。都督若要东南风时，可于南屏山建一台，名曰七星坛：高九尺，作三层，用一百二十人，手执旗幡围绕。亮于台上作法，借三日三夜东南大风，助都督用兵，何如？”

这是《三国演义》中诸葛亮火烧赤壁的故事。诸葛亮借奇门遁甲天书，呼风唤雨，好不厉害！这里所指的天书其实指的是《周易》。

在前面三章中，我们可以得出结论，《易》实在是一门自然科学和一门社会科学的集大成者。只是后人多不理解，以致成神秘的物事了。在火烧赤壁这段中，也是利用了自然天气的变化来因事利导。至于作法，无非是一种心理暗示：敌人再强大，我也有神灵相助！诸葛亮善于利用天气和人心，是个实在的气象学家和心理学家。但被后人文学化后，就变成“多智近妖了”，实在是遗憾。

在现实的场景中，如果你要计算1+1等于几，你可能心算就完事了，但如果你要计算23456*879548等于几，你可能就要求助于计算器了。同样的道理，如果你有什么难事，难以决断，也可求助于《易》。它是一门指导人行事的学说，前提是你要先找一个切入点。这个点就是你要先投一把钱币。

投钱币的随机性和《易》中的变化之道有种联系，可以预先设定为64种范式，也就是64卦。把你当前的状态代入进去，以据求解，就可得到解决的办法了。

得到卦象的办法：

取用六枚铜钱，置于龟壳或竹筒中，单手拖握，单手封口，上下摇晃6下以上，将封口手放下，轻轻将器物内的铜钱依次倒出，铜钱正面为阳，背面为阴共用六铜钱衍生八八六十四卦，前三铜钱为外卦，即为先倒出来的三个铜钱一二三，后三铜钱为内卦，即为后倒出来的三个铜钱四五六，每三个铜钱有八种变法，上八下八，刚好八八六十四种变法，即为八八六十四卦！

以得到的卦象求解，就可得到解决之道，是不是很神奇！

许多人可能听说过“蝴蝶效应”：“一只蝴蝶在巴西扇动翅膀，会在得克萨斯引起龙卷风吗？” 蝴蝶效应正式的科学术语是“对初始条件的敏感依赖性”，也就是说，每一个事关对象的偶然性因素，都包含了对象的必然发展结果的信息。用变换方法，可以把无穷大的、不可预见的东西化为可以处理的量！这同《易》占卜算卦所提供预测方法，非常的相似。有兴趣的朋友不妨一试哦。

周末影院：《掠食城市》，末世的抢食之路！

《掠食城市》，一部反乌托邦的末世电影。在未来的某一天，人们都不从事生产，只是把城市当车开，到处抢掠为生。里面一个个人们的聚集的城镇都被装上了巨大的车轮，可以四处开，真是观光旅游的必备啊

里面的场景做得无比细致和真实，构成一种强烈的视觉反差和张力。故事有点乱，有点《哈尔的移动城堡》的黑化版。有两个对立的终级大boss，你打我一炮，我打你一炮这样的开撕。场面是大，可惜有点乱打一通的感觉。

里面还有不少魔幻的元素，比如这个骷髅头，我就不知它是靠什么活的！它还养大了一个女孩，还爱上她了，最后莫名其妙地情伤死了，让人一头雾水。

比起大部分的无脑片，这部还算是脑洞不小。其实里面有很多线索可以深挖，可惜时间短，未及展开。我估计拍成电视剧的话，又是一部《冰与火之歌》了！
