每周我的心都悲伤七次--时代的歌之197

我被流放了……
在这暗无天日，
蝇营狗苟中度过。
唯一高尚的对话，
是和镜中的我交谈！

用牙齿来尝试一生的问题，
碰壁了，
断折了。
牙碎了，
往肚里吞！

就是这样，
一生都在舔舐伤口。
对人残忍，对已残酷。
对世界没有怜悯，
心底只剩悲伤。

在白天挖掘坟墓，
在夜间埋葬自身。
命运是一滩瘀血，
在叹息中酝酿癌症。

在痛苦中强颜欢笑，
在悲伤中迁徒漂泊。
人生寂寞如血，
只有勇士才能再次踏上征程。

下一步监管是否关闭矿池呢？

        以监管部门的无厘头个性来说，关闭矿池是一句话的事！请抱歉我用“无厘头”来形容，我实在找不到合适的词来界定。

9月4日中国人民银行等七部委发布了《关于防范代币发行融资风险的公告》，要求中国境内融资代币和虚拟货币的交易做出全面清理整顿工作，OKCoin币行在今天下午正式接到了监管部门的通知和指导。

OKCoin币行作为一家服务全球几百万用户的综合性数字资产交易平台，自2013年成立以来，一直不断进行技术和产品创新，以推动区块链和数字货币行业健康发展为己任，希望能够越来越好的服务广大用户，我们深感自己肩上背负的历史使命，通过4年的不断奋斗和贡献，我们也取得了很多用户的信任，OK人也一直因此而骄傲！

        基本措词一样的公告我们看了不少（看来国内交易所的老板是一个人），我们都知道OKCoin币行并没有从事过ICO代币的发行，其实并不适用于《关于防范代币发行融资风险的公告》。它也要关闭有点说不通。公告中只是笼统地形容为“监管部门”，到底是哪家部门呢，我们是一头雾水，所以我们只能有着看周星驰电影无厘头的感受了。

        下一步就是剑指矿池了！全球排名前十的矿池中国占据了七成吧。它会否成为监管的阴影地带呢？我估计是否定的。所以，鱼池啊，国池啊，，蚁池啊，矿老板们赶紧洗干净点，找你谈话的时候不至于太难看了。大矿工小矿工们赶紧把矿机往山区挪挪，实在不行，找个山洞藏起来也好。

        马三立有句话，逗你玩！

人像修复大作战 （一）--PS研习社#4

        人像修复是PS中最常用的功能，也就是我们常说的“照骗”功能了！因为条件的限制，原始照片存在很多不足，比如曝光不足，太灰，层次感 不够等等，这些都是由后期来完成的。人像修复也是经常用到的了，时尚杂志，淘宝爆品，影楼婚纱都少不了它。对设计师来讲，都是一个必备的技能。

        这里介绍两种人像修复的方法：蒙版修复和通道修复。

蒙版修复


原图

第一步


ctrl + j 复制一个新图层，对其进行 滤镜 -> 高斯模糊。

第二步


对新图层添加一个蒙版，填充黑色。


再用画笔将缺陷的部分小心地涂抹掉。

第三步


将两个图层合并，对其进行 图像 -> 调整 -> 色阶 ，调整对比度和亮度
