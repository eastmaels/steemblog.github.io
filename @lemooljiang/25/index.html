用谷歌云升级下科学上网工具吧 / 网络研习社#25

在上一篇中已经成功地连上了谷歌云，当然第一件事就是给我们的科学上网工具升下级！如果以前有过经验，那还是相当容易的。可以参考下以前的一篇文章《vps + shadowsocks科学上网神器》。

配置起来也就几步：
1、安装依赖和安装 shadowsocks服务端



2、配置文件

各字段的含义：
server 可以填入 0.0.0.0 或是服务器内网ip
server_port 服务器端口
local_port 本地端口
password 用来加密的口令
timeout 超时时间（单位/秒）
method 加密方式。可选择 “bf-cfb”, “aes-256-cfb”, “des-cfb”, “rc4″, 等等。默认是一种不安全的加密，推荐用 “aes-256-cfb”。

3、启动和关闭



4、配置客户端就可以了，小飞机起飞了！

有人称谷歌云是用的自己的海底电缆，所以在响应速度上会比其它的服务器快很多。据我自己的使用体会，谷歌云是比linode快个两三倍的！
