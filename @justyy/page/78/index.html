试用 Linkedin (领英) 高级帐号 (Premium)

Linkedin (领英) 算是比较靠谱的职业社交网站，在上面有很多猎头、很多知名公司的HR 无时无刻在招人。特别领英在被微软收购之后，名气就变得大了许多。

领英是免费使用的，但也有付费用户，有给猎头的，也有给想找工作的。价格并不便宜，对于想找工作的 Job Seeker，一个月可以花26英镑左右成为高级帐号 (Premium)。

还好有一个月试用，只需要绑定信用卡或者 PAYPAL就可以开通高级帐号。

高级会员名字旁边会有一个土黄色的 Linkedin 图标。

领英高级功能

Linkedin Job Seeker 的 Premium 帐号主要增加了：

5封站内信 5 InMails Per Month

一般情况下，你是不能直接联系不是你朋友的帐号。有了高级帐号，你一个月就可以联系5个人了。这个目测还是比较有用的，因为你可以直接联系公司的HR或者相关技术总监、推荐自己。

谁看了你的主页 Who’s viewed your profile

普通帐号并不是能够看到所查看你主页的人，有了这个功能，你就可以知道谁对你感兴趣了，特别是一些通过关键字查到你的 公司HR或者相关技术人员。

工作洞察 Job Insights

这个功能 主要是会根据你的简历还有你的一些浏览记录 给您推荐一些相关的工作，还可以按照相关度来排序。

精选的申请者 Featured applicant

这个功能就是在你申请工作的时候，你的名字会被加上一个 Featured applicant 的标签，比较醒目。

学习 Linkedin Learning

领英上有很多学习的视频，有课程，加入付费帐号后就可以学习了。

开放的简历 Open Profile

成为高级会员后，您的简历就可以选择公开了，任意人都可以看到并且联系你。比如在未登陆下，我的主页是：https://www.linkedin.com/in/drlai/

感觉很多功能都是很基本的，领英一个月收费这么贵就为了几个基本的功能 想想实在不值。不过如果你最近在找工作，花几十英镑升级一下估计还是有点点用的。

Coding Exercise - Find Max Consecutive Ones 编程练习题 - 最多连续的 1

Given a binary array, find the maximum number of consecutive 1s in this array.
Example 1:
Input: [1,1,0,1,1,1]
Output: 3
Explanation: The first two digits or the last three digits are consecutive 1s.
The maximum number of consecutive 1s is 3.
Note: The input array will only contain 0 and 1.
The length of input array is a positive integer and will not exceed 10,000

We count of the current consecutive 1s and the maximum count when iterating the numbers from left to right. If current element is 1, we increment the counter and compare to the maximum. Otherwise, we reset the current counter.

Using Python, this can be solved in a slightly different way - join the array as a string, split by delimiter ‘0’ and get the maximum length of the 1s.
