Image Credit: Pixabay.com

Steemit Online Tool - Earnings Report 在线工具 Steemit 收益报告


Image Credit: Pixabay.com

This tool generates a payout report so you can check the earnings between any two dates. As payout for posts less than 7 days are not finalized yet, the curator figures will be zeros. Comments are also included in the calculation. API will be provided as free-of-charge but use it at your own risk.

Tool URL: https://helloacm.com/tools/steemit/payout/

Tool Tips:

  1. The table is sortable, so you can sort by payout, net_votes or timestamp.
  2. You can book mark the tool by adding ?id parameter so you don’t have to type in your SteemId everytime.

简单来说，这个STEEMIT在线工具能让你查看任意一个时间段里你的收益情况，它会列出每篇收益不为零的帖子或者回复，并且会告诉你一个收益总数。7天内的帖子点赞收益(Curator)为零，因为还没有到结算的时候。这个工具提供了一个免费好用的API，可以很方便的返回两个日期内STEEM用户所得到的收益情况。

工具地址: https://helloacm.com/tools/steemit/list-of-payout/

用上小霸王键盘 （换换键盘，换换心情 ） Using AT keyboard on modern PC using AT-PS2-USB connector

I have used a AT keyboard from a 8-bit famiclone on my desktop via AT -> PS2 -> USB adapter. This keyboard is from 1990’s and it is one of my precious collections.

今天翻箱发现还有一个庞大的键盘，不过这是90年代小霸王学习机（SB2000）的分体式键盘，键盘接口是 AT口，可以理解成是PS/2 的大型版本。

于是带到公司，接上AT-PS2转换口，然后发现现在的台式机已经没有 PS/2 口了，所以还接上了一个 PS2-USB的口，然后把我的机械键盘（黑寡妇 Black Widow) 先放一旁，在办公室里用上了学习机的键盘，也是醉了。

真暴露年龄，我上小学那会，学习机很流行，小霸王学习机，其乐无穷啊，于是当时吵着闹着骗父母说买学习机是用来学习的，但是，你们懂的……当时的学习机可以接软驱，还有8位的DOS操作系统，甚至还能上网，这在当时简直牛X到不行了。

这么多年过去了， 偶尔还是会打开电脑上的模拟器，玩玩8位超级玛丽，不过，那种感觉再也找不回来了。

@happyukgo 到我55岁的时候，我退休金都给你支配！(在英国只靠国家的养老金 State Pension 是不够的)

在英国，每个人都会有一个NI 社保号，用于记录你的工作交税状况。英国政府HMRC税务局可以查交税记录。只要累计工作交社保税30年，那么在你退休的时候就可以领国家养老金 (State Pension)。我查了一下，我每个月大概交了200多英镑的社保税，那么在我满68岁生日那天，我就可以领取每周159.55英镑的国家养老金（一个月是693.76英镑）。

这只是预测，政府还说明了，这个数字只是依照现有的规章条例估计出来的，并且这个数字还未考虑通货膨胀。我已经工作了7年多，得再继续每个月交上这个社保税22年，那么在2039年的生日（55岁）就可以确保国家养老金的这个福利。

假设现在已经退休，如果没有房贷的压力，一个月700英镑是大概正正好，不会穷得连饭都吃不上，但是也甭想世界那么大我想去看看。挺多就是每天去酒巴坐坐喝杯酒发发呆。


Image Credit: Pixabay.com

还好，每个人允许有多份养老金，比如我们公司就有额外的养老金，每个月我的工资里4%被拿出来当养老金，公司会替员工交另外5%，每年养老基金会都会来信告诉你一年来交了多少还得交几年就能在55岁的时候开始享受。这样在我退休的时候，每月就能稍微富裕一些（养老金也是收入，也得交税）。

每个人在年轻的时候就得做好计划，在英国可没有养儿防老这种观念，孩子成年后就基本上和你没关系了。我听说英国一些家庭在孩子18岁成年后就把孩子赶出家里，如果继续住家里就得交房租。虽然很残忍，没有人情味，但是孩子较早出去锻炼也未尝不是件坏事。当然，我还听说另一个故事：养老院给孩子打电话说可能老人这几天就会离开，孩子说知道了，并嘱咐说晚上的时候不要打电话（怕影响到自己的孩子休息），只需要第二天告之便可。

年轻的时候多挣些钱，多做做计划，陪你一辈子的不是孩子，而是你的另一半。 我现在就在STEEM区块链上做保证： Hi, @happyukgo 到我55岁的时候，我退休金都给你支配！
