Adding `Easy Switch Between Utopian and Steem Posts` to Utopian Moderator Chrome Extension

Utopian Moderator & Supervisor

Utopian Moderator & Supervisor is the FIRST chrome extension that is intended to help the Utopian Moderators & Supervisors in moderating work.

New Feature

As promised earlier, I am adding a very useful feature to the version 0.0.2: Allow Easy switch between Utopian and SteemIt posts.

Configuration

Adding a Dropdown menu to allow user select the prefered ‘steem’ sites i.e. steemit.com or busy.org (more can be added in the future)

Now, as you are moderating utopian posts, you can easily switch between Utopian to the preferred steem sites by Alt+S short cut. For example, utopian.io -> busy.org and busy.org -> utopian.io

声明：本文纯瞎BB，请勿对号入座，你们都很棒，么么哒

表面上是赚到了，实际上是亏的

那天，我和媳妇说，这半年来我好像不怎么抑郁了。她说是啊，你天天忙着Steem，哪有功夫想七想八。

这几个月来，我几乎尝试过了SteemIt 上所有的可能（开了YY银行、玩Utopian、晒媳妇、写各种工具和API、办过一次比赛……），玩 SteemIt 玩得很 High，表面上是拿着业余零碎时间挣些小钱赚到了， 实际上却是亏的。为什么这么说？因为时间没有拿来做更有意义的事情，虽然短期内是得到了些好处，但长远来说，浪费了最宝贵的时间，有可能就和错过了几个亿的黄金机会。

SteemIt 当成业余爱好，记录记录生活顺便挣些广告费，这种心态是最好的，然而，很多人却本末倒置了。把SteemIt 当成是刷钱的手段。骗赞、博眼球各种手段应接不暇，失去了最原始的目的。

很久之前，我写过了《不要让 SteemIt 毁了你原本的生活》，现在看来，并不是危言耸听，很多人中毒了，再戒毒，一碰上又上瘾了，反反复复，屡试不爽。

能离开 STEEM的人才是真有本事的，因为他们不缺这钱，有更重要的事情要做。玩SteemIt 很费时间，天天刷收益，天天写水文，不仅你在浪费自己的时间，而且也在制造垃圾，浪费别人的时间。牛的人就是玩了一会，玩明白了原理，然后就慢慢离场，也许他们留下了点赞机器人，又或者他们借出能量每天收利息……

我自从玩上Utopian 后，大部分时间花在写代码了，姑且不能算作浪费时间。写代码比写文字更能让我快乐。这也让我渐渐改掉犯懒的毛病，一年前，每天下班回家，我就是上网玩DOTA游戏，看YOUTUBE各种综艺八卦节目，而现在，白天上班，晚上也算加班，一天恨不得48小时。

能离开 STEEM的人才是真有本事的，我没本事，所以姑且还是先玩着 SteemIt 吧。😂
