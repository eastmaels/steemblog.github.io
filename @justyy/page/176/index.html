其实，忘记和大家说了，我拉了一些好朋友一起来玩 STEEM，并且 我自己也有几个小号，这些号的 posting key 给我管，我索性让它们和 @justyy 机器人一起点赞。

所以，你懂的。当你得到 @justyy 点赞的时候，你肯定还得到十几个小号再给你点赞。

今天，一共有15个号，构成了YY银行足球队（包括替补）。你还有啥理由不加入银行？

加入银行股东需要 自己的 SP，新号一般会得到系统代理的20多SP 不是你的，所以不能加入股东，但是却可以加入足球队，完成梦想。

加入足球队的好处

你需要把 posting key 给我（加我微信 ACM-er) 你会得到：

  1. 每分钟自动领赏，从此你不需要自己去点那个 claim rewards.
  2. 您答应跟随 @justyy 点赞，点赞比例由程序设定，点赞收益归你。当足球队员能量少于20%将会休息不跟随点赞。
  3. 您将得到 和 股东不一样的点赞比例，当前暂时设定2-5%左右（根据能量调整），并且每24小时只能被点赞一贴。
  4. 您得到的是大约10几个机器人的点赞。其中 @justyy 等级69，对于新人升级是非常有帮助的。
  5. 您的隐私将得到保障（我不会公开说你把 KEY给我了）
  6. YY足球队只会用你的 posting key 来点赞和收益，承诺不会做其它的事情（比如发文）。

退出足球队

最好，你告诉我你要退出了，我就会将你的KEY删掉，当然你也可以更改您的密码，这样 posting key 将会重设。

这是抱团取暖的一种方式，适用于一些不爱自己点赞的人，当然你也可以考虑加入 YY银行，成为股东，两者不冲突。

通过 SP 代理工具 成为 YY银行股东，好处多多。只要代理大于10 SP给 @justyy 即可自动成为YY股东。用同样的工具输入0取消代理退出股东。来去自由，取消代理后系统需要7天才能将您代理的SP退回到您的帐号上。友情提示，不建议把所有SP都代理给银行，因为你需要留一些能量发贴。

剑桥华人孩子们的音乐晚宴 Children's New Year Music Showcase Dinner Cambridge

A picture is worth a thousand words… so I believe this post is worth of many thousand words… these pictures were taken today when we attend the Children’s New Year Music Showcase Dinner in Cambridge.

今天参加了剑桥华人孩子们新年音乐晚宴，一家四口，费用是110英镑，一共出席了60人，把整个音乐餐厅给包了下来。

音乐晚宴开始了，孩子们依次按年龄从小到大上台表演，这是表演节目列表，我家孩子第一个出场（5岁）

YY 银行开启互抱大腿模式 (mutually-beneficial voting scheme)


图片来源： 每日BING桌面壁纸 @superbing

之前说到 加息，加点赞比例，但是解决不了一个问题，就是YY银行发利息的的收入如何增加？有利息才能更好的加息啊。

于是，我想到了一招，先让我沾沾自喜一会儿 :)

那就是，开启互抱大腿模式

如何互抱大腿？

首先，你需要加入 CN 区会员列表，可以微信我 ACM-er 或者提交一个PR 到 github

如果你不满足这个条件，YY银行还是会给每日点表中的前30名点赞，只不过是低优先级，低比重，并不能保证（说白了，这不是YY银行条例，行长只是友情点赞，不负责）

然后，这个模式需要你先代理 SP 给 @justyy 只需要通过 这个工具 代理大于10SP 就能自动加入YY银行。

通过这个在线工具 查得现在一共有 83 个人委派代理SP给银行，一共17307.26 SP, 35455302.02 VESTS。

只要成为会员，并且在 YY 银行点赞的时候 （大概是你帖子成功发出的27分钟到4小时之间，过期不补），YY银行会检查你在这个时间前24小时内总共为 @justyy 点赞的权重值 w （单次点赞需要至少5%），假设 w = 300 也就是 100%点赞3次。

那么，YY银行的点赞百分比会比原来的基础上额外的增加：

也就是 17%，YY 银行现在的100%的钱（如果 VP = 100%）是 $6.78 ，17%也就是 $1.15 当然，考虑到能量损耗（现在差不多是半血量），点赞收入也会增加0.5$ 左右。这里的 $ 应该是 STEEM 7天的均价，所以一个赞多个几美元不是问题，你只需要支持一下 @justyy 即可。

PS： 如果 机器人 能量小于 40, YY机器人会休息一会….
