Utopian Moderators & Supervisors is a Chrome Extension that is aimed to provide useful information and statistics to the utopian moderators & supervisors.

New Features in v0.0.5

This version has the following changes:

Adding Unreviewed Contributions

Clicking the item in the list navigates to the Utopian corresponding category for review.

Pie Chart that gives you quick idea on which category needs help most.

Listing the Approved/Rejected Posts

Sometimes, you need to quickly revisit your decisions, so the latest reviewing posts is a good tool to go back to the posts.

Other changes

Removing the Tab ShortCuts as it is now integrated in the Unreviewed Contribution
