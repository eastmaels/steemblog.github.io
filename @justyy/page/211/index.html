If we split the permlink string by ‘-‘, the last group is obviously the timestamp which we can discard. The rest are in the form of ‘re-author’ where the last group is the original author.

SteemIt Online Tool - Recover Your Deleted Comments - 黑科技STEEMIT恢复你的评论


Image Credit: Pixabay.com

@justyy mentioned this STEEMSQL Tutorial - Can we Really Recover Deleted Comments/Posts on STEEMIT? a couple days ago and @nationalpark mentioned this technique in his post to recover the deleted comments. But I am going to present this even further and better……

Comments/Posts on Steemit could be deleted if nobody has yet upvoted or commented on them. However, deleted Steemit posts/comments are still alive in the SteemIt blockchain and in theory, these data can be recovered. This handy tool allows you to quickly retrieve some steemit data (block number and more) for deleted comments/posts. The API could be used to further dig those data in the steemit blockchain.

Tool URL: https://helloacm.com/tools/steemit/deleted-comments/

Click the Link and it will show you what are the deleted comments

@justyy 几天前 STEEMSQL 系列之 STEEMIT真的可以恢复删除的文章或评论么？ , @nationalpark 在 他的帖子里 讲了如何恢复评论，那么，我要介绍这个更牛逼哄哄的工具了……

Steemit 的文章或者评论一旦发表，就被确认了并锁死封存于区块链上不能被修改 (Immutable)。文章或者评论在还没有人点赞或者评论前是可以被删除的，只不过被STEEMIT给隐藏起来了。本工具可以列出指定STEEM用户曾经删除的评论链接。该API则返回更回详细的区块链上被删除的块信息。

工具地址: https://helloacm.com/tools/steemit/list-of-deleted-comments/

Steemit Online Tool - Earnings Report 在线工具 Steemit 收益报告


Image Credit: Pixabay.com

This tool generates a payout report so you can check the earnings between any two dates. As payout for posts less than 7 days are not finalized yet, the curator figures will be zeros. Comments are also included in the calculation. API will be provided as free-of-charge but use it at your own risk.

Tool URL: https://helloacm.com/tools/steemit/payout/

Tool Tips:

  1. The table is sortable, so you can sort by payout, net_votes or timestamp.
  2. You can book mark the tool by adding ?id parameter so you don’t have to type in your SteemId everytime.
