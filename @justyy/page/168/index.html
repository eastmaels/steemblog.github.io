我玩区块链虚拟货币的原则 - 月旦评


Image Credit: Pixabay.com

玩区块链已经半年了，正式算起来应该是 @ned 把权利下放给 CN 区几大鲸鱼起，在STEEM上发文直到前两天才开始准备POWN DOWN。我经历了比特币从不到2000英镑到1万4000英镑，也经历了SBD不到1美元到15美元，然后今天一下子又跌到8块多。

起起落落，犹如人生。很多人赚得几个亿，实现财务自由，而我自认为没那个命，开心最重要了。我的几个原则，时刻提醒自己。

不花真金白银来投资

这只是我的原则，因为我很清楚的知道我不是搞这块的料，我看不懂K线图，玩不明白追涨杀跌，啥时候该加仓，啥时该离场。

人已中年，有妻有娃，不应该失去理智的把钱投资在自己不明白的领域中。有点虚，大家都想捞点好处，我没信心我会是幸运的那个。事实证明，每次我一卖啥啥就涨，一买啥啥就跌，真是有点邪门。

其实，风险不是在于项目本身，而是在于你投入的资金是否影响你的生活。我不是土豪，所以只能拿挣来的STEEM玩一玩区块链，了解一下交易所和其它货币，我已经很满足了。

不投入真金白银，你也只不过是多赚少赚的区别。

不玩短线，长期持有

玩短线，你需要一颗强大的心脏。玩短线，像我这种一买就跌的很有可能就短期套牢了。要是很信一种币，可以拿闲钱加仓长期持有。

我媳妇就不会劝我卖，在我和她懊恼说比特币卖早后，她说反正暂时也先用不上那钱，就放那里好了。

比你会玩短线的大有人在，玩短线有点像投机，需要运气、勇气甚至是一些内部消息，而这些，我都没有，所以我并不适合玩短线。

玩短线，很容易随波逐流。

只持有少量几个币种

你不清楚一种币，不建议盲目投资，我这几天玩了几个交易所，把一些币换来换去，发现越换越少(等价USD)。举个例子，你100个SBD换成300个STEEM，然后你再换回SBD就不太可能多于100个了。

玩太多币种，很容易分散精力。毕竟对于我现在而言，玩区块链只能算是业余爱好，有点不务正业，多学习，多条腿走路才是正道。生活和工作千万不要被这个牵扯精力，很容易玩物丧志的。

教你如何抱上 utopian 大腿, 如何最大化乌托邦点赞收益？

utopian 是 STEEM的一个很成功的 项目，我觉得 utopian 把 STEEM的价格（包括SBD)拉高了不少。utopian 是个比较公正的大腿，因为现在 审稿人 moderator 也只是决定帖子是否被 utopian 录用，但真正让 utopian 来点赞的比率则是较客观的。

当然，官网投票中有一项将来会添加的功能就是让审核人以 1分到10分来评价帖子的质量，不过现在还没有实现。 帖子被录用的点赞比率实际上在开源代码里都有：

代码是用 Type Script (也就是 Javascript的父集) 实现的。可以拉到 第410行：

这里定义的 categories_pool 里的信息很重要，因为utopian 的类别很大程度决定了收益。比如 development 的就不设置点赞比例上限 MAX_VOTE_EVER 而 建议 ideas 类别的帖子最大也只有得到 5%的点赞。大家可以到源代码上去看看你写的类别的比重是多少。尽可能写些贡献较大的类别，比如 development 和 sub-project。还可以挑一些比较少人写的类别，因为每个类别有固定的奖金池，utopian 机器人每天点赞总额是固定的，会按份量分给每个类别，帖子较少的类别反而比较有可能分到点赞比重大。

我们再接下来看第556行到559行

这个 finalScore 最后就是点赞的比重范围0到100。大概起始值就是该类别平均权重除于总投票数。但是还会根据以下几种情况进行调整。

如果帐号的粉丝数小于500就多给20%，你可以注册小号来发，每篇都给20%，很多的。并且如果是第一篇帖子就多送15%。所以新注册一个小号来发至少35%。不过，千万别说是从我这里学的。
