This Feature

Version 0.0.4 adds these new features!

Friends ID List

You can specify a list of steem IDs so that along with @utopian-io they will be displayed in the general tab.

Show Approved/Rejected Statistics

You need to configure your moderator steem ID first, and it will show how many posts you have approved and rejected.

By the way, I have moderated 100 posts so far and the approved ratio is 50% (not too bad, huh?)
