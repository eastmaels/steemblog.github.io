碎碎念第365天 (The Day 365 at SteemIt)

今天是玩 SteemIt 的第整整365天，在 steemd.com/@justyy 显示（其实已经显示 13 moons 还好昨天被我截屏了）

粉丝数前不久满500，也截屏保存了。

我也算是西半球比较有名的老博主了，在带头大哥 @tumutanzi 的介绍下加入 steemit 的， 陆陆续续发贴，坚持了一年。 最开始不相信能挣钱（因为没搞明白怎么把SBD转出），到最近两个月平均每天1到两篇的速度，也挣了些零花钱弥补家用。

我一直觉得SteemIt的目的很多样化：挣些小钱、交些朋友、玩些技术、帮助些SEO、记录些生活、分享些好玩的。目的越多样化，你就会觉得 SteemIt 越好玩。

不以物喜，不以已悲。很高兴能加入这个大家庭，认识了很多朋友:

  • 足不出户就可以通过你们的分享了解世界， @sweetsssj @nicolemoker @susanlo @joythewanderer 等
  • 从技术大牛那学到书本上学不到的知识，很过瘾，感恩 @oflyhigh @jubi @myfirst @dapeng 等

我想我玩 SteemIt 虽然方方面都不太精，但是玩得还算多样化：

出租 SP 的一些经验之谈

SteemIt 在使用了@minnowbooster 几天之后发现这玩意还是有搞头的，你可以在市场里出租你的SP，也可以直接把SP代理给机器人，甚至你也可以通过 Power Up 方式投资 @minnowbooster 机器人（但是这种方式本钱就无法收回），这三种方式都是每天获取回报。

由于直接 Power Up 的方式无法取回本钱，所以我只试了在市场上代理 SP 和直接代理给机器人，为了收益，我各出租了 375 个SP

然后每天的收益是：

直观的比较就是：通过市场的代理投资回报率更大一些。

官网上显示，通过市场投资年回报率大概是　２４％

(**) NETT APR is the total annualised return received by the lender after facilitation fees and the one week delegation cooldown period have been factored in.

如果是投资给机器人：

Delegate STEEM POWER to the @minnowbooster bot and get roughly the returns you would get if you voted up only your own posts, but without the shame! You will earn and be paid daily with a prorata share of the funds paid by users for @minnowbooster to up vote their posts, less our facilitation fee.
You can always undelegate but will loose the earnings for the current unfinished day.

大概就是每天点赞的收入。

所以我的建议是：

  • 如果你要去旅游，或者很忙或者有一段时间无法上网，完全可以把多余的SP租给机器人，这样的话等于你每天都有在点赞，有收益（有没有像债主每天都在收利益）。这会以SBD的方式打到你的帐号。
  • 时不时可以通过市场以较高的利息出租你多余 的SP。这会以 STEEM的方式自动打到你的帐号上。
  • 如果想收回 SP代理，需要等待7天的时间（失去当天的收益）。当然取消的方式也比较直观，就是再通过官网 https://www.minnowbooster.net/delegate 上的代理工具但是需要输入负数表示收回。

没有时间，文章看不完，点赞手点到酸，看着VP越来越少却又想每天有收益怎么办？把SP出租出去就能极大缓解这些问题，你说呢？

很好用的 桌面版 Steem 钱包 - Vessel

今天 发现一个桌面版的 Steem 钱包 - Vessel 挺好用的，所以介绍给大家。

这个钱包是 @jesta 开发的。

钱包下载地址是： https://github.com/aaroncox/vessel/releases

WINDOWS 选 那个 vessel-setup-0.0.6-h.exe 下载安装即可。

添加多个 SteemIt 帐号

如果你有多号，就可以通过这个软件来方便统一管理。需要输入密码，但是既然作者把代码都开源了，所以你大可放心的使用。

经媳妇 @happyukgo 同意，我把她的钱包也加起来一并管理。

有一个很明显的好处就是可以在钱包里认领奖励。不过需要输入本地钱包的密码（不是你 SteemIt 帐号的密码）

钱包还可以方便 你在多个帐号里（包括交易所）转钱。

可以把 SP能量 通过 13周 Power Down 的过程变成STEEM

本地钱包也是加过密的。

交易所一般需要你填 MEMO, 这个桌面软件可以方便你保存，省得你每次都拷贝粘贴容易出错（我就有一次转到 blocktrades 转错了， 不过还好拿回了钱）

对我来说，Vessel 钱包最最好用的功能，还是这个，可以看看我借了多少SP租出去多少SP，这个在网页版里暂时还没有这个功能。

Code Review - Bound Parameter 代码审核之限制参数范围

There is a piece of bad-smelling code, I happen to read today. The following aims to compute the angle between two line vectors.

这两天又看到一个历史遗留下来写得很糟糕的代码。这个代码大概是求两条线的夹角。

This code has been in the code repro for a long time, and it is brought to attention because of a failing unit test, where the input to the Math.acos function is bigger but very close to 1. Math.acos function returns Nan in this case.

发现这代码是因为有一个单元测试不通过，DEBUG进来 发现是因为 Acos 函数在输入参数大于 1的时候返回 Nan，浮点运算有误差返回了类似 1.00000012 之类的大于1但是却很靠近1的数值。

Our ‘final’ solution is to bound the parameter to the function.
最后面的解决方法是：

It is not elegant, but it works. If the precision is a high priority, we can create a class that stores the parts of the floating number (integer and fraction) using integer type. The integers do not lose precision and we need to implement the arithmetic operations e.g. how to carry over the values.

通过这种方法把参数强制规定在-1到1期间。但是这样的代码并不优美。如果精度要求更严格，可以自定义一个数值类型的类，分开用整数来保存整数部分和小数部分，这样需要定义加减乘除的运算，但是由于整数部分参加运算没有精度损失，所以不会有这样的问题。不过得小心处理像进位这样的问题。

We also have the Math.Net library which provides a rich set of library regarding to Math vectors so this is totally re-inventing the wheel, however if this is a sandbox code, you may also consider operator overloading the vector type. The above code is easily error-prone because it is easy to mess up with the array indices.

撇开重新造轮子的问题（.NET 有像 Math.Net的库了），所以完全可以用现成的库来操作这些向量。即使你不用，你也需要考虑重载这些三维向量的加减乘除运算，上面的代码很容易下标给弄错了。不好维护。

节流、开源、投资

有这么一句话：富人的钱是省出来的。我并不是完全赞同，原因很简单，穷人再怎么省也富不了。每个月挣个几千块钱，你在北上广深试试，永远改变不了阶级。

我小时候的邻居，特别的省：衣服补了补又继续穿，晚上能不开灯就不开灯。十几年过去了，家里还是一贫如洗。孩子也无所事事。

不铺张浪费绝对是要提倡的，但是绝对不要指望节省来希望自己咸鱼大翻身。这里其实只做了一半，最为重要的是拿着省下来的钱来投资自己或者投资孩子。

有的人（排除富二代官二代）天性较为浪费：剩饭过夜就倒，衣服破了就扔，但日子越过越好。究其原因，他们愿意花钱投资自己改善环境为孩子提供更好的成长环境从而挣更多的钱，良性循环。

我之前是很不愿意（几乎没）买过书的，因为我觉得书上的知识在网上都能搜得到，为什么要花钱买？但是至从我买了书，天天花些时间（哪怕10分钟）看个几页，坚持几天下来，受益匪浅。为自己（孩子）投资绝对不会错。我小时候家庭环境并不好，但唯一父母绝对会给我和我姐买的就是学习所需的材料：书。高中的时候很多同学上晚自习的时候都用随身听听音乐，我也想要。于是我和我妈说这随身听是用来学英语的。妈妈二话不说便给了我400人民币让我自己去挑。这事我后来一直很惭愧，因为随身听买来就没有用来听过英语磁带。

开源，尽可能多条腿走路。收入来源广，业余时间多做些项目挣点外快没坏处。总收入提高了，你甚至都不需要花时间纠结于如何省这个省那个。日子还越过越顺心。如果你的精力有限，把时间花在如何提升自己的能力，进而更能开源。节流永远都不应该是你生活的主旋律。

2011年从LUTON搬家到Sheffield 把存了一年的储蓄罐打开，清理硬币，大概有500英镑，这些钱是平时买菜找零所得。

硬币分类，然后拿到银行去存，银行会拿个秤，根据重量来计数，要不然数得数半天。
