SteemTools: Enhancing SteemJS Editor and a few updates

Introduction to SteemTools

SteemTools is a Chrome Extension that you can install for free at:
https://chrome.google.com/webstore/detail/steem-tools/emjfpeecopppojbhkigjjmcahbfahhbn
It provides a set of useful Tools for Steem Users

  1. For Opera browsers, the workaround is to first install Chrome Extension Gadget: https://addons.opera.com/en/extensions/details/install-chrome-extensions/
  2. And similarly for Firefox, you can install Chrome Store Foxified before you install SteemTools: https://addons.mozilla.org/en-US/firefox/addon/chrome-store-foxified/

CONTRIBUTION

Fully Open Source: https://github.com/DoctorLai/SteemTools
You can either submit a Pull Request or open a issue (for suggestions and bug reports)
