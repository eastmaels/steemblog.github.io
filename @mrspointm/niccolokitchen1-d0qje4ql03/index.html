「探店」尼依格罗酒店 · 欣厨 Niccolo Kitchen #1

自从有了小孩，我和@mrpointp就很少有时间到成都吃喝玩乐了，来自时间、精力以及经济的压力使我们的生活质量大幅降低，因此前几天我们决定去成都缓解缓解压力。


Image Source

成都这座城市真是神奇的地方，你可以看到IFS、太古里城市的摩登景致，也可以看到武侯祠、宽窄巷子的古色古香；可以品尝源自各地的高大上美食，也可以体会背街小巷的成都味道。


Image Source

那天在一家中餐厅吃了午饭，忽然觉得难得来一趟成都，不能在吃这方面就如此草率略过，于是选了一家我“觊觎已久”的西餐厅欣厨，来个舒舒服服的下午茶。

欣厨隶属尼依格罗酒店的意大利餐厅。整体环境走的是现代简洁路线，配色干净大方，很有品质感。

大众点评一查，人均200+，幸好美团给了我们一个进店的理由，购得现价138元的下午茶甜品套餐。套餐包含七样甜点，两杯饮品，看起来还是内容满满。


Image Source

说说这家店的环境吧，用两句话形容：外形高大上，装修有气质，典型的西餐厅环境，简单、干净、整洁，服务员和服务生纯西装打扮，对于我们这种不经常光顾西餐厅的人还算得上高大上。

言归正传，讲一讲我们套餐中的几道甜品吧！

经典拿破仑

拿破仑蛋糕已经属于满大街都能买到的甜品了，但并不是每家店都能做出美味的拿破仑蛋糕。小时候吃任何蛋糕都觉得好吃，长大后吃多了就觉得蛋糕与蛋糕之间还是有区别的。


而这份拿破仑蛋糕（左一），它酥酥脆脆的酥皮决定了口感，丰富的夹心包含细滑cream配合香香脆脆的核桃，搭配酸酸的草莓不仅美观且减去部分甜腻。
