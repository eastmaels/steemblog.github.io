「电影推荐」亲爱的 —— 令人潸泪的题材，直接赢在起跑线

2014年上映了一部名叫《亲爱的》的打拐题材电影，虽然它远不及《战狼》这种类型的电影名响全国，一时间成为各大媒体、自媒体甚至老百姓热议的焦点。

但无法否定的是，这部由黄渤、赵薇、郝蕾等主演的电影仍然深入人心。


Source

时隔五年，我二刷这部电影，这一次我是以一个母亲的身份重新审视它，我的感触远远超过了2014年第一次观影时的心得。

从商业的角度讲，“打拐”这样的题材无疑会给一部电影加分，只要剧情不是太糟糕，那么这部电影肯定会被大多数人称赞。

正如爱国主义电影《战狼》系列，他们也是情怀大过电影本身的作品，也许这时候有人会站出来质疑：这是在消费观众的情感吗？

然而站在社会的角度讲，这样正能量题材的电影的涌现却是好事情，消费不消费感情是次要的。

在这个物欲横流、娱乐至上的时代，人们对物质的极致追求是经济发展的表现，却也是精神文明垮掉的开始。

《亲爱的》《战狼》《烈火英雄》这些电影的出现，就是在警醒世人，这个世界还有很多更值得关注、值得思考的东西。

因此我想表达，不要总是站在批判的角度看待电影，看待导演，我们应该看到的是一部电影带给这个社会的影响力和正能量。

《亲爱的》和《战狼》不一样，后者是不需要过多思考的，它十分鲜明地告诉观众爱国是怎样一种精神和情怀。

而前者却需要每个观众自己去思考，打拐只是载体，它想告诉观众的是这个社会对于儿童拐卖这样的恶劣事件是怎样的态度。

我想说，这样的题材，如此的演技，这部电影不得高分都难，不赢得人心更难。

「探店」三号院精致川菜 （IFS店）#2

这家店的麻婆豆腐我们能给8分，算得上色香味俱全，但不得不说算不上极佳的菜品。

红糖糍粑&芒果椰汁黑糯米



作为经典的中式小吃，红糖糍粑一直是我喜爱的美食，香甜可口的红糖搭配柔软醇香的糍粑，简直不要太好吃，芒果椰汁黑糯米是我随意点的甜品，味道也是蛮香的。

绍子蒸蛋


个人认为，绍子蒸蛋的两大必备特质就是蛋质滑嫩、汁水醇香。

图片中的绍子嫩蛋基本上还是保持了这两大特质，就着米饭特别入味。

川北凉粉


凉粉是夏天必点的菜，工艺地道的凉粉，清凉爽滑，在口中细细品味就可以感受到其原料的醇香味。

这份川北凉粉真的很好吃，首先食材比较优质，红油香辣十足，豆豉的香味更是将整道菜的味道丰富优化了。

菠萝油


菠萝油是打包自太古里的翠园茶餐厅，竟然可以把菠萝包做得那么美味，在我看来它是这顿饭最美味的一道菜。

热乎乎的菠萝油，端上来立刻香气扑鼻，尝起来更是美味十足。

下次来到三号院，我会把重心放到热菜，好好品尝一下这家川菜馆，来成都游玩的朋友也可以到IFS尝尝三号院。

「探店」三号院精致川菜 （IFS店）#1


Image Source

今天要推荐的餐厅位于成都IFS国际金融中心6楼，名叫“三号院”，对成都比较熟悉的朋友可能听说过这家餐厅。

三号院的店招牌是繁体的“號”，这样的名字颇具意境和内涵，再加上其中式复古的装修风格，用餐环境是比较雅致的。


Image source


Image Source

三号院以精致川菜为特色，一家开在成都给自己的定位是精致川菜馆，那一定是很自信的，毕竟川菜馆在成都遍地都是。

为了品尝这家餐厅各个层次的美食，我和mrpointp不仅点了几份热菜，还点了甜点和小吃，上了菜才发现我们确实是“眼大肚皮小”。

酸菜木桶鱼


我喜欢吃两种鱼，第一个是麻辣鱼，第二个当属酸菜鱼。酸菜鱼好在不仅可以吃鱼，还能喝汤。

这份酸菜鱼采用乌鱼，这也刚好对了我们的喜好，虽然乌鱼的肉质算不上十分细嫩，但还是很合我们口味的。

这道菜不足之处就是酸菜还差了点滋味，但总体来说我们是比较满意的。

麻婆豆腐


听很多人说，一家川菜馆手艺如何，其实未必要看硬菜的手艺，反而就是普通炒菜更彰显美味。

麻婆豆腐历史悠久，今天在全国都能看到麻婆豆腐的身影，它的做法也随着不同的地区而稍稍改变。

「探店」粤乐港式餐厅 #2

广式腊味煲仔饭


@mrpointp向来不喜欢吃甜甜的食物，因此每到粤菜馆吃饭他常会点的就是煲仔饭和叉烧肉。

这份腊味煲仔饭其实就是用切了几片腊肉，不过味道还是挺不错的，只是希望分量可以略大一点。

香辣炒花甲


这道菜要重点推荐，尤其是喜欢吃花甲的朋友可以尝试一下。

虽然是粤菜馆，但我认为这道菜是根据川蜀老百姓的口味改良了的，味道香辣十足，花甲肉也是质量足足的，不像有些餐厅的花甲，经常会吃到空壳。

上海小笼包


这份小笼包与我们平常吃的小笼包最大的区别就是表皮晶莹剔透，馅料饱满滑嫩，包子皮不像是发酵了的，更像我们所用的饺子皮和馄饨皮的质地。

轻咬一口，还会有灌汤包的特质，鲜香的汁水立刻滑入口中，香气四溢。

深井烧鹅


这道荤菜可算是很出名的粤菜了，常常出现在各种港剧中。

总体来说，这道菜真的主要看食客喜不喜欢吃这种略咸略甜的食物，就像本身不喜欢吃辣的朋友是不太可能爱上四川火锅的。

品尝后，这道深井烧鹅勉强处于及格线，但我相信广东或香港当地的深井烧鹅会更美味。

再说一遍，这家店位于乐山王府井商场一楼，有兴趣的朋友可以品尝一番！

「探店」粤乐港式餐厅 #1

有朋自他市来，岂能不聚餐？


约好两位好友，相聚在乐山城。每当有朋友来到我的家乡，带他们去吃美食是不可或缺的行程安排。

朋友说想最近吃太多辣，想吃些口味清淡的美食，于是我立刻想起了乐山王府井一楼的一家港式餐厅——粤乐。


很喜欢粤乐餐厅的装修风格，清新典雅，用了大量的原木色，比起一些装修高大上的港式餐厅，

这家店仿佛更能带给人更舒适的用餐体验，仅是个人看法。

餐厅把各类点心摆放在玻璃柜台里，供顾客挑选。到了这家餐厅的时候是下午三点，由于吃过了午饭，所以只是抱着尝尝鲜的心态来的。

朋友们各自点了自己最感兴趣的菜，接下来正式介绍我们点的几道菜。

云吞面


云吞又称馄饨，起源于我国北方，后来在南方广为流传，甚至是发扬光大，在四川吃馄饨更是口味丰富，酸甜麻辣滋味齐全。

而广东等沿海地带的云吞虽然做法相似，但味道有所不同，他们喜欢把面和云吞一起吃，因此取名云吞面。

图中的这碗云吞虽然毫无辣椒，但做到了云吞应有的两个字：鲜、香。

蛋黄流沙包



我很喜欢吃流沙包，尤其是带蛋黄馅儿的流沙包。在这份金灿灿的流沙包刚刚出炉的时候，又软又烫。

这时候轻轻咬一口，里面的流沙馅的温度也是刚刚好，蛋黄可以抵挡过度的甜味，让整个流沙包尝起来鲜香美味，甜而不腻。

「电影推荐」友情以上 Friend Zone #3

女孩一直以来避免与男孩踏入爱情的禁区，最核心的原因就是“你太重要了，我不想失去你”。


Image Source

也就是说，反正不谈恋爱就不会分手，做朋友就可以享受一辈子的陪伴。

但这样的界限对于一直深爱她的男孩来说实在是太折磨了，看着你与别人甜如蜜，哭鼻子的时候却要帮你擦鼻涕。

好在电影结局是美好的，女孩终于勇敢地迈出了步伐，反正现在男孩都不理她了，还不如放手一搏吧。

其实我可以理解女孩这种害怕失去最好朋友的心情，但我并不赞成她在某些时候把男孩当成临时的避风港。

那样子目的感太强烈，有时候会忽略男孩的感受，这样的感情是有点自私的。

光说电影情节了，还是想谈谈《友情以上》的整体观影感受。

总体来说，这部电影充斥着轻松诙谐的氛围，毕竟电影结局还是美好的，所以算是一部实实在在的爱情轻喜剧。

电影叙述手法比较特别，以10年后的男孩口吻回忆这些年的点滴故事，听众是三个同样深爱着朋友却不敢踏出那一步的男人，这几个男人的对话承包了整部电影近乎一半的笑点。

不知道你对这部电影感不感兴趣，但我还是想用一句话表达：它解开了我尘封多年的少女心。

（完结）
