「探店」尼依格罗酒店 · 欣厨 Niccolo Kitchen #2


抹茶慕斯蛋糕

这家西餐厅的抹茶慕斯蛋糕（右上）最大优势就是抹茶的的味道很自然，不像许多甜品店的抹茶味都过浓，就会带来一种绿茶饮料的味道。

口感极好，软而不绵、略带弹性。

法式鲜果塔

法式鲜果塔（右下）的外形小巧而精致，本以为它会过于甜腻，但事实上更偏向清香淡甜类型的甜品，唯一不足就是水果少了点。


提拉米苏蛋糕

与拿破仑蛋糕一样，提拉米苏也是最热门的甜品之一。

先不说它好不好吃，从外形上就带给人西餐厅特有的高大上感觉，这是一般甜品店做不到的。

提拉米苏蛋糕（右一）很扎实，每次只舀一小勺含在口里细细品味，能品尝到一股很地道的可可味。

水果很新鲜，葡式蛋挞与一般的蛋挞差不多，没有什么特色，不过椰汁西米露让我感到很满意。很容易就辨别出来，此椰汁与我们平时喝的椰子汁是不一样的，应该是真正的椰子原汁，甘甜可口，这是我吃过最美味的西米露之一。


伯爵红茶和拿铁

由于我们都不太喜欢和纯粹的咖啡，于是就分别点了一杯红茶和拿铁。这里原谅我忘了拍照 T_T

伯爵红茶闻起来很香，口感也不错，至于拿铁，我想说这种西餐厅的拿铁果然要比一般的甜品店更正宗，咖啡味与奶香相辅相成，很适合喜欢咖啡又不喜欢太多苦味的朋友。


这是一家味道不错的西餐厅，之后一定要来常常欣厨的正餐。

唯一觉得不满意的，不知道是否是西餐厅的特色，菜品分量实在是太少，一个甜品吃不了两三口，这一份套餐供两个人吃确实有些意犹未尽。

「探店」尼依格罗酒店 · 欣厨 Niccolo Kitchen #1

自从有了小孩，我和@mrpointp就很少有时间到成都吃喝玩乐了，来自时间、精力以及经济的压力使我们的生活质量大幅降低，因此前几天我们决定去成都缓解缓解压力。


Image Source

成都这座城市真是神奇的地方，你可以看到IFS、太古里城市的摩登景致，也可以看到武侯祠、宽窄巷子的古色古香；可以品尝源自各地的高大上美食，也可以体会背街小巷的成都味道。


Image Source

那天在一家中餐厅吃了午饭，忽然觉得难得来一趟成都，不能在吃这方面就如此草率略过，于是选了一家我“觊觎已久”的西餐厅欣厨，来个舒舒服服的下午茶。

欣厨隶属尼依格罗酒店的意大利餐厅。整体环境走的是现代简洁路线，配色干净大方，很有品质感。

大众点评一查，人均200+，幸好美团给了我们一个进店的理由，购得现价138元的下午茶甜品套餐。套餐包含七样甜点，两杯饮品，看起来还是内容满满。


Image Source

说说这家店的环境吧，用两句话形容：外形高大上，装修有气质，典型的西餐厅环境，简单、干净、整洁，服务员和服务生纯西装打扮，对于我们这种不经常光顾西餐厅的人还算得上高大上。

言归正传，讲一讲我们套餐中的几道甜品吧！

经典拿破仑

拿破仑蛋糕已经属于满大街都能买到的甜品了，但并不是每家店都能做出美味的拿破仑蛋糕。小时候吃任何蛋糕都觉得好吃，长大后吃多了就觉得蛋糕与蛋糕之间还是有区别的。


而这份拿破仑蛋糕（左一），它酥酥脆脆的酥皮决定了口感，丰富的夹心包含细滑cream配合香香脆脆的核桃，搭配酸酸的草莓不仅美观且减去部分甜腻。

「学校那些事儿」当教师成了高危职业 #3

身处教师行业，我想谈谈自己的心声。


Image Source

在我们小的时候，只有老师打学生的份，后来讲究未成年保护，讲究学生的权利，最初的目的是为了保护学生，如今却成为一些舆论抨击老师的武器。

我们总听到这样的声音：现在的老师怎么怎么…其实我想说，任何一个职业，警察、医生、教师，没有绝对的好与坏。

然而在舆论的大肆渲染下，个例总是被误当作一个群体的代表，这是典型的以管窥天。

一个职业可以塑造一个人的形象，但一个人不能毁掉一个职业的形象。

现在的学生压力和诱惑太多，也许再也回不到我们读书时那个学生心态好，就算被老师打骂了也不记恨的年代，只希望全社会能将重心略向孩子的身心教育转移，素质教育不能只是口号。

也希望身为父母的人们反省一下自己，现在的孩子身处在色彩缤纷的世界，父母的教育方式也应该有新的思考。

文章尾声，真心希望那位老师能够战胜伤痛，早日康复。

对于教育、对于教师，我们应当多一些沟通少一些质疑，多一些理解少一些指责。

家长更应当履行好“第一任老师”的重要职责，不要让舆论和暴力事件浇灭了老师对这份职业的热情。

「学校那些事儿」当教师成了高危职业 #2

然而在转变的过程中总会出现厚此薄彼的问题，尤其是大多数学校和家庭在素质教育和应试教育两者间难以平衡。


Image Source

有这么一个问题：那个初中生如此暴戾、任性，归根结底是谁的错？我们可能会异口同声地回答：家长有着无法推卸的责任。

显而易见，家长对孩子的从小引导实在太重要，如今的教育体系无法对每个学生实现应试教育和素质教育的双向教学任务。

最关键的是，就连家长也更倾向于应试教育，毕竟在他们眼里，孩子的前途是最重要的。

孩子性格的扭曲以及良知的丧失大概生活在两种可能性的环境下：

一是父母过度管理孩子，这会让孩子长期处于压抑的状态，价值观也得不到正确的树立；

二是父母过度“放养”孩子，造成孩子从小肆意妄为，对生命和法律毫无敬畏之心。其他的原因不是没有，只是占少数情况。

很明显，子不教父之过是很有道理的。就算一个孩子天生顽皮，但如果父母长期合理地教导，也不至于连是非对错都无法分清。

本次案件中的初中生，冲动是一方面，但善恶不分地将老师的管理视为仇恨，这才是问题的根源。

有的家长会感到苦恼，认为自己也是随时教训孩子要听话、要理解父母和老师，但教育并不是仅仅用嘴巴说说。

教育得讲究方式方法，正人先正己，言传与身教同样重要，家长会犯的错误凭什么让孩子不犯，家长若要溺爱孩子那凭什么让孩子学会独立，如果家长不正确引导孩子，那凭什么不允许孩子以暴力电影人物为榜样。

「学校那些事儿」当教师成了高危职业 #1

前几天，我所在的城市发生了一件骇人听闻的案件，短时间上了微博热搜。


Image Source

一名初中生用砖块猛击班主任老师的后脑勺，至今这位黄涛老师仍在ICU接受治疗。

由于这件事发生在我们的兄弟学校，我感到惊恐。更因为我也是一名老师，我感到无奈。先来说说我所了解到的这个案件的前因后果吧。

事发后，当地公安局立刻发出通告，说明此事由一个15岁的初中生造成，原因是不服从该老师日常管理。

在之后的了解中，我才知道所谓的日常管理竟然是教导学生不允许在校内骑车这样的事情。

这名老师在遭到连续九下的猛烈攻击后，当时就不省人事，他被送入医院后生死未卜，全城的人都在议论这件事，所有人都希望这位老师能够挺过来。

近日据非官方消息说，被打老师已有意识，希望官方能够给出准确的后续通报。

事件发生后，在普通市民中成了热议的焦点。

在教师行业内更是议论不绝，有人在痛骂那个学生，有人为那位老师感到悲伤，也有人讽刺道“看来只有去当幼儿园老师安全了”。

在各种声音如潮流般涌现之下，我们更应当理性地思考近些年发生的一件又一件震惊社会的校园案件。

从上世纪80年代开始，教育不仅被国家所重视，越来越多的家长也开始抓孩子的教育。

全国高学历人才大幅增长情况大概是从80后的学生群体开始显现的，我国从以前“早点进入社会赚钱养家”的观念迅速过渡到“读书才是最好的出路”的观念。

「探店」梦雪丽法式西点 #2

抹茶千层


这道抹茶千层真是比我所看到过的大多数千层蛋糕都要精细得多，可以看出真的有很多层，制作这份甜品也一定相当复杂，稍不注意就会影响蛋糕的整齐感。

除了抹茶味，它还略带一股榴莲的味道，但只有细细品味才能感觉到，抹茶味和榴莲味结合竟然这么美味。

提拉米苏


提拉米苏应该是甜品爱好者的入门级甜品，因为它很常见，几乎任何一家甜品店都会制作提拉米苏。

但绝对不是任何一家都能做出这么好看而美味的提拉米苏，我想说这家甜品店真的把颜值放在首位，就这样甜品，令人忍不住想尝一尝味道。

相比大多数提拉米苏，今天介绍的这款的优点不仅在于外表，味道更是胜了一筹，另外，可可粉放得不多其实更好吃。

黑森林裸蛋糕


这份裸蛋糕的知名度绝不亚于提拉米苏，几乎每个人都吃过这样的裸蛋糕。

那这份黑森林裸蛋糕究竟有什么值得推荐的地方呢，那就莫过于奶油了，几乎品尝过这家甜品店的朋友们都说，这里的奶油一点也不腻。

水果茶


水果茶里的水果种类繁多，有苹果、草莓、牛油果、菠萝、百香果，再加上一包绿茶，淡淡的茶味与水果的香甜结合在一起真的非常美妙。

虽然这样一壶茶可以续水也足够我们喝一下午，但不得不说它的价格实在不低，一百元一壶。

与好友一行人，就在这里晒了半天太阳，补充了大量的维生素D，很久没有享受过这样的舒适了。@acactus1013的儿子洋洋把几种蛋糕集中在盘子里慢慢享用，感觉小孩儿比大人吃得更香。


别看这家梦雪丽法式西点价格偏高，但几乎每天都络绎不绝，到了节假日更是座无虚席，很多人是亲自驱车取甜品外卖，因为这家店是不送外卖的，更因为地理位置远离城市外卖公司也不接单。


假如有一天你来到这座小城市，一定要来尝一尝这家甜品的味道。
