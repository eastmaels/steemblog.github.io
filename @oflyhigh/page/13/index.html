每天进步一点点：学习使用LVM

为了方便动态扩容，我这次准备使用LVM来管理空间，不直接在硬盘上分区、创建文件系统以及挂载了。


(图源 ：pixabay)

确定新硬盘

操作之前，先看用sudo fdisk -l 看一下我的新硬盘，可以看到新硬盘为/dev/sdc。

sudo fdisk -l

创建PV

首先要创建PV，使用的命令为pvcreate(Initialize physical volume(s) for use by LVM)，我直接在新硬盘上创建。

sudo pvcreate /dev/sdc

创建VG

创建VG使用的命令为vgcreate，这里我为VG起的名字为vg_storage。

sudo vgcreate vg_storage /dev/sdc

创建完成后可以用vgdisplay查看一下：

sudo vgdisplay

创建LV

接下来是创建LV，使用命令为lvcreate(Create a logical volume)。

创建线性的LV命令为：
lvcreate -L|--size Size[m|UNIT] VG

我使用了如下命令：

sudo lvcreate -n lv_main -L 3200M vg_storage

创建完成后，查看一下：

sudo lvdisplay

擦，我原本想创建的是3.2T，结果弄成了3.12G，囧。

扩容

既然创建错了，那就直接扩容一下吧，弄成3T吧：

sudo lvextend -L 3145728M /dev/vg_storage/lv_main

sudo lvdisplay

这次看起来没问题的

再来看看VG

sudo vgdisplay

可以看到总的PE以及已分配的PE

格式化& 挂载

接下来将lv格式化

sudo mkfs.ext4 /dev/vg_storage/lv_main

创建目录并测试挂载：

sudo mkdir /lv_main
sudo mount /dev/vg_storage/lv_main /lv_main

编辑fstab

sudo vi /etc/fstab

添加如下内容：

/dev/vg_storage/lv_main /lv_main ext4 defaults 0 0

重启，一切正常，搞定收工。

大版主职务丢了

以前玩一些零零碎碎Arduino之类的硬件时，结识了很多朋友，有些朋友是一些技术论坛的负责人或者管理员，一来二去聊得很熟，就邀请我做他们论坛的管理员或者大版主之类的。


(图源 ：pixabay)

那些小的论坛就不算啦，有些注册过就没怎么玩过，但是有两个国内知名度还算比较高的论坛，我还是很认真的管理过一段时间的，其中做管理员的那个论坛，后来觉得人气不旺，就要求退出了，只在另外一个论坛做大版主。

做大版主有很多事情的，比如说删除一些垃圾贴，给一些好帖子设置精华或者给与评分之类的，还有就是参与互动，增加人气。

还有就是组织一些线下活动之类的，包括发起活动，拉赞助，找场地，进行活动，然后写新闻稿之类的。

虽然有些累，但是这些都是我比较愿意去做的事情，也因为这个结识了好多新朋友，有些现在还保持着密切的往来。

当然，大版主也不只是付出，还有有些福利的，比如一些厂商的硬件啊板卡啊，可以优先拿来试用，试用过后写个评测之类的，然后一般来讲板卡就归自己啦。

还有就是参与论坛活动/互动会有一些积分，有时候会有积分换礼活动，可以换一些礼品或者京东卡之类的。

再有就是，论坛的母公司经常会拿出一些经费，组织一些线下活动等等，比如春游、秋游，爬爬山之类的，但是活动地点一般都选择在帝都，我懒得特意跑去参与。

不过相比这些福利/回报，我觉得最重要的还是学到了知识以及交到了朋友，至少我觉得我积极参与那段时间，学到的东西还是很多的。

不过这两年很少玩一些IOT相关硬件了，所以也很少光顾论坛了，只是偶尔在管理员版主群里和朋友们水上一两句。

今天偶尔想起了这个论坛，登录上去看看，发现我应该是至少大半年没有登录了，站内信中一堆要和我交流的朋友，我都不知道还能不能交流起来，因为当初会的东西，现在都忘记到脑后了。


(图源 ：pixabay)

再看了看，发现大版主列表里已经没有我了，有些伤感，也有些释然，不付出，占着这样一个岗位终究不太好，所以这个职务丢掉就丢掉吧~~~~

买了一块SAMSUNG 883 DCT SSD

趁着双12，给大电脑再加一块SSD，这次选择的是SAMSUNG 883 DCT SSD，接口是SATA 3(6.0Gbs)，容量是3.84T。

废了好大劲把它塞到电脑里，至于为啥费了好大劲，一是现在大电脑已经很沉了；二是当初没打算用2.5寸的盘，就直接把支架拆卸下来丢一边，研究了很久才重新装上去。

这个盘看起来和我老笔记本上再用的840 PRO挺像的，现在840对应的新款是860 PRO，而860 PRO价格要比这个贵一大块，没舍得下手。

麻辣烫

和朋友们一起出去夜宵，其中一个朋友点了一份麻辣烫，价格很贵，味道也很不错，但是感慨更多的是勾起了我许多回忆。

说起麻辣烫，20年前我只知其名，从来没尝试过。那时候的麻辣烫不像现在搞的这个复杂，就是一堆菜盆，有各种青菜、豆腐泡、豆皮、鸭血、粉丝之类的可供选择。

顾客可以选择都放那些菜，服务员用一个小盆选择顾客指定的菜，然后放到一个大笊篱，再放到滚烫的汤锅里涮一会，捞出来，一碗麻辣烫就诞生啦。

原本以为这玩意有啥好吃的，但是后来那时候有几个朋友特别喜欢吃这玩意，于是我跟着尝试了一两次，结果就喜欢上这个味道，一发不可收拾。

那时候街边的麻辣烫3元一份，再配两个发面馒头，就可以当作一顿午餐的，经济实惠又好吃。

后来麻辣烫一直吃了好几年，直到有媒体爆光说里边加了罂粟壳等容易导致上瘾的材料，我决定果断戒掉。

其实就算我不戒掉，街边的麻辣烫小铺也几乎全部消失了，估计一方面是媒体爆光导致顾客稀少，另一方面估计也是没法承受不断上涨的房屋租金和其它材料成本吧。

再次吃麻辣烫，竟然是这样的麻辣烫，虽然看起来高大上，吃起来也很好吃，但是再也吃不出来当年的感觉啦。

垃圾分类？垃圾回收？

前些天社区工作人员上门要求注册垃圾分类小程序，说是上级委派的任务，因为对社区工作人员的服务一直都很满意，所以尽管我十分不情愿，还是帮她们注册了一下。

注册之后，我一直对这个垃圾分类小程序以及背后的团队挺好奇的，该有多大的能量，能让社区工作人员为他们服务啊。

今天在社区旁边，看到了立起来一堆看起来很是高大上的垃圾箱，但是不同于别的地区的干垃圾、湿垃圾啥的分类，直接分的是金属、织物、纸张、塑料、玻璃瓶以及一个有害垃圾。

除了有害垃圾以外，这个看起来更像是一个无人值守的垃圾回收站呢，也就是说互联网化的收破烂的。

好像和垃圾分类关联不是那么大呢，也不知道回收咋算钱的，哈哈，等它正式运行起来的时候，我拎点垃圾去体验一下。

不过感觉这玩意不会成功，大多数人（好奇的除外）估计没心思跑挺远来扔点垃圾。

CD机复活了

十几年前，我花大价钱（大概2500多元）买了一套飞利浦的音响系统，带一个低音炮，外加7个音箱，所谓的7.1声道，支持CD/DVD，有收音机，有AUX接口。

听过几次后，我嫌它用起来太麻烦，另外那么一大堆音箱，感觉占地方，就被我雪藏起来了。

前段时间媳妇要早晚给孩子放些英语音频，拿手机或者IPad直接播放，声音不够响亮，接电脑音箱还麻烦，有两个蓝牙音箱放起来总感觉音质一般。

于是在媳妇的建议下就把这套音响又翻出来支上，手机耳机孔直接接音箱AUX接口，放起来声音圆润浑厚，完美极了。

可是这两天媳妇又给孩子买了些新概念的英语教材，网上却找不到效果比较好的对应音频，要么是带很长的广告，要么是要收费。

可是新概念就带教材对应CD啊，再去收费听其它网站的内容就有点浪费了，可是尝试了一下我飞利浦音响的CD，去发现仓门无法打开了，我估计是因为过了十几年，仓门电机的齿轮/皮带什么的都坏掉啦。

先不说能不能修复，我估计修复肯定费用不菲啦，我自己又懒得拆开看，那要怎么听这些新概念的CD呢？

想到的一种方法是用电脑抓轨，把CD保存成MP3等音频，但是还是感觉太麻烦了。突然想到自己还有一款20年前的便携CD，何不拿出来试试。

把便携CD找出来之后，连上电源，把CD光盘放进去，连上飞利浦音响的AUX，开机，顺畅的播放起来，相当完美。没想到20多年前的CD机还能派上用场，真好。😀
