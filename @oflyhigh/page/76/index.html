币渣今何在？

今天偶然间翻到一个8个月以前的老帖子《奇怪的梦：携18亿美元巨款亡命天涯》，帖子中写的是我创建一个渣滓币曝光网站，进而发家致富的事情，看了一遍后不禁感慨一下：我的币渣哪里去了？


(图源 ：pixabay)

不过现在似乎不需要曝光渣滓币了，因为随手一抓，10个当中至少有9.5个渣滓币。就算原来某个币曾经不那么空气，但是在这又一波的下跌趋势下，估计项目团队也掀不起什么浪花了。

在微信群里看到一个段子，在区块链领域成为百万富翁需要多久？以EOS为例，过去24小时内跌幅超过10%，说是按这个速度缩水，从亿万富翁变成百万富翁只需要43天。

哈哈，庆幸的是，我不是亿万富翁，根本不用担心自己43天后变成百万富翁，但是EOS我买入的最高点是138，现在22元，不过缩水成了不到六分之一。😭

EOS渣吗？我觉得不渣，至少团队还在做事情，BP们也都很努力，DAPP也非常火热，各种菠菜项目都搞得风生水起的。为了充值信仰，我还被一个不熟识的网友拉近了一个传销群，现在每天都信仰满满的。

历数一下我买过的币，好像发展的都不错呢，最惨的也不过是跌掉了98.5%，还剩1.5%呢，距离归零还早，不渣不渣。

想来想去，就算我真的做出了币渣，也没哪个项目可以曝光的，都是寒冬，项目方也不容易，不知道被谁割了，韭菜何苦为难韭菜。不过这样也好，币渣不存在，也没渣滓币可以曝光，那么我就不担心我因为暴富而亡命天涯了。


(图源 ：pixabay)

写下标题的时候，我突然想起了《悟空传》，因为它的作者网名就叫今何在，还记得里边的经典台词：

我要这天，再遮不住我眼，
要这地，再埋不了我心，
要这众生，都明白我意，
要那诸佛，都烟消云散!

可叹的是，天依旧遮着我的眼，地依旧埋着我的心，众生依旧各忙各的为利益没人在乎我的想法，诸佛怎么样我不清楚，不过币圈的诸多虚拟币以及项目就要烟消云散了，连同我那从未问世的币渣。

文章奖励中SBD为什么减少了？

大家可能注意到文章奖励中，SBD部分已经微乎其微了，在不久之前可还是占50%的比重呢！那么为什么SBD奖励会减少呢？


(图源 ：pixabay)

其实这种事情在STEEM的历史上已经发生过很多次了，简单来讲为了维持SBD接近一美元的价格，STEEM系统控制SBD的供应量占总供应量的一定比例。

在HF19时，SBD供应量控制在2%-5%之间，如果低于2%，那么文章奖励的流动性部分就100%发SBD(亦即SBD+SP)，如果超过5%，那么就停发SBD（以及STEEM+SP)，2%-5%之间呢，则按线性比例发放。

在HF20时，这个供应量比例改为5%-10%，规则还是相同的。所以我们知道了SBD的占比，就会知道SBD的发放比例了。

而STEEM系统中用current_median_history_price来计算STEEM的价格，所以SBD的供应量占比可以用如下算式获取：

percent_sbd = (current_sbd_supply/median_price) / (current_supply + current_sbd_supply/median_price )

最终算出来的比例为9.99%，算出来的当前SBD_PRINT_RATE为1%，与读出来的SBD_PRINT_RATE一致。这个就是当前文章奖励中SBD发放的比例了。

买肉

话说自从小猪佩奇一家被灭门之后，就好久没有吃猪肉了。灭门小猪佩奇一家的凶手（非洲猪瘟病毒）一路南下，血洗小猪佩奇的九族啊。

总之，亲戚朋友不时的发微信或者打电话过来关心一下，提示我别吃猪肉了，说得就好像我能吃得起一样。


(图源 ：pexels.com)

不过总不吃肉导致身体因为缺少肉的营养出现了各种问题，最严重的问题是容易出现幻觉，比如说总觉得STEEM要开始涨了，为了避免这种幻觉不时地出现，下定决心去买肉。

佩奇和她亲属的肉我是不想吃的，当然也不敢吃的，尽管砖家们说加热熟了没有任何问题，但是这年头还敢相信砖家吗，原本我也认为加热熟了可以放心吃，可是砖家这么一说，我反倒胆怯了。

那就买点牛肉吧，去超市看一块牛肉还不错，看起来很新鲜，于是和卖牛肉的店员说，就来这块吧。然后售货员放称上一称，告诉我125元。什么？多少钱？

我又问了一次，售货员明确地告诉我125元，并且用一种鄙视的眼光看着我，好像在说：“买不起就别买，没钱还想吃牛肉。”

然后我真的想就这样放弃，反正我已经闻到了肉的气味，估计也会多少有一些营养被我吸收，买与不买差别不是那么大了，不过看看售货员手里挥舞的锋利的大刀，又好是对我说：“不买有你好瞧的！”

哎，纠结了半天，还是买吧，我拿出我的山寨机，点了半天点开支付宝，选择信用卡付款，付了这125元，心都在滴血。花钱倒是很容易，这可怎么还啊，实在不行，只能去工地加班了。


(图源 ：pexels.com)

回去的路上，我小心地拎着我的肉，哦不对，拎着我买的牛肉，心中突然豪情万丈，咱是也吃牛肉的人了，也过上奢侈腐朽的资本主义生活了，不比那些土豪差哪去。

想必吃完肉，再不会出现幻觉了，STEEM会真的大涨吧。至于信用卡咋还，明天再考虑吧！

为什么钱包的钱变多了？ / Why is the "Estimated Account Value " in the wallet incorrect?

这两天最令大家Happy的事情，莫过于大家的STEEM钱包都鼓了！不但鼓了，而且要鼓爆炸了。就拿我的钱包来说吧，一下子有了四万六千亿美元。这在整个币圈都处于寒冬之际，无疑给大家带来很多温暖——哪怕这温暖是假的。


(图源 ：pixabay)

其实，大家都知道，这钱包的估值是没啥意义的，因为我们仅有的STEEM POWER、STEEM、SBD都在那呢，不增不减。这些才是可以变现的资产，估值就是个数值，没有一丁点意义。

不过，热闹之余，想必大家都会关心以下两个问题：

  • 到底是什么导致估值突然多了这么多呢？
  • 这是否会影响我们STEEM账户的财产安全？

估值的计算

为了说明这个问题，我们首先要了解估值是如何计算的，简单来讲，STEEM系统会根据见证人喂价来维持一个3.5日均价/中间价current_median_history_price，系统把这个价格当做STEEM对美元的价格，因为系统中认为SBD锚定1美元，所以这个价格也被系统当初STEEM对SBD的价格。

好了，简单来讲，这个就相当于黄金交易所的黄金价格，那么我们有多少黄金，就可以用这个价格估算出来我们有多少钱了。

STEEMIT钱包里估值算法也一样，用`(STEEM+SP)median_history_price + SBD`*，就是估值了。

为什么出错

通过上述分析，我们得出估值计算公式：
`Estimated Account Value = (STEEM+SP)median_history_price + SBD`*

我们看一下账户，嗯，STEEM没多，SP没多，SBD没多，那个导致估值多的就只能是一个因素了，亦即median_history_price

那我们来看看当前median_history_price是啥样子：

是否影响账户资产安全

通过上边的分析，我们知道这个估值，仅仅是对我们所持有的资产做一下估算，并不对链上数据产生影响，所以我们的STEEM、SP、SBD该是多少还是多少，账户的资产是安全的。

当然，其实我到希望我的账户资产真的多多出几万亿😂

正确的估值

我们知道了问题所在，所以如何计算正确的估值也就很简单了，大致把我们账户显示的估值除以2.8亿，就差不多啦。

比如我的账户是四万六千亿美元，那么实际估值约为：16000美元，😭哭，一下子缩水无数倍。

(其实这个估值还不是很精确，更精确的可以参考我公众号计算的结果。）
