BUG? 在非不可逆块应用get_ops_in_block API 返回结果为空

在我的一个程序中，我尝试用get_ops_in_block获取指定区块的内容，但是我发现一个诡异的事情，如果是比较新的区块，这个API返回空的结果。


(图源 ：pexels.com)

经过我一系列测试，得出结论，只能对不可逆块应用get_ops_in_block，也就是说block编号小于last_irreversible_block_num的块。

一般情况不可逆块比最新块落后约20左右个区块（一分钟左右），如果对最新块或者这期间的20左右个块应用get_ops_in_block，就啥也取不到喽。

测试步骤 &结果

  • 获取最新块(head_block_number)

    {"jsonrpc": "2.0", "method": "condenser_api.get_dynamic_global_properties", "params": [], "id": 1}

  • 调用get_ops_in_block，其中xxxx用最新块(head_block_number)

    {“jsonrpc”: “2.0”, “method”: “condenser_api.get_ops_in_block”, “params”: [xxxx], “id”: 1}

  • 返回结果如下：

    {'id': 1, 'jsonrpc': '2.0', 'result': []}

补充说明

当然，这可能就是这样设计的，不过如果我没记错的话，好像HF19的时候不是这样的，当然也可能是我记错了。

但是如果就是这样设计的，那么应该在文档里加上说明，至少我在文档里是没找到相关信息。
https://developers.steem.io/apidefinitions/#condenser_api.get_ops_in_block

我在github上提交了一个issue，看看官方咋说😀
https://github.com/steemit/steem/issues/3155

今年雾霾特别轻

昨晚出去和朋友散步，突然想到一个问题，以往供暖季来临，雾霾也应该随之而来啊，而今年进入供暖季已经快半个月了，感觉这空气质量还不错嘛。

看了一下手机上的天气预报，其中各项空气质量(粉尘、PM2.5、AQI等)都是良好，虽然我不知道这些条目的具体含义以及标准，但是良好终归是好事情，另外，从我个人感受上来讲，空气清新没有异味，想必不会太差。

而1、2年前，每到供暖季来临，雾霾是常态，空气质量数据爆表那是经常的事情，空气中弥漫一股呛人的气味，出去转一小会回来的时候就会满面尘埃，空气净化器如果开成自动档，那就会几乎24小时不停地在最高负载下工作。

还记得3年前的我一个邻居朋友，在雾霾中散步的时候，突发感慨：“不行，我再也不能忍受这雾霾了，再也不饿能忍受这样的生活了！”于是两年前，她全家成功地移民到国外一个有碧水蓝天的城市。而我，只能感慨：“贫贱不能移！”

不知道大家是否还记得柴静那个《苍穹之下》视频，一度非常火爆，但是没几天就销声匿迹。销声匿迹的背后原因，我们不做猜测，不去探究，我只希望将来随《苍穹之下》消失的还有雾霾，还我们一个碧水蓝天。

不过，我想，很快就可以升级到1.4.3了吧😂

吃吃喝喝最快乐了

昨天半夜一个朋友看到我受到1W点伤害的帖子，和我说别郁闷了，出来喝酒吧，额，已经很晚了好不好，况且我晚上真的吃得很饱。

朋友说，饱咱就少吃点，多喝两瓶溜溜缝。哈哈，看起来无法拒绝的样子，那就出去喝喽。

没点几个菜，但是酒是不能少地，用朋友的话说：“有菜无酒，抬腿就走；有酒无菜，不算慢待！”，哈哈，这磕一套一套的。

经典老雪，哥们的最爱啊。每次喝这个，我总想起一个段子：

听说外边下雪了，下面我给大家科普一下，雪花分好几种 ：雪花有，纯生的，冰纯的，淡爽的，晶尊的，勇闯天涯的，就这几种本人比较中意雪花勇闯天涯。

吃饱喝足，心情超级爽朗，果然吃吃喝喝最快乐了。

受到1W点伤害

今天突然有许久不联系的好友拉我进了一个微信群，我一看除了我以外还有另外一个我们共同的好友，我颇有些感动，一定是这个朋友想我们了，想找我们喝酒。

这也真是的，想我了就和我说呗，想找我出去吃饭喝酒，直接电话就可以啊，我是不会拒绝的，还拉啥群啊，整的怪神秘的。


(图源 ：pexels.com)

正等着他们和说聚会的时间地址呢，结果不到2分钟，拉群那个好友分享过来好几个链接，我一看都是一些分享赚购物红包之类的。

好吧，姑且不论这些分享送红包是真是假，想必我这个朋友也知道这类信息骚扰别人，讨人嫌、招人烦，如果发到一些大群里可能挨骂或者挨踢，不然他不会建个小群分享这些。

如果他弄几个微信马甲，自己建几个群，然后自己分享来分享去，我一定会为他的机智点赞，也许真能薅到羊毛呢！

可是，你一声不响地，拉我进群，然后丢这些垃圾链接进来，当我是什么？垃圾场吗？或者如果你事先和我招呼一下，我也许碍于情面就忍了，毕竟微信上每天收到的垃圾信息很多，也不在乎再多一条。

可是这个朋友并没有事先和我招呼。如果事后他在群里说明一下，表达一下谢意或者歉意，那我也只能默默忍受，毕竟这真不算事。

可是事前并没有招呼，事后并没有解释，我等了好久我才想明白，在他眼里，或者我只配做个垃圾场，偶尔丢些垃圾信息过来无所谓的。另外，想必他不会给顶头上司拉群丢这类垃圾链接。


(图源 ：pexels.com)

可是，我不想做别人的垃圾场，于是我默默地退群，然后在默默地把他拉黑，或者一直以来他并没有把我当朋友，是我自作多情喽。

柚子降价啦

看了一晚上电脑，虽然我关注的主机从7999降到6000左右，并且32G的笔记本内存也降了1000多元，512G的三星970 PRO也降了1000多元，但是想想还是没舍得入手。


(图源 ：pixabay)

逛京东逛淘宝逛得头昏眼花，于是出去逛逛水果超市吧，结果发现柚子降价了，平时几块钱一斤的柚子竟然只要0.99元，对，你没看错，不到一元钱一斤。

果断买了两个普通柚子和一个红心柚子，红心柚子略贵，3.99元一斤，但是也是很便宜的啦，以前贵的时候买，记得要10多元一斤呢。

哎，柚子咋跌成这样了呢？这还是加上仓储、运输、保存、销售等各个环节的成本呢，这让果农可咋活呀。

不过想想网上的柚子，更惨。我记得我从68元开始大幅买入的，一路买到138元，我还记得138元的时候EOS在香港开什么会议，结果会议一结束就暴跌。

原本指68、88、108、138等价位买的柚子能赚点钱，结果现在统统套牢，有心割肉又还舍不得，眼睁睁看着一笔巨款不断缩水，缩水成了鸡肋，这心啊，拔凉拔凉的，想必和柚子果农的心情是一样一样的。


(图源 ：pixabay)

如果柚子(EOS)赚钱，那么我何至于纠结入手一台电脑纠结好几个月，何至于只能买两三个柚子败火，哎。
