Amazon EC2创建实例时需要关联Key Pairs，之后可以用Key Pairs以及默认用户登录。


(图源 ：pexels.com)

但是我在Putty中指定从Amazon下载的PEM文件并尝试登录主机时，却提示我无法使用

Unable to use key file “C:\Users\oflyhigh\Downloads\TestEC2.pem” (OpenSSH SSH-2 private key (old PEM format))

那么怎样才能用Putty登录以及对应的私钥登录EC2的实例呢？原来在登陆之前，需要对私钥进行一下格式转换。

转换私钥格式使用的工具为：puttygen.exe，可以在Putty网站下载。最新版本的下载地址可以在此处找到：
https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html

操作步骤如下：

打开puttygen.exe，点击Load按钮：

选择对应的PEM文件，puttygen.exe会弹出如下提示：

关掉提示，在箭头指向处填入密码，并在下方填入确认密码

选择Save the private key保存私钥

保存的私钥格式(扩展名为).ppk，就可以在Putty中使用啦
