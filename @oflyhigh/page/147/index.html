自己动手，清洗地热

据说暖气主管道漏水已经解决，现在已经开始带热水运行了，既然如此我也应该把我的地热供暖都打开喽，但是经过快一年的闲置，地热管道里肯定一堆脏水和铁锈之类的，为了让它运行良好，应该清洗一下。


(图源 ：pexels.com)

一般洗地热都是找专门的人员用专业的设备处理，当然了价格也是不菲，为了省钱，我决定自己简单操作一下。

详细步骤如下 ，如果你家也地热供暖的话，不妨一试哦。


先打开供暖供水阀。


关闭供暖回水阀。(顺便说一下，我的泵已经坏掉不转了）


管道阀门，每个阀门对应一个房间或者一块区域。先关闭所有的阀门，只开一组。


通过放水阀门排水。


管道里的脏水，看看脏成什么样子了！


大概放出4-5盆水以后，水变得干净一些了。

继续上述步骤（单独开一个区域阀门，其它关闭），直到所有区域都清洗干净。然后打开所有的区域阀门，打开供暖回水阀，搞定。

是不是很简单啊？当然了，就是太累了，我至少放出去几十盆水，累死我啦。所有有条件的还是找专业人士花钱弄吧。

每天进步一点点：PHP输出整段文本

话说，尽管坚信PHP是世界上最好的语言，但是真的好多年没怎么用PHP了。好多年前，一个上市公司出品的几款安全软件的后台、用户管理以及在线处理付款生成注册码之类的脚本全是我用PHP做的，现在PHP如果不查手册，我只会写Hello World!了。


(图源 ：pixabay)

当年我还熟练地使用Smarty以及PHPAJAX之类的东西，现在查了一下，我擦，这都是啥玩意啊，还有新出的流行模板库，看了几个感觉一头雾水。哎，除了拿一些往事吹吹牛，PHP我真的已经变成菜鸟了。

话说不想用WordPress等CMS，也不想用各种模板库(我不会用)，那么PHP和HTML混排，一大堆文本该咋输出啊？经过我努力学习，大概有以下几种方式。

HTML和PHP混排一

<?
// PHP代码
echo "Hello World!";
?>
在此处写HTML等文本代码
随便写啊，随便写
<?
// 继续PHP代码
echo "Hello World!";
?>

HTML和PHP混排二

<?
// PHP代码
echo "Hello World!";
$name="oflyhigh";
?>
在此处写HTML等文本代码
<p>你好<? echo $name ?>，吃饭了吗？</p>
随便写啊，随便写
<?
// 继续PHP代码
echo "Hello World!";
?>

使用双引号、单引号

<?
// PHP代码
echo "Hello World!";
$name="oflyhigh";
echo("在此处写HTML等文本代码
<p>你好$name，吃饭了吗？</p>
随便写啊，随便写");
// 继续PHP代码
echo "Hello World!";
?>

echo函数中字符串用双引号包围，函数中的变量会被解析，比如上述代码中$name会替换成oflyhigh；如果是单引号，变量被当成正常的字符串处理。

如果使用双引号或者单引号包围字符串而字符串中含有和包围字符相同的字符，则需要使用\转义。

Heredoc 以及 Nowdoc

Heredoc 以及 Nowdoc 和双引号以及单引号包围的字符串类似。但是因为它本身使用的是另外一种写法，所以不会和内容部分中的引号或者单引号冲突，所以可以让内容保持原样，无需转义。

<?
// PHP代码
echo "Hello World!";
$name="oflyhigh";
echo <<<"EOD"
在此处写HTML等文本代码
<p>你好$name，吃饭了吗？</p>
随便写啊，随便写，什么"以及'都可以
EOD;
// 继续PHP代码
echo "Hello World!";
?>

其中EOD啥的叫标识符，命名规则啥的和变量名一样，需要注意的是开始的标识符后边以及结束的标识符前边同一行内不能有任何内容。据说，PHP7.3 以后会更灵活，可惜我服务器PHP没升级。

不多举例啦，举多了怕出错，读者可以自己尝试😀，更多详情，看参考链接吧。

参考链接

  • http://php.net/manual/en/language.types.string.php

发大水了

下午的时候去厨房，突然感觉哪里不对劲呢，然后一下子就反应过来了，原来地上全是水！


(图源 ：pixabay)

哪里来的水呢？我打开橱柜门，发现水槽的部位并没有漏水，然后仔细观察地下的水，发现竟然可以看到流水，好吧，我顺着水流去找源头，发现在冰箱后边。

冰箱后边的小铁门，打开后是自来水的总阀以及入户阀，我看暖气入户阀并没有什么问题，而且水是从自来水的管子上流下来的，于是怀疑是自来水管的问题，赶紧呼叫中控室报修。

结果中控室回复说是暖气的主管道漏水，已经有人报修过了，供暖公司已经在处理了，但是由于总阀门在车位的地方，而且车位还被占了，并且车主要一个多小时才能回来，暂时没法处理。我晕啊，再没法处理我就要水漫金山了！

去楼道里看了一眼，漏点在三楼中间的位置，供暖的门已经被打开，我可以看到主管道有铅笔芯那么粗的水流喷喷射而出，于是我又给供暖公司打电话，说你们得采取点措施啊，不然一会我那就彻底变成水帘洞了。

供暖公司回复说，他们会马上采取措施，把漏点的部分墙壁凿开，然后把水往外排，至于彻底停水，只能等压住供暖井的车主回来再说了。

到了晚上，我那水还哗哗淌着，并且已经从棚顶以及墙壁处开始冒水了，我又上楼去看，结果发现两个师傅正在处理，并且告诉我楼上已经处理完毕了，现在去关供暖罚，那个车主已经回来了。

我跟过去去看，看着师傅钻进井里，然后开始关闭阀门以及放掉整个单元里的暖气存水。

水哗哗哗的放了足足二十多分钟，才感觉力度小一些。看着那比碗口还粗的水流淌了这么久，我就想，如果这些水都淌进我屋子里，一定会装满吧。😭

再回到屋子里，发现漏水的力度小多了，估计是之前楼上楼道里的水，慢慢渗漏下来的吧，总算没有彻底被泡的风险了。

师傅说明天白天会彻底修复，我的要求不高，只要别把我泡上就可以啦。
(封面图我从网上找的，我这还惨到这种程度😭）

你会出售你的QQ号吗？

昨天接到一个电话，我以为是客户或者朋友打来，问他有什么事情，结果他问我： “QQ号卖吗？可以见面交易。” 我回问出多少钱？他回答说8000元，我果断地挂了电话。


(图源: https://bing.com)

自从有了这QQ号之后，来询价的信息就一直没有断过，尤其是这些年，愈加频繁。有通过QQ联系的、有通过微信联系的、有通过邮件联系的、还有通过电话联系的；出价则从2000-3000涨到5000-6000，再涨到现在7000-8000。

其实有了微信以后，QQ用的真的不多了，前阶段我还曾在STEEMIT上吐槽过我好些个7位QQ号被腾讯冻结，申诉啥的通通被驳回，对QQ我一点也无爱了，我的私人QQ号已经好多天没有登陆了。

但是这个QQ号不同，最初是我用来工作的，里边有成百上千的客户QQ，尽管好多客户现在已经没有合作关系了，但是万一骗子买了我这个QQ号之后拿去行骗怎么办？想想就不寒而栗。

另外一点，QQ钱包之类的我还绑定了银行卡，之前还开过财付通，如果没记错的话，好像还在拍拍开过网店。还有也用这个QQ邮箱注册过一些垃圾网站之类的，这些信息，也不放心交给别人啊。

好吧，就算别人买去不拿来行骗（事实上这种概率不大），也不关心的各种私人信息（我都可以解绑删除等），但是这个QQ我可是砸进去了好多钱在里边啊，什么QQ会员、超级会员我可是一直买着，并且曾一度为了成长值排名之类的一下子买了好多年，多了不说三五千大洋肯定有了。

我养了这么多年，昂贵的QQ，你竟然想七八千块钱就买下来？开哪门子国际玩笑啊。嗯，如果有人肯出10W人民币，我或许真的会慎重地考虑一下。这还是我QQ用的不多并且对QQ无爱后的价格呢，要是前三五年，20W以下免谈。


(图源 ：pixabay)

那么你会出售你的QQ号吗？多少钱你会考虑卖掉呢？😀
