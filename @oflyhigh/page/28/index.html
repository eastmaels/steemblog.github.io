携号转网 & 移动的套路

虽然微信群以及朋友圈里很少讨论，但是最近新闻媒体炒作的一个热点就是携号转网，因为从11月10日起，移动、联通、电信三大运营商携号转网服务已经正式进入试运行阶段。


(图源 ：pixabay)

炒作了无数年的携号转网，终于可以操作了，但是其实身边的朋友们反应平平，因为该换号/换网的早就换了，不换的估计要么是懒惰，要么是对原来的运营商还是有些认同感的。

其实我也想不出对我而言携号转网有啥必要性，多个网络运营商终归多一些保障，经常有到哪个地方联通信号不好而移动信号好或者反过来的情况，如果只用一个运营商那么遇到有事情接打电话的时候就会很尴尬。

尽管我本人并不需要携号转网，但是对大众而言，多个选择，多些自由，终归是好事，不过话说携号转网真的很容易操作嘛？

好奇心的驱使下，两天前，我就做了一下测试，首先发送如下短信到10086

CXXZ#用户名#证件号码

当然了，要把用户名和证件号码换成自己的信息，其中CXXZ，据我分析应该是查询携转的缩写。

结果很快就收到10086的回复，果不其然，转网，想多了：

回复了Y之后，移动给出了原因：

擦，我的信息咋还不符了呢？还有网龄回馈流量扩容又是什么鬼？往前翻记录，大概想起来是怎么回事了， 大概8、9月份大家都在传发送1999给10086，就能获得免费流量，于是我也试了一下：

看来移动早早的就在给我们布套，然后我就傻傻的钻了进去。

当然了，估计联通、电信也可能有这样的政策/对策，不过为什么不从根源上解决问题呢，如果你网络稳定、服务好、价格便宜，那么还会担心用户携号跑路吗？

你要携号转网吗？你被套路了吗？

纸上得来终觉浅：lxd初体验

还记得我写过的lxc初体验嘛？在那篇帖子中，我曾经说过lxd是lxc的进化版，这篇文章中我们来测试一下lxd。


(图源 ：pixabay)

安装

如果还没有安装lxd，那么我们需要如下步骤来安装lxd并且让当前用户可以直接使用lxd的相关命令。

安装lxd

sudo apt install lxd

登出并重新登录当前用户，以刷新组成员信息，或者使用如下指令：

newgrp lxd

在一台新的VPS上，我进行这样的操作后，也无法直接只用诸如lxc list等命令，出现下列错误：


Error: Get http://unix.socket/1.0: dial unix /var/lib/lxd/unix.socket: connect: permission denied

只好手工把用户添加到组lxd中：

sudo usermod -a -G lxd $(whoami)

可以使用groups命令来检查关联组别是否成功。

lxd初始化

在正式开始之前，我们还需要执行lxd init来进行一些初始化设置（网络、存储等等）。

lxd init

我们都选择默认设置即可（空间我设置的5G，因为我磁盘空间比较紧张)：

查询/创建/使用容器

查询

在进行完上述设置后，我们可以使用如下命令查询我们机器上的lxd容器：

lxc list

可以看到我们现在一个容器都没有呢：

注：lxc并非之前文章中的lxc，本文中代表的是lxd的客户端。

创建

下面来创建我们的第一个lxd容器：

lxc launch ubuntu:18.04 first

创建后，容器会自动启动：

再来查询一下：

lxc list

耶，有了一个(我有些后悔，初始化的时候应该关闭ipv6，看着闹心）：

使用

我们可以使用如下命令进入到对应的容器中：

lxc exec first -- /bin/bash

或者执行命令，拉取文件，推送文件等：

lxc exec first -- apt-get update
lxc file pull first/etc/hosts .
lxc file push hosts first/tmp/

至于关闭和开启可以用如下命令：

lxc stop first
lxc start first

不需要了，可以用以下命令删除：

lxc delete first

换掉了一张用了20年之久的储蓄卡

好多好多年前，储蓄卡还没有流行的时候，人们出门带钱都是用大皮包或者是蛇皮袋，既费劲又不安全。我还记得一个朋友95年带几十万去外地，买了一双女用过膝袜子，把钱一沓一沓的塞进去，然后缠在腰上。


(图源 ：pixabay)

当然彼时已经有储蓄卡这么一说了，不过大家还是觉得现金更靠谱、更安全，把所有资金放到一张卡片上，总觉得不是那么放心。

而我第一次接触储蓄卡大概是在98年，那时候看到别人都在用提款机，觉得这个东西简直太神奇了，卡片插进去，随便按几下，钱就出来了，于是去建行办了生平第一张储蓄卡——建行龙卡。

有了建行龙卡之后觉得简直是太方便了，再也不用担心现金被盗，需要现金的时候可以随时从提款机上提取——那时候提款机的最小面额还是10元，而去超市购物什么的就更爽了，不用现金就感觉不用花钱一样。

那时候周围的朋友人手一张建行龙卡，你还在用现金？还没有去办龙卡？你太OUT啦，赶紧去办吧。后来又办了一些中国银行之类的卡，但是因为周围建行网点最多，其它的卡就都闲置了。

再后来公司发工资强制要求用招行的一卡通，并且平时网购主要用招行的双币信用卡，所以用到建行卡的机会越来越少了，几乎闲置了好多年。

今天去建行办点事情，顺便想看看我的建行里还剩多少钱，结果卡片插进机器后，根本没法识别，也就无从查询了，客服小妹告诉我换个芯片卡吧，只需10元工本费。

于是在小妹的指导下，在自助柜员机上，用了接近1个小时的时间总算完成了更新资料、换卡等操作，别问我为啥用这么长时间，就是这么繁琐。

换完卡之后，拿着退出来的老卡，小妹告诉我老卡没用了，回家拿剪刀自己减掉就好了，竟然突然有一种怅然若失的感觉。


(图源 ：pixabay)

用了20年之久的卡，就这么被我说换就换了，怎么感觉自己是个喜新厌旧的人呢？/(ㄒoㄒ)/~~

双十一剁手大赛开启，你剁了吗？

又到一年一度的双十一购物节了，话说不知从何时起，双十一居然成了一个节日，尤其是在剁手党眼中，这个节日甚至比国庆、元旦、春节还要重要。


(图源 ：pixabay)

现在已经搞不清楚是京东还是淘宝最先弄的这个噱头，不过现在不管哪家都不愿放弃这个机会了，毕竟一年之中肯让用户们心甘情愿地出血的机会并不多。

双十一原本的意思是光棍节，是单身狗们用来自嘲的日子，不过电商们从这个视角出发，光棍没人爱，自己爱自己，没有男/女朋友给买东西那么就自己买，不需要给男/女朋友那就买给自己。

不过现在已经演变成不分男女老幼，不限年龄段，不限婚否的一场全民狂欢。尽管真相可能只是顾客的狂买、商家的狂欢。

早晨出门路上听到两个白发苍苍的老人也在讨论双十一，
其中的老奶奶说：“一会回家上网看看能买点啥，今天双十一东西便宜。”
老爷爷感慨到：“双十一搞多久了，有十年了吗？”
老奶奶说：“有没有十年我不清楚，不过我估计我活不到下个十年了。”
老爷爷说：“别瞎说，我们争取一起再活个十年。”

看着这对恩爱的老人，我心底不由地祝福他们再一起活十年以上，不过想到十年后，两个老人颤颤巍巍在电脑屏幕前血拼，也是醉了。

其实双十一的战火早在好多天前就已经开启了，这些天没少被朋友们拉去弄什么喵币，盖什么楼，我根本不懂这玩意咋玩，估计朋友们弄明白的也没几个，不过这并不妨碍他们把楼盖到几十层高。

京东也有类似活动，总之套路无外乎是拉人头，好好的两大电商，弄这些乱七八糟的东西，越来越像拼夕夕了。


(图源 ：pixabay)

说了这么多，双十一还是忍不住剁了几千大洋的东西，毕竟众人皆醉那么我独醒也很孤独，那么就一起剁手狂欢吧，为马总首富的地位，为东哥的XX开销爱情，贡献出自己的微薄之力！

烤串、喝酒 & 专车

自从天气变凉以后，我懒得出屋，朋友那边也懒得组织烤串之类的活动，所以算起来我们好些天没有一起撸串喝酒了。

昨晚晚餐吃的有点多，吃完饭打算随便转转促进一下消化，于是就转到朋友那里。朋友在那无聊呢，喊我一起出去吃点啥，可是我明明已经撑到了啊。

朋友说没关系，就是随便吃一点，然后喝喝酒聊聊天嘛。可是好像就我们两个喝不够热闹，于是群里又喊来几个人，一行六人，从两个方向居中会和。

最终选择了一个都没有试过的白玉串城，点了一些肉串、蚕蛹、鸡架、鸡头、鸡爪之类的食物，叫十几瓶淡爽啤酒，开喝。

这个串店看起来挺大，菜品的味道真的还很一般，不过也有可能是我们每个人都已经吃完晚饭的缘故。毕竟吃饱了并且有些撑到的情况下，吃什么也不会觉得香啦。

菜品一般，不过价钱还真的不便宜呢，这么一点点东西就花费我们300多。

回去的时候，一组开车回去，我们这一组原本计划打车回去，结果朋友说他在XX专车里有好多钱，于是叫了个别克商务专车回来。

到家后朋友发账单过来，正常的叫出租车10元就够啦，这个专车花掉朋友53.21元，这是什么专车，专门宰人的车吧？哈哈。

不过尽管串口味一般、专车也挺贵，但是和朋友们聚聚，喝喝酒聊聊天还是很开心的。
