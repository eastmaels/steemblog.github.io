解决安装Ubuntu 18.04时花屏的问题

在之前的一个帖子中提到使用技嘉主板的台机安装Ubuntu 18.04花屏，向技嘉技术支持求救后告诉我自己上网去查找解决方案，无奈只好自己查找了。


(图源 ：pixabay)

上网一查，还真别说，有不少和我一样遇到类似问题的，花屏啊黑屏啊，总之就是无法正常安装或者进入系统。但是这事好像也不能全赖主板，貌似和显卡(我显卡也是技嘉的)也有关系，总之就是和硬件不兼容是啦。

在经过一番努力探究和测试后，终于成功地把Ubuntu 18.04安装到我的台式机上，为了避免以后重装时懵圈，以及给遇到类似问题的朋友一些参考，现将安装过程记录如下。

安装时

如果我们按着正常的方式安装系统，那么会在开始安装后花屏，然后就没法继续了

解决方法，在选择系统菜单出来的时候，选中Intsall Ubuntu后不直接回车进入安装，而是按e进入grub编辑界面：

在linux选项下把后边的---去掉，并加上nomodeset参数，然后按Ctrl+X保存并退出，进入安装程序，就可以正常安装了。

启动时

安装完成后，再次进入系统，发现依旧花屏，这时候还需要编辑一下grub的参数。

重启系统，按住Shift不放，直到出来Grub启动界面，还是e进入grub编辑界面：

在linux选项下把后边的---去掉，并加上nomodeset参数（图片中后边的_是光标字符，请不要添加）

然后按Ctrl+X保存并退出，系统就会正常启动啦。

配置文件

经过上述修改我们以及可以安装并进入系统了，但是每次进入系统都要进行第二步中的修改，所以我们需要一个一劳永逸的方法。

进入系统后，执行如下命令：

sudo vi /etc/default/grub

把下边这行内容：
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"

修改为：
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash nomodeset"

按ESC后输入:wq后退出。

再执行如下指令：
sudo update-grub

经过这样三步操作后，我们就可以正常的进入系统啦。

关于nomodeset

尽管我已经解决了安装和使用的问题，但是还是好奇nomodeset到底是起什么作用呢？在Ubuntu 论坛的一个帖子上我找到如下内容：

nomodeset
The newest kernels have moved the video mode setting into the kernel. So all the programming of the hardware specific clock rates and registers on the video card happen in the kernel rather than in the X driver when the X server starts.. This makes it possible to have high resolution nice looking splash (boot) screens and flicker free transitions from boot splash to login screen. Unfortunately, on some cards this doesnt work properly and you end up with a black screen. Adding the nomodeset parameter instructs the kernel to not load video drivers and use BIOS modes instead until X is loaded.

Note that this option is sometimes needed for nVidia cards when using the default “nouveau” drivers. Installing proprietary nvidia drivers usually makes this option no longer necessary, so it may not be needed to make this option permanent, just for one boot until you installed the nvidia drivers.

简单来讲就是一些显卡和kernel不兼容啦，添加nomodeset让显卡先不加载视频驱动。

写到这里，好像似乎冤枉了技嘉主板（是技嘉显卡的锅？），不过毕竟客服告诉我不提供安装Linux的技术支持并且是我自己找资料搞定了Linux的安装，也不算冤枉他们了。😳

银行卡里突然多出一笔来路不明的“巨款”

如果你的账户突然多了一笔巨款，你是会兴奋还是惶恐?


(图源 ：pexels.com)

我觉得没有发生这样的事情之前，大多数人包括我在内都会那将是一件令人兴奋的事？。

然而当这件事情真实的发生在自己身上时，我却觉得万分惶恐，当然了，发生在我身上的不是巨款，只是一小笔款项而已。然而这事情本身，却和多出一笔巨款无异。

话说拿工行手机APP查询账户交易明细，突然发现一笔很奇怪的入账，交易场所显示的是批量业务，业务摘要显示的是份额调整，对方户名显示的是我的账户。

这是什么鬼？虽然入账金额才39.60，但是不明不白的收到这39.60，我寝食难安啊，况且这个批量业务能给我加钱，是不是也意味着能扣我钱？

去百度了一下工商银行的批量业务，找到如下内容：

批量业务指的是由工行系统大批量集中统一处理的业务，如代发工资、贷款扣款等。温馨提示：您可登录手机银行，选择“最爱-我的账户”，点击对应账户下方的“查询明细”功能查询明细，点击单笔明细后，还可以显示详细交易信息。

可是我这笔交易从明显中根本看不出来是什么情况啊。于是乎打工行的客服电话：95588 咨询，可是客服帮我了查了一下后，告诉我她也查不到更具体的信息，问我会不会是养老金，让我去柜台问问。

这就尴尬了😅，到底是哪里冒出来的这39.60呢？我百思不得其解，难道真的要为了这39.60跑一趟银行？真让人纠结啊。

哎，还是看看手机放松一下吧，嗯，好久没翻短信了，看看有没有啥新消息，咦，这是什么？

尊敬的客户，您尾号xxxx卡下持有的人民币账户国际原油（连续产品）在10月25日进行了份额调整，调整前您持有20桶，调整前价格是435.95元/桶，调整后您持有20桶，调整后价格是433.97元/桶。本次调整过程剩余39.60元不足以开仓最小交易数量，已退还至资金账户，详细调整情况您可以登录网上银行或手机银行查询。【工商银行】

擦，总算找到罪魁祸首了，原来前些天我试着买了20桶人民币账户国际原油（连续产品）玩，本以为买了就等涨等跌好了，哪曾会想到还有这么个份额调整功能呢，虽然我现在也没搞明白份额调整是啥情况。


(图源 ：pexels.com)

不过总算知道了这笔巨款的来龙去脉，不必胆颤心惊了。

秋的味道

朋友圈里看朋友发了几张照片，并配文：秋的味道。不禁想，秋天应该是什么味道呢？是落叶萧瑟的苦涩，还是天高云淡的清爽，又或者是硕果累累的甘甜呢？

于是我也随手拍一些照片，试图找寻一下秋天的味道。

我寻寻觅觅也没找到，直到看到这一堆晾晒的秋菜（大白菜和大葱），我才惊觉，这不就是我苦苦追寻的答案吗？

于是我在朋友的朋友圈里回复，秋的味道就是大白菜和大葱的味道，然后朋友上来谴责我，好好的气氛被我破坏殆尽。😅

惊：家用主板不支持Linux？

被新机器的硬件折磨了快两天，这还不算选购配件、验货、发现主板坏掉、更换主板等过程，前前后后算起来已经7、8天了。不过装硬件我不熟到也罢了，如果说我被装Linux系统折磨了大半天+大半晚你会信吗？

话说，弄好了硬件，我就兴匆匆的装系统，我打算在上边装个Ubuntu 18.04玩，因为之前在其它电脑上安装过，手头有现成的USB安装盘，所以直接插到电脑上，打算一路顺利的安装下去。

可以进了Ubuntu安装界面并选择安装后，屏幕闪过一些字符后，上半部分还算正常，下半部分花屏，并且好像进度显示只在屏幕下半部分进行，我根本没法进行任何操作。

因为是初装的硬件嘛，我果断怀疑是显卡问题（虽然进BIOS时屏幕正常），我把显卡从第三个插槽换到第一个插槽，故障依旧了。

然后发现主板上有个故障指示数码管，显示的故障码是A6，联系商家以及技术支持客服，问了大半天后才得出答案，故障码仅在主板无法点亮或者无法进入BIOS时有参考价值，其它的时候不用管。

为了判断到底是不是显卡故障，我又找到了一个Windows 10的安装盘，测试一下，正常安装，过程没有一点花屏/乱码之类的。

这就尴尬了啊，按说装Linux应该比装Windows更容易更顺畅才对，怎么可能反过来了呢？到底是哪里出了问题？难道是主板不支持Linux系统（貌似这个想法有点搞笑）

不过我还是尝试打了一下客服电话，结果客服和我说这款主板是家用主板不支持Linux系统，仅支持Windows。我果断反问：“你是在开玩笑吗？我家用就不许用Linux啦？”

客户沉默几秒后换了个说法：“这款主板不提供安装Linux技术支持，你要是想装Linux，自己上网找解决方案吧。”

我擦，对这个答案我是无语了，难道真的只能装Windows来玩纸牌、扫雷、泡泡龙吗？不，我不甘心，我一定要装上Linux系统。

于是经过我在百度上刻苦Google，又一遍一遍尝试，终于找到了办法。限于篇幅，下文继续分说吧。

DIY装机好累

今天商家补发的主板终于到货了，我终于可以开始装机了。原本打算去京东买一个DIY装机服务，大概150元左右，后来看看暴跌的数字货币以及股票，哎，还是自己装吧。


先来个合影， CPU、内存、SSD、显卡在右上角呢


拆开我的大机箱，哎，费老大劲了。


把机箱原来的风扇该到顶部，前边换了一组三个一体的风扇。


安装电源，现在的电源好贵啊，这个要1千多大洋。


主板安装好，我找了半天才找到固定主板的螺丝，原来主板不带这个螺丝，要机箱带。


最金贵的部件，CPU，安装的时候小心翼翼的，就怕弄坏。


8条内存统统插上，聊QQ再也不卡啦。


安装SSD，据说这个位置是和CPU直连的，装了两条SSD 512G+2048G。


安装风扇。


风扇完工 & 主板供电。


连接机箱的各种跳线。


安装显卡。


主板点亮了。


重新整理一下连线。


最终效果。

写起来很简单，实际上弄这个至少折腾了我两整天，因为DIY装机我真的一点不熟悉，一边对着各种部件的手册，一边小心翼翼地安装，生怕一不小心主板冒烟。

好在最终主板冒烟或者CPU被烧的情况没有发生，不过累得我腰酸背痛的。早知道这玩意这么麻烦，说什么也要花150元找京东的装机服务来搞定了，哎！

被迫办了ETC

说到ETC，大家可能会想到以太坊经典(Ethereum Classic)，不过此ETC非彼ETC啦，我指的是电子不停车收费系统(Electronic Toll Collection)，也就是上下高速无需人工抬杆、收费，直接通过并从卡里扣费。


(图源 ：https://cn.bing.com/images/)

按说这么先进的东西应该及时办理啊，毕竟出入高速排队等发卡或者缴费还是比较麻烦的，况且据说装了ETC卡，走三环(绕城高速)是免收过路费的。

另外，各大银行都狂推ETC，办理ETC送大米、送豆油、送油卡等活动层出不穷，只要去办，各种福利多多。

不过我这个人呢，很少跑高速的，甚至绕城高速也很少上，偶尔跑趟高速，都是等节假日高速免费的时候上，这时候走普通车道比ETC车道还要通畅呢。

还有就是我比较懒，总觉得多一事不如少一事，就算高速不免费时偶尔跑跑不过是多花绕城高速这段几块钱，无所谓了，倒是整什么ETC卡啊、设备啊怪麻烦的。

所以好多朋友找我让我办ETC卡，我都推脱了，一句我暂时不想办，朋友们就很知趣的不提这个了。

不过前两天，一个亲戚打电话过来，让我办ETC，我还是一句不想办，可是亲戚说是他们单位强制要求的任务，不完成不行，所以必须让我办。

可是你单位强制的任务与我有什么关系呢？又不是我的单位（咦，我的单位哪去了）。其实就算是我的单位，如果领导安排这样的任务，我也不理他，如果惹急了，我直接怼他。

于是我再次表达了不想办这个ETC卡，并和他说，很多朋友找我办这个，都被我拒绝了。可是亲戚依旧不放弃，和我说让我给他一个面子，把这个办了吧。

这就尴尬了，如果被迫办了ETC，我不爽；亲戚说到这份上，如果我拒绝了，这样也不太好；无奈只好同意让他办理，可是心情极度不爽啊。


(图源 ：https://cn.bing.com/images/)

为什么，总要发生些让人不爽的事情呢？哎😔
