吃点喝点败败火

给这俩菜来个特写。


给盘子再来个特写。

菜都上来就开吃开喝了，就没时间拍照啦😂。朋友以前没少在他的门店组织户外烧烤聚会啥的，不过今年创城不让在外边烧烤了，那样吃的喝的才叫过瘾呢。

坑人的Microsoft Access Driver

之前用MFC和ACCESS写了一个抓取和查询自己STEEM上文章的工具，这样就可以把STEEM当成自己大脑的备份了，比大脑靠谱的是，STEEM的数据不会遗忘。


(图源 ：pexels.com)

这工具一直用着挺方便的，虽然 无论是用MFC还是ACCESS都显得很傻，但是我一直以来的原则都是能用就行，至于一直计划把数据库换成SQLite，因为懒惰一直没行动。

前些天换了台电脑，然后就想着把这个工具也拿过来用，以往在其它电脑上也试过，装个Microsoft Access Driver就行了，因为我的程序编译成64位版本，所以也要装64位版本的Microsoft Access Driver。

我下载了一个Microsoft Access Database Engine 2016 Redistributable，选的64位的，结果安装时提示我Office是32位的，没法安装，晕啊，话说你管我Office用的啥版本啊，另一台机器我没装Office也正常用啊。

不过抗议是没用的，于是我把Office重装为X64版本，结果驱动倒是装上了，我的程序一查询就死机，没有道理啊，毕竟之前用得好好的。

想来想去只能是Office版本或者Microsoft Access Database Engine 2016 Redistributable版本的问题了，Office懒得降级，于是把Driver换成Microsoft Access Database Engine 2010 Redistributable。

程序总算可以用了，查询啥的都没问题，可是一更新文章内容，程序就死机，不过这我就判断不出来是我程序的问题还是API节点的问题又或者是Access Driver的问题了。

等以后不懒的时候调试一下吧，先这么用着喽。

去农商银行享受VIP服务

用农商银行收了朋友一笔钱，因为提现有每日限额并且很麻烦，她建议我开个手机网银转出来。


(农商行Logo)

以前我一直不知道农商银行是啥银行，虽然家附近就有一个大网点，后来办卡的时候才知道才知道原来农商行的前身就是农村信用联合社。

可是开卡的时候我手欠把开手机网银的选项挑掉了，这次则需要跑一次网点开个手机银行了。去了家附近的网点，发现竟然是铁将军把门，只好又用导航找了另外一个网点，开车前往了。

到了一看，哇，没几个人办业务，我拿到号后，马上就排到了，简直就是别的银行VIP级别的服务嘛。

窗口服务人员说手机APP也有限额，除非办个蓝牙盾，不过蓝牙盾是免费的，我说那就办一个呗，于是过了一会告诉我手机网银开好了，蓝牙盾也办好了。

既然都办好了，我就试试呗，结果死活连不上蓝牙盾，窗口帮我按了半天也没整明白，后来又他们又打电话问上级技术部门，总算解决了。

服务人员非常愧疚，说网点第一次办这个业务，他们也不知道咋弄，给我添麻烦了。弄得我怪不好意思的，明明是我给人家添麻烦了。

和人民币70周年纪念币失之交臂

之前预约了人民币70周年纪念币，约的日期是1月15日领取。昨天有好多事情忙，就把领取纪念币这事忘得一干二净，看别人秀了这个纪念币我才想起来。

上午抽时间匆忙赶到银行，却被告知昨天是最后一天，兑换已经结束了，这个不甘心啊。

于是找到省行的领导让她帮忙问问，结果得到答复是人民银行已经关闭了兑换接口，上缴了余钞，没法从网点兑换了。

她说要不回头她给我两张收藏，感觉没啥意思就没要。毕竟我原本申请了40张，准备过春节的时候给亲戚朋友发红包的，看着红色挺喜庆的。

错过就错过吧，我已经攒了好多各种纪念币了，有点养不起了——自己这样安慰自己。
