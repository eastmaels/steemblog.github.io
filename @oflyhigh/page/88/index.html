编译EOS v1.8.1

在EOSIO v1.8.0出来不久，BP们还没完成replay以及硬分叉（共识升级）的时候，EOS v1.8.1就出来了。

这可把BP们吓了一大跳，不过好在v1.8.1和 v1.8.0只是常规升级，从v1.8.0升级到v1.8.1无需replay，倒是虚惊一场。

There should be no need to do a replay or import from a snapshot.

话说得益于steem记录功能，我可以找到我上次编译和升级EOS本地节点的时间是19天前，然而不幸的是，我的本地节点还在replay当中，也就是说原来目测的需要两天以上，实际上至少20天以上😳。

先把v1.8.1编译一下，有了之前编译v1.8.0的经验，这次编译依旧挺顺利的，只是由于用了-P参数，这个编译的时间还是有点久，哈哈。

编译完看一下版本号：

nodeos --version

看起来没编译错。

只是我虽然编译出来了，可是replay还没完成：

估计等我完成replay，新版本也要出来了吧/(ㄒoㄒ)/~~

另外，据社区消息，EOS的共识升级(硬分叉)，定于9月23日，之后就可以在主网上部署Voice啦，到时候会不会拉一波呢？

至于现在的价格，一个字：全是眼泪啊。哎，我这哭得说都不会话了。

消费降级 / Consumption downgrading

最近外出带笔记本出去，发现笔记本电池只能用20分钟了，也就是说遇到需要用笔记本工作的情况，如果没有外接电源，那么根本满足不了需求。

想到这个笔记本(X230)已经服役近7年了，尽管除了电池问题，其它一切功能良好，也能满足我聊QQ之类的需求，但是也是时候换新了。

于是最近这段时间一直在看笔记本，打算价格合适的时候换一个，什么X1 2019啊，或者X390啊，看起来都不错的样子，实在不行T490貌似也不错。

可是本子看起来都不错，但是看了一下钱包，嗯，有点瘪啊，哎，花一大笔钱买个笔记本，好像有点心疼。

思来想去，我的X230差的无外乎电池续航时间，那还是从解决续航时间着手好了，说起来这个问题也是最好解决的，换个电池呗。

京东搜索了一下，貌似绿巨能的电池评价尚可，果断下单购入一块，就等到货啦。

原本1W+的预算，只用了不到300元搞定，看来我的小本可以再战500年啊。

不过如果STEEM 回到50元一枚，我除了要换个能拍月亮的手机，一定要换个新本，嗯，就这么愉快地决定了。

毕竟旧本已经一大堆了。
