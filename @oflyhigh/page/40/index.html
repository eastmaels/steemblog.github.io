失败的酒局

朋友打电话告诉我，他买了许多老杨家熟食，让我一起去喝一会，不过考虑到晚上有很多事情要处理，加上戒酒，就果断拒绝。


(图源 ：pixabay)

结果朋友再三劝说，哎，我这人心软，于是就是收拾收拾去朋友那里啦。到了一看果然好多熟食，鸡架、鸡爪、鸡头、干豆腐、鸡翅，都是我爱吃的。

除了熟食还有桔子、苹果、花生、瓜子，当然了，酒也是必不可少的，有雪花、龙山泉以及梅河三种品种，怎么感觉像新年茶话会呢。

我们三人又在微信群里喊了几句，可是这次没有喊来其它朋友，另外一个逢局必到朋友，据说最近家教比较严，媳妇说了再出来喝酒就 打死他，所以这次没喊出来。

于是只好我们三人开喝喽，掀开一罐龙山泉，吃几块鸡架，虽然晚餐我已经吃得很饱，但是还是觉得酒很爽口，菜很美味。

不过刚喝两分钟中不到，家人打电话过来，说我设置的一个报警器报警啦，擦，这说明我一个很重要的程序出了故障，哎，早不出故障晚不出故障，单单等我这边酒瘾勾上来了才出故障，简直气死人啦。

不过好在我是有备而来了呢，带着IPad呢，打开手机热点，用IPad连上去，然后登录SSH查看到底哪里出了故障，结果我擦，连不上去。

无奈只好再喝一口酒，吃一口菜，跑路回家了。朋友说等我回来，不过我忙完已经到晚上10点啦，也懒得出去啦。


(图源 ：pixabay)

哎，不但破了戒酒的成果，而且没喝几口酒，没吃几口菜！第一次喝酒这么失败呀。

使用smartctl 来检测硬盘信息/通电时间/错误状态等

之前的帖子中说到从淘宝买了两块希捷16T的企业级硬盘，但是总觉得有点不放心，不过关于外观以及PCBA版本号的问题，淘宝客服回复说是工艺问题，具体的就一问三不知了。


(图源 ：pixabay)

要说就此评定这硬盘有猫腻，有点过于武断了，况且希捷官网上还是可以查询到保修信息的，那么把硬盘装机器上试试吧。

hdparm

装到机器上后，最基本的操作就是看看硬盘读出的序列号和盘体序列号是否相符了。最简单便捷的操作就是使用hdparm了

sudo hdparm -i /dev/sda

至于返回信息其实我也读不懂，不过好在序列号和盘体上标识的一致：

其中-i选项的含义如下：

-i Display drive identification

如果想显示更详细的信息可以用-I选项，含义如下：

-I Detailed/current information directly from drive

更多详细内容可以用hdparm -h指令获取帮助。

smartctl

看完序列号后，我想看硬盘的一些更细节的信息，比如上电次数，上电时间等等。这时候好像hdparm就捉襟见肘了（也可能是我没会用），不过好在我还知道一个smartctl 。

如果机器上没有安装smartctl相关软件，那么需要使用如下指令安装：

sudo apt install smartmontools

安装完成后，可以查看硬盘信息啦：

sudo smartctl --all /dev/sda

返回的信息很多，比如如下的基本信息：

SMART一些相关信息

我关心的一些详细信息：

其中我较为关心的有以下两点：

4 Start_Stop_Count 0x0032 100 100 020 Old_age Always - 1
9 Power_On_Hours 0x0032 100 100 000 Old_age Always - 0

也就是说上电次数为1次，通电时间0，这个结果说明硬盘是全新的啦。（有没有造假手段我就不清楚啦）

除了查看序列号、通电时常等信息外，还可以用smartctl对硬盘进行测试，可选的测试类型有如下这些：

比如：

sudo smartctl --test=short /dev/sda

进行不同种类的测试需要长短不一的时间，测试完成后可以用如下指令看结果：

sudo smartctl --all /dev/sda

比如我进行了两次测试，结果大致如下：

郁闷的X总

X总是好几个大公司的幕后老板，什么科技公司、医疗器械、化工产品等领域均有涉及，每年赚钱无数。


(图源 ：pixabay)

不过X总的爱好并不在赚钱上，况且用X总的话说，公司嘛，交给专业的人搭理一下就好了，需要我出面的时候我就出门，平时还是做甩手老总比较悠闲。

不过悠闲的日子久了，X总就闲不住了，于是又弄了个汽车会所，用X总的话说他就喜欢摆弄机械相关的东西，修修车，结实一些朋友，这就是想要的幸福生活啊。

不过X总的汽车会所被我们这帮朋友改造成茶室、小酒桌了，经常的一帮朋友在晚上聚集到一起，喝茶、喝酒、侃大山，有时候还烤烤肉串、涮涮火锅，不亦乐乎。

我不止一次和X总说，要不就把修车功能砍掉吧，就保留会所功能，大家天天吃吃喝喝多好啊，何必又脏又累干修车的活，又不赚钱（X总修车收费很低，比如补胎、充气之类的都完全免费）。

可是X总说，如果再不做点事情，整天就是吃吃喝喝，那么与废人何异？必须要干活才能体现自己的价值啊。

不过昨天X总郁闷了，一个朋友过来换行车记录仪，原本的记录仪是用3M双面胶固定到前风挡上的，为了把原来的记录仪卸下来，X总用美工刀小心翼翼的削3M双面胶。

结果不知道哪下用力不当，前风挡的玻璃咔擦一下就开裂了，要知道，汽车玻璃是很结实的，想弄裂弄碎都很难，而且这个车是奥迪A4L，奥迪的风挡玻璃质量应该更是不错。

不管咋说，这个玻璃在X总的手里破碎了，X总肯定要负起责任的，而原厂的玻璃价格要两三千大洋，X总欲哭无泪。

为了安慰X总，我们决定让X总请客，一起出去吃烤肉喝酒，为X总洗掉霉运，于是晚上X总又和我们一大帮朋友出去吃吃喝喝到半夜，吃完喝完付完款，X总说，心情好多啦。


(图源 ：pixabay)

能让X总开心一些，虽然我又戒酒失败，但是也值得啦，朋友不开心的时候，就要安慰朋友，哪怕戒酒失败也无所谓。我突然觉得自己的形象变得高大起来了呢？

京东还是淘宝，这是一个问题？

这两天想为电脑加两块大硬盘，最终相中了希捷银河EXOS 16T的企业盘，不过看了一下价格，促销价格还要4399，也就是说两块硬盘要8798，接近小10000元了，真贵呀。

去淘宝的希捷官方旗舰店看了一下，价格也不便宜，不过这也正常，没准都是一个老板，没必要自己和自己搞价格站。

但是翻看了一下淘宝里其它商家的链接，竟然发现，有些商家这款硬盘只卖¥3458.00，两块全下来，只需¥6916元，便宜了足足1882元，近2000大洋啦。

观察了一下淘宝的店铺的情况，发现评价尚可，硬盘应该还是可以的，那么是在淘宝购买——省钱，还是在京东购买——省心呢？

为了这个问题我纠结了好几天，然后打算去京东购买时，发现这款16T的企业盘竟然卖断货了，哎，天意如此，果断去淘宝下单吧。

¥6916可不是个小数目呢，现金是没有这么多啦，只好动用我的法宝——信用卡啦，不过这个店铺刷信用卡竟然要信用卡服务费，哎，忍了吧。

期望到货的硬盘没有问题吧，如果硬盘没问题，32T的空间应该够我用好久的啦。

被建行黑了10元钱

如果你有很多张银行卡，并且银行卡还没有几个钱，那么你可能会知道有个叫做“小额账户管理费”的东东，还有一个东东叫做“银行卡账户年费”。


(图源 ：pixabay)

简单地说，你持有银行卡并且没有给银行创造什么利润，那么每月/每年都要给银行交钱的。

不过，自从2017年8月1日起银行卡年费、小额账户管理费开始取消，但是银行也有对策，这个取消要自己去银行柜台办理，如果你没有时间或者懒得办理，那么对不起交钱吧。

我之前发帖说过，前些天我换掉了一张用了20年之久的储蓄卡，这个卡是建行的储蓄卡，我早已申请了免除年费。

不过今天无意中查看消费记录，竟然发现被银行扣除了10元钱的“银行卡账户年费”，我擦，我早就申请过免除年费，为嘛还扣我的钱？

话说，10元钱可不是一个小数目，尤其是在币市这么不景气的情况下，这可是一笔巨款啊。于是打了电话银行客服咨询了这事，为啥我明明已经申请了双免，还扣我钱。

电话银行客服说申请了双免就应该免受相关费用，扣错的钱可以退回，但是只能去银行办理，电话银行没有权限处理这事。

好在家边就有建行，于是去转了一圈，结果柜台的MM弄了半天之后，告诉我虽然她给我申请了退费，但是系统不给我退，这种情况只有去开户行处理。

我擦，20多年前的开户行还在不在了我都不清楚，不过客服小妹给我一个放心的答案，还在呢，就在市政府广场附近，我晕，好远啊。


(图源 ：pixabay)

先不说有没有时间，跑一趟油钱都不止10元了，于是只好认了，被黑就被黑吧，谁让我们是弱势群体呢？/(ㄒoㄒ)/~~
