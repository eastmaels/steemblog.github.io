每天进步一点点：在新机器上运行EOS钱包操作遇到的问题及解决

今天在一台新的机器上运行EOS的命令行钱包，遇到了一些问题，记录下来，方便自己以后查找，也希望能给遇到同样问题的朋友一点帮助。


(图源 ：pixabay)

将编译出来的命令行工具(v1.8.1)复制到新机器上后，尝试解锁钱包，出现如下提示:

cleos wallet unlock

“/home/test1234/bin/keosd” launched
Unable to connect to keosd, if keosd is running please kill the process and try again.

咦，既然已经launched，为何还连不上呢？尝试直接启动keosd试试看：

keosd

提示如下：

keosd: error while loading shared libraries: libcurl-gnutls.so.4: cannot open shared object file: No such file or directory

原来keosd根本没有启动成功嘛，所谓的launched其实是骗人的😡

不过既然知道了原因，就好办了，缺啥装啥呗。
sudo apt-get install libcurl3-gnutls

装上之后运行keosd又出现如下提示：

keosd: error while loading shared libraries: libusb-1.0.so.0: cannot open shared object file: No such file or directory

继续暴力解决：

sudo apt-get install libusb-1.0-0-dev

之后再运行keosd：

keosd

一切正常：

再解锁钱包试试：

cleos wallet unlock

终于出现了密码提示窗口，看来这次launched是真的啦。

积分换了个大锅

好多年前就办了招行的信用卡，也知道招行有那么一个积分政策，好像是每消费20元人民币或者2美元就会积一分。


(图源 ：pexels.com)

因为积攒积分相对还是很难的，所以好多年前，招商银行的积分还是很值钱的，我记得很长一段时间，招行的积分价值都在1毛钱左右。

当然了，不同于虚拟币有交易所有价格，招行的积分估值我是通过它能兑换的商品来计算的，比如一床羊毛被，市场上价格是几百元，而招行兑换要几千分，那么积分估值是1毛左右啦。

我记得招行在2006年前后搞过一个最吸引人眼球的活动，就是200W积分可以换个帕萨特轿车，要知道当时的帕萨特轿车可远远不止20W元，招行积分之超值，可见一斑。

然后那时候我的积分是远远不够200W的，连个零头都没达到，只好望车兴叹了。只能换一些羊毛被、双立人刀具啊之类的，也算小有收获了。

当然还玩过什么积分抽奖，断断续续砸进去不少积分，结果从来没有中过奖，就不再投入了。

这两天偶然想起了积分这事，登录上去一看，竟然攒了快10万的积分，心说距离换帕萨特越来越接近了，可惜招行再没搞过这样有力度的活动了。

再去积分商城一看，没啥感兴趣的东西，而且标价都严重虚高，商城里标市场价1000的东西，往往市场价二三百就可以买到，看来积分的价值大幅缩水，远远不如当年了。

转了一大圈，找到一个13590积分兑换的大锅，倒不是这个大锅有多好或者多超值，而是恰巧家里的锅把手坏掉了，还买不到匹配的，那就换一个试试吧。

如果按照20元人民币积一分的比例计算，这个锅相当于我花费二十七万余元后得到的赠品，查了一下，市场价就几百元，当然了招行积分商城的标价是1980元。


(图源 ：pexels.com)

好奢侈的大锅啊，不过如果不换，估计积分贬值的会更加厉害，回头研究研究再换点啥。哎，说白了，就是个游戏罢了。
