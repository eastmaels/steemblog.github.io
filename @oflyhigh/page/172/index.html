油泼面来了

昨晚朋友又喊我出去吃刀削面，我说我请你换个地方吃油泼面吧，比刀削面好吃的很。朋友诧异地问我：“你的比特币赚钱了？”擦，真是哪壶不开提哪壶😭

不过话说，昨天我有一支股票涨停，回了一点血，两碗面的钱还是出得起的。说起这只股票，全是眼泪啊，好多年钱我一个相当好的朋友信誓旦旦地和我说他有超级可靠的内幕，让我建仓这只股票，他已贷款满仓。

我听了后，就买了一些，然后被套至今，目测还有十个涨停板就可以回本了。我的朋友前段时间更是卖掉了两处房产还贷款，凄惨程度不亚于混币圈的。

不多扯了，上图，对，就是这家。

坐下之后就有服务员给倒两碗面汤😳，果然有特色

拌好的油泼面

拌好的油泼棍棍面

怎么样，看起来就很有食欲吧？其实吃起来更香呢，并且价钱也是很便宜的，油泼面12，油泼棍棍面13，混币圈的平时节约一点也能吃的起。

Updated my witness node to v0.20.7 / 将我的STEEM见证人节点升级至v0.20.7

大概十小时以前，steem发布了新的v0.20.7版本，这个版本是一个安全更新，说是只影响序列化逻辑，不影响序列化格式以及共识逻辑，并推荐节点操作者在方便的时候尽快更新。

This is a security release that only impacts serialization logic, not the serialization format or consensus logic. We recommend all node operators update to 0.20.7 as soon as it is convenient.

既然要求尽快更新了，并且我也看到了，当然要尽快弄喽，赶紧编译一下，编译完成后看一下版本信息

steemd --version

貌似没毛病，直接替换原来的steemd文件，并重新执行，一切OK。

幸福的是，如果是从v0.20.6升级上来的，那么这次更新不需要 reindexing。

0.20.7 does not require reindexing from 0.20.6


执行起来一切正常，不过在我的见证人列表工具上显示的还是v0.20.6，这个要等我下次出块才会更新，我估算一下，大概半个多小时以后就会轮到我出块啦。

冷清的超市

昨天去家附近的家乐福超市转了一圈，虽然马上就是电商们炮制出的双十二购物节，并且圣诞元旦也快来临，但是超市里还是无比的冷清。

原本车位一位难求的停车场，到处都是车位，根本没有几辆车。超市楼下的商铺里除了售货员以外几乎没有顾客，以往一直在吆喝的售货员们都懒得吆喝了，因为没有路人，不知道吆喝给谁听。


超市里倒是装扮出浓郁的圣诞氛围。


然而纵有零星的顾客，也对这些不屑一顾。


可口可乐的新口味饮料，无人问津。


单身狗粮，不过想必单身狗们可买不起吧？


冷冷清清凄凄惨惨戚戚

为啥这么冷清呢？我猜很大程度是电商的冲击吧，另外整体经济都不景气，大家花钱的欲望也变低了吧。

另外一种可能就是，好多原本的顾客都炒币了……

AskSteem 要关站了😟

刚刚打开STEEMIT，看到 @thekyle 的新帖子：AskSteem is Shutting Down，大意是由于最近加密货币价格暴跌，AskSteem无法继续运行，将于2019年1月1日停止运行。


(图源 ：pexels.com)

因为平时习惯于使用自己的查询工具，所以对AskSteem了解的并不是很多。但是我知道它也是一个非常好用的STEEM区块链数据查询工具。

去年早些时候，我曾经写过一个帖子获取STEEM区块链数据的几种方式，其中讲到我常用的几种数据库方式获取STEEM区块链数据分别是：

  • SteemData by @furion
  • STEEMSQL by @arcange
  • AskSteem by @thekyle

SteemData几经挣扎后于2018年5月1日关站，之所以说几经挣扎，是因为@furion做了很多努力尝试挽救它，比如说建立专有的STEEM节点用于同步数据等等。我曾经写一篇帖子悼念SteemData: 《讲不出再见 / SteemData 将于5月1日停止服务》

STEEMSQL好久以前就改收费模式了，每个月10SBD，这样也很好，对于需要的人而言，这10SBD的花费还是很值得的，也能让STEEMSQL一直维持下去，以便于提供稳定、持续的服务。但是如果SBD继续下跌，不知道STEEMSQL会何去何从？

AskSteem曾一度被DTube、DLive、Busy.org作为搜索引擎，DLive出走也是好久之前的事情了，我也不确定DTube、Busy.org现在是否还在用AskSteem。

总之，又一个STEEM数据库应用要离开我们了，讲不出的难受，讲不出再见。都说大浪淘沙，说吹尽狂沙始到金，但是浪太大了，也有一些金子被浪带走了。只能希望有朝一日它们再次被新的淘金人发现吧。

使用STEEMIT知识库解决问题

昨天的帖子中，我曾提到我在STEEM上的知识库（文章库）不但没有任何贬值并且在持续增值中，结果今天就用到了我的STEEMIT知识库。


(图源 ：pexels.com)

事情是这样的，我在我的一台VPS上装了WordPress玩，出于安全考虑我将登陆链接wp-login.php换了个名字，并且对WordPress几个文件的代码做了一点修改，这样就实现了隐藏登陆链接的目的。

然而，我今天突然发现，原本的登陆链接有回来了，这是怎么回事呢？仔细一检查，原来是Wordpress升级版本，自动更新，把我原来的修改覆盖了。

这没啥，我再改回去即可。然而问题是，事情过了几个月了，我全然忘记了怎么修改，当初为了做这点修改，我记得我可是研究了大半天呢，难道要重新研究大半天？

突然记起，当初修改的步骤，我应该记到STEEEMIT上了，如果我找到当初的笔记，那么按照操作就可以了，无需再重复之前的研究工作了。

调出我的简单搜索工具，

一下子就找到当时的文章了，原来已经过去4个半月啦。双击指向处，自动为我在浏览器中打开了以下文章：

阅读了一遍，发现操作步骤非常简单，无非以下几步：

  • 修改wp-login.php
  • 重命名wp-login.php
  • 修改general-template.php

按上述步骤操作，瞬间搞定。所以STEEMIT的个人知识库还是很有价值的，毕竟现在年纪大了，记忆力大不如前，有些事情在STEEMIT上记录下来，比放到大脑里记忆靠谱多了。

当然，WordPress升级会覆盖原本的修改，说明我的修改方式不是最佳方式，不过这属于另外的话题了，我也懒得耗费时间去研究了，毕竟改起来一点也不麻烦😄。

寒冬来了

在币圈，最近听到最多词汇莫过于熊市与寒冬，尽管室外的温度已经抵达-20°，但是真正寒冷不是身体上的而是心里。

从STEEM上来看，STEEM价格跌至1.7元以下，号称锚定美元的SBD也已经脱锚价格低至不到4.2人民币，这些都无疑大幅打击大家的信心。

很多当初发帖声称无论币价如何低赚得如何少也不会离开STEEM的朋友，再也见不到当初活跃的身影了。这没什么，如果把STEEM看成一个赚钱的平台，赚的少，谁又愿意耗费时间在这上呢。

就像@deanliu今天的帖子中计算的，一个帖子显示收益10SBD，如果全部变现，那么作者收益不过才4.5美元，何况还有SP部分不能及时变现，何况很多人都到不了10SBD。

4.5美元折合人民币约30左右，这样算下来，我们不如去路边发传单、去开封菜端盘子赚的多呢。

不过STEEM真的仅仅是赚钱平台吗？我看了一下我的历史帖子，觉得收获还是蛮多的。这里记录了很多我生活中的事情，工作中解决的技术问题，以及平时学习进步的点点滴滴。

虽然我持有的SP啥的相比高点已经贬值了不知多少倍，但是我在STEEM上的知识库（文章库）不但没有任何贬值并且在持续增值中，或许，这才是我在STEEM上最大的财富吧。
