曾经以为自己是专家，原来不过是小虾

17年币圈整体都是大牛市的行情，用以往常说的一句话：“站在风口上，猪都能起飞。”我误打误撞，不小心混到币圈来，也买了一些币，眼看着几倍几十倍的上涨，我不禁有些膨胀了，觉得自己简直就是个投资专家。


(图源 ：pexels.com)

每天看着财富不断增值，要说心如止水那是骗人的，其实还是蛮激动的。激动来激动去，觉得自己还是蛮有眼光的嘛，你看虽然有几个币被套，但是我也投资了好几个10倍币甚至百倍币，说自己不是专家，自己都不相信。

因为平时一副专家范，再加上各路媒体都大肆宣传、介绍比特币等虚拟货币，所以也有一些朋友找我推荐靠谱的虚拟币。虽然我很膨胀，但是不傻，推荐虚拟币这种操作可不容易，万一推荐错了，会挨骂的。

但是碍不住朋友们苦苦哀求，我只好按我自己的了解介绍了几个我认为靠谱的币，我觉得即便不涨它个十倍百倍，但是至少应该不会亏钱吧。不过我还是声明了虚拟币有风险投资须谨慎，归零也不是不可能的，我的介绍仅供参考，不作为投资建议。

但是想必他们没把我的话听进去，而是当了耳旁风，然后我推荐的、认为不会亏钱的币，跌掉了96%，然后，我这几个朋友现在都不理我了。这充分说明我的朋友还是不错的，至少没来砍我，也没来骂我……

不过这并没有打击我作为专家的信心，大势所趋嘛，跌90%以上的虚拟币比比皆是，并非我推荐的东西不可靠。作为专家我应该寻找投资机会，在别人恐惧时贪婪，继续加仓。

然后，前些天，我又投了一些，然后今天早晨一看，有的腰斩，有的归零……


我躺在床上，这样拿着手机，想了半天之后，才想明白，原来自己根本不是什么专家，只不过是一条随波逐流的小虾罢了……

就像2008年股市的大牛市，我在6000点还一路买买买，然后用至今仍在套牢的事实证明了自己不是股神；就像300多元开始买黄金，一路买到现在，还没回本，证明自己不是贵金属交易天才。

总得要付出一点代价才会看清自己，才能避免继续膨胀下去——就是这代价有点大了😭

见证人收益大幅缩水

大约两个月以前，我曾经写了一篇文章：《揭秘见证人系列：见证人收益几何？》，里边介绍了见证人收益的来源以及大致的见证人收益情况。


(图源 ：pixabay)

两个多月过去了，见证人收益有什么大的变化了吗？其实从STEEM的经济体系来看，以STEEM作为基础货币来衡量的话，见证人收益几乎没啥变化。

那么我是纯粹的标题党吗？也不尽然，实在是STEEM对法币跌的太多了😂

以两个月前TOP20见证人为例，当时他们每天出块奖励约为283.61 STEEM，按当时约5元人民币/STEEM的，月收入约42500元😍，什么服务器成本啥的都不在话下。

我当是每天出块奖励约为3.88 个STEEM，每月约582元，虽然还远远不够支付服务器费用，但是如果不考虑成本，582元月收入也不错了😂，至少每天换一个汉堡+可乐还是够的。

然而，现在TOP20见证人的奖励几乎没变，当天每天约302.55个STEEM，不过STEEM价格跌到了1.65元左右，算下来每月收益鱼约为15000元，基本上还可以维持下去。

得益于见证人得票的增加，我现在每天见证人收益约为6.17个STEEM，折算成人民币每天约10.18，每月约305元，这价钱每天换不来汉堡+可乐了，只能刀削面+老雪花啤酒了。

如果再想到我的服务器成本，有点想哭了，我在慎重考虑似乎可以关掉备份节点服务器了，省点是点，毕竟这行情挥霍不起啊，另外就是STEEM程序现在很稳定，貌似有无备份节点区别不大。

如果你身边有做STEEM见证人的朋友，并且还不是TOP20，那你一定要对他好一点，没事请他吃吃饭，多陪他喝喝酒，毕竟他太可怜了😭

邮件啊邮件

虽然每天开邮件客户端，但是收一些重要的邮件(比如注册确认)才会点开，其它的各种通知之类的就是瞄一眼，都懒得点开了，然后每隔一段时间，比如十天半个月集中清理一下，久而久之，各个邮箱都爆满了。


(图源 ：pexels.com)

又到了该清理邮件的时候了，哎，想想就头大

比如这个邮箱，大多都是些通知邮件，我按一定规则搜索，然后大致判断一下以后，批量删除即可。

这个是我最重要的信箱之一，里边的每封邮件我都要仔细阅读并处理。

这个是我对外的主要信箱，最近生意惨淡，几乎没人联系我。

额，处理的时候发现错过好几封比较重要的邮件，看来邮箱太多人就不能太懒。话说，刚刚在这个邮箱里还发现个小惊喜，如果不是整理信箱，我都不知道有这么一回事。

另外一个对外信箱，我有点晕，怎么又攒这么多邮件啊。

另外还有其它五六个不是那么重要的信箱，这些邮件够我处理大半天的啦。这些都是业务相关的，还没去看生活相关的信箱呢，不过那里边几乎没啥重要信息啦。

今天目标就是把所有未读邮件都处理一遍，该删的删，该读的读，该回的回，但愿能完成吧。

移动公司的套路

突然接到10086的电话，我就知道推销电话又来了，恰巧不忙，那就听听她们这次又有什么新把戏了。


(图源 ：pixabay)

一个声音悦耳动听的女业务员首先向我问好，并介绍了她的工号等信息，一切听起来都挺正规的，我耐心等待他们步入正题。

首先告诉我什么打电话的通话费用降到全国每分钟一毛钱什么的，这个好像好早以前就取消漫游费了，我猜这不是重点，继续听她表演。

然后业务员和我说，现在移动公司回馈老客户，每月免费送40G全国不限速流量，免费三个月，之后继续用的话20元每月，共12个月，到期自动取消。

听起来很诱人是吧？免费3个月，到期自动取消，如果我不是警惕性超高的话，一般很容易被这两句话吸引，可惜的是中间还有一句话之后继续用的话20元每月，共12个月，也就是说第4到第12个月，每月要交20元人民币呢。

我和业务员说：“对不起，我不需要，请不用帮我办理。”
业务员很诧异：“先生，这个是非常优惠的，我看你卡上没有办理流量套餐，这个非常超值的”
我回答：“对不起，我有别的卡，流量充裕”
业务员继续：“先生你可以对比一下，我们这个是非常便宜的。”
我：“对不起，我不需要。”
业务员：“……先生，”
我：“对不起，我不需要。”
业务员：“好的，那不打扰您了，下次有其它优惠，我们再给您推荐，祝您生活愉快。”

讲真，这个业务员素质不错，推销不成也没有直接挂机；讲真，移动这个流量套餐听起来也很便宜。可是我联通每月都剩几十G的流量啊，再多流量也没有用啊。

另外我不爽的是，明明是卖套餐，何必非得弄些赠送之类的，并且赠送结束还不能自动取消，这不是赤裸裸的套路嘛，如果警惕性稍低一点，就会中招，办理一堆没用的套餐。

20元每月几十G流量套餐并不贵，但是如果你每月只用掉百分之一甚至千分之一的流量呢？那么这个套餐你还会觉得很便宜吗？尤其是，现在二十元可是能买12.5个STEEM呢，按历史高级算，这12.5个STEEM可是值600多元呢，你还觉得很便宜吗？

移动套路够深，可惜的是她错不该在市场行情这么不好的时候向我推销这个套餐……

油泼面来了

昨晚朋友又喊我出去吃刀削面，我说我请你换个地方吃油泼面吧，比刀削面好吃的很。朋友诧异地问我：“你的比特币赚钱了？”擦，真是哪壶不开提哪壶😭

不过话说，昨天我有一支股票涨停，回了一点血，两碗面的钱还是出得起的。说起这只股票，全是眼泪啊，好多年钱我一个相当好的朋友信誓旦旦地和我说他有超级可靠的内幕，让我建仓这只股票，他已贷款满仓。

不多扯了，上图，对，就是这家。

Updated my witness node to v0.20.7 / 将我的STEEM见证人节点升级至v0.20.7

大概十小时以前，steem发布了新的v0.20.7版本，这个版本是一个安全更新，说是只影响序列化逻辑，不影响序列化格式以及共识逻辑，并推荐节点操作者在方便的时候尽快更新。

This is a security release that only impacts serialization logic, not the serialization format or consensus logic. We recommend all node operators update to 0.20.7 as soon as it is convenient.

既然要求尽快更新了，并且我也看到了，当然要尽快弄喽，赶紧编译一下，编译完成后看一下版本信息

steemd --version

貌似没毛病，直接替换原来的steemd文件，并重新执行，一切OK。

幸福的是，如果是从v0.20.6升级上来的，那么这次更新不需要 reindexing。

0.20.7 does not require reindexing from 0.20.6


执行起来一切正常，不过在我的见证人列表工具上显示的还是v0.20.6，这个要等我下次出块才会更新，我估算一下，大概半个多小时以后就会轮到我出块啦。
