房子要抢购？

万科在朋友家附近新开了一个楼盘，朋友想在这个楼盘给父母换一个房子，再给自己换一个房子，这样就能和父母住一个小区里，比较方便。

朋友喊我陪她来看看，于是我们便一起来到这个楼盘的售楼处以及样板间处参观了一下。

我们去的时候沙盘处挤满了人，样板间处也都是参观的人群，更多的人则在休息区和置业顾问研究购买事宜。

和朋友看过样板间后，朋友非常满意，然后开始和置业顾问商讨贷款的问题，朋友的账户有大量的公积金并且每月还有大量的进账，可是怎么算公积金都没法花光。

另外的问题是，朋友现在名下已经有两套房子，属于被限购的人群，需要先出手一套才能购买。问题是出手房子不是一天两天的事情，尽管朋友的一套房子是学区房并且已经降价急售。

更严重的问题是，万科这个楼盘马上要开始抢购了，据说一旦开始抢购，一两天就会抢光的，所以朋友的房子回款也未必来得及。

都在喊房地产市场变冷清了，可是据我所知，沈阳最近房产市场还是非常火热的呢，哎，愁了我朋友这种真正需要换房的人啊。

绍汇大虾 / Eat shrimp with friends

去朋友那玩，她说中午请我们吃绍汇大虾，据说又可以吃虾，又可以下火锅，还可以吃扯面。

到了地方发现环境不错，大虾上来闻气味就觉得相当诱人。我们要地微辣版，吃起来刚刚好。

吃完虾以后，服务员给锅里添上高汤，就变身火锅了，把羊肉牛肉和青菜都下进去，比那些火锅店的火锅好吃多啦。

除了大虾、肉、青菜以外，扯面啊、小菜啊、小食啊、饮料啊、冰激凌等等统统免费不限量。

哎，又撑到了，这还咋减肥啊/(ㄒoㄒ)/~~

喝酒回来被查酒驾

最近体重狂飙，所以决定减肥，朋友晚上喊我出去吃东西我都断然拒绝——然后自己在家吃一些点心之类的。

话说昨晚快到10点的时候，又觉得饿了，可是翻遍家里也没找到啥可以直接吃的。

跑微信上问朋友：“刀削面去不？”朋友回答：“去！”我便去了他那，结果发现另外两个朋友也在，那就一起刀削面去吧。

我们四个开了一辆车，跑到这个号称24小时营业的刀削面馆，弄了几碗刀削面和一些小菜还有啤酒，又单独给朋友弄了一瓶可乐——他负责开车。

朋友郁闷得不得了，说这是第一次干看着别人喝酒自己不喝。不过为了避免犯错误，还是算了。

回来路上，遇到交警拦路查酒驾，朋友高兴得不得了，开车上前猛吹，交警不耐烦地摆手，让我们赶快开走。

朋友回来在微信群里感慨：“第一次看别人喝酒我不喝，然后还遇到查酒驾，好人一生平安啊。”

务农日记：给西红柿秧搭架/ Farming diary: Set up a frame for Tomato Seedlings

从造化大集买的西红柿苗全部存活，并且长势良好，非常茁壮。

不过随着它们的成长，新问题也随之而来，再长高的并且没有攀附的话，它们的抗风能力就会比较弱，成长也会受限，所以是时候为它们搭架了。

找来根长竹竿，把他们截成两段，然后都插在西红柿秧附近，再用一根竹竿横着放在它们上边，架就搭好了。

然后再用细绳把西红柿秧绑到竹竿上，就大功告成啦。万事俱备，就等着吃西红柿啦。

鸢尾变色啦

还记得我以前发过的鸢尾吗？紫色的花瓣非常漂亮。

然而最近这些鸢尾都变成黄色的啦。

我春天时栽种在路边的鸢尾也都开花了，不过不知是否是路边土壤不够肥沃，植株较小，花朵也显得单薄

至于鸢尾为啥会变色，其实并不是很神奇的事情，只不过是紫色和黄色混种啦，紫色花期早，黄色花期晚，看起来就像变色了一样。

紫色鸢尾象征爱意与吉祥，黄色鸢尾象征友谊永固、热情开朗，寓意都不错，而且都很漂亮的啦。

务农日记：南瓜发芽了/ Farming diary: The pumpkin has sprouted

还记得我前些天种南瓜的日记吗？

自从种上了南瓜，我就整天担心，万一我把种子埋深了怎么办？种子能否拱出来？如果种子长期拱不出来会不会憋死？

所以我几乎每天都去观察一下，看看有没有萌发的迹象。前些天去看几乎都是一点迹象都没有，为此，我甚至把其中一个种子挖了出来，看看它有没有死掉。😳

这几天看到种子们终于发芽了，有的甚至已经长出了茁壮的瓜秧，终于放下心来。

如果这些瓜秧都能结瓜，那么我就可以做一枚安静的吃瓜群众了。
