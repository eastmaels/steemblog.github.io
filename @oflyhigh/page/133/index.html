坚持锻炼：单杠

这两天晚上坚持走步锻炼，走累了去小区下边的小广场转悠转悠，看到健身的单杠，忍不住想试试。

不像现在的孩子们有各种娱乐，什么淘气堡啊、什么游乐场啊、什么水世界啊的，我们小时候的娱乐相对简单的很，比如说追双杠，又比如翻单杠。

我背对单杠，双手倒扣，然后往上使劲一翻，就把腿倒勾在双杠上，瞬间觉得整个世界都颠倒过来，然后胃里翻江倒海的感觉，头晕得很觉得天旋地转、天昏地暗。

坚持了三五秒，我就赶紧下来了，因为我觉得继续坚持的后果很有可能是我摔下来，趴在地上，来个嘴啃泥，哦，地上有铺着软软的垫子，啃不到泥。

虽然这时候在这附近遛弯的人没有几个，但是那边广场是一群广场舞大妈呢，我要趴在地上，还是会很惭愧的，所以我断然做出决定，还是不练单杠了，老实遛弯吧。

Chinese hamburger / 午餐：肉夹馍为什么叫肉夹馍呢？

最近A股涨势良好，今天上证指数又突破了3100点，看起来到达6000点指日可待啊，话说我的一个A股账户已经赚钱了，另外一个也回血了好多，现在已经把能动用的资本全追加进去了，满仓待涨。✊

涨了这么多，当然要犒劳一下自己，什么满汉全席啥的依旧不敢想，但是弄点肉夹馍啥的还是可以有的嘛！

肉夹馍 + 鸡腿 + 一口香 + 可乐(没拍照)，是不是很丰盛啊。

传完图片我想起几天前看过的一个小视频，里边的孩子问爸爸，明明是馍夹着肉，为什么叫肉夹馍呢？对啊，为啥叫肉夹馍呢？

百度了一下，总算从百度百科发现了原因：

肉夹馍是古汉语“肉夹于馍”的简称，中国陕西省传统特色食物之一。

想了解更多关于肉夹馍名称的来源之类的，去参考链接中百度百科对应词条看吧，我就不多写啦。

查询 & 取消(或设置) PowerDown Routes

今天在steemd.com浏览自己的账户时，发现竟然被设置了PowerDown Routes，惊出一身冷汗。


(小路，本人拍摄）

何为PowerDown Routes？

那么什么是PowerDown Routes呢？

PowerDown Routes又称为Withdraw Routes，简单来讲，就是Power Down的时候不Power Down到自己的账户，可以按照设定的比例到其它账户，并可以设置是否直接Power Up。

前段时间闹得沸沸扬扬的 @steemit 官方账户Power Down就是通过设置PowerDown Routes直接提现到交易所，这样就省却了先Power Down再Transfer到交易所得繁琐步骤了。

另外一种情况就是账户被盗，黑客除了转走你活期STEEM/SBD以外，还可能帮你设置PowerDown Routes，这样即便你通过恢复账户功能重置了密码，但是只要你继续Power Down，那么钱还会源源不绝地到达黑客账户。

如何查询PowerDown Routes？

听起来是不是很恐怖😱，我觉得也是，为什么我的账户被设置上了PowerDown Routes呢？难道是被黑客盗取了？可是他为何没有把我的活期STEEM/SBD一并偷走呢？

解决这些疑问很简单，查询一下PowerDown Routes的设置就清楚了，查询有好多方法，比如直接调用API，又比如调用一些库封装好的函数，或者最简单的方式就是使用我们的公众号查询了。

如何取消或设置PowerDown Routes？

通过上述查询，可以知道我账户的PowerDown Routes是我自己测试时手欠设置上的，不是被盗，总算可以松一口气啦。

可是作为强迫症患者，是不允许这样情形继续存在下去的，那么要如何取消或者重新设置PowerDown Routes呢？

其实设置和取消是同一个问题，我们来看一下withdraw_route的相关结构体：

和之前我们查询时结果中的结构是一样的。所以，对于一个账户而言：设置的话，只需要设置目标账户、百分比、是否自动Power UP即可；取消的话，只需把百分比设置为0即可。

使用STEEMPY设置或取消

最简单的设置方式是使用steempy——steempython附带的命令行客户端。如何使用请查阅帮助：

steempy powerdownroute --help

返回如下，应该不用我多说啥啦。

使用steem-python 函数

steem-python中设置PowerDown Routes的相关函数为：set_withdraw_vesting_route

简单的示例代码如下：

from steem import Steem
steem = Steem()
steem.set_withdraw_vesting_route("eval", 0, "oflyhigh")
steem.set_withdraw_vesting_route("exec", 0, "oflyhigh")

执行上述代码成功后，我们再来查询一下，是不是已经去掉啦？😄

注：使用steem-python之前需要安装steem-python以及导入私钥等

要变老赖了，怎么破？

前两天晚上，从外边归来，因为不是很远，大概3-5公里吧，我就打算步行回家，当作锻炼了。可是走着走着，发现外边太冷了，再继续走下去，我可能没到家就被冻死了。


(图源 ：百度图片)

于是我想着，不如打个出租车回去吧，结果站道边等半天，出租车到过去很多辆，就是没有一辆空车。我只好继续往前走，走着走着，看到路边一大堆共享单车，并且很醒目地写着：扫码开锁，一元骑单车。

咦，这个不错呀，打车的话至少10元，这个只需一元，于是研究了一会后，终于用微信成功扫码开锁，骑上了共享单车，这也是我第一次用共享单车。

骑呀骑呀，好久没有在夜晚骑自行车了，夜风拂面，说不出的轻柔。然后我看到警察在路上设卡查酒驾，这条路更顺畅了，真是爽歪歪。

约15分钟后，顺利到达小区门口，我下车锁好单车，再回到微信上，一看果然只需支付1元钱，真的超级便宜。然而我手一抖，就把这个支付页面刷没了。

我合计刷掉支付页面不是啥大事，重新进去支付呗，结果进去找了半天也没有支付的欠款的页面，只找到一个充值页面，问题是充值至少10元起。

然而我平时并不需要共享单车啊，共享单车流行这么久，我才第一次用，就是佐证，让我充10元，估计十年也未必会用光吧。

就这样，我欠了摩拜单车1元钱，不知道怎么还上。不知道这样做会被列入失信名单？会不会影响我的征信？会不会被列成老赖，然后限制我乘坐飞机、高铁啥的？好可怕啊，我该怎么办呢？
