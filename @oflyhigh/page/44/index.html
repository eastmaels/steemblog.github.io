难者不会，会者不难：Excel表格取消表头的小箭头

因为要做一些简单的统计工作，所以用Excel做了个小表格，这样弄些简单的数据非常方便。


(图源 ：pixabay)

可是让我郁闷的是表格的表头上，每一列项目上右侧都有一个小小的箭头，看起来十分不美观。

点击小箭头会出现一个排序的功能，然而我并不需要排序呀？

怎么才能把这些个小箭头都去掉呢？我在Excel的界面里找了大半天，还是一头雾水，没有搞定。

这时候我想起了表妹，好吧，此表妹非彼表妹，因为在我们一个小微信群里，就她一个是搞行政的，特别擅长做表，所以被我们戏称为表妹。

可是在群里和表妹说了这个问题之后，她并没有马上回复，估计可能忙公司的事情焦头烂额。这两天虽然不上班，可是估计事情比上班的时候还多。

于是我就想还是先问问度娘吧，结果一问，还真就找到了答案，原来在“Data”标签下，把”Filter”功能关掉就可以。

关掉后的表格就是我想要的样子：

而在我刚刚研究明白之后，“表妹”也回复了我：“你把筛选功能关掉就好了”。

哎，对“表妹”而言就是一句话的事情，我却研究了大半天，这真是：难者不会，会者不难。

把这个记录一下，万一有后来者遇到同样的问题，如果有机会看到这篇文章，就不会像我一样走了许多弯路啦。

新的带货王诞生新的带货王诞生

虽然我几乎不看各种网络主播的直播，但是我也隐约听说有个好像叫李佳琦的带货王，据说他在直播里或者微博上，晒什么产品，什么产品就大火特卖。


(图源 ：pixabay)

不过相比于人民日报，李佳琦大概是小巫见大巫了。昨天人民日报发布的一则消息彻底点燃了网络：

大致内容如下：

近日，经过上海药物所和武汉病毒所联合研究，初步发现中成药双黄连口服液可抑制新型冠状病毒（2019-nCoV）。

双黄连口服液由金银花、黄芩、连翘三味中药组成。中医认为，这三味中药具有良好的清热解毒、表里双清的作用。现代医学研究认为，双黄连口服液具有广谱抗病毒、抑菌、提高机体免疫功能的作用，是目前有效的广谱抗病毒药物之一。

或许人民日报的初衷只是为了传递一个提振人心的好消息，但是姑且不论双黄连是否有效，但是人民日报忽略了自己的影响力了。

新闻发出之后，很快被广泛传播，双黄连口服液瞬间脱销，据说有人连夜跑去24小时药店购买，有人连夜去药店门口排队等开门，有人连电商网站上家禽用的双黄连都抢，甚至双黄莲蓉馅月饼也跟着遭殃。

不过今天人民日报再次发声提醒：

特别提醒：按照世界卫生组织 ，到目前为止，还没有用于预防和治疗新型冠状病毒的药物。特定的治疗方法正在研究中，并将通过临床试验进行测试。

再次提醒：普通人请勿自行服用双黄连口服液，运用中医中药，脱离辨证论治都是不准确的！治病请遵医嘱。

只是这个提醒，是啪啪啪打抢购者的脸，还是打自己的脸呢？不多评价了，还是一起回顾一下03年SARS肆虐期间的一则学术消息吧：

秀发去无踪，头屑更出众

因为年底的时候觉得自己头发不是很长，又加上懒，就没有出去理发，反正发廊之类的服务业很早就上班，不怕没地方理发。


(图源 ：pexels.com)

不过没料到的是疫情来势汹汹，且不说发廊有没有恢复营业，就算是恢复了，我也不敢去呀。

但是这一连十来天没有出去，在家憋得无聊，我就突然觉得这头发有些长了。

越看越觉得长，洗澡的时候还浪费洗发水，我就决定修理一下，可是自己无论如何也修理不出来之前的造型。

干脆就彻底剃光吧，用电动理发器，三下五除二地搞定，搞定之后照镜子一看，不禁感慨：这光头还真是难看啊。

想起一句以前被魔改的广告语：

秀发去无踪，头屑更出众！


(图源 ：pexels.com)

大概这个可以用来形容我了吧，哈哈。

升级本地EOS节点至v2.0.1

EOSIO v2.0 系列已经释出好久啦，最新版本已经是v2.0.1。其实在v2.0.0出来的时候，我编译并测试了一下，但是同步同步着，突然就毫无预兆地卡住了。

尽管重启可以解决这个问题，但是说不定什么时候就重新犯病，所以升级到v2.0.0的计划就一直搁浅了。

不过据说v2.0.1里解决了这个BUG，那么我先跑一下试试喽。

升级

编译还是很顺利的，就是耗时有点长

编译完成后检查一下版本号：

nodeos --version

返回如下信息，版本号v2.0.1，看来没搞错。


按着EOSIO v2.0.0 Release Notes的说明，从快照启动，一切🆗！

EOS V2

EOS V2引进了许多新的特性，我觉得最重要的当属新的EOS VM了，分别是：

  • EOS VM Interpreter (eos-vm)
  • EOS VM Just In Time Compiler (JIT) (eos-vm-jit)
  • EOS VM Optimized Compiler (eos-vm-oc-enable)

其中第三个，我觉得更像是一个选项，和前两者结合使用。第三个选项应该还可以结合eos-vm-oc-compile-threads一起使用，但是我没有测试过。

文档上关于如何使用EOS VM的建议如下：

At this time, block producers should consider all running EOS VM JIT as the WASM runtime on their block producing nodes. Other non-producing nodes should feel free to use the faster EOS VM Optimized Compiler runtime instead.

更多特色，大家参考底部的Release Notes吧，不再赘述了。

EOS 价格

受BTC减半以及VOICE测试版即将上线的利好影响，EOS最近涨势还不错。

可惜的是，我大部分仓位都清空了，全是眼泪啊。

大白菜也成为紧俏物资

或许是继方便面之后吧，大白菜也成为了紧俏物资，以往稀烂贱(超级便宜)的大白菜竟然身价倍增。


(图源 ：pixabay)

先是我加的一个生鲜超市购物群，有人在群里说菜牌上显示1元多/斤的大白菜，结算时竟然是近五元一斤。然后超市的管理人员出来解释，是因为蔬菜调价菜牌还没来得及更改。

之后其它各个微信群里不少网友秀各种白菜的价格，基本上秀出来的白菜价格都在四五元以上，看来大白菜真的不是昔日的大白菜了。

今天一个团购群里，我平时时常从她家购买东西的商家，发起了一个白菜团购。说是不发国难财白菜团购，还说什么现在批发价格都超过了3元钱，还说什么因为东北像武汉捐赠了130万斤大白菜，导致白菜紧缺，这次搭钱给我们团，3元一斤。

本来挺感动的，打算买两颗大白菜备着，可是一个群友无情地发了一条信息，今天她在家附近的一个菜市场买了几十斤大白菜，2.5元/斤。

好吧，团购群里商家有没有赚钱我不知道，不过就算是2.5元/斤，大白菜也不算便宜啊，毕竟平时白菜的价格都是1元左右，秋天的时候更是便宜，基本上都2-3毛/斤。

其实鲁迅早已看穿了一切：

大概是物以稀为贵罢。北京的白菜运往浙江，便用红头绳系住菜根，倒挂在水果店头，尊为“胶菜”；福建野生着的芦荟，一到北京就请进温室，且美名其曰“龙舌兰”。


(图源 ：pixabay)

只是没想到，白菜有一天在我们这也会成为紧俏物资罢了。

方便面成了香饽饽

这段时间微信群里都在讨论什么口罩、酒精、消毒液难买。话说现在蔬菜店、地铁等地方不带口罩都不让进，所以口罩紧缺倒是可以想象的。


(图源 ：pixabay)

不过你大概不会想到方便面也会紧缺吧？之前小区微信群里几个朋友发超市的图片，说小区超市的方便面都被抢光了，大家还嗤之以鼻。

抢方便面？这是多么幼稚的人才能干得出来的事情啊。不过当越来越多的人发现方便面难以买到后，也都加入了抢方便面的大潮。

不过方便面是有它的优势，可以放置一年半载不用担心腐坏，食用起来也很方便开水一泡就可以，极端严重情况下还可以干吃，简直是居家必备良品啊。

相信方便面企业以及各大电商、中小超市等，这次一定赚个盆满钵满，积压的库存都清掉了不说，现在生产企业一定抓紧生产呢吧？


(图源 ：pixabay)

其实长期来看，囤方便面之类的大没有必要，因为这动心产能是很足的，完全可以保证供应，之所以现在显得短期就是大家抢购的缘故，过了这段时间就好了。

好吧，其实我也买了几包，以后有方便面吃啦。( ╯□╰ )
