朋友送的沙洲优黄

和朋友出去吃饭的时候偶然间提了一句相对于啤酒而言我更喜欢喝黄酒，没想到朋友就记在心里。

今天下午朋友微信上和我说别人给他一箱沙洲，他给我拿过来几瓶，让我去他的店面去取。我说太感谢啦，不过现在太冷啦，啥时候暖和点我就去拿。

结果十多分钟后，他打电话告诉我赶紧出来，我开车把酒给你送到小区北门啦，额，其实北门和他的门店也没差多远了，不过都给我送上门来，我再拖延就不讲究啦。

于是赶紧穿好衣服出去取酒，大冬天的出出进进太麻烦啦。到了北门就看到朋友的车，然后拎出来个塑料袋，里边装着4瓶酒。

我说拿两瓶就好啦，朋友说都拿着吧，他留两瓶做菜用，剩下的都给我啦。晕，竟然当料酒用了，真是浪费啊。

看他拿的塑料袋挺单薄的，我也不太放心，于是两瓶塞牛仔裤裤兜里再拎两瓶就回来了。朋友在微信上说，他的老战友还给他寄了一些白酒过来，说等到货了给我两瓶尝尝呢。

有朋友有酒的日子真好。

体验工行ATM扫码取现

昨晚路过工行，突然想起来工行卡里还有几千块钱，不如取出来留年底花吧，可是我没带银行卡，记得工行手机银行里有神马扫码取现的，我来试试能不能不用卡就把钱取出来。

到ATM机，点出扫码取现的界面，然后打开手机银行，进入扫码取现，界面提示如下：

输入金额

结果提示要添加支付卡，好吧，我按提示添加支付卡，其实就是把我的银行卡和这个扫码取现绑定的过程，中途需要短信验证码。

我按提示绑好卡之后，咦，ATM机已经自动返回主界面了，可是我还没提现呢啊。进手机网银一看，还好，钱都在呢，不用担心了，估计再操作一遍钱就出来了。

结果按提示又操作一遍扫码取现，这次不用绑卡了，结果提示我工银E支付超限，看了一下工银E支付，提示日限额5000，今日已用5000，可是我明明还没取出钱来啊。

我尝试把工银E支付限额调整为20000，可是提示我需要蓝牙盾，晕，我要啥都带，还用无卡取现嘛?!

万般无奈，只好打95588咨询，客服很有耐心，问了我的情况并查询后，告知我可以试试把工银E支付限额调整为10000，这样只需短信验证码就可以了，无需蓝牙盾。

于是我就费了半天力气把工银E支付限额调整到10000，果然用短信验证码就搞定了。然后再在ATM上扫码取现，5000到手，更多就不能提了，因为我的限额已经用光了，并没没有蓝牙盾在手，无法调额。

不得不说，现在银行为了方便客户整出了很多花样，属实也解决了生活中的很多痛点。不过如果细节做得更精致一些，那就更好啦。

比如我明明钱还没出来，就被扣掉5000的限额，尽管不影响我资产安全，但是，这明明就是BUG嘛！

退保风波

朋友当初购买房子用了贷款，然后被稀里糊涂买了房贷保险，至于房贷保险保的是啥，受益人是谁，朋友是一概不知。反正开发商的代办人让咋办就咋办呗，据说不买这个银行不放款。


(图源 ：pexels.com)

话说一晃10多年过去了，朋友有钱可以提前还贷了 ，办了提前还贷手续后，想起来房贷保险这码事，打听了一下提前还贷竟然能退保，于是打电话咨询了一下保险公司。

保险公司的人很惊讶，过了这么多年还有人想着退保的事，一般人买了就忘了，哪能一直记着这事，保险公司甚至在系统里都找不到朋友的保单，后来还是通过朋友的老身份证号找到的。最后，保险公司给朋友发一个地址，让他去那办理。

朋友喊我陪他一起去，我闲着没事，就和他跑了一圈。到保险公司后，有个小姑娘接待了他，让他填了好多表格。小姑娘拿着他保单的时候挺惊讶的，说怎么保费这么多，一般的也就几百元，朋友花了1600多元。

朋友吐槽：“还不是你们收多少我们给多少，我也不明白啊。”小姑娘尴尬地笑笑。填完单子朋友问：“钱什么时候给我啊?”小姑娘回答：“慢慢等吧，你这个保单比较老，要上边审核，大概需要一个月吧，到时候会打你银行卡。”

朋友再问：“能退多少钱啊？”小姑娘回答：“退不了几个钱，要扣手续费啥的，具体多少要等系统计算出来。”办完手续，我和朋友就回去了。

前几天朋友微信上告诉我，保险钱退回来了，才退了600多一点，他说按照保额和年限，他觉得能退1000多呢，不过后来上网查了一下，有人提及类似情况的退款，还要倒找保险公司的钱，他的心里就平衡多了。

竟然还有这种事，我说那你等于白赚好多钱啊。“可不是嘛，哪天有空咱们去喝酒庆祝吧。”朋友说到。咦，这倒是个让人无法拒绝的提议呢。😀

如何查询STEEMIT的Power Down信息 / Withdraw routes

最近社区都在关注 @steemit 以及相关账户的Power Down事件，大家都想要一个团结的、稳定的社区，所以STEEMIT公司处理不当的话可能导致众叛亲离。

这篇文章我们不讨论@steemit Power Down缘由，也不去做各种猜测，只是告诉大家如何方便的查询@steemit Power Down的相关信息。

steemd.com

Steemd.com 是 steem区块链上最好用的区块链浏览器，查询各种信息都很方便。点开https://steemd.com/@steemit ，我们就可以查看到@steemit用户的信息。

上图显示了@steemit将Power down 的SP。但是遗憾的是，它没有给出具体日期，也没有给出Power Down的总量，也没有给出Power Down到了第几周。

在下边我们可以查询到更细节的信息，但是不是那么容易解读。

使用微信公众号

使用微信公众号可以更直观的查看用户的Power Down信息。

上边显示了@steemit Power Down的总量、已经Power Down的数量、每周数量、进展到第几周、下次数量、下次Power Down时间等。

在steemd信息中，我们看到这个Withdraw routes，这个是什么意思呢？简单来讲就是Power Down到哪个账户。

我们看到约6小时前@steemit 清空了一些 routes，但是从之前的信息来看，还应该存在一个Withdraw routes ，可是steemd上无法查看到具体信息。

所以，是时候请出我们的公众号啦。用@steemit?pdr命令可以轻松查询到@steemit当前的Withdraw routes

可见，至少6小时之前，@steemit还没有放弃Power Down SP到交易所的计划。

关于STEEM分叉与STEEMIT Power Down，你的立场是？

外文区吵得沸沸扬扬，所以我觉得有必要来中文区讨论一下，让大家知道发生了什么事，以及其它人的观点。


(图源 ：pexels.com)

简单来讲，就是有人在Github上提交了代码，弄了一个Hardfork 21 “Freedom”，而这个HF要干啥呢？简单来讲就是把STEEMIT公司踢出局。

我之前简单捋了一下这事，大致可是是类似如下情况

当然可能是更复杂的剧情，我就不胡乱猜测了。

那么问题来了，STEEMIT公司其实没必要Power Down，因为他的投票权非常大，大到可以控制见证人的地步，所以如果哪个见证人有造反的苗头，STEEMIT把他投出TOP 20就好了。换言之，TOP20都是自己人，STEEMIT就高枕无忧了。

但是，万一某些见证人串通好，表面上支持STEEMIT，暗地里迅速地实施这个HF21并达到阈值，打STEEMIT个措手不及，那么STEEMIT的资产被置空了，也就无所谓投票权的问题了，就会彻底出局。

还有一种可能就是部分见证人直接跑另外一条STEEM链，类似GOLOS之类的，然后在这个链上，把STEEMIT踢出局。

好些个见证人都发布宣言，号称支持STEEMIT，绝不弄HF21，也绝不分叉出一条新链，并劝说STEEMIT停止Power Down。另外也有见证人表示你不停掉Power Down，我就不支持你啦。还有人旗帜鲜明的表示STEEMIT这几年毫无作为，坚决要把它踢飞😱。

那么如果你是见证人，你的立场会是什么呢？不过我觉得，先不说HF21的事，STEEMIT公司应该首先停止Power Down，这样会得到更多群众支持的。

升级本地EOS节点至1.6.0

EOS又双叒叕发布新版本啦，虽然不知道都改了些什么，但是感觉好厉害的样子，所以当然要升级一下喽。

我还是用的EOS-Mainnet上的版本，貌似现在好多人已经不管这个了，直接用EOSIO上的代码了。

唯一需要注意的地方是要运行一下：git submodule sync，如果是全新的克隆则无需此操作。

原文部分如下：

As part of this release, some of the URLs have changed for the submodules this git repository depends on. Users who are fetching updates into a pre-existing clone of the repository are strongly encouraged to execute git submodule sync to ensure that their workflow is not interrupted.

Build 完成看一下版本号：

nodeos –version

EOS发布1.6.0版本，不知道EOS在市场上会不会涨一波呢？不过已经发布两天了，看起来一点动静都没有呢，哎，熊市不要期望太多。
