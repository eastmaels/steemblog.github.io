Fan Error 之又见Fan Error

今天需要用到之前本子里的一点数据，于是开机，结果等2秒多之后并没有像往常一样迅速启动，而是滴滴两声之后在屏幕上提示Fan Error然后关机。


(没抓到Fan Error，弄个图凑数)

说起这个Fan Error，我实在是太熟悉了，我的T42就是牺牲于Fan Error，不过我怎么可能被这种问题难住呢？之后每次启动T42的时候，我都鼓起腮帮子对着风扇口使劲吹气。

只要把风扇吹得转动起来，T42就会正常开机。当时的朋友们嘲笑我，说我何不买个强力吸尘器，每次用吸尘器吸一下，就省得我鼓起腮帮子吹了。

吹了一段时间后，我实在吹不动了，于是换成了T60，再之后换成X61，再之后换成了X230一直使用至今，也算是超期服役了。

有了以往的经验，对付X230的Fan Error我是手到擒来，哦不对，应该是直接张嘴就解决，继续鼓起腮帮子使劲吹，X230顺利启动，✌！


(烂本本开会）

其实除了风扇，X230没有一点问题，我弄成12G内存，三星的840PRO SSD硬盘，总之一直没掉过链子，回头看看能不能换个风扇让其焕发新生。
