升级本地EOS节点至1.6.1

好些天没有去Github上看了，今天上去一看，EOS于6天前推出了v1.6.1版本，并且这个版本包含关键的安全修复，看来又要更新一下啦。

This release comprises critical security fixes

这次直接用的EOSIO上的代码了，已经彻底放弃EOS-Mainnet啦，不放弃也没办法，这都过去6天了，还是重要的安全更新，但是EOS-Mainnet一点更新的苗头都没有呢。

按照惯例要看一下版本号：

nodeos –version

返回如下：

看来没搞错，哈哈，以下为安全更新的内容：

Consolidated Security Fixes

  • net_plugin security fixes
  • Additional checktime calls to limit cpu usage
  • Limit memory usage in producer_plugin

EOS更新还是很频繁的嘛，相比之下STEEM这边倒是波澜不惊，不过想想也是，毕竟EOS还很新，而STEEM跑了2年半，相对要稳定很多嘛。
