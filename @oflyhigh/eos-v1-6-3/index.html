升级本地EOS节点至v1.6.3

好久没有升级EOS节点啦，去Github上看一下，EOS v1.7.0版本都已经发出来五天了。不过v1.7.0还是让他们别人先去试水吧🤭

倒是v1.6.3据说是修复了net_plugin中的小小内存泄漏问题，所以应该升级一下。

This release contains security bug fixes and miscellaneous fixes.

编译依旧是超级顺利的，按惯例确认一下版本号

nodeos –version

返回如下，看来一切顺利哦
