开通了新的赌博工具 / New gambling tool

很早以前就知道沪港通，但是隐约记得当时需要去券商那当面申请并签约，我这个人比较懒惰，所以一直没有去开。

今天无意中摆弄新券商的手机APP，发现在APP里竟然就可以直接签约，简直是太方便了，于是果断地进行了签约开通。

想开通这个首先要进行测评，可是我对沪港通的了解仅限于名字

十道题的测试我全是蒙的，几乎全军覆没，可是覆没后它给出正确答案（红色的），所以这个测试其实就是走形式喽。

有了正确答案我只需照抄即可， 当然100分喽。

然后就可以开通啦

开通成功啦。

不能白开啊，买它100股腾讯试试吧。

怎么到手就赔了近2000元呢？

研究半天才搞明白这是自动换汇（人民币—>港币，港币到人民币）的损失，看来这玩意也不太好玩啊。

赌博这事，赢家都是赌场，哎，可是咋就戒不了呢/(ㄒoㄒ)/~~
