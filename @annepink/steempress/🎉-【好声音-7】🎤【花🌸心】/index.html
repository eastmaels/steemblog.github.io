嗨 ～ 各位亲爱哒盆友们 … 刚听说今天是3月14日白色情人节喔 ～ 你们有啥感觉没😁 我是很少在意这些西方节日的 … 还是早上公司晨会说起才想起来…

话说2月14日男哒给女盆友的浪漫礼物之后 …在3月14日女哒就要回赠礼物给男盆友了😅这也是对西方国家传统情人节的一种延续了… 爱是相互的 … 3月是对2月情谊的回应，相信这也是白色情人节是什么意思的答案所在吧 ～

那么在这第7期好声音活动火热进行中… 我再次给大家带来这首周华健的成名曲 ，也是华语乐坛的经典歌曲 「花心」

这首歌貌似表达对‘‘花’’的留恋与爱慕之情 … 希望能与‘‘花’’同行的意思吧 … 歌词中的那句‘‘花的心藏在蕊中’’预示了花期易逝，应该珍惜每一寸光阴 … 表达了一种态度 … 一种萌动地青春 … 一种难忘的回忆 …

花心 … 似乎自古以来都是贬义词，但在这里确赋予新意！所以女人要花心思修炼自己内在气质 …花心思装扮自己的形象 … 一句话，就是花心思把自己内外兼修，花点心思让自己更具魅力哈 ～ 让自己余生生活更精彩，何乐而不为呢 … 哈哈
