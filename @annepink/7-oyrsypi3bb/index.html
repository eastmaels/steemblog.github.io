beyond乐队经典励志歌曲「不再犹豫」你熟悉吗…说的好像是我们每一个人 … 人生短暂啊 …再犹豫就真的老了… 心有梦想不再犹豫 … 想要什么 … 就去努力实现吧 ～ ～

不犹豫 … 一方面是明白了 … 一方面是不想了 。明白了 …明白的是自己应该持有什么样的态度去面对什么样的是非 …话到嘴边半句多 … 有些话本来就应该说给听得懂的人听

不想了 … 不想的是那些永远讨论不出结果的恩恩怨怨 、是是非非 … 想来想去 … 越想越憋 … 越想越烦 … 有些事情解决的方式就是不想了 … 这并不是逃避或是妥协，而是对自己的一种宽容吧

人生之中你会犹豫很多事情 … 不过终究这些事 … 或许会渐行渐远离你远去哒 …

这首歌我有唱两个版本，合唱和独唱 … 「不再犹豫」合唱版点这，「不再犹豫」独唱版点这

欢迎不再犹豫的你一起来欢唱吧 ～ 感谢支持呦🤗
