哎鸭 …每月小姐(结)如期进行 …😅

今天已是5月第三天了 … 在盆友⭕️ 里每天都有各种‘‘晒姿’’千姿百态哒 … 劳动假期亲们都过的相当开心充实噢 ~ 就像这明媚的阳光般灿烂很是欢喜啊 ~ ~哈🤗

人间4月天 ~~ 一年之计在于春啊 … 最美好的4月是女人们都稀饭的季节 … 可以换上轻薄的美衣美鞋 ~ 可以穿上漂亮哒裙子 … 去欣赏春色撩人的美景，去见想见的人喔~😁

整个4月大致是比较舒服的 …虽雨水稍多，可是跟“清明”节气在此月有关吧 .

工作 1个月的时间如忙的时候就会过地狠快，闲的时候就会过得更慢 …😂 不知不觉一年就过去了3分之1了 … 静下心来的时候想想挺有压力的，公司每月布置的业绩考核，我都没如他意达标 … 😅 按照我自己的预期想法，还是有所为滴进行着 … 虽然有些波折，但这也是难免的，关键我自己在进步啊 … 不就是钱少拿点么 … 啊这都不是很重要了 … 我做事的风格在于细节过程 ~ 嘿嘿

steem 4月新手村迎来了一周年纪念日，还发行了新手币，这个是很值得大家开心庆贺的盛举啊 ~👏 我虽然力量渺小但是多少也是我的一点小心意，该支持该报恩啦👊 过去的一年新手村很多荆棘坎坷 … 不容易地挺过来了💪… 那几位大神村中S.H.E都是新人们哒大爱啊ヽ(´з｀)ﾉ

我的成长经历虽然也不易，还不小心的过了个小寿60了 …😅 新手村的帮助就是我的动力源 …当然还有很多盆友们的陪伴 … 真心感谢一路上有你

生活 妈妈的腿已经基本康复了 … 她又恢复了以前的广场舞乐子👏 我与妈妈的母子情意也更和谐浓情啦 ~ 妈妈还把她的存款全部交给我，可见妈妈对我的信任和爱(⑉°з°)-♡有多深情 … 😭作为家里大女儿的我突然感觉无比的荣宠与幸福感~💗 愿我的妈妈和爸爸还有所有我爱的亲人们一直稳稳地愉快生活 ~

自己 我突然发现自己有个缺点就是很容易被小事影响心情，还会把这种坏心情影响到别人身上，本来不关别人事，心情一烦看哪儿都不顺眼了 …🙈其实根本就是自己问题，所以要时刻提醒自我反省自我批评态度

能做自己喜欢做的事 … 细细去体味生活的点点滴滴 … 努力地工作 … 潇洒地玩乐 … 改变一颗接纳包容万物的心态，这是最重要的!!

有些时候有些事情得到的结果都出乎我的意料，也让我深感幸运4月的我很开心 ~期待更好的5月 … 加油😊
