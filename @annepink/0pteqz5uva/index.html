二月嘛因为是春节期间 … 再加上本来就少几天 …感觉忽然一下就过去了样 … 也因为这种特殊时期 ，所有的锻炼、学习都会受影响，再加上遇到了一些令自己情绪波动较大的事情，所以很多方面都没有上月做得好。不过也不算焦虑啦 … 就是偶然会跟人比较，为神马别人能做到事情仅仅油条呢

整个二月也没去哪里玩 … 最多是带熊孩纸到秋水广场看看灯展 …欣赏八一广场及一些日常人多的地方看看夜景 ～ 因为差不多整个二月都是阴雨连绵的天气 …精神状态不佳…😔

回看往年的1、2月印象是模糊的 … 大多在总结和调整计划中度过 … 3月才真真的是一年的开始 …要加油喔 ～

下面简单总结下二月的状况

生活：妈妈因为身体不佳 … 基本隔三差五回去照看下 ～ 其实也没做啥，就是帮洗洗衣服一些琐碎事情，大部分都是和妈妈聊天拉家常 … 饭做的比较少 都是我爸做给我们吃 …因为爸妈是很明事理的家长，他们体谅我们，难得回家 …尽量他弄给我们吃 ～ 所以过年期间人反而变得有些慵懒了…

过年…一家人团聚总是好的 …发现长辈正在以一种比我们成长更快的速度衰老 …那么要珍惜在一起的时间了

工作：好的工作可以随时调整 …但也是需要持之以恒地坚持…什么是好、中等、差的工作呢？没有明确的衡量标准 …而当觉得自己与周遭不匹配时 … 要么学会适应 …调整自己的标准，选择“卧槽”😅 要么学会远离，融入和自己更同频的环境 …选择“跳槽” …但无论做怎样的选择 …自身的能力，才真正代表了拥有选择权的自由 ～ 相反 … 不去选择也是一种选择权 …选择的背后 … 探出头来的，就是代价了 …

日子总是平淡的过 … 也没神马特别发生 …往后的日子里 …希望找到自己的节奏，才是最重要的 ～ 有些时候的低落 … 难以和人言说 … 也不想逢人就絮叨的苦痛 … 竟在书中找到了共感 … 所以还是靠自己

steemit：在steemit上还算好吧 …日更照常努力做到 … 就是有时候脑袋空空啊 …不晓得该写写啥 … 所以大部分还是在线刷steemit …看看大家的帖子 …找些感觉吧 … 不过更需要多读书和运动 …好像已经从「强迫看书」 变成「主动阅读」了😅 当任务变成习惯了 … 一切都自然而然起来了 …那一刻内心是无比充盈

好了… 二月总结就这么多吧
