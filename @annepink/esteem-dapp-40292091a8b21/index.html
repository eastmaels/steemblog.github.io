今天给大家讲讲我体验Esteem dapp的感受哈 ~Esteem 是一款基于STEEM手机上的app ~ 这样使用起来就特别方便了，随时随地都能看帖子写帖子 …

首先我想说很感谢 @davidke20先生的推荐 ~ 这款dapp自我安装以来 … 用到现在没有遇到神马特别麻烦的不适应，唯一就是图片不知道怎么上载，后来他教我把图片复制出来，就可以粘贴过来了 … 诶~ 还真是不错呦 … 把这个问题解决了 …我用到现在基本没有任何问题了 …

就这前不久遇到要翻墙的问题 … 当时很失望啊 … 以为用不了了 … 没想到这个问题一下就解决了 … 多亏了 @davidke20先生用心良苦与esteem大佬及时沟通和帮助 … 才解决了咱们国内盆友们的使用难处和欲望 ~ 仅仅只需换下节点 … 就可以如愿以偿的重新使用了 …

而且我觉得这个dapp比steemit好用多了，打开dapp映入眼帘的画面感很清晰 ~ 高清画面给人以清爽质感 … 功能也十分强大全面 ，基本我是没觉得哪里不好用，而且此款dpp拥有超过50w sp，还是@davidke20大佬帮咱们抱到的大腿 ~多好的一个人喔🙏

不过以前是说不会自动保存，但可以手动保存也是很不错哒，而且我近来用时，退出编辑页面也没遇到没及时保存就没了内容啊 … 我有时候直接转换其他dapp，再回去编辑页面还是依然完整的内容妥妥哒在呢 … 非常好用啊~

而且点赞收入也是非常不错的，虽然只是要收点抽成 ，但总比没有强吧…

所以所以 … 我很稀饭用Esteem dapp喔… 绝对好用好用很好用鸭 ~~👍👍👍😄
