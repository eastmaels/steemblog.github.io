「喜欢你」这首歌也是Beyond乐队的一首经典歌曲，据说是黄家驹写给已经分手的女友的一首情歌 … 由于某些原因家驹不得不放弃深爱着的女友 … 把时间和精力放到音乐中去 … 这令他十分的愧疚 … 所以他便写了《喜欢你》这首歌，表达对失去爱情的苦楚 … 算是一封迟来的忏悔情书 … 在爱情和音乐的抉择面前 … 无奈地放弃了感情 … 此歌是家驹纯真质朴的自我反省吧 … 但却令人十分感动啊 … 为他追寻音乐的忍痛割爱之痛 …
