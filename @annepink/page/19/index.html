beyond乐队经典励志歌曲「不再犹豫」你熟悉吗…说的好像是我们每一个人 … 人生短暂啊 …再犹豫就真的老了… 心有梦想不再犹豫 … 想要什么 … 就去努力实现吧 ～ ～

不犹豫 … 一方面是明白了 … 一方面是不想了 。明白了 …明白的是自己应该持有什么样的态度去面对什么样的是非 …话到嘴边半句多 … 有些话本来就应该说给听得懂的人听

不想了 … 不想的是那些永远讨论不出结果的恩恩怨怨 、是是非非 … 想来想去 … 越想越憋 … 越想越烦 … 有些事情解决的方式就是不想了 … 这并不是逃避或是妥协，而是对自己的一种宽容吧

人生之中你会犹豫很多事情 … 不过终究这些事 … 或许会渐行渐远离你远去哒 …

这首歌我有唱两个版本，合唱和独唱 … 「不再犹豫」合唱版点这，「不再犹豫」独唱版点这

这照片是在地铁站台自拍滴…我说咱们来张自拍吧…他马上就给你‘‘捣蛋’’你瞧他那啥表情鸭 …😅 😅 😅

话说熊孩纸开学已一月了 … 但还是有些惰性懒散，尤其是这放完假的开学后 … 更是感觉他木有收心 …😖 这几周学习态度总回不到原位，无论是写作业还是背书…总是要催着做😓

今下午看到@liumei文章也谈到孩纸学习不积极主动地状态 … 有同感啊…😅 我想作为妈妈的我，很有必要再次来做开学实动，以在一定程度上调动起熊孩纸学习的积极性 … 用什么手段逼都不如他自己肯学好啊 … 孩纸大了，也总不好一直逼着，浪费大人太过精力啊…

但是这实动做不好就成了威胁恐吓或是说教了，这就没必要了，既达不到我要的效果，也破坏了母子和谐啊😅 所以面对越来越大的孩纸，这实动到底该怎么做呢 … 我被有点困惑了…

鉴于我的人生经历和所闻 … 唯恐稍不留神 …就成了炒现饭“你要好好学习啊！”的口头禅😅 这话虽说没啥问题 … 但感觉有种拿不出道理 …就只管下结论的流氓气质🤐 而且我小时候常被这句话跟着‘‘不然长大就要饭去！”颇有恐吓的效果😂

这么多年过去了… 这样子的习惯很多作为家长的我们‘‘传说’’的还挺好呢😂 虽前有“要饭去”的说法了 … 但与时俱进的多了些更具体的。不是我耸言听闻哦 … 这句话在现代家长之间广为流传呢 … 见缝插针 …张嘴就来的程度堪比“你看看人家那个小谁谁’’怎样怎样…

我跟熊孩纸是这么说的 … 不学知识不动脑子慢慢地越不用就越笨 … 最后就成笨🐷 了…当时感觉他害怕了 … 我还得意呢 …

后来有一次 … 我边嗑瓜子边看📺 电视被他瞧见了 … 他就来句：我在西瓜视频看见个妈妈嗑瓜子嗑多了就变白痴了…啊…啊…无语╯﹏╰了…他学我语气对付我来了…😰

还有一次我来了列假把一片卫生巾落在床上忘收了 …被熊孩纸无意间看见追着问我 …这是神马呀 … 我正要回答他，他马上来句…哦…我晓得了是卫生巾是吧…哦…天啊＼(^▽^)／！…

所以永远不要小瞧熊孩纸 … 如果你觉得教育人是大人对小孩纸的特权 … 那就错了 …

不管家长文化水平高低 … 我个人觉得问题也许出在家长们拿着成人的世界观来理所应当的教育孩子 … 不要太过于让小孩纸去理解大人的道理，我们小时候不理解，长大了自然就明白了，有些道理只有自己探索出来才有价值 … 相比之下，在年少时建立起一些良好的习惯反而更有意义些 …
