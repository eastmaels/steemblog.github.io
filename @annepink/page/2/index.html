说起我最喜欢的一餐饭挺多的，因为我是个不怎么挑食的人，基本神马饭菜到我嘴里都觉得好吃，只要是和爸妈团聚在一起，哪怕是咸菜萝卜干下饭都是好吃(｢･ω･)｢嘿

讲真的！我每次回到家，爸妈都会为我们准备好吃哒饭菜，就是最爱爸妈做的饭菜嘛(´-ω-`)我相信每个人都有这体会吧… 哈

但是我也有比较偏喜欢的，比如肥肠饭/鸭胗饭/辣椒炒肉饭/牛肉饭等等都是我喜欢的，这些我都会自己做的来

唯独一样菜我做不来，就是梅菜扣肉，梅菜扣肉也是我很喜欢滴，我虽然知道做这个菜的工序，但是我最怕炸那个五花肉，其实就是想偷懒，不愿动手去做😅

因为这个梅菜扣肉是我老爸的拿手菜，只有他做的出那个味，想想都觉得好吃，每次跟家人或是同事聊到我爸做的这个菜都让人垂涎三尺啊…🤭

以前这个菜属于特别节假日才能吃到，现在生活条件好了，神马菜都不是事，平常日子一样可以吃到我爸做的，只要我们想吃… 我爸就会满足我们的吃货欲望😅

这道菜是今年大年三十时老爸做给我们的，正好当时留着照片哈哈~ 正好今天可以写这篇文用上了

这也是一道勾起我们年少记忆的佳肴，小时候我们三姊妹就对它倾心已久啊😍 夹上几块带肉汁的扣肉，咀嚼着…… 那梅菜的浓郁芳香… 扣肉的软烂醇香… 你会感觉它一点都不肥腻，好吃到爆😋

大口吃饭🍚 吧啦几下… 一碗饭就下肚了…… 一脸满足的样子~ ~ 幸福的都不想做人了哈哈

要是你看到我的表情绝对足以勾起你对这道菜的好奇来🙊

话说梅菜扣肉要做到好吃，入口即化是有秘诀的：一煮二煎三蒸… 应该都知道吧！

这里就大概捋捋梅菜扣肉的最主要材料和做法步骤吧
五花肉约一斤 / 梅干菜一包 / 料酒 / 葱姜蒜 / 八角香叶 / 胡椒粉 / 淀粉 / 老抽生抽

具体成品步骤
①一块五花肉放入冷水锅里先焯水，再下入适量料酒去腥，水烧开后将浮沫提去，加入姜/香叶八角… 葱段，大火煮沸转中火再煮15分钟左右，用筷子能轻易插透五花肉即可
②煮五花肉的同时，我们可以把梅干菜用水浸泡一下再清洗干净，千万不要泡过太久以免把梅菜的香味泡没了
③把煮好的五花肉捞起沥干水，肉皮上抹上老抽上色，放置五分钟
④炒锅烧热后倒入菜籽油，烧至八成热，将五花肉皮朝下… 放入锅中炸至呈深红色，捞出凉凉
⑤再来把洗干净的梅菜，加点盐和胡椒粉一起下锅炒出香味，装起备用
⑥放凉了的五花肉切片，大概一厘米厚左右
⑦肉皮朝下码好在圆碗里
⑧生抽/老抽/鸡精/料酒/淀粉混合在一起搅拌均匀一下
⑨炒好的梅菜铺在码好的五花肉片上面，淋上刚才配好的酱汁
⑩下锅蒸熟约35分钟左右即可

梅菜扣肉应该是很多人的最爱了… 颜色酱红油亮滴… 汤汁粘稠鲜美😍 让人食指大动大快朵颐地吃肉感觉啊！

做的时候只要注意五花肉片需要浸透入味，那个肥肉的油脂还要被充分煎出，剩余的部分就不再会腻口了…… 而且瘦肉部分吸饱了汤汁，搭配梅菜的滋味，罪恶感都来不及涌出了… 已经被幸福感掩盖！哈哈哈🤭

美美哒吃上一顿这样的饭… 温润滴饱足感！

可是恰巧又在这个时候，好死不死的铜匠老婆闹上门来了！说自己的老公来了一趟这小学校神马姿势都学会了，居然还会「咬耳朵」耳朵是能咬的吗？！(笑喷)🤣🤣🤣 到底是哪个女人👩教会的呀！给我站出来！也教教我呗！哈哈哈哈…

这个时候铜匠闪亮登场了，他想要保护张一曼，却被她一巴掌打在脸上了…

受了爱情打击的铜匠突然变了一个人似的！又主动扮演「驴得水」站在了特派员面前，他想报复这一切

剧情真是一波未平一波又起啊！一向好强洒脱的张一曼自己巴掌打着自己的脸上… 想要挽回这一切… 甚至还被校长剪掉了自己心爱的长发

张一曼被剪去头发被报复时，我哭去了半抽纸巾…😭 校长一边将她如珠如宝的秀发一点点剪掉，她流着眼泪，脑海中回想起那些与昔日同事快乐的时光，她以为校长会对她手下留情，她以为大家都是一家人，不会让她那么难堪，她以为这只是小小的牺牲，她认为这一切都是值得的，为了这些家人愿意放弃掉所有，张一曼心中一定是这样想的

但当她回过神来，看到了自己近乎没有的头发，她还是崩溃了！！她疯了！！她发现这个后果原来不是她所可以承受的，原来牺牲可以是那么大，她一直付出，一直牺牲自己，到最后把自己完全消耗掉了

同样因为张一曼受了情伤的裴奎山冷眼看着一切… 甚至还恶语相向，而很刚的周铁男说出了事实跳出来想要打铜匠… 然后铜匠随着一声枪响吓到跪在地上磕头叫：爸爸爸爸… 我错了，你别开枪… 哈哈哈😂

另一面特派员又为了骗过美国人拿到的捐款和这群人合计一起策谋搞死铜匠，当然不是真的搞死他而是让他装死而已，就是想要骗过美国人

按着电影的套路，这骗局不可能再得逞的，没错！穿帮了！故事到了这里基本就接近尾声了… 校长女儿孙佳走了，裴奎山周铁男和校长决定继续留下来维持这所小学 … 而张一曼大家都以为她疯了，其实她后来捡起那把手枪的时候她就不疯了… 她是彻底放弃了… 她想去另一个世界找寻她认为绝对的自由…

期间曲曲折折的小插曲让我看到人性的改变，是真的很可怕(๑ó﹏ò๑)也让我觉得善良真的是很重要！

人之初性本善，我相信他们最初也是真的想一起把学校办好的，但是因为最后的种种导致张一曼自杀… 导致周铁男懦弱… 导致裴奎山变得一切以钱看齐，这都是人性最丑陋的地方也是现实版

君子爱钱取之有道啊！就算想让学校要变好，也应该是大家一起想正道，凑工资或其他方法筹齐才是，而不是用这种方式，毕竟蒙骗就不是很光彩的事情嘛🙅

如果最开始选择和上级领导坦白，不要想着蒙骗过去，张一曼就不会死了！可惜啊人生就是这样没有如果之说，一步又一步的妥协往往不会带来好结果，还是要有正确的原则才是

为神马就不能善良对人呢，善良做人多好！环境会改变人没有错，但是人都能善良待人，不去伤害他人，用善良决定事情的方式，坚持对的原则，就像校长女儿一直都坚持自己认为对的原则，因为努力就会有好的收获

恶人自有天收！我们要做善良的人才可以化解所有的苦难！共勉…

在当今这个社会，关于亲子教育，亲子阅读等话题越来越倍受众多家庭和学校关注了，而我也其实一直都很关注这方面的话题

前些天有幸读到@huangzuomin写的一篇文章，”关于亲子阅读”的分析，这篇文章我一字不落的全读完了，感触特别深刻

在文章中@zooman 有讲到这么一段话：「在亲子阅读中，不单单给孩子传达爱——对生命的爱、对生活的爱、对知识的爱，我们做父母的也从那些沉沦的事务中，从忙碌的现代生活之中寻回了一片清净的天地。这样，忙忙碌碌的事情就可以少做一点，多一点时间和孩子一起享受生活，我们的寿命可以长一点，心情会愉快一点，社会也许会变得更加和谐一点」

当读到这段话的时候，与我心有戚戚焉… 我一直认为阅读对孩子的探索新知识很有帮助，对孩纸学习写作也非常有利！可是我只知道阅读的重要性而没有以身作则去正确引导他，说来惭愧啊

可能就是教育的方式做的不对，在孩子很小的时候亲子阅读陪伴的少了? 孩纸对读书是否热爱没有发觉? 营造读书氛围很重要，而每次都是夜深人静的时候… 等孩子睡了才会想着来看自己要看的书

手机电脑少在孩子面前耍，换成亲子阅读时间… 跟孩子在一起阅读，不是更好

工作生活因为手机或电脑占据过多时间，每天孤独地忙碌着… 却得不到心灵的安宁！

表面上孩子在阅读，可他读没读进去并不知道，因为发现每次读完书，孩纸只是到了规定时间就放下书本，好像没有感觉出他很热情地读这本书

阅读应该是跟呼吸空气样很自然而然的平常生活方式才对，就像每天穿衣吃饭一样的习惯… 而不仅仅是为了要读书而读书

像zooman在文章中说的那样，要让孩子在阅读中感知知识的力量，于心灵深处的感受阅读带来的乐趣

我要从现在起，试着zooman说的那样去努力改变现状，与孩纸共享读书时光，于我和孩纸一起感受阅读带来的快乐！

坦白说，我还真的想活长寿些，我对生活还有很多美好的向往呢… 把祖国的花朵培育好于己于社会会更和谐美好哒！加油吧(⑉°з°)-♡~~~
