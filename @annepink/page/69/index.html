每天都要弄饭给我家小主吃 ~ 还好小主子不是嘴叼的熊孩纸 ~ 随便我弄啥他都吃的爽爽哒 ~~😁




今天弄了三个菜 … 红烧黄丫头 … 还有麻麻种的青菜和小芹菜 ~ 小芹菜加了千张清炒

今天主要介绍下我做的红烧黄丫头哈 ~

黄丫头呢是我南昌的方言 … 它的学名叫黄顙鱼有关介绍太多姿势了 … 亲可以点进去了解下哈 ~

如果是南昌人没有不小的「黄丫头」🐠 其实我南昌所说的“黄丫头”，本来叫做“黄额头”，是说这种鱼它的额头是黄色的 …现在呢人们叫它黄丫头是以讹传讹的说法 ~ 学名是黄颡(sang)鱼 …古代有说法“颡”即“额”，“黄颡鱼”即“黄额鱼” ~~ 黄丫头”也有叫黄辣丁哒 … 因为这种鱼刺很少少 … 鱼肉又特别鲜嫩 … 用红烧的做法就更容易入味好吃 ~

网上了解资料后中医有介绍👉 这类鱼有滋补健胃 … 清热解毒 … 止嗽下气的功效 ，还有能利尿消肿的作用呢 ~

我家吃的比较多 …偶尔一星期熊孩纸会嘴馋想吃 … 念念不忘哒 … 所以周末就会烧给他吃~~



具体做法是~~首先主料：黄丫头 辅料：生姜、大蒜、小米椒、还有香葱或香菜都可以

1⃣ 炒锅中油要烧高热状态 … 再加入黄丫头鱼骚煎两面 …

2⃣ 再加入生姜大蒜小米椒煸香后 … 倒入料酒烧一下 … 再加入老抽

3⃣ 加入开水比较鲜香 … 加水要没过黄丫头 … 将汤汁反复浇淋在鱼身上以便入味 ~

4⃣ 中火烧开汤汁后盖锅焖烧8分钟左右 … 口味比较浓重的也可以多闷烧下哈

5⃣ 调味 … 加入盐，耗油，鲜辣粉

6⃣ 然后转大火收汁 ~ 起锅装盘洒上葱花🆗

黄丫头据说还含有叶酸，维生素B2，维生素B12等维生素营养价值还不错呢👍


瞧熊孩纸吃货德行撒~ 为娘我还没上桌他就开吃了…😅
