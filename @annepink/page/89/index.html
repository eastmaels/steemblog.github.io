没有现金生活的一年是怎样的体验

不带现金出门要是在10年前的确相对尴尬且被动了 …更是“天方夜谭”啊
那是没有互联网支付，就算带了手机也无济于事啊 …而如今 … 互联网已经成为了我们生活的必需品
“线上支付”也是一种新的生活方式了
手机支付的领域和范围也越来越广，大到成千上万 … 小到一元几毛 …非常的方便呀

不带钱完全可以生活，前提是要有手机 … 微信 …支付宝 … 网银 … 能支付的方式太多了 …还有一个前提，就是要在城市，到农村基本就行不通了

其实 …我觉得每年国家都要印刷很多纸币，也要销毁很多旧钞，钞票上有很多细菌我就不说了，印钞销毁也需要人工费和社会资源的浪费啊。如果有一天，我们真正能实现无实体币，就是绿色社会的巨大成就 …那该多好啊！目前微信 … 支付宝已然实现了大部分支付功能
最近熊孩纸学校交小钱有些频繁，比如前2天他说学校要义捐，最低一元 …我就要给他现金了，摸摸包里都没有块币了，更别说其他纸币了…找了半天只有百元鈔怎么捐啊…最后在他存钱罐翻到几个硬币…😅

所以呢 … 个人觉得一周不带现金，不出意外的支付可能还行，一年就难说了 … 还有看到医院 … 政府网点还是需要现钞和银行卡的，早餐店部分也是现钞支付，这都需要再努力

不过，遇到准备不足或手机没电或故障还是比较头疼的！

我就不稀饭现金支付… 就我前面说的…主要原因很多细菌啊

保险小知识💁早知道(6)

问：我买了百万医疗保险

可以不买长期的重疾险了，不然 … 这两者重复了 …是吗？

答：有区别哒

1.最主要的区别是：百万医疗保险是短期保险 … 保障随时可能中断，而重疾险是长期保险提供终身的保障

2.百万医疗保险是遇到重疾先交纳医疗费用后采取报销式理赔 … 是报销型，而重疾险是确诊即理赔 … 是给付型

3.期交重疾险费率每年一致，短期的百万医疗保险每年保费会发生变动
