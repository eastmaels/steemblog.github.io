记忆中的小吃

有人说吃辣会呼吸的痛…但我就是介么无所畏惧ya!!所谓无“辣”不欢的人生才够味啊 …哈哈!

这个小零嘴叫辣椒饼，有小时候的记忆，只要稀饭这种辣味的 …一直都是南昌人特别熟悉的口味了 …

我一直印象深刻地记得，读书时的校门口就有这个辣椒饼卖，大概是附近的居民吧，放学时就会聚集在那 … 然后很多学生都会去买来边走边吃着 … 那种被辣的感觉好好喔!

时光匆匆啊 ! 许多年以后至今…这个从小就吃到大的味道还一直流传到现在，只是再也不是1元钱就能买到的价钱了!现在要15块钱才能买得到，就那么一丢丢(如图中 ) …😂

然而还是很多的人会去买，比如像我这样的吃货啦✧٩(ˊωˋ*)و✧就是很稀饭吃这个垂延欲滴的辣味啊!😋 😋 😋再配上我稀饭滴德亚酸牛奶简直绝配啊 …那个感觉太爽啦!😁

再来分享它的做法吧!辣椒饼的做法其实也很简单，就是用面粉、糯米粉、辣椒、盐和糖糅搓在一起之后，上锅去蒸熟，待放冷了之后切成想要的形状，就可以吃了

最早在记忆中辣椒饼是一大块的，再分成六小块或是长条形的排成一排，如今 … 经过改良后的辣椒饼更受到大家的青睐啦!
