怨曲虽名为怨曲 … 但有种冲出黑暗再放光的气势 … 听起来像是在表达跌到谷底又如何？大不了摔一身伤 … 好了之后还继续前进 …家驹的音乐 …很多都很励志，绝望之中给一丝希望 … 很多歌在无奈的表达后 … 总是先抑后扬 … 不被击垮的精神💪 … 每个人听过应该都会有不同的感触。前面开始音乐气氛有点悲凉 … 渐渐由悲伤进入高亢 … 劲 …不服输地精神，继续自己坚持理想到最后 …
