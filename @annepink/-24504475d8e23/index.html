我是属于刀子嘴豆腐心 心特软的那种 …心也善良

我交盆友…以实诚为本 可就是刀子嘴、豆腐心 不知用什么良方能把我暇痴改正？

如果心里有什么不愉快事情该说的是要说出来的 说过了…说完了…说开了…就没事啦 盆友们都说心直口快的人 … 么心么肺滴 … 心地特善良

我为人厚道不坑不害 有时候爱报打不平 … 不为名利 …但求无愧

缺点有时没有时间观念 … 贪玩 …有时爱忘事 不能太严格要求自己 老认为不要给自己上扣 … 让自己放松下 我可能属于男人性格 …豪爽 只要交的盆友从不弃知 …不删除 除非人家不愿意交删我 … 但我会一直保留 除非手机丟了 … 这是我常想人好所至 虽然不再联系 …我心中有 人与人的性格 … 环静有关 总之言之盆友就像我的亲人一般

这辈子 …最对不起的便是自己 还有我的心…也对不起我的爸爸妈妈 … 因为自己的冲动 单纯无知 … 毁了自己的一生 … 现在的自己好痛苦…

你认可吗 … 你信我吗 …
