说起这首歌要感谢 @windowglass糖糖哈~😚🌹

昨晚在全民看见她和 @tanlikming力名歌合唱了 … 才想起原来我也有合唱喔😅

记得那时候合唱这首歌时还闹过笑话呢 ，某人说我是粤语鸟，意思说我粤语不标准，我感觉特好笑自己🙈 唱到后面想到这话忍不住笑si我了…😅 没事 ~不怪他怎说，我会努力练唱的 … 从小就爱粤语歌的我觉得好听好玩啊…😎

言归正题哈，「大地」这首歌是献给父亲的歌，但又不仅仅是“父亲”的代名词，更是我们所居的“华夏大地” 这首歌里好像是有另一种家国情怀吧 …每一句歌词都那么的意义深刻!!

父爱沉沉 … 如默默无言的大地般 … 我的爸爸就是一位默默父亲 ~时常犯错的我被他一声大吼就会吓哭的眼泪哗啦啦流 … 但是呢，过后他叫我一句‘‘萍’’我就会好暖心感动 … 我的了解是，爸爸心情好时就是这样爱呼我的~嘻嘻

时光匆匆啊 …爸爸已是满头银发了… 但看见爸爸可爱哒面孔 … 心中永远都忘不了曾经潇洒英俊帅气逼人哒好爸爸… 🤗

谨以此歌大地献给天下所有的好爸比💗 感谢亲们聆听支持呦😚
