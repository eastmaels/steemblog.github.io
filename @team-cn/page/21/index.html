【新手村会客室#21】CN区三兄弟之一爱好英文的二哥@bring

在CN区，除了有@softmetal 软哥和@pgr 小P 这对欢喜“兄弟”外，还有一对真正的兄弟。

他们是大哥@wnh518518，二哥@bring 和三哥@xiaoshancun

上一期我们认识过过三哥@小山村了，这期让我们认识一下二哥@bring。

因为来到社区的时间比较短，加上二哥最近并不是很活跃，对他的了解并不是很多，唯一对他的印象是一位很爱好英文的前辈。

现在让我们来了解一下二哥@bring吧～

 

问：您居住在哪里？

目前的居住地也是我的出生地，并非“父母在不远游”，也并非故土难离，只是种种条件有限，上中专时才去了省内的另一个城市，此后的工作生活一直就在现在这个城市。

泰安，寓意“国泰民安”，山东省中部的一座小城市，也许很多人并不知道这个城市，但若提起泰山（Mount Taishan），您一定久闻大名，泰安市就是泰山所在之地。无论是在我的家里，还是办公室里，都能清晰地看到泰山。生长于中华民族象征的泰山脚下，倍感荣幸和自豪。

顺便说一下，泰安市有两个火车站，一个是老站，就是泰山站；还有一个是高铁站，那是泰安站，容易混肴。我曾见到有外地的游客，拿着泰山站的车票到泰安高铁站乘车。

问：您的职业?

我初中毕业后考入了一所纺织工业学校，学的是针织专业，毕业后顺利进入了一家国营大型棉纺织企业，从事的却是计量工作，也就是现在的技术、质量监督。可我一直喜欢的是英语，尽管只在初中、中专学过不长的时间，大多还是靠自学。企业效益江河日下，在考研几次未果，孩子上了小学以后，我便辞职应聘去了一家外贸公司，算是开始发挥自己的专业了。半年后，为稳重起见，考入了现在的政府机关，从事招商引资工作。这本身就是一个挑战性很大的工作，而且我的英语也渐渐有了用武之地。

身在体制内，可我连自己也觉得不像官场中人，只是不忘初心，至今还在埋头学习自己喜爱的英语，期盼着离岗后从事自己喜爱的英语方面的工作。

 

问：希望大家怎么称呼您？
这个倒无所谓，熟悉一点、比我年轻的会叫我二哥，因为我排行老二，是@xiaoshancun的哥哥。更多的倒是喜欢称呼我为老吴，年过半百，已过了知天命之年，称呼老吴倒也顺理成章，就连我的女儿也常常称呼我为老吴。

 

问：什么原因让您加入Steemit？

哥哥@wnh518518是最早加入Steemit的，2017年夏季的某一天，哥哥指着上面的一个帖子说，在这上面发表自己写的文章就能有收益，还是美元，让我试一下。我笑而不语，哪有这样的好事？天上会掉馅饼？我懒得关注。后来是哥哥的现身说法，让我明白并非空穴来风，便抱着试试看的心态准备加入其中。

 

问：您有什么爱好 ?

我一直是一个比较传统、守旧的人，不太喜欢新生事物，更不会冒险，也不只是年龄上的缘故。中规中矩，板板整整。不嗜好烟酒，不愿热闹，倒是能够静下心来攻读自己喜欢的英语，搞点写作，闲时研究一下中国象棋。这就是我屈指可数的几点爱好了。

从初中开始接触英语算起，迄今我已研究英语三十多年了，可惜一直不精。自学考试取得英语专业大专、本科毕业证书，获取学士学位后，曾立志考取英语专业的研究生，也未能如愿，只是初心不改，至今还刻苦学习英语，乐此不彼。业余时间曾为几家翻译公司做兼职翻译，也曾在做到网翻译网站从事翻译，英中升至最高级10级，中英达到9级。相信，没有白学的东西，终有一天会有用武之地。

中学时便开始记日记，也开始尝试着写点东西。非常欣赏作家赵树理说的那句话，当你觉得肚子里有话非说不可时，你便有了写作的材料了。无病呻吟，空洞无味，对别人对自己都是一种折磨。材料的来源便是你的经历、感悟，多接触社会，体验生活，多看书，才会有话可说。

中国象棋也是兴趣使然，跟别人学会基本的走法后就是自己摸索了。我曾研究过几本棋谱，也在网上与电脑和人进行实战。在棉纺厂年度中国象棋比赛中获得过第6名，在全区第一届中国象棋比赛中获得过亚军称号。现在，有时还在联众上杀上几盘，曾升至三级大师系列。

 

问：您主要分享什么类别的内容？

我写的东西大多是我的经历、所见所闻、所感所悟，还有一部分是关于英语方面的。事非经过不知难，经历过了才会有感而发，才会深深理解，才会有话可说。也可能，您看到我的帖子有那么点感伤，也太凝重，这或许与性格有关。我也会尝试把自己听到的喜欢的笑话趣事分享给您，博您一乐。Steemit上人才济济，藏龙卧虎，多才多艺者大有人在。我愿量力而行，积极参与各种活动，与您分享我的故事，也更愿聆听您的故事。

 

问：您想什么类型的朋友关注您？

有朋自远方来，不亦乐乎。三人行必有我师焉。无论来自哪里，也无论背景、年龄、性别、身份、爱好如何，既然为了一个共同的目标，走到了一起，我愿成为您的朋友，与您共享Steemit给我们带来的乐趣，共同走向美好的未来。

 

问：能谈谈您的 Steemit ID 由来吗？

虽然急于加入，可惜那时注册还很难，前辈@myfirst一哥慷慨相助，把他在2016年10月注册的@bring这一账号赠送与我，便有了我的现在。所以，尽管我的主页显示加入时间是2016年10月，实则是2017年7月。

@myfirst 一哥不知何故离开了Steemit，除了感激，也很想念。我也曾探究过bring 一次的含义，那大多是美好的意义。

 

问：还记得您刚在Steemit起步时的情况吗？

刚加入Steemit时，我信心百倍，情绪高涨，一日至少一篇文章，有时会达到4篇。那时正好赶上Steemit的大好时光，更幸运的是得到前辈@tumutanzi @myfirst @rivalhw @lemooljiang等等的厚爱，声望值迅速升至63级，也收获了我在Steemit上的第一桶金。不到半年以后，形势大转，人来人往，来来去去，Steemit渐渐变得萧条起来，直至今日也未能回过神来。

挺怀念那一段时光，只是花无永开日，凡事不可能只朝着一个方向发展，自我努力，等待时机，才是上策。

 

问：您想过放弃吗？

说实话，我曾有过一段时间的放弃，币值大跌，收益大减，我在2018年一整年的时间里，声望值只增加了0.3，发表文章也是三天打鱼两天晒网。可现在我又回来了，以一种更加平淡的心态而来。慢慢翻看着自己的所思所想，不断锻炼了自己的写作能力和英语水平，还会有收益，哪里有这样好的事情呢？

 

祝愿Steemit越来越好！祝愿中文区越来越繁荣！

以下是本期访谈实录：

steemit上的大神们都很让我敬佩，刚来时候， @tumutanzi @rivalhw 还有其他给我点赞的人（我就不一一列举了️），我都很感谢。

(能够一切都很平淡，原来小花的道行都不浅)

我想交朋友呀，没有找到steemit朋友。

学霸不但数学好，写文章也一流；

就像拥护共产党一样，要以龙虎豹为中心，坚决拥护村长的领导，村长一发话，大家干就是了。（没给team-cn投票的，抓紧时间去办啊！）

About @team-cn

@team-cn was founded by @ericet @honoru and @softmetal back in April 2018.
You might have heard of @steemonboarding, which is a new project to onboard new users to steem.
@team-cn is similar to @steemonboarding, but we mainly focus on boarding users who speak and write in Chinese. Unlike @steemonboarding which is still in really early stage, @team-cn has been running and helping new users for more than 300 days! It’s currently the largest and the most active CN community on steem.
During our 300 days, we created many new projects to better help new users:

  • @cn-activity, a project which aims to summarize the contests within the steem community daily. It helps many members to participate many contests.
  • #teamcn-homework, assigns varies of tasks to help newbies know steem better. Those who complete these tasks every week are getting rewarded.
  • @cn-voice, a project which hosts song singing/storytelling contest weekly.
  • 24/7 365 days Wechat group, since discord is banned in mainland China. We use Wechat group to communicate with members and answer any question they might have.
  • Many fun projects like team-cn snack shop @teamcn-shop, Newbies Graduation Ceremony, and Auto Upvote Service for Members, etc

For more detail, you can checkout this post: Introduction of @team-cn

@team-cn is here to help them get the target users they need.

【新手村同学会#18】一灯 @sasaadrian

一燈@sasaadrian應該是新手村上史上第一位將今次的訪談視頻拍成影片，以行動鼓勵大家多嘗試使用視頻方式，分享自己的日常或創作等等，希望大家可以將內容中得到一些靈感或啟發，也感謝一燈接受这次的採訪。

大家也可以了解一下刚加入steem时的访谈：【新手村计划#68】@sasaadrian 一位来自香港的环境保护的攀树技师

你在steemit上最感谢谁？

當然是村长@ericet 多谢村长一路的提点，村长对新人真心关照的。

(村長絕對是多位新人們的好幫手，一路以來為新手村默默付出，希望新手村在2019年繼續成長 !! )

你在steemit上有什么难忘的事情吗？

最难忘的应该都是近排的视频活动，因为需要拍视频，做运动是自己觉得必须要做的，配合了拍视频活动，就变得不能偷懒了。而且我觉得在视频上表达自己比写文字更加容易表达自己。

(對阿，拍視頻除了可以讓各位監督，不能作弊或偷懶外，還有機會得到 dute 大神們的點讚，大家如果是新加入 Steemit，又本身熱氣運動，建議也可以拍視頻，跟一燈 @sasaadrian 學習一下 !!)

你在steemit上有什么遗憾的事情吗？

什么都慢，我看文章太慢了，我写文章更慢，我讲说话也都不快，我自己的表达能力比一般人差，大家很少见我留言是因為我看完回答你們，你们已经去了月球，我仍然停留在地球，仍然停留在电邮回复的时代，我一般对面书，微信这一类的社交媒体都追不上也比较少用，又不知道怎麼玩微信红包。另外就是户外工作太过劳动，人一疲倦很难集中去做一些细微的思考尤其是写文章，所以用Steemit这社交媒体对我来说是一个大挑战。

事實上，很多村裡的用戶們都需要兼顧正職，包括小編也是，要在下班後，再花時間籌碼，寫一篇文章分享，的確是一個很奢侈的事情，所以能夠繼續堅持，相信某程度都是信仰吧 !! )

你在steemit上有什么收获？

在Steemit上面最大收获应该是认识到些朋友，比起其他平台，Steemit平台是最容易识朋友的，以我社交那么差劲为例，都可以认识很多朋友，尤其是米高 @honoru ，米路@minloulou ，拉仔 @davidke20 ，茵茵 @yanyanbebe ，小迪 @tydebbie ，廖神 @htliao 等等大家都是讲广东话的，感觉很亲切不会孤独。(沒办法因为自己普通话又不行英文又差)

(哈哈，說廣東話的好像遺漏了我，一燈說得對的，在新手村，群裡無論何時何地都很熱鬧，所以只要你願意跟大家交流，相信並沒有孤單的可能。)

你怎么看待新手村访谈这个项目？

社交平台吖嘛，当然很重要啊，有了这个访谈项目大家先互相容易了解互相认识大家。这个就是同其他平台不一样的地方。

(很讚同一燈的說話，訪談對於新人或舊人來說真的很重要，畢竟大部份用戶都是素未謀面，通過訪談大家才會更容易認識大家。)

你有什么建议给@team-cn或者CN区的？

我都建议大家可以尝试推动下用短视频，我只知这个是一个大趋势如抖音，IG，whatsapp，面书，17媒体，拍拍等各大平台都加插了玩短视频，限时动态等等。唱歌活动是一个好的开始，虽然我不知道怎麼玩，不过如果有时间我都会试下。

(都是好事，畢竟拍影片現在都很方便，唱歌比賽可能對不懂唱歌的人來說有點困難，才藝表現又如何，因為村裡每一個都身懷不同絕技，絕對可以通過視頻比併一下。)

你对后面加入的新人有什么建议吗？

新加入的朋友最紧要认清楚，这个平台是同其他的不一样，这个平台有很多不爲人知的方法才能做得好，所以一定要先互动交流，听取大家的意见，互相鼓励互相帮忙互相推动才有机会做得好。

(這應該是 Steemit裡新人的生存之道，不過太多新人，由於對平台缺乏認識及沒有支援下離開，這是很可惜的，所以如果你是剛加入 Steemit，而你又正面對對於平台不了解的煩惱，建議多交流聽取大家的建議，自然就有機會有一個更好的收益反映在你的帖文上。)

可以分享让你印象最深的活动吗？

也当然是一分钟平板支撑和俯卧撑22kill活动了，这几乎是一个转捩点，因为这是一个长期作战的活动，大家都会互相鼓励所以就多了互相交流，久而久之就加深了互相认识。

(這一個絕對是屬於我們村裡的創舉，一分鐘的平板支撐和俯卧撐22kill活動，除了帶動中文社區的人參與，就連國外人也收到感染，前來響應此活動，可見運動真的無分國界 ! )

若日后有机会办活动，你会办一个什么样的活动？

有机会的话我希望办一个跟视频有关的活动，最近我认识了一个字眼[Vtuber]在日本冒起来，V是虚拟的意思，是把真人转化成3D的虚拟人物就如(初音)这个虚拟明星一样，真人不需露面都可以成为Vtuber，我觉得这种技术很快可以接触得到。

(這個類似在網絡的世界創建了一個虛構的自己，之前小編也聽過，有些人可以訓練虛擬世界的自己，在求職時，可以利用虛擬世界的自己去面試，去買衣服，不用親自去試，可以用虛擬的自己穿上該衣服，再決定是否購買等，只可以講科技愈來愈發達，未來人類的功用可能大部份都會被科技取代。 )

使用Partiko 有甚麼感受？

一流，非常畅顺，一开始便使用到现在了，最近发现了一个功能非常强，打开回复的时候，自动跳去自己回复的位置，几百个回复位置不需要手动去搜寻了，真的很方便。

( Partiko 只可以講，只要你一試，就保證難忘，回復留言的新功能，確實又是一個為用戶們窩心的改進，希望日後會見到更多的窩心新功能推出，讓用戶們受惠。)

您認為Partiko 能否幫助 Steemit 的發展？

这个绝对是一个很大的发展，移动介面一定是最重要的一环，IG那么受欢迎是因为它够畅顺，而且有很多有趣可爱的编辑功能，只要做到好玩就自然多人使用。

(說得對，現在要搶佔用戶市場，移動端非常重要，畢竟現代人大部份都是手機不離手，當然界面設計、功能及流暢度等也很重要，不過小編也認為，往移動端發展，絕對會幫助到 Steem 日後生態上的發展 。)
