【新手村会客室#18】CN区老前辈@wongshiying

@wongshiying 在2016年就加入steem了，所以他是我们的老前辈。

这位老前辈之前沉寂了一段时间，最近开始玩steemmonsters的游戏的时候，豪投$$$购入卡包。

我被这土豪行为给震惊了，所以就把他抓到了贫穷的新手村，毕竟村里穷人太多了，需要土豪来支援一下。

这位土豪还没有村名，欢迎大家给老王起个村名~

大家来欢迎土豪@wongshiying~

 

问：您居住在哪里？
我住在马来西亚的砂拉越的诗巫。课本上教导我们，砂拉越是马来西亚的一个州，实际上我们应该是属于联邦的。

问：您的职业?

我的职业是眼镜店销售。目前也在马来西亚国家银行注册为合法财务顾问，也是马来西亚证券委员会认可的财务规划师，提供各种财务咨询，保险咨询，退休，遗嘱等服务。特别要解释一下财务规划顾问，我们不代表单一公司，我们代表客户，我们提供不同公司的服务，让客户可以更全面，更公平的比较不同公司的产品。

 

问：希望大家怎么称呼您？
称呼我的名字就好。我叫势颖，也有人叫我Jan，那是我的Christian name 的缩写。

 

问：什么原因让您加入Steemit？

加入Steemit 的原因是单纯好玩。当年也是第一个分布式的社交平台，贴文分享还有收入，相当有趣。可惜我不是一个很好的写手。

 

问：您有什么爱好 ?

我喜欢户外活动，看看大自然，拍拍照。无奈生活繁忙，许多东西都落下了。

 

问：您主要分享什么类别的内容？

我主要分享我的家庭生活，和一些我觉得我喜欢的东西。特殊的东西我比较少分享，是因为我担心读者们的误导。可能以后会提供一些案例分享吧。

 

问：您想什么类型的朋友关注您？

什么样的朋友都可以。大家互相认识。

 

问：能谈谈您的 Steemit ID 由来吗？

我没特别去想Steemit的ID，纯粹用自己的名字。

 

问：还记得您刚在Steemit起步时的情况吗？

起步的时候，算是满轰动的（在币圈啦）。那时候有不少大佬加入，投入几十万，几百万都有。那时的文章是天价，四位数，五位数不在少数。后来改版以后，奖励机制的改变，慢慢变成两位数，三位数。

问：您想过放弃吗？

我其实没想过放弃这问题。我虽然不常出现。可是偶尔还是会看看到。

 

问：您是怎样走出起步的困局的？

那时候，就有什么些什么，分享一下小东西。多跟别人交流，看看别人的文章。后来加入了中文社区，大家可以更好的交流。

 

问：您是怎样看Steemit的发展?

Steemit应该会越做越好，只是目前使用者还是偏少了。原因不明，可能还是因为普通人对区块链的应用不太认识，法律对加密货币的限制，让大家不敢恭维吧。其实我玩Steemit 也不是为了赚钱，就有就有吧。因为我也不是靠写作吃饭。纯粹的自己喜欢

【新手村计划#85】@chloe1 一位来自天津的学生

在CN区新帖闲逛的时候看到一篇非常理科的自我介绍文章，看了半天也看不懂，但是写作倒是蛮认真的，就把阿扣@chloe1 拉进了新手村。

 

问：@chloe1 您居住在哪里 ?

现在在天津，一个煎饼果子满大街，狗不理包子没人吃的地方（除了游客会尝鲜）


问：您的职业？

职业，嘻嘻嘻，我没有职业哟，暂时还是一个没皮没脸的的啃老族（捂脸ing）

 

问：希望大家怎么称呼您？

中文名馨馨、卉卉，英文名Sandra、Chloe都可以，更希望希望村长赐名！

 

问：什么原因让您加入Steemit?

加入Steemit其实是一个偶然。那是一个风和日丽的上午，我坐在电脑前打算研究研究如何科学上网，在国内想要上ins看看漂亮的小姐姐和帅气的小哥哥们，后来偶然看到了一个steemit的链接，点进去妙笔生金，但当时自己脑补成妙笔花生，在想这是个什么吃货网站么，后来就开始慢慢接触了。

 

问：您主要分享什么类别的内容？

会分享一些我平时收集到的各种小诗，分享记录我的生活，为自己的老年回忆录打稿做准备 ，偶尔的游记等等

 

问：您想什么类似的朋友关注您 ?

什么类型的都可以，有朋自远方来，都很欢迎，希望我们们有很好的互动。我身边几乎都没有学习金融的铁朋友，所以听到区块链、虚拟币这种高大上的名词时，就像我妈听到人工智能扫地机器人一样会两眼放光。

 

问：您有什么爱好 ?

很多爱好耶~喜欢看小说，听故事，喜欢和有趣的人聊天，喜欢学习各种稀奇古怪的我没接触过的知识，喜欢音乐，美术，建筑，喜欢脑洞，喜欢游泳，喜欢各种美人们。

 

问：能谈谈您的 Steemit ID 由来吗 ?

这是一件有趣的事儿，和马爸爸有关。接触steemit后没有立即注册。有一天晚上我和一个朋友（不是中国人）在聊天，他在Aliexpress上买了华为新发布的手机，我说这是个啥（原谅当时见识短浅的我只懂淘宝Taobao。其实它俩都是阿里的），他说淘宝在国外用不了，我和他开玩笑说你去问问马云到底咋回事儿。他回了一句”Who is Mayun”. 我一边震惊一边和他叽里呱啦的安利马爸爸的各种耀眼标签，后来他一本正经的说”His name is Jack Ma,not Mayun,it’s his real name”。我当然不肯，行不改名坐不改姓他真名就是马云。我说今天我叫Sandra（惯用英文名）,明天我还可以叫Linda、Lucy、Chloe等等。一段激烈的引经据典文化打架争论谁是与非后，以”Please call me another English name Chloe tomorrow. Thank you.”彪悍地赢得了胜利。后来在注册steemit的时候就用了Chloe1,数字是代表我在steemit的第一个账号。

 

之前一直以为阿扣是男生，看到照片才确定是女的～

【新手村会客室#17】快乐女孩 @meixia

每次看@meixia美霞的帖子感觉她是一个快乐，幸福的女孩。

由于最近开始在教育机构上班了，在steem上发帖次数减少了，但是她还在的，不要把她忘记了～

 

问：您居住在哪里？
目前居住在广州

问：您的职业?

教育机构上班哈哈

 

问：希望大家怎么称呼您？
霞妹吧～因为我觉得自己没长大

 

问：什么原因让您加入Steemit？

刚来Steem是朋友@vivia介绍的，她说有个网站可以写文章赚钱，所以就加入啦～

 

问：您有什么爱好 ?

我大学时候的爱好是打篮球，工作以后，就没有碰过球了现在的爱好是工作，带娃……

 

问：您主要分享什么类别的内容？

我平时都是分享生活中的所见所闻所感之事，我是一个比较细腻的人，所以生活中都有很多素材，让我去写。

 

问：您想什么类型的朋友关注您？

各种类型的朋友都可以呀，朋友不在多嘛

 

问：能谈谈您的 Steemit ID 由来吗？

我的Steem ID就是我名字的拼音，哈哈很随意取的，主要我很喜欢自己的名字啦

 

问：还记得您刚在Steemit起步时的情况吗？

刚开始来Steem也是自己默默一个人奋斗，走过没人点赞的日子，后来我的老师大诗兄，中文区大佬大伟哥关注了我，经常给我点赞支持我，慢慢地自己也走出了困局，越玩越得心应手，感谢一路支持的朋友们呢

问：您想过放弃吗？

其实我对于Steem一直挺佛系的，所以不存在放不放弃一说，只是觉得有个地方可以说说心里话，写写生活感悟，挺好的。只不过现在工作忙碌起来，没空写了，不过一个月还是会写几篇，记录记录生活。

 

问：您是怎样走出起步的困局的？

起步困局要找对群体，现在新手村就很棒呀，村长也很乐意帮助新人，这样新人也更容易在steem站稳脚跟哦。

 

问：您是怎样看Steemit的发展?

我对Steem研究不深，所以我希望Steem越来越好，最近一直都在寒冬状态，春天就要来了，马上就会柳暗花明啦 坚持写下去吧～
