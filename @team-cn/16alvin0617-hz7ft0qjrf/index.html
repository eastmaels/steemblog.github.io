【新手村同学会#16】小红 @alvin0617

嗨~各位亲们好呀

本期童鞋会邀请地男嘉宾是非常优秀的@alvin0617红红童鞋

他是个积极向上 热情阳光的男孩 他也是个运动达人喔…plank做的非常给力
有兴趣的童鞋可以去了解他的视频展示哈~ 而且新手村的每一位小伙伴都非常喜翻他 在新手村群里无人不知无人不晓啊
甚至在steemit上的大佬也很眷恋他喔~
经常看见大佬给他的文章点大赞
红红童鞋的文章写的也确实很棒~值得大家关注喔

我是@annepink萍萍~很开心我能得到村长的认可当一次主持人~嘿嘿

在本次访谈之前 大家可以先点开链接了解下红红加入steem时的情景:【新手村计划#75】@alvin0617 一位来自台湾现香港大学的交流生


以下是本期访谈实录：

你在steem上最感谢谁？

最感謝的當然是村長了 ，能讓中文區的大家有個這樣的群組可以互相交流及認識，使得steemit在華人地區的推廣達到一個正向循環的作用

(嗯是的…村长大爱新手村的每一位好伙伴 )

 

你在steem上有什么难忘的事情吗？

難忘的事莫過於剛開始認真經營steemit的第二個月時參加一位大佬的代理比賽最後順利地得到了500SP代理，對於當時的我是十分的振奮人心的 也因為這樣開始更活躍於Steemit上

(红红真的是太幸运啦
小伙伴们都很羡慕你呀~为你高兴喔 )

 

你在steem上有什么遗憾的事情吗？

其實目前沒有發生什麼遺憾的事，因為是較晚期加入的，所以還沒有經歷過認識村民是默默從steemit上淡出的憾事 不過還有一點是想再多多推廣給台灣的朋友們，目前很少台灣人在玩Steemit對我來說是比較可惜的事情

(喔~这个红红可以去努力推广一下…说不定有一天来了一大群你的台湾老乡呢… )

 

你在steem上有什么收获？

對我來說收穫有好幾種!除了因為參加很多活動不管是村內還是村外因此認識很多來自不同國家不同職業的朋友以外，也因為更了解STEEM區塊鏈，讓我更認識整個STEEM上的Dapp生態圈，再來就是因為有時候會寫雙語文章，所以也漸漸地對英文書寫不會這麼生疏，一點一點的在進步!

(是呀~~大家同感…来steemit玩会收获很多自己意想不到的惊喜 )

 

你怎么看待新手村访谈这个项目？

新手村訪談有別於一開始的新手村自我介紹，這算是個人在整個新手村計畫中第二次出現在大家面前發表一些感言，除了可以讓大家更認識到自己在STEEM上的心路歷程之外，對我來說也是一個很好的方式去檢視自己這幾個月來玩這個社交應用的心得以及心境上的轉變。

( 嗯~红红说的没错
每位小伙伴都很喜翻访谈这个项目~大家又多了一次互相了解的机会…彼此的友情更深刻啦 )

 

你有什么建议给@team-cn或者CN区的？

其實一直以來都很感謝在這個新手村當中貢獻許多的大腿們，米高@honoru 村長@ericet 拉仔@davidke20 等等，說建議實在沒有那個分量哈哈，已經很好了!

(对~新手村就是好的没话说了 暖暖哒… )

 

你对后面加入的新人有什么建议吗？

建議當然就是要乖乖的做新手村作業哈，還記得因為一開始看了作業內容發現完全看不懂於是便一頭栽入研究，發現STEEMIT上很多好玩的操作，比如代理、機器人點讚，帖子買讚等等，對於自己在STEEMIT上可以長期發展也很有幫助，不懂的多問問新手村的大家，一定都能得到回覆的，村裡有很多善心人士哈

(是呀~~大家都要做新手村的好学生…才能玩的得心应手哦~加油 )

 

可以分享让你印象最深的活动吗？

最有印象的活動莫過於茵茵，迷路辦的 plank-contest的活動了哈，是我們新手村的驕傲!因為這個活動受到了dtube上很活躍的Nathan同時吸引很多來自不同國家的Steemian一起加入我們一起做這項運動，我想這個影響力也是大家當初沒有想到的，這就是STEEMIT上令人著迷的部分吧，目前很多人都已經完成了!我也得趕緊加油把先前的部分補上

(嗯~红红加油!!茵茵和迷路一对神仙眷侣办的活动已经红遍CN区啦 不但爱护所有参加的伙伴 更有惊喜奖励呢 )

 

若日后有机会办活动，你会办一个什么样的活动？

我打算等我60級之後來辦一個活動哈哈，細節還沒有認真想過，不過會以分享大家的生活為一個方向吧，讓大家只要簡單的分享自己的日常就可以來一起共襄盛舉。

(红红棒棒哒!! 即将60级大腿啦 期待你的好活动喔 )

 

使用Partiko 有甚麼感受？

是一個非常流暢的app, 很感謝大灰灰創建了這一個應用讓STEEMIT真的無影無形的存在我們的生活當中，畢竟現代人或許人人都有一台電腦，不過卻更無法離開手機，這個app確實提升了大家使用STEEMIT的頻率。

(是的~感谢帅灰灰😎 partiko是玩转steemit最佳工具 大家都爱不释手呀🤗 )

 

您認為Partiko 能否幫助 Steem 的發展？

我相信村里有很多人甚至是希望大灰灰直接買下STEEMIT成為這個社群平台的老大的哈，Partiko在STEEMIT上的發展大家也是有目共睹的，而這一個手機端應用我想也確實能更方便地將STEEMIT推廣給社會大眾。

(帅灰灰听到了吗~大家都期待你成为最佳大佬喔
partiko是所有华人骄傲
一定会有越来越多的fans)
