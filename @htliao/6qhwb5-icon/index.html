「區塊鏈投資 」— 國家級區塊鏈項目–ICON

奕風簡介：
擁有傳統大學三個學位，深知道區塊鏈將會改變世界，故開展不同類型的加密貨幣和ICO的生意和投資，現時為數個世界級ICO項目的advisor，擅於分析加密貨幣市場和區塊鏈項目。

Facebook專頁：https://www.facebook.com/cryptowithwind

Source

今期為大家介紹的是韓國的頂級區塊鏈項目ICON（ICX），ICON是一個由各種獨立的社會以區塊鏈連接而形成的去中心化網絡，ICON已連接了韓國多家銀行，證券公司，保險公司，醫院，大學，電子商務等所屬的多種社會。以此，區塊鏈ID，支付與交易等分佈式應用（DAPP）服務也可以被跨領域使用。ICON的創新之處為核心網絡Nexus，通過多個通道連接不同的區塊鏈和其他設備，具有非常好的拓展性和互聯性，未來可見的是，會有很多社群機構連入ICON，形成這樣一個系統，互聯互通，便捷高效。

Source

團隊方面，韓國ICO都是大財團的模式，ICON的母公司旗下有Dayli Financial Group，Yello Financial Group等，均是知名的大公司，而另一團隊成員Paul Veradittakit是區塊鏈初創投資專家，不僅是ICON項目顧問，還是Origin、Orchid Labs、Enigma MPC和Cofound.it這些項目的顧問，這些項目的表現均十分驚人，團隊中更有數字經濟學之父的Don Tapscot教授作領導，為團隊加添不少分數。

總結來說，ICON韓國的第一個公鏈，擁有國家級區塊鏈的團隊，並以大財團主導，其優勢是好處是自帶流量，前期建立生態系統相對容易。ICON是韓國區塊鏈的基礎設施。如果項目順利落地，ICON必定起飛，以現價來說，ICX絕對是超值，在公鏈中的價值亦被低估，讀者們不妨留意。
