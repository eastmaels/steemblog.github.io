$5 STEEM?我不久前的預測竟然這麽快就成真了！

Source

一個多月前，我寫了篇名為《STEEM的價值有機會在2018年騰飛嗎?》的文章，那時STEEM還只是1刀多點，沒想到新年才剛過去了一天多，這預測就成真了，我們看到了五刀的STEEM！真是一份十分棒的新年禮物呢。像上個月另一篇寫的文章裡介紹的NEO在今日也有十分不錯的表現，令我感到十分欣慰呢哈哈。這些預測看似神乎其技，其實這也只是結合了基本面和技術面來推測最有可能發生的情況而已。

你問我的話，隨著SMTs的推出和發展，這說不定只是個開始，STEEM在接下來的一年裡漲到十刀以上也是很有可能的，但也不排除短期內可能會回調一點點啦。

從我加入STEEM開始，了解清楚它背後的機制後我就沒有懷疑過它的潛力，沒有信心的話我也不可能堅持每日寫作到現在，還不斷Power Up呢。你可能會說你寫的文章現在值不了多少錢，可是過去你能預計到STEEM和SBD能漲到現在這樣嗎？而且從STEEM上的閱讀、 寫作和社交你還能得到很多金錢以外的寶貴東西，而這就只是要花你每天一點時間而已，這根本就可說是天上掉餡餅的事情了。如果你是Steemit上的新人的話，你只要繼續努力，有空的話多動動腦筋，我想你在這裡一定能獲得遠比你能想像的還多的東西。

希望大家喜歡我有關各種加密貨幣的介紹和分析，覺得還有點用或者能學習到些什麽我就很開心了。 :) 謝謝各位支持我的讀者，我會繼續努力為大家帶來這方面的資訊的。最後祝大家新年快樂！ :)

我的2018年個人計劃


Source

感謝@aaronli
（我就不說你肥了）， @mrspointm， @krischy和 @wanderwithtwo的提名，時光飛逝，不經不覺就已經年尾了，2018年是頗值得期待的一年，我這樣說不只是因為我覺得我主力持有的NEO和STEEM在2018年會有好表現啦，就趁現在計劃一下，為來年我想做的事定下一個大概的框架好了。

運動＆健身


Source

畢業後都比較少運動了，年紀開始大了也容易長胖，來年真的要抓緊時間做做運動健身，健康的身體才是最重要嘛，也希望能找到朋友多玩一些球類活動呢！比較好玩嘛！

整理自己的投資 ＆ 繼續學習


Source

加密貨幣的世界真的很廣闊，無論是各種幣的基礎知識和交易策略也有很多東西要孜孜不倦地學習呢，我更希望能不斷完善自己的交易技巧和策略。坦白說我除了加密貨幣和黃金白銀的期貨外就沒玩什麽其他的投資產品了，更是對股市一竅不通呢，希望來年有多餘的閒錢（光是加密幣就已經有一大堆幣想買的，可是囊中羞澀呀）能買一點股票，令自己的投資更多元化吧。

儲蓄


Source

投資的錢都是從每日節省下來的錢一點一滴積累而來的，沒有儲蓄的習慣怎能有第一桶金呢？希望往後能每個月存個20-30%的錢，來讓自己能有用作未來投資的資本。

旅行


Source

每次看見大家旅遊的帖子，能去這麽多不同地方遊玩就超級羡慕的，我個人又十分貪玩，很想能在來年多去旅行，見識一下這個廣闊的世界。

聯繫一下舊朋友


Source
出來工作後，大家都比較忙和懶惰有些朋友都很久沒見了，希望能騰出時間聯繫一下。

以上就是我希望在來年實現的目標了。

很高興能在今年認識到Steemit這個平台和大家這些新朋友，希望我們都能有一個精彩的2018吧！

最後我想提名 @wilkinshui, @susanlo, @michelleluilui 和 @jacklcp說說他們2018的個人計劃。

2018年上半年持各種有潛質的加密貨幣會比持比特幣好？


Source

比特幣的牛市在最近半年大幅跑贏各種山塞幣，其他加密幣們沉寂了半年後才在最近一起爆發，像Ripple、Stellar Lumens、Verge和Qtum這些幣在最近都有十分誇張的升幅。像Ripple，就算比特幣在最近跌得像狗吃屎一樣一塌糊塗，也節節攀升。

這輪加密幣的牛市會就此結束嗎？我看倒不像，反倒像一個開始，看看技術圖表，現在很多幣兌BTC的圖表都有明顯的上升訊號，還是非常大型那種，什麽杯柄形呀下降楔形的。比特幣會漲還是跌我是不知道，反正我覺得未來一段時間持各種加密幣肯定會比較安全，所以我現在是一個比特幣也沒持有，除了分叉前拍快照前會在幣安買回外。

未來感覺像Neo, AdEx, Komodo和Omise Go等加密幣也會表現不錯，而Steem也在其中呢，我還剛Power up了呢！怎麼辦？我就每樣買點算了。你呢？你有心水的區塊鏈項目嗎？

ælf的基本介紹及背景資料整理

總量：二億六千萬

Source

定位：去中心化雲計算區塊鏈網絡

ælf是一個基於區塊鏈的底層的去中心化計算平台，它被設計成能擁有資源隔離特性、高性能和完善的治理及發展結構。


Source

ælf和很久以前和大家介紹過的Lisk一樣應用到側鏈技術，能有效實現資源隔離，提高網絡性能，也有著類似STEEM見證人的投票選舉制度來建立一個完善的治理體系。此外，ælf還支持跨鏈交易，也有著能自我進化的特性，適合商業應用。

ælf的ELF代幣主要是用作購買平台上的資源和支付費用，以及執行治理決策，例如在平台上部署或升級智能合約、交易和記帳節點的選舉。

Source

上圖就囊括了ælf一些主要的優勢所在。想了解更多技術細節的話，你可以在這裡閱讀他們的英文白皮書。

買賣平台：Binance, OKEx, Huobi
