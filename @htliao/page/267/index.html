近乎萬能的貴金屬－白銀

如果你有留意我昨天的銀幣收藏帖子的話就知道我對白銀這種金屬情有獨鐘，我可不會告訴你是因為我沒有錢買黃金，那麽這種金屬到底有什麽特別之處呢?以下就會為大家一一介紹了。

大家對黃金的貨幣屬性想必十分熟悉了，在數千年的人類歷史中就扮演了公認的貨幣角色，但大家又可曾想過白銀充當貨幣角色的影響範圍，可是比黃金的金本位還要廣泛，事實上，在我國近代史是擔當流當貨幣主角的正正就是白銀，你看古代的銀票，現代的銀行這些詞語就知道白銀和金錢是劃上等號的，廣東話也是，當我們沒有金錢，很窮困時，我們會說自己「無銀用」，說明在中華民族心中，銀和金錢之間不可分割的關係。

中國在古代長久以來都是採用銀本位的，在古代像西班牙這些西方國家和中國貿易都是要用白銀作為支付媒介的，很多國家如美國、法國、日本和西班牙等國家都曾發行自己的銀元來作國際間的貿易。而歷史上著名的鴉片戰爭，大家在歷史課也讀過吧，就是因為歐美的銀行家覬覦我們中國的白銀所導致的，當時大量歐洲人的白銀流入中國來換取茶葉，蠶絲和瓷器，外國的產品卻沒什麼銷路，最後那些歐美國家只好利用鴉片和戰爭來搶回那些白銀，你能想像在清末五十年間，光是簽訂各種不平等條約，中國就賠了約四萬噸白銀出去嗎？我都不能想像四萬噸白銀會是什麽樣子的了。

在我的心目中，白銀是最萬能的金屬，是貴金屬界的小訂噹，白銀的工業用途十分廣泛，由於白銀的導電性和傳熱性是在所有金屬中最高的，白銀常被用作火箭、潛水艇、自動化裝置、計算機、核裝置以及通訊系統等要求高靈敏度的儀器元件，連你的手機裡也有白銀呢。由於白銀在工業上被消耗得相當厲害，它的存量其實比黃金多不了多少，還會逐漸減少呢，現在的金銀比率還在七十多，白銀現在實在是便宜貨呢，也有在未來十幾年內白銀將會被用盡的說法。

除了工業用途外，白銀還有醫療的用途，銀的離子以及化合物對某些細菌、藻類病毒、以及真菌顯現出毒性，是不錯的殺菌材料，但對人體卻幾乎是完全無害的，所以醫生在手術室裡用到的工具也有不少是含有白銀的，如銀針、銀線縫合傷骨和結蹄組織、銀引流管等，連牙醫也有用到白銀呢。你有看古裝劇的話，你也會知道在古時給皇上吃的食物是用銀針來測驗食物是否被人下毒，美國西部被開發的時候，很多人會帶著牛奶橫越沙漠，而他們大多都會在瓶裡放一枚銀幣來防止牛奶壞掉，由些可見白銀在殺菌方面的功用。

除此之外，白銀還能用作感光材料。鹵化銀可是被廣泛地用於攝影膠卷、醫用X光照片、電子顯微鏡照相軟片等地方呢，而白銀作為女士們的首飾上的用途相信就不用我多提了。

Source

你有想過白銀曾比黃金更珍貴嗎？對於古埃及人來說，黃金其實不太值錢， 天啊，分點給我好了，我不介意重的。 對古埃及人來說，最珍貴的是白銀，因為產量極其稀少，也只有西奈半島有少量出產而已，相反那裡當時可是盛產黃金呢。對於古埃及人來說，白銀的顏色就像太陽的光芒一樣，而如果你對埃及文化有少許了解的話，你就會知道太陽對古埃及人來說有多深的意義了。他們的最高神明就是太陽神，拉。他們可是認為只有神才配使用這種金屬呢。

Source

以圖坦卡門的墓葬為例，他只有金棺，而他的祖先都至少有一具銀棺呢。法老作為太陽神之子，在地上的代表，卻沒有象徵太陽光芒的銀棺，對古埃及人來說大概只能用寒酸來形容。

最後，說完一些基本的介紹，歷史和白銀的功能後，回到大家最關心的

（原創分享）技術分析入門（6） 波浪理論

上集跟大家分享了一些圖表型態的應用，還應用到比特幣的走勢上呢！大家還記得吧？這次要和大家分享的是鼎鼎大名的波浪理論。

波浪理論是由拉爾夫．艾略特(Ralph N. Elliot)所發明，用作分析股價漲跌的規律，所以又被稱為艾略特波浪理論。

Source

《左傳．莊公十年》: 「一鼓作氣，再而衰， 三而竭。」所有市場的價格也是一樣, 上升的驅動浪就只有三浪，只後就會開始下跌的趨勢，然後再走另一個循環，周而復始。

波浪理論的基礎十分簡單，艾略特發現股價都有著「五波上升、三波下降」的規律，總共就是八個波浪，形成一個週期，這個規律在各種市場上都能應用到。

這些波浪分為驅動浪和調整浪兩種，驅動浪就是上圖中的1,3,5浪,調整浪則是2,4浪。

以下是各個浪的常見特徵

  • ①： 在眾多波浪中，1浪通常為5浪中最短的行情，因為市場還存在著賣壓，常形成底部形態的一部份。

  • ② ：2浪的調整幅度通常都比較大，常會誤導投資者令他們以為下跌趨勢還未結束。還記得以前提過的頭肩底，雙底等反轉型態吧？這些都是2浪結束後經常出現的型態呢，配合這些形態，我們就能更加確認這是2浪，放膽進場吃掉3浪驚人的漲幅了。

  • ③：一般情況下， 3浪的漲勢會是最大，持續的時間也會是最長的，成交量也大。由於漲勢猛烈，常有延長波浪的現象。我稱它為「邪惡的第三波」，當價格突破1浪的高點就是買進的時機了。

  • ④：4浪是3浪猛烈的漲勢後的調整浪,需要留意的是4浪的低點不會低於1浪的高點。

  • ⑤：5浪是升浪的尾聲,一般情況下都比3浪為短，反轉型態如頭肩頂的頭部可能會在5浪成型。

  • ⓐ: 在a浪時投資人通常會認為價格只是在作短暫的調整。實際上在5浪時都會給出一些警號，如頭肩頂形態、成交量背馳等。若果市場仍然樂觀，在a浪時價格也可能會作出水平調整。

  • ⓑ: b浪的成交量不大，可說是多頭最後的逃命線，但常被投資者錯認為上升趨勢而錯過了，錯過了就要迎接C浪的價格急泄，最後只能抱頭痛哭了。

  • ⓒ: c浪是最恐怖的，價格一般都會像瀑布般急瀉，是跌幅大，且持續時間長的全面性下跌，把大家的信心都徹底摧毀了。

8個浪走完，又會重新開始循環，展開下一個週期的1浪。

波浪理論博大精深，再去仔細研究的話還有延長浪、各種波浪的變形（平台形、鋸齒形等）、波浪裡的子浪等東西可以研究，這些就很深奧了，就不在這入門級裡提及，反正你學到這8個浪形也已經可以初步應用到趨勢的分析上，這些也比較常用，和實用。

Siacoin的基本背景資料整理

Source

Siacoin背後的開發團隊是Nebulous Labs，首席開發者和CEO是David Vorick，年紀輕輕才二十多歲的他還是Bitcoin Core的開發者和前IBM軟體開發者。在Sia的系統裡，提供硬盤存儲空間的叫作農民(Farmers)，農民可以透過出租電腦上的硬盤存儲空間給擁有SC的用戶來賺取SC，用戶則可享用價廉物美的優質雲存儲服務，是個雙贏的局面。

定位：應用智能合約的去中心化雲存儲

競爭對手：AmazonS3, Google Drive, Dropbox, 百度雲, StoreJ, MaidSAFE, Filecoin

總量：約450億個

第一個區塊奬勵300,000個Siacoin，每挖出一個塊減少一個幣。
你能在https://explore.sia.tech/ 查看現在的總量。

共識協議：工作量證明機制（POW算法）

Sia為了同時保障用戶和農民的利益，採用了簽訂合約時付費的策略，在用戶上傳文件時就打開了一條支付通道，而這支付是不可逆的，不能退款，這樣就能確保農民能收到款項了，另一方面，為了令農民不會在收取了款項，上線存了文件就關機，過河拆橋，令用戶無法下載到文件，用戶的客戶端可以降低這個農民的評級，令其他用戶在將來減少使用這個農民的服務，直至最後只有穩定可靠的農民可以賺取到SC。

Sia使用了一種叫作Reed Solumn的編碼技術, 系統標準是把文件分配給30個托管主機，只要10個主機就可以恢復文件，這樣即使有著大量的主機離線也不會造成文件的損壞。同時，去中心化的Sia系統能令雲存儲服務更快捷、更安全和更低成本。

優點:

  • 高度隱私, 只有你持有你自己的private key, 你的文件會被Sia系統分割、加密和分發到分佈式網絡, 只有你能訪問和完全擁有你自己的數據，不被任何數據供應商主宰，此舉解決了傳統的雲存儲存在的隱私問題。

  • Sia的代碼是開源的，有很多活躍的人和社區基於Sia API在構建創新的產品。

  • 超級便宜，Sia所提供的服務價格是現有的存儲供應商的十分一。

以5 TB為例，以下是各個存儲供應商的費用比較：

存儲供應商 每月存儲費用 下載帶寬成本
Sia $10 $5
Amazon S3 $115 $460
Google Cloud $100 $550
Microsoft Azure $120 $435

這你就能清楚明白Sia的優勢在哪了，一目了然。

根據IDC 2013年的報告顯示，在未來四年內，全球雲服務市場規模將增長到442億美元，其中雲存儲的市場比例將從9%增長到14%，其規模將接近62億美元，存儲市場是增長最快的雲計算服務。第三方報告顯示，2015 年全球數據總量約為 8.6ZB，預計 2016 年將在 12ZB 左右，到 2020 年將超過 44ZB，相當於全球平均每人有超過 5500GB 的數據量。
