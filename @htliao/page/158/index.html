幣災來了，這會是個黃金機會嗎？

Source

近日傳來中國政府內部有關打壓虛擬貨幣的會議記錄的壞消息，幣市的總市值更是在短短的時間內蒸發了數千億，對於這所謂的「幣災」我已是司空見慣了。這不難想像是大鯨為了把新進場、連丁點信仰也沒有的幣圈新手嚇跑所耍的手段。

最重要的是連我媽都被嚇得不輕，說連全世界的政府都要開始打壓這些不實在的東西，催促我趕快把這些幣放了，把錢拿在手裡呢。我不禁心想：那才更該買啊，這些才是真正的錢啊，證明這些政府們都開始怕了。

上一次她這樣說的時候，可是中國政府關閉國內的交易所，比特幣跌至三千美元的時候了。

看了她的反應後，我的信心更加堅定了，因為我媽一向是個挺不錯的反向指標，而且整個幣市的過去的增長簡直是爆發式的，這樣的一個較大的調整才能為後面的升幅鋪路嘛。在打這篇文章的時候，已經有不少幣的價格開始回穩，強勢點的甚至開始止跌回升了。

別人看到的是一片屍山血海，在我眼中的可是未來的金山銀礦啊！

話雖如此，所有幣都有它們自己的週期，還是找一些相對落後的幣會安全點啊，像NEO最近的暴漲不也是因為之前相對其他幣落後了不少嗎？尋找價值被低估的優質幣，看看未來有沒有利好消息，等待機會，不要FOMO買入才是王道，如果在Ripple、Tron和Verge等高位追入的相信現在已哭慘了。希望大家有好好把握這次大跌的機會，賺到點錢吧。

當然，上述的只是我的個人看法啦，切記幣市有風險，投資需謹慎！

順道宣傳一下中文區加密貨幣研究社，鼓勵大家多使用 #cn-cryptocurrency標簽，希望大家多些翻譯和寫作關於幣界的資訊，為中文區的大家提供更多虛擬貨幣資訊。

ICON的基本介紹及背景資料整理


Source

代幣符號：ICX
總量：約四億

ICON是一個巨大的生態系統，它被譽為韓國的以太坊。ICON網絡能連接韓國多家銀行証卷公司醫院保驗公司和大學等機構。在ICON的生態系統裡，任何人都能計劃一個新的區塊鏈項目，而這些不同的項目將會連接在一起型成一個超巨大、像怪物般的的生態系統。

區塊鏈ID、支付與交易等分佈式應用能被跨領域使用。

透過其人工智能軟件「DAVinCI」，ICON將會有它們自己的人工智能系統為ICON的政策、發展及擴張作出最佳決定。

ICON更會在它們的平台上結合了去中心化交易所來增強ICX代幣和其他基於ICON的代幣的流動性，已確定的就已有Bancor和Kyber Network。

共識算法：(Loop Fault Tolerant, LFT算法)
LFT是一種改良版的拜占庭容錯算法，所以不意外是的它像NEO一樣不能被分叉和有著良好的擴容性。

交易平台：Binance, Huobi

最重要的是，它代表了韓國這一個超級熱愛加密貨幣的國家啊！而我們不難發現這一年中國的NEO和日本的Monacoin都瘋漲得令人難以置信呢！ICON的主網將會在1月底上前線，到底ICON會為我們帶來更大的驚喜嗎？

不要小看STEEM所能創造的現金流

Source

這半年來我向很多朋友介紹了STEEM，但很多朋友都說要寫文章太花時間了。即使我告訴他們能在這上面賺到不少金錢，他們也是沒有多少興趣。既然都說過了，他們沒興趣我也不會再多費唇舌的。

我想有一部分原因是因為他們還不太了解虛擬貨幣和它們的潛力，以為它們真的是虛擬，只是一串數字吧。另一個原因就是他們完全對現金流沒有概念。根據壇子哥寫的這篇扒一扒投资Steemit的SP点赞收益率有多大？，視乎你在平台上的活動，持SP的人會得到10%~40%不等的年利率，我還完全沒把STEEM可能的升幅計算在內呢。其實，能有著5%回報的就已是十分不錯的資產了，即使你購買物業，需要應付麻煩的租客和法律問題也不一定能有這個回報率呢，STEEM POWER可是優質資產呢，所以我也從來沒Power down過。再者，如果你連點贊也懶的話，設個點贊機器人在我身上也可以呀。xD

這就像Facebook在初期說只要你在上面寫東西，就白白送給你股份，你還拒絕，這不是很荒謬嗎？

像我這樣，我一週就能收到26 STEEM POWER的curation rewards，一年52週就是1352 STEEM POWER，以今日STEEM 6.5美元的價格計算的話，這可是價值68,000港元。也就是說，我甚至不用寫文章，只是點贊已能每年白白得到這現金流了，我懶的話還能利用機器人來幫我完成這項工作。如果連我這樣的小子都能做到的話，這需要的，就只是你的時間和努力了，只要你肯發揮你的想法和創意，努力在這平台上發掘機會，我相信你還是能獲得不少收穫的。

在看這篇文章的你可說是十分幸運的一員，有幸在時候尚早的時候認識STEEM，誰知道Smart Media Tokens會為STEEM帶來一片怎樣的新景象，人生中的機會可不多，可不要錯過這黃金機會了。
