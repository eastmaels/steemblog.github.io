HotFoodHK #18 餃餃餃子

這次會帶各位吃貨去一間日式餃子專門店，門口的餐牌展示了用不同的餡料製作的日式餃子，看上去好像還不錯，就決定和朋友去試試了。

整間店舖的設計以紅色為主，裝潢十分日本風，店內環境也挺寛敞的還有這種拉面舖中的吧台座位，我還以為自己進了拉面舖呢。

看看餐牌，各種各樣的餃子琳琅滿目，想不到餃子也能玩出這麼多花樣。

每種餃子都想嚐一下的我們自然就點了餃子四種盛合，裡面包括海老餃子、蘑菇餃子、日本京都九條葱餃子和日本淡路島産洋葱餃子。海鹽是用來點海老餃子的。

桌上有多種餃子醬汁和調味料可供選擇。

當然是搭配啤酒來一起吃吧，有另一番風味。

晚餐當然不能只吃餃子吧，除了餃子外我們還點了一個豚骨拉麵來吃，味道不俗。

賣相吸引的招牌Chao Chao餃子，皮薄且香脆，一排有八隻一口餃子，蘸上該店的自家餃子醬汁，味道不錯。

日式牛筋沾烏冬，由日本秘製醬汁配上木魚湯底調製而成嘅牛筋煮沾汁，彈牙的烏冬配上味道濃郁的沾汁，十分惹味，烏冬還能免費無限添加呢。

地址：紅磡黃埔新天地百寶坊G28號舖

MonaCoin的基本介紹及背景資料整理

Source

今日要為大家介紹的是MonaCoin，MonaCoin是日本的第一隻加密貨幣。MonaCoin在2013年12月由一位叫Mr Watanabe的用戶所發佈的。他的身份就像比特幣的創始人中本聰般神秘，但可以肯定的是這位Mr Watanabe是個日本人。

MonaCoin是萊特幣的一個分叉，它與萊特幣主要的差異就是它的Hash算法由原本的Scrypt改成了Lyra2REv2，至於它的錢包客戶端和礦池的用法與萊特幣的基本上沒什麽不同。

Mona在2017年3月8日，開始了SegWit的投票，並於4月27日啟動了SegWit。

Source

提到MonaCoin當然就不得不提日本的文化。日本人都愛本地化的東西，支持本地的產品，所以日本當然對他們自己開發的加密貨幣充滿期待。關於MonaCoin的信息都是用日文書寫的，而且MonaCoin的名稱以及Logo都源自2channel網上流行的顏文字，這隻像得像小白貓的顏文字吉祥物就是Mona了，牠可是深受動漫宅的歡迎呢。

在一些日本的人氣社區裡，MonaCoin會被利用來進行打賞或購買一些物品，其中也包括一些同人作品和18禁的成人遊戲和漫畫。MonaCoin的社區還在Comic Market這個全球最大的同人誌即賣會的會場進行宣傳活動，在C86會場免費發放300本44頁的MonaCoin同人漫畫册子，還有相關的宣傳網站呢，由些可見MonaCoin的社區力量也挺強和積極而有活力呀。

Source

只用文字和符號就能拼出這樣複雜的圖像，很厲害呢

相信大家都知道日本人對於加密貨幣的熱衷程度，不少零售巨企如Bic Camera和Recruit Lifestyle等都接受比特幣支付，也有著越來越多的店舖接受比特幣，而且日本近日通過了一項法律允許零售商接受比特幣支付。在9月末，日本比特幣交易所巨頭BitFlyer等11家數字貨幣交易所更獲得了日本金融廳頒發的首批數字貨幣交易所運營許可。MonaCoin作為日本最受觀迎的競爭幣，也有不少店舖接受MonaCoin作為支付方式，其中就包括秋葉原的Ark Computer Store。

誕生年份：2014

總量：105,120,000

挖礦算法：Lyra2REv2

買賣平台：Bittrex, Livecoin, Bleutrade

看看MonaCoin價格的圖表，看看那條誘人的直線，MonaCoin在9個月內升了超過250倍之多，這個漲幅十分嚇人呢，可惜在它漲之前都沒有聽說過這個幣。 T^T

（原創分享）技術分析入門（10）破底翻與假突破

時光匆匆，這麽快就來到這個技術分析入門系列的最後一集了，這個系列來到這裡也要完結了，這次會為大家介紹波段操作的兩大絕招－破底翻和假突破。破底翻顧名思義就是價格破底後又翻上來的意思，而假突破就是股價突破區間後又跌回來的意思。看見這些形態後都是操作的黃金機會呢！

破底翻

破底翻就是價格在經過長期的下跌後，價格跌穿區間底部又回升到區間內。一般來說往後這個區間底部是十分可靠的，往後價格也很有可能會有不錯的升幅。

這是黃豆期貨價格的圖表，你可以看到黃豆的價格破底後又翻了上來，這個底應該有著強勁的支撐，可以在這個底的位置附近做多。

假突破

假突破就是市場情緒一片高漲，價格破頂後又跌回原來的區間的意思，這顯示買方力量不足，上方存在強勁阻力，價格有回落的風險。

這是黃金價格的圖表，你可以看見去年黃金在1300美元以上待了幾個月，但之後又跌穿這個位置，在1300美元左右形成了一個假突破。之後黃金就一直跌至1100多美元才止跌回升呢。現在這個結構好像有點被破壞了，希望往後金價能突破並站穩1300美元，那就可以看好金價了，順帶一提金價最強的阻力是1350美元附近的長期阻力線，突破了的話牛市應該就真正開啟了。

在幾個月前我為大家推介了一個台灣的財徑節目的，想了解更多關於破底翻和假突破的例子和應用的話，你可以看看在那節目上蔡森的技術分析。希望大家在看完這一連十個關於技術分析的帖子後能對技術分析多點認識吧，也希望大家以後在交易中能多運用這些工具和指標，從交易中賺點錢呢。

從四百多年前股票市場的崛起看現今的區塊鏈技術

比特幣在近日再闖高峰，掀起城中熱話。有人說比特幣是泡沬、 龐氏騙局，也有人認為比特幣是能掀起金融革命的新時代貨幣。筆者認為在我們討論比特幣和它背後的區塊鏈技術前，不妨先了解一下股票的由來。大家知道第一張股票長什麽樣子嗎？

Source

這就是世上第一張的股票了，現代股市的根源可以追溯到1602年的阿姆斯特丹。當時正值歐洲的大航海時代，各國開始興起海上冒險，探尋世界地理與尋找貿易商機。由於航海的風險和成本頗高，當時的荷蘭東印度公司，發行了股票作為未來利潤分配的保證，把與西印度群島的貿易往來所得到的利潤的分配給購買股票的人。這也成為了現今股份和派息制度的前身。

它第一天上市就升了15%，兩年不到就升了三至四成，一百年後更漲了一千倍！我想那時還在用硬幣，對紙鈔也信心不大的人們不可能完全理解這樣嶄新的東西，對這張股票的看法大概就跟現在人們看比特幣一樣，會認為這樣的一張紙怎麽可能作為資產，會覺得這只是泡沬或是一個騙局。

在全盛時期東印度公司擁有150艘大型商船、40艘戰艦、5萬名員工和近兩萬僱傭軍，簡直是富可敵國呢。雖然公司最後由於過度借貸，戰爭，公司內部的腐敗等問題而在1799年破產了，但股票和派息制度卻一直浴用到四百年後的今日，股票市場也漸趨成熟，股票更成為現代人資產配置和投資工具中一個不可或缺的角色。

我認為區塊鏈技術也是一樣，無論比特幣是不是泡沫，也改變不了區塊鏈技術將會堀起這個趨勢。區塊鏈公開不可竄改、公開、公平的特性和它的眾多應用註定它會成為未來經濟及金融中的重要一環。至於比特幣，作為區塊鏈技術的先驅者，可能它是泡沬，也可能它未能還能作為數位黃金一直生存下去，沒有人知道。

Source

從上圖中我們可以看到比特幣「泡沬」比起其他的「泡沬」來說也不是很大呀，而且沒有人能預知「泡沬」何時會爆破的，可能是一年、十年、甚至一百年？單單「泡沬」二字絕對不能成為你不去了解和投資比特幣的原因。


Source

事實上各國政府和銀行雖然嘴上不承認比特幣作為貨幣的價值，對它背後的區塊鏈技術卻十分感興趣，像杜拜已經在開發自己的加密貨幣emCash。我相信在不久的將來，我們將會見證區塊鏈技術像以前的股票市場體制一樣，掀起一場新的經濟及金融革命啊！總括來說，趁現在多去了解和找尋有價值和前景的區塊鏈項目來投資準是沒錯的。

Cardano的基本介紹及背景資料整理

Source

總量：450億

定位：可以證明其安全性和公平性的遊戲對戰平台

受到 @jubi的邀請，今日帶大家來看看Cardano這個新幣是什麽，原來它是關於對戰遊戲的呢，難怪我們的 @jubi這麽感興趣了。

Cardano這個名字是來自16世紀的意大利數學家Gerolamo Cardano。Cardano在醫學、數學和物理方面都有著顯赫的成就。有趣的是，他還是賭場中的常勝軍，在他還是學生的時候可是靠賭博來膁取學費的，厲害吧？他在死後所發表的《論賭博遊戲》一書也被認為是第一部機率論著作。


Source

開發Cardano的公司是IOHK，它的CEO是以太坊的前CEO Charles Hoskinson，上圖左一，站在V神旁邊的就是Charles了，他同時也是比特股的創始人之一，來頭不少呢，看來真的有很多超級有名的區塊鏈公司是香港公司呢。

編程語言：功能性語言Haskell(被視為最安全的編程語言之一)

Cardano是完全沒有被運營商所支配的民主P2P遊戲平台，參與Cardino遊戲的玩家必須持有平台的ADA代幣，透過和其他玩家對戰來赢取ADA代幣。利用區塊鏈的技術，遊戲能變得公平公正且不能作弊，玩家也能透過平台高性能的配對系統安全地和對手連線。如果遊戲需要莊家，也能任意選擇用某一位莊家。你能想像如果Google Play、App Store上應用到Cardano的話那它的潛力有多大了。

電子遊戲深受年青一代的青睞，而電子遊戲產業有著龐大的市場，單單像騰訊在2017年上半年來自遊戲業務獲得的收入就已達到74億美元。可以想像的是如果Cardano能在這一領域取得成功的話，它的潛力無疑是巨大的。

共識協議： 一種叫Ouroboros的新型股權證明(POS)機制，它的特點是能夠證明其安全性

買賣平台：Bittrex
