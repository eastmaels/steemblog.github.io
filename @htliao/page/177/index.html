IOTA的基本介紹及背景資料整理

Source

總量:

Source

這總量我都不懂數算了，反正對我來說是天文數字，IOTA是沒有通貨膨漲的，也就是說它的總量永遠會維持在這個數字，在Bitfinex中，IOTA的交易是以Mi為單位的，你把IOTA的i看成比特幣的最小單位－聰可能會舒服點。

IOTA是針對物聯網(Internet of Things)而設計的區塊鏈技術。幾乎所有的電子產品都能運用到IOTA，像電子書，你可以只為你讀過的頁數付款，像聽在線音樂，你能只為你聽過的秒數付款，可以計算得非常準確，能夠在所有的設備中做到真正的互通，在物聯網和共享經濟方面應該會有十分不錯的表現。

在IOTA中，所有的代幣都是在創生交易中產生的，所以就沒有挖礦的概念了。

用例：

  • 智能電網
  • 基礎設施
  • 供應鏈
  • 智能城市
  • 運輸

什麽是Tangle?

Source

IOTA採用的不是傳統的區塊鏈，而是一種叫作Tangle的架構，這架構是基於DAG(有向無環圖)的。這種架構令每個節點不再需要存儲整個網絡的數據，每個節點可以互相確認交易並驗證，允許每秒數千次的交易，沒有交易手續。

IOTA採用了文格尼茨簽名來達到量子安全，相反和它類似的Byteball是達不到量子安全的。

優勢：

  • 沒有交易手續費
  • 強大的擴展性
  • 超快的交易確認速度
  • 能進行小額交易
  • 安全的數據轉移和數據錨定

買賣平台：Bitfinex

因為IOTA的技術是比較新的，也不是其他區塊鏈的山塞版或ERC 20令牌，交易所集成IOTA需要一些時間，不少交易所好像都希望上架IOTA，未來相信IOTA會在更多的交易所上架。

到底IOTA將會改革整個物聯網世界嗎？就讓我們拭目以待了。先說一下，我可沒持有IOTA，因為我沒有Bitfinex的戶口。

還記得我不久前介紹的TenX，Adex和Neo嗎？這些天這三隻幣的表現可是十分不錯，遠超其他幣呢。

這隻AdEx單日就漲了超過50%，TenX和Neo這相信我就不用說了，有關注幣價的朋友肯定留意到的。

我做事永遠不是嘴上說說而已，這些就是我正在持有的幣。

（原創分享）技術分析入門（８）技術指標（中）

上一集跟大家介紹了MetaTrader5這款手機應用和RSI這個超好用的技術指標，大家有好好運用嗎? 特別是有參加模擬投資比賽的人呀。:P

看了上回大家的回覆，大家好像比較喜歡像bite-size tutorials這種小章節教學，也令大家好吸收（一次說得太多好像大家都吸收不了）。所以這次就先集中介紹MACD這個指標，威廉指數就留待下回了。

MACD

MACD，即指數平滑異同移動平均線，英文的全名是Moving Average Convergence/ Divergence。這指標是由Gerald Appel所發明的，其原理是利用快速和慢速兩條指數平滑移動平均線來計算出兩者之間的差離值(DIV)和差離值平均值(DEM)，再透過利用它們的聚合與分散的徵兆功能，來判斷買出或沽出時機，找到趨勢的方向。

要明白MACD指標的原理，我們先要了解EMA、快速線DIF、慢速線DEA這三條線和我們常常會見到的柱狀圖是怎麽來的。

EMA就是指數平滑移動平均線，它背後的概念是認為最近發生的資料比較重要，所以應該 用較大的比重 計算。EMA 就是按照不同天用不同的權重來計算出來的，這就避免了一般的移動平均只要其中一天的價格漲幅較大，就會大大地影響到移動平均的數值的問題。

MACD屬於中期指標，最普遍被運用的是MACD(12,26,9)，所以我們就用這個來做例子：

DIF = EMA[12] - EMA [16]
DEA = DIF的EMA[9]
柱狀圖 = DIF減去DEA

這兩條線和柱狀圖就是這樣計算出來的，是不是很簡單？

什麼？你說看了這麽多還不明白它的原理？我都快想哭了。 T^T 不明白不要緊，你只要懂得如何運用MACD指標就可以了，很簡單的，一點也不複雜(我是說真的)。

在漲勢中，短期(快速)移動平均線及中期(慢速)移動平均線間的差距會愈來愈大;如漲多整理時，兩者間的差距會縮小或交叉；在跌勢中，短期均線會跌破中期均線，且維持在中期均線之下，兩者的差距更會隨著跌勢加劇而擴大。下圖就教了你如何運用MACD指標：

source

我們現在就看看怎樣把MACD指標運用到黃金的走勢分析上吧(對啊，我最愛黃金了，不行嗎)，從上圖中，你能清楚看見在七月快速線DIV向上突破慢速線DEA，柱狀圖也由負轉正，這時就是買入的大好機會了，而柱狀圖一直維持在正數就不用急著沽出。反之，在九月柱狀圖開始由正轉負，快速線DIV也向下突破慢速線DEA，形成一個死亡交叉，往後的走勢不太樂觀，這時持貨的就要獲利了結甚至考慮沽空了。由些可見，MACD是一個超級好用的指標，既簡單，又可靠。

AdEx的基本資料介紹及整理



Source

總量：1億

定位：基於區塊鏈和智能合約的去中心化廣告交易平台

AdEx這個平台主要針對娱樂產業，特別是廣告發佈商所提供的流視頻服務。

Source

AdEx Profile

AdEx Profile是AdEx的核心特點，是一個個性化的頁面。它能讓用戶主動告知廣告商他們的消費偏好、行為和購物習慣等訊息。透過AdEx Profile，用戶能控制發佈給他們的廣告，不用再看一些與他們無關和沒興趣的廣告。另一邊廂，廣告商也能提高他們廣告定位的精準度，把廣告有效地帶給他們的潛在客戶，令發佈商也能獲得更高的廣告投資回報。

AdEx希望解決在線廣告的痛點，如被廣告屏蔽軟件屏蔽、隱私憂慮、廣告詐騙和支付方式受限等問題。

AdEx採用的區塊鏈技術能隱藏數據的來源，


Source

AdEx的團隊在創建AdEx之前還創建了視頻娛樂中心Stremio，Stremio的用戶數目還在兩年內攀升至300多萬。上圖中的就是Stremio的介面了。一旦AdEx全面啟動後，Stermio也會成為它的第一個廣告發佈商呢。

AdEx旳代幣ADX是用來買賣廣告版面和廣告時間的，它符合ERC20代幣標準，與以太坊錢包兼容。

想了解更多關於AdEx的資訊，你可以按此閲讀AdEx的英文白皮書

ADX代幣分配：

  • AdEx團隊分得10%
  • AdEx顧問分得2%
  • WINGS DAO平台分得2%
  • 如翻譯白皮書、社交網絡上的活動等奬勵項目分得2%
  • 80%於眾籌中銷售給大眾

競爭對手：Google, Facebook, BAT, NYIAX

AdEx的第一代原型將會在2018年2月面世，到底最後AdEx能不能掀起廣告界的革命呢，就讓我們拭目以待吧。

一個推廣Steemit的小構想

不知道大家有沒有向朋友介紹Steemit的經驗，我自問也向不少朋友介紹過Steemit，他們都嘗試寫過一些文章，但他們大多玩了一會兒後就放棄了。我想這大概是因為他們都懶得寫一篇長長的文章，寫作又跟他們的生活方式相差十萬八千里吧。回想一下，我在玩Steemit之前是連Facebook和Instagram也懶得更新的，也沒什麽想寫作的慾望，能堅持到現在我也對自己的意志力感到驚訝呢哈哈。

我覺得最大的問題是玩Steemit太花時間了，還要改變他們的生活習慣，像我每天都花了很多時間在閱讀文章、點贊、 回覆和寫作上，其實只是寫作也不是件容易的事了，這大概也是Steemit現在也只有三十多萬用戶的原因吧。反之Instagram就是年青一代很喜歡用的社交應用了，他們都愛把自己的生活照片，加上一兩句短短的Caption，上載到這個有著超過七億活躍用戶的社交平台上。我想他們如果每個Instagram的帖子能有數美元就會很滿足了。

Source

很早以前發現Steemit也有Steepshot這個手機應用了，但我自己是沒用過的，這裡有人用過嗎？Steemit只以博客的形式來推廣我想是比較難令用戶人數顯著增長的，也只會有一些老博主有興趣加入，對年輕一輩是沒多大吸引力的，而年青一代可是社交媒體的主要用戶。我的想法是透過一些流行的網上譠壇先令他們開始使用這個有著類似Instagram介面的手機應用接觸Steemit，分享生活點滴，然後再激發他們寫文章的興趣，這樣應該會容易得多，也大概是唯一能令Steemit的用戶變成過百萬、過千萬人數的方法吧，到時Steem也大概會價值十刀、百刀了，畢竟大多數人都是懶惰的，尤其是美女，很難叫她們花時間去寫文章吧，自拍照對她們來說就容易得多了，有了美女，還怕沒有男用戶嗎？

有了這個想法，我就開始著手看看Steepshot這個應用了，可是我卻發現我找不到下載的方法（我是用蘋果的手機的），按了宮網的連接後就到這個白俄羅斯的店面了，可是卻找不到這個應用，害我的推廣大計都沒辦法進行下去。 T^T

這就是我的想法了，大家有其他推廣Steemit的好辦法嗎？可以留言說說，我覺得不錯的話你的留言會獲得我的親自點贊哦。

談談比特幣錢包

上一個帖子跟大家介紹了在香港的一些購買比特幣的途徑，這個帖子就說說比特幣錢包的種類和安全性。


Source

首先錢包基本可分為熱錢包和冷錢包兩種，區別就在於它們有否連接互聯網。

熱錢包

一直聯網，無需下載整個比特幣網絡的數據。

優點：

  • 支持多種設備，十分方便

  • 便於使用

缺點：

  • 較容易被黑客或其他惡意人士攻擊

冷錢包

冷錢包也叫作離線錢包，故名思義就係不連接互聯網，一般只需要一部不聯網的電腦專門用來存儲整個比特幣網絡上的數據，離線生成比特幣的地址和私鑰，像Trezor這種硬件錢包也屬於冷錢包。

優點：

  • 完全離線，能有效對抗黑客和其他有惡意企圖的攻擊

  • 長期持有幣的好地方

缺點：

  • 對新手來說配置困難
  • 不便於日常交易

常見的比特幣錢包

Blockchain.info

最流行的比特幣錢包之一，用戶可通過瀏覽器和手提電話很容易地發送和接收比特幣，也可以自己持有私鑰。

Tenzor

Source

Trezor是一款很有名的硬件錢包，方便之餘還提供高水準的安全性，缺點就是價錢較貴了。

Exodus

Exodus是一款桌面錢包，Exodus可以容納多種加密貨幣，它還和Shapeshift合作，你能在錢包內交易各種加密貨幣，十分方便，介面還很酷呢！

紙錢包

Source

紙錢包就只是一張寫上你公鑰和私鑰的紙，這些鑰以二維碼方式打印，以後想交易就只需對它進行掃碼，把紙錢包保存在安全的地方是保存比特幣最安全的方式。

你可以在這比較看看不同錢包的評價和資訊：
https://www.cryptocompare.com/wallets/#/overview

每款錢包都有各自的優點和缺點，如果你是長線投資比特幣的用家，你一定要熟悉各種錢包，考慮周詳後選擇適合你的錢包，好好保存你的數字資產。
