為什麽我買了點AdEx?

Source

AdEx是一個基於區塊鏈的廣告交易平台，對它沒有一點認識的話可以看看我以前寫的這個帖子，它將會在二月發佈Beta版本，一切順利的話在釋出前說不定會漲一波呢！SALT就是一個很好的例子了，AdEx的Market Cap還只是排在第148位，潛力挺大的。想像一下，它如果能擠進前百名或者甚至前五十名，它能漲多少？

ZCash會是今年最穩妥的加密貨幣投資之一嗎？

Source

前幾天發了篇文章後被大家嚷著要做一個新的預測，那我現在就為大家介紹一隻幣吧，它就是ZCash，其他市值較大的幣如Monero, Dash, Ripple這些都漲得十分誇張，ZCash算是在大市值的幣中比較落後的，連波場這種新貴都第六名了，而ZCash的市值都跌出二十名開外了。

看看Zec/Btc的圖表，出現了一個頭肩底，也升穿了下降軌，配合大成交量，這可和之前STEEM圖表中的杯柄形一樣，是一個典型的見底訊號啊！想起可能的潛在回報我都快流口水了。

看完技術面，我們當然也要看看基本面來看看有什麼實在的東西能支撐它未來的升勢吧。ZCash在今年將會進行兩個大型升級- 「Overwinter」和「Sapling」，據說會使其本身和其後的網絡升級為用戶帶來更強的安全性，還會激活Sapling協議升級，改善可擴展性，實現交易保護，支持移動錢包和加入如POS算法等較新的共識協議和私有智能合約。

ZCash感覺是今年比較穩妥的投資，據我看風險應該很小，也有翻幾倍的機會，你也別想一買就立刻翻倍，給它個一兩個月的時間吧，當然你就不要幻想它能像那些空氣幣一樣能翻個幾十幾百倍了，想追求更高回報的當然可以各種市值小的潛力幣都買一點，但也不妨考慮在你的portfolio中配置一點ZCash，較進取的話也可以買它的分叉Komodo，之前也介紹過了。

Chronobank的基本介紹及背景資料整理


Source
總量：現時流通量約為71萬

受到社區大紅人 @jubi的邀請，來為大家介紹一下Chronobank這個有趣的項目。Chronobank是一個針對招聘行業的區塊鏈項目，它的平台叫作LaborX，是一個基於以太坊智能合約的勞動力交易平台，而它的代幣分為兩種，分別是TIME和Labor-Hour代幣，Labor-Hour代幣是與所在國家的平均時薪掛勾的，而TIME代幣持有者則可以獲得生成新的Labor-Hour代幣的發行費，這個基於時間的虛擬貨幣是一個十分有趣的概念，名符其實的「時間就是金錢」呢。

在傳統的招聘行業裡，中介收取十分昂貴的費用來向企業介紹它們需要的人材，而且費時失事。Chronobank就看準這點，希望透過創建一個能讓僱主輕易訪問和為未來員工評級的機制來為短期招聘行業帶來革命性的改變，它能點對點地連接工人與企業，簡單來說就是招聘行業的Uber，運用區塊鏈技術的優勢為他們帶來雙贏的局面。

$5 STEEM?我不久前的預測竟然這麽快就成真了！

Source

一個多月前，我寫了篇名為《STEEM的價值有機會在2018年騰飛嗎?》的文章，那時STEEM還只是1刀多點，沒想到新年才剛過去了一天多，這預測就成真了，我們看到了五刀的STEEM！真是一份十分棒的新年禮物呢。像上個月另一篇寫的文章裡介紹的NEO在今日也有十分不錯的表現，令我感到十分欣慰呢哈哈。這些預測看似神乎其技，其實這也只是結合了基本面和技術面來推測最有可能發生的情況而已。

你問我的話，隨著SMTs的推出和發展，這說不定只是個開始，STEEM在接下來的一年裡漲到十刀以上也是很有可能的，但也不排除短期內可能會回調一點點啦。

從我加入STEEM開始，了解清楚它背後的機制後我就沒有懷疑過它的潛力，沒有信心的話我也不可能堅持每日寫作到現在，還不斷Power Up呢。你可能會說你寫的文章現在值不了多少錢，可是過去你能預計到STEEM和SBD能漲到現在這樣嗎？而且從STEEM上的閱讀、 寫作和社交你還能得到很多金錢以外的寶貴東西，而這就只是要花你每天一點時間而已，這根本就可說是天上掉餡餅的事情了。如果你是Steemit上的新人的話，你只要繼續努力，有空的話多動動腦筋，我想你在這裡一定能獲得遠比你能想像的還多的東西。

希望大家喜歡我有關各種加密貨幣的介紹和分析，覺得還有點用或者能學習到些什麽我就很開心了。 :) 謝謝各位支持我的讀者，我會繼續努力為大家帶來這方面的資訊的。最後祝大家新年快樂！ :)
