Eidoo的基本介紹及背景資料整理


Source

代幣符號：EDO
總量：約九千萬

Eidoo這個項目來自瑞士，它結合了混合型交易所和多幣種錢包兩大功能，希望能為熱愛加密貨幣的用戶提供一個安全的一站式解決方案，成為他們的瑞士軍刀。

現今區塊鏈資產的應用由於技術門檻過高，難以全面普及，用戶體驗也有不少提升的空間。Eidoo的願景正正是大大簡化用戶和他們的區塊鏈資產間的互動，錢包將支援比特幣、以太幣和所有ERC20代幣，未來還會增加萊特幣、零幣等主流幣種。Eidoo能提供一整套產品和服務，它希望能將加密貨幣真正帶進人們的生活並將各種技術加以整合，讓用戶能透過Eidoo以簡單安全的方式存儲、購買、出售、轉讓、交換他們的區塊鏈資產，使Eidoo成為他們唯一需要的東西。

買賣平台：Binance, Bitfinex, Hitbtc

雖然這個項目的概念十分簡單，乍看沒什麼特別，但如果真能做到它所說的東西，成功整合了大部份的加密貨幣並令區塊鏈資產普及的話，這可不容忽視啊！

VeChain的基本介紹及背景資料整理


Source
代幣符號：VEN/ VET
總量：約八億六千萬

VeChain的中文名稱是唯鏈，它是一個基於區塊鏈的商品及訊息平台。它在國內區塊鏈企業中可說是擁有著最豪華的陣容，因為它是區塊鏈即服務(Blockchain-as-a-Service, BaaS)公司BitSE下的一個重大項目，這間公司的CTO更是Qtum的創始人帥初。

Source

至於為什麼有VEN和VET這兩個代幣符號呢？這是因為VeChain將進行品牌形象重塑，以後就沒有VEN，只有VET了。

唯鏈這個項目早於兩年前就開始了，並在兩年間累積了大量的商業應用案例，感覺這個幣有著眾多的業務支持，難怪能在這短短一兩個月漲那麽多。

唯鏈落地案例分享：
奢侈品：與歐洲某知名奢侈品品牌合作，從生產管理、渠道管理、防偽追蹤、消費者連接四個方面對奢侈品從源頭開始控制“授權正品”。消費者通過APP掃描商品內置芯片，判斷商品真偽、了解商品歷史、申明產品數字化所有權。
食品安全：為上海外高橋國際酒類展示交易中心打造酒類追溯服務平臺。消費者可以通過各類智能設備對酒類產品進行鑒別和溯源信息的查詢。
汽車：和歐洲的戰略合作夥伴Viseo、Mircrosoft壹起為國際知名車企，共同開發針對汽車行業的應用概念驗證。
供應鏈：為全球最大的貨運代理公司K＋N提供的Baas (Blockchain-as-a-Service)服務，以區塊鏈技術從倉儲段為起點，對其所服務的全球各大品牌的貨運商進行追蹤管理。
農業：與區塊鏈、物聯網、人工智能等技術深度融合，把綠色大棚在區塊鏈上進行註冊，為綠色農業認證提供可信數據基礎。

Source

Source
陸揚(Sunny Lu), BitSE公司的COO（首制運營官）兼聯合創始人

Source
整個系統中存在著兩種代幣，分別是VET代幣和THOR Power，取THOR Power這個名字的原因是像Marvel電影中的雷神Thor一樣把制約區塊鏈大型商業應用的障礙徹底粉碎，聽上去很酷吧？這個設計令我想起NEO和GAS，但利息的收入看起來更為可觀，而VET持有量達到一萬（力量節點）、五萬（閃電節點）和十五萬（戰錘節點）的節點能得到更多的奬勵。在使用唯鏈區塊鏈的服務（如轉脹和執行智能合約）時將會消耗THOR Power。
交易平台：Binance, Huobi

個人感覺這個項目是很有錢途的，而且還有利息，很吸引啊，就看這唯鏈能不能在物流和供應鏈領域這一塊掀起區塊鏈的革命了。
