談談比特幣錢包

上一個帖子跟大家介紹了在香港的一些購買比特幣的途徑，這個帖子就說說比特幣錢包的種類和安全性。


Source

首先錢包基本可分為熱錢包和冷錢包兩種，區別就在於它們有否連接互聯網。

熱錢包

一直聯網，無需下載整個比特幣網絡的數據。

優點：

  • 支持多種設備，十分方便

  • 便於使用

缺點：

  • 較容易被黑客或其他惡意人士攻擊

冷錢包

冷錢包也叫作離線錢包，故名思義就係不連接互聯網，一般只需要一部不聯網的電腦專門用來存儲整個比特幣網絡上的數據，離線生成比特幣的地址和私鑰，像Trezor這種硬件錢包也屬於冷錢包。

優點：

  • 完全離線，能有效對抗黑客和其他有惡意企圖的攻擊

  • 長期持有幣的好地方

缺點：

  • 對新手來說配置困難
  • 不便於日常交易

常見的比特幣錢包

Blockchain.info

最流行的比特幣錢包之一，用戶可通過瀏覽器和手提電話很容易地發送和接收比特幣，也可以自己持有私鑰。

Tenzor

Source

Trezor是一款很有名的硬件錢包，方便之餘還提供高水準的安全性，缺點就是價錢較貴了。

Exodus

Exodus是一款桌面錢包，Exodus可以容納多種加密貨幣，它還和Shapeshift合作，你能在錢包內交易各種加密貨幣，十分方便，介面還很酷呢！

紙錢包

Source

紙錢包就只是一張寫上你公鑰和私鑰的紙，這些鑰以二維碼方式打印，以後想交易就只需對它進行掃碼，把紙錢包保存在安全的地方是保存比特幣最安全的方式。

你可以在這比較看看不同錢包的評價和資訊：
https://www.cryptocompare.com/wallets/#/overview

每款錢包都有各自的優點和缺點，如果你是長線投資比特幣的用家，你一定要熟悉各種錢包，考慮周詳後選擇適合你的錢包，好好保存你的數字資產。

RSI

RSI，即相對強弱指數，英文的全名是Relative Strength Index，是用來判斷價格是強還是弱的指標，進而預測未來漲跌機率的工具。RSI在1978年由美國的機械工程師威列斯·威爾德(Welles Widler JR.)所發明，是被廣泛使用的分析工具。

我們先看看RSI的公式，很簡單直接的。

Source

由這公式可見，RSI的數值介乎0至100之間，RSI高於50表示買方較為強勢，反之低於50表示空頭佔優。

Source

以這股價的六日漲跌為例子，漲的3+2+2+2=9，跌的=4+1=5

所以這股價的六日RSI = 9/(9+5) = 64.3，顯示買方較為強勢呢

RSI背離

RSI背離的意思是RSI的走勢和價格走勢相反，意味著反轉行情可能隨時出現。

上次在黃金突破1350美元時我就是觀察到RSI背離的訊號，配合我觀察到的金價議息前下跌的規律、在帖子裡提過的阻力線和威廉指標(這個會在之後的帖子提及)，我基本上是肯定金價在那個位置是會作出一個比較大的回調的，而現實也證明了這些分析是奏效的。

超買和超賣

當RSI 超過80時就表示價格超買，反之低過20時就表示價格超賣，反轉行情有機會在不久後出現，但大家需要小心的是超買還可以更超買，超賣還可以更超賣，不能單純地靠RSI來告訴你反轉行情何時會出現，最好還是配合以前說過的陰陽燭和圖表形態來判斷。

如何利用短週期的RSI和長週期的RSI的交叉訊號

這次用了14日和7日的RSI作例子。

黃金交叉：短週期RSI向上升穿長週期RSI，買進訊號
死亡交叉：短週期RSI向下跌穿長週期RSI，沽出訊號

TenX的基本資料介紹及整理

Source

定位：支持跨鏈交易的借記卡支付系統

PAY代幣總量：約二億零五百萬

TenX是一個來自新加坡的區塊鏈項目，它的公司以前叫做OneBit，它有一個宏大的願景，就是連接不同的區塊鏈，讓任何人都能在現實世界使用加密貨幣和加密數字資產，讓數字貨幣和實體經濟接軌，簡單直接而暴力的構思。新加坡作為亞洲的主要國際金融中心之一，還真的挺適合TenX的業務發展的。

創辦人：Toby Hoenisch

Tenx有自己的手機應用程式，把電子錢包和提款卡結合了，我也剛到App Store下載了，可惜目前好像只支持比特幣，期待它未來的更新。

TenX讓我想起了我有在用的Xapo錢包，以前玩水龍頭用的，Xapo錢包也是有金融卡功能的，但TenX看來在各方面都比它優勝。

優勢：

  • 與Visa和Master Card這兩家國際支付巨頭達成了戰略合作協議

  • 可以在全球近200個國家，超過3600萬個線上、線下零售點使用

  • TenX卡用戶每進行一次購買就可獲得0.1%的代幣奬勵

  • PAY代幣持有者可以獲得通過TenX電子錢包花費的整個支付量的0.5%

  • 使用費用便宜

  • 幣種之間的兌換完全免費

  • 銀行卡的收費非常低（實體卡15美元，虛擬卡只要1.5美元，基本上免年費）

Source

Source

TenX採用了一個名為COMIT的協議來連結不同的區塊鏈，能支持像比特幣、以太幣、達世幣和各ERC20代幣等區塊鏈數字資產。據TenX聲稱，目前超過95%的區塊鏈都能經由COMIT所連結，COMIT的特點令TenX能為世界各地的用戶提供即時和低成本的支付服務和為企業帶來新的商機。
