「區塊鏈投資」— 解決區塊鏈網絡流量問題---NKN

Source

各位Steemian朋友你們好，這次我和香港本地的一位專欄作家合作推出有關加密貨幣的新系列 - 「區塊鏈投資」，刊登的文章已經過作者本人同意，希望把近期最熱門的項目和最新的消息和分析帶給大家，對加密貨幣方面有興趣的請多多支持哦。

奕風簡介：
擁有傳統大學三個學位，深知道區塊鏈將會改變世界，故開展不同類型的加密貨幣和ICO的生意和投資，現時為數個世界級ICO項目的顧問，擅於分析加密貨幣市場和區塊鏈項目。

今天介紹的NKN希望透過區塊鏈解決網絡流量的問題，網絡節點自由平等的提供網絡服務，用區塊鏈經濟模型激勵節點參與。NKN認為目前網絡是中心化的，網絡效率不高，個人隱私得不到保障。NKN把自己的項目和以太坊以及FileCoin做類比：以太坊解決了去中心化的計算問題（提供智能合約），FileCoin提供去中心化的存儲（目前FileCoin項目已經延期很長時間，何時上線不得而知。順道說一句，存儲和區塊鏈的結合難度不小），NKN解決的是網絡傳輸的問題。
簡單的說，NKN想打造基於區塊鏈經濟的去中心化的網絡。

為了讓網絡中彼此連接的節點形成共識，NKN借鑒了Cellular Automata theory。筆者對此理論理解是：網絡中的單個節點和網絡中的其他節點相連，只要遵循簡單統一的規則，以及相連節點的狀態，可以保證所有節點在一定的時間後形成穩定統一狀態。白皮書中提到BFT的共識算法需要O（N！）的複雜度，PBFT需要O（N*N）的複雜度，而使用某些模型的元胞自動機的複雜度在O（N）。有算法概念的小伙伴都知道，O（N）的算法複雜度是線性算法，性能很高的算法。簡單的說，採用Cellular Automata算法，眾多網絡節點能在線性時間內快速達成共識。

NKN由四部分組成，下圖解釋NKN的不同部份。DDTN（Decentralized Data Transmission Network）是數據傳輸層。DDTN可以理解成符合元胞自動機狀態的一些節點組織成的數據傳輸網絡。NKN在POR共識機制的基礎上，使眾多網絡節點共識。並使用NKN代幣激勵更多網絡節點參與，計在2019年第一季度上。

Source

團隊方面，創始人李彥博曾負責區塊鏈系統“DNA” 的核心設計和開發，也是前Onchain的聯合始創人，而其聯合始創人Zheng Li曾在Google 和Nokia 領導多個網絡技術創新項目，技術團隊既有理論基礎派，又有實戰經驗派，有類似目的開發經驗，此項目絕對值得留意。

「區塊鏈投資」— OneLedger---跨鏈的價值

Source

各位Steemian朋友你們好，這次我和香港本地的一位專欄作家合作推出有關加密貨幣的新系列 - 「區塊鏈投資」，刊登的文章已經過作者本人同意，希望把近期最熱門的項目和最新的消息和分析帶給大家，對加密貨幣方面有興趣的請多多支持哦。

奕風簡介：
奕風，擁有傳統大學三個學位，深知道區塊鏈將會改變世界，故開展不同類型的加密貨幣和ICO的生意和投資，現時為數個世界級區塊鏈項目的顧問，擅於分析加密貨幣市場和區塊鏈項目。

如果說共識機制是區塊鏈的核心，那麼跨鏈技術則是區塊鏈實現價值傳遞的關鍵。現階段各條公鏈，無論是比特幣還是以太坊，均是獨立的價值網絡，隨著公鏈的不斷增多，區塊鏈世界將會出現越來越多類似的信息孤島，這嚴重阻礙區塊鏈生態的發展。跨鏈技術則是連接各個孤島的橋樑，是實現價值網絡的關鍵， OneLedger項目或會顯示跨鏈的價值。

OneLedger是一種跨鏈的共識協議，通過建立側鏈的方式，幫助使用OneLedger的個人或者企業，輕鬆實現跨鏈交互。與此同時，OneLedger為企業開發者專門提供了SDK開發工具，可針對不同商業主體，進行模塊化和定制化的跨鏈開發，所有交互行為均在側鏈上進行，這極大的提高了跨鏈交易的效率，有效的保證了跨鏈交易的可安全性。

簡單來說就是：OneLedger希望為商業和個人提供方便，將區塊鏈技術整合到日常活動中。他們計劃通過使用OneLedger協議更容易地創建側鏈（私人和公共區塊鏈）來做到這一點。將使這些側鏈與其他區塊鏈兼容，從而允許數據存儲在各種區塊鏈中，同時支持各個鏈上的共識。

OneLedger為企業級開發者提供了專門的SDK模塊，使得開發者可以進行模塊化和定制化的跨鏈DAPP開發。SDK工具包極大減輕了DAPP開發難度，通過OneLedger智能合約系統，DAPP可以快速上線任一公鏈。OneLedger猶如微信小程序一樣，非常適合入門級開發者進行DAPP的輕度開發和應用。

團隊方面，創始人&CEO David Cao擁有3年以上的區塊鏈經驗以及10年的企業架構師經驗, David為幾家財富500強企業做了大量的技術項目。他還在IBM多倫多實驗室工作，開發DB2和WebSphere Commerce核心引擎。團隊工程師來自Morgan Stanley、Nokia、Amazon及IBM等，均有相當豐富的區塊鏈技術開發經驗，且其GITHUB上代碼有專門的技術團隊進行審核，無論在軟硬件資源方面都為OneLedger項目開發和落地提供了充分的保證。One Ledger的天使投資人是重量級項目Polymath的CEO，也是團隊顧問成員之一。

從OneLedger的理念來看，當未來越來越多的區塊鏈企業出現時，將資產進行跨鏈轉移會遇到困難，而OneLedger開發的SDK工具，通過底層協議來達成企業區塊鏈應用跨鏈互通，這無疑是未來亟需的一個方向。可以說，在OneLedger身上，我們看到了未來跨鏈項目為商業服務的雛形。
