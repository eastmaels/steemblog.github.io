CN區有一位大叔，他於三個多月前做了一個奇怪舉動，突然想用自己「眾矢之的」的黑暗影響力，來做一件大家都不知道會否成功的事　—　「西瓜集志社」。顧名思義，這一位大叔便是人稱瓜叔 @aellly 的西瓜 。

他做這一件事，不算是一個活動，但卻得到很多人的支持及參與，差不多每次在不同的 cn 區微信群中，都會看到「瓜叔，有廣告，請代發，謝謝!」。跟著西瓜便會將這人的帖子，加在「今日頭條」及「廣告時間」中。他這一個做法，有點像精選文章，外人看著這樣的帖，必定會感到西瓜是一個大懶蟲。但原來每一個人發給他的帖，他都會看，並且融匯後總結成一句十多二十字的推薦字。

雖然這在很多人眼中是一件「懶事」，但新手村因為要整理「飛鴿傳書」，因此也思想模擬過西瓜集志社的做法。驚覺!! 原來看每一篇文章，並每一篇文章也寫簡評推薦，每篇最少需要5-10分鐘時間。西瓜集志社有時五六篇，有時十多篇帖文。每天隨時要花上半小時至一小時才能完成一帖。

這樣「吃力不討好」的怪事，竟然在Steemit CN區有人在做；更奇怪的事，是有人提供文章積極參與。這樣的怪事，或許正如博主名稱「西瓜集志社」，他不是「雜誌」，內容散亂的綜合帖；而是「集志」，凝聚「集合」博主「心志」的精華帖，一同去欣賞欣賞：

「區塊鏈投資」 — 可穿戴的區塊鏈--Sidera

奕風簡介：
擁有傳統大學三個學位，深知道區塊鏈將會改變世界，故開展不同類型的加密貨幣和ICO的生意和投資，現時為數個世界級ICO項目的顧問，擅於分析加密貨幣市場和區塊鏈項目。

Facebook專頁：https://www.facebook.com/cryptowithwind/

Source

自2018年起，大量的區塊鏈項目出現在ICO市場，挑選優質的ICO便成為了一大學問。在市場中，無可否認地，很多ICO項目只有概念，並沒有真實的技術支持，更不用談項目的盈利模式，他們所倚靠的只是大量的宣傳和團隊的營銷能力，筆者定義這些項目為「垃圾項目」，絕無投資價值，故此必須有一套對ICO的分析系統，並必須配合ICO的發展而定期調整分析系統，因為ICO的市場發展得很快，一些新的技術會將舊的技術「汰弱留強」，故需要不斷地更新區塊鏈項目的市場發展，才能找到最優質的投資項目。

在真正的區塊鏈項目中，筆者認為大部份的項目仍然是企業級應用，主要利用區塊鏈技術提升商業效能，較少項目的客戶群是普羅大眾。今期介紹的項目是Sidera，它的客戶群正正是大眾，Sidera的目標是發展與可穿戴設備（如智能手錶）等的區塊鏈系統，造結合了區塊鏈技術和物聯網的網絡。在Sidera的生態中，它不單是發展技術層面，此項目更希望發展真實的可穿戴區塊鏈設備，其實就好像是Apple Watch， Apple Watch之間能夠自行溝通和交易，這是在物聯網領域中的真正「落地」項目。隨此之外，它的技術是用Proof of Load這個區塊鏈的新協議所調節，在技術上有突破性。

此項目的特別之處在於它將線上的技術和線下設備連結在一起，情況有點像Apple的Iphone和App store，相對其他項目更能夠容易建立其生態，技術亦會令公眾受惠。團隊方面，Sidera絕對是世界級的團隊，有多名來自世界不同國家的技術人員和資深企業家，其中數名是可穿戴設備發展的專家。Sidera已與多間電子商貿平台達成協議，對產品銷售已準備就緒，絕對是值得期待的「落地」項目。

（以上屬個人意見，絕不構成投資勸誘或買賣建議。）

「區塊鏈投資」— 既暴漲30倍，又收息的Vechain

奕風簡介：
擁有傳統大學三個學位，深知道區塊鏈將會改變世界，故開展不同類型的加密貨幣和ICO的生意和投資，現時為數個世界級ICO項目的顧問，擅於分析加密貨幣市場和區塊鏈項目。

Facebook專頁：https://www.facebook.com/cryptowithwind/

Source

區塊鏈市場很大，有時侯筆者未能夠接觸和消化每天大量的資訊和項目，故此筆者有一班志同道合的「幣友」，一起發掘和研究潛質的區塊鏈項目。前幾天聽了一名「幣友」分享了一個優質的區塊鏈項目，筆者再自行研究，不單這項目之前已有30倍的升幅，更是一個穩定的「收息」的區塊鏈項目，這項目是唯鏈Vechain。

VeChain 平台是一個基於區塊技術的全球賬本型信息交互協作雲平台採用Internet of thing (IOT) 技術。通過 API 與應用層對接，把現實世界中的人、事或物數字化，實現信息的互通互聯。。通過跨平台、跨企業、跨行業、跨國界的互聯協作，創造全新的商業模式，為協作參與方提供“信任服務”，行業應用涉及酒類、奢侈品、汽車、零售、綠色農業認證、冷鏈、物流等。簡單而言，VeChain的主要功能是用區塊鏈作商品認證服務。

Vechain現時與LV、PWC(全球知名四大會計所之一) 、D.I.G(中國最大的紅酒入口商，現今30%進口中國的紅酒都經D.I.G作銷售) 、DNVGL(一家全球品質保證和風險管理公司) 和K&G(全球最大的海運貨運代理商)，Vechain協助企業從生產開始對產品作出監控，並通過區塊鏈系統，在生產、運輸、進口和銷售各環節逐層監管產品流向，保證產品的品質和安全，有效提升客戶對企業的信心。故此，Vechain的應用範圍廣泛，適合不同行業的產品追蹤和品質檢定。

另一方面，Vechain的另一賣點是token economy。在Vechain的系統中有兩種token，分別是VET和Thor Power。這系統與NEO有點相似，只要你持有一定數量的VET，你會定期收到利息，即Thor Power。如果你持有一定數量的VET，你更可以成為Vechain的節點。作為Vechain的節點，你會定期收到Thor Power作為回報，暫時的比例為0.00032THOR per VET per day。

Vechain的CEO及聯合創辦人是陸揚，同時他也是Bitse的COO，曾任LV中國區首席信息官，Vechain的其中一名顧問是比特海洋(BitOcean) 的CEO，BitOcean是一間獲得日本政府發出許可營運牌照的加密貨幣ATM公司，將來在擴展業務時，VET在ATM流通性也會上升。Vechain從ICO階段已升超過30倍，Vechain剛完成主網上線，全球市值第17位，安全性十足，筆者認為是十分有潛質的項目，會繼續留意。
（以上屬個人意見，絕不構成投資勸誘或買賣建議。）
