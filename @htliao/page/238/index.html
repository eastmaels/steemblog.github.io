DigixDAO的基本介紹及背景資料整理

Source

總量：二百萬
代幣符號：DGD

大家在早一輪幣市的大跌中可能都會有留意到這一隻逆市暴漲的DigixDAO，那麽它究竟是做什麽的呢？說起來，它還和我另一種很喜愛的投資產品有關呢，沒錯，它就是黃金了！DigixDAO是一個由去中心化自治機構(Decentralized autonomous organization,DAO)創建的21世紀以太坊金本位金融平台。

Source

平台上有兩種幣，DGX是錨定黃金的，1枚DGX代幣價值一克黃金，以能提現實物黃金的PoA資產卡以智能合約轉發為電子憑證，而在他們托管庫中的金條都必須符合LBMA標準，而且和DGX的發行量是以一比一的方式以物理金條的形式來進行備份的，而DGD就代表著平台的股權，是一種分紅收益的幣種，分紅來源是平台上交易DGX代幣的手續費，每筆交易將會收取0.13%作為手續費。

全球黃金市場的交易量可是十分龐大，絕對是天文數字，試想像一下如果DigixDAO平台上的交易量能夠在世界知名的紐約金屬交易所(COMEX)、 上海黃金交易所、倫敦黃金市場等龐然大物中分一杯羹的話，那DGD代幣會收到多少分紅，DGD的價格也自然會水漲船高。

Source

DigixDAO在審計、保管和供應這三方面都做得滴水不漏，令人十分放心DGX代幣背後是真的有一克黃金作為擔保的，你可以閱讀以下這段文字來了解一下他們怎樣確保DGX代幣的價值：

我们的黄金供应商是新加坡上市公司方圆集团(Valuemax)，他们出售的黄金都是经过伦敦金银协会(LBMA)认证合格的；为我们提供审计服务的是成立于1828年的必维国际检验集团（Bureau Veritas Inspectorate），他们会定期对黄金进行独立审计。而Malca-amit则是一个国际性的保险库服务供应商，在全球主要城市都有保险库。他们的新加坡保险库靠近新加坡樟宜国际机场，位于新加坡自由港，面积达25，000平方米，高度安全和可空调控制温度的保险库，黄金在他们的保管下是相当安全的。

買賣平台：Binance, Huobi, OKEx

Source

這樣看來這DGX代幣說不定比USDT靠譜多了，黃金的價值早被人類數千年的歷史反覆證明了真金不怕洪爐火呢，比起波動性高的加密貨幣來說可說是穩妥多了，不妨在Portfolio中配置一部分，說不定DGX和DGD都能有一定避驗的作用呢。還不用擔心聯儲局的印鈔機每日瘋狂印鈔呢。另一方面，能帶來被動數入的DGD代幣也很吸引啊。

Stremio關閉了在它上面的所有廣告，離AdEx Beta推出已不遠了嗎？

Source

Stremio這個視頻平台在幾日前宣佈把平台上所有廣告下架，為AdEx的beta推出作準備，這意味著AdEx的Beta版離正式釋出已經不遠了，肯定會在2月內如期推出，說不定對AdEx的幣價有一定的刺激作用。

什麼是Stremio？Stremio是一個像Kodi的視頻平台，它有數百萬用戶，倚賴平台上廣告作為收入，AdEx的團隊本身就有很多是Stremio團隊的原班人馬，在未來的一段時間相信它會特別為AdEx上的廣告作出特別的安排呢，有興趣的話，你還可以下載它的桌面版和手機應用來試試看呢。

LitePay能幫助萊特幣再闖高峰嗎？

Source

今日萊特幣漲了不少，因為近日有謠言稱LitePay將於2月26日上線，並於二月第一個星期開放用戶註冊，LitePay可說是萊特幣版的PayPal，容計商家接受萊特幣作為支付模式，商家收到款項後能立即和銀行進行結算。

Source

其實比特幣也有類似的BitPay，但手續費實在太高，難以普及，而萊特幣的速度和手續費這兩方面都比比特幣要優勝很多。

近日的幣災很可怕，甚至看新聞都有韓國的學生不幸地因為投資失利自尋短見。事實上，我認為這次幣災對整個幣市的長遠發展來說是十分健康的。雖然短期不排除各種加密貨幣的價格還會繼續震盪，甚至再測試一下低位，但我猜長期的趨勢還是不變的，在2018年底比特幣價格很有可能會超過5萬美元，而其他各種幣的升幅可能會更誇張啊，而萊特幣極大機會就是其中的一員啊。

Bittrex將會在2月9日下架FUN和RISE

Source

Bittrex將會在2月9日下架FUN和RISE，持有這2種幣的朋友需要趕快提取到自己的錢包或把幣賣掉了。FUN我在之前的文章也有介紹過，是一隻頗有潛力的幣種，而RISE是像STEEM一樣運行DPOS的，感覺跟LISK有點像，是做平台的。

值得一提的是，真正有潛力和有價值的幣不會因為下架的問題而失去它們應有的價值，過往Bittrex也曾下架過Nano（以前的Raiblocks，品牌重塑後變成了Nano）和我們O哥最愛的BitShares，下架後不一樣漲得十分誇張嗎？

近日因為各種謠言和壞消息，幣價像瀑布一樣暴瀉，但我相信2018會是幣市輝煌的一年，你有看好的區塊鏈項目的話還是不建議你急著把手上的幣賣掉哦。

Dragonchain的基本介紹及背景資料整理

Source

總量： 約四億三千萬
代幣符號：DRGN

提起Dragonchain，我腦海裡就浮現了迪士尼這三個字，光是這三個字已有衝動買這幣了吧?，所以它的ICO反應也十分熱烈，一共籌了1300萬美元。Dragonchain這項目在2014至2016年期間都在西雅圖辦公室進行孵化，如今已經是一個獨立的項目了。Dragonchain有著混合鏈的性質（也就是公有鏈和私有鏈的混合啦），它的願景是建立一個更安全和靈活的區塊鏈商業服務平台。

Source

Source

孵化器是Dragonchain獨有的一個新概念，這個名稱也很可愛呢，它旨在幫助區塊鏈初創項目以最快的速度成型並實現商業化。平台上已經有幾個項目了，包括：

  • Look Lateral – Liquid Art
  • Seed2You
  • LifeID
  • IDPay
  • ClevX

至於DRGN代幣的用途和以太幣差不多，就不詳述了。

Source

系統還有一個叫做Dragon Slumber Score的系統跟據用戶持有代幣的時間和總量來奬勵代幣持有者，積分高的還能優先參與平台上的ICO。

那麽在哪裡買呢？你能在KuCoin上買到DRGN代幣。
