為大家介紹幾個小魚或許能用得著的項目


Source

Steemit上的機會實在不少，只要肯動動腦筋，即使沒有大鯨關照還是能有一定收益的。今日想為大家介紹幾個小魚或許能用得著的項目，希望能對大家有幫助，快快變成大魚甚至海豚。廢話就不多說了，馬上為大家介紹吧：

Qurator

這個很簡單，就是交2 SBD做入會費，通過審核就能每日得到upvote了，會藉好像是永久的，應該只要稍為勤力每日發一篇帖子很快就能輕易回本。你可以看 @qurator 的這篇英文文章 了解一下詳情。

Minnow Support Project

Minnow Support Project可說是這麼多個支持小魚的項目中最有名的，是由見證人 @aggroed發起的，只要註冊和follow它們的curation trail就能召喚它們的機械人點贊，收到一大堆upvotes。在我還是小魚的時候也經常用呀，你可以看看 @chris77d 很久以前寫的這篇教學，只是我太久沒有用，也不知道那些規則變了多少。

Busy.org

Busy.org在最近很有名，你看我都有在用，它是根據你的followers加起來的SP來決定投票的%的，有大鯨follow你的話可是必用的呢！雖然看上去不多，但它的機器人都是幾乎第一時間投票的，這就意味著你不但能得到這票中所有的author rewards和curator rewards，還能吃掉不少後面的curation rewards呢。我們的劉美女已經寫過一篇帖子來介紹了。

Utopian.io

這也是最近超受歡迎的，它除了會奬勵那些為Steemit上的開源項目作出努力的頁獻者外，即使你只是找bug或提出一些有建設性的idea也會得到upvote呀，就像最近的新介面不是很多bugs嗎？我們的 @jubi就曾經發了一篇帖子介紹過呀。

這個收益可說是相當不錯，中文區裡的 @shieha, @stoodkev 和 @rosatravels都有用過，你看這收益我都快流口水了。

OriginalWorks

這個可說是最簡單了，就只是在帖子是呼叫 @originalworks，它就會判定你的內容是不是原創而給出upvote，雖然金額不多，但對像我這樣的窮小子來說，蚊子再小也是肉呀！它最近還好像辦了不少寫作比賽，奬勵也很豐富呢。

Whalepower

這個在之前的帖子也介紹過，就不多說了，最近他們還有一個為期三個月的計劃來支持新人，你有加入他們的Discord Channel嗎？

這些例子還只是冰山一角呀，懂一點英語的不妨出外闖闖，看看在中文區外的世界有多廣闊，與其不斷刷屏期待大鯨的投票（現在中文區裡都一隻不剩了），還是自己努力去尋找新機遇的好，說不定哪日你還能得到某些大鯨的重視，變成一條大魚來讓我抱大腿呢。

Kyber Network的基本介紹及背景資料整理

Source

總量：約2.26億
定位：高流動性且無需信任的去中心化交易平台

Source

Kyber Network的CEO是上圖中的Loi Luu，他在以太坊社區很活躍，開發過Oyente，也是另外一個開源項目SmartPool的聯合創始人。核心技術團隊也有著多年的工作經驗，不少成員也是來自新加坡，也有著在亞洲首屈一指的新加坡國立大學的教授支持。

Source

Melonport Oyente

簡單來說，Kyber就是一間提供加密資幣和數字資產即時交易和兌換服務的去中心化交易所，這馬上令我想起BitShares，因為自己也有使用過Openledger，感覺也不太難使用。可惜在現在可能是因為這概念對一般人來說比較難懂的原因吧？中心化交易所仍然是主流，但這些平台易受攻擊、有著被盗風險、不透明和監管風險等問題，相信會令持幣者和區塊鏈技術愛好者逐漸對去中心化交易所更感興趣和探用它們的服務。

Kyber Network還打算引入衍生工具，以期貨和期權等形式來讓用戶在將來能夠基於未來的需求為他們的資產進行對沖操作。

競爭對手：Bittrex, Poloniex, Kraken, EtherDelta, 0x, BitShares

優點：

  • 安全可信
    平台上的每項操作都採用了智能合約來進行，是一個無需雙方信任的系統，用戶的資金也不是像中心化交易所那樣由平台所持有的（所有的幣都在儲備庫裡面）。

  • 高流動性
    所有代幣採用了全額儲備來提高流動性。

  • 兼容性
    與現有的智能合約兼容，能無縫結合。

V神擔任了這個項目的顧問，他曾公開表示自己只擔任了2個項目的顧問，而另一個就是OmiseGO了。

在2018年, Kyber Network將會發佈他們的正式版本，有點期待呢。

買賣平台：Binance, Coinnest, Liqui, Kucoin

Stratis的基本介紹及背景資料整理

Source

定位：區塊鏈即服務(Blockchain as a Service, BaaS)解決方案的平台

總量：約9800萬

Stratis的口號是讓開發區塊鏈應用變得更簡單，它們把重點放在企業應用上，透過它們的平台和Strat代幣，Stratis旨在降低使用區塊鏈技術的門檻，為開發者提供了完整的工具包，即使是小型公司也可以創建並且管理自己的區塊鏈網絡，利用區塊鏈進行它們自己的實驗。用戶不但從構建區塊鏈模式到加密算法再到發行利率等一切設置都能簡單上手操作，還能訂制自己的側鏈(sidechains)。

共識協議：股權證明機制（POS算法）

STRAT的通脹率很低，以每個區塊生產1 STRAT的速度來計算，每年的通脹率約為0.53%。

Source

他們的CEO Chris Trew有著超過10年的後台IT技術經驗，是C#和ASP.NET的專家，曾參與Blitz項目，也有在頂級金融公司工作的經驗，你可以在這裡看看他們的團隊陣容。

買賣平台：Bittrex, Poloniex, Binance, Livecoin

Source

Stratis最近還發佈了注重隱私保護的Breeze錢包的Alpha版本，該錢包利用了TumbleBit的技術，透過混幣和鏈下交易來保護用戶交易時的隱私，錢包暫時支持比特幣和Strat代幣，對Stratis來說應該是個重大的利好消息。
