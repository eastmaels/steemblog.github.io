Bancor的基本介紹及背景資料整理

Source

代幣符號：BNT
總量：約七千九百萬

Bancor是被設計來解決小型數字貨幣流動性不足的問題，他們的團隊來自以色列，希望透過儲備金的制度來解決小型代幣的流動性問題，只要這些代幣使用Bancor作為它們其中一種儲備金來連接到Bancor網絡，Bancor就能作為它們的流通中介，而Bancor團隊本身會在ICO中募集到的20%ETH資金用作其儲備金。

在各種ICO如雨後春筍般冒出的今天，一些較新和小型的加密幣難以與以一個合理的兌換價格與主流貨幣進行兌換，相信很多投資者最大的擔憂就是自己投的幣種能不能上主流交易所。Bancor協議就是一種針對這個問題而開發的以太坊的底層貨幣協議。除了解決流動性問題外，還同時具有用戶自發產生貨幣(User Generated Currency)、貨幣互換者(Token Changers)、指數基金和去中心化一籃子貨幣（Decentralized Token Baskets）等功能。

交易平台：Binance, Bittrex, Bancor Network

Bancor還擁有獨立的第三方審計來證實其數據、信息等的真實性，以增加投資者的信心。Bancor的合作夥伴還包括了NEO、 Status、 Aragon、Gnosis等熟悉的名字呢！

Eidoo的基本介紹及背景資料整理


Source

代幣符號：EDO
總量：約九千萬

Eidoo這個項目來自瑞士，它結合了混合型交易所和多幣種錢包兩大功能，希望能為熱愛加密貨幣的用戶提供一個安全的一站式解決方案，成為他們的瑞士軍刀。

現今區塊鏈資產的應用由於技術門檻過高，難以全面普及，用戶體驗也有不少提升的空間。Eidoo的願景正正是大大簡化用戶和他們的區塊鏈資產間的互動，錢包將支援比特幣、以太幣和所有ERC20代幣，未來還會增加萊特幣、零幣等主流幣種。Eidoo能提供一整套產品和服務，它希望能將加密貨幣真正帶進人們的生活並將各種技術加以整合，讓用戶能透過Eidoo以簡單安全的方式存儲、購買、出售、轉讓、交換他們的區塊鏈資產，使Eidoo成為他們唯一需要的東西。

買賣平台：Binance, Bitfinex, Hitbtc

雖然這個項目的概念十分簡單，乍看沒什麼特別，但如果真能做到它所說的東西，成功整合了大部份的加密貨幣並令區塊鏈資產普及的話，這可不容忽視啊！
