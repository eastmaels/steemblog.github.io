「區塊鏈投資」 — 區塊鏈基礎設施服務平台Lambda

奕風簡介：
擁有傳統大學三個學位，深知道區塊鏈將會改變世界，故開展不同類型的加密貨幣和ICO的生意和投資，現時為數個世界級ICO項目的advisor，擅於分析加密貨幣市場和區塊鏈項目。

Facebook專頁：https://www.facebook.com/cryptowithwind/


Source

在區塊鏈的世界中，其中一個樽頸位是數據的儲存，區塊鏈上有無限的數據，如數據儲存到特定的儲存器或雲端，數據的安全性便成為一大問題，亦失去區塊鏈「去中心化」的意義。故此，今天為大家介紹的區塊鏈基礎設施Lambda正正為解決這個問題而生。

Lambda是一個高速、安全、可擴展的區塊鏈基礎設施項目，通過對Lambda Chain和Lambda FS的邏輯解耦和分別實現，向去中心化應用提供可無限擴展的數據存儲能力，並實現了數據完整性證明、多鏈數據協同儲存、跨鏈數據管理、數據隱私保護等服務。Lambda將數據處理分解為Batch Layer 和Speed Layer處理，讓數據處理和運算的過程更加穩固。

作為區塊鏈的基礎服務，Lambda Chain通過Sharding技術，提供了每秒鐘數万筆的數據請求和響應能力 (RPS)，並且可以隨著系統規模的擴大而增長。在未來日子，通過子鏈技術添加更多的加密和存儲服務類型，Lambda還提供了面向未來無限擴充的技術服務能力。此外，Lambda 項目還具有數據結構靈活、編程接口強大、系統可擴展等特點。在數據索引技術、多鏈數據協同處理、跨鏈數據管理、隱私保護、存儲持有性證明、分佈式智能合約 (存儲過程計算)、查詢優化技術等領域，Lambda項目也在全力進行探索和發展。

在團隊方面，Lambda的始創人何曉陽是中國著名基礎軟件專家，之前他所創立的OneAPM曾在在一年半的時間內完成經緯中國、成為資本和啟明創投等知名投資機構領投的四輪融資，創造了企業級市場融資的奇蹟。其他團隊成員包括多名頂級數據科學專家和程式員，團隊實力強勁。

基於Lambda解決區塊鏈基礎問題的實用性，項目團隊在7月初宣佈已獲得泛城資本、策源數字基金、BlockVC、Bluehill、DATA基金會等多家頂級機構的融資。之前6月19日Lambda宣布已經獲得由真格基金、大都會資本、Dfund、星耀資本參與的數千萬人民幣融資。Lambda獲得如此多的支持，絕對不容忽視，筆者會密切追蹤。

大行「幣」行情 — 金管局開展區塊鏈貿易融資平台

各位Steemian朋友你們好，這次我和香港本地的一位專欄作家合作推出有關加密貨幣的新系列 - 大行「幣」行情，刊登的文章已經過作者本人同意，希望把近期最熱門的項目和最新的消息和分析帶給大家，對加密貨幣方面有興趣的請多多支持哦。

奕風簡介：
奕風，擁有傳統大學三個學位，深知道區塊鏈將會改變世界，故開展不同類型的加密貨幣和ICO的生意和投資，現時為數個世界級區塊鏈項目的顧問，擅於分析加密貨幣市場和區塊鏈項目。

Facebook專頁：https://www.facebook.com/cryptowithwind/

Source

近日加密貨幣市場一直低沉，惟筆者發現各國政府的官方機構均投入區塊鏈的板塊，其中一個最令人興奮的消息是香港金管局將於下月上線一個區塊鏈支持的貿易融資平台，將包括匯豐和渣打在內的21家銀行連接起來。此系統由中國平安集團設計，將成為政府主導的首批旨在升級9萬億美元全球貿易融資行業的最重要的項目之一。

區塊鏈將大幅縮減日常貿易融資和供應鏈融資交易所需的時間和文書工作，因為它使得核實這一複雜流程中的每一步，以及確認相關各方的資質變得更容易。許多銀行已經在主導開發使用區塊鏈的貿易融資平台。今年5月，匯豐宣布，它已使用區塊鏈進行了全球首例商業上可行的貿易融資交易。德意志銀行(Deutsche Bank)和荷蘭合作銀行(Rabobank)也在聯手推出類似業務。

平安集團旗下的金融科技公司金融壹賬通(OneConnect)利用其已在中國內地部署的技術設計了香港貿易融資平台，其中一個關鍵特點是，它將讓小公司更多地獲得貿易和供應鏈融資。由於註冊新客戶所需的盡職調查成本增加，世界各地的小公司通常難以獲得銀行服務。區塊鏈技術能夠以低成本獲取各種公司數據，以評估潛在客戶。這將使該貿易融資平台能夠承接更多的小企業。

在新的貿易融資平台上，欺詐也將更容易被發現。公司經常要求銀行提供的資金超過貿易所需的資金。一旦區塊鏈中記錄了一筆貿易，所有各方將能夠查看所必需的融資水平，從而使獲得多餘資金不那麼容易。

作為亞洲金融中心的香港，此次的動作明確了對區塊鏈技術的關注方向，充分體現了香港政府對香港區塊鏈在金融領域的實際應用的重視程度，也讓區塊鏈產業向規範化、制度化邁進。

在筆者看來，其實這段時間可以看出，雖然現在加密貨幣市場冷清，但是國家層面並沒有放棄對數字貨幣和區塊鏈的研究，甚至想出更多辦法來促進其發展。區塊鏈的項目已逐漸落地，加上近日各國政府對加密貨幣和區塊鏈的態度漸趨開放，連中國國家主席習近平在六月的一個科技研討會會議演講時，亦親口確認中國確認支持區塊鏈的發展。區塊鏈的時代看來已悄悄地來臨了。

「談天說幣」— 航母級項目---Celer Network

各位Steemian朋友你們好，這次我和香港本地的一位專欄作家合作推出有關加密貨幣的新系列 - 「談天說幣」，刊登的文章已經過作者本人同意，希望把近期最熱門的項目和最新的消息和分析帶給大家，對加密貨幣方面有興趣的請多多支持哦。

眾所周知，可擴展性不足是目前區塊鏈技術發展所難以迴避的一大難題，而鏈下狀態通道是解決區塊鏈擴展性不足的非常有前景的發展方向之一，特別是可以幫助到已有的性能不足的公鏈。Celer Network期望通過一套完整的分層架構在以下四個方面去解決區塊鏈這的可擴展性問題：1.支持任意 DApp 和鏈下智能合約狀態快速更新的通用性狀態通道與側鏈。2. 具有最優證明的分佈式支付網絡路由算法。3. 高性能 DApp 開發架構與運行環境。4.創造網絡效應，給運營節點提供充足資金，以及保障線下狀態安全性的新經濟模型。Celer Network 旨在通過以上技術革新實現將互聯網規模帶入區塊鏈，幫助區塊鏈商業化落地的宏大願景。

為保證 Celer Network能夠滿足鏈下擴容商業化落地的願景，該系統必須在保證資金的流動性的基礎上，盡量保證系統的可擴展性和可用性。因此，團隊綜合考慮了網絡中Token的流動性、鏈下擴容的可擴展性和可用性三大因素，提出了能使三者相互平衡和協調的Token Economy，即 cEconormy(見下圖)。此Token Economy根據區塊鏈系統的功能和目標來巧妙平衡網絡效應、流動性、可用性和安全性等彼此衝突的因素，確保了未來Token的價值。

Source

「飛鴿傳書」由我 @htliao 發起，新手村 @team-cn 策劃的一個「好文好博客推薦活動」。
在 Steemit 的模式下，有心的新手的博客容易被忽略，連帶著博文亦會被低估價值 (平均每帖少於 $0.4點贊) ，我們希望透過這個活動，鼓勵好作者繼續創作好文章。

編輯小組每天會推薦最少兩篇價值被低估的博客作品， 我 @htliao 及 @team-cn 均會全力點贊，今天為週日，以推介cn區博客訪談。

https://steemitimages.com/0x0/https://res.cloudinary.com/hpiynhbhq/image/upload/v1517926589/kwhddrcppdfqee9jiuwh.jpg



@liumei

首先介紹的是個人博主 @liumei 小花，她加入Steemit時，是剛剛ned代理SP給中文區大佬的黃金時期，發一篇文最少都會有幾美元，不會有幾分的情況。她現在看來，不到10美元的文是受很多因素的限制的，不僅限於文章水準的高低，更多的取決於關係網。小花的文章沒有刻意的主題，甚麼內容都有，最希望的事，便是與Steemit的朋友交流。
她說「有夥伴來點贊評論真的很有存在感，從未謀過面，但在Steemit上就好像已認識的一樣，我不想放棄和這些很厲害的人的接觸。多給別人贊；去留言；會發現很多有意思有用的資訊；還有很重要，要寫好自己的文。」

@yanyanbebe


我們稱呼 @yanyanbebe 為茵茵，因為她的新手訪談推出後，新手村的微信群「涼亭」突然出現「米路流星雨」，他的男朋友 @minloulou 大派紅包 及 斧頭。不知今天 #cn-curation 推薦後，會否引發新一輪的「黃金紅包潮」。

茵茵的文章走旅遊及美食博客方面，會寫中文、英文及韓文，加入了短短一個月，已有多位大佬來到她的帖下點贊。並且短短加入Steemit 28天，便已達50級的聲望。這應該要拜她男朋友 @minloulou 教了不少 Markdown排版方式有關，令茵茵的文章及圖片佈局，看得相當舒服。

【新手村計畫#60】@yanyanbebe 一位來自香港美女銷售秘書

本項目由我 @htliao 發起， @team-cn 組成編輯小組，成員包括 @abcallen @ericet @honoru @itchyfeetdonica @softmetal ，每天推薦cn區價值被低估的好文好作者，推動大家互相交流，若您有意成為當中被收錄的文章，歡迎在帖子內加入 #cn-curation 標籤，以便編輯小組發掘作品。

這篇帖子的SBD/Steem收益將會放入【新手村活動基金】，用於支持CN區活動。

本文由 @team-cn 編輯推薦，【新手村會客室】每星期會訪問一位Steemit資深博客，今天我們請來剛滿一年的黃鳥，他認為在Steemit裡“內容”與“社交”是相輔相成的，大家通過每個人長期的內容文字，大概瞭解在你文字下你是怎樣的一個人，大家有共同的興趣、價值觀或者共同的利益（比如賺錢）然後關注（follow）你產生社交，然後由此建立社交關係，持續關注你更多的文章內容。黃鳥這一篇訪談，好像上了一課，易讀易明，定能讓大家更高一層樓，在Steemit上更進一步：
