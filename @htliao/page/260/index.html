暴風雨即將來襲？黃金價格的啟示和我最近的投資決定

在談我最近的投資決定前，先看看YouTube這最近特朗普回應伊朗及北韓問題的影片。放心吧這影片很短的，只有幾十秒，花不了你多少時間。

https://www.youtube.com/watch?v=QBlrcUsMhps

此影片由Guardian Wires上載

認真讀書？不，我只對感興趣的事認真。 |谷哥點名#10

這次的谷哥點名的主題是認真，提起認真我就立刻想起不論小時候還是現在父母都再三叮囑我要認真讀書，聽得我耳朵都快長繭了。就連現在，我媽幾乎每天都想盡千方百計催促我讀碩士學位，我當然是把她的話當耳邊風。

其實我並不否定讀書的重要性，特別是掌握好語文能力、批判性思考和邏輯思維這些基本的能力，知識也是無價的，但香港也把那些名校、醫生律師等專業人士捧得太高了，好像你不立志入名牌大學，將來當這些專業就總顯得低人一等，沒有前途了。

每人也有自己的道路，認真讀書追求學問這條路也不一定適合每一個人，就像你要求畢加索去當醫生或要求愛因斯坦去當律師，這根本就是浪費人家的天賦吧？看見不少朋友都選擇繼續進修攻讀碩士博士，我常常在想這有必要嗎？你又不是立志當專業人士，要那麼高的學歷幹什麼？

「天下熙熙，皆為利來 ；天下壤壤，皆為利往」，我相信大部分的學生都是認為考取高分數就等於將來有一份高薪厚職的工作，對他們將來的「錢途」大有幫助，年輕一代更有不少人的理想就是當公務員，既穩定又舒服。他們並不是真的有興趣追求學問或知識，這在我眼中是愚昧的，因為我不認為當一個「打工仔」能使你發財。

就像 @linuslee0216在他之前的文章-赏月？其实谁又会在意呢裡所說的，他有些中學同學讀書成績十分爛，最後卻擁有其他同學難以企及的財富，在這世界要成功絕對不是只有認真讀書這條方程式，而我想真正能致富或實現財務自由的人都不是單靠讀書或一份好工作的，必然有自己創業、製造現金流或者投資的門路，而偏偏現今讀書方式就只會令你墨守成規，為你的思想打上框框，把創意都扼殺在搖籃裡，我猜這也是無論政府有多大力推動，香港的創意產業也停濟不前的原因。



我是個好動、喜歡新事物的人，我不想當一個隨波逐流的人，成為社會上可有可無的一塊小齒輪，要我整天坐著做著大公司公式化的工作，每天加班工作，乾脆就殺了我算了。慶幸在幾年前開始對加密貨幣和投資感興趣，就自己潛心鑽研，現在也算是略有小成，能為自己帶來一份不錯的額外收入。

回想起當年考中學文憑試的時候我已是這樣了，都快考公開試了我還飛到北海道去玩，我猜能這樣不把公開試當一回事的人應該不多吧，但我就是這樣的了，我無法忍受身邊的事物或人給我定下枷鎖，強迫我去作什麼。慶幸自己也有點小聰明，即使沒把多少時間放在讀書上最後也能順利入讀大學，順利畢業。

雖然讀書不太認真，但我在學習有用或我感興趣的東西時我可是很認真的呢，像當初學習倉頡輸入法、 鉛筆素描、日語、 技術分析、 投資工具和比特幣等加密貨幣都能連續花上好幾個小時孜孜不倦地學習和練習呢，認真學習不同的知識和技能可能會在將來大派用場呢。像現在我最認真在做的事就是Steemit了，早在我得到這份代理SP前已每天都花了不少時間在這上面，也從來沒有Power down，為什麽？因為我看到一個像比特幣一般的機遇，我知道現在不認真、 不努力一下的話往後是絕對會後悔的！

時光飛逝，把你的認真放在你真正感興趣的事物，至少你往後不會追悔莫及，而且出來的結果可能會比你所想的還好呢。認真做好每件你感興趣或認為有意義的事吧。

Monero的基本介紹及背景資料整理

定位：匿名加密貨幣
總量: 約1800萬，達到後每年以低於1%的通脹率通脹
誕生年份: 2014
共識協議：工作量證明機制（POW算法）
算法：CryptoNight
競爭對手：Zcash, Dash

Monero，中文譯名為門羅幣，Monero這名字來自於世界語的「錢幣」。和比特幣不同，Monero的賣點在於保護隱私。在區塊鏈上你無法找到任何地址和交易金額，更無法透過地址查看地址餘額和交易紀錄。


Source

Bytecoin是第一個使用CryptoNote的虛擬貨幣，也是所有基於CryptoNote協議的加密貨幣的始祖。

基於CryptoNote的貨幣: Bytecoin, Quazarcoin, Aeon, Monero, BoolBerry, FantomCoin, DuckNote, MonetaVerde

在Monero中，每一個地址都是95字元的字串，由 public view key 與 public spend key 組成。 在交易時,另一方會使用你的public view key 與 public spend key 再加上一些隨機資料來產生一條獨特的一次性公鑰，所有人都能夠在區塊鏈上看見一次性公鑰，但只有接收者及得到密匙的人能看到完整交易信息，而接收者最後可透過他的private spend key 花費掉這些交易輸出。

Monero採用了環狀簽名(RingCT)，環狀簽名是用來混淆交易輸出的來源的,它的原理是藉由混合一群簽署者的簽名來製造一份無法分辨來源的簽名，從而授權一份交易。這也是Monero的交易都難以被追緃的原因。

Monero的匿名性說不定會令它在暗網上受到犯罪分子的歡迎，全球最大的在線毒品交易市場AlphaBay就在去年宣佈支持Monero交易。

優點:

  • 匿名性
    在Monero的系統中，交易的來源、目標或被交易的金額都是無法被追蹤的，從而達到官方宣傳標語－「安全，隱私，無可追蹤」的效果。

  • 去中心化
    Monero所採用的CryptoNight算法有著抗ASIC的特性，減低專業礦機和礦場對整個網絡的襲斷性，普通用戶用家用電腦也能參與挖礦。

買賣平台: Poloniex, Bittrex, HitBTC, BTer, Shapeshift, Coin-Swap

隨著加密貨幣的普及，交易的隱私性說不定會更被重視呢，那時人們可能就會開始關注門羅幣了。

順道宣傳一下中文區加密貨幣研究社，鼓勵大家多使用 #cn-cryptocurrency標簽，希望大家多些翻譯和寫作關於幣界的資訊，為中文區的大家提供更多虛擬貨幣資訊。

HotFoodHK #16 COW BEAN

在中秋節的晚上，我們到了這間COW BEAN用餐。這是一間以牛牛和豆豆為主打的親子餐廳，環境舒適，食物走西日路線，標榜健康，以豆子作主題，眾多菜式會以十多款豆子如鷹咀豆、黑豆、 紅腰豆等作配菜。這些豆有不同功效，像紅腰豆就可以降低膽固醇，幫助控制血糖水平。店內裝潢用了綠色植物、牛奶盒似的屋狀設計，客人就像在農莊裡享用美食一般。

店外擺放了自家培製的咖啡豆

在享用主菜前先嚐嚐這個焗蒜香脆包，外表特別，還預先切好了，外脆內軟，配著牛油和大蒜的香濃滋味，好吃得令人停不了口。

餐湯，看上去不錯呢，配上豆子，十分健康。

小食拼盤，有兩隻烤雞翼、兩隻日式吉列炸蠔及兩塊薯餅，口感酥脆，十分可口。

輕煎白鱈魚柳 配鮮菌忌廉汁，既厚身又嫩滑的鱈魚柳質感白嫩，味道不俗，配上各色各樣的豆子，這樣的口感組合更加有層次及惹味。

火山岩燒紐西蘭西冷牛扒 配燒汁，牛扒份量十足，表層帶點焦脆，咬下去口感鬆軟，十分美味。

炸雞蕃茄忌廉汁燴飯，味道不俗。

輕煎帶子黑松露龍蝦汁意粉，意大利麵的軟硬度適中，醬汁很香，還有2粒大帶子在上面呢。

地址：紅磡黃埔新天地第5及6期地庫AEON Style B03號舖
