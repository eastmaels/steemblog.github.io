比特幣在這兩日內會派五顆糖果！

Source

這兩天比特幣在經歷了之前的暴跌後又重拾升勢了，看來是和接下來的一大堆免費糖果有關，其中包括了Bitcoin File、Bitcoin God、SegWit2x、Quantum Bitcoin和Bitcoin Cash Plus，而其中Bitcoin God和Bitcoin File就在剛剛已經進行快照了。

Source

在這眾多的分叉幣中還包括了在之前取消了、鼎鼎大名的SegWit2x，實在值得期待呢！據說現在B2X的期貨價格已來到900多美元了。

Neblio的基本介紹及背景資料整理

Source

定位：為企業和大型機構提供區塊鏈解決方案的技術平台

總量：約1300萬

儘管現在越來越多區塊鏈項目和技術如雨後春筍般出現，現在還是不多傳統企業開發和部署區塊鏈應用。Neblio就是一個針對企業設計的區塊鏈平台，Neblio非常注重降低企業用戶的開發門檻，Neblio平台計劃為開發者準備好一套RESTful API，這能有效解決企業面對的開發難度和缺乏工具包等障礙。開發者能直觀地使用現今主流的編程語言來進行開發。

Neblio還會為客戶提供諮詢服務，幫助他們利用Neblio技術架構來滿足他們的商業需求。

支持語言：Java, Python, Node.Js, Go, PHP, Objective-C, .NET

共識協議：股權證明機制（POS算法）

買賣平台：Binance, Cryptopia, Hitbtc

競爭對手：Stratis

Stratis的Strat幣價最近的表現也相當不錯，有著顯著的升幅，說到為企業提供區塊鏈解決方案，已經在這方面有著一定探索和社區支持的Stratis會是Neblio的最大競敵。你也可以看看我之前介紹Stratis的文章。

FunFair的基本介紹及背景資料整理

Source

定位：去中心化的博彩遊戲平台

總量：約100億

簡單來說，FunFair這個項目就是建立在以太坊智能合約上的賭場，團隊希望利用區塊鏈技術撘建一個安全、透明和公平的賭場，享受遊戲的樂趣，而它的FUN代幣將會被用作遊戲的籌碼，賭場許可證持有人會以Fun代幣來資助其經營，遊戲開發商也會得到Fun代幣作為報酬。

一般人都不太相信在線博彩，認為它們不可能提供公平的機會，網站也有機會被黑掉或有提款的困難，而這些常見的問題都能透過區塊鏈技術解決掉。

買賣平台： Binance

FunFair平台不僅接受以太幣，還會接受ZCash、比特幣及其他ERC20代幣，十分方便，說不定真能吸引到來自世界各地的一大堆賭客來玩呢！

Decentraland的基本介紹及背景資料整理

Source

總量：約28億

簡單來說Decentraland就是區塊鏈上的The Sims或Minecraft，是建立在以太坊上的虛擬現實平台。它的代幣叫作MANA，在這個虛擬世界裡，領地由直角坐標(x,y)來劃定，用戶能通過基於區塊鏈上的土地賬本證明他們的所有權 ，他們需要通過MANA代幣來購買土地、股務、產品應用，而內容提供者則能獲得消費者所消費的MANA代幣，你看之前以太坊的那個CryptoKitties都能弄得這麽成功，都要花個百萬來買一隻虛擬貓了，Decentraland的概念可是更有趣得多呀。

未來Decentraland說不定還能應用到虛擬實境(VR)、擴增實境(AR)和混合實境(MR)等技術，真正創造出一個能與現實互動的、區塊鏈上的虛擬世界呢。
