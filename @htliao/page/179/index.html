比特幣一如所料地暴跌，是時候買入嗎？

Source

9日前，我寫了個帖子隨筆：淺談投資，為什麽我把手上的比特幣全放了，這才短短九天，都應驗了，我們來比較看看現實與我的預測：

ⓐ浪底部是在3900多， ⓑ浪的頂部是在4700多，而現在的ⓒ浪底部還不知道出現了沒有，走勢和轉角位和預測的都十分接近。這次看得頗準確的，令我也不禁沾沾自喜。希望之前有看過我文章的有作出相應的決定吧。

這次的暴跌是非常合理的，也沒什麼好驚訝，在這兩年比特幣由無人問津，到很多我身邊的朋友都來問我怎樣買時，我就知道這次的暴跌肯定會發生，把來湊熱鬧的投機者震走，把他們的信心全摧毀才會開始下一個升浪。

我不是先知，更不是上帝，從現在起我也沒太大把握往後的趨勢會是怎樣，沒有人能準確預測浪底在哪裡，但反正我是從幾百美元開始買的，現在也應該開始分段進場了，不長線拿著也該進場博一博反彈，就算再跌下去，也多了幾個幣，沒差。

看一下圖表，情況再壞一點的話可能會跌至2500美元左右的，但我不太想撈底，反正最後還是會升的。

你問我有買嗎？那肯定是有的！

去中心化交易所會是未來的趨勢嗎？

看完 @tumutanzi的中国关闭交易所，比特币的世界末日来了吗？，有一個觀點挺贊同的，就是比特幣不會因交易所的問題而倒，而交易轉向在像比特股這樣的去中心化交易所進行也可能是未來的趨勢，比特股社區在中國的規模本來就不小，隨著更多交易所的關閉，去中心化交易所顯然是一個不錯的選擇，說不定會令比特股社區人數大幅增加。如果你連比特股是什麽也不知道的話，你可以看一下我上個月寫的這個帖子，了解一下這個同樣是由Dan所開發的比特股的背景和理念。

香港的年青人都不懂投資嗎？

前天由於公司的安排，參加了一個由香港青年協會和錢家有道舉辦的關於投資理財的工作坊，不用上班一天，不錯呢。

那個工作坊旨在通過一些桌上遊戲、影片和分享來令參加者認識到投資理財的重要性，他們還請來了香港挺受歡迎的YouTuber達哥為他們拍影片呢。

https://www.youtube.com/watch?v=pl_wPqzD6d0&list=PLOBVjsktn0CxQLsEVjmDP5yM6FpEU6ifY&index=2

上載者: undergroundDV

這次工作坊的對象就是我們這些剛畢業不久的員工，在大家分享自己每月怎花自己的薪水時，我發現很多人的儲蓄習慣只是把部份的薪水存在銀行，或根本是月光族（每月都把薪水花光），連股票也沒投資，更不用說像期貨、ETF、期權、虛擬貨幣等更進階的投資工具了，他們甚至連運用定期存款賺多一點少少的利息也沒做。

這令我很驚訝，心想；難道香港的年輕一代都不懂投資嗎？因為我認為這是十分不明智的，貨幣只會不斷貶值，我認為因為害怕風險而不去投資就跟認輸一樣，是穩賠的，就像把錢包敞開，任由老奸巨滑的銀行家和通脹悄悄地偷走你的金錢一樣。

相信沒有人會否認金錢的重要性吧？不然你馬上就可以辭去你的工作，去環遊世界了。可是在我看來很多人對金錢的觀念就跟在念幼稚園沒分別，連基本的投資概念也沒有，卻不打算自己好好學習。在大學，他們會用心學習那些出來工作後在大部份情況下根本就用不著的知識，為了應付考試，得到好的成績甚至可以通宵溫習。可是同一時間，卻對真正能幫助自己增長財富的投資和關於金錢方面的知識興致缺缺。好了，學校裡沒有教，你不會自己去學習嗎，谷歌一下有什麼知識是不能自己學習的？

Source

我一向認為，靠著「打工」所得來的微薄薪水，你只能成為社會上被剝削的一群，砧板上待宰的魚，能夠勉強應付生活，有空去去旅行就算好的了，更不用說在寸金尺土的香港買到房子了，不努力學習投資，最後老了可能就只能露宿街頭 LOL ，所以投資是一門必要的學問。

Source

瑞士格林德瓦-號稱世上最美村莊的風景

去年年尾，我到了瑞士旅遊， 最令我印象深刻的必定是瑞士的格林德瓦小鎮，想必 @liflorence應該很熟悉吧。 xD 那裡的景色真的很漂亮，既貼近大自然，也沒有半點落後的感覺。它是少女峰的前哨站，是一條蘊藏著無限魅力、風景秀麗的山谷小鎮，與蘇黎世帶給人的感覺完全不同。

剛來到格林德瓦，就看見很多滑雪愛好者帶著他們的全副裝備呢，滑雪果然是瑞士最受歡迎的活動。

整個格林德瓦被四周的雪山山峰包圍，在翠綠的山丘上，散佈著瑞士小屋，仿佛是在童話故事裡才有的城鎮，擁有著一片再看也看不膩的風景。在這個小鎮裡，到處都能看到美麗的風景，難怪被人稱為世界上最美麗的村莊。

這裡環境清幽，讓人感到十分平靜和放鬆，和香港充滿著壓迫感的生活節奏很不同。

山坡下的一座可愛教堂。

我們就是住在類似這種傳統的瑞士小旅館了，據說有百多年的歷史呢。老閱娘超親切和貼心的，把火車的班次都預先告訴我們了，還親自駕車載了我們去火車站。

格林德瓦附近山上有個非常有名的景點叫First，你需要乘坐纜車才能上去，那裡還有很多超好玩的遊玩設施呢，包括滑雪、山景卡丁車，滑翔傘、高空飛索等。我玩了滑翔傘，在高空俯瞰著下方迷人的風景簡直是人生一大享受呢！教練還拿著他的自拍神器為我拍了很多照片，當然付費的時候我就哭了。

這片景色真是令人百看不厭。

有時間的話，不妨到附近的茵特拉根小鎮看看，那裡也有著令人著迷的湖光山色呢！

看完我的簡短介紹後，希望你們喜歡這些瑞士格林德瓦周圍的風景吧，這個小鎮絕對是值得一去的地方。
