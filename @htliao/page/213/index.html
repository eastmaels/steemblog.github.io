Bibox平台上調BIX鼓勵金發放比例

Source

Bibox平台最近上調了BIX鼓勵金發放比例，每個星期會跟據用戶持有BIX代幣的數量給平台上的BIX持有者發放30%平台交易費的奬勵，奬勵於每星期五大約17:00左右的時間發放。

我也買了一點，也在昨天收到了這星期的奬勵。

如果還沒有戶口的話，歡迎使用我的推薦連結註冊啊！

像過山車般的Verge

Source

Verge在2日前宣佈與成人網站Pornhub合作前後上演過山車般的行情，價格在消息宣佈前一路暴漲，之後轉身一個急插，在低點足足跌了差不多一半。

看見短短2日就半價促銷的機會，當然不能錯過，不長線持有也要捕捉一個像樣的反彈才對嘛。

Verge的流動性和成交一向很大，非常適合短炒，上午和中午分注買了一點，最後幸好也有少許斬獲，賺了一點，其實下午跌得我都有點怕了。

想學習交易的朋友不妨了解一下Fibonacci retracement，找出價格有大機會大幅反彈的位置，十分有用呢！

Red Pulse的基本介紹及背景資料整理


Source

總量：約13億5000萬
代幣符號：RPX


Source

昨日介紹過Qlink，這次就介紹同為NEO項目的Red Pulse。Red Pulse是基於NEO的共享研究平台，被譽為中國市場情報與內容的下一代生態系統。說起來Red Pulse還是NEO的第一隻ICO！

Red Pulse簡單來說就是區塊鏈版的雪球網，說起來和Steemit還有點像，好像也是根據voting power來upvote別人的帖子和研究報告。這能直接給予分析師和供稿人可觀的報酬，他們的目標是顛覆現有的金融研究市場形態，解決現有平台準確性低、信息過載和透明度低等問題，在早期他們會專注於中國市場方面。

現在已經能註冊了呢，可惜我對這些方面不太熟悉，各位金融才俊可以來試一試。

買賣平台：Binance, Kucoin, Huobi

他們還有IOS的App可供下載。

4月18日比特幣價格即市分析

在上一次的即市分析中給出了8400-8500的目標，果然升到這個位置左右價格就掉頭向下（做槓桿交易的記得先掛單哦！），在前日的文章裡也寫到比特幣的價格大概會先調整再漲，有機會跌至7900-8000美元左右，但週線的早晨之星形態暗示長遠的價格走勢向好。

在今日清晨比特幣的價格跌至7850左右，看似比特幣完成了abc浪調整，現在應該要開始走五浪上升了，就在不久前價格突然急速拉升，漲到了8150美元左右，未來，有機會微跌一下再向上拉升呢！交易的話靜候做多的機會就可以了，9000美元關口應該不遠了！相信在五月尾前我們會再看到10000美元以上的比特幣。

交易涉及風險，以上純粹為個人看法，不構成任何投資或投機的建議哦。
