AppCoins的基本介紹及背景資料整理

Source

總量：四億五千萬
代幣符號：APPC

Source

AppCoins由移動應用程序商店Aptoide開發，顧名思義，AppCoins當然是用在智能電話的應用和應用商店裡。AppCoins能被開發人員用來宣傳他們的遊戲或應用，來換取用戶的注意力，也能被用戶用來購買應用或遊戲內的付費項目。

Source

一般在開發人員和用戶之間有幾個中介，假設開發人員想用一些奬勵來推廣他們的遊戲或應用，經過多重的中介後用戶往往就只能收到很小的一部分，而AppCoins正正能直接接通開發者和用戶，不需要中間的中介，令雙方的利益達致最大化。

Aptoide有著多達800,000個獨特程序和二億用戶，據說是全球排名第三的應用程序商店，但我可沒聽說過，可能是因為我是蘋果的用家吧。AppCoins當然不會錯過這個賣點了，聲稱他們是「第一個為二億用戶服務的ICO」，夠引人注目了吧?

Source

在AppCoins背後有著多間著名投資機構，而且AppCoins和去中心化交易所Kyber Network還是合作夥伴的關係呢。

買賣平台：Binance, Huobi
